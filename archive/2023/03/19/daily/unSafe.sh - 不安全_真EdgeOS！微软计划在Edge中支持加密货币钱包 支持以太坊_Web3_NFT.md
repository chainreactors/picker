微软不停地向 Microsoft Edge 浏览器里塞各种功能，最新要被塞进去的新功能是加密货币钱包，注意这里是 Microsoft Edge 原生支持加密货币钱包，并不是微软通过扩展程序的方式提供支持。

从网友发布的截图来看，Microsoft Edge 内置的加密货币钱包主要支持以太坊区块网络，这个倒是可以理解，毕竟现在最流行的加密货币应用都是基于以太坊的。

获得支持的包括但不限于以太坊加密钱包、DAI、NFT 以及微软提到的关键词：Web3。对加密货币社区来说微软是重量级嘉宾，微软通过 Microsoft Edge 浏览器支持加密货币和 Web3，按理说应该会对 Web3 的发展带来积极影响。

话说回来咱也不知道微软是不是觉得既然都做以太坊区块网络了，不支持 NFT 是不是不太好，NFT 的热潮已经过去，没想到 Microsoft Edge 加密货币钱包里还单独提到支持 NFT。

功能方面：Microsoft Edge 的加密货币钱包提供完整的钱包功能，包括但不限于创建钱包、密码、发送、接受以太坊和基于 ETH 的代币，甚至还支持 SWAP。

稳定币方面支持 USDT 和 USDC，当然都是基于以太坊网络的不支持 TRC20 即波场网络，发送以太坊时也支持以太坊的专用域名 ENS，不得不说微软考虑的还是挺周全的，还没上线提供的功能就这么全面了。

另外对于受支持的国家或地区，微软还提供加密货币购买服务，通过 Coinbase 和 MoonPay，这些目前主要支持美国等少部分国家。

目前尚不清楚微软何时会正式推出 Microsoft Edge 加密货币钱包，不过可以预见未来 EdgeOS 会越来越臃肿，直接学谷歌搞个类 ChromeOS 的 EdgeOS 估计也不差。
