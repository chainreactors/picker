# 8个岗位快来看看，云安全实习生校招进行中

阿里安全响应中心

阿里云安全春季实习生招聘已开启，请关注的同学尽快投递简历，毕业起止时间：

2023-11-01至2024-10-31

```
岗位列表：一、产品安全工程师二、漏洞运营工程师三、安全研发工程师四、Web安全工程师五、系统安全工程师六、安全工程师 - 安全技术能力建设/运营方向七、安全工程师 - 专有云Web安全方向八、安全工程师 - 专有云系统安全方向
```

```
一、产品安全工程师岗位描述： 深度参与云产品SecDevOps流程建设，保障阿里云平台&云产品安全，打造最安全的云计算产品。岗位职责：1. 负责阿里云核心云产品的安全架构设计、安全审计与安全测试；2. 针对云产品&开源组件进行漏洞挖掘；3. 对业界新出的漏洞进行跟踪分析和应急响应，设计修复和缓解方案，推动云产品和云平台及时修复。岗位要求：1. 熟悉常见的Web安全漏洞的原理以及对应的防范、修复方法，包括XSS、SQL注入、SSRF、业务逻辑漏洞等OWASP TOP 10安全风险；2. 具有良好的沟通、团队协作能力；3. 熟悉代码审计，有开源框架漏洞挖掘经验者优先，熟悉容器、k8s安全技术优先；4. 在各大SRC平台上有提交过漏洞、知名社区发布过文章优先；5. 参加过CTF比赛并取得名次者优先。简历投递：yunye.ry@alibaba-inc.com
```

```
二、漏洞挖掘工程师 岗位描述： 基于历史的安全风险，进一步反向验证企业的安全水位。岗位职责：1. 负责漏洞运营工作，对各类漏洞进行分析评估、复现利用，并与业务沟通相关风险；2. 关注市面主流软件的通用漏洞，研究企业关注的组件风险；3. 以黑客视角对企业资产进行攻击模拟演练，举一反三发现更多风险。岗位要求：1. 对安全有浓厚的兴趣，有正确的安全职业价值观，喜欢探索；2. 精通常见的Web安全漏洞的原理及测试过程，有Java、Python、PHP任一方面漏洞挖掘经验；3. 熟悉Java、Python、PHP等任一编程语言，并有工具开发经验；4. 加分：熟悉CVSS漏洞评级体系，有漏洞的标准化分类分级管理经验优先；5. 加分：对通用漏洞（CVE、CNVD等）领域关注的同学优先；6. 加分：在各大SRC平台上有提交过漏洞、在知名社区发布过文章、在大型攻防演练项目中有实践经验者优先。简历投递：xianxian.xxx@alibaba-inc.com
```

```
三、安全研发工程师岗位职责： 负责云平台安全防御、检测、运营、资产与风控平台等中台能力研发与能力设计， 挑战公有云、混合云等复杂场景下的安全能力建设， 解决高并发、大数据处理平台及高性能、高稳定性安全中间件与基础设施等技术难题， 从而构建完整的零信任安全中台能力体系。岗位要求1. 有扎实的JAVA/Golang基础，熟悉常见设计模式，有良好的编码习惯，注重代码质量与编码风格；2. 熟悉MySQL、Redis等数据库的应用及原理，拥有良好的算法、数据结构、操作系统、网络通信等计算机专业基础能力；3. 良好的沟通能力与团队精神，追求卓越，关注主流技术发展趋势，乐于分享。优先和加分项（其一）：1. 对安全攻防有一定的认识，参与ctf比赛并获得优异成绩；2. 有较丰富的安全、云原生或中间件项目研发经验；3. 了解编译原理，对缺陷静态分析技术有深入理解与实践；4. 了解虚拟化技术、虚拟化安全，对云原生技术有一定理解和实践。简历投递：renyi.cry@alibaba-inc.com
```

```
四、Web安全工程师 岗位要求： 负责阿里云产品和云平台的的安全漏洞发现并提供修复解决方法。岗位职责：1. 熟知Web安全，了解当前流行的Web漏洞（XSS, SQL Injection, CSRF, etc）2. 了解Java开发框架（Struts, Spring, iBATIS, Hibernate, etc）和常见的Java安全漏洞；3. 具备Web安全攻防的经验；4. 熟识计算机网络，至少掌握Perl/Python/Shell一种脚本语言；5. 有CTF比赛经验或者CVE漏洞提交者优先；6. 学习能力强，对新事物保有好奇心，并能快速适应新环境；良好的沟通能力和团队协同能力；   能与他人合作，共同完成目标；对所在领域有热情，相信方法总比困难多，善于独立思考并反思总结。简历投递至shengping.wsp@alibaba-inc.com
```

```
五、系统安全工程师 岗位要求： 负责阿里云产品和云平台的的安全漏洞发现并提供修复解决方法。岗位职责：1. 熟知操作系统安全，了解主流操作系统之上的二进制组件漏洞或容器K8S系统漏洞原理；2. 具备系统安全安全攻防的经验，具备漏洞的EXP、POC能力如Linux提权，RCE等；3. 熟识计算机网络，熟悉C/C++、Golang至少一种语言的常见安全漏洞原理；4. 至少掌握Perl/Python/Shell一种脚本语言； 5. 有CTF比赛经验或者CVE漏洞提交者优先，有过顶会Paper者优先，有协议漏洞经验者优先；6. 学习能力强，对新事物保有好奇心，并能快速适应新环境；良好的沟通能力和团队协同能力；   能与他人合作，共同完成目标；对所在领域有热情，相信方法总比困难多，善于独立思考并反思总结。简历投递至shengping.wsp@alibaba-inc.com
```

```
六、安全工程师 - 安全技术能力建设/运营方向 岗位描述： 参与阿里云安全检测能力与安全防御能力的建设，提升安全检测与防御的效果。 参与前沿安全技术预研与创新，并探索新安全技术在阿里云内部落地。岗位要求：1. 熟悉掌握WEB常见安全漏洞原理、检测方法、利用手段及对应修复方案；2. 熟悉常用的风险发现、防御工具的原理，包括黑白灰盒扫描器、WAF、RASP、SCA等；3. 具有简单的研发能力、SQL能力，通过脚本自动化完成日常告警、扫描的自动化运营工作；4. 具有良好的沟通、团队协作能力；5. 在各大SRC平台上有提交过漏洞、在知名社区发布过文章者优先；6. 有CTF经验或在护网项目中有攻防经验者优先；7. 具备Java或Go语言代码审计能力，有实际漏洞挖掘的经验者优先。简历投递：yukai.hyk@alibaba-inc.com
```

```
七、安全工程师-Web安全方向岗位描述： 负责阿里云专有云平台产品的SDL工作，包含应用的黑白盒工具检测能力、人工黑白盒测试， 以及云产品的安全架构评审、云产品安全风险测试和解决方案。岗位要求：1. 熟悉常见的Web安全漏洞、中间件等安全漏洞的原理、挖掘方法和修复方案；2. 了解常见源代码审计工具，如Checkmarx、Fortify SCA、CodeQL等；3. 有CTF比赛经验或者CVE漏洞提交者优先，有开源框架漏洞挖掘经验者优先；4. 有黑盒、白盒扫描工具开发和规则编写经验者优先；5. 熟练使用Golang/python/shell至少一种语言。简历投递至xiaowei.wuxiaowei@alibaba-inc.com
```

```
八、安全工程师-系统安全方向岗位描述： 负责阿里云专有云平台的系统、网络、架构安全风险的发现， 安全检测能力的实现，安全风险解决方案的制定和落地。岗位要求：1. 熟悉linux操作系统安全相关技术，了解linux常见组件漏洞或容器K8S系统漏洞原理；2. 熟悉容器docker，kubernetes的网络配置、安全机制及相关配置；3. 熟悉常见的Web漏洞原理及修复、防御加固方案；4. 熟练使用Golang/python/shell至少一种语言；5. 有CTF比赛经验或者CVE漏洞提交者优先。简历投递至xiaowei.wuxiaowei@alibaba-inc.com
```
