### 1、右键搜

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-d315311f9d7a543b68756faf58541a6b.png)](https://lrting.top/wp-content/uploads/2022/07/frc-d315311f9d7a543b68756faf58541a6b.png)

* 不知道有多少人像我一样想把在网页中的右键生成一个万能菜单，想跳转到哪里就去哪里，那你一定要试试这款插件。
* 安装这款插件后，他自带了很多菜单配置。

+ 可以划词后直接右键选择就可以直达搜索引擎搜索。
+ 在页面中右键可以直接跳转到任何你想去的网页。
+ 可以右键图片创建图片短网址、图片生成二维码。

拿我自己举例我习惯性将`github`,`掘金`等链接添加到这个右键菜单中，这样就不用我去地址栏查找或书签栏翻阅就可以直接跳转。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-afccb19425bda3445b0439bd170db5e6.png)](https://lrting.top/wp-content/uploads/2022/07/frc-afccb19425bda3445b0439bd170db5e6.png)

### 2、Dark Reader

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-1c2d76aea4578ae6a84751fc52584f0e.png)](https://lrting.top/wp-content/uploads/2022/07/frc-1c2d76aea4578ae6a84751fc52584f0e.png)

* 作为一个暗黑模式爱好者在浏览一个网站的时候不支持跟随系统模式切换是不是有点难受？不用担心，这款插件可以让你在不同的网页有属于你自己的**暗黑模式**。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-42fdad94b6ab76460c0dc5a4b6f460ac.png)](https://lrting.top/wp-content/uploads/2022/07/frc-42fdad94b6ab76460c0dc5a4b6f460ac.png)

* 不仅仅可以记住你所浏览的网页中所设置的**亮度、对比度、滤镜、灰度**，甚至还可以改变当前**网页字体的样式**，去试试吧～会有不同的感觉喔。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-bc58df1708f33ef299c31bcbc3cf9bec.png)](https://lrting.top/wp-content/uploads/2022/07/frc-bc58df1708f33ef299c31bcbc3cf9bec.png)

### 3、AdBlock

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-f94a96798625a1d74b25069f1d2bf4ab.png)](https://lrting.top/wp-content/uploads/2022/07/frc-f94a96798625a1d74b25069f1d2bf4ab.png)

* 这款插件可以让你在`YouTube`、`Facebook`、`Twitch`和其他你任意喜爱的网站上拦截广告和弹窗。
* 我们在浏览一些博客论坛或者谷歌、百度搜索的时候难免会有一些不太舒服的广告，这时候我们只需要开启这个插件就可以让他们统统消失。

### 4、SuperCopy

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-8992d7c31f7c4bc38fb5636c57a3204c.png)](https://lrting.top/wp-content/uploads/2022/07/frc-8992d7c31f7c4bc38fb5636c57a3204c.png)

* 大家有没有遇见一种很恶心的情况就是当你在某网址翻阅文献的时候想复制里面的资料却弹出来一个需要开通会员或者付费才能复制的弹窗，想当年写论文的时候头皮发麻。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-a688bedc510c1f374a93c99b1ecc2498.png)](https://lrting.top/wp-content/uploads/2022/07/frc-a688bedc510c1f374a93c99b1ecc2498.png)

* 这种时候这个插件就派上他的用场了，只需要我们点击右上角的插件小图标，页面中就可以复制你想要复制的任意资料。

### 5、沙拉查词

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-fc68deb2f8feffa056567da7c747e436.png)](https://lrting.top/wp-content/uploads/2022/07/frc-fc68deb2f8feffa056567da7c747e436.png)

* `Saladict` 沙拉查词是一款专业划词翻译扩展，为交叉阅读而生。大量权威词典涵盖中英日韩法德西语，支持复杂的划词操作、网页翻译、生词本与 `PDF` 浏览。
* 只要在网页中用鼠标划动词语就会出来一个小沙拉杯的图标，当我们鼠标`hover`上去的时候就会出现这个词的所有不同平台的翻译，甚至连造句都有。
* 你可以把这次的翻译添加到生词本或是钉在屏幕中方便查看与学习。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-d87c863f5fa5c5b33d5bdb8d07231130.png)](https://lrting.top/wp-content/uploads/2022/07/frc-d87c863f5fa5c5b33d5bdb8d07231130.png)

### 6、JSON Formatter

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-c0a84287e342ee5b24a7228eabfa78d8.png)](https://lrting.top/wp-content/uploads/2022/07/frc-c0a84287e342ee5b24a7228eabfa78d8.png)

* 这款插件应该就不用我多说了吧，相信每个前端都会有一款 `Json` 格式化插件，这个插件用起来也很简单，只需要添加到拓展程序即可，当我们下次打开一个`json`文件的时候就会自动帮我们格式化完成。

### 7、Octotree

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-2c7ab356c7c3dd1a54638bd86d60a6ff.png)](https://lrting.top/wp-content/uploads/2022/07/frc-2c7ab356c7c3dd1a54638bd86d60a6ff.png)

* 作为一个程序员，每天看`GitHub`已经成了一种习惯，使用这个插件可以将`GitHub`仓库中的文件自动展开目录呈现在左侧，方便查看。
* 当我们点击目录树中的文件后，页面也会自动跳抓到对应的文件，再也不用看源码时打开在线`ide`或者来回点返回了。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-f8d0873e0b37b249a52f4cc92920befa.png)](https://lrting.top/wp-content/uploads/2022/07/frc-f8d0873e0b37b249a52f4cc92920befa.png)

### 8、GitHub Hovercard

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-bda4ecf863a854943eca90ca4b85f096.png)](https://lrting.top/wp-content/uploads/2022/07/frc-bda4ecf863a854943eca90ca4b85f096.png)

* 如果你想在`GitHub`中快速了解到一个作者或者一个仓库甚至一个`issues`的信息的时候，你可以使用这款插件。
* 我们只需要将鼠标`hover`到作者名称、`issues`上就可以立马了解到对应的信息，这对于我这种经常翻`issues`的人来说简直是神器，可以马上筛选到我想要找的资料。

### 9、Markdown Web Clipper

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-bba8f01a0ee5037906972a2a5687cd5c.png)](https://lrting.top/wp-content/uploads/2022/07/frc-bba8f01a0ee5037906972a2a5687cd5c.png)

* 如果你是一个作者或者是一个号主，那么`Markdown`你肯定不陌生，在看到一篇好的文章想要转载或者记录到本地的时候，使用这个插件可以让你快速保存`Markdown`。
* 你只需要打开你想要下载的`Markdown`所在页面，点击一下这个插件的黑色小图标，就可以复制或者下载这篇文章啦～

### 10、ImageAssistant

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-b62d7a5e7bdf7f825eaab887471f2ff3.png)](https://lrting.top/wp-content/uploads/2022/07/frc-b62d7a5e7bdf7f825eaab887471f2ff3.png)

* 如果你跟我一样是个壁纸收藏控，是个经常需要收集图片的人，那么这款插件肯定也适合你。
* 只需要开启后在网页右键点击**提取本页所有图片**，那么这个网页的所有可以保存和不可保存的图片都会将一览无遗，然后你就可以选择合适的图片来保存。
* 不仅如此，还可以设置你想要的尺寸，会自动帮你筛选出符合尺寸大小的图片，甚至还可以批量保存。

[![推荐10个 Chrome 生产力插件（不看后悔系列）](https://lrting.top/wp-content/uploads/2022/07/frc-be4587cb86691a8742c11697f7150d91.png)](https://lrting.top/wp-content/uploads/2022/07/frc-be4587cb86691a8742c11697f7150d91.png)
