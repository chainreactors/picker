## How to use

LFI-FINDER is an open-source tool available on GitHub that focuses on detecting [Local File Inclusion](https://www.kitploit.com/search/label/Local%20File%20Inclusion "Local File Inclusion") (LFI) vulnerabilities. Local File Inclusion is a common security [vulnerability](https://www.kitploit.com/search/label/Vulnerability "vulnerability") that allows an attacker to include files from a web server into the output of a web application. This tool automates the process of identifying LFI [vulnerabilities](https://www.kitploit.com/search/label/vulnerabilities "vulnerabilities") by analyzing URLs and searching for specific patterns indicative of LFI. It can be a useful addition to a security professional's [toolkit](https://www.kitploit.com/search/label/Toolkit "toolkit") for detecting and addressing LFI vulnerabilities in web applications.

This tool works with geckodriver, search url for LFI Vuln and when get an root text on the screen, it notifies you of the successful payload.
