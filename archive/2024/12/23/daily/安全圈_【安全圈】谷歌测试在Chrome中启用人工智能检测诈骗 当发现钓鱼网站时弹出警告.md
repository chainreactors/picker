# 【安全圈】谷歌测试在Chrome中启用人工智能检测诈骗 当发现钓鱼网站时弹出警告

网络钓鱼

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGyljhwIGlibiakO7ePnPiaOItbvyicUucibGHQ1DVDiat3DpOQ7cxI2YloYdxOfxkp0nqUILzEx61nuCoJxPw/640?wx_fmt=png&from=appmsg)

目前谷歌在 Chrome 浏览器中部署的安全措施主要是谷歌安全浏览服务 (Google Safe Browsing)，该服务基于云端数据库识别和分析恶意网站、钓鱼网站和危险程序。

但安全浏览服务的缺陷在于数据并不能实时更新因此总会有漏网的钓鱼网站，基于这种考虑谷歌尝试在 Chrome 中部署人工智能用来识别诈骗。

在 Chrome Canary 版中谷歌新增实验性选项「客户端检测品牌和诈骗检测意图」，该功能使用人工智能来分析设备上的网页识别是否存在欺诈行为。

这个实验性选项在开启后暂时还不起作用，不过从描述来看通过人工智能进行欺诈检测应该还是有实际意义的，尤其是谷歌可能会通过现有的钓鱼网站数据库进行模型训练。

但这里还存在隐私问题那就是人工智能模型需要读取用户访问的每一个网站，从这方面来说或许后续该功能为可选开启而不是默认开启状态。

按照谷歌在 Chrome 中的惯例隐私设置，谷歌应该希望用户将人工智能模型的使用数据发送到服务器上供谷歌改善模型，到时候关注隐私的用户可以检查相关设置，根据自己的需要选择开启或禁用。
