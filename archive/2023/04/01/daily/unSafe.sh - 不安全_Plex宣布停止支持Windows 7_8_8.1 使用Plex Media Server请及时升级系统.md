媒体播放与服务器软件 Plex 日前在官方论坛透露后续发布的 Plex Media Server 将不再支持 Windows 7、Windows 8、Windows 8.1、macOS 10.11、macOS 10.12，如果用户仍然使用这些老旧版本的操作系统，那将无法升级自己的 Plex Media Server。

[![Plex宣布停止支持Windows 7/8/8.1 使用Plex Media Server请及时升级系统](https://img.lancdn.com/landian/2023/03/98096.png)](https://img.lancdn.com/landian/2023/03/98096.png)

具体来说 Plex 将在 4 月 10 日发布 Plex Media Server 1.32.0 版，自此版本开始不再支持上述提到的系统，好消息是旧版本的 Plex Media Server 仍然可以使用，坏消息是继续使用旧版本意味着不安全。

[之前 LastPass 被黑就是因为其核心工程师使用旧版 Plex Media Server](https://www.landiannews.com/archives/97690.html)，旧版本存在安全漏洞，黑客通过漏洞入侵了这名工程师的电脑，进而窃取了 LastPass 核心系统的密钥，引发巨大安全问题。

Plex 鼓励用户至少升级到 Windows 10 Version 1607 (周年更新版) 或 macOS 10.13 High Sierra，这样 Plex 可以继续发布关键安全更新提供支持。

话说回来，Windows 10 v1607 消费者版和企业版都已经停止支持无法获得安全更新，目前此版本的分支版本仍然提供支持的是 Windows 10 LTSB 2016 版，也就是说非 LTSB/LTSC 缺乏微软提供的更新，那还是不安全的。

[本周游戏平台 Steam 也宣布停止支持 Windows 7/8/8.1 了](https://www.landiannews.com/archives/98022.html)，所以仍然使用旧版本的用户还是尽早安排升级吧。
