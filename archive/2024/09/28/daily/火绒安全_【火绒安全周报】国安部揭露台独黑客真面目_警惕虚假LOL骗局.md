# 国安部揭露台独黑客真面目

#

近期，国家安全部揭露了一个名为“匿名者64”的黑客组织的真实面目。该组织自今年以来，频繁对大陆及港澳地区实施网络攻击，企图通过控制有关门户网站、户外电子屏幕和网络电视等传播平台，散布诋毁大陆政治制度和大政方针的谣言。经调查，“匿名者64”实为台湾资通电军下属机构，该机构专门负责对大陆实施网络认知战、舆论战。国安部指出，“匿名者64”所宣称的攻击成果多为虚构或夸大，而且其攻击网站多为虚假或废弃网站。目前，国安部已锁定涉案人员，并依法对其中三人立案侦查。国安部呼吁社会各界加强网络安全防范，及时向国家安全机关或有关部门举报违法活动，共同维护网络空间主权与国家安全。

# 黑客冒充谷歌人员窃取2.43亿美元

#

近期，有研究人员揭露了一起网络钓鱼攻击事件。有一黑客团伙通过冒充Google和Gemini支持人员，诱骗受害者重置双因素身份验证（2FA），成功窃取2.43亿美元加密货币。据悉，这些黑客因重复使用洗钱地址和在社交媒体平台发布的奢侈生活方式被追踪调查。目前，已有嫌疑人被捕，受害者的部分资金得以追回。美国司法部指控其中两名涉案人员共谋窃取加密货币和洗钱。该事件的调查仍在进行中。

# 警惕虚假英雄联盟骗局

#

随着英雄联盟2024全球总决赛的火热进行，一批网络犯罪分子开始借机传播Lumma Stealer恶意软件。据悉，犯罪分子通过社交媒体广告诱导玩家下载伪装成英雄联盟游戏的Lumma Stealer恶意软件，以窃取用户的敏感数据进行身份盗窃或网络钓鱼攻击。Bitdefender Labs报告称，目前已有约4000名欧洲游戏玩家受害，受害者多数为成年男性玩家。建议各位电竞爱好者下载游戏时认准官方渠道，切勿轻信网络广告，确保个人信息安全。

# 金融行业巨头遭受网络攻击致使服务中断

#

近日，金融行业巨头速汇金（MoneyGram）遭遇一起网络安全事件。该公司于本周一证实，其自上周五起发生的服务中断是系统遭遇网络攻击所致。速汇金称，他们在发现问题后迅速展开调查，并立即采取了相应的保护措施。该公司表示，他们会通过与外部网络专家和相关执法部门合作，应对此次攻击。本周四，速汇金在社交媒体上宣布其官方网站与应用程序均已恢复服务，并承诺将继续处理因服务中断而积压的交易。

# 迪士尼因数据泄露事件宣布弃用Slack平台

#

据报道，迪士尼公司受今年7月重大数据泄露事件的影响，决定逐步停止团队协作平台Slack的使用。今年7月，黑客组织NullBulge声称窃取了迪士尼内部Slack平台上的1.1TB数据，其中包括未发布项目、原始图像、代码等敏感信息。此前，Rhysida勒索软件组织也曾泄露迪士尼旗下Insomniac Games在Slack平台上的机密数据，被盗的数据涉及金刚狼、蜘蛛侠2等游戏的开发资料。CNBC指出，迪士尼目前正在迁移到更为简化的企业协作工具，并计划在下一个财政季度结束前，使大部分业务部门停止使用Slack平台。
