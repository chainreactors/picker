所属地 上海

## 全球动态

### 1.苹果：App调用部分API需给出详细理由

苹果公司宣布，从今年秋开始，开发者提交的App若需要调用某些API，需要在提交之前给出详细的调用理由，以确保开发人员不会滥用 API 进行用户的指纹识别。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/apple/apple-says-new-app-store-api-rules-will-limit-user-fingerprinting/)】

### 2.夏威夷社区学院向勒索软件团伙支付费用以防止数据泄露

夏威夷社区学院是一所经过认证的公立社区学院，在夏威夷岛上设有两个校区，是夏威夷大学 （UH） 的一部分，该大学拥有超过 50，000 名学生。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/hawaii-community-college-pays-ransomware-gang-to-prevent-data-leak/)】

### 3.澳大利亚和美国发布有关 Web 应用程序威胁的警告

澳大利亚网络安全中心 （ACSC）、美国网络安全和基础设施安全局 （CISA） 和美国国家安全局于2023年7月年27日发布了防止 Web 应用程序访问控制滥用指南。【外刊-[阅读原文](https://www.infosecurity-magazine.com/news/australia-us-warning-web-app/)】

### 4.欧盟《数据法》解析：非个人数据跨境流动的充分性保障

欧盟《数据法》已经非常接近通过。其中第27条专门规制非个人数据的跨境流动。很长一段时间内，第27条第1款表述为数据处理服务提供者应当采取“合理”（reasonable）措施，防止特定类型的非个人数据跨境流动。【[阅读原文](https://www.secrss.com/articles/57241)】

### 5.国内首个《云原生安全配置基线规范》标准正式发布

本标准中的要求主要包括对Kubernetes主节点API Server、控制管理器、调度器、etcd的安全配置要求，对工作节点kube-proxy、kubelet、工作负载的安全配置要求，以及针对附加项的CNI和网络策略配置的要求。【[阅读原文](https://www.secrss.com/articles/57210)】

### 6.美股上市公司重磅新规！重大网络安全事件需在4天内披露

美国证券交易委员会（SEC）于26日通过新规，要求上市公司在4天内披露所有可能影响其利润的网络安全违规事件。如果立即披露会带来严重危及国家安全或公共安全，可延迟披露。【[阅读原文](https://www.secrss.com/articles/57203)】

## 安全事件

### 1.全球宕机！以色列最大的炼油厂遭遇黑客攻击

以色列最大的炼油厂运营商BAZAN Group的网站遭遇黑客入侵，全球大范围宕机。BAZAN集团总部位于海法湾，前身为石油精炼厂有限公司，每年原油炼化规模约980万吨，年收入超135亿美元，员工超过1800人。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/israels-largest-oil-refinery-website-offline-after-ddos-attack/)】

### 2.谷歌： 安卓补丁漏洞让 "N-days "与 "0-days"同样危险

近日，谷歌发布了年度零日漏洞报告，展示了 2022 年的野外漏洞统计数据，并强调了 Android 平台中长期存在的问题，该问题在很长一段时间内提高了已披露漏洞的价值和使用。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/google-android-patch-gap-makes-n-days-as-dangerous-as-zero-days/)】

### 3.软件公司Ivanti披露安全漏洞，已被积极利用

Ivanti 披露了一个影响端点管理器移动版 （EPMM） 的新安全漏洞，跟踪为 CVE-2023-35081（CVSS 分数：7.8），该漏洞在野外被威胁参与者作为漏洞利用链的一部分加以利用。【外刊-[阅读原文](https://securityaffairs.com/148957/hacking/ivanti-epmm-flaw.html)】

### 4.CISA警告梭子鱼ESG攻击中使用的潜艇后门

美国网络安全和基础设施安全局（CISA）发布了有关恶意软件变种的警报，该变种被跟踪为SUBMARINE Backdoor，用于利用梭子鱼邮件安全网关（ESG）设备中的CVE-2023-2868漏洞进行攻击。【外刊-[阅读原文](https://securityaffairs.com/148942/malware/submarine-backdoor-barracuda-esg-attacks.html)】

### 5.STARK#MULE用美国军事主题的文件诱饵瞄准韩国人

正在进行的网络攻击活动通过使用美国军方主题的文档诱饵来诱骗他们在受感染的系统上运行恶意软件，从而将目光投向了讲韩语的个人。【外刊-[阅读原文](https://thehackernews.com/2023/07/starkmule-targets-koreans-with-us.html)】

### 6.Twitter 更名为“X”会触发边缘安全警报Microsoft

Microsoft Edge警告说，这是一个潜在的安全问题，而且它正在按预期工作。这与一项名为“渐进式Web应用程序图标更改”的安全功能有关，旨在提醒用户应用程序图标或名称更改，可能表明存在骗局。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/twitters-rebranding-to-x-triggers-microsoft-edge-security-alert/)】
