### 界面简洁、效率至上、体验优秀！

相比很多同类的工具，**He3 工具箱**的 UI 界面算得上是比较精美的，也足够的简洁舒适，同时它还提供了非常方便的快捷键，随时随地呼出界面，并支持“模糊搜索”，可以很快地找到你所需要的工具。

#### 一键呼出 He3 工具箱

默认情况下，按下 `Alt` + `空格` (Windows) 或 `Option` + `空格` (Mac) 即可立即调出 He3 工具箱。按下 `Ctrl`+`K` (Win) 或 `⌘`+`K` (Mac) 则可以快速搜索。(快捷键都可以在设置里修改)

#### 方便的模糊搜索 + 高效预览结果

即使分类清晰，但要在几百个工具中找到自己需要的并非容易的事。He3 工具箱提供了一个非常好用智能的“**模糊搜索**”功能，你只需输入一部分关键字，它就能智能地列出相关的小工具，帮助你用最快的速度解决问题。

![搜索工具](https://img.iplaysoft.com/wp-content/uploads/2023/he3/he3_search.jpg)

而且，很多工具还提供了搜索列表「预览结果」的功能，比如**查询 IP 地址**、**查看当前时间戳**、颜色生成器等工具时，它会直接在搜索列表里就显示出内容，你甚至都不必打开工具就能直接看到结果。

![搜索预览结果](https://img-dp.iplaysoft.com/dispatch/b303fc5de2831de3a76196ba3e674145.gif)

#### 剪贴板智能识别+自动填写

He3 还有一大亮点就是会智能识别[剪切板](https://www.iplaysoft.com/tag/%E5%89%AA%E8%B4%B4%E6%9D%BF)的内容。比方说，当你先复制一段 BASE64 编码的字符串，然后打开「**BASE64 解码**」工具时，它就会**自动读取剪贴板**里的内容填写进去帮你解码；当你拷贝了一串 JSON 代码后，打开 He3 它就会自动推荐 JSON 相关的一些工具等等。

![自动读取剪贴板](https://img-dp.iplaysoft.com/dispatch/95ad48a84b5d86559218b729609735e3.gif)

#### 自定义主页、拖拽调整排序、收藏

由于收录的工具众多，因此 He3 还给用户提供了「个性化首页」，可以显示出用户最近使用过的、最常使用的，以及收藏过的工具，方便你更快地找到它们来使用。

![拖放排序](https://img-dp.iplaysoft.com/dispatch/696d565acc8c3045df100a1dc7e64203.gif)

遇到常用的工具你也可以进行收藏，日后能更快地找到它们来使用。界面上所有的工具也都可以通过鼠标拖拽来调整排序等。
