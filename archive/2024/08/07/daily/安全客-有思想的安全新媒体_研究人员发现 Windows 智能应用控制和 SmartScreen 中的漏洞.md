зҪ‘з»ңе®үе…Ёз ”з©¶дәәе‘ҳеҸ‘зҺ°дәҶ Microsoft зҡ„ Windows Smart App Control е’Ң SmartScreen дёӯзҡ„и®ҫи®ЎејұзӮ№пјҢиҝҷдәӣејұзӮ№еҸҜиғҪдҪҝеЁҒиғҒиЎҢдёәиҖ…иғҪеӨҹеңЁдёҚеҸ‘еҮәд»»дҪ•иӯҰе‘Ҡзҡ„жғ…еҶөдёӢиҺ·еҫ—еҜ№зӣ®ж ҮзҺҜеўғзҡ„еҲқе§Ӣи®ҝй—®жқғйҷҗгҖӮ

жҷәиғҪеә”з”ЁжҺ§еҲ¶ пјҲSACпјү жҳҜ Microsoft еңЁ Windows 11 дёӯеј•е…Ҙзҡ„дёҖйЎ№дә‘й©ұеҠЁзҡ„е®үе…ЁеҠҹиғҪпјҢз”ЁдәҺйҳ»жӯўжҒ¶ж„ҸгҖҒдёҚеҸ—дҝЎд»»е’ҢеҸҜиғҪдёҚйңҖиҰҒзҡ„еә”з”ЁеңЁзі»з»ҹдёҠиҝҗиЎҢгҖӮеҰӮжһңжңҚеҠЎж— жі•еҜ№еә”з”ЁиҝӣиЎҢйў„жөӢпјҢе®ғдјҡжЈҖжҹҘеә”з”ЁжҳҜеҗҰе·ІзӯҫеҗҚжҲ–жҳҜеҗҰе…·жңүжңүж•Ҳзҡ„зӯҫеҗҚпјҢд»Ҙдҫҝжү§иЎҢгҖӮ

дёҺ Windows 10 дёҖиө·еҸ‘еёғзҡ„ SmartScreen жҳҜдёҖз§Қзұ»дјјзҡ„е®үе…ЁеҠҹиғҪпјҢеҸҜзЎ®е®ҡзҪ‘з«ҷжҲ–дёӢиҪҪзҡ„еә”з”ЁзЁӢеәҸжҳҜеҗҰе…·жңүжҪңеңЁзҡ„жҒ¶ж„ҸгҖӮе®ғиҝҳеҲ©з”ЁеҹәдәҺеЈ°иӘүзҡ„ж–№жі•иҝӣиЎҢ URL е’Ңеә”з”ЁдҝқжҠӨгҖӮ

вҖңMicrosoft Defender SmartScreen иҜ„дј°зҪ‘з«ҷзҡ„ URLпјҢд»ҘзЎ®е®ҡе®ғд»¬жҳҜеҗҰе·ІзҹҘеҲҶеҸ‘жҲ–жүҳз®ЎдёҚе®үе…Ёзҡ„еҶ…е®№пјҢвҖқRedmond еңЁе…¶ж–ҮжЎЈдёӯжҢҮеҮәгҖӮ

вҖңе®ғиҝҳдёәеә”з”ЁзЁӢеәҸжҸҗдҫӣеЈ°иӘүжЈҖжҹҘпјҢжЈҖжҹҘдёӢиҪҪзҡ„зЁӢеәҸд»ҘеҸҠз”ЁдәҺзӯҫзҪІж–Үд»¶зҡ„ж•°еӯ—зӯҫеҗҚгҖӮеҰӮжһң URLгҖҒж–Үд»¶гҖҒеә”з”ЁжҲ–иҜҒд№Ұе·Іе»әз«ӢдҝЎиӘүпјҢеҲҷз”ЁжҲ·дёҚдјҡзңӢеҲ°д»»дҪ•иӯҰе‘ҠгҖӮеҰӮжһңжІЎжңүеЈ°иӘүпјҢеҲҷиҜҘйЎ№зӣ®е°Ҷиў«ж Үи®°дёәйЈҺйҷ©иҫғй«ҳпјҢе№¶еҗ‘з”ЁжҲ·еҸ‘еҮәиӯҰе‘ҠгҖӮ

иҝҳеҖјеҫ—дёҖжҸҗзҡ„жҳҜпјҢеҪ“еҗҜз”Ё SAC ж—¶пјҢе®ғдјҡжӣҝжҚўе’ҢзҰҒз”Ё Defender SmartScreenгҖӮ

Elastic Security Labs еңЁдёҺ The Hacker News еҲҶдә«зҡ„дёҖд»ҪжҠҘе‘ҠдёӯиЎЁзӨәпјҡвҖңSmart App Control е’Ң SmartScreen еӯҳеңЁи®ёеӨҡеҹәжң¬зҡ„и®ҫи®ЎејұзӮ№пјҢиҝҷдәӣејұзӮ№е…Ғи®ёеңЁжІЎжңүе®үе…ЁиӯҰе‘Ҡе’ҢжңҖе°‘з”ЁжҲ·дәӨдә’зҡ„жғ…еҶөдёӢиҝӣиЎҢеҲқе§Ӣи®ҝй—®гҖӮ

з»•иҝҮиҝҷдәӣдҝқжҠӨзҡ„жңҖз®ҖеҚ•ж–№жі•д№ӢдёҖжҳҜдҪҝз”ЁеҗҲжі•зҡ„жү©еұ•йӘҢиҜҒ пјҲEVпјү иҜҒд№ҰеҜ№еә”з”ЁзЁӢеәҸиҝӣиЎҢзӯҫеҗҚпјҢжҒ¶ж„ҸиЎҢдёәиҖ…е·Із»ҸеҲ©з”Ёиҝҷз§ҚжҠҖжңҜжқҘеҲҶеҸ‘жҒ¶ж„ҸиҪҜд»¶пјҢжӯЈеҰӮжңҖиҝ‘еңЁВ HotPageВ дёӯжүҖиҜҒжҳҺзҡ„йӮЈж ·гҖӮ

дёӢйқўеҲ—еҮәдәҶдёҖдәӣеҸҜз”ЁдәҺжЈҖжөӢи§„йҒҝзҡ„е…¶д»–ж–№жі• вҖ“

* еЈ°иӘүеҠ«жҢҒпјҢж¶үеҸҠиҜҶеҲ«е’ҢйҮҚж–°еҲ©з”Ёе…·жңүиүҜеҘҪеЈ°иӘүзҡ„еә”з”ЁзЁӢеәҸд»Ҙз»•иҝҮзі»з»ҹпјҲдҫӢеҰӮпјҢJamPlusВ жҲ–е·ІзҹҘзҡ„ AutoHotkey и§ЈйҮҠеҷЁпјү

* дҝЎиӘүж’ӯз§ҚпјҢж¶үеҸҠдҪҝз”ЁзңӢдјјж— е®ізҡ„ж”»еҮ»иҖ…жҺ§еҲ¶зҡ„дәҢиҝӣеҲ¶ж–Үд»¶жқҘи§ҰеҸ‘з”ұдәҺеә”з”ЁзЁӢеәҸдёӯзҡ„жјҸжҙһиҖҢеҜјиҮҙзҡ„жҒ¶ж„ҸиЎҢдёәпјҢжҲ–иҖ…еңЁз»ҸиҝҮдёҖе®ҡж—¶й—ҙеҗҺи§ҰеҸ‘жҒ¶ж„ҸиЎҢдёәгҖӮ

* еЈ°иӘүзҜЎж”№пјҢж¶үеҸҠжӣҙж”№еҗҲжі•дәҢиҝӣеҲ¶ж–Үд»¶пјҲдҫӢеҰӮи®Ўз®—еҷЁпјүзҡ„жҹҗдәӣйғЁеҲҶд»ҘжіЁе…Ҙ shellcodeпјҢиҖҢдёҚдјҡеӨұеҺ»е…¶ж•ҙдҪ“еЈ°иӘү

* LNK StompingпјҢж¶үеҸҠеҲ©з”Ё Windows еҝ«жҚ·ж–№ејҸ пјҲLNKпјү ж–Үд»¶еӨ„зҗҶж–№ејҸдёӯзҡ„й”ҷиҜҜжқҘеҲ йҷӨ Web ж Үи®° пјҲMotWпјү ж Үзӯҫе№¶з»•иҝҮ SAC дҝқжҠӨпјҢеӣ дёә SAC дјҡйҳ»жӯўеёҰжңүж Үзӯҫзҡ„ж–Үд»¶гҖӮ

вҖңе®ғж¶үеҸҠеҲ¶дҪңе…·жңүйқһж ҮеҮҶзӣ®ж Үи·Ҝеҫ„жҲ–еҶ…йғЁз»“жһ„зҡ„LNKж–Үд»¶пјҢвҖқз ”з©¶дәәе‘ҳиҜҙгҖӮвҖңеҪ“еҚ•еҮ»ж—¶пјҢиҝҷдәӣLNKж–Үд»¶иў«explorer.exeдёҺи§„иҢғж јејҸдҝ®ж”№гҖӮиҝҷз§Қдҝ®ж”№еҜјиҮҙеңЁжү§иЎҢе®үе…ЁжЈҖжҹҘд№ӢеүҚеҲ йҷӨ MotW ж ҮзӯҫгҖӮ

вҖңеҹәдәҺеЈ°иӘүзҡ„дҝқжҠӨзі»з»ҹжҳҜйҳ»жӯўе•Ҷе“ҒжҒ¶ж„ҸиҪҜд»¶зҡ„ејәеӨ§еұӮпјҢвҖқиҜҘе…¬еҸёиЎЁзӨәгҖӮвҖңз„¶иҖҢпјҢеғҸд»»дҪ•дҝқжҠӨжҠҖжңҜдёҖж ·пјҢе®ғд»¬д№ҹжңүдёҖдәӣејұзӮ№пјҢеҸҜд»ҘйҖҡиҝҮдёҖдәӣе°ҸеҝғжқҘз»•иҝҮгҖӮе®үе…Ёеӣўйҳҹеә”д»”з»ҶжЈҖжҹҘе…¶жЈҖжөӢе Ҷж Ҳдёӯзҡ„дёӢиҪҪеҶ…е®№пјҢиҖҢдёҚжҳҜд»…дҫқиө–ж“ҚдҪңзі»з»ҹеҺҹз”ҹе®үе…ЁеҠҹиғҪеңЁжӯӨеҢәеҹҹжҸҗдҫӣдҝқжҠӨгҖӮ
