# hhash

简单小巧的哈希计算工具，MD5/SHA1/SHA256/SHA512，支持字符串、文件哈希计算，支持按照指定顺序计算哈希值

## Usage

Usage:

hhash string|file md5|sha1|sha256|sha512... [file]

hhash 12345 md5

hhash 12345 md5 sha1

hhash hash.zip sha256 file

hhash hash.zip sha256 md5 file

## About

简单小巧的哈希计算工具，MD5/SHA1/SHA256/SHA512，支持字符串、文件哈希计算，支持按照指定顺序计算哈希值
