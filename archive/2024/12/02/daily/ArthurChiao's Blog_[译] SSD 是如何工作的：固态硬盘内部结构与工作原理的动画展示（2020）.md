з§°дёәвҖңеӣәжҖҒвҖқжҳҜзӣёеҜ№дәҺж—ӢиҪ¬пјҲrotationalпјүзЈҒзӣҳпјҲд№ҹе°ұжҳҜжҷ®йҖҡ HDD зЎ¬зӣҳпјүйӮЈз§ҚвҖңеҠЁжҖҒвҖқиҖҢиЁҖзҡ„гҖӮ

жң¬ж–Үе°Ҷж·ұе…ҘеҲ°иҝҷдёӘиҠҜзүҮеҶ…йғЁпјҢзңӢзңӢе®ғжҳҜеҰӮдҪ•е·ҘдҪңзҡ„гҖӮ

# 1 еӯҳеӮЁжқҗж–ҷ & з»“жһ„пјҡ`Charge Trap`

е°Ҷ SSD иҠҜзүҮж”ҫеӨ§еҲ°**зәізұізә§**пјҢе°ұиғҪзңӢеҲ°е®ғеӯҳеӮЁз”өиҚ·зҡ„**еҹәжң¬з»“жһ„**гҖӮ

* ж №жҚ®жҠҖжңҜи·Ҝзәҝзҡ„дёҚеҗҢпјҢеӯҳеӮЁз»“жһ„/жқҗж–ҷзҡ„йҖүжӢ©д№ҹдёҚеҗҢпјҢ
* жң¬ж–Үд»Ӣз»Қзҡ„жҳҜжҜ”иҫғж–°зҡ„дёҖз§ҚпјҢз§°дёә **`Charge Trap`**пјҲз”өиҚ·жҚ•иҺ·пјҢжҲ–з”өиҚ·йҷ·йҳұпјүпјҢ
  е®ғдҪҝз”Ёзҡ„жҳҜ**ж°®еҢ–зҹҪ**пјҲsilicon nitrideпјүпјҢиҝҷжҳҜдёҖз§Қ**з»қзјҳдҪ“**гҖӮ

дёӢеӣҫдёӯзҡ„вҖңе·ҘвҖқеӯ—з»“жһ„е°ұжҳҜ Charge TrapпјҢе®ғзҡ„еҹәжң¬еҺҹзҗҶжҳҜ**е°Ҷз”өеӯҗеҗёйҷ„еҲ°ж°®еҢ–зҹҪдёҠ**пјҢ
еҗёйҷ„зҡ„з”өеӯҗж•°йҮҸдёҚдёҖж ·пјҢз”өиҚ·зҡ„й«ҳдҪҺе°ұдёҚдёҖж ·пјҢд»ҺиҖҢеҸҜд»Ҙз”ЁдәҺиЎЁзӨәдёҚеҗҢзҡ„ж•°еӯ—пјҢ

![](/assets/img/how-ssd-works/charge-trap-1.png)

еӣҫдёӯй»„иүІйғЁеҲҶе°ұжҳҜеҗёйҷ„зҡ„з”өеӯҗпјҢ

* иҫғиҖҒзҡ„жҠҖжңҜеҸӘиғҪеӯҳеӮЁ**2 дёӘдёҚеҗҢзҡ„з”өиҚ·зә§еҲ«**пјҢеҚіз”өеӯҗеҫҲеӨҡжҲ–еҫҲе°‘пјҢ
  еӣ жӯӨ**еҸӘиғҪиЎЁзӨәдёӨз§Қж•°еҖј**пјҢд№ҹе°ұжҳҜ **`1bit`** `0` е’Ң `1`пјӣ
* иҫғж–°зҡ„ Charge Trap еҸҜд»ҘеӯҳеӮЁ **8 дёӘжҲ– 16 дёӘз”өиҚ·зә§еҲ«**пјҢ
  д№ҹе°ұжҳҜжҜҸдёӘ Charge Trap еҸҜд»ҘиЎЁзӨә **`3bit жҲ– 4bit`**гҖӮ

иў«еҗёйҷ„зҡ„з”өиҚ·еҸҜд»Ҙ**дҝқжҢҒеҮ еҚҒе№ҙ**д№Ӣд№…пјҢиҝҷд№ҹжҳҜе®ғиў«з§°дёәз”өиҚ·йҷ·йҳұзҡ„еҺҹеӣ гҖӮ

# 2 SSD иҠҜзүҮзЎ¬д»¶з»„жҲҗ

дёӢйқўд»Һе°ҸеҲ°еӨ§пјҢзңӢзңӢжҳҜеҰӮдҪ•еҹәдәҺ Charge Trap иҝҷж ·дёҖдёӘжңҖеҹәжң¬еҚ•е…ғжһ„е»әеҮәдёҖдёӘжңҖз»Ҳзҡ„ SSD иҠҜзүҮзҡ„гҖӮ

## 2.1 `Charge Trap` -> еҹәжң¬еӯҳеӮЁеҚ•е…ғ `Memory Cell`

Charge Trap жҳҜ SSD зҡ„еҹәжң¬еӯҳеӮЁеҚ•е…ғ вҖ”вҖ” **`memory cell`** вҖ”вҖ” зҡ„ж ёеҝғгҖӮ

еңЁжң¬ж–ҮжҺҘдёӢжқҘзҡ„еҶ…е®№дёӯпјҢжҲ‘д»¬еҒҮи®ҫдёҖдёӘ charge trap ж”ҜжҢҒ 8 дёӘдёҚеҗҢзҡ„з”өиҚ·зә§еҲ«пјҢд№ҹе°ұжҳҜиҜҙеҸҜд»ҘиЎЁзӨә 3bitпјҢ
жҜ”еҰӮеҗёйҷ„зҡ„з”өеӯҗеҫҲе°‘еҜ№еә” `111`пјҢеҗёйҷ„зҡ„з”өеӯҗеҫҲеӨҡеҜ№еә” `000`гҖӮ

дёӢйқўз®ҖеҚ•д»Ӣз»ҚдёӢиҜ»еҸ–е’ҢеҲ йҷӨж•°жҚ®еҜ№еә”зҡ„еә•еұӮж“ҚдҪңгҖӮ

### 2.1.1 иҜ»еҸ–ж•°жҚ®

иҜ»еҸ–дёҖдёӘ memory cell еӯҳеӮЁзҡ„ж•°жҚ®пјҢе°ұжҳҜ**жөӢйҮҸиҝҷдёӘ Charge Trap дёҠзҡ„зҡ„з”өиҚ·йҮҸ**пјҢ

![](/assets/img/how-ssd-works/charge-trap-2.png)

иҝҷйңҖиҰҒе…ҲйҖҡиҝҮ control gate й”Ғе®ҡиҜҘ Charge TrapпјҢз„¶еҗҺдҝЎжҒҜе°ұеҸҜд»Ҙд»Һдёӯй—ҙзҡ„дј иҫ“зәҝйҖҒдёҠеҺ»гҖӮ
еҗҺйқўдјҡиҜҰз»Ҷд»Ӣз»ҚгҖӮ

### 2.1.2 еҲ йҷӨж•°жҚ®

еҲ йҷӨдёҖдёӘ memory cell еӯҳеӮЁзҡ„ж•°жҚ®пјҢе°ұжҳҜ**жё…йҷӨиҝҷдёӘ Charge Trap дёҠзҡ„зҡ„з”өиҚ·йҮҸ**пјҢ
дҪҝе…¶еӣһеҲ°жңҖдҪҺз”өе№іпјҲ`111`пјүгҖӮ

## 2.2 зәөеҗ‘е ҶеҸ  `Memory Cell` -> `String`

жңүдәҶиғҪиЎЁзӨә 3bit зҡ„еҹәжң¬еҚ•е…ғпјҢжҺҘдёӢжқҘжҲ‘д»¬е°Ҷ N дёӘ cell еһӮзӣҙе ҶеҸ иө·жқҘпјҢ
е°ұеҫ—еҲ°дёҖдёӘз§°дёә **`String`**пјҲвҖңдёІвҖқпјүзҡ„з»“жһ„гҖӮ

дёӢеӣҫжҳҜ 10 дёӘ memory cell е ҶеҸ жҲҗзҡ„ stringпјҢ

![](/assets/img/how-ssd-works/vertical-string-2.png)

дёҖдёӘ String еҶ…зҡ„жүҖжңү cell е…ұдә«йЎ¶йғЁзҡ„ **`bit line`**пјҲвҖңbit дј иҫ“зәҝвҖқпјҢиҜ»еҸ–жҲ–еҶҷе…Ҙ cell ж•°жҚ®зҡ„зәҝпјүпјҢ

![](/assets/img/how-ssd-works/bit-line.png)

дёҖдёӘ String жңүеҫҲеӨҡ cellпјҢдҪҶе®ғд»¬е…ұдә«еҗҢдёҖж № bit lineпјҢ
еӣ жӯӨпјҢеңЁд»»дёҖж—¶й—ҙеҸӘиғҪжҝҖжҙ» String дёӯзҡ„дёҖдёӘ cellгҖӮдёәжӯӨпјҢйңҖиҰҒеј•е…ҘдәҶ **`control gate`**гҖӮ

* control gate жҺ§еҲ¶ String дёҠзҡ„е“ӘдёӘ cell еҸҜд»ҘиҜ»еҶҷж•°жҚ®пјҢжӯӨж—¶з§°дёәвҖңжҝҖжҙ»вҖқзҠ¶жҖҒпјӣ
  еҰӮдёҠеӣҫжүҖзӨәпјҢиҜ»еҸ–з¬¬ 10 еұӮзҡ„ cell дҝЎжҒҜж—¶пјҢе°ұжҝҖжҙ»з¬¬ 10 еұӮзҡ„ control gateпјҡ
* дҪҶжіЁж„ҸпјҢcontrol gate еҸӘжҳҜз”ЁжқҘ**жҝҖжҙ» cell**пјҢиҖҢдёҚжҳҜз”ЁжқҘиҜ»еҸ– cell зҡ„дҝЎжҒҜпјҡ
  жҜ”еҰӮеңЁиҜ»ж•°жҚ®еңәжҷҜпјҢиў«жҝҖжҙ»зҡ„ cell дјҡе°Ҷе®ғдҝқеӯҳзҡ„дҝЎжҒҜйҖҡиҝҮ String дёӯеҝғзҡ„ж•°жҚ®зәҝпјҲжҜҸдёӘвҖңе·ҘвҖқеӯ—зҡ„дёӯеҝғзәҝпјүеҸ‘йҖҒз»ҷйЎ¶йғЁзҡ„ **`bit line`**гҖӮ

## 2.3 жЁӘеҗ‘е ҶеҸ  `Memory Cell` -> `Page`

е°ҶеӨҡдёӘ String ж°ҙе№іиҝһеҲ°дёҖиө·пјҢе°ұеҫ—еҲ°дёҖдёӘдәҢз»ҙ cell з©әй—ҙгҖӮ

жЁӘеҗ‘зҡ„жҜҸдёҖжҺ’ memory cellпјҢз§°дёәдёҖдёӘ **`Page`**пјҲвҖңйЎөвҖқпјүпјҢеҰӮдёӢеӣҫжүҖзӨәпјҡ

![](/assets/img/how-ssd-works/memory-cell-string-page.png)

## 2.4 String+Page з»„жҲҗ 2D еӯҳеӮЁзҹ©йҳө -> Row

String+Page з»„жҲҗзҡ„ 2D еӯҳеӮЁзҹ©йҳөпјҢз§°дёә **`Row`**пјҲиҷҪз„¶еңЁиҝҷйҮҢзӣҙи§үдёҠеҸ«вҖңPageвҖқжӣҙеҗҲйҖӮпјҢеҗҺйқўдјҡзңӢеҲ°иҝҷдёӘеҗҚз§°зҡ„з”ұжқҘпјүпјҢ

![](/assets/img/how-ssd-works/row.png)

### 2.4.1 bit line е’Ң control gate

еҶҚжқҘзңӢдёӢ bit-line/control-gate е’Ң String/Row зҡ„е…ізі»пјҢ

![](/assets/img/how-ssd-works/32x-bit-lines.png)

* жҜҸдёӘ String жңүзӢ¬з«Ӣзҡ„ bit lineпјӣ
* жҜҸдёӘ Row дёҠзҡ„жүҖжңү cell е…ұдә«дёҖдёӘ control gateпјҢ

### 2.4.2 иҜ»еҶҷдёҖдёӘ Pageпјҡд»…йңҖдёҖж¬Ў control gate ж“ҚдҪң

з”ұдёҠеӣҫеҸҜзҹҘпјҢеҗ‘ Row еҶҷе…ҘжҲ–иҜ»еҸ–ж•°жҚ®ж—¶пјҢ**жЁӘеҗ‘зҡ„ cell иғҪеҗҢж—¶иў«жҝҖжҙ»**пјҢе®ғд»¬иғҪйҖҡиҝҮйЎ¶дёҠзҡ„ bit lines е№¶иЎҢдј иҫ“гҖӮ
