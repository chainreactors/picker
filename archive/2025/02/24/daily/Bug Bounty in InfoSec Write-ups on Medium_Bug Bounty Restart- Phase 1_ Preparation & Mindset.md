Hello Everyone,

As I mentioned in my last blog, I’ve decided to restart my bug bounty journey. If you’re also restarting or even starting from scratch, this may help you in some way. So, let’s begin!

Not a Paid member yet ? [Click here to read this story for free](https://omarora1603.medium.com/57ca877a9278?sk=9d714d8e406afdcb5c3ee66a32dda3d2)

## 1. Mindset

Many people start bug bounties with the mindset of earning a lot of money. However, we’ve been there, done that, and learned from it. This time, we’re approaching bug bounty with a different perspective — one focused on learning new things and applying our knowledge to real targets. Now, let’s start with the preparation.

## 2. Revisiting The Basics

![]()

Back to Basics By **Demosea on** [**Giphy**](https://giphy.com/stickers/Demosea-skincare-demosea-back-to-basics-J61bijsi5drIlBLSn9)

So first we are going to brush up all the important basics concepts, for this I am going to read a book —

The book is [Bug Bounty Bootcamp](https://www.amazon.com/Bug-Bounty-Bootcamp-Reporting-Vulnerabilities/dp/1718501544?s=books&sr=1-1&language=en_US&ref_=as_li_ss_tl) by [Vickie Li](https://www.amazon.com/Vickie-Li/e/B094KXD5YW/ref%3Ddp_byline_cont_book_1), She is highly reputed in the bug bounty industry, and I’ve heard great things about this book. With the help of this book we can brush up all the old topics and also learn new things with it. And if you decide to read it too don’t forget to make notes as you read it, They will be very useful for your journey.
