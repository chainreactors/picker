## 甲骨文 SUSE 和 CIQ 成立 Open Enterprise Linux Association

[![Redhat](https://icon.solidot.org/images/topics/topicredhat.png?123)](/search?tid=99 "Redhat")
[![SuSE](https://icon.solidot.org/images/topics/topicsuse.png?123)](/search?tid=103 "SuSE")

[Wilson](/~Wilson) (42865)发表于 2023年08月11日 14时49分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75769&appkey=1370085986&title=%E7%94%B2%E9%AA%A8%E6%96%87%20SUSE%20%E5%92%8C%20CIQ%20%E6%88%90%E7%AB%8B%20Open%20Enterprise%20Linux%20Association "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自红女王**

因 Red Hat 改变了其企业级发行版 Red Hat Enterprise Linux (RHEL)的源码发布方式，增加了 AlmaLinux、Rocky Linux 和 Oracle Linux 等构建 RHEL 完美克隆版本的难度。现在甲骨文、SUSE 以及 Rocky Linux 的服务合作伙伴 CIQ 宣布成立 Open Enterprise Linux Association，致力于通过提供开放企业 Linux 源代码促进兼容 RHEL 发行版的开发。SUSE 此前宣布将创建维护一个 RHEL 兼容分支，现在决定与其它 RHEL 兼容发行版项目进行联手。
https://openela.org/
