# 戴尔在 Microsoft Ignite 大会上宣布了多项人工智能和网络安全方面的新进展

本文是翻译文章，文章原作者 Megan Crouse，文章来源：techrepublic

原文地址：<https://www.techrepublic.com/article/dell-ai-cybersecurity-solutions-microsoft-ignite-2024/>

译文仅供参考，具体内容表达以及含义原文为准。

**戴尔在Microsoft Ignite上宣布了新的人工智能和网络安全进展，包括面向 Azure 的 APEX 文件存储和 Copilot 服务。**

戴尔在微软Ignite大会上发布了几项新的互联服务，Ignite大会是由微软主办的年度大会，专为开发人员和IT专业人士设计。其中几项服务旨在消除部署生成式人工智能时的猜测或安全顾虑，尤其是在微软的 Copilot+ PC 上。

**戴尔负责管理Azure的APEX文件存储**

Azure的APEX文件存储已经有一段时间了。不过，从 2025 年上半年的公开预览版开始，客户将可以选择由戴尔对其进行全面管理。与戴尔提供的客户管理服务不同，APEX File Storage 可以作为 Azure 原生 ISV 服务使用。

Microsoft Azure管理员也可以添加戴尔APEX保护服务。APEX Protection 服务使用零信任原则、机器学习、分析和取证工具来保护关键数据免受勒索软件的攻击。适用于 Microsoft Azure 的戴尔 APEX Protection 服务将于 2025 年上半年发布。

**Copilot和Azure人工智能服务旨在简化人工智能的应用，尤其是在微软的人工智能PC上**

微软正在大力推广其 Copilot AI 助手和 Copilot+ PC，戴尔也推出了相应的产品： Dell Services for Microsoft Copilot Studio（低代码）和 Azure AI Studio（专业代码）。

这些产品旨在帮助客户以最符合其需求和技能组合的方式利用人工智能功能。它们可以识别用例，帮助客户创建定制的 Copilot 或其他人工智能解决方案。总之，我们的目标是提供能让人工智能项目更容易启动的框架。

对于微软的 Copilot+ PC，Dell Accelerator Services for Copilot+ PC 根据员工角色和公司目标提供 “优化使用建议”。它还为人工智能实验提供了一个环境。这包括有关 NPU 利用率如何影响客户组织的 Al 工作负载的数据，以及有关优化 Windows 11 设置和 IT 团队的 Microsoft Intune 管理的提示。

戴尔专业服务副总裁斯科特-比尔斯（Scott Bils）在11月13日的新闻发布会上表示，戴尔的目标是 “最大限度地提高Copilot+的体验和成果”。

**在现有 APEX 和 MDR 产品中增加安全服务**

其他公告重点关注安全服务。针对 MS Azure 的戴尔 APEX 保护服务现已作为戴尔托管服务提供。APEX 保护包括安全存储、加密、MFA、用于攻击检测和恢复的人工智能分析以及用于关键数据的网络保险库。

戴尔ISG和电信产品营销高级副总裁Varun Chhabra在新闻发布会上表示：“针对MS Azure的戴尔APEX保护服务提供了一种‘简化方法’，通过Azure中的强化数字保险库保护和保障关键数据免受勒索软件和网络威胁的侵害。”

最后，戴尔的托管检测和响应现在可以与微软 Defender XDR 互操作。这标志着戴尔现有安全咨询和建议解决方案的Windows友好型扩展。

Bils指出：“戴尔将这一行业领先的能力与我们的认证安全专家结合起来，帮助客户建立和部署、实施Microsoft Defender XDR解决方案，并在出现漏洞时提供支持。”

**戴尔为CMMC防御认证提供更多帮助**

戴尔为微软的美国联邦承包商推出了网络安全成熟度模型认证对标服务。该认证是一项帮助企业符合国防部 CMMC 标准的服务。CMMC 2.0 版将于 2025 年推出，并从 2024 年 12 月开始纳入国防部合同。这项服务可以帮助企业制定路线图，以符合 2.0 版的要求。
