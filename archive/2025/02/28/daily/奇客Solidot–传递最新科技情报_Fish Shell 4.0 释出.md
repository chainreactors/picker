## Fish Shell 4.0 释出

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2025年02月27日 23时00分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=80672&appkey=1370085986&title=Fish%20Shell%204.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自异形：走出阴影**

有 20 年历史的 Unix shell Fish 释出了 4.0 版本。Fish Shell 4.0 的一大变化是用 Rust 重写了核心 C++ 代码，意味着依赖关系和构建方式的巨变，但这对终端用户没有直接影响。其它变化包括：新的 bind 键符号，相同种子将生成与之前版本不同的随机值，Ctrl-C 将默认清除命令行，Windows Subsystem for Linux 调整，Fish 能作为自安装二进制文件构建，等等。
GitHub:fish 4.0.0 (released February 27, 2025)
