# ChatGPT+数字水印，信源密信强大的安全底座+灵活的应用集成能力

企业资讯
[行业](https://www.4hou.com/category/industry)
2023-02-17 16:27:11

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)105559

收藏

导语：持续“狂飙”的ChatGPT似乎无所不能，从无缝对话，到作曲、小说、文章甚至编剧，ChatGPT展现了AI的惊人能量，为世界带来了不可思议的新能力。

持续“狂飙”的ChatGPT似乎无所不能，从无缝对话，到作曲、小说、文章甚至编剧，ChatGPT展现了AI的惊人能量，为世界带来了不可思议的新能力。但事情很快就有了反转，有些人开始担心这项新的技术会被坏人所利用。而在版权界，生成式人工智能始终被“抄袭”、“剽窃”等舆论裹挟。以至于一些专家学者或是画家开始抵制人工智能学习作品，在此背景下，“数字水印”诞生了。“以技术对抗技术”，数字水印一时间受到市场关注。

信源密信是北信源2013年立项研发的战略级安全通讯产品，是保障工作及相关人员的“安全通信+协同办公+开放开发平台”为基础的移动安全通信平台。满足了数字化转型过程中数据资产的安全需求及万物互联时代的安全通信需求，保护了国家秘密、工作秘密、商业秘密和个人隐私安全。今年初，信源密信顺利通过信通院“办公即时通信软件安全能力”最高级评测，成为首批入选“卓越级”的产品。

信源密信以私有化部署，支持全面适配信创环境，为用户提供“即时通讯、协同办公、应急指挥、任务管理、智能化流程、AI机器人、应用开发、万物互联、互联互通”等多层次的安全聚合平台服务，成为跨终端、全方位、安全可信的通信平台。经过多次版本升级，已经形成以信源密信为安全底座，应用于各种业务场景的衍生产品体系，比如智慧党建安全系统、好家好社区安全平台、网络安全应急指挥系统等，在数字中国建设、企业数字化转型、支持国家信创产业发展中起到了即时通讯安全底座的作用。

信源密信另一大优势是其开放开发聚合平台功能，这也为其与ChatGPT的快速集成提供了便利。信源密信提供DDIO接口，为用户提供开放开发功能，用户可通过DDIO接口改造已有的调度、办公类IT应用系统，连接人与系统，打通数据孤岛，系统间共享信源密信的资源和安全通道通信。目前，信源密信已通过开放接口实现ChatGPT的技术验证对接，达成基于信源密信会话模式的ChatGPT服务，为用户提供消息对话、业务集成、新闻资讯等一系列服务。未来百度文心一言若支持开放对接，信源密信作为安全通讯底座也能实现与其进行快速对接。

信源密信实现与ChatGPT技术验证对接的同时，在数据与信息安全保障方面也采取了一些措施。在“数字水印”技术上，信源密信真正发挥了其底层技术能力，为了防止重要消息泄露，加入了用于特殊处理拍照和截屏的安全数字水印，包括明水印和暗水印。明水印，直接将内容加载到页面上，拥有抵御未经授权的删除、嵌入和检测等攻击的能力。暗水印，隐藏在明水印下方，肉眼无法直接查看。如想查看暗水印，需要将其设置的暗水印界面截图（如果是照片、文件、视频无需截图），发送给相关的技术人员。通过图片处理工具查看图片中的图层即可甄别水印情况，进行溯源。

信源密信后台还可进行水印的自定义设置，应用灵活。信源密信+ChatGPT可发挥信源密信强大的安全底座+灵活的应用集成功能。后期信源密信也会逐渐完善敏感信息识别和拦截等业务。比如在消息的询问检查上做一些关键词过滤，或者通过语义分析来捕捉不当问题等…… 作为一款即时通讯软件，信源密信在信息传递过程中能起到闸口的作用。

信源密信凭借其快速集成能力和包括数字水印在内的强大安全功能，现已成功服务于国家部委、军工、部队、科研院校、大型央企、金融机构等重要单位，被相关单位选为“指定即时通讯平台”，成为国家重要单位、重点工程、重大活动优选的安全通信平台，装机量达千万台。
