真的是活久见系列！微软今天宣布 Windows 11 资源管理器将原生支持更多压缩格式，在过去许多年里，资源管理器原生支持的只有 zip 格式，无论是压缩还是解压都只有 zip 格式。

也正是基于这种原因，蓝点网分享的文件从来都是用 zip 打包，因为无论是在 Windows、Mac、Linux、Android、iOS 上，zip 格式都可以解压，实际上算是减少了某些用户的不变。

还有以前的笑话：下载 WinRAR 结果下载的文件是 WinRar.rar 还是没法用，后面这些问题都要解决了。

[![终于！Windows 11资源管理器将原生支持rar/7z/tar/gz/tgz等压缩格式](https://img.lancdn.com/landian/2023/05/98798.png)](https://img.lancdn.com/landian/2023/05/98798.png)

微软宣布 Windows 11 资源管理器将原生支持.tar、.gz、.rar、.7z 等格式，虽然没说那么细，但既然.tar/.gz 都支持了，那 Linux 上的其他压缩格式例如.tar.gz、.tgz 等格式肯定也是支持的。

坏消息是资源管理器只支持解压这些文件，不过应该问题不大，毕竟真有压缩文件需求，使用默认的 zip 其实也够了，如果追求更好的压缩率或选项，那可以考虑自己装个 7-Zip。

此外微软还对压缩方面的操作进行了优化，后续用户在 Windows 11 上压缩文件，将可以获得更好的性能。

微软没有提到这些变化具体何时支持，乐观估计 6 月份发布更新的时候，这些功能应该都会一并支持。
