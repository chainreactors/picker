SAP Document and Reporting Compliance, outbound invoicing option for Brazil that enables the communication between your source system and tax authorities servers, through the cloud. The source system is integrated to the outbound service landscape.

As NF3-e solution is linked to the product *SAP Document and Reporting Compliance, outbound invoicing option for Brazil*, we have heard feedback from some customers who have already installed the product. We have modified the structure of the documentation to make it clearer for the NF3-e customers and to support the implementation of the NF3-e solution as much as possible.

As the SAP Document and Reporting Compliance, outbound invoicing option for the Brazil product guide you will find information about more than one functionality, these direct links will help you find what the NF3-e refers to, thus facilitating the implementation of the solution.

For product installation, follow the landscape steps, but note that in the below chapters there are NF3-e specific documentation, they are:

General Guide: [SAP Document and Reporting Compliance, outbound invoicing option for Brazil](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/b2f5ad5578574616855dda8638e2ee92.html?locale=en-US&state=DRAFT)

* Development -> [REST APIs](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/960e56e26bc44fee93c0802be433297a.html?locale=en-US&version=Cloud) -> NF3-e Specifics

* Integration -> [NF3-e Specifics](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/d36583c051474e189a13e61e7c2032d5.html?locale=en-US&version=Cloud)

* Service Consumption at Application Runtime -> Configuring Outbound nota fiscal > SAP Application Users > [Scheduling Jobs for NF3-e](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/0e95d0e867cb43da9edc9dd4d2856596.html?locale=en-US&version=Cloud)

* Service Consumption at Application Runtime -> Using Outbound Nota fiscal -> [NF3-e Specifics](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/90712b6534eb47ddbacbda55a62d8a29.html?locale=en-US&version=Cloud)

These documentations will always be updated as development changes. Direct links will help you follow these changes more efficiently without having to go through SAP Home for all products (links will stay the same).

Stay tuned for accessing the latest available version, or the corresponding version for your product.

SAP Note 2962963 details the SAP Notes required for NF3-e.

In this blog post we covered all NF3-e direct links to make the implementation of the *SAP Document and Reporting Compliance, outbound invoicing option for Brazil - NF3-e* easeir for our customers.
