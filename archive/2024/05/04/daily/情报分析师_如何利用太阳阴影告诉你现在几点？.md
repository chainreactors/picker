# 如何利用太阳阴影告诉你现在几点？

原创

DMT

情报分析师

先说个小学知识

身边最常见的自然现象——太阳，其实可以成为你的私人时钟？没错，只要留意太阳阴影的位置就能大致估算出现在的时间。那么，具体该怎么做呢？

首先，你需要一个可以投射出清晰阴影的物体，比如一根直立的棍子或一根绳子。当太阳照射到这个物体时，它会在地面上投射出一个阴影。这个阴影的位置会随着时间的推移而移动。

早晨，太阳从东方升起，所以阴影会朝向西方。随着时间的推移，阴影会慢慢向北方移动。到了中午，太阳处于天空的最高点，阴影将直接指向北方。之后，阴影又会开始向南方移动，直到太阳在西方落下，阴影指向东方。 通过观察阴影的移动，你可以估算出现在的时间。

比如，如果阴影刚好指向东方，那么大概是早晨；如果阴影指向北方，那么大概是中午；如果阴影指向西方，那么大概是傍晚。当然，这种方法并不精确到分钟，但对于大致估算时间非常有用。

此外，你还可以通过阴影的长度来估算时间。早晨和傍晚，太阳与地面的角度较大，因此阴影会相对较长；而中午时分，太阳几乎直接照射在地面上，因此阴影会相对较短。

在情报分析师的眼中，当照片或视频中无法直接看到时间信息时有一个巧妙的办法——利用阴影来推测日期或时间。

在多次的深入研究中都不得不依赖这种方法来试图构建一个可靠的时间脉络，虽然有时候这确实是一项极其考验技巧的挑战。

阴影不仅能提供关于光线方向和角度的线索，还能帮助理解当时是一天中的哪个时段，甚至是在一年中的哪个季节。这种方法的运用，让时间的神秘面纱得以揭开一角，揭示出那些隐藏在影像背后的故事和时光痕迹。

敲黑板、认真听

首先，地球围绕太阳旋转，我们称之为地球的公转，地球产生的轨迹形状像一个椭圆。虽然我们每年绕太阳旋转一次，但地球也围绕一个稍微偏离中心的垂直轴旋转。倾斜的轴，加上围绕太阳的自转，使地球四季。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibc8eM75nvVhGL91VlcSZX4RRZk3E3eH2w4JpFq2Z8ULXE5z2mhIq8wbg/640?wx_fmt=png)

由于地球的自转和围绕太阳的轨迹是极其可预测的，知道在任何给定的日期和时间，太阳都处于特定位置。在北半球的夏天，太阳在天空中很高，而冬天的太阳则低得多，朝向地平线。

在互联网上，有几种工具可以计算太阳在任何给定日期或时间的确切位置。要将其用于计算，唯一需要的是与地平线（天顶或高度）和太阳方向（方位角）相比的角度。有了这两条信息，任何人都可以计算出任何给定阴影的可能日期和时间。

由于地球的轨迹及其自转，每年有两个日期太阳处于相同的位置，称为春分点。这些日期约为3月20日和9月22日。看看春分的时刻，看到以下模式出现：

| 纬度 | 三月分点 | 九月分点 |
| --- | --- | --- |
| 90 N | 0 度 | 0 度 |
| 60 N | 30 度 | 30 度 |
| 50 牛 | 40 度 | 40 度 |
| 25 N | 65 度 | 65 度 |
| 0 N | 90 度 | 90 度 |
| 25 秒 | 65 度 | 65 度 |
| 50 秒 | 40 度 | 40 度 |
| 60 秒 | 30 度 | 30 度 |
| 90 秒 | 0 度 | 0 度 |

在开始计算时间或日期之前，我们需要了解一些重要的信息：

1. 越靠近赤道就越困难，因为太阳的轨迹比其他任何地方都更平坦。
2. 最简单的计算是当图像拍摄时阴影在图像中水平显示的位置，或者有一种简单的方法来测量长度。如果有迹象表明场景可能倾斜，则稍微旋转它以使自己更容易是没有问题的。
3. 有一定角度或朝你走来的阴影不容易测量，除非它落在能够准确测量长度的表面上。
4. 如果阴影完全可见并且足够清晰，可以测量则是最好的。
5. 当阴影不完全可见时，仍然可以使用它来计算可能的时间，因为只需要一个方向，但是查找日期可能会带来问题，因为需要阴影的长度。
6. 尽量不要使用表面扭曲或不均匀的阴影，除非您有能力通过使用3D建模来重新创建场景来使用它。
7. 物体和它投射的阴影都应该在一个框架内测量，这样就可以确定物体高度和阴影长度之间的关系。
8. 当使用与胶片剪辑不同的帧时，距离可能会有所不同，因此可能会使测量不太准确。
9. 需要一些参考点来确定阴影落下的确切方向。

如果满足所有这些条件，那么确定拍摄该照片或视频的日期和时间不仅是可能的，而且甚至相当容易。

#####

基本时间计算

#####

接下来通过一个实例来探讨一个有趣的时间推理问题。这里有一张在2018年6月7日星期四拍摄的照片。

既然日期已经明确，你可能会好奇，这张照片具体是在哪个时刻拍摄的呢？

接下来将一起探索如何揭开这个时间之谜。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcEI4l1pn2MjbEuM1XoD55z3sHOxMibaENumspYajMsXvU567D61sF9Tg/640?wx_fmt=png)

首先，需要追踪这个图像的踪迹。经过反向图像搜索有了答案：这张照片的场景是位于布鲁塞尔原子塔脚下的一件艺术建筑物。

然而，仅有这些信息还不够，还需要进一步探索，确定这件艺术建筑物的具体位置，以及拍摄这张照片的最佳角度。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcbVYNOkzL4NjWkq4MxvybpdUCsOjvrbl6MiaFsyIr9icSkHAiaWibyxRh2g/640?wx_fmt=png)

*图：原子塔底部的欢迎标志*

测量艺术建筑物的角度，给一个5度的偏差。可以通过使用谷歌地球专业版中的标尺来相当容易地测量这一点。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcEp66Kialc7ibWjqWAko3iafI5w9zia2ZVg9kQ99mCtMItYGzKpmBFqGoXQ/640?wx_fmt=png)

图：在谷歌地球专业版中测量方位角

看着图像本身，可以看到站在它前面的女人投射出的阴影。知道太阳正站在南方，方位角为185度。将图稿的底部（绿色）与阴影的方向（红色）对齐时，看到它们几乎平行运行。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcB3ibMK49YFUfgpSP307CthnlMJPwdwmYZEzXhdeqVSlebAp2ylvj2dA/640?wx_fmt=png)

图：估计阴影的方向

如果想更精确，可以放大图像本身，以检查阴影是否确实与艺术品本身一致，或者稍微偏离。当观察阴影时，似乎有轻微的差异，但差异是如此之小，以至于这可能是半度的偏差。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcibvcOTIibo49ibV4daEAsapibWqQl4YJZCOXMrX2QTOsvbDPEL7oFZg0ZQ/640?wx_fmt=png)

图：查看所有细节

为了找出这是什么时间，打开了网站SunCalc。

首先需要找到确切的位置，为此，可以简单地搜索“原子塔”（A）会给一个可能位置的列表。选择布鲁塞尔的地点，然后通过单击日期字段将日期设置为2018年6月7日（B）。然后可以移动顶部栏（C）上的滑块，直到接近所需的方位角。

为了尽可能接近，可以通过单击时间字段并相应地进行调整来手动调整分钟数。网站上给出的时间是本地时间，并自动调整为夏令时。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcsm07xxk9eRdWOSNlwibjaC64PDxytvnCQibNichhLUWica8g6CyVibmqFBw/640?wx_fmt=png)

图：查找正确的方位角

在匹配太阳的高度和方位角后，发现这张照片的拍摄时间约为当地时间13：53。实际时间在14：00左右，只有几分钟的时差。

不同角度的阴影

#####

当观察视频或照片中的阴影时，若它们未能保持水平对齐，这无疑为精确测量增加了不小的难度。要知道，我们所生活的世界是一个三维的舞台，阴影的投影与我们的视角息息相关。

想象一下，当这些影子悄然落在你面前，有时想要准确捕捉其角度，简直是难上加难。下面将展示两个在同一区域拍摄的实例，希望更直观地理解这一现象。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcvOZEhibuicMWs2EW2obPCrPuZdmPJtib0ITul9uupQp3bgKGsT5mwL3xg/640?wx_fmt=png)

阴影是水平的

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibciboicHIYeEsbial1xJGQcC0Yf62riaTiawHLeRPXucMpQR9QwRVZQ4Y53yg/640?wx_fmt=png)

阴影正朝你落下

通过使用环境元素作为指导，可以相当准确地测量阴影的距离。尝试找到可以确定确切大小的项目，找到可用于尽可能准确地测量阴影的东西。街道瓷砖可能非常有用，例如，将它们绘制成网格。

再看一个例子，在卢浮宫，阴影没有水平绘制在图像上。仍然可以将瓷砖用作网格，并计算阴影在照片中水平放置时的位置，从而使计算成为可能。

首先，需要确定阴影在图像中的位置，并绘制一条水平线，以便稍后用于指导，这样就可以确定如果阴影逆时针旋转。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcLKkMtIDKG5BYicbse9gibwup0dxeRAC5NRq2wSsXxOiaJ7BQ8an35w3vw/640?wx_fmt=png)

可视化阴影应该在哪里进行测量

接下来，用正确的比例重新创建网格，重新创建阴影的位置和长度。这可以是一条简单的线，从物体，或者在这种情况下是一个人，到阴影的最远点。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibc3V3kdjYkkCfFvxX3KOaXuQzqHW0BfibDib6HbtaFr8ZbyxbLawIWNjLw/640?wx_fmt=png)

图：重新创建网格和阴影

这个圆不仅仅是一个简单的几何形状，它的中心，正是整个图案的灵魂所在。要找到这个中心，不妨想象一束阳光从高空倾泻而下，投射出一个清晰的影子。那个影子的顶点，正是所要寻找的圆心。

接下来，聚焦于这个中心点，想象有一条无形的线从它出发，一直延伸到原始阴影的边缘。这条线的长度，就是你所要绘制的圆的半径。

在画布上，可以清晰地看到，这条半径正好触及到阴影的末端，而整个圆则是以黄色为标记。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibceFJy9xuWtSoRehpHwfCZoMlx2BfjSvY2DJHY6yDM81my1MAoBq13DQ/640?wx_fmt=png)

绘制可测量的旋转阴影

在将照片中的水平线和圆的半径对齐后，找到了一个新点，如果它逆时针旋转，阴影将落在该点。

在 Photoshop 中测量原始图像时，发现长度为 830 像素。当将其与图像中原始阴影的长度进行比较时，发现它只有625像素长，因此大约30%太短而无法用于精确计算。

影子计算器

#####

如果不清楚阴影（例如建筑物投射的阴影）在特定区域中的样子，则可以使用免费提供的工具重新创建它。

到目前为止发现的最简单的一个是影子计算器，可以选择在Google地图上创建虚拟建筑物，在地图上放置点以创建形状。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcCG9ovSV5bQOjO1BiaO7ezqh77T5c5lotiaI64eFsFrYdicCJOIKAhEytA/640?wx_fmt=png)

图：阿姆斯特丹大坝宫殿的快速示例

形状中的每个点都可以调整其高度，从而使更复杂的形状成为可能。通过调整屏幕右侧的日期和时间，阴影及其形状将在地图上进行调整。

注意事项

#####

在浏览图像和视频，寻找阴影时，有几件事需要小心。如果是游戏新手，则在某些情况下可能会导致进入不同的日期或时间。有一件事要小心，那就是天空中的云彩。穿过云层的阳光可以散射阳光，云层充当扩散.这将使阴影的边缘更难区分，实际上需要的信息来确定正确的高度。

除此之外还可能使很难将阴影与沥青等较暗的表面区分开来。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcr7zlH8PBQXuv1zyHJEklAvkJic2OH6TGa9SSf0C7Tib3PftibcaNC90wQ/640?wx_fmt=png)

图：云朵像扩散器

另一件需要警惕的事情是反射表面。特别是在现代城市，有大玻璃外墙，有反光窗户。看下面这个例子，可以清楚地看到太阳被反射在大型建筑中，因此坐在摄影师左边的某个位置。

但由于建筑物的反射表面，所有的阴影都朝我们落下。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibcW4RlaXEztwbq4pLib1ibSoRpklwbqO7UKaic69ShIXXQ8o7CettzNfB1A/640?wx_fmt=png)

要确定太阳的确切方向，必须放大到照片的右下角。走在那里的女人没有被太阳的反射照亮。通过使用这些信息可以确定太阳的正确位置。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmUwwrT0IjhMsRkiaKGdb2aibck0afLjgztyRoAFecPtA4ElHIVR2htDQpo3LsEfgadIzybwVph4lPRg/640?wx_fmt=png)
