# 资本在往哪里看？盘点三季度网安融资事件

原创

邱天

安全牛

尽管企业服务数据库公司Crunchbase的数据显示，2024年第三季度，网络安全初创公司的风险投资下降了51%，降至21亿美元，但这是紧跟在融资额达到43亿美元的第二季度。这一季度是自2022年以来该行业最强劲的融资季度。事实上，三季度网安领域融资还算活跃，其中有四家企业的融资过亿美元。

融资热点的是产业发展的晴雨表，安全牛对三季度的典型融资事件做了一个回顾，旨在让读者对整个网安产业热点有个全局的了解。

**典型融资事件回顾**

**Kiteworks**

**融资金额：4.56亿美元**

**所属领域：安全内容通信**

Kiteworks 是一家知名的网络安全公司，专注于安全内容通信和治理。Kiteworks提供一个强大的平台，帮助组织管理和保护内部和外部交换的敏感信息。

2024 年9月，Kiteworks 完成了 4.65 亿美元的融资，公司估值超过10 亿美元，成为独角兽。这项投资资金由Insight Partners和Sixth Street Growth。此次交易包括至少2.28亿美元的股权融资，以及一笔二级出售交易，即现有投资者出售股份的交易。

公司网址：

https://www.kiteworks.com/

**Abnormal Security**

**融资金额：2.84亿美元**

**所属领域：电子邮件安全**

Abnormal Security是一家知名的网络安全公司，成立于2018年，总部位于加利福尼亚州旧金山。该公司专注于电子邮件安全，提供针对各种电子邮件威胁的高级保护，包括网络钓鱼、恶意软件、勒索软件和社会工程攻击。

Abnormal Security于8月宣布完成2.84亿美元D轮融资，估值达到51亿美元。本轮融资由Wellington Management主导，现有投资者Greylock Partners、Menlo Ventures、Insight Partners和CrowdStrike Falcon Fund参与。

公司网址：

https://www.kiteworks.com/

**Vanta**

**融资金额：1.5亿美元**

**所属领域：合规自动化**

Vanta是一家领先的自动化合规和信任管理平台公司，成立于2018年，旨在为各类组织简化和集中安全工作流程。Vanta总部位于加利福尼亚州旧金山，旨在简化与实现和维持各种安全标准相关的复杂流程。

Vanta于7月完成了1.5亿美元的C轮融资，估值达到24.5亿美元。此次融资由红杉资本主导，此外还包括新的投资者高盛另类投资部门Growth Equity、摩根大通，以及现有投资者Atlassian Ventures、Craft Ventures、CrowdStrike Ventures、HubSpot Ventures、Workday Ventures和Y Combinator。

公司网址：

https://www.vanta.com/

**Chainguard**

**融资金额：1.4亿美元**

**所属领域：软件供应链安全**

Chainguard 是一家专注于软件供应链安全的公司，其使命是通过提供端到端的软件供应链安全解决方案，帮助企业确保其软件的完整性和安全性。

Chainguard 在7月份完成了由 Redpoint Ventures、Lightspeed Venture Partners 和 IVP 主导的1.4亿美元 C 轮融资，使公司的总融资额达到 2.56 亿美元。现有投资者包括 Amplify、Mantis VC、红杉资本和 Spark Capital 也参与了这一轮融资。

公司网址：

https://www.chainguard.dev/

**Gcore**

**融资金额：6000 万美元**

**所属领域：安全解决方案**

Gcore 是一家全球知名的服务提供商，专注于云计算、边缘计算、内容交付、托管和安全解决方案。公司提供全面的安全解决方案，包括 DDoS 保护、入侵检测系统（IDS）和全面的网络安全服务。

Gcore于7月从机构和战略投资者那里获得了 6000 万美元的 A 轮融资。此轮融资由 Wargaming 主导，Constructor Capital 和 Han River Partners 参与，这是公司成立十多年来首次获得外部融资。

公司网址：

https://gcore.com/

**Intezer**

**融资金额：3300 万美元**

**所属领域：安全运营自动化**

Intezer 是一家成立于 2015 年，总部位于纽约的网络安全公司，专注于自主安全运营。该公司致力于自动化安全警报的分类和事件响应任务，利用人工智能提升安全运营的效率。

Intezer 在9月份获得了 3300 万美元的 C 轮融资，使其总融资额增至 6000 万美元。此轮融资由 Norwest Venture Partners 主导，所有现有投资者均参与，包括英特尔资本、OpenView、Magma 和 CyberArk 创始人 Alon Cohen。

公司网址：

https://www.intezer.com/

**Lakera**

**融资金额：2000万美元**

**所属领域：AI应用安全**

Lakera 是一家前沿的网络安全公司，专注于为生成式人工智能（GenAI）应用提供实时安全解决方案。该公司由前谷歌和 Meta 的工程师创立，总部位于瑞士苏黎世，因其创新的 AI 安全方法而迅速获得认可。

Lakera在7月完成了一轮 2000万美元的 A 轮融资。由欧洲风险投资公司 Atomico 主导，Citi Ventures、Dropbox Ventures 以及现有投资者 redalpine 也参与，这笔投资使 Lakera 的总融资额达到 3000 万美元。

公司网址：

https://www.lakera.ai/

**Lineaje**

**融资金额：2000 万美元**

**所属领域：供应链安全**

Lineaje 是一家专注于持续的软件供应链安全管理。该公司成立的使命是增强软件应用的安全性和治理，提供创新解决方案，帮助组织管理与开源和专有软件组件相关的风险。

Lineaje 于7月宣布完成了一轮具有里程碑意义的2000万美元 A 轮融资，由 Prosperity7 Ventures、Neotribe 和 Hitachi Ventures 主导，现有投资者 Tenable Ventures 也参与了。

公司网址：

https://lineaje.com/

**Command Zero**

**融资金额：2100 万美元**

**所属领域：网络调查**

Command Zero是一家创新的网络安全初创公司，专注于提升网络调查的效率和有效性。成立于 2022 年，总部位于德克萨斯州奥斯汀，Command Zero 旨在解决安全运营团队在应对网络威胁时面临的重大瓶颈。

Command Zero在7月份以隐身状态获得 2100 万美元的种子融资，从此公开其业务和产品。此轮融资由 Andreessen Horowitz 主导，Insight Partners 及 60 多位网络行业思想领袖和高管参与。

公司网址：

https://www.commandzero.io/

**Harmonic Security**

**融资金额：1750万美元**

**所属领域：AI应用安全**

Harmonic Security是一家前瞻性的网络安全公司，专注于为生成式人工智能时代提供零接触数据保护解决方案。该公司成立的目的是应对因快速采用生成式人工智能技术而产生的独特安全挑战，旨在使组织能够在保护敏感数据的同时安全地进行创新。

Harmonic Security在10月份获得了1750万美元的 A 轮融资，以将其"零接触数据保护"能力带入企业。自去年10 月推出以来，总融资额已超过 2600万美元，企业客户数量已超过十家。

公司网址：

https://www.harmonicsecurity.com/

**P0 Security**

**融资金额：1500 万美元**

**所属领域：云原生安全**

P0 Security是一家充满活力的网络安全公司，成立于2022年，总部位于加利福尼亚州旧金山。该公司专注于云原生可观察性、安全性和网络，特别强调云访问和权限管理。

P0 Security 在9月完成的A轮融资中获得了 1500 万美元，使其自成立以来的总融资额增至 2000 万美元。此笔投资由 SYN Ventures 主导，云安全领导者 Zscaler 和现有投资者 Lightspeed Venture Partners 参与。

公司网址：

https://p0.dev/

**Apono**

**融资金额：1550 万 美元**

**所属领域：云安全**

Apono是致力于通过其先进的即时（Just-in-Time, JIT）和适度（Just-Enough）访问管理解决方案来增强云访问安全。Apono于2022年创立，迅速在现代云环境中专门针对特权访问管理（PAM）领域确立了领导地位。

Apono 于10月在新纪元资本合伙公司的主导下完成了 1550万美元的 A 轮融资，参与者包括 Mindset Ventures、Redseed Ventures、Silvertech Ventures、早期种子投资者等。

公司网址：

https://www.apono.io/

**RunSafe Security**

**融资金额：1200 万**

**所属领域：嵌入式系统安全**

RunSafe Security 是一家专注于嵌入式系统安全的公司，其使命是帮助企业保护其嵌入式系统和物联网设备，防止恶意软件和未经授权的代码执行。

RunSafe Security 在9月份获得了 1200 万美元的 B 轮融资。这轮融资包括新老投资者的参与，将加速新产品的开发和市场扩展至 EMEA 及亚太地区。

公司网址：

https://runsafesecurity.com/

**Tamnoon**

**融资金额：1200 万美元**

**所属领域：云安全**

Tamnoon专注于托管云安全修复，旨在通过混合的人机智能方法来应对云安全管理的复杂性。

Tamnoon于9月获得了 1200 万美元的 A 轮融资。此轮融资由网络安全投资公司 Bright Pixel Capital（前身为 Sonae IM）主导，新投资者 Blu Ventures 和 Mindset Ventures 以及现有投资者 Merlin Ventures、Secret Chord Ventures、Inner Loop Capital 和 Elron Ventures 也参与。

公司网址：

https://tamnoon.io

**Heeler Security**

**融资金额：850 万美元**

**所属领域：云安全**

Heeler Security是一家创新的应用安全初创公司，专注于改变组织保护其应用程序免受不断演变的网络威胁的方式。Heeler于2023年6月成立，旨在通过其独特的方法和技术来应对现代应用安全的复杂性。

Heeler Security 在 Norwest Venture Partners 的主导下获得了 850 万美元的种子融资，Storm Ventures也积极参与。

公司网址：

https://www.heeler.com/

**Hydden**

**融资金额：440 万美元**

**所属领域：身份攻击面管理**

Hydden是一家新兴的网络安全公司，专注于身份攻击面管理，旨在增强对混合IT环境中身份的可见性和治理。Hydden成立于2024年，旨在应对与身份管理相关的复杂性和风险，特别是在组织面临越来越多的人类和非人类账户时。

Hydden 在9月完成了由 Access Venture Partners主导的440万美元种子融资。其他投资者包括CISO Rinki Sethi和Lucas Moody的风险基金 Lockstep、Service Provider Capital 以及几位网络安全领域的天使投资者，如 Andy Grolnick 和 Paul Trulove。

公司网址：

https://hydden.com/

**Opnova**

**融资金额：375 万美元**

**所属领域：安全运营自动化**

Opnova是一家创新的网络安全初创公司，专注于通过其先进的代理AI平台自动化IT、安全和合规操作。Opnova于2024年9月正式推出，旨在应对操作工作流程中重复任务带来的重大挑战，这些任务可能会妨碍效率并增加错误的风险。

Opnova在9月份获得了由 Faber、ScaleX 和 Preface Ventures 共同主导的 375 万美元预种子融资，旨在通过解决重复、单调但又至关重要的任务，重新定义 IT 运营。

公司网址：

https://opnova.ai/

**三季度网安领域融资特点分析**

上述网络安全创业公司的融资信息，我们可以发现三季度网络安全领域的融资呈现以下特点：

* **电子邮件安全备受关注。** Abnormal Security 专注于 AI 驱动的邮件安全，获得 2.84 亿美元融资，反映了邮件仍是主要的网络攻击途径。
* **软件供应链安全持续受重视。** Chainguard 、Lineaje 等公司获得亿元级融资,说明供应链安全问题的重要性与日俱增。
* **云安全和云原生安全的关注度不减。** Gcore 、Apono、P0 Security 、Heeler Security 等公司的融资表明，云环境下的安全挑战需要新的解决方案。
* **AI 应用安全成为新的增长点。** Lakera 、Harmonic Security 瞄准生成式 AI 带来的安全风险,获得市场认可。
* **合规自动化助力企业提效。**Vanta 通过简化合规流程获得 1.5 亿美元融资，突显了自动化合规的市场需求。
* **身份安全管理成为必争之地。** Apono 、Hydden 聚焦身份管理领域，尤其是非人类账户的安全问题日益突出。
* **自动化成为安全运营的大势所趋。** Intezer 、Opnova 等公司运用 AI 实现安全运营自动化,提升效率。
* **多个细分赛道齐头并进，**网络安全创新方兴未艾。从云、数据到身份、应用，再到供应链、嵌入式系统等，创业公司覆盖网络安全的方方面面，不断涌现新的解决方案。
