## 微软 .NET 运行时加入初步支持 RISC-V 处理器的代码

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2023年02月24日 18时27分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74228&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%20.NET%20%E8%BF%90%E8%A1%8C%E6%97%B6%E5%8A%A0%E5%85%A5%E5%88%9D%E6%AD%A5%E6%94%AF%E6%8C%81%20RISC-V%20%E5%A4%84%E7%90%86%E5%99%A8%E7%9A%84%E4%BB%A3%E7%A0%81 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自时间的地图**

微软开源 .NET 运行时项目合并了初步支持 RISC-V 处理器的代码。代码已经在 RISC-V 上成功进行了测试，甚至还运行了简单的 hello world 应用。微软目前没有让 Windows 支持 RISC-V 架构的计划，代码也不是微软程序员递交的，而是三星程序员，这是他在三星工作的一部分。
https://www.phoronix.com/news/Microsoft-dotNET-RISC-V
