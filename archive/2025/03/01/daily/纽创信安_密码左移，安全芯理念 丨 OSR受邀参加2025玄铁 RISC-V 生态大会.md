# 密码左移，安全芯理念 丨 OSR受邀参加2025玄铁 RISC-V 生态大会

原创

OSR

纽创信安

2025年2月28日，阿里达摩院2025玄铁 RISC-V 生态大会于北京举办，全球顶尖行业智库、企业决策者与 RISC-V 精英汇聚，解码玄铁生态前沿突破，共探产业协同路径。

OSR作为玄铁优选伙伴，公司 CEO 樊俊锋博士受邀分享了关于 RISC-V 集成密码全栈解决方案的前沿探索，特别是“密码左移”理念的创新实践，为芯片安全提供了全新的视角与解决方案。

![image.png](https://mmbiz.qpic.cn/sz_mmbiz_png/eEicdLAdfSLLcLVibR3Mxt19tNDj3T7cb0GQtzqHSoNk6YuIpsLr3Ejk2HBmscZ5NO1yxibBialrVDC3N5SrpmkSfw/640?from=appmsg)

**密码左移，以始为终的芯片安全设计**

密码左移‌是芯片安全领域的全新范式，其本质是‌“将防御阵地前置到数字世界的起点，并链接芯片安全的全生命周期”‌。传统安全模式如同在数据洪流中“筑坝堵漏，哪里有漏堵哪里”。但安全不应该是亡羊补牢，而是未雨绸缪，密码左移是‌在芯片诞生之初，就注入安全的基因‌——以密码技术为信任锚点，形成从 IP Vendor 到应用生态的全链路防护体系。

![image.png](https://mmbiz.qpic.cn/sz_mmbiz_png/eEicdLAdfSLLcLVibR3Mxt19tNDj3T7cb00HQqp2vouibWPaSz4cexTxaeMUlRRO5ic2oAMfN0SSmIKTM950icLERRw/640?from=appmsg)

**生态共建，0SR+玄铁一站式安全解决****方案**

OSR同玄铁生态共建、深度协同，以‌“安全标准全域穿透+芯片矩阵场景适配”‌为核心，基于国密认证、FIPS 140-3、PSA、SESIP等国内与国际安全标准，覆盖低能耗 MCU 芯片、高性能应用芯片、安全芯片、车规芯片等多类芯片，形成"端-边-云-车"全场景的全栈芯片安全解决方案，打造智能化时代万物互联的安全底座。

![image.png](https://mmbiz.qpic.cn/sz_mmbiz_png/eEicdLAdfSLLcLVibR3Mxt19tNDj3T7cb0uPWcj5vsSCKGdQxkFLzrh1XhjBeJWH3lLibFqJpxUGo0tmzFibHCvz3w/640?from=appmsg)

**创新实践，轻量级eHSM+整体解决方案发布**

樊俊锋博士在演讲中正式发布OSR携手玄铁推出的轻量级eHSM+安全整体解决方案。OSR在信息安全领域深耕多年，具备密码服务的全栈体系优势，旗下 eHSM 已获ISO 26262 ASIL-D管理体系、ASIL-B产品认证、ASIL-D软件流程认证，实现‌信息安全与功能安全的芯片级融合‌，以密钥为中心，以轻量化、高性能、全生命周期安全为核心优势，助力客户芯片成功。

![image.png](https://mmbiz.qpic.cn/sz_mmbiz_png/eEicdLAdfSLLcLVibR3Mxt19tNDj3T7cb0zFe0uXGPcqEiaibpGa36LXsKCjzwQvxwrx5xGfAlPhBJs4FJmjndXMGQ/640?from=appmsg)

樊俊锋博士在演讲中展望了未来安全技术趋势，他指出，随着AI技术对云端与终端数据隐私保护要求的指数级提升，芯片安全技术将加速向‌可信根（ROT）‌与‌可信执行环境（TEE）‌双核心架构演进，构建完整的软件生态链。RISC-V 生态的开放性使得“密码左移”更加容易实现，进一步降低了导入成本，随着 RISC-V 生态的持续发展，硬件安全将成为其显著特色，推动行业实现更高效、安全的技术创新。

**关于纽创信安：**

深圳市纽创信安科技开发有限公司（简称“纽创信安”或“OSR”)成立于2014年，总部位于深圳，在北京、上海设有分/子公司。公司专注于新一代密码关键技术的创新突破与核心产品研发，为云计算、数据要素以及网络安全领域提供创新的商用密码产品和解决方案，致力于成为全球领先的密码服务商。

纽创信安是国家高新技术企业和深圳市高新技术企业，通过了ISO9001质量管理体系认证、ISO20000信息技术服务管理体系认证、ISO27001信息安全管理体系认证。公司注重研发投入和技术创新，业务覆盖从“芯”到“云”的密码应用场景，提供的产品和服务包括高安全高性能密码算法IP、片内安全子系统、芯片安全检测设备、高安全国密产品、密码态势感知平台和密码靶场等，目前已获得17张密码型号证书，其中信创安全三级密码卡和密码态势感知平台均为国内首创。

纽创信安拥有深厚的技术积淀和产学研融合基础，与清华大学、浙江大学等知名高校开展了产学研转化合作，在密码算法设计分析、密码芯片设计实现、密码算法快速实现、同态密码等领域处于国际领先水平。公司是全国信息安全标准化技术委员会、工业和信息化部密码应用推进标准工作组成员单位，牵头和参与制定了多项密码领域国家标准或行业标准。
