## Firefox 108 发布

[![Firefox](https://icon.solidot.org/images/topics/topicfirefox.png?123)](/search?tid=39 "Firefox")

[Wilson](/~Wilson) (42865)发表于 2022年12月14日 17时43分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73649&appkey=1370085986&title=Firefox%20108%20%E5%8F%91%E5%B8%83 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自勇者物语**

Mozilla 释出了 Firefox 108。主要新特性包括：默认使用 Import maps，允许网页控制 JavaScript imports 的行为；在 Windows 11 下后台标签使用的进程将使用效率模式限制资源使用；shift+esc 快捷键将打开进程管理器快速识别使用太多资源的进程；改进帧调度；支持 Web MIDI API；一系列安全修正，等等。
https://www.mozilla.org/en-US/firefox/108.0/releasenotes/
https://github.com/WICG/import-maps#readme
