# 今天给大家推送一份文件，列举了美国国际开发暑、司法部、农业部、商务部、国防部、教育部、能源部、卫生与公共服务部、国土安全部、住房和城市发展部、劳工部、国务院、内政部、财政部、交通部、退伍军人事务部、环境保护署、总统行政办公室、联邦通信委员会、总务管理局、联邦航空航天局、国会图书馆、小型企业管理局、社会保障局、以及各种基金会和委员会的对外援助项目。

# 本文是一份关于联邦财政援助计划分析的指导文件，旨在支持M-25-13指令。所有提供联邦财政援助的联邦机构被要求在2025年2月7日前完成附带的电子表格，并提交给OMB（美国行政管理和预算局）。该信息适用于任何有资金或活动计划至2025年3月15日的项目。机构被鼓励尽快完成此任务，因为OMB将为后续期间设定额外的截止日期。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wzxONx3Tzbrxgkf8Kbm8iabeq414Oa8wJSibSvackvkicO2kKGQpn0unxCn1swOWInESwYA2F8ANZow/640?wx_fmt=png&from=appmsg)

**提交要求**

* **截止日期**：所有联邦机构必须在2025年2月7日前完成并提交电子表格。
* **适用范围**：涉及所有有资金或活动计划至2025年3月15日的项目。
* **后续跟进**：OMB将设定后续期间的额外截止日期。

**电子表格填写指南**

* **定位或筛选项目**：使用提供的电子表格，通过辅助列表编号定位或筛选机构的联邦项目。
* **填写信息**：在C-G列中填写每个项目的请求信息。
* **添加未列出项目**：如果电子表格中没有列出某些项目，机构应添加额外行以包含这些项目。

**各列具体内容**

* **列B**：子机构或组成部分。
* **列C-G**：预填充的每个辅助列表项目的相关信息。
* **列H**：负责监督该项目的高级政治任命官员的电子邮件。
* **列I**：该项目是否有任何待发布的资金公告。
* **列J**：该项目是否有任何预计在2025年3月15日前的资金义务或拨款。
* **列K**：该项目是否有任何强制性法律要求在2025年3月15日前拨款或支付资金。
* **列L**：下一个义务或支付资金的估计日期。
* **列M-T**：回答每个问题（是/否）。
* **列U**：提供有关项目或活动的任何其他相关信息。

**具体项目示例**

* **美国国际开发署（USAID）项目**：

+ 外国援助计划、合作社发展计划、海洋货运报销计划等。

* **阿巴拉契亚地区委员会项目**：

+ 阿巴拉契亚区域发展、阿巴拉契亚地区发展、阿巴拉契亚发展高速公路系统等。

* **消费者产品安全委员会项目**：

+ 维吉尼亚·格雷姆·贝克游泳池和水疗安全、尼古拉斯和扎卡里·伯特纪念一氧化碳中毒预防计划等。

* **国家和社区服务公司项目**：

+ 美国老年志愿者计划、州委员会支持赠款、国家和国家赠款等。

* **三角洲地区管理局项目**：

+ 三角洲地区管理局。

* **丹奈利委员会项目**：

+ 丹奈利委员会计划。

* **农业部项目**：

+ 农业研究、植物和动物疾病、害虫控制和动物护理、野生动物服务等。

**提交方式**

* **提交地址**：将电子表格发送至FFAPlans@omb.eop.gov。
* **审查与反馈**：OMB将审查每个机构的提交，并通过机构高级财政援助官员和提供的联系点传达更多信息。

### 总结

本文是一份详细的指导文件，要求所有提供联邦财政援助的联邦机构在规定的时间内完成并提交一个特定的电子表格。该表格用于收集有关各个项目的详细信息，包括项目的负责人、资金状况、法律要求等。机构需要根据提供的模板填写相关信息，并在指定的截止日期前提交。此外，文件还提供了多个具体项目的示例，以便机构更好地理解和执行任务。核心观点在于确保所有相关机构能够按时提供准确的信息，以便OMB进行后续的审查和管理。
