иҜҘжјҸжҙһиў«и·ҹиёӘдёәВ **CVE-2024-6327**пјҲCVSS иҜ„еҲҶпјҡ9.9пјүпјҢдјҡеҪұе“Қ Report Server зүҲжң¬ 2024 Q2 пјҲ10.1.24.514пјү еҸҠжӣҙж—©зүҲжң¬гҖӮ

иҜҘе…¬еҸёеңЁдёҖд»Ҫе…¬е‘ҠдёӯиЎЁзӨәпјҡвҖңеңЁ 2024 е№ҙз¬¬дәҢеӯЈеәҰ пјҲ10.1.24.709пјү д№ӢеүҚзҡ„ Telerik Report Server зүҲжң¬дёӯпјҢеҸҜиғҪдјҡйҖҡиҝҮдёҚе®үе…Ёзҡ„еҸҚеәҸеҲ—еҢ–жјҸжҙһиҝӣиЎҢиҝңзЁӢд»Јз Ғжү§иЎҢж”»еҮ»гҖӮвҖқ

еҪ“еә”з”ЁзЁӢеәҸеңЁжңӘиҝӣиЎҢе……еҲҶйӘҢиҜҒзҡ„жғ…еҶөдёӢйҮҚе»әж”»еҮ»иҖ…еҸҜд»ҘжҺ§еҲ¶зҡ„дёҚеҸ—дҝЎд»»зҡ„ж•°жҚ®ж—¶пјҢе°ұдјҡеҸ‘з”ҹеҸҚеәҸеҲ—еҢ–зјәйҷ·пјҢд»ҺиҖҢеҜјиҮҙжү§иЎҢжңӘз»ҸжҺҲжқғзҡ„е‘Ҫд»ӨгҖӮ

Progress SoftwareиЎЁзӨәпјҢиҜҘжјҸжҙһе·ІеңЁ10.1.24.709зүҲжң¬дёӯеҫ—еҲ°и§ЈеҶігҖӮдҪңдёәдёҙж—¶зј“и§ЈжҺӘж–ҪпјҢе»әи®®е°ҶжҠҘиЎЁжңҚеҠЎеҷЁеә”з”ЁзЁӢеәҸжұ зҡ„з”ЁжҲ·жӣҙж”№дёәе…·жңүжңүйҷҗжқғйҷҗзҡ„з”ЁжҲ·гҖӮ

з®ЎзҗҶе‘ҳеҸҜд»ҘйҖҡиҝҮд»ҘдёӢжӯҘйӘӨжЈҖжҹҘд»–д»¬зҡ„жңҚеҠЎеҷЁжҳҜеҗҰе®№жҳ“еҸ—еҲ°ж”»еҮ» вҖ“

иҝ‘дёӨдёӘжңҲеүҚпјҢиҜҘе…¬еҸёдҝ®иЎҘдәҶеҗҢдёҖиҪҜд»¶дёӯзҡ„еҸҰдёҖдёӘе…ій”®зјәйҷ·пјҲCVE-2024-4358пјҢCVSSВ иҜ„еҲҶпјҡ9.8пјүпјҢиҜҘжјҸжҙһеҸҜиғҪиў«иҝңзЁӢж”»еҮ»иҖ…ж»Ҙз”Ёд»Ҙз»•иҝҮиә«д»ҪйӘҢиҜҒе№¶еҲӣе»әжөҒж°“з®ЎзҗҶе‘ҳз”ЁжҲ·гҖӮ
