Hello learners!

You asked, we listened! Here begins the expert live session journey for SAP Sales Cloud!

In this 2 day Live Session you get to learn and  interact with our expert who will be presenting the various capabilities this solution provides. This will help you with certification readiness as well, for SAP Certified Application Associate Consultant certification (C\_C4H410). Each session is for 3 hrs.

(Note: This is for the Sales Cloud NGAP version, not Sales Cloud Version 2)

[![](/legacyfs/online/storage/blog_attachments/2023/05/SAP-live-session-sales.jpg)](https://saplearninghub.plateau.com/learning/user/common/viewItemDetails.do?componentTypeID=EXPERT_LED&componentID=LS_SALESCLOUDESSENTIALS_EN&revisionDate=1681437240000&catalogCurrencyCode=USD&returnLink=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D&backto=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D)

This live session is relevant for Administrators, Consultants, Business users or anyone interested in learning about SAP Sales Cloud.

You can register and book this session on Learning Hub in two ways:

(Please Note:  You must have a subscription and  login with your S-user ID on **SAP Learning Hub**.)

1. Here is the direct link to the Live Session : [SAP Sales Cloud Essentials](https://saplearninghub.plateau.com/learning/user/common/viewItemDetails.do?componentTypeID=EXPERT_LED&componentID=LS_SALESCLOUDESSENTIALS_EN&revisionDate=1681437240000&catalogCurrencyCode=USD&returnLink=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D&backto=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D)

2. Login to Learning Hub and scroll down  and click on  Browse Learning Content.  Here you can search for “Sales Cloud Essentials” and you will find it

[![](/legacyfs/online/storage/blog_attachments/2023/05/livesession.jpg)](https://saplearninghub.plateau.com/learning/user/common/viewItemDetails.do?componentTypeID=EXPERT_LED&componentID=LS_SALESCLOUDESSENTIALS_EN&revisionDate=1681437240000&catalogCurrencyCode=USD&returnLink=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D&backto=%2Flearning%2Fuser%2Fcatalogsearch%2FcatalogSearchDispatchAction.do%3FsearchType%3DfilteredSearch%26keywords%3D)

Course Details:

This program consists of two sessions – you will receive one confirmation email with all login details attached.

SAP Sales Cloud Essentials Session #1 of 2 – Time: 22nd May 2023, 16:00 CEST to 19:00 CEST

SAP Sales Cloud Essentials Session #2 of 2  – Time: 23rd May 2023, 16:00 CEST to 19:00 CEST

Recordings will be available and published after the session on Learning Hub.

If you have a **Partner Edge subscription**, you can register by clicking [here](https://partneredge.sap.com/en/library/education/psd/2023/apr/e_oe_te_w_ssce_32411.html)
