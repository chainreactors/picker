иҝҷдёӘжјҸжҙһиў«иҝҪиёӘзј–еҸ·дёә CVE-2025-27607пјҢеҲқе§Ӣзҡ„йҖҡз”ЁжјҸжҙһиҜ„еҲҶзі»з»ҹпјҲCVSSпјүиҜ„еҲҶдёә 8.8пјҢеҪұе“ҚиҜҘиҪҜд»¶еҢ…зҡ„ 3.2.0 зүҲжң¬е’Ң 3.2.1 зүҲжң¬пјҢе…¶ж №жәҗеңЁдәҺдёҖдёӘзјәеӨұзҡ„дҫқиө–йЎ№й—®йўҳгҖӮ

е®үе…Ёз ”з©¶е‘ҳ omnigodz еңЁеҜ№дҫӣеә”й“ҫж”»еҮ»иҝӣиЎҢе®һйӘҢз ”з©¶ж—¶еҸ‘зҺ°дәҶиҝҷдёӘжјҸжҙһгҖӮ

иҜҘз ”з©¶е‘ҳеҸ‘зҺ°пјҢpython-json-logger иҪҜд»¶еҢ…еңЁе…¶ pyproject.toml ж–Үд»¶дёӯеЈ°жҳҺдәҶдёҖдёӘеҗҚдёә msgspec-python313-pre зҡ„дҫқиө–йЎ№пјҢдҪҶиҝҷдёӘдҫқиө–йЎ№еңЁ Python еҢ…зҙўеј•пјҲPyPIпјүдёҠе№¶дёҚеӯҳеңЁпјҢд№ҹжңӘиў«д»»дҪ•е®һдҪ“жіЁеҶҢгҖӮ

вҖңеңЁжҲ‘зҡ„з ”з©¶иҝҮзЁӢдёӯпјҢжҲ‘еҸ‘зҺ°дәҶдёҖдёӘеҪұе“ҚPackage ManagerдёҠеҸ‘еёғзҡ„иҪҜд»¶еҢ…зҡ„йҮҚеӨ§зјәйҷ·пјҢвҖқomnigodz и§ЈйҮҠйҒ“пјҢвҖңжҲ‘иғҪеӨҹзЎ®е®ҡ PyPi иҪҜд»¶еҢ… python-json-logger дҪҝз”ЁдәҶдёҖдёӘеҸҜйҖүдҫқиө–йЎ№пјҢиҖҢиҝҷдёӘдҫқиө–йЎ№ж—ўдёҚеңЁ PyPi еӯҳеӮЁеә“дёӯпјҢд№ҹжңӘиў«д»»дҪ•е®һдҪ“жіЁеҶҢгҖӮвҖқ

иҜҘжјҸжҙһдә§з”ҹзҡ„еҺҹеӣ жҳҜпјҢmsgspec-python313-pre дҫқиө–йЎ№е·Іиў«е…¶жүҖжңүиҖ…еҲ йҷӨпјҢдҪҝеҫ—д»»дҪ•дәәйғҪеҸҜд»ҘдҪҝз”ЁиҝҷдёӘеҗҚз§°гҖӮиҝҷе°ұйҖ жҲҗдәҶдёҖз§Қжғ…еҶөпјҢеҚіжҒ¶ж„ҸиЎҢдёәиҖ…еҸҜд»ҘеҸ‘еёғдёҖдёӘеҗҢеҗҚзҡ„еҢ…еҗ«жңүе®ід»Јз Ғзҡ„иҪҜд»¶еҢ…гҖӮ

****жҰӮеҝөйӘҢиҜҒпјҲPoCпјү****

еҪ“з”ЁжҲ·еңЁ Python 3.13 зҺҜеўғдёӯдҪҝз”Ёе‘Ҫд»Ө pip install python-json-logger [dev] е®үиЈ… python-json-logger еҸҠе…¶дҫқиө–жҖ§гҖҒйЎ№ж—¶пјҢд»–д»¬дјҡеңЁдёҚзҹҘдёҚи§үдёӯдёӢиҪҪе№¶жү§иЎҢжҒ¶ж„ҸиҪҜд»¶еҢ…гҖӮ

дёәдәҶеңЁдёҚйҖ жҲҗеҚұе®ізҡ„жғ…еҶөдёӢжј”зӨәиҝҷдёӘжјҸжҙһпјҢиҜҘз ”з©¶е‘ҳдёҙж—¶еҸ‘еёғдәҶдёҖдёӘеҗҢеҗҚзҡ„йқһжҒ¶ж„ҸиҪҜд»¶еҢ…пјҢз„¶еҗҺе°Ҷе…¶еҲ йҷӨпјҢжңүж•Ҳең°йҳ»жӯўдәҶжҒ¶ж„ҸиЎҢдёәиҖ…еҲ©з”ЁиҜҘжјҸжҙһгҖӮиҜҘз ”з©¶е‘ҳдҝқз•ҷдәҶиҝҷдёӘиҪҜд»¶еҢ…еҗҚз§°зҡ„жіЁеҶҢпјҢд»ҘйҳІжӯўе…¶д»–дәәжҒ¶ж„ҸдҪҝз”Ёе®ғгҖӮ

вҖңиҝҷдёӘеҸ‘еёғзҡ„иҪҜд»¶еҢ…жІЎжңүд»»дҪ•жҒ¶ж„ҸеҶ…е®№пјҢеӣ дёәжҲ‘дёҚжғіиҝқеҸҚ PyPi.org еҲ¶е®ҡзҡ„д»»дҪ•ж”ҝзӯ–пјҢд№ҹдёҚжғіж„ҹжҹ“ python-json-logger иҪҜд»¶еҢ…зҡ„д»»дҪ•з”ЁжҲ·пјҢвҖқ иҜҘз ”з©¶е‘ҳиЎЁзӨәгҖӮ

python-json-logger иҪҜд»¶еҢ…жһҒе…¶еҸ—ж¬ўиҝҺпјҢж №жҚ® PyPI зҡ„ BigQuery ж•°жҚ®еә“жҳҫзӨәпјҢе…¶жҜҸжңҲзҡ„дёӢиҪҪйҮҸи¶…иҝҮ 4300 дёҮж¬ЎгҖӮиҝҷз§Қе№ҝжіӣзҡ„дҪҝз”Ёжҳҫи‘—ж”ҫеӨ§дәҶиҜҘжјҸжҙһеҸҜиғҪйҖ жҲҗзҡ„еҪұе“ҚгҖӮ

иҝҷдёӘй—®йўҳзҡ„еҮәзҺ°жҳҜеӣ дёәпјҢе°Ҫз®ЎеӨ§зәҰдёҖдёӘжңҲеүҚйҖҡиҝҮжҸҗдәӨи®°еҪ• 1ce81a3 д»ҺйЎ№зӣ®еӯҳеӮЁеә“дёӯеҲ йҷӨдәҶиҜҘдҫқиө–йЎ№пјҢдҪҶеңЁ 3.2.1 зүҲжң¬д№ӢеҗҺпјҢиҝҷдәӣжӣҙж”№д»ҺжңӘиў«жҺЁйҖҒеҲ°ж–°зүҲжң¬дёӯпјҢдҪҝеҫ—иҝҷдәӣзүҲжң¬зҡ„з”ЁжҲ·йқўдёҙжјҸжҙһйЈҺйҷ©гҖӮ

жҜ”еҲ©ж—¶зҪ‘з»ңе®үе…ЁдёӯеҝғеҸ‘еёғдәҶдёҖеҲҷиӯҰе‘ҠпјҢз§°пјҡвҖңжҲ‘д»¬ејәзғҲе»әи®®еңЁз»ҸиҝҮе…ЁйқўжөӢиҜ•еҗҺпјҢдёәе…·жңүжңҖй«ҳдјҳе…Ҳзә§зҡ„жҳ“еҸ—ж”»еҮ»зҡ„и®ҫеӨҮе®үиЈ…жӣҙж–°гҖӮвҖқ

****зј“и§ЈжҺӘж–Ҫ****

иҜҘжјҸжҙһе·ІеңЁ 3.3.0 зүҲжң¬дёӯеҫ—еҲ°дҝ®еӨҚпјҢз”ЁжҲ·еә”з«ӢеҚіеҚҮзә§еҲ°иҜҘзүҲжң¬гҖӮжӯӨеӨ–пјҢPyPI з®ЎзҗҶе‘ҳе·ІйҮҮеҸ–жҺӘж–Ҫйҳ»жӯўзјәеӨұиҪҜд»¶еҢ…зҡ„еҗҚз§°иў«дҪҝз”ЁпјҢйҳІжӯўе…¶иў«жҒ¶ж„ҸиЎҢдёәиҖ…йҮҚж–°еҚ з”ЁгҖӮз”ұдәҺиҝҷдәӣдҝқжҠӨжҺӘж–ҪпјҢиҜҘжјҸжҙһзҡ„дёҘйҮҚзЁӢеәҰиҜ„зә§е·Ід»Һ вҖңй«ҳвҖқ йҷҚиҮі вҖңдҪҺвҖқгҖӮ

е»әи®®дҪҝз”ЁеҸ—еҪұе“Қеә“зҡ„з»„з»ҮжЈҖжҹҘе…¶дҫқиө–йЎ№пјҢеҚҮзә§еҲ°е·Ідҝ®еӨҚзҡ„зүҲжң¬пјҢе№¶зӣ‘жҺ§жҳҜеҗҰеӯҳеңЁеҸҜз–‘жҙ»еҠЁгҖӮеҰӮжһң msgspec-python313-pre д»ҚжҳҜжӮЁзҡ„дҫқиө–йЎ№д№ӢдёҖпјҢеә”з«ӢеҚіе°Ҷе…¶еҲ йҷӨгҖӮ
