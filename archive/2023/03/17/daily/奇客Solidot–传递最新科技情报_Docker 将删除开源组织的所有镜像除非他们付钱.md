## Docker 将删除开源组织的所有镜像除非他们付钱

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")
[![商业](https://icon.solidot.org/images/topics/topicbiz.png?123)](/search?tid=54 "商业")

[Wilson](/~Wilson) (42865)发表于 2023年03月16日 11时29分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74406&appkey=1370085986&title=Docker%20%E5%B0%86%E5%88%A0%E9%99%A4%E5%BC%80%E6%BA%90%E7%BB%84%E7%BB%87%E7%9A%84%E6%89%80%E6%9C%89%E9%95%9C%E5%83%8F%E9%99%A4%E9%9D%9E%E4%BB%96%E4%BB%AC%E4%BB%98%E9%92%B1 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自喀迈拉空间**

Docker 向所有 Docker Hub 用户发去邮件，如果他们是以组织的名义创建账号，那么他们的账号将被删除，所有镜像也将一并删除，除非他们升级到一个付费的团队方案——其年费为 420 美元。Docker Hub 被很多开源项目用于发布镜像，而绝大部分开源项目是没有多少资金的。此举引发了争议，很多人开始将其镜像迁移出 Docker 转到其它平台或自托管。没有自己的基础设施依赖于私有平台都会存在风险。
https://blog.alexellis.io/docker-is-deleting-open-source-images/
