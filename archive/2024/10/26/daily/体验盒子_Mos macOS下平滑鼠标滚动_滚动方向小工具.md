## Mos

一个用于在MacOS上平滑你的鼠标滚动效果的小工具, 让你的滚轮爽如触控板。

## 特性

* 疯狂平滑你的鼠标滚动效果
* 支持分离触控板/鼠标事件, 单独翻转鼠标滚动方向。
* 滚动曲线的自定义调整。
* 支持区分应用处理, 黑/白名单系统。
* 用于监控滚动事件的图形化呈现窗口。
* 基于 Swift4 构建。

## 下载及安装

### Homebrew安装

Mos可通过[Homebrew](https://brew.sh/)来安装:

|  |  |
| --- | --- |
| 1 | $ brew install mos |

应用将被安装至
`/Applications/Mos.app`
。

若有新版本,可用以下命令升级:

|  |  |
| --- | --- |
| 1  2 | $ brew update  $ brew reinstall mos |

重新启动应用即可。
