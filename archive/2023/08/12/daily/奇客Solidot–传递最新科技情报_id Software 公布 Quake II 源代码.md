[Wilson](/~Wilson) (42865)发表于 2023年08月11日 11时49分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75766&appkey=1370085986&title=id%20Software%20%E5%85%AC%E5%B8%83%20Quake%20II%20%E6%BA%90%E4%BB%A3%E7%A0%81 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自夏日永别**

id Software 公布了 Quake II 源代码。源代码采用 GNU GPLv2.0 许可证，托管在 GitHub 上。id Software 表示它不再提供任何支持，但感兴趣的用户可以利用现有的社区资源。Quake II 源代码包含了新版 API 用于在服务器和游戏模组之间通信，引入了一个瘦 client game 模组，类似 Quake III Arena 的 cgame 模组，以及一个 2023 年版的网络协议。
https://github.com/id-Software/quake2-rerelease-dll
