## Google 开始向所有 Chrome 用户推送隐私沙盒 API

[![Google](https://icon.solidot.org/images/topics/topicgoogle.png?123)](/search?tid=26 "Google")

[Wilson](/~Wilson) (42865)发表于 2023年07月21日 15时54分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75577&appkey=1370085986&title=Google%20%E5%BC%80%E5%A7%8B%E5%90%91%E6%89%80%E6%9C%89%20Chrome%20%E7%94%A8%E6%88%B7%E6%8E%A8%E9%80%81%E9%9A%90%E7%A7%81%E6%B2%99%E7%9B%92%20API "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自团圆奇遇**

随着 Chrome 115 的发布，Google 宣布开始向所有 Chrome 用户启用隐私沙盒相关 API。隐私沙盒（Privacy Sandbox）将取代第三方 Cookie，它跟踪用户的兴趣，允许广告商根据兴趣展示广告，而用户可以管理兴趣并对其归类分组。隐私沙盒代表着广告行业的一次巨大变化。Chrome 115 标志着隐私沙盒相关的 API GA（普遍可用），开发者可对其进行测试。Google 的目标是到 8 月底对 99% 的用户启用隐私沙盒相关 API，第三方 Cookie 将会继续保留一段时间。从 2024 年开始，Google 将对 1% 的 Chrome 用户禁用第三方 Cookie，然后到 2024 年底完全淘汰第三方 Cookie。
https://techcrunch.com/2023/07/20/google-starts-the-ga-rollout-of-its-privacy-sandbox-apis-to-all-chrome-users/
