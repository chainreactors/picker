## Firefox 139.0 释出

[![Firefox](https://icon.solidot.org/images/topics/topicfirefox.png?123)](/search?tid=39 "Firefox")

[Wilson](/~Wilson) (42865)发表于 2025年05月26日 23时52分 星期一 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81397&appkey=1370085986&title=Firefox%20139.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自巴比伦Ⅰ：女人**

Mozilla 释出了 Firefox 139.0。主要新特性包括：显著改进了 HTTP/3 连接的上传性能；支持扩展页的整页翻译；粘贴时保留 PNG 图像的透明度；支持 WebAuthn largeBlob 扩展；默认禁用的实验性功能如优先任务调度、View Transition API 等等。
