иҝҷдәӣе®үе…ЁжјҸжҙһеҜ№еңЁ XenServer е’Ң Citrix Hypervisor е№іеҸ°дёҠиҝҗиЎҢ Windows иҷҡжӢҹжңәзҡ„дјҒдёҡзҺҜеўғжһ„жҲҗйҮҚеӨ§йЈҺйҷ©гҖӮ

## Xen Windows PV й©ұеҠЁзЁӢеәҸзјәйҷ·

иҝҷдәӣжјҸжҙһжәҗдәҺеҜ№ Windows PV й©ұеҠЁзЁӢеәҸдёӯз”ЁжҲ·е…¬ејҖзҡ„и®ҫеӨҮзҡ„жқғйҷҗиҝҮеӨҡпјҢзү№еҲ«жҳҜеҪұе“ҚдёүдёӘж ёеҝғз»„д»¶пјҡXenConsгҖҒXenIface е’Ң XenBusгҖӮ

ж №жҚ®е®үе…Ёе…¬е‘ҠпјҢиҝҷдәӣз»„д»¶вҖңжІЎжңүе®үе…ЁжҸҸиҝ°з¬ҰпјҢеӣ жӯӨйқһзү№жқғз”ЁжҲ·е®Ңе…ЁеҸҜд»Ҙи®ҝй—®вҖқгҖӮ

XenCons й©ұеҠЁзЁӢеәҸжјҸжҙһ пјҲCVE-2025-27462пјү йҰ–ж¬ЎеңЁзүҲжң¬ 9.0.0 дёӯеј•е…ҘпјҢиҮӘйҰ–ж¬ЎеҸ‘еёғд»ҘжқҘдёҖзӣҙе®№жҳ“еҸ—еҲ°ж”»еҮ»гҖӮ

XenIface пјҲCVE-2025-27463пјү е’Ң XenBus пјҲCVE-2025-27464пјү й©ұеҠЁзЁӢеәҸеңЁжүҖжңүзүҲжң¬дёӯйғҪе®№жҳ“еҸ—еҲ°ж”»еҮ»пјҢиҝҷдҪҝеҫ—жӯӨй—®йўҳжҲҗдёәеҪұе“Қдј—еӨҡдјҒдёҡйғЁзҪІзҡ„е№ҝжіӣй—®йўҳгҖӮ

еҸ—еҪұе“Қзҡ„зі»з»ҹеҢ…жӢ¬еңЁ XenServer 8.4 е’Ң Citrix Hypervisor 8.2 CU1 LTSR дёҠиҝҗиЎҢзҡ„ Windows иҷҡжӢҹжңәгҖӮ

е…·дҪ“иҖҢиЁҖпјҢж—©дәҺ 9.0.9065 зҡ„ XCP-ng PV жҖ»зәҝгҖҒXCP-ng жҺҘеҸЈе’Ң XCP-ng PV жҺ§еҲ¶еҸ°зүҲжң¬е®№жҳ“еҸ—еҲ°ж”»еҮ»пјҢиҖҢж—©дәҺ 9.1.11.115 зҡ„ XenServer/Citrix PV жҖ»зәҝзүҲжң¬е’Ңж—©дәҺ 9.1.12.94 зҡ„ PV жҺҘеҸЈзүҲжң¬д№ҹеӯҳеңЁйЈҺйҷ©гҖӮ

иҝҷдәӣжјҸжҙһдҪҝ Windows жқҘе®ҫдҪңзі»з»ҹдёӯзҡ„йқһзү№жқғз”ЁжҲ·иғҪеӨҹе°Ҷзү№жқғеҚҮзә§еҲ°жқҘе®ҫеҶ…ж ёзҡ„жқғйҷҗгҖӮиҝҷиЎЁзӨәдёҘйҮҚзҡ„е®үе…ЁжјҸжҙһпјҢеӣ дёәе…·жңүжңүйҷҗи®ҝй—®жқғйҷҗзҡ„ж”»еҮ»иҖ…еҸҜд»Ҙе®Ңе…ЁжҺ§еҲ¶еҸ—еҪұе“Қзҡ„иҷҡжӢҹжңәгҖӮ

иҝҷдәӣжјҸжҙһзҡ„ CVSSv4.0 иҜ„еҲҶдёә 5.9пјҢж №жҚ®дёҖдәӣиҜ„дј°иў«еҪ’зұ»дёәвҖңдҪҺвҖқйЈҺйҷ©пјҢдҪҶе®һйҷ…еҪұе“ҚеҫҲдёҘйҮҚгҖӮ

еҲ©з”Ёиҝҷдәӣзјәйҷ·зҡ„ж”»еҮ»иҖ…еҸҜд»Ҙд»Ҙзі»з»ҹзә§жқғйҷҗжү§иЎҢд»»ж„Ҹд»Јз ҒпјҢд»ҺиҖҢеҸҜиғҪжі„йңІж•Ҹж„ҹж•°жҚ®гҖҒе®үиЈ…жҒ¶ж„ҸиҪҜд»¶жҲ–е°ҶеҸ—ж„ҹжҹ“зҡ„ VM з”ЁдҪңеңЁзҪ‘з»ңеҶ…жЁӘеҗ‘з§»еҠЁзҡ„жһўиҪҙзӮ№гҖӮ

еҲ©з”ЁеӘ’д»ӢжҳҜжң¬ең°зҡ„пјҢиҝҷж„Ҹе‘ізқҖж”»еҮ»иҖ…еҝ…йЎ»е·Із»ҸеҜ№ Windows жқҘе®ҫзі»з»ҹе…·жңүдёҖе®ҡзә§еҲ«зҡ„и®ҝй—®жқғйҷҗгҖӮ

дҪҶжҳҜпјҢжӯӨйҷҗеҲ¶е№¶дёҚиғҪжҳҫи‘—йҷҚдҪҺеЁҒиғҒпјҢеӣ дёәи®ёеӨҡж”»еҮ»еңәжҷҜж¶үеҸҠйҖҡиҝҮзҪ‘з»ңй’“йұјгҖҒжҒ¶ж„ҸиҪҜд»¶жҲ–е…¶д»–жҸҗдҫӣеҝ…иҰҒз«Ӣи¶ізӮ№зҡ„еӘ’д»ӢиҝӣиЎҢеҲқе§Ӣе…ҘдҫөгҖӮ

## зј“и§ЈжҺӘж–Ҫ

Citrix е’Ң XenServer еҸ‘еёғдәҶйҖӮз”ЁдәҺ Windows зҡ„ XenServer VM Tools зүҲжң¬ 9.4.1 жқҘи§ЈеҶіиҝҷдәӣжјҸжҙһгҖӮ

жӣҙж–°еҗҺзҡ„е·Ҙе…·еҢ…еҗ«зү№е®ҡзҡ„з»„д»¶зүҲжң¬пјҢеҢ…жӢ¬ xenbus 9.1.11.115гҖҒxeniface 9.1.12.94 е’Ңе…¶д»–дҝ®иЎҘзҡ„й©ұеҠЁзЁӢеәҸгҖӮ

з®ЎзҗҶе‘ҳеә”з«ӢеҚійҖҡиҝҮеӨҡз§ҚеҸҜз”Ёжё йҒ“е°ҶжүҖжңү Windows VM жӣҙж–°еҲ°жңҖж–°зҡ„ XenServer VM Tools зүҲжң¬пјҡд»Һ Citrix ж”ҜжҢҒзӣҙжҺҘдёӢиҪҪгҖҒWindows Update жңәеҲ¶жҲ– Management Agent иҮӘеҠЁжӣҙж–°еҠҹиғҪгҖӮ

дҪҝз”Ё Windows Update зҡ„з»„з»Үеә”йӘҢиҜҒ вҖңйҖҡиҝҮ Windows Update з®ЎзҗҶ Citrix PV й©ұеҠЁзЁӢеәҸвҖқ жҳҜеҗҰе·ІеҗҜз”ЁгҖӮ

еҜ№дәҺж— жі•з«ӢеҚідҝ®иЎҘзҡ„зҺҜеўғпјҢеҸҜд»ҘдҪҝз”Ё PowerShell зј“и§Ји„ҡжң¬пјҢиҜҘи„ҡжң¬еҸҜд»ҘйҖҡиҝҮеңЁжіЁеҶҢиЎЁдёӯжҸ’е…ҘйҖӮеҪ“зҡ„е®үе…ЁжҸҸиҝ°з¬ҰжқҘжү«жҸҸжјҸжҙһжҲ–еә”з”Ёдёҙж—¶дҝ®еӨҚзЁӢеәҸгҖӮ

дҪҶжҳҜпјҢжӯӨи„ҡжң¬д»…и§ЈеҶі XenIface й©ұеҠЁзЁӢеәҸжјҸжҙһпјҢеә”иў«и§Ҷдёәдёҙж—¶жҺӘж–ҪгҖӮ

е…ій”®еҹәзЎҖи®ҫж–ҪиҝҗиҗҘе•Ҷеә”дјҳе…ҲиҖғиҷ‘иҝҷдәӣжӣҙж–°пјҢеӣ дёәиҷҡжӢҹеҢ–зҺҜеўғйҖҡеёёжүҳз®Ўд»»еҠЎе…ій”®еһӢеә”з”ЁзЁӢеәҸе’Ңж•Ҹж„ҹж•°жҚ®зі»з»ҹгҖӮ
