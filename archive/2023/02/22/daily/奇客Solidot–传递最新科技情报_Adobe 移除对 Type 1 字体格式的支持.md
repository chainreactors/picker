## Adobe 移除对 Type 1 字体格式的支持

[![IT](https://icon.solidot.org/images/topics/topicit.png?123)](/search?tid=9 "IT")

[Wilson](/~Wilson) (42865)发表于 2023年02月21日 22时52分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74195&appkey=1370085986&title=Adobe%20%E7%A7%BB%E9%99%A4%E5%AF%B9%20Type%201%20%E5%AD%97%E4%BD%93%E6%A0%BC%E5%BC%8F%E7%9A%84%E6%94%AF%E6%8C%81 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自树上银花**

Adobe 本月将释出一系列更新，从其应用中移除对 PostScript 字体格式 Type 1 的支持。PostScript 是 Adobe 为专业数字排版开发的轮廓字体规范编码的字体文件，它在 1984 年创造了 Type 1 和 Type 3 字体格式，而 Type 1 提供了当时最佳的质量和性能，在多年里 Type 1 是数字排版的黄金标准。但进入 1990 年代中期后，Type 1 遇到了各种挑战：它不支持大的字符集，Unicode 字符编码标准出现了，跨平台兼容性，等等。
OpenType 字体逐渐取代了 Type 1，而 Adobe 也最终淘汰了它。
https://typenetwork.com/articles/the-end-of-type-1
