# 微软11月安全更新多个产品高危漏洞通告

## ****一、漏洞概述****

11月13日，绿盟科技CERT监测到微软发布11月安全更新补丁，修复了89安全问题，涉及Windows、Microsoft SQL Server、Microsoft Office、Azure、Open Source Software、Microsoft Visual Studio、System Center等广泛使用的产品，其中包括权限提升漏洞、远程代码执行漏洞等高危漏洞类型。

本月微软月度更新修复的漏洞中，严重程度为关键（Critical）的漏洞有4个，重要（Important）漏洞有85个。其中包括2个存在在野利用的漏洞：

Windows NTLM 哈希泄露欺骗漏洞(CVE-2024-43451)

Windows 任务计划程序权限提升漏洞(CVE-2024-49039)

请相关用户尽快更新补丁进行防护，完整漏洞列表请参考附录。

参考链接：

https://msrc.microsoft.com/update-guide/releaseNote/2024-Nov

## ****二、重点漏洞简述****

根据产品流行度和漏洞重要性筛选出此次更新中包含影响较大的漏洞，请相关用户重点进行关注：

**Windows NTLM 哈希泄露欺骗漏洞****（****CVE-2024-43451****）****：**

Windows NTLM存在哈希泄露欺骗漏洞，攻击者通过滥用MSHTML，诱导用户与恶意文件进行交互：如单击或右键查看，从而泄露用户的NTLMv2哈希。攻击者可以利用这些哈希进行身份验证。该漏洞已公开披露，且存在在野利用，CVSS评分6.5。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-43451

**Windows 任务计划程序权限提升漏洞****（****CVE-2024-49039****）：**

Windows任务计划程序中存在权限提升漏洞，经过身份验证的攻击者可以运行特制的应用程序以利用此漏洞，能够让攻击者从低权限环境中实现容器逃逸并获得较高权限访问，从而可以执行通常仅限于特权账户的RPC功能。该漏洞已存在在野利用，CVSS评分8.8。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-49039

**Windows Kerberos 远程代码执行漏洞****（****CVE-2024-43639****）：**

Windows Kerberos中存在远程代码执行漏洞，由于Windows Kerberos中存在数字截断错误，未经身份验证的攻击者可以通过发送HTTP请求使Kerberos代理访问指定的恶意服务器。使Kerberos代理在解析返回的特制数据包时解析不当，从而造成远程代码执行。此漏洞影响所有受支持版本的Windows Server，可能导致系统间的蠕虫式传播。CVSS评分9.8。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-43639

**.NET 和 Visual Studio 远程代码执行漏洞****（****CVE-2024-43498****）：**

.NET 和 Visual Studio存在远程代码执行漏洞，由于.NET和Visual Studio存在类型混淆错误，未经身份验证的攻击者可以向.NET应用程序发送特制的请求或将特制的文件加载到桌面应用程序中，从而实现在目标系统上执行任意代码。CVSS评分9.8。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-43498

**Microsoft Windows VMSwitch 权限提升漏洞****（****CVE-2024-43625****）：**

Microsoft Windows VMSwitch存在权限提升漏洞，由于 Microsoft Hyper-V中的VmSwitch组件存在释放后重使用错误（Use-After-Free），攻击者可通过向VMswitch驱动程序发送一系列特定的网络请求，从而获得系统的SYSTEM权限。CVSS评分8.1。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-43625

**Active Directory 证书服务权限提升漏洞****（****CVE-2024-49019****）：**

Active Directory 证书服务存在权限提升漏洞，由于在Active Directory证书服务中处理身份验证请求时存在错误，本地用户可以通过滥用内置默认版本证书模板绕过身份验证从而获取域管理员权限。该漏洞已公开披露，CVSS评分8.8。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-49019

**Microsoft Exchange Server 欺骗漏洞****（****CVE-2024-49040****）：**

Microsoft Exchange Server 存在欺骗漏洞，由于对用户提供的数据进行了不正确的处理，攻击者在发送给本地收件人的电子邮件中可以伪造发件人的电子邮件地址，进行欺骗攻击。该漏洞已公开披露，CVSS评分7.5。

官方通告链接：

https://msrc.microsoft.com/update-guide/en-US/vulnerability/CVE-2024-49040
