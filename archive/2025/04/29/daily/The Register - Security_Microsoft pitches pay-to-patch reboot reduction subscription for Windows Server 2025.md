## Redmond reckons $1.50/core/month hotpatch service is worth it to avoid eight Patch Tuesday scrambles each year

![icon](/design_picker/d518b499f8a6e2c65d4d8c49aca8299d54b03012/graphics/icon/vulture_red.svg)
[Simon Sharwood](/Author/Simon-Sharwood "Read more by this author")

Mon 28 Apr 2025  //
06:37 UTC

![](/design_picker/d2e337b97204af4aa34dda04c4e5d56d954b216f/graphics/icons/social_share_icon.svg)

Microsoft has announced its preview of hotpatching for on-prem Windows Server 2025 will become a paid subscription service in July.

Hotpatching allows software updates to be applied without requiring a reboot. It’s a common approach that’s available for the Linux kernel, VMware products, and the Xen hypervisor, among others. Admins like it because it allows security updates to be implemented without requiring reboots – and therefore without having to find a change window or convenient moment for an outage.

> This feature will be a game changer; you may finally get to see your family on the weekends

Microsoft already offers hotpatching for Windows Server: Azure Edition and version 2022 running in its Azure cloud. The MS-DOS titan says its Xbox team is a big user.

In August 2024, the Windows behemoth [announced](https://techcommunity.microsoft.com/blog/windowsservernewsandbestpractices/hotpatching-is-now-available-in-preview-on-windows-server-2025-evaluation-vms-in/4203451) a preview of the same functionality for Windows Server 2025 running in Azure.
