**摘要：**互联网的诞生给人类带来了翻天覆地的变化，深刻地改变着人类的生产方式和生活方式。互联网治理不同于传统的规制、管制。互联网治理主体强调多方共治，治理手段注重多管齐下，治理机制强调平衡协调。互联网治理领域随着现实世界的数据化、网络化、智能化的深化而不断发展，治理的主体和客体也不断扩展。中国互联网治理实践可以分为 3个阶段：自由发展阶段、探索阶段和综合治理阶段。互联网治理从被动管理到主动治理，继而实行综合治理。互联网治理的核心问题是保障互联网技术造福于人，各国唯有秉持携手构建网络空间命运共同体的理念，共建共治共享网络空间，才能确保新兴的互联网科技向上向善。

2024 年是互联网诞生 55 周年，也是中国全功能接入世界互联网 30 周年。三五十年的时间相对于人类文明而言虽然短暂，但互联网却给人类带来了翻天覆地的变化，深刻地改变着人类的生产方式和生活方式。互联网由最初的一项技术，快速衍生出一个产业，进而推动人类社会进入信息网络时代。随着互联网的技术进步和普及应用，互联网治理也不断加强和深入，为互联网的健康发展保驾护航。时至今日，网络空间与现实世界深度泛在融合，互联网治理的范围和手段也不断拓展。本文拟从法律的视角剖析互联网治理的理念和实践，以期更好地理解互联网治理的内涵和发展。

# **1、互联网治理理念的演进**

“治理（Governance）”一词常用于社会管理领域。20 世纪 80 年代，中国针对严峻的社会治安问题提出了社会治安综合治理理念，强调运用政治的、经济的、行政的、法律的、文化的、教育的等多种手段，整治社会治安，保障社会稳定 。20 世纪 90 年代，经济学领域引入了“公司治理”的概念，以协调公司与所有利益相关者之间的利益关系，也体现了综合治理理念 。

在 21 世纪初，治理理念被引入到网络空间，提出了互联网治理（Internet Governance）的概念。联合国互联网治理工作组对“互联网治理”的定义为：互联网治理是政府、私营部门和社会公众根据各自的作用制定和实施的，旨在规范互联网发展和使用的共同原则、准则、规则、决策程序和方案 。美国学者劳拉·德拉迪斯认为，互联网治理的主要内容包括维持互联网正常运转所必需的技术支持和制度安排，以及围绕此类制度涉及和实践的互联网政策。互联网治理本质上是公共利益决策的一部分，互联网治理的架构实际上也是一种权力分配。可见，互联网治理的内容主要聚焦于技术及与技术应用相关的政策制定和实施，它是网络空间治理的基础层级。

从上述社会治理、公司治理和互联网治理的内涵看，治理不同于传统的规制、管制。就互联网治理而言，治理主体强调多方共治，除了政府，企业、社会组织和社会公众等在内的各利益相关方也应共同参与，并根据各自角色承担相应治理责任；治理手段注重多管齐下，主张利用经济、法律、行政、技术、自律等多种手段开展有效治理；治理机制强调平衡协调，注重各利益相关主体之间的权力制衡和相互依存 。习近平总书记指出：“随着互联网特别是移动互联网发展，社会治理模式正在从单向管理转向双向互动，从线下转向线上线下融合，从单纯的政府监管向更加注重社会协同治理转变。” 从国内角度来看，互联网治理属于国家治理体系的重要组成部分，是国家和社会管理理念的重大革新。从国际角度来看，网络治理这种公共利益管理方式的转变，体现为从国家自上而下的等级官僚机制转向多方利益主体平行参与的国际组织全球协调机制 。

**1.2　中国立法中的互联网治理概念辨析**

中国现行立法对互联网治理相关概念的使用还没有完全统一，人们常常将“互联网治理”与“网络治理”等概念混同使用。本文对中国与互联网有关的法律、行政法规、部门规章、司法解释等法律文件中关于“互联网治理”概念的使用情况进行粗略统计（如表 1 所示），发现中国立法对“互联网”和“网络”两个概念并没有做严格区分。但从“互联网服务提供者”和“网络服务提供者”这两个概念的使用情况来看，近些年立法者越来越倾向于使用“网络”这一更为宽泛的概念。

互联网治理是一个动态的概念，它的内涵和外延随着互联网的发展而不断变化。就目前而言，互联网治理的核心内容仍是基于互联网而引发的，且与互联网强相关。例如，人工智能技术正带来深刻的科技革命和产业革命，但人工智能带来的治理问题也多是在互联网的场景下出现的，如生成式人工智能服务的治理问题。因此，“互联网治理”这一概念并没有完全过时，其仍然具有很强的包容性。

当然，由于数据化、网络化、智能化的发展，网络的范围已经超出了互联网本身，网络信息基础设施还包括各种电信网与通信系统、广电网与各种传播系统、工控网等信息通信技术基础设施。因此，“网络治理”或“网络空间治理”具有更广泛的内容，是包括网络空间基础设施、标准、法律、社会文化、经济、发展等多方面内容的一个范畴 。特别是“网络空间”这一概念与现实世界相对，在两者越来越深度融合的背景下，“网络空间治理”显然具有更为宽泛的包容性和适应性，建议后续立法应考虑互联网治理法律概念的统一。

中国立法对网络治理的内涵和范围的扩张做了全面的回应，这主要体现在网络治理的主体和客体的范围不断扩张方面。

# **2、互联网治理领域的扩张**

**2.1　互联网治理领域**

随着互联网的快速发展，网络空间与现实世界不断融合、相互渗透，互联网治理涉及的领域如同现实世界一样复杂多样，治理的需求不断扩大，主要包括：

一是与基础设施和互联网重要资源管理有关的问题。主要包括域名系统和互联网协议（Internet Protocol，IP）地址管理、根服务器系统管理、技术标准研制、互传和互联、网络基础设施建设及语文多样性等问题 。

二是与互联网使用有关的问题。主要包括垃圾邮件、电子商务、网络安全、数据安全、虚假信息和网络犯罪等问题 。

三是与互联网有关、但影响范围远超互联网的问题。主要包括知识产权、人权、人工智能、国际贸易、环境保护、发展中国家能力建设等问题。

**2.2　互联网治理主体的扩张**

与社会治安综合治理一样，互联网治理也是一项系统工程。实现互联网治理目标需要各利益相关方共同参与，因而互联网治理的主体具有显著的多样性。从国内立法而言，互联网治理主体主要可以分为 4 类，即政府、互联网企业、社会组织和社会公众。

**2.2.1　政府**

在互联网发展初期，技术社群对互联网的发展贡献甚巨，IP 地址、域名等互联网基础资源主要由技术社群的精英们在管理，他们对政府介入互联网存在本能的反感和抵触。最具代表性的事件是美国电子前沿基金会创始人约翰·佩里·巴洛（John Perry Barlow）于 1996 年 2 月 8 日发表了《网络空间独立宣言》，宣称网络空间自然而然独立于国家主权，政府没有道德上的权力来统治网络空间 。

随着互联网的普及和深入发展，以及网络安全问题的日益突出，政府在互联网治理中的作用日益重要，并逐渐占据主导地位。政府负责网络公共政策的制定、执行和监督，为互联网的发展创造有利环境，包括制定法律和标准，确立最佳实践；推动技术和标准的研发和应用，促进网络能力建设；推动信息基础设施建设，普及网络服务；打击网络犯罪；解决网络纠纷；促进国际和区域合作等。《中华人民共和国网络安全法》《中华人民共和国数据安全法》《中华人民共和国个人信息保护法》均强调政府在互联网治理中的作用。例如，《中华人民共和国网络安全法》第 16 条明确规定，国务院和省、自治区、直辖市人民政府应当统筹规划，加大投入，扶持重点网络安全技术产业和项目，支持网络安全技术的研究开发和应用，推广安全可信的网络产品和服务，保护网络技术知识产权，支持企业、研究机构和高等学校等参与国家网络安全技术创新项目。第 8 条明确规定，国家网信部门负责统筹协调网络安全工作和相关监督管理工作；国务院电信主管部门、公安部门和其他有关机关依法在各自职责范围内负责网络安全保护和监督管理工作；县级以上地方人民政府有关部门的网络安全保护和监督管理职责，按照国家有关规定确定。

**2.2.2　互联网企业**

当今的互联网主要是由包括互联网企业在内的众多市场主体共同发展起来的。互联网基础设施大多由大型互联网企业运营，与互联网治理有关的海量信息和数据也大多掌握在大型互联网企业手中。没有互联网企业的合作，互联网治理不可能取得成效。

互联网企业应依法承担网络治理的主体责任，在互联网治理中的作用主要体现在：参与制定行业自律规范；确立最佳实践；研究和发展技术、标准，促进创新；协助政府起草法律，参加国家和国际政策的制定；促进网络能力建设；参与网络纠纷的解决等。例如，《中华人民共和国网络安全法》明确规定，网络运营者应当遵守法律、行政法规，尊重社会公德，遵守商业道德，诚实信用，履行网络安全保护义务，接受政府和社会的监督，承担社会责任；履行安全保护义务；制定网络安全事件应急预案；为公安机关、国家安全机关依法维护国家安全和侦查犯罪的活动提供技术支持和协助；配合监督检查等。

**2.2.3　社会组织**

在互联网治理领域，行业协会、科研机构等社会组织也在发挥越来越大的作用，例如，《中华人民共和国网络安全法》涉及的社会组织就包括网络相关行业组织、检测认证机构、高等学校、教育培训机构、大众传播媒介等。

由于互联网具有很强的技术性，互联网应用更新迭代快，政府的治理相对滞后，这就给行业组织提供了更多的发挥空间，其在互联网治理中的作用越来越突出。《中华人民共和国网络安全法》明确规定，网络相关行业组织的职责包括：加强行业自律，制定网络安全行为规范，指导会员加强网络安全保护，提高网络安全保护水平，促进行业健康发展；参与网络安全国家标准、行业标准的制定；建立健全本行业的网络安全保护规范和协作机制，加强对网络安全风险的分析评估，定期向会员进行风险警示，支持、协助会员应对网络安全风险等。又如，高等学校可以参与网络安全国家标准、行业标准的制定，参与网络安全相关的教育与培训，可以采取多种方式培养网络安全人才，促进网络安全人才交流。

**2.2.4　社会公众**

正因为有众多网民的参与，网络才变得更加重要，网络社会才得以形成，并成为现实社会的一部分。只有依靠网民的共同参与，群策群力，才能实现互联网治理目标。构建和谐的网络社会需要提高社会公众的网络能力和网络素养，培养公民网络安全意识，做到文明上网，理性对待网络舆论。社会公众还可以通过网络参与政治、经济、社会、文化活动，促进实现各种公益目标，发展和推广最佳实践，促进企业承担社会责任等。我国每年开展的“网络安全宣传周”活动，以“网络安全为人民，网络安全靠人民”为主题，目的在于提高社会公众网络安全素养，维护一个安全可信的网络环境。

互 联 网 是 全 球 性 的 信 息 基 础 设 施， 除 了国内主体，互联网治理还离不开政府间和非政府间的国际合作，离不开国际组织的参与。联合国、经济合作与发展组织、亚太经合组织、世 界 贸 易 组 织、 世 界 知 识 产 权 组 织、 欧 盟、非洲联盟等国际组织都在积极参与国际互联网治理。

**2.3　互联网治理客体的扩张**

互联网治理客体的范围也呈现明显的扩张特点，由最初的 IP 地址、域名等互联网基础资源，进一步发展到网络行为、数据、信息等。

**2.3.1　IP 地址、域名等互联网基础资源**

IP 地址、域名是互联网基础资源。其中，IP 地址是 IP 协议提供的一种统一的地址格式，它为互联网上的每一个网络和每一台主机分配了一个逻辑地址。有了 IP 地址，我们在上网时就能够高效且方便地从千千万万台计算机和网页中选出自己所需的对象来。由于 IP 地址由数字串组成，难以记忆，故而设计出域名。域名是互联网上识别和定位计算机的层次结构式的字符标识，与该计算机的 IP 地址相对应，极大地方便了用户上网。

对 IP 地址、域名的治理涉及域名系统和 IP地址管理、根服务器系统管理、技术标准研制、互联互通、语文多样性等方面，我国专门出台了《互联网 IP 地址备案管理办法》《互联网域名管理办法》等规定。由于中国是互联网的接入国，因此没有经历互联网早期以互联网基础资源为主要对象的治理阶段，而是直接面临因互联网普及而带来的网络信息和网络行为的治理问题。

**2.3.2　网络行为**

网络行为是网络治理的重要客体。网络行为主要包括：一是网络信息处理行为，包括信息生产、发布、上传、下载、收集、使用、转让、删除等；二是网络交易行为，包括电子合同、数据交易等；三是网络服务消费行为，包括网络新闻、网络娱乐、网络教育、网络学习等；四是网络技术行为，包括网络开发、维护、升级、运行、管理、保障等；五是网络监管行为，包括网络监测、网络审查、违法违规网络行为的处置等。网络行为法律规范主要涉及网络行为的成立、效力、法律后果等。我国立法对网络合法行为和非法行为划定了基本边界。

**2.3.3　数据**

数据是指任何以电子或者其他方式对信息的记录。在计算机科学中，数据是指所有能输入到计算机并被计算机程序处理的符号的介质的总称，是用于输入电子计算机进行处理，具有一定意义的数字、字母、符号和模拟量等的通称 。在计算机系统中，数据以二进制信息单元 0、1 的形式表示。在网络上，网络行为和网络信息都会被记录为网络数据。

如今，数据已经成为数字经济的重要生产要素，数据资源成为关涉国家安全的重要资源。数据治理是互联网治理的重要内容，主要包括数据的开发利用和数据安全管理两个方面。我国出台了《中华人民共和国数据安全法》《中华人民共和国个人信息保护法》等，对数据和个人信息安全治理做了全面系统的规定。

**2.3.4　信息**

信息与数据既有联系又有区别，二者是不可分离的。信息是数据所承载的含义；数据是信息的表现形式和载体，是信息的外壳，表现为符号、文字、数字、语音、图像、视频等形式。

互联网为信息的制造和传播提供了便利，网络自媒体的形式和内容层出不穷，人工智能、深度伪造等技术让虚假信息更容易被制造和传播，这为网络信息内容的治理带来了极大的挑战。网络信息的传播常常涉及社会稳定和国家安全，我国尤为重视。《中华人民共和国网络安全法》对此做了专章规定，中央网信办还出台了《网络信息内容生态治理规定》，以维护网络信息内容安全，建设良好的网络生态。

# **3、中国互联网治理实践的演进**

自 1994 年中国全功能接入国际互联网以来，中国的互联网治理随着互联网技术的发展和应用的普及而不断演进。中国对于网络运行安全和数据安全的治理，紧跟形势发展，与世界互联网发达国家的治理方式和手段大体类似；而对于互联网信息内容的治理，则具有显著的中国特色，下面着重就互联网信息内容的治理演进展开进一步分析。

总体而言，中国互联网治理实践的发展大体可以分为 3 个阶段：自由发展阶段、探索阶段和综合治理阶段。互联网治理理念从初期的自由发展到逐渐加强治理，从被动管理到主动治理，继而发展到综合治理。

**3.1　自由发展阶段**

2000 年之前，中国互联网大体处于自由发展阶段。这一时期中国大力推进信息化和互联网基础设施建设，积极推广和普及个人电脑和互联网。人们刚开始接触互联网这一新技术，对互联网的了解非常有限，截至 1999 年底，中国网民数量仅为 890 万人。该阶段的互联网应用非常有限，主要是网易、搜狐、新浪等门户网站，网络信息传播类似于传统媒体，主要采取单向输出，仅供网民浏览，人们之间的网络交流主要通过电子邮件和 BBS 论坛进行。

在立法方面，国家仅出台了《中华人民共和国计算机信息系统安全保护条例》《中华人民共和国计算机信息网络国际联网管理暂行规定》等少量有关信息化和计算机信息安全的行政法规和规章，内容主要是关于计算机信息系统安全保护及国际联网、域名注册方面的管理规定，条文数量少，内容简单。

这一阶段，互联网治理问题尚不突出，治理的重点是网络安全、国际联网安全等网络运行安全问题，对于互联网信息内容的治理尚未引起太多关注。

**3.2　探索阶段**

2000—2015 年，中国互联网治理大体属于探索阶段。这一时期 Web 2.0、云计算兴起，移动互联网全面发展，3G/4G 技术快速迭代，智能手机迅速普及，截至 2015 年底，中国网民规模达 6.88 亿人，互联网普及率为 50.3%，超过全国人口半数。QQ、微信等即时通信软件极大地便利了人们的交流，博客、微博等自媒体社交平台快速发展，极大便利了信息的生成和传播，由此也带来了网络治理的巨大压力。

各相关主管部门为了应对网络治理新问题，开始推出网络治理专门立法，主要有全国人民代表大会常务委员会制定的《关于维护互联网安全的决定》《关于加强网络信息保护的决定》《中华人民共和国电子签名法》《刑法修正案（七）》等法律，国务院制定的《互联网信息服务管理办法》《中华人民共和国电信条例》《计算机软件保护条例》《互联网上网服务营业场所管理条例》《信息网络传播权保护条例》等行政法规，此外就是大量的部门规章。总体来看，这个时期的网络立法层次低且分散，欠缺上位法和体系化设计；立法较为滞后，不能适应网络社会快速发展的需要；立法重管理而轻治理，重义务而轻权利。

这一阶段，互联网内容治理问题非常突出，各主管部门在自己的职责范围内纷纷出台规章或规范性文件，作为治理依据。对于网络平台信息的治理，主要采取“通知－删除”规则和“避风港”规则，互联网服务提供者如发现或被告知存在违法信息，则有义务采取必要措施予以处置；如果不知道或未发现存在违法信息，则可以免责。
