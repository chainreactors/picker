# Oracle发布2024年7月的安全公告

原创

CNVD

CNVD漏洞平台

安全公告编号:CNTA-2024-0010

2024年7月16日，Oracle发布了2024年7月份的安全更新，修复了其多款产品存在的386个安全漏洞。受影响的产品包括：Oracle Database Server数据库（8个）、Oracle Application Express（1个）、Oracle Essbase（2个）、Oracle GoldenGate（1个）、Oracle NoSQL Database（1个）、Oracle REST Data Services（1个）、Oracle TimesTen In-Memory Database（1个）、Oracle
Commerce（7个）、Oracle Communications Applications（20个）、Oracle
Communications（95个）、Oracle Construction and Engineering（4个）、电子商务套装软件Oracle
E-Business Suite（10个）、Oracle Enterprise Manager（5个）、Oracle Financial Services Applications（60个）、中间件产品Fusion
Middleware（41个）、Oracle Analytics（17个）、Oracle HealthCare Applications（5个）、Oracle
Hyperion（3个）、Oracle Insurance Applications（10个）、Oracle
Java SE（7个）、Oracle JD Edwards（8个）、Oracle MySQL数据库（37个）、Oracle PeopleSoft（11个）、Oracle Retail Applications（5）、Oracle Siebel CRM（12个）、Oracle Supply Chain（7个）、Oracle Systems（2个）、Oracle Utilities Applications（2个）和Oracle
Virtualization（3个）。

本次安全更新涉及的漏洞中，共包括200个高危漏洞，324个可被远程利用漏洞。CNVD提醒广大Oracle用户，请及时下载补丁更新，避免引发漏洞相关的安全事件。
