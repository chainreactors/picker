据某些学术研究人员发布的消息，推特目前已经撤销此前免费提供的专用 API 接口，该接口每天会随机性的提供 10% 的推文给学术研究人员分析错误信息和仇恨言论等。

此前推特已经调整 API 政策，将面向普通开发者的免费 API 权限撤销，然后大幅度提高 API 价格，如果不付费那就无法继续使用推特 API 和获得数据。

[![推特撤销学术研究的API权限 要么付费4.2万美元/月要么删除所有数据](https://img.lancdn.com/landian/2023/04/98285.png)](https://img.lancdn.com/landian/2023/04/98285.png)

现在这种情况已经到了学术研究领域，推特给出两种选择：要么研究人员支付 42000 美元 / 月的 API 使用费，这样可以继续获得 0.3% 推文的访问权限；要么停止使用 API 并删除已经获得的数据。

如果研究人员选择后者，则需要向推特提供删除数据的屏幕截图😯😯😯理论上说，研究人员要是把数据复制一份到另一个文件夹，再把源文件夹删除并截图发给推特，也是可行的吧？推特这是啥操作。

目前推特并未就此事发布回应，但学术研究专用的 Decahose API 已经无法使用，对于大多数学术研究人员来说，支付每月高达 4.2 万美元的月费是不太现实的，除非是规模非常大的、预算足够的学术研究机构估计才能花得起这个钱。

当然还有一种可能是埃隆马斯克压根没想着学术研究人员能够付钱，关闭这个 API 后对研究推特平台上的虚假信息、虚假账号、平台透明度有重大影响，**既然都不能研究了那自然无法再提供透明度或虚假账号的数据，那后续推特数据只有马斯克自己说了算，外界是无法再进行预估的。**
