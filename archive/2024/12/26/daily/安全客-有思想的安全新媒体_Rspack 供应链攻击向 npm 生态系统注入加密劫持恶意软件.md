##### иҜ‘ж–ҮеЈ°жҳҺ

жң¬ж–ҮжҳҜзҝ»иҜ‘ж–Үз« пјҢж–Үз« еҺҹдҪңиҖ… do sonпјҢж–Үз« жқҘжәҗпјҡsecurityonline

еҺҹж–Үең°еқҖпјҡ<https://securityonline.info/rspack-supply-chain-attack-injects-cryptojacking-malware-into-npm-ecosystem/>

иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

д»ҘдҪҝз”Ё Rust зј–еҶҷзҡ„й«ҳжҖ§иғҪ JavaScript жҚҶз»‘зЁӢеәҸиҖҢй—»еҗҚзҡ„ Rspack з”ҹжҖҒзі»з»ҹе·ІжҲҗдёәдҫӣеә”й“ҫж”»еҮ»зҡ„жңҖж–°еҸ—е®іиҖ…гҖӮиҝҷж¬ЎжјҸжҙһеҪұе“ҚдәҶдёӨдёӘе№ҝжіӣдҪҝз”Ёзҡ„ npm иҪҜд»¶еҢ… @rspack/core е’Ң @rspack/cliпјҢе®ғд»¬иў«зҜЎж”№дёәеҢ…еҗ«еҠ еҜҶеҠ«жҢҒжҒ¶ж„ҸиҪҜд»¶гҖӮ

еҪ“ж”»еҮ»иҖ…иҺ·еҫ—жңӘз»ҸжҺҲжқғзҡ„ npm еҸ‘еёғжқғйҷҗж—¶пјҢ@rspack/core е’Ң @rspack/cli зҡ„ 1.1.7 зүҲжң¬йҒӯеҲ°дәҶз ҙеқҸгҖӮвҖңиҝҷдәӣжҒ¶ж„ҸзүҲжң¬еҢ…еҗ«ж—ЁеңЁжү§иЎҢжңӘз»ҸжҺҲжқғзҡ„еҠ еҜҶиҙ§еёҒжҢ–жҺҳзҡ„ж··ж·Ҷи„ҡжң¬гҖӮвҖқж №жҚ® Socket з ”з©¶е°Ҹз»„зҡ„еҲҶжһҗпјҢжҒ¶ж„ҸиҪҜд»¶жҳҜйҖҡиҝҮе®үиЈ…еҗҺи„ҡжң¬жү§иЎҢзҡ„пјҢиҜҘи„ҡжң¬дјҡеңЁе®үиЈ…иҪҜд»¶еҢ…ж—¶иҮӘеҠЁиҝҗиЎҢгҖӮ

Rspack е·ІеңЁеҫ®иҪҜгҖҒдәҡй©¬йҖҠгҖҒDiscord е’ҢйҳҝйҮҢе·ҙе·ҙзӯүйЎ¶зә§е…¬еҸёдёӯиҝ…йҖҹжөҒиЎҢиө·жқҘпјҢд»… @rspack/core е°ұжӢҘжңүжҜҸе‘Ё 37 дёҮж¬Ўзҡ„дёӢиҪҪйҮҸгҖӮиҝҷж¬ЎжјҸжҙһжңүеҸҜиғҪеҪұе“ҚеҲ°жҲҗеҚғдёҠдёҮдҪҝз”ЁеҸ—жҚҹиҪҜд»¶еҢ…зҡ„ејҖеҸ‘дәәе‘ҳе’ҢдјҒдёҡгҖӮ

ж”»еҮ»иҖ…еңЁ @rspack/core зҡ„ support.js ж–Үд»¶е’Ң @rspack/cli зҡ„ config.js ж–Үд»¶дёӯжіЁе…ҘдәҶеҠ еҜҶеҠ«жҢҒи„ҡжң¬гҖӮдё»иҰҒжҒ¶ж„ҸеҠҹиғҪеҢ…жӢ¬пјҡ

* **еҠ еҜҶеҠ«жҢҒ**пјҡ и„ҡжң¬дёӢиҪҪе№¶жү§иЎҢ XMRig Cryptomining зЁӢеәҸпјҢеҲ©з”Ёи®Ўз®—иғҪеҠӣжҢ–жҺҳ Monero еҠ еҜҶиҙ§еёҒгҖӮдҪҝз”Ё-cpu-max-threads-hint=75зӯүеҸӮж•°жқҘйҷҗеҲ¶CPUдҪҝз”ЁйҮҸпјҢйҒҝе…Қиў«еҸ‘зҺ°гҖӮ
* **ж•°жҚ®еӨ–жё—**пјҡ жҒ¶ж„Ҹд»Јз Ғд»Һ http://80.78.28.72/tokens е’Ң http://ipinfo.io/json зӯүеӨ–йғЁжңҚеҠЎеҷЁиҺ·еҸ–дҝЎжҒҜпјҢж”¶йӣҶ IP ең°еқҖе’Ңең°зҗҶдҪҚзҪ®ж•°жҚ®зӯүз”ЁжҲ·зҺҜеўғиҜҰз»ҶдҝЎжҒҜгҖӮиҝҷз§ҚдҫҰжҹҘеҸҜдёәе®ҡеҲ¶ж”»еҮ»жҸҗдҫӣдҫҝеҲ©гҖӮ
* **йҡҗи”ҪжҲҳжңҜ**пјҡ йҮҮз”Ёж··ж·ҶжҠҖжңҜжқҘйҖғйҒҝжЈҖжөӢе’ҢеҲҶжһҗгҖӮжҒ¶ж„ҸиҪҜд»¶йҖҡиҝҮиҪҜд»¶еҢ…зҡ„е®үиЈ…еҗҺи„ҡжң¬е°ҶиҮӘиә«еөҢе…ҘеҲ°ејҖеҸ‘зҺҜеўғдёӯпјҢдҪҝејҖеҸ‘дәәе‘ҳж— жі•еҜҹи§үе…¶еӯҳеңЁгҖӮ

Rspack еӣўйҳҹйҖҡиҝҮд»ҘдёӢжҺӘж–Ҫиҝ…йҖҹзј“и§ЈдәҶиҝҷдёҖй—®йўҳпјҡ

* еәҹејғ 1.1.7 зүҲжң¬пјҢе°ҶжңҖж–°зҡ„ dist-tag йҮҚе®ҡеҗ‘еҲ° 1.1.6 зүҲжң¬гҖӮ
* дҪҝжүҖжңү npm е’Ң GitHub ж Үи®°еӨұж•ҲпјҢд»ҘйҳІжӯўжӣҙеӨҡжңӘз»ҸжҺҲжқғзҡ„и®ҝй—®гҖӮ
* еҸ‘еёғ 1.1.8 зүҲпјҢеҲ йҷӨжҒ¶ж„Ҹд»Јз Ғе№¶жҒўеӨҚеҜ№еҸ—еҪұе“ҚиҪҜд»¶еҢ…зҡ„дҝЎд»»
