# 微软在Edge浏览器测试加密货币钱包

导语：​微软在Edge浏览器中测试内置加密货币钱包。

微软在Edge浏览器中测试内置加密货币钱包。

日前，微软开始在Edge浏览器中测试内置的以太坊加密货币钱包，允许用户发送和接收加密货币和NFT（非同质化代币、数字藏品）。用户可以共享公钥给其他用户以接收加密货币，私钥用来授权加密货币的交易支付。

Albacore最先在推特上分享了Edge浏览器加密货币钱包的截图，并指出这可能是微软Edge浏览器的新特征。目前，该功能仅限于Edge浏览器开发者用户，处于有限测试阶段。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230318/1679112155553356.png "1679112155553356.png")

微软称，这是一个非监管的钱包，也就是说用户可以完全控制其加密货币。微软并不会访问用户的密码和恢复密钥。该钱包内嵌在Edge浏览器中，用户使用过程中无需安装扩展因此使用过程中非常容易。

测试中，测试用户可以使用自己的资金。如果有资金损失，微软并不承担任何责任。目前这仍然是一个保密项目，没有更多公开的项目信息。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230318/1679112156121209.png "1679111895198720.png")

图 微软 Edge加密货币钱包

BleepingComputer 测试了这一新功能，在测试过程中，用户被要求生成一个口令来确保其加密货币资产的安全，并被要求设置一个12个词的恢复密钥以在用户忘记密码时恢复其加密货币钱包账户。然后，钱包会生成一个以太坊地址允许用户通过以太坊网络接收资金。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230318/1679112163130989.png "1679111928126464.png")

图 设置Edge加密货币钱包恢复密钥

与其他加密货币钱包类似，微软的Edge加密货币钱包支付多个以太坊账户，用户可以在多个账户之间切换。测试人员还可以在Edge加密货币钱包中设置默认的加密货币钱包或者扩展钱包。

Edge 的加密货币钱包还可以连接到去中心化应用，并提供了最新的加密货币行情和新闻。目前，微软还与Consensys合作提供以太坊与Dai 稳定币、Uniswap、USD Coin和Tether币的兑换。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230318/1679112164116644.png "1679111960112745.png")

图 Edge加密货币钱包支持兑换的加密货币

此外，根据Edge浏览器开发者版本中的字符串可以看出，该钱包未来可能还提供对比特币钱包的支持。

微软发言人称，微软会经常探索新的特征以增强用户体验，但目前无更多关于加密货币钱包的信息。
