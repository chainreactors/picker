# HPE 为小型企业推出一体化安全网关

惠普企业推出了HPE Networking Instant On Secure Gateway,这是一种多功能网络解决方案,旨在支持中小型企业保护其网络免受网络威胁。

HPE Networking Instant On Secure Gateway 旨在为可能缺乏专用安全团队或广泛 IT 资源的中小企业提供全面的企业级保护。该系统将防火墙保护、入侵检测和预防、VPN支持和广域网等功能以可访问和可管理的格式组合在一起,适用于没有专业知识的组织。

HPE Aruba Networking全球中小型企业副总裁兼总经理Amol Mitra表示:“如今的中小企业面临着保护其网络免受复杂网络威胁、IT资源有限和快速发展数字环境的不断压力。

“如果没有专门的安全团队或大量预算,许多中小企业很难有效地保护他们的网络。传统的安全解决方案往往不足,使这些企业变得脆弱。

“HPE Networking Instant On Secure Gateway 解决了这一差距,无需广泛资源即可提供中小企业级保护,使其能够在瞬息万变的数字世界中保持安全。

该设备作为第一道防线,使用智能监控和过滤来确保只允许安全,授权的活动,并主动阻止恶意威胁。通过将基本安全功能集成到一个解决方案中,它旨在为技术容量有限的企业简化和现代化网络安全。

安全网关有两种模式,可满足不同的操作需求。HPE Networking Instant On Secure Gateway SG1004 配备一个 1 个设备、4 端口、有线安全网关,具有基本路由、广域网弹性、VPN、防火墙和入侵检测和防疫系统 (IDS/IPS) 选项。SG2505P 变型提供多齿轮连接、五端口配置、4 类 POE 支持为接入点供电,总功率预算为 64W,并包含 SG1004 中提供的所有功能。

HPE Aruba Networking亚太及日本副总裁Mark Ablett评论了新产品对该地区业务的重要性。“亚太地区和日本的中小企业部门不仅对当地经济至关重要,也是全球发展最快、最具活力的细分市场之一。

HPE Networking Instant On Secure Gateway 为这些企业提供了先进且易于管理的安全解决方案,可保护其网络并简化运营。这一创新是使中小企业能够在日益互联的世界中充满信心地创新和扩张的关键。

新的 Secure Gateway 解决方案为 SMB 客户带来了诸多好处。通过持续实时监控和自动更新IDS/IPS签名,提供增强的保护,旨在防范不断变化的安全威胁。将多个关键安全功能整合到一个平台中,旨在减少对各种工具和供应商的依赖。

该公司强调,该产品可以为企业节省成本,因为统一安全流程会减少硬件支出和对第三方安全供应商的需求。内置高可用性功能有助于最大限度地减少停机时间,自动事件响应功能专为加速威胁解决而设计。自动化策略执行和详细的审计跟踪也通过数据保护和日志自动化支持法规遵从。

对于管理,Secure Gateway 利用基于云的系统,使中小企业能够通过统一和直观的界面监督其整个网络。这种设计消除了对高级IT技能的要求。此外,订阅模式旨在透明,消除隐藏费用,并使小型企业更容易获得企业级安全性。

HPE Networking Instant On Secure Gateway 现已面向全球客户。
