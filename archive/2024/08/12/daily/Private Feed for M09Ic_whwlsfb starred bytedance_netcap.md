# netcap

## 简介

netcap(内部称为xcap)是一个基于bcc，可对含skb作为参数的系统函数，或者基于DPDK的mbuf抓包分析的工具。简单试用如下：

```
netcap skb -f icmp_rcv@1 -e "host 10.227.0.72" -i eth0
```

## 安装依赖

netcap工具依赖安装`bcc`包。
