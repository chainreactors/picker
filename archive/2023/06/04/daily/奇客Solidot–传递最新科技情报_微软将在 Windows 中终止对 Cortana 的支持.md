## 微软将在 Windows 中终止对 Cortana 的支持

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Wilson](/~Wilson) (42865)发表于 2023年06月03日 14时05分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75141&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E5%B0%86%E5%9C%A8%20Windows%20%E4%B8%AD%E7%BB%88%E6%AD%A2%E5%AF%B9%20Cortana%20%E7%9A%84%E6%94%AF%E6%8C%81 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自头号书迷**

微软将在 Windows 中杀死语音助手 Cortana。微软表示，从 2023 年底开始，将不再支持 Windows 中的 Cortana 作为独立应用。用户仍然可以在 Windows 11 中获得 Windows Copilot 等 AI 助手。
Cortana 最早是在 2014 年作为 Windows Phone 8.1 的虚拟助手提供给用户的，以与苹果的 Siri 竞争。 2015 年随 Windows 10 在桌面上推出。类似今天无处不在的 Copilot 助手，微软在各种应用中整合了 Cortana。当 Cortana 无法与亚马逊 Alexa 竞争之后，微软撤退了。Cortana 从 Windows 中剥离成为一个独立应用。现在这个应用也要被杀死了。
https://support.microsoft.com/zh-cn/topic/%E5%9C%A8-windows-%E4%B8%AD%E7%BB%88%E6%AD%A2%E5%AF%B9-cortana-%E7%9A%84%E6%94%AF%E6%8C%81-d025b39f-ee5b-4836-a954-0ab646ee1efa
