## Rust 庆祝十周年，发布 Rust 1.87

[![开发者](https://icon.solidot.org/images/topics/topicdevelopers.png?123)](/search?tid=11 "开发者")

[Wilson](/~Wilson) (42865)发表于 2025年05月16日 17时17分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81310&appkey=1370085986&title=Rust%20%E5%BA%86%E7%A5%9D%E5%8D%81%E5%91%A8%E5%B9%B4%EF%BC%8C%E5%8F%91%E5%B8%83%20Rust%201.87 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自金斯顿城·卷一：巫师之印**

Rust 语言的第一个稳定版本 1.0 于 2015 年 5 月 15 日发布。为庆祝发布十周年，Rust 项目的开发者在荷兰的 Utrecht 举办了一场活动，宣布发布 Rust 1.87。新版本的主要新特性包括：标准库加入匿名管道（Anonymous Pipes），安全架构 intrinsics，通过 asm！内联汇编可跳转到 Rust 代码中的标记块，稳定 API 等等。
blog.rust-lang.org/2025/05/15/Rust-1.87.0/
