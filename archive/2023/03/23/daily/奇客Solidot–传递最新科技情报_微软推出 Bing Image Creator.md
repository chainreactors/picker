## 微软推出 Bing Image Creator

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")
[![人工智能](https://icon.solidot.org/images/topics/topicAI.png?123)](/search?tid=151 "人工智能")

[Wilson](/~Wilson) (42865)发表于 2023年03月22日 11时50分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74457&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E6%8E%A8%E5%87%BA%20Bing%20Image%20Creator%20 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自飞行村**

微软官方博客宣布推出 Bing Image Creator，它的 AI 文本图像生成工具。和 Stable Diffusion 和 MidJourney 类似，用户可以通过输入一段文字描述，让 AI 生成相应的图像。Bing Image Creator 是基于 OpenAI 的 DALL∙E 模型。Bing Image Creator 可以作为一个工具独立使用，也可以通过 Bing chat 使用，它还将支持 Microsoft Edge。
https://www.bing.com/create
https://blogs.microsoft.com/blog/2023/03/21/create-images-with-your-words-bing-image-creator-comes-to-the-new-bing/
