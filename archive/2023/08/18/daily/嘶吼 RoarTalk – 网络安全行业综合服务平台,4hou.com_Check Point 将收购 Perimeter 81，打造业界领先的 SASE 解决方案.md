# Check Point 将收购 Perimeter 81，打造业界领先的 SASE 解决方案

[行业](https://www.4hou.com/category/industry)

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)70885

导语：Check Point 将收购 Perimeter 81，打造业界领先的 SASE 解决方案。

2023 年 8 月，全球领先的网络安全解决方案提供商 Check Point® 软件技术有限公司（纳斯达克股票代码：CHKP）签署了收购安全服务边缘 (SSE) 先驱公司 Perimeter 81 的最终协议。通过此次收购，Check Point 将能够帮助企业更快速地在远程用户、站点、云、数据中心和互联网端获得安全访问能力，同时打造市场上最安全、最快速的 SSE 解决方案。根据协议条款，Check Point 将在无现金、无债务的基础上以约 4.9 亿美元的总价收购 Perimeter 81。

如今，混合办公模式已成为全球企业采用的标准运营模式，数字化转型正不断地将数据中心迁移到云端。这一重大转变需要运用一种先进的网络安全架构，从而无缝适应现代分布式企业的需求。Gartner 预计，到 2026 年，安全访问服务边缘 (SASE) 市场规模将超过 200 亿美元，这凸显了企业采用综合安全架构的必要性。

Perimeter 81 创立于 2018 年，拥有 200 多名员工，为全球超过 3,000 家客户提供服务。该公司在 Forrester Wave 零信任平台厂商评估中被评为领导者，创建了一种兼顾云端和设备保护的创新 SSE 方法。Perimeter 81 提供了一套独特的能力，包括零信任访问以及用户、分支机构和应用之间的全网状连接，支持远程用户随时随地快速、安全地上网，同时解决云隐私侵犯问题。相较于传统解决方案长达数周的部署时间，Perimeter 81 部署简单快速，只需一小时便可完成，不仅易于使用，而且有助于简化管理。

Check Point 计划将 Perimeter 81 的安全能力集成到 Check Point Infinity 架构中，从而在业界领先的威胁防御技术的支持下，为网络、云端和远程用户提供统一的安全解决方案。Perimeter 81 能够有力支持 Check Point 实现交付业界最快速、最安全 SASE 解决方案的愿景。

Check Point 软件技术公司首席执行官 Gil Shwed 表示：“随着混合办公模式的出现和云转型的兴起，对延伸到网络边界之外的安全服务的需求与日俱增。通过利用 Perimeter 81 的能力并将其集成到 Check Point Infinity 平台中，我们将加速推进公司愿景，即通过全面的综合性协同解决方案提供最佳安全防护。”

Perimeter 81 首席执行官 Amit Bareket 表示：“这一战略性举措预示着我们的公司、合作伙伴和客户将迎来激动人心的新时期。Check Point 是网络安全领域的全球领导者，拥有 30 多年的丰富行业经验。通过加入 Check Point，我们将有望交付市场上一流的 SASE 平台。我们的强强联手标志着整个行业朝着全面、可扩展的现代化安全防护迈进了重要一步，用户与市场期待我们共同带来的积极影响。”

Check Point 对 Perimeter 81 的收购取决于惯例成交条件，预计将于 2023 年第三季度完成。

**关于 Check Point 软件技术有限公司**

Check Point 软件技术有限公司 (www.checkpoint.com.cn) 是一家面向全球政府和企业的领先网络安全解决方案提供商。Check Point Infinity 解决方案组合对恶意软件、勒索软件及其他威胁的捕获率处于业界领先水准，可有效保护企业和公共组织免受第五代网络攻击。Infinity 包含四大核心支柱，可跨企业环境提供卓越安全保护和第五代威胁防护：Check Point Harmony（面向远程用户）；Check Point CloudGuard（自动保护云环境）；Check Point Quantum（有效保护网络边界和数据中心）；Check Point Horizon (以预防为中心的统一安全管理和防御平台) — 所有这一切均通过业界最全面、直观的统一安全管理进行控制。Check Point 为十万多家各种规模的企业提供保护。

**关于 Perimeter 81 Ltd.**

Perimeter 81 (www.perimeter81.com) 提供了一个强大易用的融合式网络和安全平台，可帮助办公室或远程用户无缝连接至本地或云端资源。该平台以易于使用的云原生服务形式整套交付，并通过统一控制台进行管理。借助该平台，企业无需任何硬件，即可在几分钟内通过全球骨干专网构建安全的企业网络。Perimeter 81 的解决方案现已被全球超过 3,000 家客户和合作伙伴（包括托管服务提供商和渠道经销商）所采用。该公司目前拥有 200 多名员工。
