自战争爆发后诸如甲骨文、戴尔、IBM 等公司已经关停在俄罗斯的办事处，同时也不再为俄罗斯提供软件、硬件的销售，但通常情况下在合同约定的时间内这些公司还会继续提供售后服务。

微软的做法与上面这些公司略有区别，微软将俄罗斯的办事处的员工裁掉，但没有关闭办事处而是表示暂停新的销售活动，同时继续履行与俄罗斯客户的合同义务。

只不过微软禁止俄罗斯客户访问 Windows Update，实际上导致 Windows 10、Windows 11、Windows Server 等无法获得自动更新，尽管客户自己下载补丁安装是可以的。

另一方面微软竟然致函俄罗斯客户要求他们续费许可证，范围仅限于未受制裁的俄罗斯私营企业，至于原因大概是不想完全失去俄罗斯市场以及看着被盗版实在是心疼。

此前由于西方制裁俄罗斯已经将盗版技术合法化，即与国防和安全相关的紧急情况下，是可以进行盗版的，法律也授权政府在未经专利持有人同意的情况下扣押资产，不过这也仅限于对俄罗斯实施制裁的国家的法律实体。

制裁和盗版合法化让俄罗斯几乎可以获得各种软件的许可副本，微软称这种情况下该公司损失了 1.25 亿美元。所以现在重新要求私营客户续费许可证就是这个原因，只不过实操起来也是非常困难的。

困难的原因是因为一方面有些私营公司已经不太可能再向微软支付许可费用，另一方面想支付的话也没什么合适的渠道，所以这种情况下微软想要继续收许可费也是大问题。
