Microsoft正在调查导致Windows任务栏和开始菜单无响应并触发 Outlook 和 Teams 登录问题的问题。

据外媒披露称，Windows用户一直在报告单击时看不到 Windows 开始菜单、无法启动现代应用程序以及 Windows 搜索功能被破坏的问题。其他人也报告在尝试登录 Azure Active Directory (Azure AD) 以激活 Office 365应用程序时遇到问题。 [[阅读原文]](https://www.bleepingcomputer.com/news/microsoft/microsoft-investigates-bug-behind-unresponsive-windows-start-menu/)
