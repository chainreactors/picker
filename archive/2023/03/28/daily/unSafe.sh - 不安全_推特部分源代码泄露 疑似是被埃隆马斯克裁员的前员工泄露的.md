上周五推特向 Github 提交 DMCA 申请要求 Github 删除某账号上传的数据，这部分数据是什么呢：推特部分服务的源代码。

根据相关文件说明，推特在发现其部分系统的源代码于上周五被上传到 Github 且可以公开访问，Github 接到 DMCA 请求后，在同一天完成审核并删除了存储库，避免源代码继续外泄。

然而数据这东西只要发布到网上，就不可能彻底删除的，没人知道有多少用户已经下载了这部分源代码并在其他地方传播，尤其是这些源代码可能已经在 Github 上流传了几个月。

[![推特部分源代码泄露 疑似是被埃隆马斯克裁员的前员工泄露的](https://img.lancdn.com/landian/2022/12/96339.png)](https://img.lancdn.com/landian/2022/12/96339.png)

目前还不清楚这些源代码泄露的时间，**但推特自己认为可能已经上传了几个月**，为了追查来源，推特正在试图让法院下令 Github 披露上传源代码的用户信息。

从法庭申请文件里可以看到推特的要求还不少：要求 Github 披露最初上传源代码的用户个人信息、要求 Github 披露其他可能下载这些源代码的用户的个人信息。

**也就是说如果你之前 “不小心” 看到那个存储库并且顺手下载了一份，那么你的个人身份信息可能就会被披露。**

当然目前法庭还未同意推特的请求，至少对 Github 来说被要求提供下载存储库的用户身份信息不太能接受，但具体还得看法院怎么决定。

另外关于泄露者的身份，有消息称推特高管认为源代码是推特的一名前员工泄露的，这名员工在 2022 年底被解雇，也就是马斯克接管推特后采取的大裁员中被裁的，因此可能是出于报复心理才泄露了源代码。

但目前也不清楚这部分源代码具体涉及推特的哪些系统，如果是关键系统的话，黑客有可能会通过分析源代码寻找漏洞然后入侵推特。

对了，最后还是要提醒一下各位，无论是删库跑路还是私自下载源代码并私密/公开分享都是犯罪行为，所以千万不能因为头脑发热就做违法的事。
