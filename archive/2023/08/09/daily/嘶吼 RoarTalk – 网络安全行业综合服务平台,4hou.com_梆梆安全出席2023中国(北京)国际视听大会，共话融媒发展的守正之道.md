# 梆梆安全出席2023中国(北京)国际视听大会，共话融媒发展的守正之道

梆梆安全
[行业](https://www.4hou.com/category/industry)
2023-08-08 15:50:17

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)95112

收藏

导语：8月6日，以“智慧广电 未来视听”为主题的2023中国(北京)国际视听大会(CIAC2023)在北京亦创国际会展中心圆满闭幕。产学研各界重量级嘉宾通过视听网络安全发展论坛等主题论坛展开专业研讨，聚焦行业领域的新需求、新研究、新应用，描绘新视听行业发展蓝图。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230808/1691480875120584.png "1691480848143365.png")

作为移动应用安全领域的代表企业，梆梆安全受邀参与「视听网络安全发展论坛」，全面布局“第四大运营商—广电”业务市场，研发中心总监张智发表《移动应用安全防护与监测在融媒体系统中的应用》主题演讲，结合梆梆安全在移动应用安全领域的业务实践，为视听科技中如何保护个人隐私信息、防范虛假内容传播提供建设性思考。

![image.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230808/1691480876134898.png "1691480860943756.png")

张智介绍到，在新的传播环境下，智能手机和移动设备快速普及，“媒体融合+产业互联”伴随短视频、直播受众的飞速增长而不断深入，为融媒体发展提供了更加广阔的平台，移动音视频等相关产品因直观性、互动性、接地气更强，覆盖越来越多用户，也将软件的相关安全风险推向了风口浪尖：

* 攻击者想尽方法通过线上渠道窃取用户信息；
* 盗版仿冒、移动应用界面劫持等安全事件屡见不鲜；
* 文字、音视频未经允许被复制、传播，开发者的知识产权无法有效保护；
* 无人直播、一机多号等现象层出不穷，媒体内容的真实性有待商榷;
* ......

对此，梆梆安全推出移动应用安全防护与监测在融媒体系统中的解决方案：从应用自身保护、运行环境安全监测、服务端API安全防护三个方面，覆盖应用的开发过程、承载环境及运行过程，为用户提供移动安全建设新范式。

**01  移动新媒体的安全从APP加固开始**

面向行业数字化转型过程中的各位移动应用形态，提供全生命周期的移动应用安全保护，有效防御应用反编译(逆向分析、权限破解、盗版仿冒)、二次打包（植入广告、病毒、流氓软件），内存注入和动态调试等攻击行为（信息窃取、钓鱼欺诈、外挂、充值通道暴露）等安全风险。

**02  终端环境监测，让风险无所遁形**

将梆梆安全已有的安全能力（应用威胁监测、环境脆弱性监测、行为异常管理等）聚合，通过轻量级APP探针采集移动端环境与风险特征，在服务端进行安全事件综合研判，使客户及时掌握其 APP 运行时的风险，并最终完成用户定义的处置行为。

**03  把好服务端的大门，对所有风险说“不”**

梆梆安全基于“端到端的全渠道风险防护”技术理念，结合行业业务终端安全接入具体需求，推出“动静结合、横向端到端联动+实时防御、纵向全渠道联动”的 API 安全解决方案，利用动静结合的防御思想，实现端到端的风险联动防御，通过多渠道的情报共享，实现多渠道的协同防御，全面打造“安全合规、架构稳定、产品易用”的终端安全环境，构建高质量融媒体业务安全体系。

保护您的软件——既是梆梆安全的使命，也是梆梆安全作为网络安全企业应当承担的社会责任。截至目前，梆梆安全已打造适配融媒体行业的安全产品、方案和服务，帮助行业客户构建全方位、多层次的移动应用安全体系。梆梆安全未来将不断深入洞察行业用户需求，以数字信息化推动新发展，支撑数字中国整体战略的落地。
