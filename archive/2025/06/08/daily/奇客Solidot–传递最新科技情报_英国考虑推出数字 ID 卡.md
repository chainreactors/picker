## 英国考虑推出数字 ID 卡

[![英国](https://icon.solidot.org/images/topics/topicBritain.png?123)](/search?tid=145 "英国")

[Wilson](/~Wilson) (42865)发表于 2025年06月07日 18时42分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81494&appkey=1370085986&title=%E8%8B%B1%E5%9B%BD%E8%80%83%E8%99%91%E6%8E%A8%E5%87%BA%E6%95%B0%E5%AD%97%20ID%20%E5%8D%A1 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自终极之门**

英国政府正考虑为所有人成年人推出数字 ID 卡，此举旨在应对非法移民危机。被称为 BritCar 的数字 ID 将用于核实个人在英国的生活和工作权利，它储存在智能手机上，关联政府数据，可用于核实福利和监控福利欺诈。构建 BritCar 的系统预计将耗费 4 亿英镑，它将作为一款免费使用的手机应用提供给用户，预计每年的运营成本为 1000 万英镑。
