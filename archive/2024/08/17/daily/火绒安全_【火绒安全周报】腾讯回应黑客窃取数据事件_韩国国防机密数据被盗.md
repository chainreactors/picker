# 腾讯回应黑客窃取数据事件

#

近日，名为"Fenice"的黑客声称窃取了腾讯14亿用户数据，包括邮箱、电话号码和QQ号等。泄露数据为44GB的压缩包，解压后可达500GB。对此，腾讯方相关人士在微博回应称此消息为虚假信息，同时表示过去两年，类似谣言多次被海外黑客炒作，数据口径不断膨胀，并被恶意关联到国内多个互联网产品和公司。建议广大用户提高网络安全意识，加强个人信息保护。

# 澳大利亚黄金矿商遭遇勒索软件攻击

#

近日，澳大利亚黄金矿商Evolution发布公告称，公司于8月8日发现其IT系统遭受勒索软件攻击。经过Evolution公司与其外部网络安全专家联合调查，此次攻击事件已得到有效控制，预计不会对其运营产生重大影响。Evolution公司强调，其在事件处理过程中以保护人员健康、安全和隐私以及公司系统和数据为重点。目前，Evolution公司已向澳大利亚网络安全中心报告了此次攻击事件。

# 北美汽车经销商IT系统被攻击

#

北美知名汽车经销商AutoCanada在加拿大经营84家经销商，涵盖28个品牌，并在美国伊利诺伊州设有分部。近日，AutoCanada公司披露了一起影响其IT系统的网络安全事件。事件发生后，该公司迅速采取措施保护其网络和数据安全，并与网络安全专家合作展开全面调查。AutoCanada公司表示，尽管其业务运营仍处于开放状态，但在相关系统完全恢复前，仍存在运营中断风险。目前，该事件正在进一步调查中，其具体范围、性质及影响尚不明确。

# 韩国国防机密数据被盗

#

据报道，韩国发生国防机密数据泄露事件，被泄露的数据包括K2主战坦克和"Baekdu"、"Geumgang"侦察机在内的关键军事技术资料。据悉，K2坦克的一家备件供应商工程师携带图纸等敏感资料跳槽后，其新雇主试图将技术出口，导致K2坦克数据泄露；而"Baekdu"和"Geumgang"侦察机的数据泄露是朝鲜黑客攻击韩国国防承包商所致。此次泄露的数据会被朝鲜用于发展无人机技术和监视规避技术，对韩国国防构成严重威胁。目前，韩国政府已要求立即采取措施加强网络安全，并呼吁紧急修订法律以保护国家安全。

# 两名网络罪犯因生活奢侈被调查

#

据报道，两名外籍男子因其资金来源不明的奢侈生活方式被美国联邦调查局调查。调查显示该两名男子涉嫌共用名为"Makein"的账户，参与暗网平台WWH及相关论坛的非法活动，包括非法交易登录凭据、个人身份信息、恶意软件、伪造的身份证明文件和财务凭证，甚至还提供网络犯罪培训。联邦调查局指控两名嫌疑人共谋贩运未经授权的访问设备和拥有15个或更多未经授权的访问设备，并要求法院批准刑事起诉书。目前，其中一人已被逮捕，另一人情况不明。
