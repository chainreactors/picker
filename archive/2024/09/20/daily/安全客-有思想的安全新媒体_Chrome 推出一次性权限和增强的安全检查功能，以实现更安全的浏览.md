# Chrome 推出一次性权限和增强的安全检查功能，以实现更安全的浏览

译文仅供参考，具体内容表达以及含义原文为准。

谷歌宣布将为其 Chrome 浏览器推出一组新功能，让用户在上网时更好地控制他们的数据，并保护他们免受在线威胁。

“使用最新版本的 Chrome，您可以利用我们升级后的安全检查，更轻松地选择退出不需要的网站通知，并且仅向网站授予一次选择权限，”这家科技巨头表示。

对 Safety Check 的改进使其能够在后台自动运行，通知用户它已采取的操作，例如撤销他们不再访问的网站的权限，以及标记可能不需要的通知。

它还旨在通知用户需要解决的安全问题，同时自动撤销 Google 安全浏览识别的可疑网站的通知权限。

“在桌面上，如果您安装了任何可能对您构成安全风险的 Chrome 扩展程序，安全检查将继续通知您，然后将您带到扩展程序页面并显示一个带有快速控件的摘要面板以删除它们，”Chrome 产品经理 Andrew Kamau 说，

这家搜索和广告公司补充说，安全检查除了为用户提供启用 Google 安全浏览保护的选项外，还能够在存储在 Google 密码管理器中的用户名或密码涉及数据泄露时发出警告。

其他一些关键更新包括直接在 Pixel 和 Android 设备上的通知抽屉上取消订阅不需要的网站通知的能力，以及授予 Android 和桌面版 Chrome 的一次性权限。

“使用此功能，您可以选择仅向网站授予一次特定权限——例如访问您的摄像头或麦克风，从而帮助您更好地管理您的在线隐私，”Kamau 说。“一旦你离开网站，Chrome 就会撤销权限。在您再次明确授予这些权限之前，该网站将无法使用这些权限。
