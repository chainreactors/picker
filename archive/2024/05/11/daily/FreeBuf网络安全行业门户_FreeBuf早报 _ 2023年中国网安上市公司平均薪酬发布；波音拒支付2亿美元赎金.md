所属地 上海

## 全球动态

### 1. 天津发布自贸试验区数据出境管理负面清单

天津自贸试验区管委会、天津市商务局9日发布《中国（天津）自由贸易试验区数据出境管理清单（负面清单）（2024年版）》，规范和促进数据有序跨境流动，为企业数据出境提供精准指导。【[阅读原文](https://news.youth.cn/hotnews_41880/202405/t20240510_15244886.htm)】

### 2. 2023年中国网安上市公司平均薪酬与绩效排行

国内网络安全上市公司2023年的人均薪酬中位数是30.71万（去年是29.1万），平均数是30.8万（去年是29.5万）。【[阅读原文](https://www.secrss.com/articles/65982)】

### 3. 德国警方开始在当地部分地区使用面部识别技术

柏林检察官办公室告诉德国媒体 Netzpolitik，德国东部萨克森州和柏林的警方已经部署了这套系统，它可以在几秒钟的时间延迟内处理面部图像。【外刊-[阅读原文](https://cybernews.com/news/police-live-facial-recognition-germany/)】

### 4. 埃森哲赢得7.89亿美元的合同，为美国海军网络安全能力提供支持

全球专业服务公司埃森哲（NYSE： ACN）的子公司埃森哲联邦服务公司（AFS）已获得一份价值7.89亿美元的合同，以加强美国海军海上力量的网络安全。【外刊-[阅读原文](https://cybersecuritynews.com/accenture-wins-789-million-contract/)】

### 5. CrowdStrike 和 NinjaOne 宣布建立合作伙伴关系，以弥合 IT 和 SOC 团队之间的差距

CrowdStrike 和 NinjaOne 是端点管理、安全和可见性 IT 平台服务的领跑者，目前二者已宣布建立战略合作伙伴关系。【外刊-[阅读原文](https://cybersecuritynews.com/crowdstrike-ninjaone-bridge-gaps/)】

### 6. LockBItSupp否认自己就是霍罗舍夫

LockBit 勒索软件团伙的头目网络名叫 LockBitSupp，他在接受采访时称国际执法部门犯了一个错误：他不是德米特里·尤里耶维奇·霍罗舍夫，执法部门将错误的人附加到了他的名称上。【外刊-[阅读原文](https://therecord.media/lockbitsupp-interview-ransomware-cybercrime-lockbit)】

## 安全事件

### 1. 采用模块化设计逃避检测，安全公司披露恶意载入器 HijackLoader

安全公司 Zscaler 近日发布报告，披露了一款采用“模块化设计”的恶意载入器 HijackLoader，这款载入器可以加装各种模块以进行脚本注入、远程命令执行等操作，同时还能够根据用户设备端情况“智能”逃避检测。【[阅读原文](https://www.ithome.com/0/766/954.htm)】

### 2. 波音公司去年 43GB 数据泄露，拒绝向黑客支付 2 亿美元赎金

波音公司本周三披露细节，称 2023 年 11 月公司遭到勒索软件攻击，黑客索要高达 2 亿美元（的赎金，最终公司决定拒绝支付。【[阅读原文](https://www.ithome.com/0/766/908.htm)】

### 3. 影响 4900 万客户，戴尔通知发生数据泄露：涉及姓名、住址等订单信息

戴尔公司5月9日通过电子邮件方式通知用户，表示其个人信息可能已经泄露，推荐用户尽快更改，以保护个人隐私和财产。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/dell-warns-of-data-breach-49-million-customers-allegedly-affected/)】

### 4. 微软再次证实 Windows Server 四月更新会导致新问题

微软证实 Windows Server 四月更新可能会导致 Local Security Authority Subsystem Service (LSASS)进程崩溃，域控制器重启的问题。LSASS 服务被用于处理安全策略、用户登录、访问令牌创建和密码更改。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/microsoft/microsoft-april-windows-server-updates-also-cause-crashes-reboots/)】

### 5. 广泛使用的Telit Cinterion调制解调器容易受到短信接管攻击

Telit Cinterion 蜂窝调制解调器中的安全漏洞广泛用于工业、医疗保健和电信等行业，可能允许远程攻击者通过短信执行任意代码。【外刊-[阅读原文](https://www.bleepingcomputer.com/news/security/widely-used-telit-cinterion-modems-open-to-sms-takeover-attacks/#google_vignette)】

### 6. 研究人员发现针对云托管 AI 模型的“LLMjacking”计划

###

网络安全研究人员发现了一种新的攻击方式，该攻击利用被盗的云凭据来针对云托管的大型语言模型 （LLM） 服务，目的是向其他威胁参与者出售访问权限。【外刊-[阅读原文](https://thehackernews.com/2024/05/researchers-uncover-llmjacking-scheme.html)】
