# CNBC专访CertiK联创顾荣辉：从形式化验证到AI赋能，持续拓展Web3.0信任边界

近日，CertiK联合创始人、哥伦比亚大学教授顾荣辉接受全球知名财经媒体CNBC阿拉伯频道专访，围绕形式化验证的行业应用、AI在区块链安全中的角色，以及新兴技术风险等议题，分享了其对Web3.0安全未来的深刻洞察。

顾荣辉表示，形式化验证是构建区块链安全基础的关键，其作用在于以数学方法验证代码行为是否符合预期逻辑。这种方法比传统审计工具更为严谨，也更具挑战性。他曾多次强调，区块链的本质不只是技术创新，更是信任的重构，而真正可信的系统，必须是可被验证的。

# **形式化验证：构建区块链安全底座**

形式化验证作为高门槛、高强度的审计方式，要求深度的技术积累和对底层逻辑的理解。顾荣辉在专访中指出，这种方法虽然复杂，但在保障区块链安全性和可控性方面具有不可替代的优势。

他与形式化验证的渊源可以追溯回其求学阶段。2016年，他与耶鲁大学教授邵中合作，成功研发出全球首个完全经过形式化验证的多核操作系统内核CertiKOS，并被谷歌评价为无懈可击（Unbreakable）。这一成果不仅验证了该技术在现实世界的应用作用，也为其走入行业实践打下基础。

此后，顾荣辉于2017年创立CertiK，将形式化验证引入区块链场景。如今这一技术已广泛应用于如ZK等多个注重隐私与安全的头部项目，以及蚂蚁集团等大金融机构。顾荣辉在专访中表示，为提升形式化验证效率，CertiK内部目前已构建了如AI智能分析等辅助工具，利用其自动识别代码特征、生成逻辑证据。

在推动学术成果落地的同时，顾荣辉也始终密切关注行业面临的新型威胁。针对“超级计算机或量子计算可能威胁区块链安全”的担忧，他坦言这类技术具备潜在风险，但现阶段仍不足以构成实质威胁。“超级计算机的能力尚不足以攻破现有区块链系统”，顾荣辉认为，“更重要的是，行业已在积极构建抗量子算法和防御机制，拥有相关解决方案是可能的。”

![]()

# **AI应用：重塑Web3.0安全格局**

顾荣辉在专访中特别强调，AI正在重塑区块链安全格局，尤其在智能合约审计与链上威胁识别方面已展现巨大潜力，“正确应用AI，可以为区块链安全体系带来实质性的提升。”

他进一步表示，CertiK的研究显示，Web3.0项目中的恶意行为手法日益复杂，一些钓鱼项目可长时间追踪用户并实施精准诈骗。但通过AI模型的快速识别能力，这类攻击如今往往能在数小时内被识别与处置。

目前，CertiK已将AI广泛应用于漏洞挖掘与审计流程中。通过解析链上预警数据，辅助识别漏洞成因与攻击路径；同时利用AI技术进行已知漏洞模式扫描、代码逻辑分析与报告生成，构建出以“人机协同”为核心的智能审计体系，为人工审计团队提供可信的数学支撑与效率加成。

但随着AI应用的日益普及，攻击者同样可利用其强化攻击手段。顾荣辉认为，这意味着安全对抗的门槛被抬高，行业需要更强大的安全解决方案。

# **技术愿景：拓展Web3.0信任边界**

顾荣辉始终认为，Web3.0发展的下一个阶段，将从“可用”迈向“可信”。效率与创新并非终点，责任与信任才是技术演进的核心价值。只有嵌入可验证的安全机制，区块链才能真正成为值得信赖的全球金融基础设施。

这正是CertiK自创立以来所坚持的使命——以形式化验证为核心，打造涵盖投资孵化、安全审计、合规咨询等全生命周期的安全产品体系，为项目提供可验证的安全性，为行业设立更高的安全标准。

对顾荣辉而言，这不仅是企业战略，更是人生选择。从耶鲁求学到哥伦比亚执教，从学术研究到创业实践，他始终专注于将形式化验证从理论转化为实践基石，以技术驱动行业信任。

作为哥大教授和CertiK首席执行官，顾荣辉横跨学术与商业两大领域，目标却始终聚焦于守护Web3.0安全。他曾提及其教学理念植根于将现实洞察融入学术讨论，旨在为网络安全领域的新一代构建认知与能力基础。

“保障Web3.0的安全性，不仅是一项技术挑战，更是一种社会责任。”顾荣辉多次公开表示，“我们的目标，是让这个行业更安全，也更值得信赖。”

在技术创新与合规建设的交汇点上，顾荣辉正以学者的严谨、创业者的勇气与行业建设者的担当，不断拓宽Web3.0的信任边界，引领行业走向可验证、可持续发展的未来。
