## 印度封杀 Matrix 客户端 Element

[![审查](https://icon.solidot.org/images/topics/topicCensorship.png?123)](/search?tid=120 "审查")

[Wilson](/~Wilson) (42865)发表于 2023年05月05日 18时48分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74870&appkey=1370085986&title=%E5%8D%B0%E5%BA%A6%E5%B0%81%E6%9D%80%20Matrix%20%E5%AE%A2%E6%88%B7%E7%AB%AF%20Element "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自机器岛**

印度以传播恐怖主义的名义封杀了 14 款消息应用，其中之一是 Matrix 客户端 Element。Matrix 是一个端对端加密、去中心化的实时通讯系统，它没有中心服务器，通过网桥与其它平台互通。Matrix 客户端在功能上类似于浏览器，只是用于连接 Matrix 通信网络，并不能控制上面的内容。开发 Element 的公司表示它没有收到该决定的提前通知，它猜测印度对 Element 的封杀可能是出于对去中心化和联邦式服务如 Matrix 的误解。Element 只是一个客户端，印度用户完全可以使用其它客户端访问 Matrix。Element 表示它坚信端对端加密是一种事关通信安全和隐私的基本人权，部分政府在打击恐怖主义等名义下破坏端对端加密是对基本人权的攻击。
https://element.io/blog/india-bans-flagship-client-for-the-matrix-network/
