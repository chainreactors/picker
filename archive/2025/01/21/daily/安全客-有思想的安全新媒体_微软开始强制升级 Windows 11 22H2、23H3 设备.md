# 微软开始强制升级 Windows 11 22H2、23H3 设备

![Windows 11]()

微软已开始向运行 Windows 11 22H2 和 23H2 家庭版和专业版的符合条件的非托管系统强制推出 Windows 11 24H2。

根据该公司生命周期策略网站的说法，Windows 11 23H2将于2025年10月结束支持，而Windows 11 22H2家庭版和专业版已于2024年10月结束支持。

“我们已经进入了分阶段推出 24H2 版本的新阶段。运行 Windows 11 家庭版和专业版、23H3 版和 22H2 版的符合条件的设备将逐步更新到 24H2 版，”该公司在周四更新的 Windows 发布健康页面上说。

“这一自动更新仅针对不受 IT 部门管理的设备。用户可以选择重启设备的时间或推迟更新。”

Windows 11 24H2也被称为Windows 11 2024更新，于10月份开始作为一个完整的操作系统交换，向运行Windows 11 22H2和23H2的符合条件的设备推出。

如果你不想等待自动升级，可以进入 “设置”>“Windows 更新”，点击 “检查更新”，手动检查更新是否可用。如果您的设备已准备好接受更新，您将看到 “下载并安装 ”Windows 11 24H2 的选项。

想要第一时间获得最新功能更新的用户也可以进入 “设置”>“Windows 更新”，启用 “获得最新更新 ”选项。

如果你还没有准备好安装微软推荐的 24H2 升级，也可以从 “设置”>“Windows 更新 ”中选择暂停更新的时间，从而暂停下载和安装更新。不过，在达到该时间限制后，您必须安装最新更新。

微软还应用了一些保障措施，以防止 Windows 11 24H2 功能更新提供给配置或软件不兼容的系统。

其中包括启用了自动 HDR 的 PC、运行 Dirac 音频改进软件的 PC、《Asphalt 8》（空降）和《刺客信条》游戏、部分华硕设备，以及带有集成摄像头或运行安全考试浏览器和 Easy Anti-Cheat 应用程序的设备。

雷德蒙德还提供了分步指南和支持文档，帮助用户解决在 Windows 11 24H2 升级过程中遇到的问题。
