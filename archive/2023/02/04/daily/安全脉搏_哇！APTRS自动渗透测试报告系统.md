**工具简介**

APTRS（自动渗透测试报告系统）是 Python 和 Django 中的自动报告工具。该工具允许渗透测试人员直接创建报告，而无需使用传统的 Docx 文件。它还提供了一种跟踪项目和漏洞的方法。

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-195332-1675413692.png)

**安装使用**

已经在 Kali Linux 2022.2/3、Ubuntu 20.04.5 LTS、Windows 10/11 上使用 Python 3.8.10 进行了测试。

Windows安装

```
git clone https://github.com/Anofcyber/APTRS.git
cd APTRS
install.bat
run.bat
```

Linux安装

```
git clone https://github.com/Anofcyber/APTRS.git
cd APTRS
install.sh
run.sh
```

**下载地址**

https://github.com/Anof-cyber/APTRS
