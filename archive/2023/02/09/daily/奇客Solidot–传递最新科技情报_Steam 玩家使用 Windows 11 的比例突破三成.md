## Steam 玩家使用 Windows 11 的比例突破三成

[![Windows](https://icon.solidot.org/images/topics/topicwindows.png?123)](/search?tid=33 "Windows")

[Wilson](/~Wilson) (42865)发表于 2023年02月08日 17时31分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74075&appkey=1370085986&title=Steam%20%E7%8E%A9%E5%AE%B6%E4%BD%BF%E7%94%A8%20Windows%2011%20%E7%9A%84%E6%AF%94%E4%BE%8B%E7%AA%81%E7%A0%B4%E4%B8%89%E6%88%90 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自梦书迷宫**

根据 Valve 公布的最新 Steam 硬件和软件调查，Steam 玩家使用 Windows 11 的比例突破三成，主要是使用的 Windows 10 的玩家比例略有减少。Windows 11 增加了 1.91 个百分点达到 30.33%，而 Windows 10 减少 1.96% 但仍然有 63.46%，刚刚停止安全更新的 Windows 7 还有 1.6%， macOS 有 2.61%，Linux 维持 1.38% 不变。三款最流行的 GPU 都属于英伟达：GTX 1650、GTX 1060 和 RTX 3060L（笔记本版本）。
