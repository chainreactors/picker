## Linux 和开源软件由谁开发

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")
[![Linux](https://icon.solidot.org/images/topics/topiclinux.png?123)](/search?tid=7 "Linux")

[Wilson](/~Wilson) (42865)发表于 2023年02月27日 14时11分 星期一 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74241&appkey=1370085986&title=Linux%20%E5%92%8C%E5%BC%80%E6%BA%90%E8%BD%AF%E4%BB%B6%E7%94%B1%E8%B0%81%E5%BC%80%E5%8F%91 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自虚拟现实**

开源云数据平台公司 Aiven 分析了 GitHub 代码项目，发现最主要的代码贡献者来自于 AWS、英特尔、Red Hat、Google 和微软等科技公司。Aiven 根据贡献者的数量、贡献者的项目以及递交数量三个指标进行分析，结果是微软和 Google 并列第一，Red Hat 第三，英特尔第四，之后是 AWS 和 IBM。在 2022 年第四季度，微软开发者递交的代码数量最多为 128,247，Red Hat 第二 125,012，Google 只有 94,961。而 Google 参与开源项目的人数最多有 5,757 人，微软 5,513 人，Red Hat 3,656 人。对 Linux Kernel 5.16 到 6.1 时代码作者的分析有类似的发现，开发者主要来自 AMD（主要是驱动代码多）、英特尔、Google、Linaro、主要的 Arm Linux 开发组织，Meta 和 Red Hat 等企业，只有 7.5% 的内核开发来自个人开发者。
https://www.theregister.com/2023/02/24/who\_writes\_open\_source/
