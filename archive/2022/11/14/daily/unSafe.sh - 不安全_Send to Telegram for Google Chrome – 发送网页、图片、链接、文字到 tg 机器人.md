## Send to Telegram for Google Chrome

Send to Telegram for Google Chrome 在 [GitHub](https://github.com/phguo/Send-to-Telegram-Chrome-extension) 开源，在设置 tg bot 的 API Token 之后，就可以将网络内容发送给机器人。

安装扩展之后，需要重启浏览器，否则无法看到右键菜单。

### 具体步骤

首先，创建机器人依旧很简单，和 @[Botfather](https://t.me/BotFather) 聊天，创建一个 bot，然后获取 API Token。

其次，与另外一个机器人 @[myid](https://t.me/myidbot) 聊天，然后回复 /getid 获得用户 ID。

最后，点击浏览器扩展栏上的 Send to Telegram for Google Chrome 图标，输入 API TOken 和 ID，就完成了设置。

记得重启下浏览器，否则不好用 😂

### 使用

打开需要保存的标签页，直接点击 Send to Telegram for Google Chrome 按钮，就会将当前网页的网址发送给机器人。

![Send to Telegram for Google Chrome - 发送网页、图片、链接、文字到 tg 机器人 1](https://static1.appinn.com/images/202211/screen-appinn2022-11-13-15-39-31.jpg!o "Send to Telegram for Google Chrome - 发送网页、图片、链接、文字到 tg 机器人 2")

另外，还可以选中文本，或者直接对图片右键，点击 **Push this image to Telegram Bot**，很快就能在机器人里收到了。

但注意如果是图片的话，收到的将是图片的 URL，而不是图片本身，这点略有遗憾，如果直接收到图片就完美了。
