As always, we continue to bring you new and easier ways to build apps using our pro-code solutions. Following a cloud mindset, we are introducing many features as we complete them. Keep your suggestions coming as this helps us prioritize our development efforts. We look forward to ongoing innovation and collaboration for the rest of the year.

If you want to meet with fellow developers and the SAP engineers who build our professional development tools, I encourage you to join our monthly SAP Fiori elements and SAP Fiori tools roundtable group.
[https://blogs.sap.com/2021/07/31/peer-to-peer-learning-at-the-sap-fiori-elements-and-sap-fiori-tools...](https://blogs.sap.com/2021/07/31/peer-to-peer-learning-at-the-sap-fiori-elements-and-sap-fiori-tools-roundtable/)
You can email me if you would like an invitation.

We publish the SAP Fiori development newsletter every other month to share new information with developers in the SAP ecosystem about SAPUI5, SAP Fiori elements, SAP Fiori tools, and SAP Business Application Studio.

To receive this content in your inbox, please [subscribe to receive the SAP Fiori development newsletter](https://www.sap.com/cmp/nl/sap-fiori-development-newsletter/index.html).

oliver.graeff, raz.korn, and peter.spielvogel.

**SAP Fiori tools 2302 update has more for OData V4 and the flexible programming model, better troubleshooting, more open source content**

The latest SAP Fiori tools update focuses on making annotations easier, expanding support for flexible programming model and OData V4, and releasing more open-source modules that can be reused by the community.

**The SAP Fiori elements feature showcase with RAP and ABAP CDS: what’s new in Release 2302?**

Continuous innovation in SAP Fiori elements and ABAP CDS annotations means more new features more regularly. This blog post discusses the latest UI features that are possible with ABAP CDS annotations.
[https://blogs.sap.com/2023/02/23/the-sap-fiori-elements-feature-showcase-with-rap-and-abap-cds-whats...](https://blogs.sap.com/2023/02/23/the-sap-fiori-elements-feature-showcase-with-rap-and-abap-cds-whats-new-in-release-2302)

**Want to know what’s new in SAPUI5 and SAP Fiori elements? Check the What’s New Viewer**

In addition to the SAPUI5 demo kit, the What’s New Viewer allows you to see all the new updates on one page. Filter options let you focus on the version, type of change, or valid-as-of date.

**SAP Business Application Studio update for 2301**

While SAP Business Application Studio releases new functionality every two weeks, we summarize the innovations every few months. This time, we cover updates to the storyboard, run configurations, usability, and visual editor for application logic.
<https://blogs.sap.com/2023/01/26/product-updates-for-sap-business-application-studio-release-2301/>

**SAP Fiori elements for OData V4 knowledge resources**

Given all the interest in SAP Fiori elements for OData V4 and the flexible programming model, we have prepared a list of resources that will help you navigate our documentation and other sources to learn about our latest development innovations.
<https://blogs.sap.com/2023/02/24/sap-fiori-elements-for-odata-v4-knowledge-resources/>
