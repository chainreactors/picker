Its finally here! The much awaited topic of diving deep into our new SAP Service Cloud Version2 solution!

In this 4 day live session you will be able to interact with an expert and learn about the configurations, features and capabilities this new solution offers.

This live session is relevant for Administrators, Consultants, Business users or anyone interested in learning about SAP Service Cloud Version 2.

You can register and book this session on Learning Hub in two ways:

(Please Note:  You must have a subscription and  login with your S-user ID on **SAP Learning Hub**.)
