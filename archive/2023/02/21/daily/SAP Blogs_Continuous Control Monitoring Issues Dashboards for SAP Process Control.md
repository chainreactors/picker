SAP Analytics Cloud is a software as a service product SAP customers use for business intelligence, augmented analytics, predictive analytics, and enterprise planning.

Depending on their data source, SAP customers have the option of choosing a live connection or importing data into SAP Analytics Cloud.

In this blog, we will focus on how SAP Process Control users can visualize continuous control monitoring issues in SAP Analytic Cloud.

Below is a diagram displaying how the live data connection accesses on-premise data sources.

**Live Data Connection to SAP S/4HANA On-Premise Using a Direct CORS Connection Via Unified Connectivity**

![](/legacyfs/online/storage/blog_attachments/2023/02/1-83.png)

With SAP Process Control 12.0 support pack 19, backend support is delivered for continuous control monitoring issues dashboards in SAP Analytics Cloud. The following is a selection of the dashboards:

![](/legacyfs/online/storage/blog_attachments/2023/02/1-v2.png)

![](/legacyfs/online/storage/blog_attachments/2023/02/3-27.png)

SAP Process Control and SAP Analytics Cloud (through live data connection) support:

* the resolution of continuous control monitoring issues across enterprises

* connections to SAP Analytics Cloud in a nearly real-time manner while storing data in the original source system. The metadata are stored inside SAP Analytics Cloud. On the other hand, none of the data that is generated by SAP Process Control is stored in SAP Analytics Cloud.

* encrypted and protected connections

**\***For more information, see SAP note [3239484](https://launchpad.support.sap.com/#/notes/3239484).
