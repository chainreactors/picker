# 新增4场｜网安出海正当时——2025全球展会攻略一览

喵酱

安全村SecUN

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNlhxVOKd6KcJ0O4ox1Y2IFSqSdhxlsiatKkL22icGicuePYrqviciav2lFkBF239ic3JViaA3OabQtgrd7Q/640?wx_fmt=png&from=appmsg)

**在全球化的浪潮中，中国网络安全企业以其创新技术和专业服务，正逐步成为国际舞台上的重要力量。**

为了更好地让海外客户了解和认识中国的网安企业，斯元Z-ONE精心策划了2025年的全球展会攻略。这一年，我们将带领您走进一系列国际网络安全盛会，让海外客户直观感受中国网安企业的实力与魅力。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpOU7lJB2Hc8jjcWhEvKcy2pfgzEbWach4OqnCIVF5trXriaoz1vibRnIDA1LG5ib8YnK4LXmV5y8LP3w/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxleeYbWP3yiabR8uB3ndm7uwfHvUOyHIjrq6F7dC2EDSZgpVNC7ubqLA/640?wx_fmt=jpeg&from=appmsg)

**Cybersec Asia·泰国曼谷·1月**

Cybersec Asia是经验丰富的网络安全专家和新一代初创企业与同行分享知识、共同应对网络安全挑战的平台。各行各业的企业和机构将学习如何提高网络复原力和保护其核心。

Cybersec Asia 2025 将于2025年1月22日至23日在泰国曼谷举行。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNP8hrwn9jia90LbNxTqjsexlib0P0HibFnVrKuBxicaSiaIk8DWrYqVjxIu7TVtpG2zWr3X4RouCBWEkA/640?wx_fmt=jpeg&from=appmsg)

**Nullcon·印度果阿·2月&3月**

Nullcon 是一个著名的安全会议，起源于印度果阿，并已扩展到柏林（2024）。它为安全专业人士、政府官员和行业领袖提供了一个平台，以从全球安全专家呈现的创新研究中获得见解。会议专注于进攻和防御技术、新兴漏洞，并包括捕旗（CTF）比赛、研讨会以及涵盖各种安全主题的多个分会场活动。

Nullcon将于2025年3月1-2日举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxwHcaENMDiannOMVcvoEAuYfTUcpdses0yz1AwI1ibTD1IDyG3ECfVRtw/640?wx_fmt=png&from=appmsg)

**Cyber Security Summit·新加坡·3月**

Cyber Security Summit是全球系列活动的一部分，在多个大洲的 10 多个国家吸引了大量观众，汇聚超过 500 名首席信息安全官、首席信息官、IT 安全负责人及其他网络安全专业人士，本次峰会将为您提供有效防御威胁、适应变化条件以及在不断发展的数字世界中推进组织网络安全战略所需的重要工具和见解。

Cyber Security Summit 新加坡站将于2025年3月6-7日举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxVJicZn2p63G4qt7rhDPdQ7PaiaDO1QcaZbzpZHaqztB9sMTEDPSgYbAw/640?wx_fmt=png&from=appmsg)

**Cyber Security Summit·菲律宾马尼拉·3月**

Cyber Security Summit是全球系列活动的一部分，在多个大洲的 10 多个国家吸引了大量观众，汇聚超过 500 名首席信息安全官、首席信息官、IT 安全负责人及其他网络安全专业人士，本次峰会将为您提供有效防御威胁、适应变化条件以及在不断发展的数字世界中推进组织网络安全战略所需的重要工具和见解。

Cyber Security Summit 菲律宾站将于2025年3月20日举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxiadg5xFQohu1aTbWar4Zq8er7pAXqVO91Vp9AapXL7LWova9U5kdvhA/640?wx_fmt=png&from=appmsg)

**Black Hat Asia·新加坡·4月**

Black Hat Asia 2025将重返新加坡滨海湾金沙，为期四天的活动包括针对各种技能水平的专业网络安全培训课程、峰会日和为期两天的主会场会议。Black Hat Asia 2025 还将举办简报会，由来自世界各地的专家介绍网络安全风险、发展和趋势方面的最新研究成果，在阿森纳举办数十场开源工具演示会，提供强大的商务大厅、交流机会、社交活动等。

Black Hat Asia 2025将于2025年4月1-4日在新加坡举办。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxbuesvfwKQtUichSdheY3nvkbas27ybCdvO1wZQweQ0rkgic88QoDK2zw/640?wx_fmt=jpeg&from=appmsg)

**GITEX ASIA·新加坡·4月**

GITEX ASIA致力于为数字经济构建全新商业生态，涵盖公共服务、大型企业、中小企业、数字城市、智慧出行、零售革新、金融科技和智能制造等多个领域。它将成为连接亚洲市场与全球网络的桥梁，汇聚全球科技巨头、独角兽企业创始人、顶级投资者、人工智能先锋、政府要员以及尖端研发机构和学术精英。

这个亚洲最大规模的科技盛会将云集1,000家全球领先企业和创新创业公司，吸引20,000多名高质量科技买家，汇聚来自120多个国家和地区的250多位活跃投资者，以及引领行业前沿的科技思想领袖。

GITEX ASIA 将于2025年4月23-25日在新加坡举办。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNP8hrwn9jia90LbNxTqjsexSLqmL3K2LpQbpMCepOwNfezAhyntHV3H0lBA1oUFMdcv9UvXSRwQlQ/640?wx_fmt=jpeg&from=appmsg)

**RSA Conference·美国旧金山·4月&5月**

RSA Conference是全球最大和最具影响力的网络安全专业人士聚会之一。它提供了一个全面的议程，包括主题演讲、小组讨论和实践课程，涵盖广泛的主题，包括密码学、风险管理和新兴威胁。大会还举办了一个广泛的展览，展示最新的安全产品和创新。

RSA Conference将于2025年4月28-5月1日在美国旧金山举办。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxWkj3FlbicIhCe9Vvqiaold8Mojpc4O2TnTIWIAu7icEeLlE0z8naxyKow/640?wx_fmt=jpeg&from=appmsg)

**GISEC Global·迪拜·5月**

GISEC Global是中东和非洲最大的网络安全活动，为行业、政府和思想领袖提供了一个聚集在一起的机会，以应对全球安全专业人员工作组中最紧迫的威胁、问题和对策。

GISEC Global 2025将于2025年5月6-8日举行。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxEcpNicPG63LDB43rUFaEd5MtQms8fwNpoCGVr4jOhia1gbO8lwcbtcJg/640?wx_fmt=png&from=appmsg)

**Cyber Security Summit·泰国曼谷·5月**

Cyber Security Summit是全球系列活动的一部分，在多个大洲的 10 多个国家吸引了大量观众，汇聚超过 500 名首席信息安全官、首席信息官、IT 安全负责人及其他网络安全专业人士，本次峰会将为您提供有效防御威胁、适应变化条件以及在不断发展的数字世界中推进组织网络安全战略所需的重要工具和见解。

Cyber Security Summit 泰国站将于2025年5月20日举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxPvzNJ3yQ5ic9QSCCtzXnpy2gwj98G1SrbN1mvqaQRuw3QWyjJa3gehg/640?wx_fmt=png&from=appmsg)

**CYDES 2025·马来西亚·7月**

CYDES 2025 是第三届世界综合性网络防御、网络安全和新兴技术盛会，将在充满活力的马来西亚布城国际会议中心举行。作为一项国家级活动，CYDES 是一个展示专业知识、解决方案和尖端技术的强大平台，旨在应对当前和未来的网络威胁。

CYDES 2025 暂定于2025年7月1-3日在马来西亚举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxaVgzDn1KQjOZn0N1lBS84sK9m20gt0obykkC9G4kGlw3h5719Yg1yg/640?wx_fmt=png&from=appmsg)

**Cyber Security World Asia·新加坡·10月**

Cyber Security World Asia，是亚洲最大、最专业的云计算及网络安全展之一，围绕网络安全与云行业技术以及对政府、能源、金融服务、医疗等行业的影响，举办顶级内容和讨论。

Cyber Security World Asia将于2025年10月8-9日在新加坡举办。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpM90kXj6xc5L0kjkq6icZqsKIvEcOMmbkcD5rRWrEoEyFxKclz7bA0iaicEWxhLO5S3WUgAnyVQJrFfg/640?wx_fmt=jpeg)

**GITEX Global·迪拜·10月**

GITEX GLOBAL 2025，是全球最具影响力和标志性的规模大型科技展览之一。自成立以来40多年，它一直是科技创作者、投资者和爱好者合作的首要平台，向大家展示在人工智能、网络安全、移动性、可持续科技等领域的科技巨头和创新初创企业。

GITEX Global将于2025年10月13-17日在迪拜举办。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxBHj3XnEqCeHgwyQ07fVsfbR7yBqD3dCRLogRSI3Wx04THA1GL17p5A/640?wx_fmt=png&from=appmsg)

**GovWare·新加坡·10月**

GovWare 是新加坡网络信息和连接平台，提供多渠道接触点，以推动社区情报共享、培训和战略合作。三十多年来，GovWare 作为一个值得信赖的纽带，将亚洲及其他地区的政策制定者、技术创新者和终端用户团结在一起，推动关于最新趋势和关键信息流的相关对话。

GovWare 2025将于2025年10月21-23日在新加坡举办。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNP8hrwn9jia90LbNxTqjsex8Miad2ryF6JukMia1JdB04BibficWYLK3BvV6Qc9UCzUcrMcmKQADQQiaQQ/640?wx_fmt=jpeg&from=appmsg)

**Africa Cyber Defense Forum·肯尼亚内罗毕**

Africa Cyber Defense Forum是一个公共和私营部门合作的大陆平台。论坛通过其精心策划的世界级内容，吸引顶级政府、军事和执法官员、技术专家、商业及其他社会领袖，共同塑造非洲网络空间的大陆、区域和行业议程。

![](https://mmbiz.qpic.cn/mmbiz_jpg/ua7rPK0FKpNP8hrwn9jia90LbNxTqjsexQsiaZIQjBbqBbiaycw3EdNyxIJ2dTxYGgKseWOAbt47lXPjI2GqGibgHg/640?wx_fmt=jpeg&from=appmsg)

**Black Hat MEA·沙特阿拉伯利雅得**

Black Hat MEA是在沙特阿拉伯利雅得举办的领先网络安全会议和展览。该活动汇集了来自世界各地的网络安全专业人士、尖端技术、解决方案提供商和决策者，将几个月的网络交流浓缩在短短三天时间里。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNTsHwQJiayMwDyd1o1UtRwxrYExPfxrvErsEjyIuibfHLWP2znicYFbP1F3mPc3OCia5TFYd8uXuXvqw/640?wx_fmt=png&from=appmsg)

**IndoSec·印度尼西亚雅加达**

在IndoSec上，人们可以向最有影响力的安全思想领袖学习最新的案例研究、流行观点、使用案例、最先进的创新技术、最佳实践等，以缓解当前的威胁情况，为数字领域建立更强大的防御。网络安全峰会提供了富有洞察力的主题演讲、深入的小组讨论和前沿解决方案展示，为安全和业务负责人提供了一个理想的网络安全框架。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpNP8hrwn9jia90LbNxTqjsexJvDfazcdPFT7Rjq3UCEMCicUZGe5l5doVOEKGyV8UKaUX8Ds8EpjPGw/640?wx_fmt=png&from=appmsg)

**Segurinfo·阿根廷布宜诺斯艾利斯**

Segurinfo 是拉丁美洲领先的网络安全会议之一，专注于信息安全实践、隐私法律和企业防御策略。它汇集了来自该地区的首席信息安全官、IT 经理和网络安全专业人士。

![](https://mmbiz.qpic.cn/mmbiz_png/ua7rPK0FKpOU7lJB2Hc8jjcWhEvKcy2pMuq1cBZ7nvh6JEgphKicB45mwp23hxmoL6x1ibSeyg3nvImq6mibMHGIA/640?wx_fmt=png&from=appmsg)

**中国网安科技沙龙·香港·全年**

斯元致力于推动中国网安出海，持续在全球多个国家和地区举办闭门活动。China Cybersecurity Tech Workshop 香港站，是斯元为促进国内与港澳的行业交流及生态对接，辅助加速器成员的业务落地定期举办的沙龙活动。
