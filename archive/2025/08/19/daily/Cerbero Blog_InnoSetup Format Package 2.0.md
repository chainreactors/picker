# InnoSetup Format Package 2.0

Version 2 of the [InnoSetup Format package](https://cerbero.io/packages/innosetupformat/) is now available and includes support for the latest releases of InnoSetup.

![](/wp-content/uploads/2025/08/inno/1.png)

In addition to the format itself and file extraction, make sure to install the [IFPS Format package](https://cerbero.io/packages/ifpsformat/) to inspect the code of setup scripts.
