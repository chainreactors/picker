иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

зҫҺеӣҪзҪ‘з»ңе®үе…Ёе’ҢеҹәзЎҖи®ҫж–Ҫе®үе…ЁеұҖ пјҲCISAпјү е·Іеҗ‘е…ЁзҗғиҒ”йӮҰжңәжһ„е’Ңз»„з»ҮеҸ‘еҮәиӯҰе‘ҠпјҡжҒ¶ж„ҸиЎҢдёәиҖ…жӯЈеңЁз§ҜжһҒеҲ©з”Ёдә”дёӘж–°еҸ‘зҺ°зҡ„е®үе…ЁжјҸжҙһгҖӮиҝҷдәӣжјҸжҙһжңҖиҝ‘иў«ж·»еҠ еҲ°е·ІзҹҘеҲ©з”ЁжјҸжҙһ пјҲKEVпјү зӣ®еҪ•дёӯпјҢж¶өзӣ–дёҖзі»еҲ—жөҒиЎҢзҡ„иҪҜд»¶пјҢе№¶еҜ№жңӘдҝ®иЎҘзҡ„зі»з»ҹжһ„жҲҗйҮҚеӨ§йЈҺйҷ©гҖӮ

**Apache HugeGraph-Server иў«еӣҙж”»**

е…¶дёӯжңҖд»ӨдәәжӢ…еҝ§зҡ„жҳҜ CVE-2024-27348пјҢиҝҷжҳҜ Apache HugeGraph-Server дёӯзҡ„иҝңзЁӢд»Јз Ғжү§иЎҢ пјҲRCEпјү жјҸжҙһгҖӮжӯӨжјҸжҙһзҡ„дёҘйҮҚ CVSS иҜ„еҲҶдёә 9.8пјҢе…Ғи®ёж”»еҮ»иҖ…з»•иҝҮе®үе…ЁжҺӘж–Ҫе№¶еңЁеҸ—еҪұе“Қзҡ„зі»з»ҹдёҠжү§иЎҢжҒ¶ж„Ҹд»Јз ҒгҖӮжҰӮеҝөйӘҢиҜҒжјҸжҙһзҡ„еҸҜз”ЁжҖ§еҠ еү§дәҶиҝҷз§Қжғ…еҶөпјҢдҪҝеЁҒиғҒиЎҢдёәиҖ…жӣҙе®№жҳ“еҸ‘иө·ж”»еҮ»гҖӮ

**Microsoft е’Ң Oracle дә§е“Ғд№ҹжҲҗдёәзӣ®ж Ү**

Microsoft дә§е“Ғд№ҹеӨ„дәҺзӣ®ж Үд№ӢдёӯгҖӮCVE-2020-0618 жҳҜ Microsoft SQL Server Reporting Services дёӯзҡ„ RCE жјҸжҙһпјҢCVE-2019-1069 жҳҜ Microsoft Windows Task Scheduler дёӯзҡ„жқғйҷҗжҸҗеҚҮзјәйҷ·пјҢйғҪжӯЈеңЁиў«з§ҜжһҒеҲ©з”ЁгҖӮ

Oracle з”ЁжҲ·д№ҹдёҚиғҪе№ёе…ҚгҖӮCVE-2022-21445 е’Ң CVE-2020-14644 иҝҷдёӨдёӘдёҘйҮҚ RCE жјҸжҙһеҲҶеҲ«еҪұе“Қ Oracle JDeveloper е’Ң Oracle WebLogic Server пјҢд№ҹе·Іж·»еҠ еҲ° KEV зӣ®еҪ•дёӯгҖӮ

**дҝ®иЎҘзҡ„зҙ§иҝ«жҖ§**

з§ҜжһҒеҲ©з”ЁиҝҷдәӣжјҸжҙһеҮёжҳҫдәҶеҸҠж—¶дҝ®иЎҘзҡ„жһҒз«ҜйҮҚиҰҒжҖ§гҖӮCISA е·ІиҰҒжұӮиҒ”йӮҰжңәжһ„еңЁ 2024 е№ҙ 10 жңҲ 9 ж—Ҙд№ӢеүҚи§ЈеҶіиҝҷдәӣзјәйҷ·гҖӮдҪҶжҳҜпјҢејәзғҲе»әи®®жүҖжңүз»„з»Үе°Ҫеҝ«еә”з”Ёеҝ…иҰҒзҡ„жӣҙж–°пјҢд»ҘдҝқжҠӨе…¶зі»з»ҹе’Ңж•°жҚ®е…ҚеҸ—жҚҹе®ігҖӮ
