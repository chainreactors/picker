[早前意大利隐私监管机构以涉嫌侵犯隐私为由，封禁流行的人工智能对话应用 ChatGPT](https://www.landiannews.com/archives/98094.html)。该监管机构称禁令是临时性的，直到 OPENAI 满足监管机构要求以及符合欧盟通用数据保护条例（GDPR）为止。

大利隐私监管机构认为 OPENAI 没有遵守欧盟通用数据保护条例 (GDPR) 处理意大利用户的数据，例如监管机构认为 OPENAI 缺乏法律依据来证明大量收集和存储个人数据以训练 ChatGPT 的算法 (即没有证明这是必要的)，同时监管机构还认为 OPENAI 不准确的处理用户数据。

还有个关键问题可能会引起欧盟其他成员国的共同关注：ChatGPT 注册账号时不需要验证年龄，即未成年人也可以注册使用，而 ChatGPT 提供内容没有经过分级处理，可能会向未成年人提供不适合的内容。

[![OPENAI封锁意大利全境IP地址应监管机构要求阻止用户继续访问ChatGPT](https://img.lancdn.com/landian/2023/03/98055.png)](https://img.lancdn.com/landian/2023/03/98055.png)

基于以上情况，目前 OPENAI 已经通过 CloudFlare 的访问限制功能识别并阻止所有意大利 IP 地址访问 ChatGPT。

当属于意大利的 IP 地址访问 ChatGPT 时，OPENAI 会弹出提示：应监管机构要求禁止意大利用户访问。如果是已经开通 ChatGPT Plus 的用户，则 OPENAI 会自动进行退款，同时用户开通的订阅计划会被自动暂停，并不会自动取消订阅，直到意大利监管机构认为 ChatGPT 可以使用后，OPENAI 会恢复用户的订阅。

另外理论上说意大利用户是可以通过 VPN 绕过 OPENAI 限制的，但 OPENAI 设置了风控规则：如果用户账户最初是通过意大利 IP 地址注册的，则使用 VPN 也可能会被限制，也就是说意大利用户可能得首先使用 VPN 换成其他受支持区域的 IP 地址，然后重新注册一个账户。

OPENAI 在公开声明中表示：我们致力于保护人们的隐私，我们相信我们提供的 ChatGPT 符合 GDPR 和其他隐私法的要求。我们将与监管机构合作，尽快恢复意大利用户的访问权限。
