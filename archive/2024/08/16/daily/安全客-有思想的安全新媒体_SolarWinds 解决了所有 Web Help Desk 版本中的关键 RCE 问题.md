## SolarWinds и§ЈеҶідәҶе…¶з”ЁдәҺе®ўжҲ·ж”ҜжҢҒзҡ„ Web Help Desk и§ЈеҶіж–№жЎҲдёӯзҡ„дёҖдёӘдёҘйҮҚиҝңзЁӢд»Јз Ғжү§иЎҢжјҸжҙһгҖӮ

SolarWinds дҝ®еӨҚдәҶ SolarWinds зҡ„ Web Help Desk е®ўжҲ·ж”ҜжҢҒи§ЈеҶіж–№жЎҲдёӯзҡ„дёҖдёӘдёҘйҮҚжјҸжҙһпјҢиў«и·ҹиёӘдёә CVE-2024-28986пјҲCVSS иҜ„еҲҶ 9.8пјүгҖӮиҜҘжјҸжҙһжҳҜдёҖдёӘ Java еҸҚеәҸеҲ—еҢ–й—®йўҳпјҢж”»еҮ»иҖ…еҸҜеҲ©з”ЁиҜҘй—®йўҳеңЁжҳ“еҸ—ж”»еҮ»зҡ„дё»жңәдёҠиҝҗиЎҢе‘Ҫд»ӨпјҢд»ҺиҖҢеҜјиҮҙиҝңзЁӢд»Јз Ғжү§иЎҢгҖӮ

SolarWinds е°Ҷ WHD жҸҸиҝ°дёәдёҖз§Қз»ҸжөҺе®һжғ зҡ„её®еҠ©еҸ°зҘЁеҠЎе’Ңиө„дә§з®ЎзҗҶиҪҜд»¶пјҢиў«еӨ§еһӢдјҒдёҡе’Ңж”ҝеәңз»„з»Үе№ҝжіӣдҪҝз”ЁгҖӮ

*вҖңеҸ‘зҺ° SolarWinds Web Help Desk е®№жҳ“еҸ—еҲ° Java еҸҚеәҸеҲ—еҢ–иҝңзЁӢд»Јз Ғжү§иЎҢжјҸжҙһзҡ„еҪұе“ҚпјҢеҰӮжһңеҲ©з”ЁиҜҘжјҸжҙһпјҢж”»еҮ»иҖ…е°Ҷе…Ғи®ёж”»еҮ»иҖ…еңЁдё»жңәдёҠиҝҗиЎҢе‘Ҫд»ӨгҖӮиҷҪз„¶е®ғиў«жҠҘе‘ҠдёәжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„жјҸжҙһпјҢдҪҶз»ҸиҝҮеҪ»еә•жөӢиҜ•еҗҺпјҢSolarWinds ж— жі•еңЁжІЎжңүиә«д»ҪйӘҢиҜҒзҡ„жғ…еҶөдёӢйҮҚзҺ°е®ғгҖӮвҖңдҪҶжҳҜпјҢеҮәдәҺи°Ёж…ҺиҖғиҷ‘пјҢжҲ‘д»¬е»әи®®жүҖжңү Web Help Desk е®ўжҲ·йғҪеә”з”ЁиҜҘиЎҘдёҒпјҢиҜҘиЎҘдёҒзҺ°е·ІжҺЁеҮәгҖӮвҖқ*

жјҸжҙһ CVE-2024-28986 дјҡеҪұе“ҚжүҖжңү Web Help Desk зүҲжң¬гҖӮиҜҘиҪҜд»¶е…¬еҸёж•Ұдҝғе®ўжҲ·еҚҮзә§еҲ° WHD 12.8.3 Web Help Desk пјҲWHDпјү зҡ„жүҖжңүзүҲжң¬пјҢз„¶еҗҺе®үиЈ…дҝ®иЎҘзЁӢеәҸгҖӮ

иҜҘжјҸжҙһжҳҜз”ұиҜҘе…¬еҸёе®үе…Ёе…¬еҸёзҡ„з ”з©¶дәәе‘ҳеҸ‘зҺ°зҡ„гҖӮиҜҘе…¬еҸёиҝҳж„ҹи°ўInmarsatж”ҝеәң/Viasatзҡ„еҚҸеҠ©гҖӮ
