## 微软考虑广告和订阅支持的廉价 PC

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[WinterIsComing](/~WinterIsComing) (31822)发表于 2022年11月04日 18时13分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73269&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E8%80%83%E8%99%91%E5%B9%BF%E5%91%8A%E5%92%8C%E8%AE%A2%E9%98%85%E6%94%AF%E6%8C%81%E7%9A%84%E5%BB%89%E4%BB%B7%20PC "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自帽子里的天空**

根据微软最近发布的招聘广告，软件巨人正在为新成立的 Windows Incubation 团队招募工程师，该团队的一项任务是在云端优先的世界为 Windows 构建新方向。而微软考虑的一个方向是打造订阅和广告支持的连接云端的廉价 PC，将深度整合 Windows 365。根据微软最近公布的季度财报，它的 Windows OEM 收入同比下降了 15%，但商业产品和云服务收入增加了 8%，设想 Windows 着眼于基于云端的未来在情理之中。
https://www.theregister.com/2022/11/02/microsoft\_windows\_pc\_ads/
