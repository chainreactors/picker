##### иҜ‘ж–ҮеЈ°жҳҺ

жң¬ж–ҮжҳҜзҝ»иҜ‘ж–Үз«

иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

е®үе…Ёе…¬еҸёProofpoint еҸ‘зҺ° TA577 з»„з»Үж”№еҸҳдәҶж”»еҮ»зӯ–з•ҘгҖӮй»‘е®ўзҺ°еңЁдҪҝз”ЁзҪ‘з»ңй’“йұјз”өеӯҗйӮ®д»¶зӘғеҸ– NT LAN Manager ( NTLM ) иә«д»ҪйӘҢиҜҒе“ҲеёҢеҖјпјҢд»ҺиҖҢе®һзҺ°еёҗжҲ·жҺҘз®ЎгҖӮ

е…·дҪ“жқҘиҜҙпјҢеңЁ 2024 е№ҙ 2 жңҲ 26 ж—ҘиҮі 27 ж—Ҙзҡ„дёӨжіўж”»еҮ»дёӯпјҢиҜҘз»„з»Үеҗ‘е…Ёзҗғж•°зҷҫдёӘз»„з»ҮеҸ‘йҖҒдәҶж•°еҚғе°Ғз”өеӯҗйӮ®д»¶пјҢзӣ®зҡ„жҳҜзӘғеҸ–е‘ҳе·Ҙ NTLM е“ҲеёҢеҖјгҖӮ

NTLM е“ҲеёҢеңЁ Windows дёӯз”ЁдәҺиә«д»ҪйӘҢиҜҒе’ҢдјҡиҜқе®үе…ЁгҖӮж”»еҮ»иҖ…еҸҜд»ҘдҪҝз”ЁзӘғеҸ–зҡ„е“ҲеёҢеҖјжқҘз ҙи§ЈеҜҶз ҒжҲ–иҝӣиЎҢвҖңе“ҲеёҢдј йҖ’вҖқж”»еҮ»пјҢд»ҺиҖҢж— йңҖи§ЈеҜҶеҜҶз ҒеҚіеҸҜеҜ№иҝңзЁӢжңҚеҠЎеҷЁиҝӣиЎҢиә«д»ҪйӘҢиҜҒгҖӮ

ж”»еҮ»ж–№жі•ж¶үеҸҠеҸ‘йҖҒзңӢдјјеҜ№еҸ—е®іиҖ…д№ӢеүҚзҡ„ж¶ҲжҒҜиҝӣиЎҢеӣһеӨҚзҡ„зҪ‘з»ңй’“йұјз”өеӯҗйӮ®д»¶пјҢиҝҷз§ҚжҠҖжңҜз§°дёәзәҝзЁӢеҠ«жҢҒгҖӮз”өеӯҗйӮ®д»¶дёӯйҷ„жңүе”ҜдёҖзҡ„пјҲеҜ№дәҺжҜҸдёӘеҸ—е®іиҖ…пјүZIP еӯҳжЎЈпјҢе…¶дёӯеҢ…еҗ« HTML ж–Үд»¶пјҢиҝҷдәӣж–Үд»¶дҪҝз”Ё META еҲ·ж–°е…ғж Үи®°жқҘи§ҰеҸ‘дёҺSMBжңҚеҠЎеҷЁдёҠзҡ„ж–Үжң¬ж–Үд»¶зҡ„иҮӘеҠЁиҝһжҺҘгҖӮеҪ“ Windows и®ҫеӨҮе°қиҜ•иҝһжҺҘеҲ°жңҚеҠЎеҷЁж—¶пјҢжӯӨжҠҖжңҜе…Ғи®ёзҪ‘з»ңзҠҜзҪӘеҲҶеӯҗзӘғеҸ– NTLM е“ҲеёҢеҖјгҖӮ

Proofpoint ејәи°ғпјҢж”»еҮ»иҖ…дј йҖ’жҒ¶ж„ҸжЎЈжЎҲд»Ҙз»•иҝҮ 2023 е№ҙ 7 жңҲеҗҺжӣҙж–°зҡ„з”өеӯҗйӮ®д»¶е®ўжҲ·з«Ҝзҡ„дҝқжҠӨгҖӮеҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢж”»еҮ»зҡ„зӣ®ж ҮжӯЈжҳҜжҚ•иҺ· NTLM е“ҲеёҢеҖјпјҢиҖҢдёҚжҳҜдј ж’ӯжҒ¶ж„ҸиҪҜд»¶гҖӮ

дё“е®¶жҢҮеҮәпјҢдёәдәҶеңЁзҪ‘з»ңж”»еҮ»дёӯдҪҝз”ЁзӘғеҸ–зҡ„е“ҲеёҢеҖјпјҢеҝ…йЎ»зҰҒз”ЁеёҗжҲ·зҡ„еӨҡйҮҚиә«д»ҪйӘҢиҜҒ (MFA)гҖӮиҝҳжңүдәәе»әи®®пјҢзӘғеҸ–е“ҲеёҢеҖјеҸҜд»ҘдҪңдёәдёҖз§Қжғ…жҠҘеҪўејҸжқҘиҜҶеҲ«жңүд»·еҖјзҡ„еҸ—е®іиҖ…гҖӮ

е»әи®®зҡ„е®үе…ЁжҺӘж–ҪеҢ…жӢ¬й…ҚзҪ®з”өеӯҗйӮ®д»¶иҝҮж»Өд»Ҙйҳ»жӯўеҢ…еҗ«еӯҳжЎЈ HTML ж–Үд»¶зҡ„йӮ®д»¶пјҢд»ҘеҸҠй…ҚзҪ®йҳІзҒ«еўҷд»Ҙйҳ»жӯўдј еҮә SMB иҝһжҺҘпјҲйҖҡеёёжҳҜз«ҜеҸЈ 445 е’Ң 139пјүгҖӮеҜ№дәҺ Windows 11 з”ЁжҲ·пјҢеҫ®иҪҜеј•е…ҘдәҶдёҖйЎ№йўқеӨ–зҡ„е®үе…ЁеҠҹиғҪпјҢд»ҘйҳІжӯўйҖҡиҝҮ SMB иҝӣиЎҢеҹәдәҺ NTLM зҡ„ж”»еҮ»гҖӮ
