微软和 OpenAI 宣布扩大合作关系，在投资了 10 亿美元之后微软将在未来几年再次向 OpenAI 投资数十亿美元。早些时候有传言称微软有意向 OpenAI 投资百亿美元，最新的声明只是笼统的称为数十亿美元。OpenAI 表示它将会继续使用基于 Azure 的超算训练其模型，而 Azure 将始终是其所有产品和服务的唯一云服务商。OpenAI 称微软的产品和服务将整合其技术，其中包括 GitHub Copilot 和 Microsoft Designer。
