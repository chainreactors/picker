来自网友 @Sha1rholder 的投稿，知名壁纸引擎 Wallpaper Engine 在 Windows 11 开发版和金丝雀版中存在兼容性问题，如果用户升级最新的开发版和金丝雀版则 Wallpaper Engine 会失效并且无法使用。

根据网友测试，当升级到 [Windows 11 Dev Build 23511.1000](https://www.landiannews.com/archives/99638.html) 版和 [Windows 11 Can Build 25915.1000](https://www.landiannews.com/archives/99648.html) 版后，Wallpaper Engine 都无法正常工作。

但目前还没有 Windows Insider Beta Channel 和 Windows Insider Release Preview 通道的用户报告该错误，应该是微软改了什么东西暂时还没推送到测试版和发布预览通道。

[![Windows 11开发版和金丝雀版用户请注意：Wallpaper Engine会失效](https://img.lancdn.com/landian/2023/07/99656.png)](https://img.lancdn.com/landian/2023/07/99656.png)

上图原图下载地址：[99656](https://dl.lancdn.com/landian/media/wallpaper/99656)

Wallpaper Engine 开发者也[已经知道此事](https://steamcommunity.com/app/431960/discussions/2/3805029995437166271/#c3805029995438521440)不过无能为力，因为开发者只负责对稳定版频道的保证，开发版和金丝雀版的话，开发者无法直接修复，以避免出现某些意外的问题。

不过蓝点网相信 Wallpaper Engine 可能也已经在检查问题了，但即便是发现问题后应该也不会发新版本修复，毕竟微软改动的东西暂时还没到稳定版。

对于已经升级到新版本的小伙伴，暂时除了回滚到旧版本外别无他法，毕竟不是 Wallpaper Engine 的问题，对 Wallpaper Engine 降级是没用的。

最后需要提醒的是，虽然 Wallpaper Engine 没说这是什么问题导致的，**但蓝点网猜测大概率是和微软近期修改 Windows Shell 有关**，因为多个第三方修改任务栏和资源管理器的软件都出现问题了，Wallpaper Engine 引擎也要使用 Windows Shell，所以大概率也是类似问题。
