е№ёиҝҗзҡ„жҳҜпјҢеҜ№дәҺдҪҝз”Ё VBR зҡ„дјҒдёҡжқҘиҜҙпјҢеңЁеӨ§еӨҡж•°з®ЎзҗҶе‘ҳжңүжңәдјҡе®һж–Ҫ Veeam дёҠе‘ЁжҸҗдҫӣзҡ„дҝ®еӨҚзЁӢеәҸд№ӢеүҚпјҢдёӨе®¶е…¬еҸёйғҪжІЎжңүеҲҶдә« PoC жјҸжҙһеҲ©з”Ёд»Јз ҒжҲ–жңүе…іиҜҘжјҸжҙһзҡ„е…¶д»–иҜҰз»ҶдҝЎжҒҜгҖӮ

### е…ідәҺ CVE-2024-40711

Veeam Backup&ReplicationжҳҜдёҖз§ҚжөҒиЎҢзҡ„дјҒдёҡи§ЈеҶіж–№жЎҲпјҢз”ЁдәҺеӨҮд»ҪгҖҒеӨҚеҲ¶е’ҢжҒўеӨҚиҷҡжӢҹзҺҜеўғгҖҒзү©зҗҶжңәеҷЁе’ҢеҹәдәҺдә‘зҡ„е·ҘдҪңиҙҹиҪҪзҡ„еӨҮд»ҪгҖӮ

еӢ’зҙўиҪҜд»¶з»„з»Үд»ҘеүҚжӣҫеҲ©з”Ё VBR дёӯзҡ„жјҸжҙһпјҲдҫӢеҰӮВ CVE-2023-27532пјүжқҘи®ҝй—®з»„з»Үзҡ„еӨҮд»ҪеҹәзЎҖи®ҫж–ҪгҖӮ

CVE-2024-40711 жҳҜдёҖдёӘдёҚжҳҺжјҸжҙһпјҢеҪұе“Қ VBR зүҲжң¬ 12.1.2.172 е’ҢжүҖжңүж—©жңҹзүҲжң¬гҖӮ

вҖңCVE-2024-40711 еҸҜиғҪе…Ғи®ёж”»еҮ»иҖ…е®Ңе…ЁжҺ§еҲ¶зі»з»ҹгҖҒж“Қзәөж•°жҚ®пјҢе№¶еҸҜиғҪеңЁзҪ‘з»ңеҶ…жЁӘеҗ‘з§»еҠЁпјҢдҪҝе…¶жҲҗдёәеЁҒиғҒиЎҢдёәиҖ…зӣёеҜ№й«ҳд»·еҖјзҡ„зӣ®ж ҮпјҢвҖқCensys з ”з©¶дәәе‘ҳе‘Ёдә”жҢҮеҮәгҖӮ

д»–д»¬иҝҳжҢҮеҮәдәҶдә’иҒ”зҪ‘дёҠжҡҙйңІзҡ„2,833дёӘйқўеҗ‘дә’иҒ”зҪ‘зҡ„VeeamеӨҮд»Ҫе’ҢеӨҚеҲ¶жңҚеҠЎеҷЁпјҢдё»иҰҒйӣҶдёӯеңЁеҫ·еӣҪе’Ңжі•еӣҪ вҖ“ дҪҶе…¶дёӯжңүеӨҡе°‘е®№жҳ“еҸ—еҲ°ж”»еҮ»е°ұдёҚеҫ—иҖҢзҹҘдәҶгҖӮ

### дҝ®еӨҚеҸҜз”Ё

Veeamе·Із»ҸеңЁе‘Ёеӣӣдҝ®еӨҚдәҶCVE-2024-40711е’Ңе…¶д»–дә”дёӘеҪұе“ҚVBRзҡ„дёҚеӨӘдёҘйҮҚзҡ„зјәйҷ·пјҢе№¶ж•Ұдҝғз®ЎзҗҶе‘ҳжӣҙж–°еҲ°Veeam Backup & Replication 12.2пјҲжһ„е»ә12.2.0.334пјүгҖӮиҜҘе…¬еҸёжІЎжңүжҸҗеҸҠе…¶дёӯд»»дҪ•дёҖз§ҚеҸҜиғҪзҡ„и§ЈеҶіж–№жі•гҖӮ

еҗҢж—¶пјҢиҜҘе…¬еҸёиҝҳеҸ‘еёғдәҶй’ҲеҜ№д»ҘдёӢеҗ„з§ҚжјҸжҙһзҡ„дҝ®еӨҚзЁӢеәҸпјҡ

* йҖӮз”ЁдәҺ Linux зҡ„ Veeam Agent
* Veeam ONEпјҲйқўеҗ‘ IT е·ҘдҪңиҙҹиҪҪзҡ„зӣ‘жҺ§е’ҢеҲҶжһҗи§ЈеҶіж–№жЎҲпјү
* Veeam Backup for Nutanix AHV е’Ң Veeam Backup for Oracle Linux Virtualization Manager е’Ң Red Hat иҷҡжӢҹеҢ–пјҢд»ҘеҸҠ
* Veeam жңҚеҠЎжҸҗдҫӣе•ҶжҺ§еҲ¶еҸ°пјҲMSP е’ҢдјҒдёҡдҪҝз”Ёзҡ„з”ЁдәҺз®ЎзҗҶж•°жҚ®еӨҮд»Ҫж“ҚдҪңзҡ„и§ЈеҶіж–№жЎҲпјүгҖӮ
