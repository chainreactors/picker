Windows Insider 最大的骗局就是金丝雀通道的功能更新竟然还不如开发版，比如这段时间很多新功能都是优先推送给开发版，这搞的不少金丝雀版用户非常无奈，因为现在也没法直接降级，除非重装系统。

今天微软向开发版用户推出 Windows 11 Dev Build 23511 版，同时微软也推出了 Windows 11 Can Build 25915 版，但后者仅仅只是同步了开发版之前提供的新功能，所以说总体还是挺无聊的。

[![微软推出Windows 11 Can Build 25915版 同步开发版的部分功能更新](https://img.lancdn.com/landian/public/thumb/insider-pink.png)](https://img.lancdn.com/landian/public/thumb/insider-pink.png)

**以下是微软提供的更新日志：**

1. 同步开发版近期增加的新功能，包括现代资源管理器主页、地址栏和详细窗格，也就是那个新版图库。

2. 同步开发版近期改进的无密码登录体验、同步任务栏永不合并选项。

3. 新版 Outlook 变成预装应用，即 WebView 2 版的，之前的 UWP 版邮件和日历暂时保留。

4. 改变刷新率逻辑，如果连接多个显示器，可以在不同的显示器上设置不同的刷新率。

5. 当选择动态刷新率并启用省电模式后，Windows 将保持较低的刷新率，并且在禁用省电模式后也不会自动切换回来。

6. 解决此前版本与华硕主板的兼容性问题，现在华硕主板用户也能更新到这个版本了。

6. 解决此前版本中以太网连接可能丢失的问题。
