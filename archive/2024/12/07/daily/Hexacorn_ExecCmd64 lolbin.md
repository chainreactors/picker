# ExecCmd64 lolbin

Posted on [2024-12-06](https://www.hexacorn.com/blog/2024/12/06/execcmd64-lolbin/ "11:01 pm")  by  [adam](https://www.hexacorn.com/blog/author/adam/ "View all posts by adam")

If you have [ASRock Polychrome RGB](https://www.asrock.com/MB/AMD/X870%20Pro%20RS/index.asp#Download) installed on your system you may find this interesting executable in one of its subfolders:

```
c:\Program Files (x86)\ASRock Utility\ASRRGBLED\HAL\ExecCmd64.exe
```

It is signed:

```
Verified: Signed
Signing date: 2:11 AM 3/16/2023
Publisher: ASROCK INC.
Company: n/a
Description: n/a
Product: n/a
Prod version: n/a
File version: n/a
MachineType: 64-bit
```

And… it accepts command line arguments so can act as a nice lolbin proxy:
