# 美国最大报业集团遭遇网络攻击

#

近日，美国最大报业集团之一Lee Enterprises遭遇网络攻击事件，导致部分报纸印刷及配送服务中断。据称，该集团表示，此次攻击造成其网络系统关闭，但尚未发现重大实质性影响。该集团表示，正与执法部门合作调查事件影响并对系统进行修复。目前，其部分订阅服务及电子版报纸访问仍受限。

# Cisco否认近期遭遇勒索攻击

#

近日，国际网络设备巨头思科公司否认遭遇了Kraken勒索软件组织新的入侵。Kraken组织此前在暗网发布了其窃取的思科内部敏感数据，其中包括用户名及其关联域等信息。据称，思科已针对此事发表官方声明。思科声明称，此次事件所泄露的数据来源于已披露的2022年5月发生的安全事件，且当时事件并未对思科客户造成影响。

# 国内一物联网公司1.17TB数据被泄露

#

近日，网络安全研究人员披露了一个大规模物联网数据泄露事件。研究人员发现一个未经保护的大型数据库，其中包含1.17TB的敏感数据，涉及全球销售的IoT设备的日志、监控记录和错误报告等信息。据称，调查发现，这些数据与我国植物生长灯制造商Mars Hydro及其关联公司LG-LED SOLUTIONS LIMITED有关。该数据库在相关公司接收到通知后的数小时内被限制访问。Mars Hydro公司现已确认数据库所指向的应用程序是其产品。目前，该数据库的所有权与暴露时间等问题尚不明确。

相关链接：

https://cybersecuritynews.com/massive-iot-data-breach/

# 抖音集团因代码抄袭被判赔8266万元

#

#

近期，字节跳动公司因旗下产品代码侵权，被判决赔偿北京美摄公司经济损失及合理支出共计8266.8万元。2021年5月，北京美摄公司就字节跳动旗下抖音、剪映、轻颜相机等8款产品抄袭其音视频编辑软件代码事件向法院提起侵害技术秘密诉讼。经北京知识产权法院及北京市高院一审判决，认定侵权行为并判赔2670.4万元，双方均上诉至最高人民法院。北京美摄公司表示，最高人民法院近日作出终审判决，维持一审判决侵权认定的同时，对一审判决损害赔偿额改判为8266.8万元。

# HPE遭遇黑客入侵，部分员工信息泄露

#

慧与（HPE）公司是一家全球领先的企业IT解决方案和服务提供商。2023年，该公司的Office 365电子邮箱遭遇黑客入侵，部分员工信息被泄露。HPE公司表示，该事件仅对少部分员工造成影响，且黑客窃取的数据仅限于这些员工的邮箱。2025年1月29日起，HPE开始通知受该事件影响人员。
