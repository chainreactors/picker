## 勒索软件组织通过 PuTTy 和 WinSCP 恶意广告试图感染 Windows 管理员

[![安全](https://icon.solidot.org/images/topics/topicsecurity.png?123)](/search?tid=100 "安全")

[Wilson](/~Wilson) (42865)发表于 2024年05月20日 22时02分 星期一 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=78218&appkey=1370085986&title=%E5%8B%92%E7%B4%A2%E8%BD%AF%E4%BB%B6%E7%BB%84%E7%BB%87%E9%80%9A%E8%BF%87%20PuTTy%20%E5%92%8C%20WinSCP%20%E6%81%B6%E6%84%8F%E5%B9%BF%E5%91%8A%E8%AF%95%E5%9B%BE%E6%84%9F%E6%9F%93%20Windows%20%E7%AE%A1%E7%90%86%E5%91%98 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自光环：新斯巴达**

安全公司 Rapid7 报告，勒索软件组织通过搜索引擎展示假的 Putty 和 WinSCP 下载网站试图感染 Windows 管理员的电脑。Putty 是流行的 SSH 客户端，而 WinSCP 是 SFTP 和 FTP 客户端，它们都是系统管理员们常用的工具，而管理员通常是高价值的攻击目标。Putty 和 WinSCP 的恶意广告冒充了 WinSCP (winscp.net)的合法网站，以及拼错域名如 puutty.org, puutty[.]org, wnscp[.]net 和 vvinscp[.]net，Putty 的官网是 https://www.chiark.greenend.org.uk/~sgtatham/putty/。
https://www.bleepingcomputer.com/news/security/ransomware-gang-targets-windows-admins-via-putty-winscp-malvertising/
