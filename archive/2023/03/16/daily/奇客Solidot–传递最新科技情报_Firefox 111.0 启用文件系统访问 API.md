## Firefox 111.0 启用文件系统访问 API

[![Firefox](https://icon.solidot.org/images/topics/topicfirefox.png?123)](/search?tid=39 "Firefox")

[Wilson](/~Wilson) (42865)发表于 2023年03月15日 20时36分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74402&appkey=1370085986&title=Firefox%20111.0%20%E5%90%AF%E7%94%A8%E6%96%87%E4%BB%B6%E7%B3%BB%E7%BB%9F%E8%AE%BF%E9%97%AE%20API "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自通灵师**

Mozilla 释出了 Firefox 111.0。主要新特性包括：启用 Windows 原生通知，生成随机电邮地址和电话号码的 Firefox Relay 服务的用户可选择直接从凭证管理器创建假电邮；启用文件系统访问 API，等等。文件系统访问 API 允许操作用户本地设备上的文件，支持读写和保存，调用该 API 将会要求得到用户的明确许可，并且在标签关闭之后需要重新获得授权。
https://www.mozilla.org/en-US/firefox/111.0/releasenotes/
https://developer.mozilla.org/en-US/docs/Web/API/File\_System\_Access\_API
