数量创KCon历史新高

以下**20**大议题

将在KCon 2023的舞台上与大家见面

为大家开启一场网安技术的盛宴！

### **演讲议题现在揭晓！**

\*以下排名不分先后，最终演讲顺序以KCon官方议程通知为准

1、WebLogic漏洞挖掘思路  丁立洁（thiscodecc）

2、Attacking Debug Components in the Android Ecosystem  曲乐炜

3、Modern Linux sandboxing technology  李强、杨玉彪

4、Java表达式攻防下的黑魔法  刘镇东（yzddMr6）

5、梅塞德斯·奔驰汽车车机安全研究  王启泽

6、从 0 到 1 打造云原生容器沙箱 —— vArmor  韦伟

7、Tai-e: 基于程序分析的安全漏洞检测框架  谭添

8、Recent Progresses in Transfer-based Attack for Image Recognition  王晓森

9、AI对于代码审计和漏洞挖掘的作用  俍梁

10、Magic in Java Api  yemoli、R1ckyZ

11、CNC APT Group Operation Herbminister Revealed - 揭秘CNC APT组织Herbminister行动技战术武器  NaN、k

12、SxS also stands for Specter by Side  张云海

13、闭源安卓系统下的漏洞自动化发现之旅  Hearmen

14、内窥镜: 反混淆虚拟化加固的安卓程序  余帘

15、vSphere攻防技法分享  彭峙酿

16、CI/CD攻击场景  mx7krshell

17、从移动终端到驾驶舱：Fuzzing-WIFI&蓝牙远程协议  李立东

18、低功耗蓝牙黑客：从数字逃逸到现实  肖临风、杨大林

19、近源攻击的安全研究思路分享  马良

20、移动人脸识别安全攻防对抗之旅  谭桂涛

数字化浪潮汹涌而至，万物加速互联，AI技术亦引起新的爆点。网络新技术不断更新的同时，风险与隐患也在暗处滋生。置身于数智化的风口浪尖之上，如何迎接蜕变、破浪前行？

KCon始终关注网安前沿技术，期待以技术初心凝聚行业之力，为护航数字时代安全发展贡献绵薄之力。

本届KCon的20大演讲议题，涵盖了**AI、APT、网络攻防、车联网、IoT、移动安全、漏洞挖掘、云安全**等众多热点话题。演讲人既有来自大型国企、互联网企业、网络安全企业的一线安全研究员，也有来自高校的老师和学生，还有多位独立安全研究员。

相信他们带来的不同视角、不同行业、不同领域的议题定能为大家奉上一场安全技术盛宴，让大家乘兴而来，满载知识而归！

遗憾的是，短短2天的会议议程无法承载所收到的全部议题，因此对每一个优秀议题，我们都本着公平、公正、公开的原则，邀请了公众参与评分，并由专家顾问团严格把关评选。

对于遗憾落选的投稿演讲者，在此报以诚挚的歉意！**会务组将为您奉上投稿福利，并真诚邀请您参与本届大会**，在现场与同样痴迷于安全攻防技术的研究者们交流互动！
