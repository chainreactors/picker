其实这个问题，我早也知道，主要是主题自带了一个 lightbox，导致我如果在安装一个，就会重复弹窗，而至于重复弹窗的问题，这个东西和 envira gallery 带的 lightbox 也冲突，会出来两个弹窗。之前的解决方案一直是禁用相册的 lightbox，使用主题的功能。然而，最近发现一个情况就是手机上不管怎么改都不好使，还是重复弹窗。

只好下定决心，禁用主题的这个 lightbox，直接修改 functions.php， hook 掉相关功能：

```
// 禁用lightbox功能
function disable_olsen_lightbox() {
    // 移除lightbox相关的过滤器
    remove_filter( 'the_content', 'olsen_lightbox_rel', 12 );
    remove_filter( 'get_comment_text', 'olsen_lightbox_rel' );
    remove_filter( 'wp_get_attachment_link', 'olsen_lightbox_rel' );

    // 移除lightbox脚本和样式
    wp_dequeue_script( 'simple-lightbox' );
    wp_dequeue_style( 'simple-lightbox' );
}
add_action( 'wp_enqueue_scripts', 'disable_olsen_lightbox', 20 );
```
