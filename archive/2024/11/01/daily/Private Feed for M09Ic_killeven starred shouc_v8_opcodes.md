# V8 opcodes

This repo contains V8 opcodes and their information for NodeJS 6.0.0 - 17.4.0 (latest).

## Example

`LdaZero` opcodes and its argument information, etc.

```
{
    "LdaZero": {
        "args_count": 0,
        "args": [],
        "opcode": 0,
        "opcode_hex": "0x0"
    },
    ...
}
```
