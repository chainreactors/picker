# Free AugmentCode

[English](#english) | [中文](#chinese)

# 中文版

Free AugmentCode 是一个用于清理AugmentCode相关数据的工具，可以在同一台电脑上无限次登录不同的账号，避免账号被锁定。

## 功能特性

* 📝 修改Telemetry ID

  + 重置设备 ID 和机器 ID
  + 自动备份原始数据
  + 生成新的随机 ID
* 🗃️ 数据库清理

  + 清理 SQLite 数据库中的特定记录
  + 自动备份数据库文件
  + 删除包含 'augment' 关键字的记录
* 💾 工作区存储管理

  + 清理工作区存储文件
  + 自动备份工作区数据
