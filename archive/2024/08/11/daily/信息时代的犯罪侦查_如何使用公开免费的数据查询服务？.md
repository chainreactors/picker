**编者按**

笔者进行了部分验证测试可用。数据可用就行了，不一定要为“我”所有，这两年很多“专家”都在讨论“数据可用不可见”，事实上，实现起来有很多种方式（小编指的绝非"数据大屏"），并非绝对的悲观。

**WebService**是非常标准化的第三方服务提供接口。本文不具体介绍如何实现WebService，那是后端服务开发的内容；也不具体介绍在前端程序中如何调用第三方提供的WebService。说实话，“知道有什么很重要”，今天主要给大家提供一些公开的、免费的WebService接口服务。

**一、如何理解WebService？**

* 支付宝并没有银行的数据，它只是去调用银行提供的接口来获得数据。
* 那些天气预报的网站，也是气象局把自己的系统服务以webservice服务的形式暴露出来，让第三方网站和程序可以调用这些服务功能。
* 其他常见的还有基金和股票交易数据，也是证券交易所把自己的系统服务以webservice服务的形式暴露出来，让第三方网站和程序可以调用这些服务功能。
* 机场大屏上显示的航班信息，火车站大屏显示的车次信息，甚至很多城市公交站也能够显示车次到站和距离信息，……，实时性、吞吐性都很顺畅，基本上也都是通过WebService的方式实现的。

因此，笔者认为，了解并能够实现WebService对IT人员而言很重要。

**二、公开免费的WebService都有哪些？**

事实上提供公开、免费的公共数据服务接口很多，只是我们了解的比较少而已。笔者把了解到的情况罗列一下供大家使用。部分做了测试，并非全部持续有效（仅列出了WSDL方式的地址）。

**1、股票行情数据 WEB 服务**（支持香港、深圳、上海基金、债券和股票；支持多股票同时查询）

http://webservice.webxml.com.cn/WebServices/StockInfoWS.asmx?wsdl

**2、中国开放式基金数据 WEB 服务**

http://webservice.webxml.com.cn/WebServices/ChinaOpenFundWS.asmx?wsdl

**3、外汇-人民币即时报价 WEB 服务**

http://webservice.webxml.com.cn/WebServices/ForexRmbRateWebService.asmx?wsdl

**4、即时外汇汇率数据 WEB 服务**

http://webservice.webxml.com.cn/WebServices/ExchangeRateWebService.asmx?wsdl

**5、国内手机号码归属地查询WEB服务**

http://webservice.webxml.com.cn/WebServices/MobileCodeWS.asmx?wsdl

**6、腾讯QQ在线状态 WEB 服务（返回数据（String）Y = 在线；N = 离线 ；E = QQ号码错误......）**

http://webservice.webxml.com.cn/webservices/qqOnlineWebService.asmx?wsdl

**7、Email 电子邮件地址验证 WEB 服务**

http://webservice.webxml.com.cn/WebServices/ValidateEmailWebService.asmx?wsdl

**8、2500多个城市天气预报 WEB服务**（2500多个城市天气预报Web服务，包含2400个以上中国城市和100个以上国外城市天气预报数据。数据每2.5小时左右自动更新一次，准确可靠。）

http://webservice.webxml.com.cn/WebServices/WeatherWS.asmx?wsdl

**9、国内飞机航班时刻表 WEB 服务**（通过出发城市和到达城市查询飞机航班、出发机场、到达机场、出发和到达时间、飞行周期、航空公司、机型等信息。）

http://webservice.webxml.com.cn/webservices/DomesticAirline.asmx?wsdl

**10、火车时刻表 WEB 服务** （第六次提速最新列车时刻表）

http://webservice.webxml.com.cn/WebServices/TrainTimeWebService.asmx?wsdl

**11、IP地址来源搜索 WEB 服务**（IP地址搜索 WEB 服务包含中国和国外已知的IP地址数据，是目前最完整的IP地址数据，记录数量现已超过30万条并还在不断更新和增加中，感谢纯真网络提供IP地址数据来源。）

http://webservice.webxml.com.cn/WebServices/IpAddressSearchWebService.asmx?wsdl

**12、中文简体字<->繁体字转换 WEB 服务** http://webservice.webxml.com.cn/WebServices/TraditionalSimplifiedWebService.asmx?wsdl

**三、有关声明**

1、公开免费的数据主要强调“公开”，一般不用于商业用途。

2、作为专业技术人员，我们要研究如何获得这些服务，并安全使用这些数据。您不要想当然的以为，公开、免费的数据就是一个毫无安全感可言的静态数据库文件或者电子表格文件。

3、我们在享用这些服务的时候，尽可能按需使用，不要做一个令人讨厌的“数据爬虫”。
