# Phase 2

因为没有时间，系统的输出越来越少。明明在大语言模型迅速繁荣的这个时代，保持输出的意义越发巨大。

其实日常整理总结的习惯还在，只是更为零散和片段，也就犹豫是否有必要成文。

前一阵听 CVP 在部门员工例会上强调，最重要的事情，是要经常问自己，是否在做最重要的事情。好消息是，前半句是肯定的而且有着明确的答案。坏消息是，后半句的答案是否定的。

过于一年基础体能稳步提升，全马 PB 刷新了两次，力量指标也小幅增长。作息和饮食规律，免疫系统还没有恢复，对病菌依然敏感。

或许是得益于手指较为细长，每天练琴五分钟，一年下来也能熟练弹奏不少曲子。读谱虽然还不熟练，但总算摆脱了过去用游戏式的 app 来学习旋律的小打小闹。

阅读的题材变窄了一些，几门外语还在日常积累，插画练习画了一些但没有作品，业余学习了一下 3D 建模和作曲。

受限于时间，游戏和动画的时间被大幅压缩，PS Network 会员甚至多年来第一次暂停续费，PS5 和 Xbox Series X 主机也很久没有开机。不需要占用电视机的 Switch 偶尔还会拿起来玩个十分钟十五分钟，但 Xenoblade X DE 通关遥遥无期，至少在下半年的 FFT Remaster 上市前不太可能。

除此之外，更多的时间不得不花费在必须花费的事情上。

看似平稳的日常，正在迅速逼近一个不可持续的节点。逐渐感到如果继续这样下去，很快就会到达某一个极限。如果因为战略上的懒惰而始终做着不是最重要的事情，终有一天会再也无法扭转局面。

有哪些事情，是如果现在不做，将来会后悔的呢。有哪些事情，是必须由我去做，否则就会错过的呢。

Phase 1 已经进入尾声。
