# MoNotificationUxStub.exe lolbin

Posted on [2024-12-27](https://www.hexacorn.com/blog/2024/12/27/monotificationuxstub-exe-lolbin/ "12:16 am")  by  [adam](https://www.hexacorn.com/blog/author/adam/ "View all posts by adam")

When you run *MoNotificationUxStub.exe* on Windows Server 2025, it will try to load the following non-existing library:

```
C:\Windows\UUS\amd64\UMPDC.dll
```

We can place our own, leading to the code execution:

[![](https://www.hexacorn.com/blog/wp-content/uploads/2024/12/MoNotificationUxStub_UMPDC.png)](https://www.hexacorn.com/blog/wp-content/uploads/2024/12/MoNotificationUxStub_UMPDC.png)

Same as in the [previous case](https://www.hexacorn.com/blog/2024/12/27/mlenginestub-exe-lolbin/), TrustedInstaller rights are required to write to the folder.
