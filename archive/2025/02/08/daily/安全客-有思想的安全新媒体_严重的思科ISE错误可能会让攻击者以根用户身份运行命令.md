жҖқз§‘е·ІеҸ‘еёғиЎҘдёҒпјҢдҝ®еӨҚе…¶иә«д»ҪжңҚеҠЎеј•ж“ҺпјҲISEпјүе®үе…Ёзӯ–з•Ҙз®ЎзҗҶе№іеҸ°дёӯзҡ„дёӨдёӘдёҘйҮҚжјҸжҙһгҖӮ

дјҒдёҡз®ЎзҗҶе‘ҳе°ҶжҖқз§‘ ISE з”ЁдҪңиә«д»ҪдёҺи®ҝй—®з®ЎзҗҶпјҲIAMпјүи§ЈеҶіж–№жЎҲпјҢиҜҘж–№жЎҲе°Ҷи®ӨиҜҒгҖҒжҺҲжқғе’Ңи®Ўиҙ№еҠҹиғҪж•ҙеҗҲеңЁдёҖеҸ°и®ҫеӨҮдёӯгҖӮ

иҝҷдёӨдёӘе®үе…ЁжјҸжҙһпјҲCVE вҖ“ 2025 вҖ“ 20124 е’Ң CVE вҖ“ 2025 вҖ“ 20125пјүеҸҜиў«жӢҘжңүеҸӘиҜ»з®ЎзҗҶе‘ҳжқғйҷҗзҡ„е·Іи®ӨиҜҒиҝңзЁӢж”»еҮ»иҖ…еҲ©з”ЁпјҢеңЁжңӘжү“иЎҘдёҒзҡ„и®ҫеӨҮдёҠд»Ҙ root жқғйҷҗжү§иЎҢд»»ж„Ҹе‘Ҫд»Өе№¶з»•иҝҮжҺҲжқғгҖӮ

иҝҷдәӣжјҸжҙһдјҡеҪұе“ҚжҖқз§‘ ISE е’ҢжҖқз§‘ ISE иў«еҠЁиә«д»ҪиҝһжҺҘеҷЁпјҲISE вҖ“ PICпјүи®ҫеӨҮпјҢж— и®әи®ҫеӨҮй…ҚзҪ®еҰӮдҪ•гҖӮ

вҖңжӯӨжјҸжҙһжәҗдәҺеҸ—еҪұе“ҚиҪҜд»¶еҜ№з”ЁжҲ·жҸҗдҫӣзҡ„ Java еӯ—иҠӮжөҒиҝӣиЎҢдёҚе®үе…Ёзҡ„еҸҚеәҸеҲ—еҢ–пјҢвҖқ жҖқз§‘еңЁжҸҸиҝ°дёҘйҮҚзЁӢеәҰиҜ„зә§дёә 9.9пјҲж»ЎеҲҶ 10 еҲҶпјүзҡ„ CVE вҖ“ 2025 вҖ“ 20124 жјҸжҙһж—¶иЎЁзӨәгҖӮ

вҖңж”»еҮ»иҖ…еҸҜйҖҡиҝҮеҗ‘еҸ—еҪұе“Қзҡ„ API еҸ‘йҖҒзІҫеҝғжһ„йҖ зҡ„еәҸеҲ—еҢ– Java еҜ№иұЎжқҘеҲ©з”ЁжӯӨжјҸжҙһгҖӮжҲҗеҠҹеҲ©з”ЁиҜҘжјҸжҙһеҸҜдҪҝж”»еҮ»иҖ…еңЁи®ҫеӨҮдёҠжү§иЎҢд»»ж„Ҹе‘Ҫд»Өе№¶жҸҗеҚҮжқғйҷҗгҖӮвҖқ

CVE вҖ“ 2025 вҖ“ 20125 жҳҜз”ұзү№е®ҡ API дёӯзјәд№ҸжҺҲжқғд»ҘеҸҠеҜ№з”ЁжҲ·жҸҗдҫӣзҡ„ж•°жҚ®йӘҢиҜҒдёҚеҪ“еҜјиҮҙзҡ„пјҢж”»еҮ»иҖ…еҸҜеҲ©з”ЁжҒ¶ж„Ҹжһ„йҖ зҡ„ HTTP иҜ·жұӮжқҘиҺ·еҸ–дҝЎжҒҜгҖҒдҝ®ж”№жҳ“еҸ—ж”»еҮ»зі»з»ҹзҡ„й…ҚзҪ®е№¶йҮҚеҗҜи®ҫеӨҮгҖӮ

е»әи®®з®ЎзҗҶе‘ҳе°Ҫеҝ«е°ҶжҖқз§‘ ISE и®ҫеӨҮиҝҒз§»жҲ–еҚҮзә§еҲ°дёӢиЎЁдёӯеҲ—еҮәзҡ„дҝ®еӨҚзүҲжң¬д№ӢдёҖгҖӮ

жҖқз§‘дә§е“Ғе®үе…ЁдәӢд»¶е“Қеә”еӣўйҳҹпјҲPSIRTпјүе°ҡжңӘеҸ‘зҺ°е…¬ејҖеҸҜз”Ёзҡ„жјҸжҙһеҲ©з”Ёд»Јз Ғзҡ„иҜҒжҚ®пјҢд№ҹжңӘеҸ‘зҺ°иҝҷдёӨдёӘдёҘйҮҚе®үе…ЁжјҸжҙһпјҲз”ұеҫ·еӢӨе®үе…Ёз ”з©¶дәәе‘ҳдё№гғ»й©¬жһ—е’ҢеЎһе·ҙж–Ҝи’Ӯе®үгғ»жӢүжқңиҺұдәҡжҠҘе‘ҠпјүеңЁж”»еҮ»дёӯиў«ж»Ҙз”Ёзҡ„иҜҒжҚ®гҖӮ

е‘ЁдёүпјҢиҜҘе…¬еҸёиҝҳиӯҰе‘Ҡз§°пјҢе…¶ IOSгҖҒIOS XEгҖҒIOS XRпјҲCVE вҖ“ 2025 вҖ“ 20169гҖҒCVE вҖ“ 2025 вҖ“ 20170гҖҒCVE вҖ“ 2025 вҖ“ 20171пјүе’Ң NX вҖ“ OSпјҲCVE вҖ“ 2024 вҖ“ 20397пјүиҪҜд»¶еӯҳеңЁдёҘйҮҚжјҸжҙһпјҢж”»еҮ»иҖ…еҸҜеҲ©з”ЁиҝҷдәӣжјҸжҙһи§ҰеҸ‘жӢ’з»қжңҚеҠЎпјҲDoSпјүжқЎд»¶жҲ–з»•иҝҮ NX вҖ“ OS й•ңеғҸзӯҫеҗҚйӘҢиҜҒгҖӮ

жҖқз§‘е°ҡжңӘй’ҲеҜ№еҗҜз”Ё SNMP еҠҹиғҪзҡ„ IOSгҖҒIOS XE е’Ң IOS XR иҪҜд»¶дёӯзҡ„ DoS жјҸжҙһеҸ‘еёғиЎҘдёҒгҖӮдёҚиҝҮпјҢиҜҘе…¬еҸёиЎЁзӨәиҝҷдәӣжјҸжҙһе°ҡжңӘеңЁе®һйҷ…зҺҜеўғдёӯиў«еҲ©з”ЁпјҢе№¶жҸҗдҫӣдәҶзј“и§ЈжҺӘж–ҪпјҢиҰҒжұӮз®ЎзҗҶе‘ҳеңЁжҳ“еҸ—ж”»еҮ»зҡ„и®ҫеӨҮдёҠзҰҒз”Ёжҳ“еҸ—ж”»еҮ»зҡ„еҜ№иұЎж ҮиҜҶз¬ҰпјҲOIDsпјүпјҲе°Ҫз®ЎиҝҷеҸҜиғҪдјҡеҜ№зҪ‘з»ңеҠҹиғҪжҲ–жҖ§иғҪдә§з”ҹиҙҹйқўеҪұе“ҚпјүгҖӮ

иҜҘе…¬еҸёи®ЎеҲ’еңЁ 2 жңҲе’Ң 3 жңҲжҺЁеҮәиҪҜд»¶жӣҙж–°пјҢд»Ҙи§ЈеҶі SNMP DoS е®үе…ЁжјҸжҙһгҖӮ

9 жңҲпјҢжҖқз§‘дҝ®еӨҚдәҶиә«д»ҪжңҚеҠЎеј•ж“Һзҡ„еҸҰдёҖдёӘжјҸжҙһпјҲеӯҳеңЁе…¬ејҖзҡ„жјҸжҙһеҲ©з”Ёд»Јз ҒпјүпјҢиҜҘжјҸжҙһеҸҜи®©еЁҒиғҒиЎҢдёәиҖ…еңЁжҳ“еҸ—ж”»еҮ»зҡ„и®ҫеӨҮдёҠе°ҶжқғйҷҗжҸҗеҚҮиҮі rootгҖӮ

дёӨдёӘжңҲеҗҺпјҢжҖқз§‘иҝҳдҝ®еӨҚдәҶдёҖдёӘжһҒе…¶дёҘйҮҚзҡ„жјҸжҙһпјҢиҜҘжјҸжҙһдҪҝж”»еҮ»иҖ…иғҪеӨҹеңЁжҳ“еҸ—ж”»еҮ»зҡ„и¶…еҸҜйқ ж— зәҝеӣһзЁӢпјҲURWBпјүжҺҘе…ҘзӮ№дёҠд»Ҙ root жқғйҷҗиҝҗиЎҢе‘Ҫд»ӨгҖӮ
