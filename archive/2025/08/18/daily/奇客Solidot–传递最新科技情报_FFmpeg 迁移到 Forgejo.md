## FFmpeg 迁移到 Forgejo

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Edwards](/~Edwards) (42866)发表于 2025年08月17日 18时52分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=82068&appkey=1370085986&title=FFmpeg%20%E8%BF%81%E7%A7%BB%E5%88%B0%20Forgejo "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自布兰尼肯夫人**

FFmpeg 的开发迁移到了使用 Forgejo 的自托管平台 code.ffmpeg.org。Forgejo 是 Gitea 的分支，是一个类似 GitHub 的 Git 软件开发和版本控制平台，支持 bug 跟踪、Wiki 和代码审查等功能。迁移到 Forgejo 的一个重要原因是随着越来越多的开发者参与 FFmpeg 项目，现有的基于邮件列表的开发模式已经无法满足需求。短时间内邮件列表仍然继续使用，但随着时间的推移将会逐渐减少使用。
code.ffmpeg.org/FFmpeg/FFmpeg
ffmpeg.org/pipermail/ffmpeg-devel/2025-July/346938.html
