部分 Windows 10 和 Windows 11 用户在安装了最新预览版更新 KB5029351 后遭遇了错误信息为 UNSUPPORTED\_PROCESSOR 的蓝屏死机。微星公司现在证实该问题与其主板相关，它在官方声明中称使用 MSI 600/700 系列主板的用户遭遇了蓝屏死机，如果该补丁没有自动卸载，那么建议用户回滚 BIOS 到上一个版本，然后卸载 KB5029351。微软表示受影响的操作系统版本包括 Windows 10 21H2/22H2 和 Windows 11 22H2，它表示在多次蓝屏死机之后系统应该会自动卸载该补丁。
