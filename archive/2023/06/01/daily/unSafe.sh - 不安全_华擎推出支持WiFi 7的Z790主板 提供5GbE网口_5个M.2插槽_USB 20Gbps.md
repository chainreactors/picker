主板制造商华擎 (ASRock) 日前在 Computex 2023 上展示了一些新主板和显示器，新主板依然按照华擎以往作风，在接口配置上直接堆满，让用户根本不用考虑怎么规划后续的硬件组合。

**第一款主板是 ASRock Phantom Gaming Z790 Nova WiFi 7：**

该主板采用 20+1+1 相供电设计、支持 SPS (智能电源管理)、1 个 PCIe 5.0 SSD 插槽、4 个额外的 PCIe 4.0 M.2 SSD 插槽、4 个 SATA 接口、7 个后置 USB 接口 (含 1 个 USB 3.2 Gen 2x2 即 20Gbps 接口)、9 个前置 USB 接口 (含 1 个 USB 3.2 Gen 2x2 即 20Gbps 接口) 以及一个 5GbE 网口。

WiFi 7 方面，WiFi 7 标准将在今年夏季正式推出，目前已经定稿，所以制造商提前准备起来也没问题。注：WiFi 7 为 802.11be 。

[![华擎推出支持WiFi 7的Z790主板 提供5GbE网口/5个M.2插槽/USB 20Gbps](https://img.lancdn.com/landian/2023/05/98913-1.png)](https://img.lancdn.com/landian/2023/05/98913-1.png)

**第二款主板是 ASRock Z790 Riptide WiFi：**

该主板采用 16+1+1 相供电设计、支持 SPS (智能电源管理)、1 个 PCIe 5.0 SSD 插槽、4 个额外的 PCIe 4.0 M.2 SSD 插槽、8 个 SATA 接口、10 个后置 USB 接口、7 个前置 USB 接口 (含 1 个 USB 3.2 Gen 2x2 即 20Gbps 接口) 以及一个 5GbE 网口，同样支持 WiFi 7。

[![华擎推出支持WiFi 7的Z790主板 提供5GbE网口/5个M.2插槽/USB 20Gbps](https://img.lancdn.com/landian/2023/05/98913-2.png)](https://img.lancdn.com/landian/2023/05/98913-2.png)

**第三款主板是 ASRock Z790 Taichi Lite：**

太极系列为华擎较为便宜但规格配置依然不错的主板，其中太极 Lite 配置为 1 个 PCIe 5.0 SSD 插槽、4 个额外的 PCIe 4.0 M.2 SSD 插槽、8 个 SATA 接口、2 个雷电 4/USB4 接口、9 个后置 USB 接口 + 9 个前置 USB 接口、网口为 2.5GbE 而不是 5GbE 的。另外也不支持 WiFi 7，支持的是 WiFi 6E。

[![华擎推出支持WiFi 7的Z790主板 提供5GbE网口/5个M.2插槽/USB 20Gbps](https://img.lancdn.com/landian/2023/05/98913-3.png)](https://img.lancdn.com/landian/2023/05/98913-3.png)

**第四款是 ASRock B650 Taichi Lite：**

这款是专门为 AMD RYZEN 7000 系列 CPU 设计的主板，采用 24+2+1 相供电设计，支持 SPS、1 个 PCIe 5.0 SSD 插槽、2 个额外的 PCIe 4.0 M.2 SSD 插槽、1 个雷电 4/USB4 接口，11 个后置 USB 接口 + 7 个前置 USB 接口，网口也是 2.5GbE 的，支持 WiFi 6E。

[![华擎推出支持WiFi 7的Z790主板 提供5GbE网口/5个M.2插槽/USB 20Gbps](https://img.lancdn.com/landian/2023/05/98913-4.png)](https://img.lancdn.com/landian/2023/05/98913-4.png)

**还有一款产品是 ASRock PG558KF 55 吋 8K 显示器：**

这款显示器支持 HDR1000、最大亮度为 750 尼特，不过高端游戏党应该看不上这款显示器，因为刷新率只有 60Hz，对比度为 1200:1 也不如 OLED 显示器。

目前以上产品的价格都还不知道，不过华擎销售代表称两款 WiFi 7 的新主板将在 8 月上市，两款太极主板则会在 7 月上市。

[![华擎推出支持WiFi 7的Z790主板 提供5GbE网口/5个M.2插槽/USB 20Gbps](https://img.lancdn.com/landian/2023/05/98913-5.png)](https://img.lancdn.com/landian/2023/05/98913-5.png)
