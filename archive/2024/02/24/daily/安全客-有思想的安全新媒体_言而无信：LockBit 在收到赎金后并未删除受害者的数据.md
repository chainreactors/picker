# 言而无信：LockBit 在收到赎金后并未删除受害者的数据

不要相信网络勒索者的承诺。

作为 国际克罗诺斯行动的一部分 ，英国执法部门 透露 ，LockBit 勒索组织仍在保留受害者的数据，尽管他们收到了删除受害者数据的赎金。

英国国家犯罪局 ( NCA ) 局长格雷厄姆·比格 (Graham Biggar) 称 LockBit 是过去四年来“最活跃、最具危害性的勒索软件组织”，自成立以来，该组织已成功攻击了全球数千个组织。

犯罪分子的行为包括加密受害者网络上的设备和/或窃取数据，然后他们要求赎金以提供 解密密钥 并删除数据。该组织在诉求中声称，将严格履行承诺，因为如果不信守诺言，今后将不会再有人支付赎金。

然而，最近的事实证明，即使受害者支付了赎金，黑客仍然保留了被盗数据的备份副本。目前尚不清楚犯罪分子是否与第三方分享了这些信息，但这些数据保留在他们手中这一事实完全否定了勒索者的所有声明和承诺。

NCA 计划稍后公布在行动中获得的更多信息，包括有关 LockbitSupp 管理员和该组织财务状况的数据。Graham Biggar 在接受记者采访时强调了勒索软件威胁的严重性，并指出 2023 年是攻击次数和勒索资金量创纪录的一年。

LockBit 有可能会试图恢复其犯罪活动，但执法机构打算继续打击该组织，特别是现在他们已经掌握了有关其参与者和行动方法的足够信息。

我们之前报道过，执法人员 清理了 LockBit 基础设施，并完全控制了他们 在洋葱网络上的网站。现在，那里公布的不是受害者名单，而是有关该组织内部结构的各种秘密。
