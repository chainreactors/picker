GitHub 官方博客发表简短声明，宣布从 2023 年 2月 23 日起 GitHub Sponsors 将不再支持 PayPal 作为支付处理器。因此赞助人将无法再通过 PayPal 赞助开发者或组织。GitHub 建议赞助人更新支付方式，使用信用卡或借记卡。GitHub 官方对此没有给出更多解释。PayPal 是最流行的第三方支付服务之一，其优点是保护付款方，避免直接暴露信用卡或借记卡。
