# Memory Cleaner 重制版

原版 Memory Cleaner 还是非常好用的，但是会强制联网更新，经常连不上他的服务器就用不了了。
此版完全按照原版的内部逻辑重写所有功能，我自己还额外加了清理内存等待链（适用于游戏玩家）。
