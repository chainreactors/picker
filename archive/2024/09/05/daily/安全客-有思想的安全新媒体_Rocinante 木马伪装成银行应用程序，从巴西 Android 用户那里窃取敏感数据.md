д№ҹе°ұжҳҜиҜҙпјҢж №жҚ® Silent PushВ жңҖиҝ‘зҡ„еҲҶжһҗпјҢPegasus иў«иҜ„дј°дёәиў«з§°дёә DukeEugene зҡ„еЁҒиғҒиЎҢдёәиҖ…зҡ„дҪңе“ҒпјҢиҜҘеЁҒиғҒиЎҢдёәиҖ…д№ҹд»Ҙзұ»дјјзҡ„жҒ¶ж„ҸиҪҜд»¶иҸҢж ӘиҖҢй—»еҗҚпјҢдҫӢеҰӮ ERMACгҖҒBlackRockгҖҒHook е’Ң LootгҖӮ

ThreatFabric иЎЁзӨәпјҢе®ғзЎ®е®ҡдәҶ Rocinante жҒ¶ж„ҸиҪҜд»¶дёӯзӣҙжҺҘеҸ— ERMAC ж—©жңҹиҝӯд»ЈеҪұе“Қзҡ„йғЁеҲҶпјҢе°Ҫз®ЎжҚ®дҝЎ ERMAC жәҗд»Јз ҒеңЁ 2023 е№ҙзҡ„жі„йңІеҸҜиғҪиө·еҲ°дәҶдёҖе®ҡдҪңз”ЁгҖӮ

вҖңиҝҷжҳҜз¬¬дёҖдёӘеҺҹе§ӢжҒ¶ж„ҸиҪҜд»¶е®¶ж—Ҹд»Һжі„жјҸдёӯиҺ·еҸ–д»Јз Ғе№¶еңЁд»–д»¬зҡ„д»Јз Ғдёӯе®һзҺ°йғЁеҲҶд»Јз Ғзҡ„жғ…еҶөпјҢвҖқе®ғжҢҮеҮәгҖӮвҖңиҝҷдёӨдёӘзүҲжң¬д№ҹжңүеҸҜиғҪжҳҜеҗҢдёҖдёӘеҲқе§ӢйЎ№зӣ®зҡ„еҚ•зӢ¬еҲҶж”ҜгҖӮвҖқ

Rocinante дё»иҰҒйҖҡиҝҮзҪ‘з»ңй’“йұјзҪ‘з«ҷеҲҶеҸ‘пјҢж—ЁеңЁиҜұйӘ—жҜ«ж— жҲ’еҝғзҡ„з”ЁжҲ·е®үиЈ…еҒҮеҶ’зҡ„ dropper еә”з”ЁзЁӢеәҸпјҢиҝҷдәӣеә”з”ЁзЁӢеәҸе®үиЈ…еҗҺпјҢдјҡиҜ·жұӮиҫ…еҠ©еҠҹиғҪжңҚеҠЎжқғйҷҗд»Ҙи®°еҪ•еҸ—ж„ҹжҹ“и®ҫеӨҮдёҠзҡ„жүҖжңүжҙ»еҠЁгҖҒжӢҰжҲӘ SMS ж¶ҲжҒҜе№¶жҸҗдҫӣзҪ‘з»ңй’“йұјзҷ»еҪ•йЎөйқўгҖӮ

е®ғиҝҳдёҺе‘Ҫд»Өе’ҢжҺ§еҲ¶ пјҲC2пјү жңҚеҠЎеҷЁе»әз«ӢиҒ”зі»пјҢд»Ҙзӯүеҫ…иҝӣдёҖжӯҘзҡ„жҢҮд»Ө вҖ“ жЁЎжӢҹи§Ұж‘ёе’Ңж»‘еҠЁдәӢд»¶ вҖ“ д»ҘиҝңзЁӢжү§иЎҢгҖӮж”¶йӣҶзҡ„дёӘдәәдҝЎжҒҜиў«жі„йңІз»ҷ Telegram жңәеҷЁдәәгҖӮ

вҖңиҜҘжңәеҷЁдәәдҪҝз”ЁеҶ’е……зӣ®ж Үй“¶иЎҢзҡ„иҷҡеҒҮзҷ»еҪ•йЎөйқўжҸҗеҸ–иҺ·еҫ—зҡ„жңүз”Ё PIIгҖӮз„¶еҗҺпјҢе®ғдјҡе°ҶиҝҷдәӣдҝЎжҒҜпјҲж јејҸеҢ–пјүеҸ‘еёғеҲ°зҠҜзҪӘеҲҶеӯҗеҸҜд»Ҙи®ҝй—®зҡ„иҒҠеӨ©дёӯпјҢвҖңThreatFabric жҢҮеҮәгҖӮ

вҖңиҜҘдҝЎжҒҜдјҡж №жҚ®з”ЁдәҺиҺ·еҸ–е®ғзҡ„иҷҡеҒҮзҷ»еҪ•йЎөйқўиҖҢз•ҘжңүеҸҳеҢ–пјҢеҢ…жӢ¬и®ҫеӨҮдҝЎжҒҜпјҢдҫӢеҰӮеһӢеҸ·е’Ңз”өиҜқеҸ·з ҒгҖҒCPF еҸ·з ҒгҖҒеҜҶз ҒжҲ–еёҗеҸ·гҖӮвҖқ

иҝҷдёҖеҸ‘еұ•жҳҜеңЁиөӣй—Ёй“Ғе…Ӣејәи°ғеҸҰдёҖдёӘеҲ©з”Ё secureserver зҡ„й“¶иЎҢжңЁй©¬жҒ¶ж„ҸиҪҜд»¶жҙ»еҠЁд№Ӣйҷ…еҸ‘з”ҹзҡ„гҖӮnet domain е®ҡдҪҚиҘҝзҸӯзүҷиҜӯе’Ңи‘Ўиҗ„зүҷиҜӯеҢәеҹҹгҖӮ

вҖңеӨҡйҳ¶ж®өж”»еҮ»д»ҺжҒ¶ж„Ҹ URL ејҖе§ӢпјҢеҜјиҮҙеҢ…еҗ«ж··ж·Ҷ .hta ж–Үд»¶зҡ„еӯҳжЎЈпјҢвҖқиҝҷе®¶ Broadcom жӢҘжңүзҡ„е…¬еҸёиЎЁзӨәгҖӮ

вҖңжӯӨж–Үд»¶дјҡеҜјиҮҙдёҖдёӘ JavaScript иҙҹиҪҪпјҢиҜҘиҙҹиҪҪеңЁдёӢиҪҪжңҖз»Ҳ AutoIT иҙҹиҪҪд№ӢеүҚжү§иЎҢеӨҡдёӘ AntiVM е’Ң AntiAV жЈҖжҹҘгҖӮжӯӨжңүж•ҲиҙҹиҪҪдҪҝз”ЁиҝӣзЁӢжіЁе…ҘеҠ иҪҪпјҢзӣ®зҡ„жҳҜд»ҺеҸ—е®іиҖ…зҡ„зі»з»ҹдёӯзӘғеҸ–й“¶иЎҢдҝЎжҒҜе’ҢеҮӯжҚ®пјҢе№¶е°Ҷе…¶жі„йңІеҲ° C2 жңҚеҠЎеҷЁгҖӮ

е®ғиҝҳзҙ§йҡҸдёҖз§Қж–°зҡ„вҖңжү©еұ•иҪҜд»¶еҚіжңҚеҠЎвҖқзҡ„еҮәзҺ°пјҢиҜҘжңҚеҠЎйҖҡиҝҮж–°зүҲжң¬зҡ„В Genesis Market иҝӣиЎҢе№ҝе‘Ҡй”Җе”®пјҢиҜҘеёӮеңәдәҺ 2023 е№ҙеҲқиў«жү§жі•йғЁй—Ёе…ій—ӯпјҢж—ЁеңЁдҪҝз”ЁеңЁ Chrome Web Store дёҠдј ж’ӯзҡ„жҒ¶ж„Ҹ Web жөҸи§ҲеҷЁжү©еұ•д»ҺжӢүдёҒзҫҺжҙІ пјҲLATAMпјү ең°еҢәзҡ„з”ЁжҲ·йӮЈйҮҢзӘғеҸ–ж•Ҹж„ҹдҝЎжҒҜгҖӮ

иҜҘжҙ»еҠЁиҮӘ 2023 е№ҙе№ҙдёӯд»ҘжқҘдёҖзӣҙжҙ»и·ғпјҢй’ҲеҜ№еўЁиҘҝе“Ҙе’Ңе…¶д»–жӢүдёҒзҫҺжҙІеӣҪе®¶пјҢиў«еҪ’е’ҺдәҺдёҖдёӘеҗҚдёә Cybercartel зҡ„з”өеӯҗзҠҜзҪӘз»„з»ҮпјҢиҜҘз»„з»Үеҗ‘е…¶д»–зҪ‘з»ңзҠҜзҪӘеӣўдјҷжҸҗдҫӣжӯӨзұ»жңҚеҠЎгҖӮиҝҷдәӣжү©еұ•дёҚеҶҚеҸҜдҫӣдёӢиҪҪгҖӮ

вҖңжҒ¶ж„Ҹзҡ„ Google Chrome жү©еұ•зЁӢеәҸе°ҶиҮӘе·ұдјӘиЈ…жҲҗеҗҲжі•еә”з”ЁзЁӢеәҸпјҢиҜұйӘ—з”ЁжҲ·д»ҺеҸ—ж„ҹжҹ“зҡ„зҪ‘з«ҷжҲ–зҪ‘з»ңй’“йұјжҙ»еҠЁе®үиЈ…е®ғпјҢвҖқMetabase Q Ocelot еЁҒиғҒжғ…жҠҘеӣўйҳҹзҡ„ Karla Gomez зҡ„е®үе…Ёз ”з©¶дәәе‘ҳ Ramses VazquezВ иҜҙгҖӮ

вҖңе®үиЈ…жү©еұ•зЁӢеәҸеҗҺпјҢе®ғдјҡе°Ҷ JavaScript д»Јз ҒжіЁе…Ҙз”ЁжҲ·и®ҝй—®зҡ„зҪ‘йЎөдёӯгҖӮжӯӨд»Јз ҒеҸҜд»ҘжӢҰжҲӘе’Ңж“ҚзәөйЎөйқўеҶ…е®№пјҢд»ҘеҸҠжҚ•иҺ·ж•Ҹж„ҹж•°жҚ®пјҢдҫӢеҰӮзҷ»еҪ•еҮӯжҚ®гҖҒдҝЎз”ЁеҚЎдҝЎжҒҜе’Ңе…¶д»–з”ЁжҲ·иҫ“е…ҘпјҢе…·дҪ“еҸ–еҶідәҺзү№е®ҡзҡ„жҙ»еҠЁе’ҢжүҖй’ҲеҜ№зҡ„дҝЎжҒҜзұ»еһӢгҖӮ
