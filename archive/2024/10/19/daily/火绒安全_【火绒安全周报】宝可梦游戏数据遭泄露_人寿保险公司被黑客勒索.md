# 美国人寿保险公司被黑客勒索

#

据报道，美国人寿保险公司Globe Life遭遇了一起黑客勒索事件。该事件与Globe Life公司系统今年6月遭到的攻击有关。近日，黑客向该公司提出勒索要求，企图以不公开6月攻击事件所盗取的数据为条件索取赎金。据悉，被盗数据涉及5000多名用户，包括姓名、电话号码、电子邮件地址等敏感信息。Globe Life公司正在与美国执法部门合作调查这一事件，并表示6月的攻击事件并非勒索软件攻击所致。目前，Globe Life公司认为此次事件不会对其运营和财务状况产生重大影响。

# 宝可梦游戏数据遭遇泄露

#

据报道，《宝可梦》游戏的开发商Game Freak近日遭遇网络攻击，导致2606名员工的个人信息和大量内部资料被泄露，其中包括《宝可梦》游戏的源代码和未公布项目的开发细节。据悉，此次攻击曝光了该公司第十代宝可梦的代号及下一款宝可梦游戏正在为即将推出的Switch继任者进行测试的信息。事件发生后，Game Freak公司对其服务器进行了彻底检查和重建。目前，该公司已就此事件道歉，并表示正与受影响者联系，同时承诺将会加强安全基础措施以防范未来类似事件的发生。

# 仿冒网站乱象频发

#

近日，一起假冒新闻网站的案件引发社会广泛关注。经记者调查，不法分子利用低技术门槛和低成本制作假冒网站，冒充正规媒体或企事业单位，欺骗用户进行非法牟利。尽管相关部门近年来在持续打击这一乱象，但由于仿冒网站的服务器多设在境外，且具有较强的隐蔽性，监管难度极大。专家建议，可以采用人工智能和大数据筛查手段加强监管力度；搜索引擎应加大对官方网站的展示力度，挤压仿冒网站生存空间；相关部门也应普及鉴别网站真伪的知识，增强公众防范意识。

# 数据中心公司CEO面临65年监禁

#

近日，美国司法部披露一起当地一家数据中心公司CEO被起诉的案件。该CEO涉嫌对美国联邦政府进行重大欺诈并向美国证券交易委员会（SEC）提交虚假陈述。据悉，该CEO通过创建虚假机构，为其所在数据中心公司伪造最高级别的数据中心认证，以此欺骗美国证券交易委员会签订合同并支付1000多万美元的费用。这名CEO被指控犯有重大欺诈罪和虚假陈述罪。若罪名成立，他将面临最高65年监禁。目前，美国相关部门正在对此案件进行调查。

# 思科公司疑似遭遇数据泄露

#

据报道，全球网络设备巨头思科公司近日疑似遭遇数据泄露事件。名为“IntelBroker”的威胁行为者在黑客论坛声称已于10月6日入侵思科，并公开出售窃取的思科内部数据。据悉，此次泄露的数据包括源代码、客户信息及思科机密文档等敏感内容。思科公司表示正在针对该事件展开调查，但并未发现系统被入侵的迹象。目前，思科已与执法部门合作调查此次事件，并承诺一旦确认客户机密信息被盗，将立即通知相关客户。
