# Microsoft发布2024年7月安全更新

原创

CNVD

CNVD漏洞平台

安全公告编号:CNTA-2024-0009

7月10日，微软发布了2024年7月份的月度例行安全公告，修复了多款产品存在的139个安全漏洞，产品包括Windows 11、Windows 10、Windows Server 2022、Windows Server 2019和Windows Server 2008等。

利用上述漏洞，攻击者可进行欺骗，绕过安全功能限制，获取敏感信息，提升权限，执行远程代码，或发起拒绝服务攻击等。CNVD提醒广大Microsoft用户尽快下载补丁更新，避免引发漏洞相关的网络安全事件。
