# 美军步兵AR装备的现况和未来发展

原创

多米特

情报分析师

步兵是最为传统的兵种，从第一次世界大战到今天，战斗方式并没有什么重大的改变。唯一不同的，大概只有步枪的射速变的更快; 过去挂在身上各个部位的装备，逐渐整合到战术背心上、夜视仪让士兵拥有夜间视野，无线电能更快速传递与接收命令。但在战场上，卧倒、手榴弹、火力压制与小组战术仍是一般步兵最熟悉的东西。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR8cr74EX0qzojO8oQ38BezmFccjv703l8tYRzgBicSs8icsA68RxsKZaPg/640?wx_fmt=png)

*美国陆军与微软公司正在研发的「整合式视觉增强系统」（Integrated Visual Augmentation System， IVAS），让步兵真正跨入信息化作战的领域。图/取自美国陆军网站*

不过美国陆军与微软公司正在研发的“整合式视觉增强系统”（Integrated Visual Augmentation System， IVAS），即将改变这种情况，让步兵真正跨入信息化作战的领域。

步兵信息化作战的AR装备

美国陆军这款IVAS的前身，是微软公司所推出的Hololens 2眼镜，属于一种增强现实（Augmented Reality， AR）技术。与虚拟现实（Virtual Reality， VR）建构一个完全虚拟的世界不同，扩增实境是在真实的世界中，添加更多原本不存的影像，藉由真假世界的融合，与人类产生互动。

这最好的例子是任天堂所推出的宝可梦手机游戏，玩家可以借由手机画面，看到在现实世界中不存在的宝可梦角色，进而在街头四处抓宝。IVAS也利用类似的技术，让士兵可以看到在真实战场上不存在的信息，进而强化战斗力。

其实这并非什么新颖的概念，在战机上已使用很久的抬头显示器（HUD）就是这种扩增实境的雏形。抬头显示器会将飞行员所需要的信息，投影到一块特殊玻璃上，让飞行员不用低头查看飞行仪表，就能在飞行或空战过程中，掌握所有情况，还能辅助锁定敌机与瞄准炸射目标。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR8ticT4d0QxEwthlx91ib9kYTOfnz4Yj6bzKEr9W9dW8CZLdCNTqXqT9Kw/640?wx_fmt=png)

*美国陆军这款IVAS的前身，是微软公司所推出的Hololens 2眼镜，属于一种增强现实（Augmented Reality， AR）技术。图/取自美国陆军网站*

抬头显示器后来更发展成头盔显示器，所有的资料直接投影到面罩上，更为便利，飞行员能直接以目光追踪并锁定目标，快速发射飞弹。只是这种系统在过去非常昂贵，IVAS企图利用民间公司所发展的产品，以较便宜的价格，大量配备在一般步兵身上。

整套IVAS的硬件部份，包括了一个安装在护目镜上的抬头显示器，这个护目镜上还配备了各式侦测器，提供夜视、热影像与测距功能。这个多功能护目镜连接到士兵身上的穿戴型电脑上，并由电池供电。整体重量约1.54公斤，其中有差不多1.08公斤，都在头部的护目镜上。同时也整合了个人无线电与无线网络系统，让士兵能从抬头显示器上，看到整个战场的实时立体地图，即使是在黑夜或烟雾迷漫的战场上，也能透露夜视、热影像系统来侦察敌军，快速标定威胁所在位置，实时分享作战信息。

除了利用本身的侦测器获得信息，IVAS还能连结步枪上的光学瞄准仪，让士兵不用冒着被击中的风险，只要将步枪伸出墙角或掩体，就能从护目镜上的抬头显示器，看到步枪所瞄准的目标，并立即开火。这样的功能大幅增强士兵在狭小环境中的作战能力，在城镇战中特别有用。

此外IVAS还能连结装甲运兵车或步兵战斗车上的360度全景摄影机，让士兵们不用冒险下车，就能看到周边的情势。这有助于机械化步兵随时掌握战况，提前拟定作战计划，不会在一开始下车战斗时，处于完全陌生的环境中。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR88tf0AicWZg28TV7CElibYg06IYm8Bo9NZJfE1Q4TbDNzNgQgibEkPQZtA/640?wx_fmt=png)

*IVAS还能连结装甲运兵车或步兵战斗车上的360度全景摄影机，让士兵们不用冒险下车，就能看到周边的情势。图/取自美国陆军网站*

IVAS测试情况引起疑虑

由于IVAS本身就是一种增强现实的装备，所以非常适合用于日常训练，内建「小队沉浸式虚拟训练系统」（Squad Immersive Virtual Trainer， SiVT），能将训练内容与虚拟的敌人，直接投影在每个士兵的抬头显示器上，让士兵看到未来的战场情况。演训结果，还可透过战术云端套件（Tactical Cloud Package， TCP）上传到云端系统，供教官分析。

可以说一套IVAS就可取代过去的通讯、侦搜索、图资与训练等系统，也因此受到美国陆军的青睐，不遗余力发展推动，并与微软公司签下了十年提供12万套IVAS的合约，总价高达220亿美元。

不过IVAS计划也受到诸多的批评，首先是民间的Hololens 2眼镜，是否适合军事使用，从一开始就受到质疑。军用版的发展也是一波三折，初期的产品并不稳定。目前美国陆军已小规模采购五千套IVAS 1.0版，供基层部队进行作战测试与评估（DOT&E），结果出现许多问题。有士兵表示，使用IVAS数小时以后，容易出现晕昡、想吐、眼睛疲劳与肩颈酸痛等症状，而且汗水或雨水很容易渗入护目镜中，形成的水气会严重影响抬头显示器的清晰程度。同时故障率偏高，夜视系统不稳定的问题仍未完全解决。

许多批评者认为，在家中使用Hololens 2眼镜玩射击游戏，与在真实的战场上使用IVAS进行战斗，是完全两回事，美国陆军低估了两者的差异性。IVAS夜视系统的视野虽然可达80度，较传统的夜视镜好上许多，但夜视镜只有在夜间战斗时使用，IVAS却必需全时间配戴，在日光充足的时候，反而影响了士兵的周边视野。

这也导致测试时，部分士兵的射击反应速度变慢，击中敌方目标的数目也下降。此外IVAS的操作方式并不够直觉，整合各种侦测器的护目镜，体积很大，也容易曝露士兵位置。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR8bfm720N3rfL8RibkronMKrkY6bhibwuNqftJXALqt41Wak3kCaPapRzQ/640?wx_fmt=png)*许多批评者认为，在家中使用Hololens 2眼镜玩射击游戏，与在真实的战场上使用IVAS进行战斗，是完全两回事，美国陆军低估了两者的差异性。图/美联社*

不少受测者都认为，他们更喜欢现有的「网络战士」（Nett Warrior）系统与新型的双筒式夜视镜ENVG-B。前者是供基层指挥官使用的战术信息系统，可以将作战指令，战场信息直接传送到士兵身上的终端机; 后者正在逐步取代传统式的厚重夜视镜，能提供更清晰的夜视与热影像功能。

这两者虽然未整合成一个单一系统，却能彼此配合使用，在ENVG-B读入Nett Warrior所提供的信息，也因此受到基层士兵的好评。包括美国国会也对IVAS的发展有疑虑，不只删除了部份预算，还继续加码采购ENVG-B，以防IVAS最后走向失败。

前景不明的IVAS展示步兵改革方向

不过美国陆军对于IVAS计划仍然非常有信心，要求微软公司持续发展。目前已经在IVAS 1.0的基础上，推出IVAS 1.1，新增改良过的低光度侦测器，能更有效侦测到视线不良处的敌人，而IVAS 1.2则重新设计了整个护目镜的外观，让全套系统的重量减轻。

美国陆军计划在未来继续采购IVAS 1.1与IVAS 1.2，虽然不会马上变成部队的标准装备，但将扩大测试的范围。负责的官员说，IVAS初期的表现虽然不尽人意，不过就像一开始的移动电话，大又不便，后来却发展出小巧成熟的iPhone，IVAS也具备一样的潜力。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR8Pvq9hh9icmWT46Adtz6XCqoiaT88d04m5iaUx0RbEXOoxKEvhPIQDkQqg/640?wx_fmt=png)*美国陆军计划在未来继续采购IVAS 1.1与IVAS 1.2，虽然不会马上变成部队的标准装备，但将扩大测试的范围。图/取自美国陆军网站*

美国陆军会如此坚持这个计划的原因，是IVAS只是整个拼图的一小部份。未来随着各式无人载具的普及运用，配备IVAS的士兵，可以随时取得无人机或地面战斗机械人的画面，协助判断敌军目标，下达开火指令，其扩展的侦搜与作战能力，将极为惊人。

另外IVAS的所有信息都以云端共享的方式储存，随着人工智能技术的快速发展，这些储存在云端的信息，可由人工智能协助处理。不只能更快找出敌人位置，分析目前的战场情势，将来甚至能介入指挥链，提供基层指挥官最佳的战术建议，并协助指挥每一个士兵。

因此IVAS虽然前景不明，有可能成为被美军放弃的诸多实验性计划之一，但已描绘出传统步兵的改革方向。过去步兵常被认为是最没有技术性的兵种，却也是最难以取代的部队，因为无论科技如何进步，最后都需要步兵来清理战场与占领目标。

![](https://mmbiz.qpic.cn/mmbiz_png/Mkvak7WiccmXkJ0EfUVP4oRHrXLuHmnR8tiaJWggtuTOuZTvUVDltr9ZZich3Dm1q8MqJ9UgJeCYRicSibyhM4iaWexQ/640?wx_fmt=png)

随着科技日新月异，现在能投资在步兵身上的装备与训练，已经非常多，绝不是一把步枪与一柄刺刀而已。 国军会坚持一定要保留刺枪术，背后的真正原因，应该是装备、师资、场地、训练经费都严重不足，而刺枪术是最便宜、又是能最快展现训练成果的方法。
