得益于 [Ubuntu](https://www.iplaysoft.com/ubuntu.html)、[深度](https://www.iplaysoft.com/deepin.html)、[优麒麟](https://www.iplaysoft.com/ubuntukylin.html)、[Mint](https://www.iplaysoft.com/linux-mint.html) 等一众优秀 Linux 发行版的存在，现在除了[服务器](https://www.iplaysoft.com/go/vps)之外，个人桌面级用户的使用率也越来越高了，特别在极客、[程序员](https://www.iplaysoft.com/tag/%E5%BC%80%E5%8F%91)群体里还是比较受欢迎的。

作为重要的通讯工具之一 ，**QQ for Linux** 在多年前发布后就一直没更新了，而且网页版 QQ 也已关停。用户除使用 Wine、[虚拟机](https://www.iplaysoft.com/tag/%E8%99%9A%E6%8B%9F%E6%9C%BA)外并没更好的 QQ 版本可用。但随着[腾讯](https://www.iplaysoft.com/tag/%E8%85%BE%E8%AE%AF)“推翻重做”了 [苹果版 Mac QQ](https://www.iplaysoft.com/qq-for-mac.html) 后，新一代的技术终于也惠及到其他平台，带来了**全新架构的 Linux QQ** 版本……

「Linux v3.0 全新重构版发布」

### Linux QQ 重生回归！发布全新 NT 技术架构版本

继腾讯发布了重构版的 [QQ for Mac](https://www.iplaysoft.com/qq-for-mac.html) 之后，腾讯终于将这个全新名为“QQ NT”技术架构应用到 Linux 平台上了，正式宣告船新的 **Linux 版 QQ** 正式到来！这对于开发者、[运维](https://www.iplaysoft.com/tag/%E8%BF%90%E7%BB%B4)人员等、以及需要经常使用 [Linux](https://www.iplaysoft.com/os/linux-platform) 系统[工作](https://www.iplaysoft.com/tag/%E5%B7%A5%E4%BD%9C)的朋友绝对是一大好消息。

![新版 Linux QQ](https://img.iplaysoft.com/wp-content/uploads/2019/linux-qq/linux_qq.jpg)

这也使得诸如 UOS、[深度](https://www.iplaysoft.com/deepin.html)、[优麒麟](https://www.iplaysoft.com/ubuntukylin.html)等一系列国产 Linux 操作系统在使用 QQ 进行网络沟通[办公](https://www.iplaysoft.com/tag/%E5%8A%9E%E5%85%AC)时能获得更加友好、更加接近 Windows 和 Mac 用户的使用体验，同时也拥有更加现代化的外观。

![QQ for Linux](https://img-dp.iplaysoft.com/dispatch/447e81455586881334df74bcfd6cd125.gif)

毕竟 Linux 近年来在国内的生态也已经日益完善，越来越多的大众也因各种理由开始[学习](https://www.iplaysoft.com/tag/%E5%AD%A6%E4%B9%A0)和接触 Linux。像 [WPS for Linux 办公软件](https://www.iplaysoft.com/wps-for-linux.html)、[搜狗输入法](https://www.iplaysoft.com/sogou-pinyin.html)、[QQ 音乐](https://www.iplaysoft.com/go/qqmusic)等各种各样的工具都应有尽有，尽管跟 [macOS](https://www.iplaysoft.com/os/mac-platform) 和 Windows 的生态依然没法比，但目前的发展势头还是很不错的。而这个全新版本 Linux QQ 的出现也是大好事一件。

![QQ Linux](https://img.iplaysoft.com/wp-content/uploads/2019/linux-qq/linux_qq_v3a.jpg)

新版的 Linux QQ 3.0 支持 x64 (x86\_64、amd64)、arm64 的处理器架构，每种架构都提供了 .deb, .rpm, .AppImage 等格式的安装包。可支持绝大多数主流的 Linux 发行版，比如 [Debian](https://www.iplaysoft.com/debian.html) 系 ([Ubuntu](https://www.iplaysoft.com/ubuntu.html))、红帽系 ([CentOS](https://www.iplaysoft.com/centos.html)、[Fedora](https://www.iplaysoft.com/fedora.html)、RHEL) 、[Deepin](https://www.iplaysoft.com/deepin.html)、UOS、[Mint](https://www.iplaysoft.com/linux-mint.html) 等等，大家按需选择即可。

### QQ Linux 3.0

![Linux QQ 搜索](https://img.iplaysoft.com/wp-content/uploads/2019/linux-qq/linux_qq_v3b.jpg)

从实际体验来看，全新原生的 [QQ Linux](https://www.iplaysoft.com/qq-linux.html) 整体感觉跟 Mac 版 QQ 比较相似，功能不多也不花哨，总体也是简洁实用的风格。大家喜欢的 QQ [截图](https://www.iplaysoft.com/tag/%E6%88%AA%E5%9B%BE)、聊天记录全局搜索、文件传输等功能都有。

![QQ 截图 Linux](https://img.iplaysoft.com/wp-content/uploads/2019/linux-qq/linux_qq_v3c.jpg)

对于大多数普通办公或沟通需求的朋友来说， 新版本的 **QQ for Linux** 已经能很好地满足要求了，应该没有太多必要在去使用[虚拟机](https://www.iplaysoft.com/tag/%E8%99%9A%E6%8B%9F%E6%9C%BA)安装 Windows QQ 的了。

### 写在后面：

当然，一个 **Linux QQ** 或许并不会改变 Linux 桌面端在大众用户市场的形势，毕竟大家需要的不仅是 [QQ](https://www.iplaysoft.com/qq.html)，可能还要 [Office 办公软件](https://www.iplaysoft.com/go/officepost)、[PhotoShop](https://www.iplaysoft.com/adobe-creative-cloud.html) 等一系列杀手应用。

但好的是，各种软件在 Linux 上基本都能找到有不错的替代品，比如 [金山 WPS](https://www.iplaysoft.com/wps-for-linux.html)、[GIMP](https://www.iplaysoft.com/gimp.html) 等。而现在“多平台”包含 Linux 的软件数量也已经非常的多了，而且在专业应用方面，比如[编程](https://www.iplaysoft.com/tag/%E7%BC%96%E7%A8%8B)开发和运维，Linux 的生态其实非常不错了。如果你敢兴趣，也不妨都去玩玩。
