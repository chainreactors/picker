Here is a step-by-step procedure to check if the adaptation project is active.

1. Open the app that you want to check
2. Run the SAPUI5 Diagnostic by using the following shortcut: CTRL+SHIFT+ALT+S
3. Expand the "Flexibility section"
4. If the adaptation project is active, you should see the custom namespace. For example: "com.acme"

![](/legacyfs/online/storage/blog_attachments/2022/10/diag-tool.png)

Note: the function is available starting from SAP\_UI 7.55. Check the prerequisites for the "Flexibility" feature. This tab only displays apps that use sap.ui.fi library, and SAPUI5 needs to be in debug mode. See details on the Help Portal > SAPUI5: UI Development Toolkit for HTML5 > Essentials > Troubleshooting > Diagnostics [https://help.sap.com/docs/ABAP\_PLATFORM\_NEW/468a97775123488ab3345a0c48cadd8f/6ec18e80b0ce47f290bc264...](https://help.sap.com/docs/ABAP_PLATFORM_NEW/468a97775123488ab3345a0c48cadd8f/6ec18e80b0ce47f290bc2645b0cc86e6.html?locale=en-US)
