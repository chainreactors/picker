# At 17000 curl commits

[December 27, 2022](https://daniel.haxx.se/blog/2022/12/27/at-17000-curl-commits/) [Daniel Stenberg](https://daniel.haxx.se/blog/author/daniel/)

Today, another 1,000 commits have been recorded as done by me in the curl source code git repository since [November 2021](https://daniel.haxx.se/blog/2021/11/12/16000-curl-commmits/). Out of a total of 29,608 commits to the [curl source code repository](https://github.com/curl/curl), I have made 17,001. 57.42%.

The most recent one was [PR #10019](https://github.com/curl/curl/pull/10019).

In 2022, I have done 56% of all the commits in the curl source repository. I am also the only developer who works full time on curl all the time.

In 2022, 179 individuals authored commits that were merged into curl. 115 of them did that for the first time this year. Over curl’s life time, a total of 1104 persons have authored code merged into curl.

Do I ever get bored? Not yet. I will let you know if I do.
