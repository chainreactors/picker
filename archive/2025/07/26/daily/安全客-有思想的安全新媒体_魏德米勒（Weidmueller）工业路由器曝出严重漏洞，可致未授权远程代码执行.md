з”ұ CERT@VDE еҚҸи°ғеҸ‘еёғзҡ„жңҖж–°е®үе…Ёе…¬е‘ҠжҳҫзӨәпјҢйӯҸеҫ·зұіеӢ’пјҲWeidmuellerпјүж——дёӢе№ҝжіӣеә”з”ЁдәҺе·ҘдёҡиҮӘеҠЁеҢ–е’Ңе…ій”®еҹәзЎҖи®ҫж–ҪйўҶеҹҹзҡ„ **IE-SR-2TX зі»еҲ—е®үе…Ёи·Ҝз”ұеҷЁ**дёӯпјҢеӯҳеңЁеӨҡйЎ№й«ҳеҚұжјҸжҙһгҖӮиҝҷдәӣжјҸжҙһе·Іиў«еҲҶй…Қдә”дёӘ CVE зј–еҸ·пјҢеҸҜиғҪиў«иҝңзЁӢж”»еҮ»иҖ…еҲ©з”ЁиҝӣиЎҢжңӘжҺҲжқғе‘Ҫд»Өжү§иЎҢгҖҒзј“еҶІеҢәжәўеҮәе’Ң Web жҺҘеҸЈжіЁе…Ҙзӯүж”»еҮ»пјҢиҝӣиҖҢиҺ·еҸ–и®ҫеӨҮзҡ„ root жқғйҷҗгҖӮ

е…¶дёӯдёӨйЎ№жјҸжҙһвҖ”вҖ”**CVE-2025-41663** е’Ң **CVE-2025-41687**вҖ”вҖ”йЈҺйҷ©зӯүзә§жңҖй«ҳпјҢCVSS иҜ„еҲҶдёә 9.8**пјҲеҚұжҖҘзә§пјү**гҖӮ

**вҖў CVE-2025-41663** е…Ғи®ёеӨ„дәҺдёӯй—ҙдәәпјҲMITMпјүдҪҚзҪ®зҡ„ж”»еҮ»иҖ…пјҢйҖҡиҝҮ u-link з®ЎзҗҶ API еҗ‘жқҘиҮӘ WWH жңҚеҠЎеҷЁзҡ„е“Қеә”дёӯжіЁе…ҘжҒ¶ж„Ҹе‘Ҫд»ӨгҖӮиҷҪз„¶иҜҘж”»еҮ»дҫқиө–дәҺе®ўжҲ·з«Ҝй…ҚзҪ®дёҚе®үе…Ёзҡ„д»ЈзҗҶи®ҫзҪ®пјҢдҪҶдёҖж—ҰжҲҗеҠҹпјҢж”»еҮ»е‘Ҫд»Өе°Ҷд»Ҙй«ҳжқғйҷҗжү§иЎҢгҖӮ

е…¬е‘ҠдёӯжҢҮеҮәпјҡвҖңжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„иҝңзЁӢж”»еҮ»иҖ…иӢҘеӨ„дәҺдёӯй—ҙдәәдҪҚзҪ®пјҢеҸҜжіЁе…Ҙд»»ж„Ҹе‘Ҫд»ӨпјҢе№¶д»ҘжҸҗеҚҮзҡ„жқғйҷҗжү§иЎҢгҖӮвҖқ

**вҖў CVE-2025-41687** еҗҢж ·дёә 9.8 еҲҶпјҢжәҗдәҺ u-link з®ЎзҗҶ API дёӯзҡ„ж Ҳзј“еҶІеҢәжәўеҮәжјҸжҙһгҖӮж”»еҮ»иҖ…ж— йңҖиә«д»ҪйӘҢиҜҒпјҢдҫҝеҸҜиҝңзЁӢеҲ©з”ЁиҜҘжјҸжҙһе…ЁйқўжҺҘз®Ўи®ҫеӨҮгҖӮ

жӯӨеӨ–пјҢе…¬е‘ҠиҝҳжҠ«йңІдәҶеҸҰеӨ–дёүйЎ№й«ҳеҚұжјҸжҙһпјҢCVSS иҜ„еҲҶдёә 8.8пјҡCVE-2025-41661гҖҒCVE-2025-41683 е’Ң CVE-2025-41684гҖӮй—®йўҳдё»иҰҒеҮәзҺ°еңЁ Web з®ЎзҗҶз•ҢйқўпјҢжәҗдәҺиҫ“е…ҘйӘҢиҜҒдёҚи¶ід»ҘеҸҠзјәд№Ҹ CSRFпјҲи·Ёз«ҷиҜ·жұӮдјӘйҖ пјүйҳІжҠӨпјҢеҸҜиғҪиў«е·Ізҷ»еҪ•жҲ–з”ҡиҮіжңӘзҷ»еҪ•зҡ„ж”»еҮ»иҖ…еҲ©з”ЁпјҢжү§иЎҢд»»ж„Ҹе‘Ҫд»Өе№¶иҺ·еҸ– root жқғйҷҗгҖӮ

**вҖў CVE-2025-41661** еӣ зјәеӨұ CSRF йҳІжҠӨжңәеҲ¶иҖҢеӯҳеңЁйЈҺйҷ©пјӣ

**вҖў CVE-2025-41683** еӯҳеңЁдәҺ event\_mail\_test жҺҘеҸЈпјҢиҫ“е…ҘжңӘжӯЈзЎ®иҝҮж»Өпјӣ

**вҖў CVE-2025-41684** еҲҷж¶үеҸҠ tls\_iotgen\_setting жҺҘеҸЈпјҢеӯҳеңЁзұ»дјјй—®йўҳгҖӮ

иӢҘиҝҷдәӣжјҸжҙһиў«жҲҗеҠҹеҲ©з”ЁпјҢж”»еҮ»иҖ…дёҚд»…еҸҜ**е№Іжү°е·ҘдёҡзҪ‘з»ңзҡ„жӯЈеёёиҝҗиЎҢ**пјҢз”ҡиҮіеҸҜиғҪеҲ©з”Ёиў«ж”»йҷ·зҡ„и·Ҝз”ұеҷЁдҪңдёәи·іжқҝпјҢеҸ‘иө·**жӣҙе№ҝжіӣзҡ„е…Ҙдҫөжҙ»еҠЁ**гҖӮ

йӯҸеҫ·зұіеӢ’е·ІеҸ‘еёғиЎҘдёҒдҝ®еӨҚзӣёе…ій—®йўҳпјҢж¶үеҸҠдә§е“Ғзҡ„еӣәд»¶**жӣҙж–°еҰӮдёӢпјҡ**

**вҖў IE-SR-2TX-WLпјҡ**еҚҮзә§иҮі v1.49

**вҖў IE-SR-2TX-WL-4G-EUпјҡ**еҚҮзә§иҮі v1.62

**вҖў IE-SR-2TX-WL-4G-US-Vпјҡ**еҚҮзә§иҮі v1.62

еңЁеҸ‘еёғиЎҘдёҒзҡ„еҗҢж—¶пјҢйӯҸеҫ·зұіеӢ’иҝҳжҸҗйҶ’з”ЁжҲ·еә”йҒөеҫӘзҪ‘з»ңе®үе…ЁжңҖдҪіе®һи·өпјҡеҸҠж—¶жӣҙж”№й»ҳи®ӨеҜҶз ҒгҖҒд»…е°Ҷи®ҫеӨҮжҡҙйңІдәҺеҸҜдҝЎзҪ‘з»ңгҖҒйҮҮз”Ёе®үе…Ёй…ҚзҪ®зӯүпјҢд»ҘжңҖеӨ§зЁӢеәҰйҷҚдҪҺжҪңеңЁж”»еҮ»йқўгҖӮ
