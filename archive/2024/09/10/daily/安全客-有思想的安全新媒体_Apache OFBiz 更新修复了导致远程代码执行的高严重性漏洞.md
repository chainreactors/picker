жң¬ж–ҮжҳҜзҝ»иҜ‘ж–Үз« пјҢж–Үз« еҺҹдҪңиҖ… Ravie LakshmananпјҢж–Үз« жқҘжәҗпјҡThe Hacker News

еҺҹж–Үең°еқҖпјҡ<https://thehackernews.com/2024/09/apache-ofbiz-update-fixes-high-severity.html>

иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

Apache OFBiz ејҖжәҗдјҒдёҡиө„жәҗи§„еҲ’ пјҲERPпјү зі»з»ҹдёӯи§ЈеҶідәҶдёҖдёӘж–°зҡ„е®үе…ЁжјҸжҙһпјҢеҰӮжһңжҲҗеҠҹеҲ©з”ЁиҜҘжјҸжҙһпјҢеҸҜиғҪдјҡеҜјиҮҙеңЁ Linux е’Ң Windows дёҠжү§иЎҢжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„иҝңзЁӢд»Јз ҒгҖӮ

иҜҘй«ҳдёҘйҮҚжҖ§жјҸжҙһиў«и·ҹиёӘдёәВ CVE-2024-45195пјҲCVSS иҜ„еҲҶпјҡ7.5пјүпјҢеҪұе“Қ 18.12.16 д№ӢеүҚзҡ„жүҖжңүиҪҜд»¶зүҲжң¬гҖӮ

вҖңжІЎжңүжңүж•ҲеҮӯжҚ®зҡ„ж”»еҮ»иҖ…еҲ©з”Ё Web еә”з”ЁзЁӢеәҸдёӯзјәе°‘зҡ„и§ҶеӣҫжҺҲжқғжЈҖжҹҘеңЁжңҚеҠЎеҷЁдёҠжү§иЎҢд»»ж„Ҹд»Јз ҒпјҢвҖқRapid7 е®үе…Ёз ”з©¶е‘ҳ Ryan Emmons еңЁдёҖд»Ҫж–°жҠҘе‘ҠдёӯиҜҙгҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢCVE-2024-45195 з»•иҝҮдәҶдёҖзі»еҲ—й—®йўҳпјҢеҢ…жӢ¬ CVE-2024-32113гҖҒCVE-2024-36104 е’Ң CVE-2024-38856пјҢйЎ№зӣ®з»ҙжҠӨиҖ…еңЁиҝҮеҺ»еҮ дёӘжңҲдёӯи§ЈеҶідәҶиҝҷдәӣй—®йўҳгҖӮ

жӯӨеҗҺпјҢCVE-2024-32113 е’Ң CVE-2024-38856 йғҪйҒӯеҲ°дәҶе№ҝжіӣзҡ„з§ҜжһҒеҲ©з”ЁпјҢеүҚиҖ…иў«з”ЁжқҘйғЁзҪІ Mirai еғөе°ёзҪ‘з»ңжҒ¶ж„ҸиҪҜд»¶гҖӮ

Rapid7 иЎЁзӨәпјҢжүҖжңүдёүдёӘиҫғж—§зҡ„зјәзӮ№йғҪжәҗдәҺвҖңдҪҝжҺ§еҲ¶еҷЁе’Ңи§Ҷеӣҫең°еӣҫзҠ¶жҖҒдёҚеҗҢжӯҘзҡ„иғҪеҠӣвҖқпјҢиҝҷдёӘй—®йўҳеңЁд»»дҪ•иЎҘдёҒдёӯйғҪд»ҺжңӘеҫ—еҲ°е®Ңе…Ёи§ЈеҶігҖӮ

иҜҘжјҸжҙһзҡ„дёҖдёӘеҗҺжһңжҳҜпјҢж”»еҮ»иҖ…еҸҜиғҪдјҡж»Ҙз”Ёе®ғжқҘжү§иЎҢд»Јз ҒжҲ– SQL жҹҘиҜўпјҢе№¶е®һзҺ°ж— йңҖиә«д»ҪйӘҢиҜҒзҡ„иҝңзЁӢд»Јз Ғжү§иЎҢгҖӮ

жңҖж–°зҡ„иЎҘдёҒвҖңйӘҢиҜҒдәҶеҰӮжһңз”ЁжҲ·жңӘз»Ҹиә«д»ҪйӘҢиҜҒпјҢи§Ҷеӣҫеә”иҜҘе…Ғи®ёеҢҝеҗҚи®ҝй—®пјҢиҖҢдёҚжҳҜзәҜзІ№еҹәдәҺзӣ®ж ҮжҺ§еҲ¶еҷЁжү§иЎҢжҺҲжқғжЈҖжҹҘгҖӮ

Apache OFBiz зүҲжң¬ 18.12.16 иҝҳи§ЈеҶідәҶдёҖдёӘе…ій”®зҡ„жңҚеҠЎеҷЁз«ҜиҜ·жұӮдјӘйҖ  пјҲSSRFпјү жјҸжҙһпјҲCVE-2024-45507пјҢCVSSВ иҜ„еҲҶпјҡ9.8пјүпјҢиҜҘжјҸжҙһеҸҜиғҪйҖҡиҝҮеҲ©з”Ёзү№еҲ¶зҡ„ URL еҜјиҮҙжңӘз»ҸжҺҲжқғзҡ„и®ҝй—®е’Ңзі»з»ҹеҸ—жҚҹгҖӮ
