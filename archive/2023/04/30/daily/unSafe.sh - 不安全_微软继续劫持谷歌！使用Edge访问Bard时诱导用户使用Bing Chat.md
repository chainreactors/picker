Microsoft Edge 浏览器目前以臃肿和广告多著称，不过微软对此似乎并不在意，只要能继续获得市场份额那就是值得的，比如现在微软尝试通过 Microsoft Edge 劫持 Google Bard。

Bard 是谷歌对标 ChatGPT 和 Bing Chat 开发的对话式人工智能机器人，就目前实际体验和用户关注度来说，Bard 远不如 ChatGPT 和 Bing Chat，但这并不妨碍微软通过 Microsoft Edge 继续劫持 Bard，正所谓落井下石，现在 Bard 处于弱势那微软就要疯狂打压。

[![微软继续劫持谷歌！使用Edge访问Bard时诱导用户使用Bing Chat](https://img.lancdn.com/landian/2023/04/98461-1.png)](https://img.lancdn.com/landian/2023/04/98461-1.png)

当用户使用 Microsoft Edge 浏览器访问 Bard 时，地址栏会弹出提示 “将 Bard 与 AI 驱动的 new Bing 进行比较”，这个做法着实有些虾仁猪心，因为点击后不是跳转到 new Bing，而是利用分屏功能同时打开 Bard 和 new Bing。

[![微软继续劫持谷歌！使用Edge访问Bard时诱导用户使用Bing Chat](https://img.lancdn.com/landian/2023/04/98461-2.png)](https://img.lancdn.com/landian/2023/04/98461-2.png)

正如前文所说，目前 Bard 体验远不如 Bing Chat，所以对比使用的话，用户大概率会被 “诱拐” 使用 new Bing 而不是 Bard，所以微软这招还是挺狠的。

目前微软在想方设法围堵谷歌，例如 Microsoft Edge 访问 Chrome 官网时弹出提示要求用户继续使用 Microsoft Edge，此外微软也在 Microsoft Edge 里投放其他各种各样的广告，但从市场份额来说 Microsoft Edge 现在增长已经停滞，这倒是有可能是太臃肿导致用户重新使用 Chrome 或 Firefox。
