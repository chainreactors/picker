## Python 3.14 将支持模板字符串

[![Python](https://icon.solidot.org/images/topics/topicPython.png?123)](/search?tid=114 "Python")

[Wilson](/~Wilson) (42865)发表于 2025年04月22日 15时50分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81111&appkey=1370085986&title=Python%203.14%20%E5%B0%86%E6%94%AF%E6%8C%81%E6%A8%A1%E6%9D%BF%E5%AD%97%E7%AC%A6%E4%B8%B2 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自图书馆员与追寻鹅妈妈**

Python Steering Council 接受了 PEP 750 (Template Strings) ，预计 10 月释出的 Python 3.14 将加入对模板字符串的支持。模板字符串（(t-strings）是一种新型的字符串，能生成结构化数据而不是原始字符串，允许 Python 库作者构建自己的自定义模板处理逻辑。Python 自 2015 年起就有 f-strings，t-strings 为开发者提供了更多灵活性。
discuss.python.org/t/pep750-template-strings-new-updates/71594/130
