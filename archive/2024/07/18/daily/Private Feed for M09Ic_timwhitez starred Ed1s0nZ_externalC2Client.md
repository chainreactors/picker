# externalC2Client

External C2 Client , 免杀效果自测。

## 配置

第一步: Cobalt Strike listeners新建External C2 listener;
第二步: 填写端口;
[![](https://github.com/Ed1s0nZ/externalC2Client/raw/main/%E9%85%8D%E7%BD%AE.png)](https://github.com/Ed1s0nZ/externalC2Client/blob/main/%E9%85%8D%E7%BD%AE.png)
第三步: 修改main.go中: var address = `127.0.0.1:8080` 的IP和端口为Cobalt Strike External C2 listener的IP和端口;

## 编译

### Mac编译

第一步: brew install mingw-w64
第二步: GOOS=windows GOARCH=amd64 CC=/opt/homebrew/Cellar/mingw-w64/11.0.0/bin/x86\_64-w64-mingw32-gcc CGO\_ENABLED=1 go build -ldflags="-w -s" -ldflags="-H windowsgui" main.go

### Windows编译

第一步: 安装mingw64环境
第二步: go build -ldflags="-w -s" -ldflags="-H windowsgui" main.go
