**重点关注漏洞**

经过赛博昆仑CERT的分析，这里列出部分值得关注的漏洞，详细信息如下：

* ****CVE-2025-21391 Windows Storage 权限提升漏洞****

  在野 在Windows Storage中发现一个逻辑漏洞，该漏洞可以导致任意文件删除。成功利用此类漏洞可导致攻击者在目标系统上以 SYSTEM 权限执行代码。该漏洞已有在野利用。建议用户尽快测试并部署此更新。
* ****CVE-2025-21418 Windows Ancillary Function Driver for WinSock 权限提升漏洞****

  在野 在Windows AFD驱动中发现一个堆溢出漏洞。成功利用此类漏洞可导致攻击者在目标系统上以 SYSTEM 权限执行代码。该漏洞已有在野利用。建议用户尽快测试并部署此更新。
* **CVE-2025-21376 Windows Lightweight Directory Access Protocol (LDAP) 远程代码执行漏洞**

  在Windows LDAP服务端中发现了一个会导致远程无身份验证攻击的漏洞。攻击者可以通过发送特制LDAP请求，在高权限下执行任意代码。本漏洞需要攻击者赢得竞争条件，但一旦成功，可以在受害机器上以系统服务权限执行任意代码。建议用户迅速测试并部署此更新。
* **CVE-2025-21379 DHCP Client Service 远程代码执行漏洞**

  在Windows DHCP客户端服务中发现了一个释放后重用漏洞。该漏洞需要攻击者对受害者进行中间人攻击。成功利用此类漏洞可导致攻击者在目标系统上以 SYSTEM 权限执行代码。建议用户尽快测试并部署此更新。
* **CVE-2025-21381 Microsoft Excel 远程代码执行漏洞**

  在Excel中发现一个不可信指针解引用。攻击者可以通过诱使本地打开excel文件触发漏洞执行任意代码。建议用户避免下载或打开来源不明的Excel文件以预防可能的攻击并尽快部署更新。
