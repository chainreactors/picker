微软推出Security Copilot，将GPT-4应用于安全领域。

3月28日，微软发布人工智能赋能的安全分析工具——Security Copilot，将AI技术应用于网络安全领域，为用户提供类ChatGPT的助理功能。Security Copilot可以帮助安全分析师快速响应威胁，以机器级速度处理信号，在几分钟内评估风险的暴露程度。

Security Copilot 将OpenAI的大语言模型GPT-4与来自微软的安全相关模型融合在一起。其中安全模型融合了来自微软的强大的威胁情报能力，包括每日超过650万亿的信号。Security Copilot还融合了其他微软安全工具的数据和功能，包括Sentinel、Defender、Intune，可以为每个组织提供定制化的指南。Security Copilot运行在Azure的超大规模基础设施上，可以提供企业级的安全和隐私体验。

Security Copilot可以通过类似ChatGPT的接口回答用户安全相关的问题，并从交互中持续学习以适应不同企业环境并给出最佳响应。Security Copilot可以帮助安全团队更加高效地处理复杂安全问题，其主要目标是通过加速威胁情报总结和翻译来增强安全分析师的能力，使其可以在分析web流量时更快地识别出恶意活动。Security Copilot还可以通过关联威胁情报活动信号和检查攻击数据以检测之前被忽视的威胁。

微软安全副总裁Vasu Jakkal称，在安全领域，分秒必争。在Security Copilot的帮助下，安全防护人员可以在几分钟内对安全事件作出响应。Security Copilot还可以根据自然语言的调查经验来给出每一步的指导，加速安全事件调查和响应的速度。

OpenAI称GPT-4虽然在许多现实场景中仍然比不过人的能力，但在许多专业和学术领域评估时达到了与人类相同的能力。Security Copilot可简化安全团队的复杂性，放大安全团队的能力，帮助安全团队穿过web流量中的噪声识别出恶意活动。

两周前，微软发布了Microsoft 365 Copilot助理，也是基于GPT-4的，可以在不同的365应用中提高用户的工作效率，包括Word、Excel、PowerPoint、Outlook、Teams、Viva 和Power等。

微软称目前Security Copilot已经进入发布了内部预览版，未来将发布公开预览版本。

更多参见微软官方博客：https://blogs.microsoft.com/blog/2023/03/28/introducing-microsoft-security-copilot-empowering-defenders-at-the-speed-of-ai/
