## Java ShellCode Loader

基于Java实现的ShellCode加载器，兼容32位及64位平台。

核心原理是利用Jna来调用Windows API，实现shellcode的注入。

运行环境：Jre >= 1.5

注：本项目已经内置在[yzddmr6/As-Exploits](https://github.com/yzddmr6/As-Exploits)的ShellCodeLoader模块中。
