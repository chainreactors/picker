## Steam 从明年起将停止支持 Windows 7、8 和 8.1

[![游戏](https://icon.solidot.org/images/topics/topicgames.png?123)](/search?tid=13 "游戏")

[Wilson](/~Wilson) (42865)发表于 2023年03月28日 21时33分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74514&appkey=1370085986&title=Steam%20%E4%BB%8E%E6%98%8E%E5%B9%B4%E8%B5%B7%E5%B0%86%E5%81%9C%E6%AD%A2%E6%94%AF%E6%8C%81%20Windows%207%E3%80%818%20%E5%92%8C%208.1 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自临渊而立**

Valve 通过其支持文档宣布它的 Steam 客户端从明年 1 月 1 日起将停止支持 Windows 7、8 和 8.1。之后 Steam 客户端将停止在这些操作系统上运行。为了继续运行 Steam 以及运行通过 Steam 购买的游戏，用户需要更新到新版本的 Windows。Valve 解释说，Steam 客户端是基于 Google Chrome 的嵌入式版本，而 Chrome 在旧版本的 Windows 上不再工作。未来版本的 Steam 将需要 Windows 10 及以上版本。Google 今年初发布的 Chrome 109 是最后一个支持 Windows 7 和 Windows 8/8.1 的版本。
https://help.steampowered.com/zh-cn/faqs/view/4784-4F2B-1321-800A
