# 专业认可！再度上榜《CCSIP 中国网络安全产业全景册》

星阑科技

![](https://mmbiz.qpic.cn/mmbiz_gif/Cc8QqLUKOeiaFHTFtiatmEIxZQcXOHfyr6GOBM88IeMm28ybjSAHEJKicuQxPxN5L5NFZ5mza2NOnuokf9ant2fUQ/640?wx_fmt=gif)

近日，国内领先的网络安全行业媒体FreeBuf咨询正式发布《CCSIP（China Cyber Security Panorama）2022 中国网络安全行业全景册（第五版）》。**星阑科技凭借硬核产品技术能力、专业服务水平以及优秀市场表现力，实力入围该图册云原生安全（容器安全）、数据安全治理（解决方案）、API安全三大细分领域，技术和业务能力再获认可。**

![](https://mmbiz.qpic.cn/mmbiz_jpg/Cc8QqLUKOegsvmFOn82icMyFQlMllRicNSzBehiclcwWVkaMuuEzicQGFoKICMISQTLSCtWtg7iaGhZEOFiccwuBoA0Q/640?wx_fmt=jpeg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/Cc8QqLUKOegsvmFOn82icMyFQlMllRicNSPibO9p7Q1IicbWicAyzficvyXZXRjsOcXWMsYxkw1zkbqFuicOh2ERYgn8w/640?wx_fmt=jpeg)

![](https://mmbiz.qpic.cn/mmbiz_jpg/Cc8QqLUKOegsvmFOn82icMyFQlMllRicNSXy76fshprXEDEszzCjazvlN9BluLmj8koHxafd41to6Z97UHNdpupQ/640?wx_fmt=jpeg)

为了更好地为企业提供网络安全产品选型参考，帮助企业了解中国网络安全技术与市场的发展趋势，FreeBuf咨询启动《CCSIP2022中国网络安全行业全景册（第五版）》。本次全景册面向广大国内安全厂商，由FreeBuf咨询顾问、市场分析师、安全智库专家组成专业的评审团队，对安全厂商进行归类与评审。经FreeBuf咨询团队审核后，将符合要求的厂商信息入驻至《CCSIP 2022中国网络安全产业全景册（第五版）》，为企业安全建设及产品选型提供参考。

此次入选FreeBuf《CCSIP 2022 中国网络安全行业全景册（第五版）》，不仅是业内专业机构和专家对星阑科技的高度认可，更是对星阑科技发展的激励。

作为网络安全的新锐力量，星阑科技自成立以来就一直基于AI深度感知和强大的自适应机器学习技术，帮助用户迅速发现并解决面临的安全风险和外部威胁，公司创始团队深耕于信息安全行业多年，聚集了从清华、中科院、北美高校等知名研究人才以及亚马逊、阿里、明略科技等企业人才，秉承“智能安全、赋能业务”的理念，凭借世界一流的攻防能力和成熟的云原生体系技术栈，为客户带来高质量的安全产品和服务，用户遍及政府、金融、互联网、政府、能源等各大行业。

当前，数字化浪潮席卷而来，数字化安全风险愈加突出。对于企业而言，安全风险已经成为吞噬企业利润的最大潜在威胁。随着云计算、大数据、人工智能的蓬勃发展，越来越多的应用开发深度依赖于API之间的相互调用，API安全受到广泛重视。为解决API安全问题，星阑科技从攻防能力、大数据分析能力及云原生技术体系出发，提供全景化API识别、API高级威胁检测、复杂行为分析等能力，构建API Runtime Protection体系。创新研发的API安全产品“萤火”，可以从资产-风险-事件-数据等多个维度实现对API使用情况的实时监控、异常访问的可视化。对流量中的API自动聚合、分类、自定义标签化管理，并以可视化的方式便于管理员进行探索和调查，从而实现API安全统一监测。该产品可以充分适配传统IT架构及云原生容器化、微服务架构，全面覆盖企业API安全风险。

山积而高，泽积而长，未来，星阑科技将保持初心继续努力，紧跟网络安全市场发展趋势，聚焦不断升级的新需求，利用自身优势不断推陈出新，继续为企业网络安全保驾护航，为中国网络安全贡献力量。
