ж №жҚ®и¶ӢеҠҝз§‘жҠҖзҡ„жҠҘе‘ҠпјҢиҝҷж¬ҫж–°зҡ„еӢ’зҙўиҪҜд»¶дјјд№ҺжЁЎд»ҝдәҶеӢ’зҙўиҪҜд»¶з”ҹжҖҒзі»з»ҹдёӯзҡ„зҹҘеҗҚе“ҒзүҢLockBitпјҢд»ҘиҜҜеҜјеҸ—е®іиҖ…зӣёдҝЎд»–д»¬еҸ—еҲ°дәҶдёҖдёӘжӣҙзҹҘеҗҚзҡ„з»„з»Үзҡ„ж”»еҮ»гҖӮвҖңжҠҘе‘Ҡз§°пјҡвҖқеңЁж”»еҮ»зҡ„жңҖеҗҺйҳ¶ж®өпјҢи®ҫеӨҮзҡ„еЈҒзәёдјҡеҸҳжҲҗдёҖеј жҸҗеҲ° LockBit зҡ„еӣҫзүҮгҖӮйүҙдәҺ LockBit еЈ°еҗҚзӢји—үпјҢиҝҷз§Қзӯ–з•ҘдјҡеўһеҠ еҸ—е®іиҖ…зҡ„еҺӢеҠӣпјҢиҝ«дҪҝд»–д»¬ж»Ўи¶іиөҺйҮ‘иҰҒжұӮгҖӮ

з„¶иҖҢпјҢи¶ӢеҠҝз§‘жҠҖзҡ„еҲҶжһҗжҳҫзӨәпјҢиҝҷз§ҚеӢ’зҙўиҪҜд»¶е№¶йқһ LockBitпјҢиҖҢжҳҜдёҖз§ҚдҪҝз”Ёдәҡй©¬йҖҠ S3 еӨ–жі„ж•°жҚ®зҡ„ж–°з—…жҜ’гҖӮж”»еҮ»иҖ…еҲ©з”Ё AWS зҡ„ S3 Transfer Acceleration (S3TA) зӯүеҠҹиғҪе°ҶеҸ—е®іиҖ…ж•°жҚ®дёҠдј еҲ°д»–д»¬жҺ§еҲ¶зҡ„ S3 еӯҳеӮЁжЎ¶пјҢд»ҺиҖҢе®һзҺ°жӣҙеҝ«зҡ„иҝңи·қзҰ»ж•°жҚ®дј иҫ“гҖӮжҠҘе‘Ҡи§ЈйҮҠиҜҙпјҡвҖңS3TAдҪҝз”ЁжҲ·иғҪеӨҹе®һзҺ°жӣҙеҝ«зҡ„иҝңи·қзҰ»ж•°жҚ®дј иҫ“вҖҰвҖҰеӢ’зҙўиҪҜд»¶дҪҝз”ЁзЎ¬зј–з Ғзҡ„дёҖеҜ№еҮӯжҚ®еңЁж”»еҮ»иҖ…жҺ§еҲ¶зҡ„AWSиҙҰжҲ·дёҠеҲӣе»әдёҖдёӘдәҡй©¬йҖҠS3жЎ¶вҖқгҖӮ

иҜҘеӢ’зҙўиҪҜд»¶жҙ»еҠЁдёӯжңҖд»ӨдәәйңҮжғҠзҡ„еҸ‘зҺ°д№ӢдёҖе°ұжҳҜеңЁеӢ’зҙўиҪҜд»¶д»Јз Ғжң¬иә«дёӯдҪҝз”ЁдәҶзЎ¬зј–з Ғзҡ„ AWS еҮӯжҚ®гҖӮи¶ӢеҠҝз§‘жҠҖеҸ‘зҺ°пјҢиҝҷдәӣеҮӯжҚ®иў«з”ЁдәҺеҲӣе»ә S3 жЎ¶е’ҢеӨ–жөҒиў«зӣ—ж•°жҚ®гҖӮвҖңжҠҘе‘ҠжҢҮеҮәпјҡвҖқеӨ§еӨҡж•°ж ·жң¬йғҪеҢ…еҗ«зЎ¬зј–з Ғзҡ„ AWS еҮӯжҚ®пјҢиў«зӣ—ж•°жҚ®иў«дёҠдј еҲ°з”ұеЁҒиғҒиЎҢдёәиҖ…жҺ§еҲ¶зҡ„дәҡй©¬йҖҠ S3 еӯҳеӮЁжЎ¶гҖӮ

иҝҷз§ҚжҠҖжңҜдҪҝж”»еҮ»иҖ…иғҪеӨҹз»•иҝҮдј з»ҹзҡ„зҪ‘з»ңйҳІеҫЎзі»з»ҹпјҢеҲ©з”ЁеҗҲжі•зҡ„дә‘жңҚеҠЎжқҘзӘғеҸ–ж•°жҚ®гҖӮйҡҸзқҖдә‘еҹәзЎҖи®ҫж–Ҫи¶ҠжқҘи¶ҠжҲҗдёәдјҒдёҡдёҚеҸҜжҲ–зјәзҡ„дёҖйғЁеҲҶпјҢе®ғд№ҹдёәж”»еҮ»иҖ…жҸҗдҫӣдәҶж–°зҡ„еҲ©з”ЁйҖ”еҫ„гҖӮеӢ’зҙўиҪҜд»¶дёҚд»…дјҡеҠ еҜҶж–Үд»¶пјҢиҝҳдјҡе°Ҷж–Үд»¶дёҠдј еҲ°дә‘з«ҜпјҢдҪҝеҸ—е®іиҖ…жӣҙйҡҫжҒўеӨҚгҖӮ

иҜҘеӢ’зҙўиҪҜд»¶зҡ„дёҺдј—дёҚеҗҢд№ӢеӨ„еңЁдәҺе®ғзҡ„еӨҡе№іеҸ°жҖ§гҖӮе®ғдҪҝз”Ё Go зј–зЁӢиҜӯиЁҖпјҲGolangпјүзј–еҶҷпјҢиғҪеӨҹеҗҢж—¶й’ҲеҜ№ Windows е’Ң macOS зҺҜеўғгҖӮGolang зҡ„и·Ёе№іеҸ°иғҪеҠӣдҪҝеЁҒиғҒиЎҢдёәиҖ…иғҪеӨҹејҖеҸ‘еҸҜж„ҹжҹ“еӨҡз§Қзі»з»ҹзҡ„еӢ’зҙўиҪҜд»¶гҖӮи¶ӢеҠҝз§‘жҠҖзҡ„еҲҶжһҗжҠҘе‘Ҡејәи°ғдәҶиҝҷдёҖзӮ№пјҢжҢҮеҮәпјҡвҖңGolang дёәејҖеҸ‘дәәе‘ҳжҸҗдҫӣдәҶдёҖдёӘеҚ•дёҖзҡ„д»Јз Ғеә“пјҢеҸҜд»ҘдёәеӨҡдёӘдёҚеҗҢе№іеҸ°зј–иҜ‘дҫқиө–е…ізі»вҖқгҖӮ

иҝҷдҪҝеҫ—еӢ’зҙўиҪҜд»¶зҡ„з”ЁйҖ”жһҒе…¶е№ҝжіӣпјҢиҖҢдё”йҡҫд»ҘйҒҸеҲ¶пјҢеӣ дёәе®ғеҸҜд»ҘиҪ»жҳ“ең°еңЁдёҚеҗҢзҡ„ж“ҚдҪңзі»з»ҹе’ҢзҪ‘з»ңзҺҜеўғдёӯдј ж’ӯгҖӮ

иҷҪз„¶ AWS е·ІжҡӮеҒңиҜҘеӢ’зҙўиҪҜд»¶дҪҝз”Ёзҡ„и®ҝй—®еҜҶй’ҘпјҢдҪҶи¶ӢеҠҝз§‘жҠҖе»әи®®дјҒдёҡдҝқжҢҒиӯҰжғ•гҖӮйҷӨдәҶжӣҙж–°иҪҜд»¶еӨ–пјҢдјҒдёҡиҝҳеә”е®ЎжҹҘиҝңзЁӢи®ҝй—®зӯ–з•ҘпјҢзЎ®дҝқеҸӘжңүеҸ—дҝЎд»»зҡ„з”ЁжҲ·жүҚиғҪи®ҝй—®е…ій”®зі»з»ҹгҖӮ

жӯӨеӨ–пјҢи·ҹиёӘдёҺжҒ¶ж„Ҹжҙ»еҠЁзӣёе…іиҒ”зҡ„ AWS еёҗжҲ· ID д№ҹжҳҜдёҖдёӘжңүд»·еҖјзҡ„е…ҘдҫөжҢҮж Ү (IOC)гҖӮйҖҡиҝҮзӣ‘жҺ§иҝҷдәӣеёҗжҲ·ж ҮиҜҶз¬ҰпјҢдјҒдёҡеҸҜд»ҘжӣҙеҘҪең°жЈҖжөӢе’Ңеә”еҜ№жҪңеңЁзҡ„дә‘еЁҒиғҒгҖӮ
