# 【安全圈】Mozilla再次发文称禁止谷歌搜索向浏览器分成将威胁火狐等独立浏览器的生存

**关键词**

浏览器

谋智基金会(Mozilla)日前再次发文针对谷歌反垄断案进行评价，目前美国司法部有想法强迫谷歌出售 Chrome 浏览器以及禁止谷歌搜索通过向浏览器付费成为默认搜索引擎。

火狐浏览器等众多浏览器的主要营收来源就是谷歌，即这些浏览器通过将谷歌设置为默认搜索引擎从而获得谷歌提供的搜索分成，因此如果法院禁止谷歌再通过这种方式给浏览器付费，将对火狐等众多浏览器造成威胁。

在博客中谋智基金会强调自己始终都是竞争的捍卫者和改革的倡导者，旨在为数字市场创造公平的竞争环境，该基金会赞同美国司法部为改善消费者的搜索竞争所做的努力，但此案的结果将对市场产生重大影响。

按照美国司法部拟议的补救措施，火狐浏览器等规模较小的独立浏览器需要从根本上重新审视整个运营模式，而禁止谷歌搜索实施分成模式有可能在无意中加强少数强大参与者 (浏览器) 的地位，这样不仅不会对搜索竞争带来有意义的改进，而且还关乎到开放和可互操作网络的未来。

此前火狐浏览器曾与雅虎合作将 Yahoo Search 设置为默认搜索引擎，但用于大多数用户并不喜欢雅虎搜索以及后来雅虎搜索被爆出数据泄露问题，在 2017 年火狐浏览器重新与谷歌续签合同并将谷歌搜索设置为默认搜索引擎。

同时现在市场上虽然还有很多浏览器，但主流浏览器核心排版引擎仅有 3 个：Apple WebKit、Google Chromium Blink 和 Mozilla Firefox Gecko。

苹果的 WebKit 内核仅在苹果生态系统例如 macOS、iOS 和 iPadOS 中使用，谋智基金会认为当前能够提供跨平台挑战的有且仅有 Mozilla Gecko 内核 (该内核支持 Windows、macOS、Linux，也在开发 iOS 版)

谋智基金会为什么要提浏览器引擎呢？因为美国司法部拟议的做法可能会摧毁火狐浏览器的商业模式，如果火狐浏览器被摧毁那意味着市场上的浏览器引擎只有苹果 WebKit 和谷歌 Blink，而在 Windows 和 Linux 平台谷歌 Blink 内核为成为唯一的主宰。

这种情况下像火狐浏览器不仅会变成牺牲品，同时还会导致 Mozilla Gecko 引擎被摧毁，因此美国司法部拟议的做法还会再无意中损害浏览器与浏览器引擎之间的竞争。

谋智基金会不希望看到市场份额从一家市值万亿美元的科技公司转移到另一家科技公司，真正的改变是需要在搜索引擎、浏览器、浏览器引擎领域消除竞争并创造一个促进竞争、创新和消费者选择的市场。

最后该基金会敦促法院考虑采取补救措施以实现最初的反垄断目标，但不要损害独立浏览器、浏览器引擎，以免最终也会损害整个网络。
