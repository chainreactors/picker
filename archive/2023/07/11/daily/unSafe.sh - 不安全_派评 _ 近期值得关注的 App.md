## 不容错过的新 **App**

虽然少数派一直在为大家发现和介绍各平台上的优质 App，但仍有不少设计、功能、交互、体验都非常优秀的 App，还没有被我们发掘和介绍。它们可能是一款老 App，也可能是近期上架的新 App，我们会在这里介绍给你。

## **值得关注的 App**

### Threads：马斯克真的慌了？

* 平台：iOS / Android
* 关键词：社交网络

> 至截稿日，推特暂时放开了已经于此前关闭的 Tweetdeck 和 API 限制。

在社交应用市场，「文字分享社交平台」这一领域 Twitter 一直都是领头羊之一，领跑整个市场。 不过也不知道马老板怎么想的，开始限制用户每天刷推的时间。而在这个限制发布不久，吆喝了一段时间的 Meta 竞品，也就是 Threads ，横空出世了。

![DCb8byZ9moafvExUjmJcG6o3nYS](https://cdn.sspai.com/editor/u_/cilt9vdb34t9ooh06700?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

时间点非常巧，结合 Threads 很多适配都还没有做得很完美的情况，很难不认为这是扎老板看到 Twitter 对用户的限制后让底下的人加班加点推动 Threads 上线的。

就 Threads 的操作逻辑来看，说它是 Twitter 竞品一点问题没有。刷新后需要上滑到顶才能看到最新消息，线性的回复体系，甚至混乱的时间线以及偶尔突然出现在你时间线的不知名的女性（或男性）博主，都像是给 Twitter 换了一个皮肤然后放出来说「这是我们 Meta 自己做的社交软件，我们不知道什么是 Twitter」。

![](https://cdn.sspai.com/2023/07/10/b6aaf03f3b33f09a678adbe24ca56363.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

使用 Twitter 的时候，你的好友圈其实是与其他社交软件割裂开的，Facebook 与 Instagram 在一定程度上可以做到好友的互通，但是他们都与 Twitter 隔绝开来。到了 Threads 上，你甚至不用在意前期好友转移的问题，Meta 已经帮各位想好了完美的解决办法—— Instagram 好友即 Threads 好友。俩兄弟直接共享好友列表，无需有各种麻烦的跳转，只需要轻轻点一下「关注」按钮就能立刻拥有所有的 Ins 好友。

显然它还是不够完美。我前面也提到了，它目前更像是看到马老板的骚操作之后赶工放出的产品。有许多问题没有改善。首先要提到的就是自适应刷新率的问题。Android 版的 Threads 在最开始是没有适配自适应刷新率的，所以导致很多高刷手机用起来会感觉有点卡，手动将其拉到高刷后问题就解决了。其次，相对于 Twitter ，Threads 目前还未拥有网页端，对上班摸鱼来说还是不太方便的。还有一点，也是最影响使用体验的一点，就是目前没有像 Instagram 一样能够将时间线切换成「已关注的人」，时间线非常混乱。

作为 Instagram 的变体，Threads 的初期表现尚可。发布的几天内也在 Android 平台有着不少次更新。我的 Ins 好友也大部分都转移到了 Threads 的平台上活跃。当然这也可能是出自对一个大公司新产品的新鲜感。在 Twitter 日益收紧用户权限的今天，Threads 的开放态度让我觉得它是有趣的。所以起码这一场「八角笼格斗」中，我认为扎克伯格略胜一筹。

你可以在 [App Store](https://apps.apple.com/us/app/threads-an-instagram-app/id6446901002) 下载 Threads。

### Run 11：能够接入 GPT 的现代化 Windows 运行对话框

* 平台：Windows
* 关键词：GPT、系统组件

[@Noah\_Choi](https://sspai.com/u/69zxdqx8)：Windows 11 虽然采用了更加现代美观的 UI 设计，但由于各种各样的原因，系统中还是存在着很多看起来具有「年代感」的组件，比如我们经常会用到的「Win+R」运行功能。想要等待微软不断更新这些历史组件怕是遥遥无期，不如来看一下 Run 11 这款开源的第三方替代品。

Run 11 由 FireCube 工作室开发，它采用了与 Windows 11 更加匹配的 Fluent Design，看起来毫不违和。在功能方面，Run 11 支持打开系统功能和第三方应用程序，同时也能够调用管理员权限来打开程序，使用起来与 Windows 原生运行功能基本没有任何差异。

![OArDbXBSgosmDGxee2EcdmdcnTg](https://cdn.sspai.com/editor/u_/cilta05b34t9o1ls51ug?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

微软之前公布了 Windows Copilot 功能，也就是将 GPT 集成到了 Windows 系统中，前段时间的 Windows 11 预览版通道也针对这一功能进行了一个「微小」的更新，不过实际上只是套壳 Edge，并没有什么实质性功能。不过你可以借助 Run 11 曲线救国，在填写 OpenAI 的 API 密钥之后，你就能把 Run 11 当作一个类似于 Copilot 的助手。

![ZkJ8bHF7roZAzCxlyzVcfF5Xnyb](https://cdn.sspai.com/editor/u_/cilta0db34t9og6cs1o0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

你可以直接在输入框中让它「打开设置」「打开网络」甚至是「打开微信」。不过这个功能也有一定的限制，比如我们修改了应用程序的安装位置，那么 Run 11 还是会尝试在默认路径下寻找（通常是 Program Files 文件夹）；另外，在中文环境下可能不会识别的那么准确。

Ruh 11 直接顶替 Windows 原生系统组件，并且由于 UI 更加现代，我使用 Windows 11 原生运行组件时反而会有割裂感。如果你也想替换掉原生运行组件，可以在 [GitHub](https://github.com/FireCubeStudios/Run)、[Microsoft Store](https://www.microsoft.com/store/apps/9NQGV64S5136) 免费下载安装，或使用 `winget install "Run by FireCube`" 安装。

### 50 音起源：现代化 Android 平台特性适配

* 平台：Android / iOS
* 关键词：日语、学习、外语

[@克莱德](https://sspai.com/u/clyde)：在最近更新的版本中，日语学习工具《50 音起源》对「边到边」、主题图标以及小组件三大平台特性进行了适配。

**关联阅读：**

* [干掉「毒瘤」只是第一步：怎样的 Android 应用才算「好应用」？](https://sspai.com/post/65930)
* [具透 | 下个 Android 大版本更新，Google 想为你「众筹」一套图标包](https://sspai.com/post/74800)
* [Google 做不好的「小组件」，MIUI 做对了什么？ ｜ 少数派会员 π+Prime](https://sspai.com/prime/story/android-widget-problems)

其中「边到边」为兼容性适配，主要确保了应用在系统深色主题开启时导航栏背景能够正确应用配色（而不是黑漆漆的背景），主题图标特性则面向部分保留了该特性的操作系统，如 Google Pixel、三星 One UI、摩托罗拉 MYUI 等，开启后桌面启动器的应用图标会根据 Material You 动态取色方案保留单色应用图标，保证主屏的整体视觉观感。

![](https://cdn.sspai.com/2023/07/10/b9f42ebb18d14d600080e326bb5dc9df.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

主题图标与深色主题下的「边到边」特性适配

小组件方面，《50 音起源》Android 版新增了 2×2 和 4×2 两种网格尺寸的可选组件，前者为日期及对应日语表达方式卡片，方便我们在查看当前日期的同时学习对应日期的日语表达方式，后者则是包含词汇和日期的复合内容卡片；功能方面，目前《50 音起源》的 4×2 桌面小组件以信息展示和单词卡片识记功能为主，每 15 分钟还会自动切换新单词。

值得一提的是，由于开发者直接选择了当下最新的 Jetpack Compose Glance 进行构建，两款小组件均拥有更为标准的布局和尺寸，放在主屏上时无论圆角、边距等细节都没有瑕疵。

![ND9Jb1l3xovUcux9lAcc1DoTnCc](https://cdn.sspai.com/editor/u_/cilta1lb34t9oqlqkvf0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

《50 音起源》的可调节小组件

同时新 API 自然对 Android 平台的最新特性也提供了完整支持。《50 音起源》的 4×2 小组件在摆放至桌面后能够根据系统深色、浅色主题设置自动切换样式，可以根据小组件摆放位置的桌面背景色彩动态调整小组件文本颜色，同时还支持尺寸调节——在调节尺寸的过程中，小组件内部所呈现的内容和内容排版都会根据当前尺寸进行动态调整。

你可以在 [Play 商店](https://play.google.com/store/apps/details?id=com.kevinzhow.kanaoriginlite)和国内各大安卓应用商店下载《50 音起源》。

### Opera One：老牌浏览器全新升级

* 平台：macOS / Windows / Linux
* 关键词：浏览器

[@化学心情下2](https://sspai.com/u/liuxiaofengone)：虽然 Opera 最近将浏览器品牌升级成了 Opera One，但其本质上依旧是原来的主品牌浏览器——其实从关于页面不难看出他只是 Opera 100 换了名字而已。但对于我这种有近乎十年没有使用 Opera 的用户而言，Opera One 带给我的新鲜感可不仅仅只是换个名字那么简单。

首先就是那个从十年前就存在的左边侧边栏，在十年前的 Opera 上，左边侧边栏中就可以实现类似的 widget 小部件方便在上网浏览时使用，而现在这个侧边栏则可以帮我快速打开 Twitter、Instagram 等——与其在桌面系统上通过 PWA 将其安装到系统中，到不如让这些社交服务直接在自己上网冲浪时顺手刷来得更方便。

![M8JLbgDqAoUXKIx8Jyyc6zrpnBd](https://cdn.sspai.com/editor/u_/cilta25b34t9oqlqkvfg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

而侧边栏中除了集成这些社交应用之外，还可以连接在线流媒体服务，比如在 Opera One 可以直接点击侧边栏上的播放器按钮来边上网边听歌——虽然这依旧是调用这些流媒体服务的网页版来实现，但至少省去了安装桌面客户端打开的麻烦。

![K56abTo06osXZQxNnpOcmbQpnpb](https://cdn.sspai.com/editor/u_/cilta2lb34t9ooh0670g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

AI 则依旧是这一次 Opera 品牌升级的热点，Opera 这一次直接在侧边栏集成了 AI 助手入口，并且相比于存在较高使用门槛的 ChatGPT，更是直接加入了其自家品牌的 AI：Aria，并且不受制于网络环境就可以免费使用，而实际使用上和 ChatGPT 类似，以对话的方式进行交互，信息回答的准确性上还是相当不错的。

![BoyCbuNSLofzSwxUatlcuTPvnHg](https://cdn.sspai.com/editor/u_/cilta2tb34t9oqlqkvg0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

除了左侧栏带来了一系列的功能创新之外，作为历史上首个推出多标签页浏览的浏览器，Opera One 也对多标签页浏览进行了诸多创新，加入了「标签页岛」的概念，对于基于同网站中打开的分页标签页会自动加入到相同的「标签页岛」中，不同的标签页岛通过「岛」前面的色块进行标记，点击色块就可以实现同一个「标签页岛」的标签展开或者隐藏，这样设计的最大好处在于可以实现标签页的快速分类，更好地找到需要浏览的标签页。

![QtqXbSPwuoNxS3xqD5acytBInRW](https://cdn.sspai.com/editor/u_/cilta3db34t9oqlqkvgg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

在 Opera One 中，整个 UI 被分成了多个模块，整体 UI 会根据整体的页面内容和功能变化动态地进行调整。同时针对页面切换上的动态效果进行了重新设计，整体操作和使用流程上会从视觉上看上去更「流畅平滑」。

![](https://cdn.sspai.com/2023/07/10/ae97f3af3b33e1d7e7d3d7fc64d02861.gif)

总的来说，对于新品牌下的 Opera One 浏览器，你依旧可以看到诸多创新的功能点，比如对于多标签页浏览的优化，充分利用的侧边栏来辅助进行浏览操作，如果你是一个纯粹的视觉体验党，那么这个涅槃重生的老牌浏览器绝对值得一试，你可以在其[官网](https://www.opera.com/zh-cn/one)上免费下载使用。

### 专注旅人 2.0：专注地「登上」一座山

* 平台：iOS / iPadOS
* 关键词：专注

[@Snow](https://sspai.com/u/ul0vcezb)：《专注旅人》是一款采用精致手绘风的专注类应用，每一次专注过程，都被它描绘成一次登山之旅。这儿没有什么蛇虫鼠蚁激流勇进，而是拥抱大自然的惬意徒步，在自然声效与舒缓音乐中，缓缓登上山巅，在这期间你说不定还能捕捉夜空流星，或是获得林间萌物的陪伴。回首望去这一路的坚持，似乎没有那么艰辛了。

在开启旅程前，你需要选择一位登山旅人并为他/她/它命名。接着设定专注目标、专注时间、季节场景和专注模式即可开启登山之旅。应用提供了「一般专注」和「深度专注」两种模式，前者在你将应用置于后台时仍会继续登山之旅，后者则会立刻暂停开始烤火。在登山期间，你也可以随时点击屏幕暂停，此时旅人会就地搭起篝火，取暖的同时顺便吃点东西补充能量。长按屏幕，当右上角「✓」周围一圈进度条跑满，即可结束登山之旅，完成的进度会体现在地区上，上下滑动地图你可以清晰地了解自己目前的进度。

![SlTcbEEUUoFecQx8jcBcUJYEnPh](https://cdn.sspai.com/editor/u_/cilta55b34t9ooh06710?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

在旅程之中，点击左上角的音乐按钮，你还可以自定义声效与背景音乐，专注旅人支持连接 Apple Music 服务，让你可以选择自己的音乐作为登山伴侣。应用提供了 4 种自然音效，包括森林、溪流、雷雨以及夏夜蝉鸣，音效本身质量还可以，但与应用本身提供的山路和季节场景设计存在一定冲突，毕竟在雪山上听见蝉鸣声声是有一些诡异的吧，希望开发者后续能再做优化。

![YANwbGAp9oB99nxTL6ec7l2mnRf](https://cdn.sspai.com/editor/u_/cilta5db34t9oqlqkvh0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

专注旅人还支持多人专注，应用可以利用 Game Center 邀请好友一起参与登山之旅。如果你像我一样没有好友，则可以通过 iMessage 邀请加入，不过应用目前尚未开通其他邀请...
