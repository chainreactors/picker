## 微软高管称语音将成为下一代 Windows 的主要输入方式

[![输入设备](https://icon.solidot.org/images/topics/topicinputdevices.png?123)](/search?tid=79 "输入设备")

[Edwards](/~Edwards) (42866)发表于 2025年08月15日 17时11分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=82057&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E9%AB%98%E7%AE%A1%E7%A7%B0%E8%AF%AD%E9%9F%B3%E5%B0%86%E6%88%90%E4%B8%BA%E4%B8%8B%E4%B8%80%E4%BB%A3%20Windows%20%E7%9A%84%E4%B8%BB%E8%A6%81%E8%BE%93%E5%85%A5%E6%96%B9%E5%BC%8F "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自诺比与错乱的时间线**

微软 Windows 部门主管 Pavan Davuluri 在一则公司视频中表示，随着 AI 彻底改变用户与计算机的交互方式，下一代 Windows 将变得“更具环境感、更普适、更多模态（more ambient, pervasive, and multi-modal）”。Davuluri 表示，语音将与键盘和鼠标一起成为 Windows 主要的输入方式，操作系统将具有情境感知功能，通过自然语言理解屏幕内容和用户意图。他表示，随着平台日益智能化，Windows 界面将在五年内发生根本性的变化。这种转变将依赖于本地处理能力和云计算能力去提供无缝的体验，用户可以在打字或墨迹式画图的同时与计算机对话。
