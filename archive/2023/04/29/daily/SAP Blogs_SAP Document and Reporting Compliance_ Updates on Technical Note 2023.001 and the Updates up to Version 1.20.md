Hi everyone,

The SAP Document and Reporting Compliance, inbound and outbound invoicing options for Brazil team has released updates on mapping and validation rules for XML files.

![](/legacyfs/online/storage/blog_attachments/2023/04/image-min.png)

SAP Document and Reporting Compliance: Updates on Technical Note 2023.001

This release complies with SEFAZ **Technical Note 2023.001 and the updates up to version 1.20**. The update changed the XML layout by adding new fields in the Fuel group. It also created new ICMS tax groups and added ICMS fields in the Total group.

You can implement these changes in the inbound invoicing option and outbound invoicing option. Follow the instructions on the following SAP Notes.

If you use **outbound invoicing option for Brazil**:

* SAP Note [3311653](https://launchpad.support.sap.com/#/notes/3311653)

* SAP Note [3311652](https://launchpad.support.sap.com/#/notes/3311652)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

If you use **inbound invoicing option for Brazil**:

* SAP Note [3317013](https://launchpad.support.sap.com/#/notes/3317013)

* SAP Note [3319139](https://launchpad.support.sap.com/#/notes/3319139)

* SAP Note [3325410](https://launchpad.support.sap.com/#/notes/3325410)

* SAP Note [3328353](https://launchpad.support.sap.com/#/notes/3328353)

The new version will be effective on May 1, 2023, according to the tax authority.

For a step-by-step guide on how to use the SAP Document and Reporting Compliance, inbound and outbound invoicing option for Brazil, refer to the following SAP Help Portal pages:

* [SAP Document and Reporting Compliance, inbound invoicing option for Brazil](https://help.sap.com/docs/DC_NFe/75478ef83fbe47b6836f33609268796b/ffaf0ea3291d4ddd9be5e5ba2c641e27.html)

* [SAP Document and Reporting Compliance, outbound invoicing option for Brazil](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/b2f5ad5578574616855dda8638e2ee92.html)

You can implement the changes in the SAP Electronic Invoicing for Brazil (SAP NFE) solution. Follow the instructions on the following SAP Notes.

If you use **SAP NFE outbound in the higher releases**:

* SAP Note [3311653](https://launchpad.support.sap.com/#/notes/3311653)

* SAP Note [3311652](https://launchpad.support.sap.com/#/notes/3311652)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

If you use **SAP NFE outbound in the lower releases**:

* SAP Note [3315005](https://launchpad.support.sap.com/#/notes/3315005)

* SAP Note [3314894](https://launchpad.support.sap.com/#/notes/3314894)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

If you use **SAP NFE inbound in higher and lower releases**:

* SAP Note [3317026](https://launchpad.support.sap.com/#/notes/3317026)

* SAP Note [3325411](https://launchpad.support.sap.com/#/notes/3325411)
