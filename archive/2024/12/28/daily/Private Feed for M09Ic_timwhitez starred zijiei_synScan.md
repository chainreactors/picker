# synScan

GoLang 全端口 **syn** 扫描
自定义TCP/IP头
syn扫描需要监听网卡，如果报错可能是权限问题

```
sudo ./masscan [ip] [rate]  // rate不填默认1500 速率取决于网络和网卡太大容易丢包
```
