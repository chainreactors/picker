# 如何调查一个人的社交媒体账户

原创

001

情报分析师

在数字时代，社交媒体已成为我们日常生活的一部分。我们分享生活点滴、表达观点、与朋友互动。但你知道吗，这些在线行为不仅仅是简单的信息交流，它们还能揭示很多信息。

本文将深入了解一个人的社交媒体账户如何成为信息分析的窗口。该分析还可能揭示个人潜在的行为、人物关系等。通过这种方式，可以更深入地理解一个人并预测他们的下一步行为。

**一、用户名分析**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XlbJQsU3PsgQnYtWq4Ny3eWOVUWNBqtX8RbTJsuYFw7jicIUwjdA20Ow/640?wx_fmt=png&from=appmsg)

有时，一个用户名便是我们探索的起点，它引领我们深入挖掘背后的故事。举个例子，在新罕布什尔州，一位共和党州众议员正是通过这样的方式，成为了 Reddit 上一个极具争议的男性社区的创始人。这个社区名为 The Red Pill，而揭开其背后架构师身份的调查，正是从用户名“pk\_atheist”开始的。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0X3J883Jymh2dLNctpuvsmue4L3eaahGz0ZaUXLYuowvJHW9ic4kqCyWw/640?wx_fmt=png&from=appmsg)

在数字世界中，用户名成为了个人身份的一种标识。一些人倾向于在不同的平台和电子邮件服务中保持用户名的一致性，仅做微小的调整。

然而，对于那些重视网络安全的人士，如新罕布什尔州的某位代表，他们则倾向于在每次注册新服务时都采用全新的用户名，并在不再使用时果断放弃。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XwIXicgaGI7lveAArWr1bwGBONUlIZ5IVVSqCe6qvv2KT2bdhHINSeew/640?wx_fmt=png&from=appmsg)

尤其是那些避开主流社交媒体的人，他们可能会在一些不太显眼的地方留下踪迹，比如评论区域、论坛帖子等，这些线索可能会引导发现更多的信息和其他相关账户。

除了搜索引擎搜索，还可以利用一些付费的专业服务。这些服务可能需要一定的费用。大多数机构都会使用 Nexis，它在查找公共记录和法庭文件方面非常有效，但遗憾的是它在电子邮件/用户名搜索方面表现不佳。Nexis 对于专注于美国的研究尤其有用。

Pipl 和 Spokeo 是发现的非常有用的工具之一，它们能够将电话号码、财产记录等现实世界的信息与电子邮件和用户名等在线信息进行交叉对比，而且它们的服务范围覆盖全球。这些付费搜索引擎通常提供电话和财产记录，但它们也能识别出即使账户已经关闭仍然存在的 Facebook 和 LinkedIn 个人资料。它们还能连接到人们可能已经遗忘的账户，比如旧博客，甚至是亚马逊愿望清单——这些都是了解一个人的兴趣、购买习惯和愿望的宝贵资源。

当然，这些服务也可能会提供一些错误的信息，所以我通常会从它们的结果开始调查，然后使用其他方法进行进一步的验证。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XIibywghab6kGpnTJnluUyc4XNrETMictdF0d55aicWlztBO7P2TFVv9Zg/640?wx_fmt=png&from=appmsg)

当遇到一个可能与研究的对象相关的用户名或电子邮件地址时，会利用一些在线工具，比如 namechk 或 namecheckr，来检查这个用户名在不同平台上的注册情况。

这些工具原本是为了方便营销人员快速查看他们想要注册的用户名是否在多个平台上可用。然而，它们同样适用于这样的调查工作，帮助确认正在调查的用户名是否在其他平台上也有注册。需要明确的是，一个用户名在多个平台上的存在并不直接意味着这些账户都属于同一人。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XFDYHGKe07dcMNaas6ibhp9pIWxpspOOj5xn2kK3ymxI1O2u4tngLvDw/640?wx_fmt=jpeg&from=appmsg)

为了深入挖掘用户名信息，可以利用 haveibeenpwned.com 和 Dehashed.com 这样的网站。这些平台通过搜索数据泄露事件中的数据，帮助用户快速验证电子邮件地址，并可能发现新的潜在关联账户。这种方法为进一步的用户信息核查提供了便捷途径。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0X34IYW5azWeF4FRIR1gf6IUib29rbl1G8kDbmBtbOXjguRmpNmuQ8yrA/640?wx_fmt=png&from=appmsg)

**二、发布照片分析**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XlbJQsU3PsgQnYtWq4Ny3eWOVUWNBqtX8RbTJsuYFw7jicIUwjdA20Ow/640?wx_fmt=png&from=appmsg)

当用户名不足以提供充分信息时，图片往往能提供更有力的证据。个人头像是验证用户在不同账户间身份的另一种有效方法。

这里就要提一下反向图片搜索功能。

谷歌的反向图片搜索功能非常强大，但有时其他搜索引擎，尤其是俄罗斯的 Yandex，可能会提供更精确的搜索结果。

使用 Reveye 这款 Chrome 浏览器扩展，它允许通过右键点击图片，在包括 Google、Bing、Yandex 和 Tineye 在内的多个搜索引擎上进行图片匹配搜索。Reveye 的图片搜索扩展还具备一个便捷的功能，可以从图片中提取特定元素进行搜索，这为图像识别提供了更多可能性。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0X0U3fumb3N4P5fziaz2TKX30phPgM9v3dNo2mTn0DsAba3TicKH0Q6ONA/640?wx_fmt=png&from=appmsg)

尽管反向图像搜索是一个强大的工具，但它也有其局限性。

例如，上述搜索引擎在从X抓取图片方面表现不佳，对于从 Instagram、Facebook 等社交平台获取搜索结果也几乎无能为力。

在进行身份验证时，经常遇到人们在不同时间点的形象差异。

在这种情况下，识别照片中的特定特征，如痣、面部毛发或其他独特标志，就显得尤为重要。

Face++ 这样的面部识别工具来辅助判断，它允许用户上传两张照片，然后计算它们属于同一人的概率。在一些案例中，这个工具能够在相隔十年的照片中成功识别出同一个人。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XdbDynEg8iaesakmKsZQHgkhOxEyWiczT7v7LMA9gPJAGTzwOEzic16ycQ/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XdeV5VNFeRuibjx6bhW3MiamaKhnw9Kcf1YcvTmibSGtwegXibyPibRFkRAw/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0X5Ebag0I0VMzCYwFBOZDsmib4rduroNpyypHUb0xE7qwVJ4VDo0hdx3Q/640?wx_fmt=png&from=appmsg)

在追踪分析一个人的社交媒体账号时，可能会注意到他们通常会更加努力地隐藏自己的个人资料照片，或者他们可能会使用伪造的照片。在这种情况下，对照片进行编辑和翻转可能有助于逆向追踪他们的行为模式。

然而，不仅仅是个人头像可以提供线索。随着人们对个人及家庭隐私的日益重视，他们仍然乐于分享那些他们感到自豪的事物的照片。通过分析汽车、房屋或宠物等物品的照片来识别人物。从这个角度来看，照片成为了连接账户和背后人物的桥梁，帮助你围绕目标构建起一个网络。这是社交媒体账户调查中的关键步骤。

举个例子吧，曾经试图确认一名男子的社交账户，因他曾在俄亥俄州代顿市的一家酒吧外枪杀了9人。他的X账户透露了他的意识形态，但他的昵称@iamthespookster是独一无二的，与警方公布的他的真名并不相符。

事实上，他的一个受害者是他的跨性别兄弟姐妹，这个信息并未出现在公共记录中，也没有向外界公开，这使得确定关键人物的身份变得更加困难。但是，在他的个人资料中，有一张宠物狗的照片，而这只宠物狗也出现在他未被报道的跨性别兄弟的账户横幅图片上。这个发现帮助我们建立了两个账户之间的联系。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XJLAlBNOialeTXkybMWGAWj9S7uic52CickodDEEjppRSsJFQxIpVRVurw/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XJ4iceLFDPLz3VkbvfLiamicvBVUzIFPBZneprkv9vxMtmoz6CI2y7o1sA/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XcZicDs1DWaIWlpLiaDTfAjxCGtiap4oficSYJ4pc0TZrlxLxsyKLiaFReUw/640?wx_fmt=png&from=appmsg)

在调查过程中，狗并不是唯一能提供帮助的细节。那张图片实际上是俄亥俄州枪手的父亲所发布的，帮助核实了枪手的个人账户以及他家人的账户。

如果在社交媒体上有账户，有时候甚至能看出你的出生日期，即使你没有在个人资料中公开分享，或者自己发布过。验证社交媒体账户的一个有效方法是浏览可疑账户在相应月份和日期的帖子，寻找生日祝福的迹象。

即使他们自己的页面上没有发布任何内容，父母也经常会在他们的页面上发布关于孩子生日的信息。这种方法可以帮助确认账户的真实身份。

**三、人物关系分析**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XlbJQsU3PsgQnYtWq4Ny3eWOVUWNBqtX8RbTJsuYFw7jicIUwjdA20Ow/640?wx_fmt=png&from=appmsg)

一个人在社交媒体上的联系网络可以揭示他们的个性和偏好。通过观察一个人在网上互动的人群，可以对他们的生活方式和倾向有所了解。

在调查一个人的社交媒体账户时，一定会考虑到与个人联系的相关性。社交媒体平台算法会推荐一些账户让关注。这些推荐受到手机联系人列表、在现有账户中的联系人、位置和其他因素的影响。

因此，查看一个账户最早的关注者和好友总是能提供一些有价值的信息。这是一个有用的工具，可以帮助探索大型账户之间的联系，以及下载小型账户的时间线和收藏夹等信息。对于处理更大规模的数据集，我通常会依赖那些有 API 访问权限的开发人员的帮助。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0Xl1SwC4Veh0F0qvibfD0OYLLBRibGj4ocf468wiaLsSOJnIpQ8ibeREEWqw/640?wx_fmt=png&from=appmsg)

以The Columbia Bugle为例，这是一个在X上广受欢迎的匿名极右翼账户，它自豪地宣称自己的推文曾被唐纳德·特朗普的官方账号转发过两次。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0X5PCKfUqe9MjUVnAMCmqUjibfibEIz5gjDfH2SIGciapWpvA0lAWWCicT6A/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0Xy2Y7TYP3UkePj9wuYr6ECX7Oz6HQxuNuq4kZTbmIwC7LrpQV4oicib7Q/640?wx_fmt=png&from=appmsg)

另外一个例子，以 Max Delarge 为例，这是一个自称是 The Columbia Bugle 编辑的X账户。Max Delarge 最初关注的是一些特定的圣地亚哥新闻来源和体育账户。考虑到 The Columbia Bugle 发布的许多推文都包含了来自圣地亚哥特朗普集会和加州大学圣地亚哥分校活动的视频，可以合理推测，运营这个账户的人很可能居住在圣地亚哥地区。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmVMhKRia32ibsRoGu0TAutF0XeNxJRDSrWE1DB6jEicxnw8tmSfLOAa2tdvFrrpRJlRcO0rI7rPnrHfA/640?wx_fmt=png&from=appmsg)
