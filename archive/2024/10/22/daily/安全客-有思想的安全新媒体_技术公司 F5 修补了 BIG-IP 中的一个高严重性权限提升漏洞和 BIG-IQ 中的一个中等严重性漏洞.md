![]()

F5 и§ЈеҶідәҶ BIG-IP е’Ң BIG-IQ дјҒдёҡдә§е“Ғдёӯзҡ„дёӨдёӘжјҸжҙһпјҢеҲҶеҲ«и·ҹиёӘдёә CVE-2024-45844 е’Ң CVE-2024-47139гҖӮ

жӢҘжңүз»ҸзҗҶи§’иүІжҲ–жӣҙй«ҳжқғйҷҗзҡ„з»ҸиҝҮйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҸҜеҲ©з”Ё CVE-2024-45844 жјҸжҙһжҸҗеҚҮжқғйҷҗе№¶е…Ҙдҫө BIG-IP зі»з»ҹгҖӮ

вҖңжӯӨжјҸжҙһеҸҜиғҪе…Ғи®ёе…·жңүз»ҸзҗҶи§’иүІжқғйҷҗжҲ–жӣҙй«ҳжқғйҷҗгҖҒеҸҜи®ҝй—®й…ҚзҪ®е®һз”ЁзЁӢеәҸжҲ– TMOS Shell (tmsh) зҡ„з»ҸиҝҮйӘҢиҜҒзҡ„ж”»еҮ»иҖ…жҸҗеҚҮжқғйҷҗе№¶жҺ§еҲ¶ BIG-IP зі»з»ҹгҖӮиҜҘе…¬е‘Ҡз§°пјҡвҖқжІЎжңүж•°жҚ®е№ійқўжјҸжҙһпјӣиҝҷеҸӘжҳҜдёҖдёӘжҺ§еҲ¶е№ійқўй—®йўҳгҖӮ

иҜҘе…¬еҸёеҸ‘еёғзҡ„ 17.1.1.4гҖҒ16.1.5 е’Ң 15.1.10.5 зүҲжң¬и§ЈеҶідәҶиҝҷдёҖжјҸжҙһгҖӮ

дёәзј“и§ЈиҝҷдёҖй—®йўҳпјҢдјҒдёҡеә”е°ҶеҜ№ BIG-IP й…ҚзҪ®е®һз”ЁзЁӢеәҸе’Ң SSH зҡ„и®ҝй—®йҷҗеҲ¶еңЁеҸ—дҝЎд»»зҡ„зҪ‘з»ңжҲ–и®ҫеӨҮдёҠпјҢе№¶йҳ»жӯўйҖҡиҝҮиҮӘ IP ең°еқҖзҡ„и®ҝй—®гҖӮ

иҜҘе…¬е‘ҠиҝҳжҢҮеҮәпјҡвҖңз”ұдәҺиҝҷз§Қж”»еҮ»жҳҜз”ұз»ҸиҝҮйӘҢиҜҒзҡ„еҗҲжі•з”ЁжҲ·е®һж–Ҫзҡ„пјҢеӣ жӯӨжІЎжңүеҸҜиЎҢзҡ„зј“и§ЈжҺӘж–Ҫе…Ғи®ёз”ЁжҲ·йҖҡиҝҮ SSH и®ҝй—®й…ҚзҪ®е®һз”ЁзЁӢеәҸжҲ–е‘Ҫд»ӨиЎҢгҖӮе”ҜдёҖзҡ„зј“и§ЈжҺӘж–ҪжҳҜеҸ–ж¶ҲдёҚе®Ңе…ЁеҸ—дҝЎд»»з”ЁжҲ·зҡ„и®ҝй—®жқғйҷҗгҖӮеңЁе®үиЈ…еӣәе®ҡзүҲжң¬д№ӢеүҚпјҢеҸҜд»ҘдҪҝз”Ёд»ҘдёӢйғЁеҲҶдҪңдёәдёҙж—¶зј“и§ЈжҺӘж–ҪгҖӮиҝҷдәӣзј“и§ЈжҺӘж–Ҫе°ҶйҖҡиҝҮ SSH еҜ№ BIG-IP й…ҚзҪ®е®һз”ЁзЁӢеәҸе’Ңе‘Ҫд»ӨиЎҢзҡ„и®ҝй—®йҷҗеҲ¶еңЁд»…еҸ—дҝЎд»»зҡ„зҪ‘з»ңжҲ–и®ҫеӨҮпјҢд»ҺиҖҢйҷҗеҲ¶дәҶж”»еҮ»йқўгҖӮвҖқ

* йҳ»жӯўйҖҡиҝҮз®ЎзҗҶз•Ңйқўи®ҝй—®й…ҚзҪ®е®һз”ЁзЁӢеәҸе’Ң SSHгҖӮ
* йҳ»жӯўйҖҡиҝҮиҮӘиә« IP ең°еқҖи®ҝй—®й…ҚзҪ®е®һз”ЁзЁӢеәҸе’Ң SSH

иҜҘе…¬еҸёи§ЈеҶізҡ„з¬¬дәҢдёӘй—®йўҳжҳҜдёҖдёӘеӯҳеӮЁзҡ„и·Ёз«ҷи„ҡжң¬ (XSS) жјҸжҙһпјҲи·ҹиёӘзј–еҸ·дёә CVE-2024-47139пјүпјҢе®ғдјҡеҪұе“Қ BIG-IQ жјҸжҙһгҖӮжӢҘжңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„ж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҝҷдёӘжјҸжҙһпјҢд»ҘеҪ“еүҚзҷ»еҪ•з”ЁжҲ·зҡ„иә«д»ҪиҝҗиЎҢ JavaScriptгҖӮ

вҖңз»ҸиҝҮйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҸҜйҖҡиҝҮеңЁ BIG-IQ з”ЁжҲ·з•ҢйқўдёӯеӯҳеӮЁжҒ¶ж„Ҹ HTML жҲ– JavaScript д»Јз ҒжқҘеҲ©з”ЁжӯӨжјҸжҙһгҖӮеҰӮжһңж”»еҮ»жҲҗеҠҹпјҢж”»еҮ»иҖ…е°ұеҸҜд»ҘеңЁеҪ“еүҚзҷ»еҪ•з”ЁжҲ·зҡ„дёҠдёӢж–ҮдёӯиҝҗиЎҢ JavaScriptгҖӮвҖқе…¬е‘ҠдёӯеҶҷйҒ“пјҢвҖңеҰӮжһңжҳҜжӢҘжңүй«ҳзә§ Shell (bash) и®ҝй—®жқғйҷҗзҡ„з®ЎзҗҶз”ЁжҲ·пјҢж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁжҲҗеҠҹеҲ©з”ЁжӯӨжјҸжҙһжқҘе…Ҙдҫө BIG-IP зі»з»ҹгҖӮиҝҷжҳҜдёҖдёӘжҺ§еҲ¶е№ійқўй—®йўҳпјҢдёҚеӯҳеңЁж•°жҚ®е№ійқўжјҸжҙһгҖӮвҖқ

BIG-IQ йӣҶдёӯз®ЎзҗҶзүҲжң¬ 8.2.0.1 е’Ң 8.3.0 и§ЈеҶідәҶиҝҷдёҖжјҸжҙһгҖӮ

дёәеҮҸе°‘жјҸжҙһпјҢз”ЁжҲ·еә”еңЁдҪҝз”Ё BIG-IQ з•ҢйқўеҗҺжіЁй”Җе№¶е…ій—ӯжөҸи§ҲеҷЁпјҢе№¶дҪҝз”ЁеҚ•зӢ¬зҡ„жөҸи§ҲеҷЁиҝӣиЎҢз®ЎзҗҶгҖӮзӣ®еүҚиҝҳжІЎжңүе·ІзҹҘзҡ„жјҸжҙһеҲ©з”Ёж–№жі•гҖӮ

зӣ®еүҚиҝҳдёҚжё…жҘҡиҝҷдәӣжјҸжҙһжҳҜеҗҰе·ІеңЁйҮҺеӨ–иў«еҲ©з”ЁгҖӮ
