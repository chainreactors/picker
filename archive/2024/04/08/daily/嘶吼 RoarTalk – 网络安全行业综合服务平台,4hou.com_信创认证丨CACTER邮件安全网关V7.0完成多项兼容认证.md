# 信创认证丨CACTER邮件安全网关V7.0完成多项兼容认证

导语：祝贺！CACTER邮件安全网关系统V7.0与中科可控、麒麟软件、统信软件、达梦数据、人大金仓、鲲鹏、飞腾、龙芯和海光等等众多国产服务器、操作系统、数据库和CPU，成功完成兼容适配认证工作。

随着信息技术的蓬勃发展，信息安全的重要性日益凸显。与此同时，国产操作系统和芯片技术日益成熟，并广泛应用于多个行业信息化建设中。为了契合我国信息化建设中硬件与操作系统的创新发展需求，CACTER邮件安全网关积极响应信创号召，致力于全面适配信创生态，为信息安全保驾护航。

近期，由广东盈世计算机科技有限公司自主研发的**CACTER邮件安全网关系统V7.0，已成功完成与中科可控、麒麟软件、统信软件、达梦数据、人大金仓、鲲鹏、飞腾、龙芯和海光等等众多国产服务器、操作系统、数据库和CPU的兼容适配认证工作**。经过严格的测试，结果显示CACTER邮件安全网关系统V7.0在上述多种数据库、服务器、操作系统和CPU环境下运行表现出色，稳定可靠，符合企业级生产可用标准。

此次兼容适配认证的顺利完成，充分展现了CACTER邮件安全网关系统V7.0的卓越性能，切实满足了不同行业客户的高标准需求，能够为各行各业提供坚实稳固、高效安全的邮件安全防护屏障。

CACTER邮件安全将不断加大在邮件安全产品功能性、技术前瞻性和环境适应性方面的研发投入，力求打造更加专业、适配性更强的邮件安全产品，为我国的邮件安全事业添砖加瓦。

**CACTER邮件安全网关**

CACTER邮件安全网关基于自主研发神经网络平台NERVE2.0恶意邮件检测能力，可对接收，外发以及域内垃圾邮件、钓鱼邮件、病毒邮件、BEC诈骗邮件等恶意邮件进行全方位检测拦截（阻断或隔离），**反垃圾准确率高达99.8%，误判率低于0.02%**。

可选择软件/硬件/云服务三种部署方式，不影响邮件系统的正常运行，为邮件系统提供独立安全的防护，支持多类型邮箱系统，支持为Coremail、Exchange、O365、Gmail、IBM Domino、lotus notes等各类主流邮箱系统提供反垃圾反钓鱼反病毒服务。

![Coremail邮件安全网关层层过滤技术架构图（蓝色版）.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20240407/1712459983162724.jpg "1712459983162724.jpg")

**独家域内安全解决方案**

CACTER邮件安全网关基于多项自主研发的世界领先级反垃圾邮件技术，针对用户账号被盗后，【域内互发】垃圾钓鱼邮件场景，推出独家域内安全解决方案：支持域内垃圾邮件过滤检测、域内发信行为管控和告警，确保钓鱼邮件、病毒邮件、垃圾邮件精准拦截，异常发信行为及时发现，以保障邮件系统不受恶意邮件威胁。

![网关-576x370px.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20240407/1712460000632673.png "1712460000632673.png")

**高级威胁邮件事后处置方案**

Coremail邮件安全大数据情报中心将整合学习全网邮件安全情报，CACTER邮件安全网关可以接收并执行Coremail邮件安全大数据情报中心下发的高级恶意威胁邮件召回指令，网关程序将自动召回已经投递至邮件系统的高级恶意威胁邮件，并通知管理员，守住邮件系统安全最后一道防线。

![cacter邮件安全网关-架构图（绿色）.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20240407/1712460036107449.png "1712460036107449.png")
