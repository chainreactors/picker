ж №жҚ® Siemens зҡ„е…¬е‘ҠпјҢж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҝҷдәӣжјҸжҙһз»•иҝҮиә«д»ҪйӘҢиҜҒпјҢи®ҝй—®жҲ–зҜЎж”№ж•°жҚ®еә“еҶ…е®№пјҢз”ҡиҮіиғҪеӨҹд»Ҙ вҖңNT AUTHORITY\NetworkServiceвҖқ жқғйҷҗеңЁж“ҚдҪңзі»з»ҹ shell дёӯжү§иЎҢд»Јз ҒгҖӮиҝҷдәӣй—®йўҳжәҗдәҺдёҚе®үе…Ёзҡ„ж—§д»Јз ҒжЁЎејҸпјҢзӣ®еүҚе·Іеҫ—еҲ°дҝ®жӯЈгҖӮ

Siemens иЎЁзӨәпјҡвҖңV3.1.2.2 д№ӢеүҚзүҲжң¬зҡ„ TeleControl Server Basic еӯҳеңЁеӨҡдёӘ SQL жіЁе…ҘжјҸжҙһпјҢж”»еҮ»иҖ…еҸҜеҲ©з”ЁиҝҷдәӣжјҸжҙһеҜ№еә”з”ЁзЁӢеәҸзҡ„ж•°жҚ®еә“иҝӣиЎҢиҜ»еҶҷж“ҚдҪңгҖҒйҖ жҲҗжӢ’з»қжңҚеҠЎж”»еҮ»пјҢе№¶еңЁж“ҚдҪңзі»з»ҹ shell дёӯжү§иЎҢд»Јз ҒгҖӮвҖқ

жҜҸдёӘжјҸжҙһйғҪжңүе…¶еҜ№еә”зҡ„ CVE зј–еҸ·пјҢSiemensеҲ—еҮәдәҶж•°еҚҒдёӘеҸ—еҪұе“Қзҡ„еҶ…йғЁеҮҪж•°пјҢдҫӢеҰӮ CreateTraceгҖҒVerifyUserгҖҒAuthenticate е’Ң RestoreFromBackupгҖӮжүҖжңүиҝҷдәӣжјҸжҙһзҡ„йҖҡз”ЁжјҸжҙһиҜ„еҲҶзі»з»ҹпјҲCVSSпјүиҜ„еҲҶйғҪеҫҲй«ҳпјҡ

1.CVE-2025-27495пјҲCVSS иҜ„еҲҶ 9.8пјүвҖ”вҖ” еҸҜйҖҡиҝҮ CreateTrace еҮҪж•°иҝӣиЎҢеҲ©з”ЁпјҢе…Ғи®ёжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„иҝңзЁӢд»Јз Ғжү§иЎҢгҖӮ

2.CVE-2025-27539 / CVE-2025-27540пјҲCVSS иҜ„еҲҶ 9.8пјүвҖ”вҖ” еҪұе“Қз”ЁжҲ·иә«д»ҪйӘҢиҜҒж–№жі•пјҢеӯҳеңЁж•°жҚ®еә“иў«е®Ңе…Ёж”»йҷ·зҡ„йЈҺйҷ©гҖӮ

3.CVE-2025-29905 иҮі CVE-2025-32870вҖ”вҖ” й’ҲеҜ№еҗ„з§ҚеҶ…йғЁжӣҙж–°е’Ңз®ЎзҗҶеҮҪж•°пјҢжүҖжңүиҝҷдәӣжјҸжҙһйғҪеҸҜиғҪиў«е·ІйҖҡиҝҮиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҲ©з”ЁгҖӮ

еӨ§еӨҡж•° CVE еңЁ CVSS v3.1 дёӯзҡ„иҜ„еҲҶдёә 8.8 жҲ– 9.8пјҢеңЁ CVSS v4.0 дёӯзҡ„иҜ„еҲҶдёә 8.7 жҲ– 9.3гҖӮ

Siemens е»әи®®з”ЁжҲ·еҚҮзә§еҲ° V3.1.2.2 жҲ–жӣҙй«ҳзүҲжң¬пјҢеҸҜеңЁе…¶ж”ҜжҢҒй—ЁжҲ·зҪ‘з«ҷиҺ·еҸ–иҜҘзүҲжң¬гҖӮ

дёҺжӯӨеҗҢж—¶пјҢSiemens ж•Ұдҝғе®ўжҲ·пјҡ

1.д»…е…Ғи®ёеҸ—дҝЎд»»зҡ„ IP ең°еқҖи®ҝй—®з«ҜеҸЈ 8000гҖӮ

е…¬е‘ҠдёӯжҢҮеҮәпјҡвҖңдҪңдёәдёҖйЎ№йҖҡз”Ёзҡ„е®үе…ЁжҺӘж–ҪпјҢSiemens ејәзғҲе»әи®®дҪҝз”ЁйҖӮеҪ“зҡ„жңәеҲ¶жқҘдҝқжҠӨеҜ№и®ҫеӨҮзҡ„зҪ‘з»ңи®ҝй—®гҖӮвҖқ
