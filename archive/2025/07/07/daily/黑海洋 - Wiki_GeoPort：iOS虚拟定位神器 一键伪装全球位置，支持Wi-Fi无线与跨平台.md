## GeoPort概览

GeoPort 是一款开源虚拟定位工具，用户只需几次点击即可把 iPhone“瞬移”到世界任意角落；它兼容最新 iOS 17 与 iOS 18，并提供 Windows、macOS、Linux 全平台客户端，支持 USB 与 Wi-Fi 双模式连接，方便测试、娱乐与社交分享。

## GeoPort核心卖点

* **全球定位伪装**：内置高精度 GPS 模拟引擎，可在地图上自由选择城市、国家或地标，对社交媒体、游戏和位置依赖型应用实现实时欺骗。
* **Fuel Mode 油价助手**：专为澳大利亚车主打造，实时抓取 Frugal Fuels 与 KwikiMart 油价，帮你一键切换城市，比价更省钱。

* **一键开启开发者模式**：即使 Apple 将入口深藏，GeoPort 仍可自动为连接的 iOS 设备开启 Developer Mode；若设备已设置锁屏密码，只需临时关闭即可完成设定。
* **Wi-Fi 无线模拟**：首次用 USB 绑定后即可彻底摆脱数据线，通过局域网远程控制 iPhone 进行定位伪装。
* **跨平台支持**：官方提供 Windows、macOS（ARM/x86）与 Linux 64 位安装包，测试 App 或“打卡”不受设备限制。

## GeoPort安装与使用要点

1. 访问官网或 GitHub 下载对应系统的安装包并运行。
2. 连接 iOS 设备，按提示授予信任；Windows 端需预先安装 iTunes 以启用 USB 服务。
3. 在主界面搜索或拖动地图，选定目标坐标后点击「Teleport」，即可立即在手机上生效。忘记恢复原位时，只需再次连接点击「Stop Location」。

## GeoPort技术架构与开源优势

GeoPort 基于 Flask 与 pymobiledevice3 开发，利用纯 Python 协议栈与 iDevice 服务通信，因而架构跨平台且后续维护门槛低。
代码托管于 GitHub，由作者 davesc63 持续迭代，并吸收 iFakeLocation 社区反馈，加入对新系统与无线连接的支持。

## GeoPort适用场景

* **APP QA 测试**：验证打车、外卖、社交产品的定位逻辑与异常处理。
* **游戏与社交打卡**：Pokémon GO、旅行相册、朋友圈分享异地风景，安全又有趣。
* **本地生活服务比价**：结合 Fuel Mode，快速查找各州最划算油价并进行导航。

> GeoPort 倡导合理合规使用虚拟定位，避免触犯游戏或服务条款，并尊重他人隐私。
