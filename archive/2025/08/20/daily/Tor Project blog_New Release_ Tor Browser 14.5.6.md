# New Release: Tor Browser 14.5.6

by [ma1](/author/ma1)
| August 19, 2025

![](/new-release-tor-browser-1456/lead.png)

Tor Browser 14.5.6 is now available from the [Tor Browser download page](https://www.torproject.org/download/) and also from our [distribution directory](https://www.torproject.org/dist/torbrowser/14.5.6/).

This version includes important [security updates](https://www.mozilla.org/en-US/security/advisories/) to Firefox.

## Send us your feedback

If you find a bug or have a suggestion for how we could improve this release, [please let us know](https://support.torproject.org/misc/bug-or-feedback/).

## Full changelog

The full changelog since [Tor Browser 14.5.5](https://gitlab.torproject.org/tpo/applications/tor-browser-build/-/raw/maint-14.5/projects/browser/Bundle-Data/Docs-TBB/ChangeLog.txt) is:

* All Platforms
  + Updated NoScript to 13.0.9
  + [Bug tor-browser#44048](https://gitlab.torproject.org/tpo/applications/tor-browser/-/issues/44048): Backport Bug 1979608
  + [Bug tor-browser#44093](https://gitlab.torproject.org/tpo/applications/tor-browser/-/issues/44093): Rebase Tor Browser stable onto 128.14.0esr
  + [Bug tor-browser#44100](https://gitlab.torproject.org/tpo/applications/tor-browser/-/issues/44100): Backport Security Fixes from Firefox 142
  + [Bug tor-browser-build#41515](https://gitlab.torproject.org/tpo/applications/tor-browser-build/-/issues/41515): Bump version of Conjure to include P173 improvements
* Windows + macOS + Linux
  + Updated Firefox to 128.14.0esr
  + [Bug tor-browser#44041](https://gitlab.torproject.org/tpo/applications/tor-browser/-/issues/44041): Letterboxing causes greyed out alert background to be mis-aligned
* Android
  + Updated GeckoView to 128.14.0esr
  + Updated Zstandard to 1.5.7
  + Bug 43346: Remove the "[android] Stop PrivateNotificationService" patch [tor-browser]
  + [Bug tor-browser#43645](https://gitlab.torproject.org/tpo/applications/tor-browser/-/issues/43645): Swiping away doesn't always disconnect from tor
* Build System
  + Windows + Linux + Android
    - Updated Go to 1.23.12
