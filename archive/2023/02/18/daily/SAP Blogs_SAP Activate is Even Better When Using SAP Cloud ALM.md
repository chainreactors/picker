SAP Activate provides methodology and end-to-end guidance for implementing SAP solutions. SAP Cloud ALM provides functionality to extend the same great Activate methodology through project management and execution support.

In this blog, I will point out how several SAP Cloud ALM features are used to facilitate SAP Activate tasks and procedures. For a video on the same subject, see the [SAP Activate Minute - Cloud ALM Features in Activate](https://www.youtube.com/watch?v=p7aJwWWF4xg&list=PLpQebylHrdh7EjybM29pPxT72p35fDMkO&index=4).

I will use the roadmap viewer to visualize the relationships between SAP Activate and SAP Cloud ALM. However, the same activate content is also available in Cloud ALM via a project and the Tasks app.

Tasks that have actions that can be performed in SAP Cloud ALM can be easily found using the filter on the left side of any Activate roadmap. The actions are complementary to the procedure steps or listed as procedure notes.

To find all the Cloud ALM related tasks in a roadmap, select the product SAP Cloud ALM. [Example](https://go.support.sap.com/roadmapviewer/#/group//roadmapContentPage/82b2db84548d41209cda972f0fac428b:p5).

![](/legacyfs/online/storage/blog_attachments/2023/02/Product-CALM.jpg)

Cloud ALM apps are embedded in the procedure text or procedure notes when the functionality can help simplify the task. [Example1](https://go.support.sap.com/roadmapviewer/#/group//roadmap/82b2db84548d41209cda972f0fac428b/node/A5C716BA730940B8A9C818C24EEE195E) [Example2](https://go.support.sap.com/roadmapviewer/#/group//roadmap/82b2db84548d41209cda972f0fac428b/node/FA163ED752201EDABFE83D4F5A945D51)
