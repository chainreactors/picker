## 不容错过的新 App

虽然少数派一直在为大家发现和介绍各平台上的优质 App，但仍有不少设计、功能、交互、体验都非常优秀的 App，还没有被我们发掘和介绍。它们可能是一款老 App，也可能是近期上架的新 App，我们会在这里介绍给你。

### Inure：应用管理器也能有这样的视觉享受

* 平台：Android
* 关键词：应用管理器

[@Noah\_Choi](https://sspai.com/u/69zxdqx8)：Android 玩家的设备中一定会安装一类 App，那就是应用管理器，我们可以方便查看 API 等级、应用架构，权限更高的应用管理器甚至还能直接控制广播器、权限设置等等。比较知名的应用管理器有 Skit、LibChecker 等等，他们都是大家耳熟能详的产品，功能也相对完备。Inure 这款产品并不打算和他们从功能性上正面竞争，而是在无比丝滑的操作体验上夺得焦点。

![boxcnSFS078iJKgvoPhdYbErYQe](https://cdn.sspai.com/editor/u_/cfgcj95b34tejph9cph0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Inure 是一款完全开源的应用，当然也继承了开源应用的优良传统，比如积极适配 Material You 设计，包括完全响应系统调色盘的动态取色功能。应用所有操作都有对应的动态效果，比如在设置界面，每一个设置项图表都会平滑移动、设置列表则会淡出弹出；点击开关按钮，小白点则会有一个弹跳效果。

![boxcnUq97gSNFly5VjZsVaUANXf](https://cdn.sspai.com/editor/u_/cfgcj9db34tejvlnki50?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

除此之外，如果一个选项中有滑动操作，Inure 还会展示进度条动效。为了更好地调教这些动画，Inure 还在设置中单独开辟了动作选项，我们可以在这里更改动画类型、动效开关等等，整体细节满到快要溢出来了。

![boxcnWyv1SDswBu1SnQbSxhLxKc](https://cdn.sspai.com/editor/u_/cfgcj9tb34tejvlnki5g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

当然了 Inure 也不只是一个花瓶，在 Skit 中具备的功能，它绝大部分也都能实现，而且它还提供了终端和 Shell 命令行支持，只不过想要更完备的命令行功能可能需要 Root 权限才可以。

![boxcn5Es4SefGjGxKPbqhGPlUdd](https://cdn.sspai.com/editor/u_/cfgcja5b34tej9i9b2ag?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

比较有趣的是，Inure 允许用户给应用输入备注，而打字的地方竟然是一个支持文本格式化的编辑器，比如可以加粗、斜体、调整颜色甚至是上下角标。根据开发者提供的路线图，Inure 曾经还计划支持音频播放功能，之后也会完善专用文本编辑器，可见其野心之大。

![boxcndqb5TlueXW7eZg2snoDFAj](https://cdn.sspai.com/editor/u_/cfgcjalb34tej9i9b2b0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

整体而言，Inure 给我的感觉是醉翁之意不在酒，它更像是为其他应用在动效上做一个标杆，让用户知道不仅风格设计要好看，动态效果也很重要。如果你也想体验一下这款动效丝滑的应用管理器，可以在 [Google Play](https://play.google.com/store/apps/details?id=app.simple.inure)、[GitHub](https://github.com/Hamza417/Inure)、[F-Droid](https://f-droid.org/en/packages/app.simple.inure/) 和 [IzzyOnDroid](https://apt.izzysoft.de/fdroid/index/apk/app.simple.inure) 上下载安装 Inure。

### Brethap：自定义每日呼吸训练

* 平台：Android
* 关键词：呼吸训练、健康与运动

[@Peggy\_](https://sspai.com/u/5isr02uh)：随着越来越多人对于身心健康的重视，大家开始尝试偶尔从繁重的工作和生活压力当中跳脱出来，注重通过各类方式来体察自己的内心，向内探索自己的各种训练和教程都颇受欢迎，ASMR 视频也受众颇多，同时冥想训练、正念训练也进入了大众的视野。Brethap 是一款辅助我们在冥想时控制呼吸的应用。

使用过 Apple Watch 的用户都知道，正念是被作为内置应用整合在手表当中的，如果你正在使用 Android 手机，同时也没有额外购置智能手表。那么 Brethap 就可以作为一个很好的辅助工具来帮助我们在正念或是冥想的过程当中控制呼吸方式，当然，即便你有智能手表，也可以安装 Brathap 的 Wear OS 版本。

![boxcnOGBJdDxCAe1tHUjm3sTBze](https://cdn.sspai.com/editor/u_/cfgcjatb34tejvlnki60?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Brethap 允许我们设置 5 个不同的呼吸方案，我们可以为每一种方案进行命名，从而方便后续选择对应的方案。而具体到每一种方案，Brethap 都可以进一步设置吸气时间、憋气时间和呼气时间。

![boxcnSApIoHkGD3yII5JkGODZ6f](https://cdn.sspai.com/editor/u_/cfgcjb5b34tejph9cphg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

由于 Brethap 主要是在手机端使用，每次吸气、呼气的提示不像手表那样可以直接通过震动感知，如果你不想在呼吸的过程当中还要抱着手机，那么 Brethap 还支持自定义各类提醒的声音。

Brethap 也内置了统计功能，通过内置的日历，我们可以清晰地看到我们哪天进行了呼吸训练，当天的呼吸训练共计进行了多少次，每次时常是多久。

![boxcniyIJcngOgbfT0kjsFMZQje](https://cdn.sspai.com/editor/u_/cfgcjbdb34tejvlnki6g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

你可以通过 [Play Store](https://play.google.com/store/apps/details?id=com.jithware.brethap&hl=en_US&gl=US) 或是 [F-Droid](https://f-droid.org/packages/com.jithware.brethap/) 下载体验 Brethap，相比之下，F-Droid 版本功能更加精简。

### 事线：规划你的时间线

* 平台：iOS / iPadOS
* 关键词：任务管理

[@Snow](https://sspai.com/u/ul0vcezb)：我们在做项目任务的时候，通常不是一蹴而就的，而需要分阶段拆解和长时间跟进。以往我们可能会在日历或待办中设置阶段性目标，按部就班地完成，但这样的流程在我们需要回溯，或是多项目协调的时候就会变得有些混乱，近期上架 App Store 的「事线」或许可以帮你换种方式解决问题。

事线是一款以时间线为基础的项目架构应用，应用首页是项目清单，点击右上角的「+」新建项目，项目支持自定义名称、颜色、图标以及收集分类，之后就可以开启你的时间线了。项目中的每一个事件，除了支持文字编辑外，还可以添加图片、链接以及时间标记，借助于添加事件的先后顺序和时间标记，应用会自动将所有事件列为一条时间线，你可以快速厘清时间先后逻辑，判断优先跟进事项以及清晰了解事项间的时间间隔。

![boxcnxD12Kn3jwBrVrouYCZERNg](https://cdn.sspai.com/editor/u_/cfgcjblb34tejhhcq4s0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

此外，单个事件还支持追加记录、引用等功能：追加记录可以让你及时补充信息，工作链更为完整；引用则可以在时间顺序基础上再增加一套互联关系，更便于跟进过程中回溯的需求。如果你有团队协作的需求，还可以将任意事项或整个项目以图片的形式分享出去，方便你的工作伙伴一同跟进。

如果你需要同时跟进多个项目，可以在应用第二栏中自建「视图」，快速将多条时间线「收束」在一起，方便你在一条时间线上查阅近期需要跟进的所有事项。

![boxcnSrJihMTS3PE1b2YWjzUGZd](https://cdn.sspai.com/editor/u_/cfgcjc5b34tejph9cpi0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

应用包含一项售价为 30 元的 Pro 服务，付费后可以解锁更丰富的项目配色、支持多图插入和密码保护等功能，由于目前以上服务对主要功能影响不大，可以按需付费。

你可以在 [App Store](https://apps.apple.com/app/id1662454742) 上免费下载「事线」。

### Unread 3.3 更新：新增添加网页文章功能和保存到 Readwise Reader

* 平台：iOS / iPadOS
* 关键词：RSS、聚合阅读

[@Vanilla](https://sspai.com/u/whoops)：Unread 作为一款老牌的 RSS 阅读器，算得上一众竞品中功能迭代比较保守的一个代表。不过，在近日的 3.3 版本更新中，Unread 也新增了添加网页文章的功能，相当于是多了一个稍后读（Read It Later）的模块。另外，Unread 也在新版本中添加了对刚推出不久的 Readwise Reader 的适配，所有文章都可以通过 Article Action 直接添加到 Readwise Reader 中。

将网页文章添加到 Unread 中有两种方式。第一种是通过系统的共享菜单，在 Safari 、Edge 等浏览器或者其它第三方应用中，通过共享菜单中的「Save to Unread」就可以执行保存操作；第二种是直接在 Unread 中长按任意文章中的网页链接，然后点击「ADD TO SAVED ARTICLES」即可。

![boxcniqSUIT3ZFKtuTPkK0hehub](https://cdn.sspai.com/editor/u_/cfgcjctb34tejph9cpig?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

当网页文章被保存到 Unread 后，文章保存的位置会根据你使用服务的不同而不同。如果你使用的是 Unread Cloud、本地账户或者 NewsBlur，那么网页文章都会被保存到 SAVED ARTICLES 一栏；如果你使用的是 Feedbin，那么网页文章会被保存到 Pages 一栏；如果你使用的是 Feedly，那么网页文章会被保存到 READ LATER 一栏。

![boxcnVlGHJEEWgTJ3Jgrcp4RtMd](https://cdn.sspai.com/editor/u_/cfgcjd5b34tejhhcq4sg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Readwise Reader 是 Readwise 在去年年底推出的稍后读工具，Unread 在 3.3 版本中新增了 Readwise Reader 的文章动作，可以将 Unread 中的文章一键推送到 Readwise Reader 中。

在使用这个文章动作前，需要先通过「SETTINGS」>「ARTICLE ACTIONS」启用 Readwise 服务。该服务需要通过 token 来验证，而 token 可以通过点击 GET ACCESS TOKEN 来跳转到相应的网页来获取，再复制粘贴到 Unread 中验证即可。

![boxcnCIQfOwGIOF2y9jwEtAv69b](https://cdn.sspai.com/editor/u_/cfgcjddb34tejvlnki70?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

使用 Readwise 文章动作也有多种方式；第一种是长按文章列表中的任意一篇文章，第二种是长按任意文章中的网页链接，第三种是在任意文章中向左滑动，第四种则是通过外接键盘的键盘快捷键 `⌥ Option+R`，就可以在弹出的菜单中选择「Readwise」动作。

![boxcnZDE9VO6klOzUWn5FLpWkVb](https://cdn.sspai.com/editor/u_/cfgcjdlb34tejhhcq4t0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Unread 可以在 [App Store](https://apps.apple.com/cn/app/unread-2/id1363637349) 上免费下载使用，免费用户可以使用 RSS 订阅和阅读的基础功能。如果订阅 Premium 会员，可以解锁自定义小组件、文章快捷动作、缓存网页文本和图片、定制图标、保存网页文章等高级功能，订阅的价格为每月 20 元或者每年 98 元。

### Things 3.17：用 Apple 快捷指令完成更多

* 平台：iOS / iPadOS / macOS / watchOS
* 关键词：任务管理

[@化学心情下2](https://sspai.com/u/liuxiaofengone)：对于 Apple 用户而言，Things 这款待办事项应用的易用性自然不必多言，开发团队也总可以在第一时间利用好 Apple 生态系统中的诸多新特性，比如在 v3.17 这个新版本上，就让我们在管理待办事项时可以更为简单。

通过 Apple 的快捷指令可以扩展应用的使用边界。比如在此前我们就已经可以通过一些简单的快捷指令来实现一些基础操作，例如创建待办事项、打开待办列表、显示项以及运行 Things URL。而在 v3.17 中，支持的动作扩展到了直接创建项目、创建标题，甚至对待办事项直接进行编辑。这使得我们可以让快捷指令实现更为丰富的操作流。

![boxcn9tPoSfah7RLW591ivVKL9c](https://cdn.sspai.com/editor/u_/cfgcjdtb34tejvlnki7g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

当然，因为支持的动作变多了，Things 为了方便我们操作甚至创建了快捷指令库，这样我们既可以直接调用库中的模板，也可以基于模板来创建自己的快捷指令：

首先我们打开 [Using Apple Shortcuts](https://culturedcode.com/things/support/articles/2955145/) 这个快捷指令库，在这里面我们可以看到多组的快捷指令模板，比如创建任务，就可以实现将文本添加到待办事项、创建一个购买清单或者保存当前位置到我的任务中。这里我们可以选择一个，例如选择这个创建必需品清单。

![boxcnc4xTRA6kTGRWHSQsEpj35d](https://cdn.sspai.com/editor/u_/cfgcje5b34tejhhcq4tg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

点击上方的添加后，会自动打开 Apple 快捷指令应用，点击设置后，首先会要求你选择一个父级目录，选择完成后点击「添加快捷指令」即可，这时候回到快捷指令的「所有快捷指令」中，这个购物清单就已经创建好了。
