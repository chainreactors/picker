## GrimResource зҡ„е·ҘдҪңеҺҹзҗҶ

дҪңдёәжӯӨжҙ»еҠЁзҡ„дёҖйғЁеҲҶпјҢж”»еҮ»д»ҺзҪ‘з»ңй’“йұјз”өеӯҗйӮ®д»¶жҲ–жҒ¶ж„ҸзҪ‘з«ҷдёӯеҢ…еҗ«зҡ„ ZIP ж–Үд»¶ејҖе§ӢгҖӮ

ZIP еҢ…еҗ«еёҰжңү Windows иҜҒд№ҰжҲ– PDF еӣҫж Үзҡ„ж–Үд»¶гҖӮе®һйҷ…дёҠпјҢе®ғжҳҜдёҖдёӘз®ЎзҗҶдҝқеӯҳзҡ„жҺ§еҲ¶еҸ° пјҲMSCпјү ж–Үд»¶пјҢдёҖз§Қз”ЁдәҺеңЁ MMC дёӯдҝқеӯҳй…ҚзҪ®е’Ңи®ҫзҪ®зҡ„ж–Үд»¶зұ»еһӢгҖӮ

MSC жңҖиҝ‘еңЁеЁҒиғҒиЎҢдёәиҖ…дёӯи¶ҠжқҘи¶ҠеҸ—ж¬ўиҝҺгҖӮжӯЈеҰӮ Elastic еЁҒиғҒе’Ңе®үе…Ёжғ…жҠҘдё»з®Ў Jake King жүҖи§ЈйҮҠзҡ„йӮЈж ·пјҢе®ғе§ӢдәҺ Microsoft еҸ‘еёғдәҶеҜ№й»ҳи®ӨжҺ§д»¶зҡ„еӨ§йҮҸжӣҙж”№пјҢиҝҷдәӣжӣҙж”№еҸҜз”ЁдәҺжү§иЎҢз”өеӯҗйӮ®д»¶дёӯзҡ„жңүж•ҲиҙҹиҪҪгҖӮвҖңжҲ‘д»¬ејҖе§ӢзңӢеҲ°дҪҝз”Ё MSIгҖҒISO е’Ң LNK ж–Үд»¶иҝӣиЎҢе”ҫжүӢеҸҜеҫ—зҡ„еҲ©з”ЁгҖӮдҪҶжӣҙе…Ҳиҝӣзҡ„еӣўйҳҹејҖе§ӢеҲ©з”Ё MSC дҪңдёәеҲқе§ӢиҪҪдҪ“пјҢвҖңд»–иҜҙгҖӮ

вҖңиҝҷжҳҜдёҖз§Қйқһеёёжңүи¶ЈгҖҒеҠҹиғҪејәеӨ§зҡ„ж–Үд»¶ж јејҸпјҢ[иҖҢдё”] дёҺи®ёеӨҡз»Ҹеёёиў«ж»Ҙз”Ёзҡ„жӣҙеёёи§Ғзҡ„ж–Үд»¶ж јејҸзӣёжҜ”пјҢе®ғеҸ—еҲ°зҡ„е…іжіЁиҫғе°‘пјҢвҖқд»–иЎҘе……йҒ“пјҢе№¶жҢҮеҮәпјҢвҖңMMC жңүи®ёеӨҡдҪ еҸҜд»ҘеҲ©з”Ёзҡ„жҢҒд№…жҖ§жңәеҲ¶ вҖ” дёҖдәӣж—§зҡ„жјҸжҙһгҖӮ

еҲ©з”ЁжӯӨзұ»жјҸжҙһзҡ„дёҖз§ҚжҠҖжңҜжҳҜВ GrimResourceпјҢе®ғдәҺ 7 жңҲз”ұ Elastic йҰ–ж¬ЎеҸ‘зҺ°гҖӮGrimResource еҲ©з”Ё Windows иә«д»ҪйӘҢиҜҒеҚҸи®®еҹҹж”ҜжҢҒ пјҲAPDSпјү еә“дёӯеӯҳеңЁ 6 е№ҙзҡ„и·Ёз«ҷзӮ№и„ҡжң¬ пјҲXSSпјү й—®йўҳпјҢеңЁ MMC дёӯеҗҜз”Ёд»»ж„Ҹд»Јз Ғжү§иЎҢгҖӮеңЁжӯӨжҙ»еҠЁдёӯпјҢж”»еҮ»иҖ…дҪҝз”Ёе®ғжқҘж¶ҲйҷӨж„ҹжҹ“иҝҮзЁӢдёӯзҡ„дёҖдёӘжӯҘйӘӨпјҡиҖҢдёҚжҳҜи®©еҸ—е®іиҖ…еҚ•еҮ» MSC ж–Үд»¶дёӯзҡ„жҒ¶ж„Ҹй“ҫжҺҘпјҢеҸӘйңҖжү“ејҖ MSC ж–Үд»¶е°ұдјҡи§ҰеҸ‘еөҢе…ҘејҸ JavascriptгҖӮ

з„¶еҗҺпјҢжҒ¶ж„Ҹ Javascript дёӢиҪҪе№¶иҝҗиЎҢдёҖдёӘеҗҲжі•зҡ„гҖҒзӯҫеҗҚзҡ„ Microsoft еҸҜжү§иЎҢж–Үд»¶вҖ”вҖ”вҖңdfsvc.exeвҖқвҖ”вҖ”йҮҚе‘ҪеҗҚдёәвҖңoncesvc.exeвҖқгҖӮдҪҶжҳҜпјҢеҰӮжһңж–Үд»¶жҳҜе®Ңе…ЁиҜҡе®һзҡ„пјҢе®ғжҖҺд№ҲиғҪз”ЁжқҘдёӢиҪҪжҒ¶ж„ҸиҪҜд»¶е‘ўпјҹ

## жҝҖжҙ» AppDomainManager жіЁе…Ҙ

дҪҝз”Ё Microsoft зҡ„ .NET Framework жһ„е»әзҡ„жүҖжңүеә”з”ЁзЁӢеәҸйғҪиҝҗиЎҢдёҖдёӘжҲ–еӨҡдёӘеә”з”ЁзЁӢеәҸеҹҹпјҢиҝҷдәӣеҹҹз”ұ вҖңAppDomainManagerвҖқ зұ»еҲӣе»әе’Ңз®ЎзҗҶгҖӮеңЁ AppDomainManager жіЁе…ҘдёӯпјҢж”»еҮ»иҖ…дҪҝз”ЁжҒ¶ж„Ҹд»Јз ҒеҲӣе»әдёҖдёӘ AppDomainManager зұ»пјҢз„¶еҗҺж¬әйӘ—зӣ®ж Үеә”з”ЁзЁӢеәҸеҠ иҪҪиҜҘзұ»пјҢиҖҢдёҚжҳҜеҗҲжі•еә”з”ЁзЁӢеәҸгҖӮиҝҷеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®дёүдёӘзү№е®ҡзҡ„зҺҜеўғеҸҳйҮҸпјҲAPPDOMAIN\_MANAGER\_ASMгҖҒAPPDOMAIN\_MANAGER\_TYPE е’Ң COMPLUS\_VERSIONпјүжқҘе®ҢжҲҗпјҢжҲ–иҖ…еғҸжң¬жҙ»еҠЁдёӯзҡ„жғ…еҶөдёҖж ·пјҢдёҠдј дёҖдёӘиҮӘе®ҡд№үй…ҚзҪ®ж–Үд»¶пјҢиҜҘж–Үд»¶еҸӘжҳҜжҢҮзӨәеә”з”ЁзЁӢеәҸиҝҗиЎҢе…¶жҒ¶ж„Ҹзҡ„ AppDomainManagerгҖӮ

вҖңжӮЁе®һйҷ…дёҠжҳҜеңЁе‘ҠиҜүе…¬е…ұиҜӯиЁҖиҝҗиЎҢж—¶ пјҲCLRпјүпјҲWindows ж“ҚдҪңзі»з»ҹзҡ„дёҖйғЁеҲҶпјҢе‘ҠиҜүж“ҚдҪңзі»з»ҹеҰӮдҪ•еҠ иҪҪе’ҢеӨ„зҗҶ .NET еә”з”ЁзЁӢеәҸпјүеңЁжӮЁиҝҗиЎҢ .NET иҝӣзЁӢж—¶еҢ…еҗ«жҒ¶ж„Ҹ DLLпјҢвҖқRapid7 жё—йҖҸжөӢиҜ•йҰ–еёӯе®үе…ЁйЎҫй—® Nicholas Spagnola и§ЈйҮҠиҜҙгҖӮвҖңе®ғжңүж•Ҳең°е…Ғи®ёжӮЁе°ҶеҮ д№Һд»»дҪ• .NET еә”з”ЁзЁӢеәҸиҪ¬жҚўдёәзҰ»ең°еӯҳзҡ„дәҢиҝӣеҲ¶ж–Үд»¶пјҢвҖқжҲ– lolbinгҖӮ

вҖңзӣ®еүҚпјҢDLL ж—ҒеҠ иҪҪжҳҜжү§иЎҢжҒ¶ж„ҸиҪҜд»¶зҡ„жңҖеёёз”Ёж–№жі•пјҢвҖқNTT з ”з©¶дәәе‘ҳеҶҷйҒ“пјҢвҖңдҪҶ AppDomainManager жіЁе…ҘжҜ” DLL ж—ҒеҠ иҪҪе®№жҳ“еҫ—еӨҡпјҢе№¶дё”дәәд»¬жӢ…еҝғжңӘжқҘеҸҜиғҪдјҡеўһеҠ жјҸжҙһеҲ©з”ЁгҖӮ

з”ұдәҺеҸ‘зҺ°жӯӨзұ»жҒ¶ж„ҸжіЁе…Ҙйқһеёёеӣ°йҡҫпјҢеӣ жӯӨ King жҺЁиҚҗдәҶдёҖз§ҚйҳІеҫЎж–№жі•пјҢеҸҜд»ҘеңЁжӯӨзұ»ж”»еҮ»ејҖе§Ӣд№ӢеүҚйҳ»жӯўе®ғд»¬гҖӮ

вҖңдҪ еңЁиҝҷйҮҢзңӢеҲ°зҡ„жңҖйҮҚиҰҒзҡ„дәӢжғ…жҳҜиғҪеӨҹд»ҺдёҖејҖе§Ӣе°ұйҳ»жӯўжңүж•ҲиҪҪиҚ·зҡ„жү§иЎҢпјҢвҖқд»–иҜҙгҖӮдҫӢеҰӮпјҢеңЁиҝҷдёӘжңҖж–°зҡ„жҙ»еҠЁдёӯпјҢвҖңиҝҷдәӣжҳҜеј•е…Ҙ ZIP ж–Үд»¶зҡ„йұјеҸүејҸзҪ‘з»ңй’“йұјж”»еҮ»гҖӮжӮЁеҸҜд»ҘеңЁ MMC зә§еҲ«е®һж–ҪдёҖдәӣеҹәжң¬зҡ„жҺ§еҲ¶жҺӘж–ҪпјҢдҪҶ [йў„йҳІ] е®һйҷ…дёҠеҸӘжҳҜеҪ’з»“дёәеӣҙз»•з”өеӯҗйӮ®д»¶еҚ«з”ҹзҡ„иүҜеҘҪе®һи·өгҖӮ
