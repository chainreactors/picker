# la57setup.exe & OOBEFodSetup.exe lolbin

Posted on [2024-12-26](https://www.hexacorn.com/blog/2024/12/26/la57setup-exe-lolbin/ "11:44 pm")  by  [adam](https://www.hexacorn.com/blog/author/adam/ "View all posts by adam")

When you run *la57setup.exe* or *OOBEFodSetup.exe* on Windows Server 2025, they will try to load a non-existing library from the path below:

```
C:\Windows\System32\Dism\MetaDeployProvider.dll
```
