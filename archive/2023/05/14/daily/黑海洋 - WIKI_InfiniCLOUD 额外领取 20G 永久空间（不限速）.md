日本 TeraCloud 网盘更改为 InfiniCloud 名称，并简单提升一些服务，它是一款 WebDAV 网盘。

## 额外领取25G永久空间

官网地址 [https://infini-cloud.net/](https://blog.upx8.com/go/aHR0cHM6Ly9pbmZpbmktY2xvdWQubmV0Lw)

注册成功后，在`我的页面`（[https://infini-cloud.net/ja/modules/mypage/usage/](https://blog.upx8.com/go/aHR0cHM6Ly9pbmZpbmktY2xvdWQubmV0L2phL21vZHVsZXMvbXlwYWdlL3VzYWdlLw)）中`Get more capacity`一栏输入介绍代码（**NRK76**）即可**额外**获取5G永久空间（也就是20+5G）。注意，如果你不输入邀请码，只能有20G。

登录账号后，在地址栏分别输入下面两个链接(改成自己地用户ID)后回车，在页面上点击确认按钮就可以了。

* **日文活动链接**

```
https://infini-cloud.net/ja/modules/bonus/code=202304_10GB_jp/userid=用户ID
```

* **英文活动链接**

```
https://infini-cloud.net/ja/modules/bonus/code=202304_10GB_en/userid=用户ID
```

领取成功之后会显示如下图，一个是英文的，一个是日文的，每个链接可以领取10G（273年时效，可以算成永久）空间，经过以上操作，你就可以领取45G永久空间了

## 新老版本区别

* 老版本（TeraCloud）新用户10G永久空间，最大传输速率100M；
* 新版本（InfiniCloud）新用户20G永久空间，最大传输速率200M。

## 不受影响的方面

* 用户名称
* 登录密码
* 文件浏览器网址
* 外部连接信息（WebDAV 连接 URL、连接 ID、应用密码）
* 用于链接与 WebDAV 兼容的应用程序的 REST API URL
* 订阅费
