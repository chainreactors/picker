# 连续三年！Fortinet再次位列《Gartner 企业级有线和无线局域网基础设施魔力象限报告》“远见者”

导语：​网络安全领导者Fortinet®（NASDAQ：FTNT），近日宣布连续第三年入围《Gartner®企业级有线和无线局域网基础设施魔力象限报告》“远见者”象限。

网络安全领导者Fortinet®（NASDAQ：FTNT），近日宣布连续第三年入围《Gartner®企业级有线和无线局域网基础设施魔力象限报告》“远见者”象限。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230110/1673344127199308.png)

这一殊荣源于Fortinet 旗下FortiSwitch 和 FortiAP等有线和无线局域网产品组合的强劲实力，以及人工智能（AI）等先进自动化技术驱动的实时网络风险可见性、无需额外许可成本的安全性以及无缝集成实时威胁情报的安全平台等一系列强大功能优势，全方位满足用户不断发展的安全需求，为企业数字化转型加速保驾护航。

Fortinet 有线和无线局域网产品组合由安全驱动型组网架构提供强劲支持，使有线和无线网络功能无缝整合至FortiGate 下一代防火墙，助力用户紧密集成网络基础设施和安全架构，快捷交付全新数字化创新成果，无惧各类网络安全风险。此外，Fortinet 有线和无线局域网产品组合与 FortiAIOps 解决方案的无缝集成，支持跨局域网、无线局域网和广域网交付AI驱动的一致性能洞察，赋能网络运营中心（NOC）团队尽享潜在安全风险实时可见性的同时，确保顺畅高效的网络用户体验。

**Fortinet 为用户打造功能丰富的有线和无线局域网解决方案**

作为当前唯一一家提供网络和安全全面融合的有线和无线局域网高级解决方案供应商，Fortinet 赋能用户以下强大优势：

以FortiGate为中心实现有线与无线局域网高级特性，无需额外支付高昂许可费用。摒弃以往独立无线局域网控制器与短期许可证的捆绑采购模式，无需额外支付许可费用，即可启用 Fortinet 有线和无线局域网管理或 NAC（网络访问控制）功能。Fortinet将交换机和无线局域网控制器原生集成至 FortiGate 平台，无需搭建overlay架构，助力用户大幅降低总体拥有成本（TCO）及运营成本。

凭借单一管理平台即可轻松简化局域网和网络安全的运维管理。随着组网架构日益复杂化，Fortinet 有线和无线局域网解决方案支持全面融合网络和安全管理，简化日常网络运维，缩短IT 部署周期，减少任务切换数量，助力管理员轻松管理网络安全性、局域网、无线网、SD-WAN 和 5G/LTE 网关组件。

支持与业界表现最佳的网络安全网格平台 Fortinet Security Fabric 无缝集成。仅需部署Security Fabric平台，即可实现对Fortinet 有线和无线局域网解决方案的一站式轻松管理，全面掌控局域网、广域网乃至云端网络环境实时动态。

FortiOS 单一操作系统为以上创新功能提供强劲支撑，助力用户统一进行安全性和局域网的部署、管理和运维。这种由Fortinet专用操作系统支持的紧密集成，支持用户跨整个网络收集一致的威胁情报，并通过 AI/ML 引擎进行集中关联和分析。

Fortinet始终专注有线和无线解决方案的持续创新开发，为当今企业高速发展打造更加出色的高级网络和安全解决方案，并随不断扩展的混合网络架构为用户交付持续一致的可用性和安全性。

Fortinet产品执行副总裁兼首席营销官John Maddison认为，“当前，有线和无线局域网网络环境与安全性密不可分。作为业内唯一一家能够提供安全有线和无线局域网解决方案的行业领军者，Fortinet 始终引领网络和安全融合创新发展。我们坚信，Fortinet连续三年位列《Gartner企业级有线和无线局域网基础设施魔力象限报告》远见者象限，充分印证了Fortinet解决方案能够实现有线、无线局域网和安全性全面融合的独特优势。”
