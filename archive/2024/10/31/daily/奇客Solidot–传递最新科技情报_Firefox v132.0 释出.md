## Firefox v132.0 释出

[![Firefox](https://icon.solidot.org/images/topics/topicfirefox.png?123)](/search?tid=39 "Firefox")

[Wilson](/~Wilson) (42865)发表于 2024年10月30日 14时23分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=79627&appkey=1370085986&title=Firefox%20v132.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自临渊而立**

Mozilla 释出了 Firefox v132.0。主要新特性包括：在部分地区的 Windows 上支持 Microsoft PlayReady 加密媒体播放；Windows 和 macOS 支持高色域（Wide Color Gamut）WebGL；SVG 硬件加速；移除对 HTTP/2 Push 的支持；恢复支持通过 USB 进行远程调试，安全修正等等。
https://www.mozilla.org/en-US/firefox/132.0/releasenotes/
