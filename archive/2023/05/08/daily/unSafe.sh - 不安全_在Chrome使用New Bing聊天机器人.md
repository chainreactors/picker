在之前New Bing公测的时候，博主就对其感兴趣了。不过由于微软的某些原因，我迟迟没有等到内测的名额。好在最近几天，New Bing向所有用户开放了，我才得以使用其进行聊天。而且流畅度比ChatGPT好多了。但是有个缺点就是只能在Edge浏览器上使用。在这期文章中，我来和大家一起突破微软的限制，在Chrome浏览器使用New Bing

## 准备材料

* Chrome 浏览器
* 微软账户（可选）

## 部署步骤

1. 打开这个插件的下载链接：<https://chrome.google.com/webstore/detail/user-agent-switcher-and-m/bhchdcejhohfmigjafbampogmaanbfkg> ，然后点击“添加至Chrome”按钮，进行插件安装

[![](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs@main/20230505100641.png)](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs%40main/20230505100641.png)

2. 按照下图指示，设置UA

[![](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs@main/20230505100737.png)](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs%40main/20230505100737.png)

3. 点击“Test UA”按钮，测试UA。当看到和插件设置的UA一样的时候即为成功

[![](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs@main/20230505100836.png)](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs%40main/20230505100836.png)

4. 随意问一个问题，可以看到是和在Edge一样是正常使用的

[![](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs@main/20230505100939.png)](https://cdn.jsdelivr.net/gh/Misaka-blog/imgs%40main/20230505100939.png)
