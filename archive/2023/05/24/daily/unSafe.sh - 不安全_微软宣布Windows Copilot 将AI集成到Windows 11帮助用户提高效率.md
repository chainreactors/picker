Build 2023 即将开幕，在开幕前微软已经发布了部分产品预告，其中与 Windows 相关的就是将在 6 月份推出的 Windows Copilot --- 基于人工智能的 Windows 副驾驶。

根据微软说明 Windows Copilot 将在 Windows 11 任务栏中提供，旨在帮助 Windows 用户更轻松地查找和修改设置，例如当发出自然语言命令 “调整我的设置以便让我集中精力” 时，AI 会自动开启 Windows 11 专注模式，屏蔽非重要的通知，避免打扰用户。

[![微软宣布Windows Copilot 将AI集成到Windows 11帮助用户提高效率](https://img.lancdn.com/landian/2023/05/98793.png)](https://img.lancdn.com/landian/2023/05/98793.png)

Windows Copilot 还支持分析内容，其载体是剪贴板，当用户将一段内容复制到剪贴板时，就可以让 Windows Copilot 总结内容或者解释说明。

除此之外，Windows Copilot 也将成为通过 Bing Chat 的门户，用户可以在 Windows 11 上提出简单或复杂的问题，由 Bing Chat 提供更好的回答。

上述功能将在 6 月份提供，到时候微软可能会提前进行测试，具体效果怎么样那还需要等到 6 月观察。

这次集成 Bing AI 不再是之前那种糊弄人的做法了，之前微软也宣称将 AI 集成到 Windows 11，结果只是搜索栏里添加 Bing Chat 的快捷方式，点击后使用 Microsoft Edge 浏览器打开 Bing Chat 页面。
