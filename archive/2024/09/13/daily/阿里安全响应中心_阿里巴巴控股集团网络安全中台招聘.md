**集团安全部-算法专家（NLP）-网络安全**

![](https://mmbiz.qpic.cn/mmbiz_png/tCS9QJPdcGcYYvTr8E8ZQlmNNO18VR85jTYoDdbdl8j3F7kEl41aYJf5R9yCXlib8faJLiaGgIv3Xg3ysfQKFMIg/640?wx_fmt=other&from=appmsg&wxfrom=5&wx_lazy=1&wx_co=1&tp=webp)

**职位描述**

负责阿里集团网络安全算法能力建设和优化，包括并不限于：

1.负责集团数据防泄漏和资产保护算法能力建设：开展数据资产分类分级、检索匹配、信息抽取、数据流转追踪、UEBA用户行为分析、事件溯源等相关算法研发，构建业界领先的一体化数据安全能力。

2.建设安全领域大模型，模型能力具备业界竞争力，完成大模型在数据安全“识别-检测-溯源”全链路的落地，提升安全水位、业务效率和用户体验。

3.持续跟进AI算法前沿技术，用创新的思路和技术方案解决业务带来的安全挑战。

**职位要求：**

1.计算机、数学或统计学相关专业硕士及以上学历；

2.熟悉自然语言处理、机器学习算法，有大模型、多语言、多模态、端云协同相关实践经验优先；

3.熟练掌握Python/Java/C++中至少一门语言，有扎实的数据结构和算法基础；

4.在顶会或者期刊上发表过论文者优先，在权威比赛中获奖者优先；

5.在高影响力的开源项目中，做出过贡献者优先；

6.具备优秀的问题分析和解决能力，对新技术有好奇心和热情，具备良好的沟通协作能力

**简历投递：**

https://talent-holding.alibaba.com/off-campus/position-detail?positionId=1190601

**集团安全部-Java高级安全研发工程师/专家**

![](https://mmbiz.qpic.cn/mmbiz_png/tCS9QJPdcGcYYvTr8E8ZQlmNNO18VR85jTYoDdbdl8j3F7kEl41aYJf5R9yCXlib8faJLiaGgIv3Xg3ysfQKFMIg/640?wx_fmt=other&from=appmsg&wxfrom=5&wx_lazy=1&wx_co=1&tp=webp)

**职位描述**

从事生产网安全产品的 Java 底层技术研发

**职位要求：**

1. 具备3年以上的 Java 工程开发经验，熟悉常见 Java 应用框架并理解其原理，具备过硬的编码能力，有中间件开发经验加分

2. 深入理解 Java 虚拟机，从事过 JVMTI 相关开发，有 Agent 开发经验加分

3. 熟悉操作系统原理，有系统软件开发经验加分，有 Rust 经验更佳

4. 了解常见的 Web 安全漏洞/ Java 安全漏洞及其防护机制

5. 强烈的责任心和主动性，能够在无监督的情况下自主工作

6. 具备良好的沟通能力和团队合作精神

**工作地点：**

杭州
