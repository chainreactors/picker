е·ІдёәеҪұе“Қ NMP Web+ зүҲжң¬ 3.2.5 еҸҠжӣҙж—©зүҲжң¬зҡ„й—®йўҳеҲҶй…ҚдәҶдёүдёӘ CVEпјҡ

* **CVE-2025-49151 вҖ“ зЎ¬зј–з Ғзҡ„ JWT еҜҶй’ҘпјҲCVSS 9.1 дёҘйҮҚпјүпјҡ**вҖң*еҸ—еҪұе“Қзҡ„дә§е“ҒеҸҜиғҪе…Ғи®ёжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…з”ҹжҲҗдјӘйҖ зҡ„ JSON Web д»ӨзүҢ пјҲJWTпјү д»Ҙз»•иҝҮиә«д»ҪйӘҢиҜҒпјҢ*вҖқе…¬е‘ҠжҳҫзӨәгҖӮиҝҷдёӘе…ій”®жјҸжҙһжәҗдәҺеңЁиә«д»ҪйӘҢиҜҒжңәеҲ¶дёӯдҪҝз”ЁзЎ¬зј–з Ғзҡ„гҖҒдёҺе®үе…Ёзӣёе…ізҡ„еёёйҮҸпјҢе…Ғи®ёж”»еҮ»иҖ…дјӘйҖ  JWT е№¶еңЁжІЎжңүжңүж•ҲеҮӯжҚ®зҡ„жғ…еҶөдёӢи®ҝй—®зі»з»ҹгҖӮ
* **CVE-2025-49152 вҖ“ жҢҒд№…дјҡиҜқд»ӨзүҢ пјҲCVSS 7.5 Highпјүпјҡ**вҖң*еҸ—еҪұе“Қзҡ„дә§е“ҒеҢ…еҗ«дёҚдјҡиҝҮжңҹзҡ„ JSON Web д»ӨзүҢ пјҲJWTпјүпјҢиҝҷеҸҜиғҪе…Ғи®ёж”»еҮ»иҖ…и®ҝй—®зі»з»ҹпјҢ*вҖқе…¬е‘ҠиӯҰе‘ҠиҜҙгҖӮдёҖж—ҰйўҒеҸ‘пјҢд»ӨзүҢе°Ҷж— йҷҗжңҹдҝқжҢҒжңүж•ҲпјҢеҰӮжһңж”»еҮ»иҖ…и®ҫжі•жӢҰжҲӘжҲ–з”ҹжҲҗжңүж•Ҳд»ӨзүҢпјҢеҲҷеҸҜд»Ҙдёәе…¶жҸҗдҫӣжҢҒд№…и®ҝй—®жқғйҷҗгҖӮ
* **CVE-2025-49153 вҖ“ и·Ҝеҫ„йҒҚеҺҶеҲ°д»»ж„Ҹд»Јз Ғжү§иЎҢпјҲCVSS 9.8 дёҘйҮҚпјүпјҡ**вҖң*еҸ—еҪұе“Қзҡ„дә§е“ҒеҸҜиғҪе…Ғи®ёжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…иҰҶзӣ–ж–Үд»¶е№¶жү§иЎҢд»»ж„Ҹд»Јз ҒпјҢ*вҖқе…¬е‘ҠжҢҮеҮәгҖӮиҝҷжҳҜдёүдёӘжјҸжҙһдёӯжңҖдёҘйҮҚзҡ„дёҖдёӘпјҢеҸҜиғҪе…Ғи®ёеңЁжІЎжңүд»»дҪ•иә«д»ҪйӘҢиҜҒзҡ„жғ…еҶөдёӢиҝңзЁӢжү§иЎҢд»Јз ҒпјҢд»ҺиҖҢжҸҗдҫӣеҜ№зі»з»ҹзҡ„е®Ңе…ЁжҺ§еҲ¶гҖӮ
