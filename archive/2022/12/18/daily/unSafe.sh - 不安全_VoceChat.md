## Have a Private Social Space Hosted on Your Site

As a top alternative to centralized chat services, Voce chat is a superlight Rust powered open-core chat app that prioritizes private hosting.

[![Google Play Icon](https://voce.chat/img/icon.app.google.play.png)](https://play.google.com/store/apps/details?id=com.privoce.vocechatclient "download Android APP from Google Play")[![APK Icon](https://voce.chat/img/icon.app.apk.png)](https://vocechat.s3.amazonaws.com/vocechat.android.apk "download Android APP from APK file")[![App Store Icon](https://voce.chat/img/icon.app.ios.png)](https://apps.apple.com/app/vocechat/id1631779678 "download iOS APP from app store")

### Open API and SDK for you to host anywhere. No central service.

Modern features, open API, open sourced SDK, iterate based on users feedback. Trusted by over 4,000 customers.

[Test Live API](https://dev.voce.chat/api/swagger)

![Feature Icon](https://voce.chat/_next/image?url=%2Fimg%2Ficon.secure.png&w=96&q=75)

#### Security

* All data is stored on your own, private, server.
* The server is developed in Rust with critical business process tests with 80%+ coverage.
* Read and delete functionality.

![Feature Icon](https://voce.chat/_next/image?url=%2Fimg%2Ficon.lightweight.png&w=96&q=75)

#### Lightweight

* The server is only 10MB in size and can easily run on a NAS and Raspberry Pi.

![Mobile Demo Picture](https://voce.chat/_next/image?url=%2Fimg%2Fmobile.png&w=640&q=75)

![Feature Icon](https://voce.chat/_next/image?url=%2Fimg%2Ficon.open.api.png&w=96&q=75)

#### OpenAPI

* Utilizes OpenAPI standard, with easy integration with third-party applications and tools.
* With complete documentation and testing.

![Feature Icon](https://voce.chat/_next/image?url=%2Fimg%2Ficon.cross.platform.png&w=96&q=75)

#### Cross Platform

* Developed with Flutter + React.
* Support Android, iOS, MacOS, Web, and Windows platforms.

### Deploy our free-trial version through docker:

For other ways of installation, check out our [documentation](https://doc.voce.chat).

`docker run -d --restart=always \
-p 3000:3000 \
--name vocechat-server \
privoce/vocechat-server:latest`After starting the server, visit http://localhost:3000/ to use the app.
