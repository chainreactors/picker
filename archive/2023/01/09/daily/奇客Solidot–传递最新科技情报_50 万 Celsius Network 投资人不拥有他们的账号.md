## 50 万 Celsius Network 投资人不拥有他们的账号

[![比特币](https://icon.solidot.org/images/topics/topicBitcoin.png?123)](/search?tid=149 "比特币")

[Wilson](/~Wilson) (42865)发表于 2023年01月08日 21时54分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73833&appkey=1370085986&title=50%20%E4%B8%87%20Celsius%20Network%20%E6%8A%95%E8%B5%84%E4%BA%BA%E4%B8%8D%E6%8B%A5%E6%9C%89%E4%BB%96%E4%BB%AC%E7%9A%84%E8%B4%A6%E5%8F%B7 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自怒火重燃**

破产的加密货币贷款机构 Celsius Network 的逾 50 万投资者不拥有他们的账号。审理破产案的法官 Martin Glenn 裁决这些加密货币属于 Celsius 而不是储户。法官发现很少有用户去阅读的 Celsius 使用条款意味着加密货币资产是 Celsius 的财产。这一裁决突出了不受监管的加密货币行业的狂野西部性质。行业专家指出，有很多加密货币平台的使用条款与 Celsius 类似，用户在将资产储存在缺少监管的平台时需要明白所承担的风险。纽约检察长 Letitia James 本周提起诉讼，指控  Celsius 创始人 Alex Mashinsky 欺骗了数十万客户。
https://www.washingtonpost.com/technology/2023/01/05/celsius-crypto-bankruptcy-ruling/
