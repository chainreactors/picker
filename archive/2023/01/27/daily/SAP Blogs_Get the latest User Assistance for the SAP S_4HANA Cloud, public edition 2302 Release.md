The SAP S/4HANA Cloud, Public Edition 2302 Release is now available. To get an overview for all the new features available in SAP S/4HANA Cloud, public edition check out the following information:

* You will find an overview on the most important innovations in Bert Schulze’s blog [Highlights of the SAP S/4HANA Cloud, Public Edition Release](https://blogs.sap.com/2023/01/25/highlights-of-the-sap-s-4hana-cloud-public-edition-2302-release/).

* A full list of all new and changed features is available in the [What's New Viewer.](https://help.sap.com/whats-new/7d3d11840a6543329e72391cf4d48e2d)

* You can get an overview on all news and changes relevant for your company’s SAP S/4HANA Cloud, public edition system via the [Release Assessment and Scope Dependency (RASD)](https://release-assessment-v2.cfapps.eu10.hana.ondemand.com/cp.portal/site#Shell-home).

To get all available information and details on the new release, visit our [SAP S/4HANA Cloud](https://help.sap.com/docs/SAP_S4HANA_CLOUD) product page on the SAP Help Portal.
