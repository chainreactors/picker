# 360智脑开放API接口 周鸿祎：大模型赋能百行千业才能引领工业革命

企业资讯
[行业](https://www.4hou.com/category/industry)
2023-06-28 16:35:09

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)66119

收藏

导语：“大模型不是操作系统，未来将成为数字化系统标配”

“大模型不是操作系统，未来将成为数字化系统标配”6月28日，三六零(601360.SH，下称“360”)集团创始人周鸿祎在GPT产业联盟成立大会上发布了360企业级AI大模型战略，并宣布360智脑API平台正式开放，将率先为20个行业提供解决方案。GPT产业联盟大会由中国互联网协会、中国信息通信研究院云大所、360集团联合举办。

周鸿祎表示，数字化已经上升为国家战略，人工智能的发展要服务产业数字化，所以企业级市场是大模型未来的蓝海市场。但是，公有大模型直接用于企业级客户存在几点不足，如缺乏行业深度、数据安全隐患、无法保障内容可信、训练和部署成本高等。

周鸿祎透露，360将以“安全、可信、可控、易用”四大原则为指导，联合生态伙伴为企业级客户打造行业化、专有化大模型，“让万事通变成城市通、行业通和企业通”。

![图片1.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230821/1692599686202244.png "1687941228722352.png")

周鸿祎同时宣布面向企业客户和开发者等生态伙伴开放“360智脑”API，“大模型要放低身段，真正赋能百业千行，才能成为工业革命级的生产工具，引领工业革命”。据了解，360智脑行业解决方案将率先在安全、传媒、文旅、政务、能源等近20个行业落地，为企业级用户构建包括“办公写作大脑、决策分析大脑、知识管理大脑、客户服务大脑、文旅招商大脑”在内的“五个智慧大脑”。

![图片2.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20230821/1692599689176243.png "1687941260115123.png")

会上周鸿祎首次系统阐述了360企业级AI大模型的四大方法论。他判断，大模型未来在企业的落地形态一定是多个垂直小规模的大模型的组合。企业从数字化到智能化要分多步走，从松耦合到深度融合、多步迭代。企业应用大模型的落地场景要“小切口、大纵深”，第一步是从办公场景切入，办公场景能做到“松耦合”，不破坏现有业务系统，与现有系统无缝融合。他认为，大模型不仅要服务企业领导，更要服务企业员工，真正发挥解决痛点和刚需的作用，让从上到下每个人都能用起来，实现AI普惠。

会上，中国互联网协会副理事长黄澄清致辞中表示，生成式人工智能给我国带来了机遇，也给数据安全、生产生活等多个领域带来风险和挑战。中国信息通信研究院纪委书记李德文在致辞中强调，我们要以高度负责任的态度推动GPT产业发展，持续深化伦理风险和治理体系的研究。加大相关技术投入，让GPT不仅可用，还要可靠、可信。
