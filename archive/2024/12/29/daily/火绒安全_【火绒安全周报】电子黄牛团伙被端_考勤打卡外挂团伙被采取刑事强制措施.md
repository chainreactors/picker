# 代抢挂专家号电子黄牛团伙被端

#

近日，公安机关成功摧毁两个长期倒卖医院专家号进行非法牟利的“电子黄牛”团伙。据报道，这些“电子黄牛”利用外挂程序，通过破解医院挂号系统验证机制、抓包、调整请求发送频次等手段，恶意抢占专家号，致使患者无法通过医院挂号系统正常预约挂号。该行为严重扰乱医疗秩序，同时使“电子黄牛”得以高价转卖专家号，进行非法牟利。经过3个月的调查，警方成功捣毁犯罪团伙2个，抓获犯罪嫌疑人12名，并查扣大量作案装备。目前，已有4名涉案人员被依法追究法律责任。警方提醒公众，购买黄牛票将增加个人信息泄露风险，并呼吁患者通过官方渠道满足就诊需求。

# 网飞起诉博通与VMware公司专利侵权

#

2024年12月23日，美国视频流媒体巨头Netflix（网飞）正式对芯片制造商Broadcom（博通）及其子公司VMware提起专利侵权诉讼。此次纠纷涉及的技术主要用于虚拟环境中的资源监控和优化。网飞方指控，VMware自2012年知晓相关专利信息后仍持续进行侵权行为，并通过其开发和销售的云服务产品，以直接和间接的方式侵犯了网飞公司持有的五项专利权。此外，网飞方称，VMware在其云服务产品的销售和使用手册中，涉嫌故意引导用户进行侵权操作。鉴于博通已在2023年成功收购VMware，网飞方认为博通应承担相应连带责任。目前，网飞公司已向法院申请判决博通和VMware构成侵权，并要求其赔偿相关损失。

# 考勤打卡外挂团伙被采取刑事强制措施

#

近日，四川公安网安部门成功捣毁多个开发、售卖远程打卡外挂的犯罪团伙。据报道，有网民利用打卡外挂软件绕过安全风控机制，非法获取打卡应用程序数据，通过修改打卡软件和手机位置参数，伪造考勤打卡数据，严重扰乱了企业的经营秩序和信息网络安全。警方就此奔赴多地展开调查，共计查封涉案工作室6家，抓获犯罪嫌疑人17名，扣押手机80余部、电脑主机51台，查获外挂硬件辅助设备200余个。目前，相关犯罪嫌疑人已被采取刑事强制措施。根据我国《刑法》相关规定，提供此类侵入、非法控制计算机信息系统的程序、工具，情节严重将面临刑事处罚。

# 犯罪分子以帮忙为名，行诈骗之实

#

据新华社报道，近日四川、河南等多地曝出一种与医保码相关的诈骗案件。犯罪分子假借第三方平台工作人员身份，以下载注册医保APP、帮忙激活电子医保卡为名，套取群众个人信息，实施网络账号注册、转账、洗钱等非法行为。国家医保局提醒，医保码一人一码、专码专用，需本人激活才安全有效；同时强调，医保局及其下属部门从未授权任何社会人员开展相关工作，并建议广大群众在享受便捷医保服务的同时，务必提高警惕，谨防上当受骗。若发现相关线索，及时向公安机关报案或向医保部门举报。

# 联合国大会通过《联合国打击网络犯罪公约》

#

2024年12月24日，联合国大会一致通过《联合国打击网络犯罪公约》。该公约为20多年来经谈判达成的首个国际刑事司法条约，将于2025年在越南首都正式开放签署，并在第40个签署国批准后90天生效。据称，《联合国打击网络犯罪公约》旨在加强国际合作，预防和打击网络犯罪。公约承认滥用信息和通信技术所带来的重大风险，并强调技术援助、能力建设以及各国和其他利益攸关方之间合作的必要性，以保护人民及其在线权利，维护全球网络安全与稳定。
