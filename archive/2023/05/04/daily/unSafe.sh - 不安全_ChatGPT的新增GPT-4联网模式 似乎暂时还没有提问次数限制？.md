[上周蓝点网提到 OPENAI 将 ChatGPT 的联网模式名称修改了](https://www.landiannews.com/archives/98458.html)，原本为 Browsing Alpha，修改后为 Default (GPT-3.5) with browsing (Alpha)。

之前 OPENAI 是没有说明联网模式具体是基于哪个版本的，当时蓝点网猜测 GPT-4 有提问次数限制，联网模式没有，那联网模式肯定是 GPT-3.5，这次改名算是证实了。

不过现在来看我们的猜测还是不对，OPENAI 修改名称是为了新模式的推出，也就是刚推出的 Web Browsing (GPT-4) Alpha 版。

[![ChatGPT的新增GPT-4联网模式 似乎暂时还没有提问次数限制？](https://img.lancdn.com/landian/2023/05/98510-1.png)](https://img.lancdn.com/landian/2023/05/98510-1.png)

[![ChatGPT的新增GPT-4联网模式 似乎暂时还没有提问次数限制？](https://img.lancdn.com/landian/2023/05/98510-2.png)](https://img.lancdn.com/landian/2023/05/98510-2.png)

这与 GPT-3.5 的联网模式有什么区别呢？其实本质上没太大区别，就是与你对话的机器人换成了 GPT-4，联网查询内容还是基于浏览模式插件 (包括返回的内容)，但如果用户继续对话的话，那就是 GPT-4 来回答了。

比较有趣的是 GPT-4 联网模式似乎没提到提问次数限制，而非联网模式的 GPT-4 至今仍然是每 3 小时内最多提问 25 次，不知道这是 OPENAI 忘了改了还是希望更多用户使用并测试联网模式，所以暂时还没加上限制。

***注：联网模式属于插件功能，插件功能目前正在逐步开放中，并非所有用户都能看到联网模式。***
