YARA [release candidate 1 for version 4.3.0](https://github.com/VirusTotal/yara/releases/tag/v4.3.0-rc1) brings a new option for [XOR strings](https://isc.sans.edu/diary/YARA%2BXOR%2BStrings%2Ban%2BUpdate/25346): --print-xor-key

This option prints out the XOR key that matches an XOR string (0x41 in this example):
