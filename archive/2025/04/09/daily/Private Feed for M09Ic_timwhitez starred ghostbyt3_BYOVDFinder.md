# BYOVDFinder

BYOVDFinder helps you check which drivers from loldrivers.io are not blocked by the current HVCI (Hypervisor Code Integrity) blocklist on your system. This is particularly useful for BYOVD (Bring Your Own Vulnerable Driver) attack paths where vulnerable drivers are permitted to load despite HVCI being enabled.

> This project has also been updated with real-time analysis of LOLDrivers against Microsoft's HVCI blocklist. You can find the project here: [byovd-watchdog](https://github.com/pwnfuzz/byovd-watchdog). For a better viewing experience, visit: [https://byovd-watchdog.pwnfuzz.com](https://byovd-watchdog.pwnfuzz.com/).
