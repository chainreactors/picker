**报名时间：7月19日-8月14日**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/1brjUjbpg5xglQajuOjgBrlvo6TCWHBBAooECRaXhwrhymWicHLCGh5gTmzzLl3hsxEKfY1RbeKOMVkXCEypqkQ/640?wx_fmt=png&from=appmsg)

本次大赛以团队竞赛为主，围绕数字化转型中常见的风险场景，搭建专业网络安全竞技平台，创新升级比赛赛制，遴选顶尖网络安全人才，为数字时代的智慧城市建设保驾护航。

本次大赛分为初赛与决赛两个阶段，初赛主要囊括**“理论考试+CTF场景实操”**两大板块，理论题目涉及网络安全政策法规、安全防护、密码学等多个方面，场景实操则采用“CTF夺旗赛”模式，综合考察参赛者不同维度的网络安全综合技术水平。

在初赛脱颖而出的20支优秀团队，将与4支特邀战队以及2支极客少年挑战赛优胜战队共同进入决赛阶段。线下决赛在往届城市网络安全演习的基础上拓宽赛道，聚焦网络安全产业高质量发展需求，采用**“大模型安全检测”“实网攻防演练”“靶场演练”**三种竞赛模式展开，充分检验选手的实战实践能力，促进人才培养与市场需求紧密结合，持续为国家输出高质量的网络安全人才。

**参赛须知**

|  |  |
| --- | --- |
| **1** | **参赛人员：**  面向全国广泛招募参赛对象，所有参赛人员须拥有中国国籍（不含港澳台地区）。 |
| **2** | **队伍组成：**  参赛者须以团队为单位报名，每支队伍须设队长1名，队长必须是参赛队员，负责参赛队伍的组织管理。每支队伍参赛队员不超过4人（含队长）。参赛队员只能隶属于一支队伍，比赛晋级过程中如果有队员中途退出，不能补位，队长退出可由队内其他队员担任。 |
| **3** | **命名规则：**  参赛队名不得违反国家法律和社会公德，不得侵犯他人的品牌权益、知识产权、名誉权等合法权利。若违规，组委会有权要求参赛队伍修改战队名称，如拒绝更改，组委会有权取消其参赛资格。 |
| **4** | **报名形式：**  由队长统一进行在线报名，所预留的手机号需为**“i春秋”****账号，**报名完成后可通过专题页的报名查询功能查看是否已报名成功。 |
