# Fastly 重磅升级 DDoS 防护，护航企业网络安全

发布时间 : 2025-04-09 14:50:59

本文是翻译文章，文章原作者 Melania Watson，文章来源：securitybrief

原文地址：<https://securitybrief.asia/story/fastly-unveils-advanced-real-time-ddos-attack-insights>

译文仅供参考，具体内容表达以及含义原文为准。

![Story image]()

Fastly公司宣布对其分布式拒绝服务（DDoS）防护服务进行一系列更新，推出了新功能，旨在为各组织机构提供有关分布式拒绝服务攻击缓解工作的实时可见性。

Fastly公司今日发布的新 “攻击洞察”（Attack Insights）功能，使安全团队能够在 DDoS攻击发生时观察其活动，实时评估缓解策略，并验证自动化规则在阻止恶意流量的同时，是否保障了对应用程序和应用程序编程接口（API）的合法访问。

Fastly公司首席产品官 Kip Compton 表示：“安全团队和平台团队需要相信，自动化防护措施能够在不影响关键业务流量的情况下有效阻止攻击。通过 Attack Insights 功能，我们为各组织机构提供了所需的透明度，使他们能够对缓解攻击的工作充满信心，并优化安全响应措施，而且无需具备深厚的安全专业知识。”

DDoS攻击旨在通过大量涌入的流量使应用程序和 API 不堪重负，可能会导致严重的服务中断和经济损失。虽然自动化防御手段被广泛应用，但一个普遍存在的挑战是，难以了解其实际效果。

Fastly公司的这些增强功能旨在通过提供有关攻击以及所采用的缓解方法的详细、实时数据来解决这一问题。

Fastly公司的新功能包括：

****1.********事件（Events）****：帮助识别和区分单个 DDoS 攻击，为安全团队提供实时威胁背景信息。

****2.********事件详情（Event Details）****：针对每起事件，提供详细的洞察信息，如请求样本、流量汇总以及缓解措施。

****3.********规则详情（Rule Details）****：展示临时缓解规则及其具体属性，使团队能够评估规则的有效性，同时确保合法用户流量不受影响。

网站优化和性能解决方案提供商 Yottaa 公司的客户服务副总裁 Ben Trafton 对这些新功能表示欢迎。

Trafton 称：“Fastly公司的 DDoS 防护服务对我们的客户来说非常有效，这给我们留下了深刻印象，现在我们真的可以看到它在实际行动中的强大能力。Attack Insights功能以及能够在几秒钟内查看自动生成的缓解规则的能力，将加快我们的报告速度，这对于我们帮助客户保障安全的使命来说极具价值。”

Fastly公司表示，其更新后的 DDoS 防护服务使用户能够直观地看到攻击时间线，并访问详细的攻击数据，而无需进行手动配置或具备高级安全技能。

该解决方案使用了包括属性揭露（Attribute Unmasking）在内的专有技术，以区分合法流量和恶意活动。

除了这些新的可见性功能外，Fastly公司还强调了其致力于通过可预测的定价来简化 DDoS 防护服务，并且对攻击流量不收取额外费用。

Fastly公司的 DDoS 防护服务只需单击一下即可激活，使各组织机构能够在不增加复杂性或成本的情况下加强其安全防护态势。

Attack Insights 功能现已向现有的 Fastly公司 DDoS 防护服务客户开放，客户可直接通过 Fastly 控制面板访问该功能。
