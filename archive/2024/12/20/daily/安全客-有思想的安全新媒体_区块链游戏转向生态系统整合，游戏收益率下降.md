# 区块链游戏转向生态系统整合，游戏收益率下降

![Game controllers. Image created by Decrypt using Canva]()

据一家专注于推动游戏领域技术发展的组织称，区块链游戏正在进入一个新的成熟阶段，越来越多的玩家可以更加无缝地访问游戏。

然而，随着传统游戏开发商的加入，该领域也发生了一些变化，可能会带来巨大的转变。

根据区块链游戏联盟（Blockchain Game Alliance，BGA）本周发布的一份行业调查报告，索尼（Sony）和Square Enix等大型工作室正通过第二层解决方案进入这一领域，而这一变化恰好与潮汐相吻合。

然而，公众的看法仍然是一个关键障碍。

报告显示，66%的项目仍在努力建立可信度，与区块链游戏与骗局有关的误解作斗争。

尽管如此，玩家进入区块链游戏的便利性已经得到改善，上手摩擦率从去年的79.5%下降到2024年的53.9%。

这主要归功于更好的用户体验，让玩家在接触区块链游戏时不再那么麻烦。

BGA总裁塞巴斯蒂安-博尔格特（Sebastian Borget）在报告中写道：“这一增长的核心是玩家赋权原则。”

Borget 谈到了所有权的说法，这也是吸引玩家关注区块链体验的主要原因。报告显示，超过 71% 的受访者认为这是 “区块链游戏带来的最大好处”。

“自上一轮市场周期以来，人们在用户体验和用户界面方面投入了更多精力，优先考虑类似于玩家对Web2游戏的期望的无缝体验，”报告中的一段话写道。

该联盟指出，这一趋势将在2025年扩大，新的重点是玩家体验而不是金融机制。

区块链游戏平台正在从公开的加密元素（如代币经济和NFT市场）转向无缝集成，Web3功能将在后台运行。

**玩赚游戏转向新元**

随着2025年的到来，区块链游戏面临着新的元，区块链游戏联盟（Blockchain Game Alliance）与Decrypt分享了其预测。

BGA表示，从传统的中心化游戏服务器向基于区块链的基础设施过渡，表明了游戏是如何接受所有权叙事的。

传统游戏的稳定性和数据管理依赖于区域服务器网络，而区块链集成则通过链上资产跟踪和交易实现了真正的数字所有权。

在这方面，2020 年出现的 “玩到赚 ”模式也发生了巨大转变，更加关注如何将链上特性用于游戏。

这种转变催生了基于功绩和活动驱动的迭代，Web3 社区中更为人熟知的 Zyori Andrew Campbell 指出。

坎贝尔在报告中写道：“最初的‘游戏赚钱’模式存在两个关键缺陷：超通胀奖励随着用户的采用而不受控制地增长，以及缺乏足够的代币汇来产生通缩压力。”

**完全链上游戏和自主世界**

报告还提出了两个新术语：自主世界（AW）和完全链上游戏（FOCGs）。这两个术语将游戏描述为与链密切相关并完全在链上生存的自主体验。

这包括游戏规则、玩家物品和游戏内经济，而不仅仅是在链上保留部分内容，同时在常规的集中式服务器上运行游戏的主客户端。

自主世界通过将 “游戏状态和逻辑元素置于链上”，使游戏能够 “根据社区意见进行改进”，Sovrun（前身为 BreederDAO）的参谋长 Gillian Pua 告诉 Decrypt。

随着人工智能的进步与区块链基础设施的融合，这方面也会影响到在不久的将来，链上游戏可以如何完全运行。

Pua说：“随着链上透明度的提高，AW也为人工智能代理的蓬勃发展提供了完美的基础，使智能自适应游戏能够对玩家的实时行为做出响应。”
