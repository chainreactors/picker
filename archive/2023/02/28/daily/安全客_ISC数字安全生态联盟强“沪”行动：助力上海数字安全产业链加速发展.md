# ISC数字安全生态联盟强“沪”行动：助力上海数字安全产业链加速发展

ISC

安全客

2月24日，第三届上海数字创新大会开幕式在上海市普陀区中以（上海）创新园会议中心拉开帷幕。本次大会由上海市经济和信息化委员会、上海市科学技术委员会、上海推进科技创新中心建设办公室、上海市普陀区人民政府主办，上海市普陀区科学技术委员会、桃浦智创城、中以（上海）创新园、360集团承办，ISC平台等支持。

开幕式上，**上海普陀区副区长蒋龙，360数字安全集团副总裁、ISC品牌主理人卜思南，上海市信息安全行业协会副会长张照龙共同发布了《ISC上海数字安全创新能力全景图谱》。**这份图谱是ISC数字生态联盟助力上海提升城市数字安全运营能力，促进创新链、产业链融合，带动上海相关产业链加速发展的新范本。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Ok4fxxCpBb67OibJ6YibTakMT8zFFSibvqRtKoq6dHOdYmS8v1iazTa4pIOFPskibQdibqcWy8dPRtJfmmByY89UnaRA/640?wx_fmt=jpeg)

**双链融合**

**打造城市级数字安全创新图谱**

**▲▲▲**

近年，上海提出建设具有世界影响力的国际数字之都，全面推进城市数字化转型建设，并始终重视统筹发展与安全。在此过程中，上海普陀区依托自身扎实的产业基础、丰富的科创资源、独特的区位优势，持续规划建设普陀网络安全产业园。

为加快推动上海数字安全产业“圈地”发展，普陀区政府联合ISC平台，通过对上海城市数字安全体系建设全生命周期所涉及的创新技术、产品加以系统梳理，绘制了《ISC上海数字安全创新能力全景图谱》。图谱涵盖数据安全、安全运营、端点安全、供应链应用安全、工业互联网安全、威胁检测与响应、攻击面与资产管理、云安全、零信任、安全访问服务边缘SASE、流量安全、移动安全、身份安全、密码管理**14大热点安全领域近百家上海优质创新企业，包含360数字安全集团、四叶草安全、山石网科、格尔软件、指掌易、赛讯验证等。**

![](https://mmbiz.qpic.cn/mmbiz_png/Ok4fxxCpBb67OibJ6YibTakMT8zFFSibvqRw8XaXia8GF5wtia6BQuux4BovQuPpibSXyceXwziaAuqdqICZRKVgFM3Ww/640?wx_fmt=png)

这份崭新的数字安全维基百科图谱通过创新产品、技术与产业的深度融合，系统性地展示出上海数字安全创新领域的代表性厂商。未来将根据上海产业需求及发展持续进行动态调整，为上海政府、企业单位、投资人以及相关机构及时掌握区域产业发展现状、预测产业发展方向提供科学参考，全面赋能各数字化业务场景下的安全建设。

**全国标杆**

**ISC数字安全生态联盟持续助力城市“强链”**

**▲▲▲**

“安全” 是党的二十大报告中的高频词。身处数字时代，在日益严峻的国际安全形势和不可避免的数字经济内在安全风险双重挑战之下，提升数字安全能力成为发展数字经济的战略底座。

为服务数字安全国家战略，构建数字安全良性生态。**ISC基于自身十年能力沉淀，正式成立数字安全生态联盟。**通过聚拢产业链上中下游生态伙伴，做好需求分析和系统研究，培育壮大产业新增长点，同时加强政府、企业、高等院校、科研院所等各主体间的沟通交流，促进生态合作，构建多方参与、优势互补、融合发展的产业生态体系。

360数字安全集团副总裁、ISC品牌主理人卜思南表示，**此次与上海“牵手”是ISC数字安全生态联盟城市合作计划的首站**，未来联盟将为服务数字安全国家战略，帮助多个城市引进、培育数字安全龙头企业和创新独角兽企业，带动千亿级规模数字安全上下游产业链聚集，助力各城市数字经济加速发展。为抢抓城市产业集群发展机遇，卜思南呼吁更多数字安全创新企业加入联盟，实现资源整合，携手共建城市数字安全产业。
