# 【安全圈】迪士尼宣布弃用 Slack 平台

安全圈

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGylgOvEXHviaXu1fO2nLov9bZ055v7s8F6w1DD1I0bx2h3zaOx0Mibd5CngBwwj2nTeEbupw7xpBsx27Q/640?wx_fmt=other&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1&wx_co=1)

**关键词**

数据泄露

**一、 背景**

今年7月黑客组织 NullBulge 公布了该娱乐巨头数千个 Slack 频道的数据，包括计算机代码和未发布项目的详细信息。

消息称这些数据来自迪士尼的 Slack 工作场所通讯工具，**涉及 4400 多万条信息。**

该公司曾在 8 月份表示，正在调查未经授权从其一个通信系统中泄露**超过 1 TB 数据的事件。**

SentinelOne 的威胁情报和恶意软件分析团队称，NullBulge 通过利用 GitHub 和 Hugging Face（协作编码平台）上的代码来破坏软件供应链，并诱使用户下载恶意文件。

[【安全圈】迪士尼遭黑客入侵超1TB资料外泄，外泄资料涉及项目计划和财务数据](http://mp.weixin.qq.com/s?__biz=MzIzMzE4NDU1OQ==&mid=2652063000&idx=4&sn=bd9c3923d3541ecb86edcc720a34cefa&chksm=f36e6958c419e04e25e9e17769cd33421fe0140a9cc7772946bc5fbb2fb12f2cc6e52712bf33&scene=21#wechat_redirect)

**二、 措施**

迪士尼首席财务官休・约翰斯顿（Hugh Johnston）表示，**公司的大部分业务将在今年晚些时候停止使用 Slack 服务。**
