## Getting Started

### Installation

*Only need latest version of Impacket*

1. Clone the impacket repository

   ```
   git clone https://github.com/fortra/impacket
   ```
2. Install imapcket

   ```
   cd imapcket && sudo pip3 install .
   ```
3. Enjoy it :)

   ```
   git clone https://github.com/XiaoliChan/zerologon-Shot.git
   ```
