## GitHub 用户数突破 1 亿

[![开发者](https://icon.solidot.org/images/topics/topicdevelopers.png?123)](/search?tid=11 "开发者")

[Wilson](/~Wilson) (42865)发表于 2023年01月29日 23时28分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73981&appkey=1370085986&title=GitHub%20%E7%94%A8%E6%88%B7%E6%95%B0%E7%AA%81%E7%A0%B4%201%20%E4%BA%BF "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自帕迪多街车站**

GitHub 宣布其用户数突破 1 亿。GitHub 成立于 2008 年，2013 年用户数突破 300 万，2018 年达到 2800 万，2022 年 10 月突破 9000 万，三个月后突破了 1 亿。GitHub 称这 1 亿用户是 1 亿开发者，这个数字很多人有不同看法，它的用户未必就是开发者，而一名开发者可能有多个账号，如个人账号和工作账号，真正的开发者人数可能只是这一亿的一小部分。GitHub 称在 2015 年三分之一的用户来自北美，今天仅仅印度一个国家的用户就超过 1000 万。
https://github.blog/2023-01-25-100-million-developers-and-counting/
