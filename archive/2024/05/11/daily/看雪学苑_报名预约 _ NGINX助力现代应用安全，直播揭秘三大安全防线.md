随着数字经济的蓬勃发展，现代应用架构已成为企业和组织发展的主流。然而，现代应用和微服务的快速增长也带来了潜在的安全漏洞和威胁。

为了有效应对不断增长的安全威胁和安全“左移”的趋势，NGINX作为被广泛应用于现代应用前端和K8s集群入口的软件负载均衡，提供了关键的安全防护能力，对现代应用的顺利交付起到了至关重要的作用。

根据最新调查显示，90%的企业和组织正在发展现代应用架构，而53%的用户更倾向于使用构建在NGINX开源版基础上的应用安全插件。为了满足用户需求，NGINX团队最新推出了NGINX App Protect 5.0版本，不仅支持加载在NGINX Plus上使用，还可基于NGINX开源版部署，为更广泛的现代应用提供高级安全防护支持。

F5 NGINX 解决方案顾问

超过 16 年以上 IT 从业经验，目前主要负责银行、证券等金融客户。在加入 F5 公司之前，曾经就职于 Cisco、Radware 等公司。精通大型数据中心网络、安全和应用交付的架构设计。拥有多家银行、证券、支付、制造业、运营商等用户的网络架构设计、应用交付和应用安全设计经验。拥有三个方向的 CCIE（CCIE RS/DC/SP) ，Vmware VCP5，CKA，CKS 和 AWS SAP 等多个认证。

NGINX App Protect提供全面的WAF安全防护和DoS防御解决方案，有效保护应用和API免受高级七层攻击。其部署环境非常灵活，无论是本地部署还是云端，都能满足用户的安全需求。

此外，F5 NGINX 根据流量处理的三个阶段总结了现代应用的三大安全防线：

* 防线 1 —— 检测并防范拒绝服务 (DoS) 攻击
* 防线 2 —— 抵御 OWASP 十大安全风险和恶意 bot 程序
* 防线 3 —— 使用 NGINX 原生功能以及集成的第三方单点登录 (SSO) 解决方案，对应用和 API 客户端进行身份验证和授权
