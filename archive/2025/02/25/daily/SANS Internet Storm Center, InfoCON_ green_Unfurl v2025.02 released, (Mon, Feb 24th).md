# [Unfurl v2025.02 released](/forums/diary/Unfurl%2Bv202502%2Breleased/31716/)

**Published**: 2025-02-24. **Last Updated**: 2025-02-24 20:41:04 UTC
**by** [Jim Clausing](/handler_list.html#jim-clausing) (Version: 1)

[0 comment(s)](/diary/Unfurl%2Bv202502%2Breleased/31716/#comments)

I've been a big fan of Ryan Benson's [unfurl](https://dfir.blog/introducing-unfurl/)[1] tool since he released it a little over 5 years ago. Unfurl is a tool that can parse/decode URLs including things like embedded timestamps and IP addresses. It can be run in gui form via a web browser or as a command-line tool (my preference). Well, last week, Ryan released an update to [v2025.02](https://dfir.blog/unfurl-parses-obfuscated-ip-addresses/)[2,3] of unfurl and added the ability to decode BlueSky URLs (among other bugfixes). I've also updated my [docker container](https://hub.docker.com/repository/docker/clausing/dfir-unfurl/general)[4] to run the command-line version of unfurl as well.
