# 推动网络安全走向云化 微步三度上榜Cloud 100 China榜单

![](https://mmbiz.qpic.cn/mmbiz_gif/Yv6ic9zgr5hRYwmkFFVSsK0fQGJBGqwl6iaBoFgqTpPricWCuX7uIb4Rj7eibLo3ibOiaOtqo7vXEnibKhxuInrceOoibg/640?wx_fmt=gif&wxfrom=5&wx_lazy=1&tp=webp)

9月5日，2024 Cloud 100 China 榜单正式发布，这是靖亚资本和崔牛会联合推出的第三届榜单。**微步凭借多年技术引领和稳定增长连续三年上榜**，在持续推动网络安全走向云化的过程中展示了卓越的技术实力和发展潜力。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Yv6ic9zgr5hQanMJ5HF9YmbjCc3sU29QdibzVCnicn6wHQia4afAL9FPuaylm7iaJRZuLFV88ibXOgNA2O6gvsLWPZ5Q/640?wx_fmt=jpeg&from=appmsg)

Cloud 100 China 榜单，持续关注中国 Cloud 行业最具潜力的公司，以公开、公平、透明的标准遴选出其中的 Top100。本届评选基于过去一年融资环境和企业经营的挑战，进一步增加对企业运营指标的关注，更多地考察 Cloud 企业的业务增长能力和盈利能力，从而呈现出一份能代表2024年最优秀的中国 Cloud 企业的榜单。

自2015年创立至今，微步一直是国内安全 SaaS 规模化发展的倡导者、践行者和推动者，利用威胁情报和人工智能等先进技术，打造了国内首个基于海量情报数据的安全云，以此构建“**云+流量+边界+端点**”新一代智慧安全运营产品及服务体系，帮助客户建立全生命周期的威胁监控体系和安全响应能力。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Yv6ic9zgr5hQanMJ5HF9YmbjCc3sU29QdpwwiaA14uxaajLWk25Fx7An2w0iapDeTjjXGX8hvficpRqwn9Cicbzj7sQ/640?wx_fmt=jpeg&from=appmsg)

以终端安全为例，在由被动防御向主动防御阶段转变过程中，SaaS 化式 EDR 逐渐成为未来终端最有效的防护方式之一。微步新一代终端安全管理平台 OneSEC 基于微步安全云的专业 EDR 模块，通过安装在终端上的轻量级 Agent 实时采集终端全量行为日志数据，结合云端动态 IOA 与静态 IOC 海量情报，运用图关联检测技术可从多个维度交叉检测、综合评判，让内存马、无文件攻击、域前置等高级攻击手法无处遁形。

在部署模式上，OneSEC 采用云端 SaaS 模式，无需采购硬件，企业只需申请一个账号，即可将分散在全国的办公终端纳入统一管理。目前，来自金融、央企、大型互联网企业等头部客户纷纷接受并认可 OneSEC 的安全云服务，充分体现了微步 SaaS 化终端产品的安全性和高可靠性。据统计，OneSEC 在金融行业的成单客户中，SaaS 部署占比高达 93.3%。

随着企业数字化转型的深入，网络安全逐步走向云化已成必然趋势。此次再登Cloud 100 China 榜单是业界对微步安全云化产品技术实力、创新能力的高度认可。未来微步将坚持安全 SaaS 模式，以云端持续运转的安全能力为核心，发力创新技术和产品的同时，更关注客户的使用体验与效果，助力广大政企客户提升安全运营实战水平，护航数字时代的发展。
