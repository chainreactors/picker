дёҠе‘ЁпјҢеңЁ2024е№ҙ7жңҲ19ж—Ҙдё–з•ҢеҚҸи°ғж—¶0409еҲҶпјҢи‘—еҗҚзҡ„йҳІз—…жҜ’еҲ¶йҖ е•ҶCrowdStrikeеҸ‘еёғдәҶе…¶е№ҝжіӣдҪҝз”Ёзҡ„Falconе№іеҸ°зҡ„жӣҙж–°пјҢеҜјиҮҙе…Ёзҗғзҡ„Microsoft WindowsжңәеҷЁеҸ‘з”ҹеҙ©жәғгҖӮ

еҪұе“ҚиҢғеӣҙжһҒдёәе№ҝжіӣгҖӮдҫӣеә”й“ҫе…¬еҸёInterosдј°и®ЎпјҢеӨ§зәҰжңү674,620дёӘCrowdStrikeе’ҢMicrosoftзҡ„зӣҙжҺҘдјҒдёҡе®ўжҲ·е…ізі»еҸ—еҲ°еҪұе“ҚгҖӮжҚ®MicrosoftиЎЁзӨәпјҢжңү850дёҮеҸ°WindowsжңәеҷЁеӨұиҙҘгҖӮиҝҷж¬ЎдәӢд»¶йҷӨдәҶйҖ жҲҗеӨ§йҮҸзҡ„ITиЎҘж•‘ж—¶й—ҙеӨ–пјҢиҝҳеј•еҸ‘дәҶе…ЁзҗғиҲӘзҸӯе’ҢиҲӘиҝҗзҡ„е»¶иҜҜпјҢиҝҷжҳҜз”ұдәҺе№ҝжіӣзҡ„Windowsзі»з»ҹж•…йҡңжүҖйҖ жҲҗзҡ„гҖӮ

CrowdStrikeиҮід»ҠйҖҸйңІзҡ„еҺҹеӣ жҳҜпјҡвҖңдёҖдёӘйҖ»иҫ‘й”ҷиҜҜеҜјиҮҙеҸ—еҪұе“Қзі»з»ҹеҮәзҺ°зі»з»ҹеҙ©жәғе’Ңи“қеұҸпјҲBSODпјүвҖқгҖӮ

жӯӨж¬Ўеҙ©жәғжәҗдәҺеҸҜиғҪиў«зҜЎж”№зҡ„ж•°жҚ®пјҢиҝҷдәӣж•°жҚ®дёҚзҹҘдҪ•ж•…иҝӣе…ҘдәҶеҗҚдёәChannel Fileзҡ„Falconй…ҚзҪ®ж–Үд»¶дёӯпјҢиҜҘж–Үд»¶жҺ§еҲ¶зқҖCrowdStrikeе®үе…ЁиҪҜд»¶зҡ„е·ҘдҪңж–№ејҸгҖӮ

Channel FilesдјҡйҡҸзқҖж—¶й—ҙз”ұCrowdStrikeжӣҙж–°пјҢе№¶жҺЁйҖҒеҲ°иҝҗиЎҢе…¶иҪҜд»¶зҡ„зі»з»ҹдёҠгҖӮеҸҚиҝҮжқҘпјҢиҝҷдәӣжңәеҷЁдёҠзҡ„FalconдҪҝз”Ёж–Үд»¶дёӯзҡ„дҝЎжҒҜжқҘжЈҖжөӢе’Ңеә”еҜ№еЁҒиғҒгҖӮиҝҷжҳҜFalconеҹәдәҺиЎҢдёәзҡ„жңәеҲ¶зҡ„дёҖйғЁеҲҶпјҢиҜҘжңәеҲ¶иҜҶеҲ«гҖҒзӘҒеҮәжҳҫзӨәе№¶йҳ»жӯўи®Ўз®—жңәдёҠзҡ„жҒ¶ж„ҸиҪҜд»¶е’Ңе…¶д»–дёҚеҸ—ж¬ўиҝҺзҡ„жҙ»еҠЁгҖӮ

еңЁиҝҷдёӘжЎҲдҫӢдёӯпјҢж•°зҷҫдёҮеҸ°иҝҗиЎҢFalconзҡ„Windowsз”өи„‘жҺҘж”¶дәҶдёҖдёӘй…ҚзҪ®ж–Үд»¶пјҢиҝҷдёӘж–Үд»¶дҪҝе®үе…ЁиҪҜд»¶еҰӮжӯӨеӣ°жғ‘пјҢд»ҘиҮідәҺж•ҙдёӘзі»з»ҹеҙ©жәғгҖӮеҪ“йҮҚеҗҜеҸ—еҪұе“Қзҡ„жңәеҷЁж—¶пјҢе®ғеҮ д№Һдјҡз«ӢеҚіеҗҜеҠЁFalconпјҢз„¶еҗҺеҶҚж¬Ўеҙ©жәғгҖӮ

ж №жҚ®CrowdStrikeзҡ„иҜҙжі•пјҢWindowsжңәеҷЁдёҠзҡ„Channel FilesеӯҳеӮЁеңЁд»ҘдёӢзӣ®еҪ•дёӯпјҡ

C:\Windows\System32\drivers\CrowdStrike\

иҝҷдәӣж–Үд»¶зҡ„е‘ҪеҗҚи§„еҲҷд»ҘвҖқC-вҖңејҖеӨҙпјҢеҗҺйқўи·ҹзқҖдёҖдёӘе”ҜдёҖзҡ„ж ҮиҜҶеҸ·гҖӮеҮәй—®йўҳзҡ„ж–Үд»¶еҗҚд»ҘвҖқC-00000291-вҖңејҖеӨҙпјҢеҗҺйқўи·ҹзқҖе…¶д»–ж•°еӯ—пјҢе№¶д»Ҙ.sysжү©еұ•еҗҚз»“е°ҫгҖӮдҪҶCrowdStrikeжҢҮеҮәпјҢиҝҷдәӣе№¶дёҚжҳҜеҶ…ж ёй©ұеҠЁзЁӢеәҸпјӣдәӢе®һдёҠпјҢе®ғд»¬жҳҜдҫӣFalconдҪҝз”Ёзҡ„ж•°жҚ®ж–Үд»¶пјҢFalconзЎ®е®һеңЁй©ұеҠЁзә§еҲ«иҝҗиЎҢгҖӮ

д№ҹе°ұжҳҜиҜҙпјҢжҚҹеқҸзҡ„й…ҚзҪ®ж–Үд»¶дёҚжҳҜй©ұеҠЁеҸҜжү§иЎҢж–Үд»¶пјҢдҪҶе®ғжҳҜз”ұCrowdStrikeй«ҳеәҰдҝЎд»»зҡ„д»Јз ҒеӨ„зҗҶзҡ„пјҢиҜҘд»Јз Ғиў«е…Ғи®ёеңЁж“ҚдҪңзі»з»ҹдёҠдёӢж–ҮдёӯиҝҗиЎҢпјҢеҪ“еқҸж–Үд»¶еҜјиҮҙиҝҷж®өд»Јз ҒеҮәй”ҷж—¶пјҢе®ғеёҰдёӢдәҶж•ҙдёӘе‘Ёеӣҙзҡ„ж“ҚдҪңзі»з»ҹвҖ”вҖ”еңЁиҝҷдёӘж•…дәӢдёӯе°ұжҳҜMicrosoft WindowsгҖӮ

вҖңChannel File 291жҺ§еҲ¶FalconеңЁWindowsзі»з»ҹдёҠиҜ„дј°е‘ҪеҗҚз®ЎйҒ“жү§иЎҢзҡ„ж–№ејҸгҖӮе‘ҪеҗҚз®ЎйҒ“з”ЁдәҺWindowsдёӯзҡ„жӯЈеёёгҖҒиҝӣзЁӢй—ҙжҲ–зі»з»ҹй—ҙзҡ„йҖҡдҝЎгҖӮвҖқCrowdStrikeеңЁе…¶е‘Ёжң«еҸ‘еёғзҡ„жҠҖжңҜж‘ҳиҰҒдёӯи§ЈйҮҠйҒ“гҖӮ

й…ҚзҪ®жӣҙж–°и§ҰеҸ‘дәҶдёҖдёӘйҖ»иҫ‘й”ҷиҜҜпјҢеҜјиҮҙж“ҚдҪңзі»з»ҹеҙ©жәғ вҖңеңЁ04:09 UTCеҸ‘з”ҹзҡ„жӣҙж–°ж—ЁеңЁй’ҲеҜ№ж–°и§ӮеҜҹеҲ°зҡ„гҖҒеңЁзҪ‘и·Ҝж”»еҮ»дёӯз”ұеёёи§ҒC2жЎҶжһ¶дҪҝз”Ёзҡ„жҒ¶ж„Ҹе‘ҪеҗҚз®ЎйҒ“гҖӮй…ҚзҪ®жӣҙж–°и§ҰеҸ‘дәҶдёҖдёӘйҖ»иҫ‘й”ҷиҜҜпјҢеҜјиҮҙж“ҚдҪңзі»з»ҹеҙ©жәғгҖӮвҖқ

жҚўиЁҖд№ӢпјҢCrowdStrikeеҸ‘зҺ°жҒ¶ж„ҸиҪҜд»¶ж»Ҙз”ЁWindowsзҡ„дёҖдёӘеҗҚдёәе‘ҪеҗҚз®ЎйҒ“зҡ„еҠҹиғҪпјҢд»ҘдёҺиҜҘжҒ¶ж„ҸиҪҜд»¶зҡ„жҢҮжҢҘе’ҢжҺ§еҲ¶пјҲC2пјүжңҚеҠЎеҷЁиҝӣиЎҢйҖҡдҝЎпјҢиҝҷдәӣжңҚеҠЎеҷЁйҖҡеёёжҢҮзӨәжҒ¶ж„ҸиҪҜд»¶жү§иЎҢеҗ„з§ҚдёҚиүҜиЎҢдёәгҖӮCrowdStrikeжҺЁйҖҒдәҶдёҖдёӘй…ҚзҪ®ж–Үд»¶жӣҙж–°д»ҘжЈҖжөӢе№¶йҳ»жӯўиҝҷз§Қз®ЎйҒ“зҡ„ж»Ҙз”ЁпјҢдҪҶй…ҚзҪ®ж–Үд»¶еҚҙеј„е·§жҲҗжӢҷпјҢз ҙеқҸдәҶFalconгҖӮ

е°Ҫз®ЎжңүдәәзҢңжөӢй”ҷиҜҜжҳҜз”ұдәҺChannel FileдёӯеҢ…еҗ«з©әеӯ—иҠӮжүҖиҮҙпјҢдҪҶCrowdStrikeеқҡз§°жғ…еҶөе№¶йқһеҰӮжӯӨгҖӮ

вҖңиҝҷдёҺChannel File 291жҲ–е…¶д»–д»»дҪ•Channel FileдёӯеҢ…еҗ«зҡ„з©әеӯ—иҠӮж— е…ігҖӮвҖқиҝҷе®¶зҪ‘з»ңе®үе…Ёе…¬еҸёиЎЁзӨәпјҢе№¶жүҝиҜәиҝӣиЎҢиҝӣдёҖжӯҘзҡ„ж №жң¬еҺҹеӣ еҲҶжһҗд»ҘзЎ®е®ҡйҖ»иҫ‘зјәйҷ·жҳҜеҰӮдҪ•еҸ‘з”ҹзҡ„гҖӮ

е…·дҪ“еҜјиҮҙй”ҷиҜҜзҡ„ж №жң¬еҺҹеӣ е°ҡжңӘжӯЈејҸжҠ«йңІвҖ”вҖ”CrowdStrikeзҡ„CEOд№”жІ»В·з§‘иҢЁпјҲGeorge KurtzпјүеҲҡеҲҡиў«иҰҒжұӮе°ұжӯӨдәӢеҗ‘еӣҪдјҡдҪңиҜҒвҖ”вҖ”дёҚиҝҮеғҸGoogle Project Zeroзҡ„жі°з»ҙж–ҜВ·еҘҘжӣјиҝӘпјҲTavis Ormandyпјүе’ҢObjective-SeeеҲӣе§Ӣдәәеё•зү№йҮҢе…ӢВ·жІғеҫ·е°”пјҲPatrick Wardleпјүиҝҷж ·зҡ„е®үе…Ёдё“е®¶д»ӨдәәдҝЎжңҚең°дәүиҫ©иҜҙпјҢеҶ’зҠҜжҖ§зҡ„Channel Fileд»Ҙжҹҗз§Қж–№ејҸеҜјиҮҙFalconи®ҝй—®дәҶдёҚеӯҳеңЁзҡ„дҝЎжҒҜпјҢи§ҰеҸ‘дәҶеҙ©жәғгҖӮ

зңӢиө·жқҘFalconд»ҺеҶ…еӯҳдёӯзҡ„дёҖдёӘиЎЁиҜ»еҸ–жқЎзӣ®пјҢе№¶еңЁеҫӘзҺҜдёӯдҪҝз”ЁиҝҷдәӣжқЎзӣ®дҪңдёәиҝӣдёҖжӯҘе·ҘдҪңзҡ„еҶ…еӯҳжҢҮй’ҲгҖӮеҪ“иҮіе°‘жңүдёҖдёӘжқЎзӣ®еӣ дёәй…ҚзҪ®ж–Үд»¶иҖҢдёҚжӯЈзЎ®жҲ–дёҚеӯҳеңЁпјҢиҖҢжҳҜеҢ…еҗ«еһғеңҫеҖјж—¶пјҢеҶ…ж ёзә§еҲ«зҡ„д»Јз Ғе°ҶиҝҷдёӘеһғеңҫи§Ҷдёәжңүж•ҲеҖјпјҢд»ҺиҖҢе°қиҜ•и®ҝй—®жңӘжҳ е°„зҡ„еҶ…еӯҳгҖӮ

еӨ„зҗҶеҷЁе’Ңж“ҚдҪңзі»з»ҹжҚ•иҺ·дәҶиҝҷдёӘй”ҷиҜҜзҡ„и®ҝй—®пјҢе№¶еј•еҸ‘и“қеұҸжӯ»жңәпјҢеӣ дёәеңЁйӮЈдёӘзӮ№дёҠж“ҚдҪңзі»з»ҹзҹҘйҒ“еңЁйқһеёёдҪҺзҡ„еұӮйқўеҸ‘з”ҹдәҶж„ҸеӨ–зҡ„дәӢжғ…гҖӮеңЁиҝҷз§Қжғ…еҶөдёӢеҙ©жәғеҸҜиғҪжҳҜжӣҙеҘҪзҡ„йҖүжӢ©пјҢиҖҢдёҚжҳҜе°қиҜ•з»§з»ӯиҝҗиЎҢе№¶иҰҶзӣ–ж•°жҚ®пјҢйҖ жҲҗжӣҙеӨҡзҡ„жҚҹе®ігҖӮ

жІғеҫ·е°”е‘ҠиҜүThe RegisterпјҢеҙ©жәғиҪ¬еӮЁе’ҢеҸҚжұҮзј–жё…жҘҡең°иЎЁжҳҺеҙ©жәғиө·еӣ дәҺиҜ•еӣҫдҪҝз”ЁжңӘеҲқе§ӢеҢ–зҡ„ж•°жҚ®дҪңдёәжҢҮй’ҲвҖ”вҖ”дёҖдёӘйҮҺжҢҮй’ҲпјҢдҪҶжӣҙе…·дҪ“зҡ„з»ҶиҠӮд»Қз„¶жңӘзҹҘгҖӮ

вҖңдҪҶжҲ‘д»¬д»Қз„¶дёҚзҹҘйҒ“зЎ®еҲҮзҡ„еҺҹеӣ пјҢдёәд»Җд№ҲChannel Fileдјҡи§ҰеҸ‘иҝҷдёҖзӮ№пјҢвҖқд»–иҜҙгҖӮ

The RegisterдёҺзҪ‘з»ңе®үе…Ёиө„ж·ұдәәеЈ«Omkhar ArasaratnamдәӨи°ҲпјҢд»–жҳҜOpenSSFзҡ„жҖ»з»ҸзҗҶпјҢе…ідәҺдәӢжғ…жҳҜеҰӮдҪ•еҮәй”ҷзҡ„гҖӮ

ArasaratnamиҜҙпјҢзЎ®еҲҮзҡ„еҺҹеӣ д»Қз„¶жҳҜжҺЁжөӢжҖ§зҡ„пјҢеӣ дёәд»–жІЎжңүжҺҘи§ҰеҲ°CrowdStrikeзҡ„жәҗд»Јз ҒжҲ–WindowsеҶ…ж ёгҖӮ

CrowdStrikeзҡ„Falcon SensorжңүдёӨдёӘз»„д»¶пјҡдёҖдёӘжҳҜеҗҚдёәCSAgent.sysзҡ„з»ҸиҝҮж•°еӯ—зӯҫеҗҚгҖҒеҫ®иҪҜжү№еҮҶзҡ„й©ұеҠЁзЁӢеәҸпјҢеҸҰдёҖдёӘжҳҜз”ЁдәҺз”ЁжңҖж–°зҡ„е®үе…ЁдҝЎжҒҜжӣҙж–°иҪҜд»¶зҡ„Channel FilesгҖӮ

вҖңCrowdStrikeжүҖеҒҡзҡ„еҹәжң¬дёҠжҳҜд»–д»¬жңүдёҖдёӘз»ҸиҝҮзӯҫеҗҚзҡ„й©ұеҠЁзЁӢеәҸпјҢз„¶еҗҺеҠ иҪҪдәҶеӨ§йҮҸзҡ„иҝҷдәӣйҖҡйҒ“й…ҚзҪ®пјҢвҖқArasaratnamиҜҙгҖӮвҖңжҲ‘д»¬дёҚзҹҘйҒ“Channelй…ҚзҪ®ж–Үд»¶еҲ°еә•еҢ…еҗ«д»Җд№ҲгҖӮиҝҷжҳҜж–Үд»¶дёӯжүҖеҗ«еҶ…е®№д»ҘеҸҠCSAgent.sysеҰӮдҪ•и§ЈйҮҠе®ғзҡ„з»“еҗҲгҖӮвҖқ

ж №жҚ®дёҖдёӘе Ҷж Ҳи·ҹиёӘпјҢд»–и§ЈйҮҠиҜҙпјҢCSAgent.sysеӣ жү§иЎҢжүҖи°“зҡ„еқҸжҢҮй’Ҳи§ЈйҷӨеј•з”ЁиҖҢиў«з»ҲжӯўгҖӮе®ғиҜ•еӣҫд»Һең°еқҖ0x000000000000009cи®ҝй—®еҶ…еӯҳпјҢдҪҶиҝҷдёӘең°еқҖе№¶дёҚеӯҳеңЁгҖӮ

вҖңйӮЈжҳҜе®ғдёҚеә”иҜҘи®ҝй—®зҡ„еҶ…еӯҳеҢәеҹҹпјҢвҖқArasaratnamиҜҙгҖӮ

вҖңзҺ°еңЁпјҢеҪ“дҪ жңүдёҖдёӘеғҸиҝҷж ·зҡ„дҪҺзә§зЁӢеәҸж—¶пјҢдҪ е°ұдјҡйҷ·е…ҘдёӨйҡҫеўғең°пјҢеӣ дёәжҖ»дҪ“дёҠеҶ…ж ёеә”иҜҘиҙҹиҙЈж“ҚдҪңзі»з»ҹеҒҡи®ёеӨҡдҪҺзә§зҡ„дәӢжғ…пјҢеҢ…жӢ¬еҲҶй…ҚеҶ…еӯҳпјҢвҖқArasaratnamиҜҙгҖӮ

вҖңжүҖд»ҘеҰӮжһңеҶ…ж ёе®һиҙЁдёҠжҳҜеңЁе°қиҜ•и®ҝй—®е®ғдёҚеә”иҜҘи®ҝй—®зҡ„еҶ…еӯҳпјҢйӮЈд№Ҳд»Һж“ҚдҪңзі»з»ҹзҗҶи®әзҡ„и§’еәҰжқҘзңӢпјҢжӯЈзЎ®зҡ„еҒҡжі•жҳҜеҒҮи®ҫеҲҶй…Қзҡ„жүҖжңүеҶ…еӯҳйғҪдёҚе®үе…ЁпјҢеӣ дёәеҰӮжһңиҝһеҶ…ж ёйғҪдёҚзҹҘйҒ“пјҢйӮЈи°ҒдјҡзҹҘйҒ“е‘ўпјҹеҹәжң¬дёҠеҒңжӯўзі»з»ҹиҝҗиЎҢгҖӮвҖқ

дҪ жҳҜеҗҰдәҶи§Јеҙ©жәғжҳҜеҰӮдҪ•еҸ‘з”ҹзҡ„жӣҙеӨҡдҝЎжҒҜпјҹдҪ еҸҜд»ҘзӣҙжҺҘиҒ”зі»жҲ‘д»¬иҝҷйҮҢжҲ–жҢүз…§иҝҷйҮҢзҡ„иҜҙжҳҺеҢҝеҗҚиҒ”зі»гҖӮ

Arasaratnamи§ЈйҮҠиҜҙпјҢжғ…еҶөзҡ„еӨҚжқӮжҖ§иҝҳеңЁдәҺWindowsй©ұеҠЁжһ¶жһ„зҡ„е·ҘдҪңж–№ејҸгҖӮ

вҖңе·ҘдҪңж–№ејҸжҳҜй©ұеҠЁзЁӢеәҸеҸҜд»Ҙи®ҫзҪ®дёҖдёӘеҸ«еҒҡboot-startзҡ„ж Үеҝ—пјҢвҖқд»–иҜҙгҖӮ

вҖңйҖҡеёёпјҢеҰӮжһңдҪ жңүдёҖдёӘиЎЁзҺ°жңүзӮ№buggyзҡ„й©ұеҠЁзЁӢеәҸпјҢеҜјиҮҙеғҸиҝҷж ·зҡ„ж•…йҡңпјҢWindowsеҸҜд»ҘйҖҡиҝҮз®ҖеҚ•ең°дёӢж¬ЎдёҚеҠ иҪҪиҜҘй©ұеҠЁзЁӢеәҸжқҘиҮӘеҠЁжҒўеӨҚгҖӮдҪҶеҰӮжһңе®ғиў«ж Үи®°дёәboot-startпјҢиҝҷеә”иҜҘжҳҜдёәе…ій”®й©ұеҠЁзЁӢеәҸдҝқз•ҷзҡ„пјҢжҜ”еҰӮдҪ зҡ„зЎ¬зӣҳй©ұеҠЁзЁӢеәҸпјҢWindowsе°ұдёҚдјҡе°Ҷе…¶д»ҺеҗҜеҠЁеәҸеҲ—дёӯеҲ йҷӨпјҢдјҡдёҖж¬ЎеҸҲдёҖж¬Ўең°жҢҒз»ӯеӨұиҙҘпјҢиҝҷе°ұжҳҜжҲ‘д»¬еңЁCrowdStrikeж•…йҡңдёӯзңӢеҲ°зҡ„жғ…еҶөгҖӮвҖқ
