## GNU Texinfo 7.0 发布

[![GNU](https://icon.solidot.org/images/topics/topicgnu.png?123)](/search?tid=44 "GNU")

[WinterIsComing](/~WinterIsComing) (31822)发表于 2022年11月08日 22时08分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73305&appkey=1370085986&title=GNU%20Texinfo%207.0%20%E5%8F%91%E5%B8%83 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自乌鸦从高塔坠落**

GNU 项目的文档系统排版软件包 Texinfo 释出了一个大更新版本 v7.0。主要新特性是支持输出格式 LaTeX 和 EPUB 3，以及大量其它变化如改进 HTML 输出格式支持，修正了字符编码问题，加强了 DocBook 输出。LaTeX 是基于 TeX 的排版系统，以生成高印刷质量的科技和数学、物理文档著称。EPUB 3 是 EPUB 电子书格式的最新一代。
https://lists.gnu.org/archive/html/info-gnu/2022-11/msg00002.html
