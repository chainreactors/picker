支持 GitHub 文件 , 图片、Releases、archive、gist.github.com、raw.githubusercontent.com等文件代理加速下载服务.

### 使用方法：

在链接/后面拼接github.com或者raw.githubusercontent.com等域名链接就可以，包含图片、文件等资源。

**示例：**https://ghproxy.com/github.com/fatedier/frp/releases/download/v0.33.0/frp\_0.33.0\_linux\_amd64.tar.gz
