### SQL Chat是什么

SQL Chat 是一个基于聊天的 SQL 客户端，使用自然语言向数据库提问和查询数据库，小白也可以直接使用母语进行操作使用数据库，可以利用SQL Chat 来查询、操作本地和远程数据库，也可以询问数据库问题或者进行优化等等操作，目前支持MySQL、PostgreSQL、MSSQL三种数据库，项目由 Next.js 构建，国内可直接部署到腾讯云 Web 应用托管服务，也可一键部署到 Vercel，同样支持私有化部署，使用前需要自备OpenAI api key。

### 关于数据保密问题

* 所有数据库连接配置都本地存储在浏览器中，您也可以访问设置到清除数据。
* 只有数据库模式提供给 OpenAI API，表数据不会提供给 OpenAI API。
* 如果使用 sqlchat.ai, 它会记录下这些匿名对话。
