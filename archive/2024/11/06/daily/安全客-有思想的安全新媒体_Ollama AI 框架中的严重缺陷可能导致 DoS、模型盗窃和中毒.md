Oligo Security з ”з©¶е‘ҳ Avi Lumelsky еңЁдёҠе‘ЁеҸ‘еёғзҡ„дёҖд»ҪжҠҘе‘ҠдёӯиҜҙпјҡвҖңжҖ»зҡ„жқҘиҜҙпјҢиҝҷдәӣжјҸжҙһеҸҜд»Ҙи®©ж”»еҮ»иҖ…йҖҡиҝҮеҚ•дёӘ HTTP иҜ·жұӮжү§иЎҢеҗ„з§ҚжҒ¶ж„ҸиЎҢдёәпјҢеҢ…жӢ¬жӢ’з»қжңҚеҠЎпјҲDoSпјүж”»еҮ»гҖҒжЁЎеһӢдёӯжҜ’гҖҒжЁЎеһӢзӣ—зӘғзӯүгҖӮвҖқ

Ollama жҳҜдёҖж¬ҫејҖжәҗеә”з”ЁзЁӢеәҸпјҢе…Ғи®ёз”ЁжҲ·еңЁ WindowsгҖҒLinux е’Ң macOS и®ҫеӨҮдёҠжң¬ең°йғЁзҪІе’Ңж“ҚдҪңеӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүгҖӮиҝ„д»ҠдёәжӯўпјҢе®ғеңЁ GitHub дёҠзҡ„йЎ№зӣ®еә“е·Іиў«еҲҶеҸү 7,600 ж¬ЎгҖӮ

е…ӯдёӘжјҸжҙһзҡ„з®ҖиҰҒиҜҙжҳҺеҰӮдёӢ

* **CVE-2024-39719**пјҲCVSS еҫ—еҲҶпјҡ7.5пјү- ж”»еҮ»иҖ…еҸҜеҲ©з”ЁиҜҘжјҸжҙһдҪҝз”Ё /api/create з«ҜзӮ№зЎ®е®ҡжңҚеҠЎеҷЁдёӯжҳҜеҗҰеӯҳеңЁж–Үд»¶пјҲе·ІеңЁ 0.1.47 зүҲжң¬дёӯдҝ®еӨҚпјү
* **CVE-2024-39720** (CVSS еҫ—еҲҶпјҡ8.2) вҖ“ дёҖеҖӢи¶Ҡз•Ңй–ұи®ҖжјҸжҙһпјҢж”»ж“ҠиҖ…еҸҜеҲ©з”Ё /api/create з«Ҝй»һе°ҺиҮҙжҮүз”ЁзЁӢејҸзөӮжӯўпјҢеҫһиҖҢе°ҺиҮҙ DoS жғ…жіҒпјҲе·ІеңЁ 0.1.46 зүҲжң¬дёӯдҝ®еҫ©пјү
* **CVE-2024-39721** (CVSS еҫ—еҲҶпјҡ7.5) вҖ“ з•¶еӮійҖҒжӘ”жЎҲгҖҢ/dev/randomгҖҚдҪңзӮәијёе…ҘжҷӮпјҢйҮҚиӨҮиӘҝз”Ё /api/create з«Ҝй»һжңғе°ҺиҮҙиіҮжәҗиҖ—зӣЎпјҢжңҖзөӮе°ҺиҮҙ DoS зҡ„жјҸжҙһпјҲе·ІеңЁзүҲжң¬ 0.1.34 дёӯдҝ®еҫ©пјү
* **CVE-2024-39722** (CVSS еҫ—еҲҶпјҡ7.5) вҖ“ api/push з«ҜзӮ№дёӯзҡ„и·Ҝеҫ„йҒҚеҺҶжјҸжҙһпјҢиҜҘжјҸжҙһдјҡжҡҙйңІжңҚеҠЎеҷЁдёҠеӯҳеңЁзҡ„ж–Үд»¶д»ҘеҸҠйғЁзҪІ Ollama зҡ„ж•ҙдёӘзӣ®еҪ•з»“жһ„пјҲе·ІеңЁ 0.1.46 зүҲдёӯдҝ®еӨҚпјү
* дёҖдёӘеҸҜиғҪеҜјиҮҙйҖҡиҝҮ /api/pull з«ҜзӮ№д»ҺдёҚеҸ—дҝЎд»»зҡ„жәҗдёӯжҜ’жЁЎеһӢзҡ„жјҸжҙһпјҲж—  CVE ж ҮиҜҶпјҢжңӘдҝ®иЎҘпјү
* дёҖдёӘеҸҜиғҪеҜјиҮҙйҖҡиҝҮ /api/push з«ҜзӮ№еҗ‘дёҚдҝЎд»»зӣ®ж Үзӣ—еҸ–жЁЎеһӢзҡ„жјҸжҙһпјҲж—  CVE ж ҮиҜҶпјҢжңӘдҝ®иЎҘпјү

еҜ№дәҺиҝҷдёӨдёӘжңӘи§ЈеҶізҡ„жјҸжҙһпјҢOllama зҡ„з»ҙжҠӨиҖ…е»әи®®з”ЁжҲ·йҖҡиҝҮд»ЈзҗҶжҲ–зҪ‘з»ңеә”з”ЁйҳІзҒ«еўҷиҝҮж»ӨжҡҙйңІеңЁдә’иҒ”зҪ‘дёҠзҡ„з«ҜзӮ№гҖӮ

Lumelsky иҜҙпјҡвҖңиҝҷж„Ҹе‘ізқҖпјҢй»ҳи®Өжғ…еҶөдёӢпјҢе№¶йқһжүҖжңүз«ҜзӮ№йғҪеә”иҜҘжҡҙйңІеңЁдә’иҒ”зҪ‘дёҠгҖӮиҝҷжҳҜдёҖдёӘеҚұйҷ©зҡ„еҒҮи®ҫгҖӮдёҚжҳҜжҜҸдёӘдәәйғҪзҹҘйҒ“иҝҷдёҖзӮ№пјҢд№ҹдёҚжҳҜжҜҸдёӘдәәйғҪдјҡиҝҮж»Ө HTTP и·Ҝз”ұеҲ° OllamaгҖӮзӣ®еүҚпјҢиҝҷдәӣз«ҜзӮ№йғҪеҸҜд»ҘйҖҡиҝҮ Ollama зҡ„й»ҳи®Өз«ҜеҸЈи®ҝй—®пјҢиҝҷжҳҜжҜҸж¬ЎйғЁзҪІзҡ„дёҖйғЁеҲҶпјҢжІЎжңүд»»дҪ•еҲҶзҰ»жҲ–ж–ҮжЎЈж”ҜжҢҒгҖӮвҖқ
