# 免费IP查询api接口

要稳定当然要收费的API合适，但是免费的API也能将就使用。下面分享一些免费的API

## IP API

ip-api.com 是一家天花板IP业务公司，提供免费api接口，收费版更是丝滑。

接口示例

* http://ip-api.com/json/117.136.12.79?lang=zh-CN

## 百度IP地址查询

接口示例

* http://opendata.baidu.com/api.php?query=117.136.12.79&co=&resource\_id=6006&oe=utf8

## IP信息网

接口示例

* https://ip.useragentinfo.com/json?ip=117.136.12.79

他家还有ipv6的，但是需要自己做一下解析

## VORE

个人搭建的免费的代码接口，稳定性比不上大厂

* https://api.vore.top/api/IPdata?ip=117.136.12.79

## 太平洋

速度挺快的  不支持跨域

* http://whois.pconline.com.cn/ipJson.jsp?ip=39.100.139.241&json=true

## saintic

速度快  支持跨域

* https://open.saintic.com/ip/rest
  实际上 /ip 下还有其他接口
  /ip/myip 仅查询访问者IP；
  /ip/addr 查询访问者IP及信息，可用查询参数 ip 改为查询其他值。

## useragentinfo

速度快  支持跨域

* https://ip.useragentinfo.com/json

## ipapi

速度快  支持跨域

* https://ipapi.co/json
