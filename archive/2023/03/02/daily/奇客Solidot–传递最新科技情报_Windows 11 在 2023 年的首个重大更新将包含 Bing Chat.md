## Windows 11 在 2023 年的首个重大更新将包含 Bing Chat

[![Windows](https://icon.solidot.org/images/topics/topicwindows.png?123)](/search?tid=33 "Windows")
[![人工智能](https://icon.solidot.org/images/topics/topicAI.png?123)](/search?tid=151 "人工智能")

[Wilson](/~Wilson) (42865)发表于 2023年03月01日 20时10分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74268&appkey=1370085986&title=Windows%2011%20%E5%9C%A8%202023%20%E5%B9%B4%E7%9A%84%E9%A6%96%E4%B8%AA%E9%87%8D%E5%A4%A7%E6%9B%B4%E6%96%B0%E5%B0%86%E5%8C%85%E5%90%AB%20Bing%20Chat "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自飞行村**

微软官方博客宣布了 Windows 11 在 2023 年的首个大更新，Windows 11 v22H2 用户现在可以通过手动检查 Windows Update 下载，或者等待 3 月的例行安全更新（第二个星期二）。其中最引入瞩目的功能是整合了 AI 的新必应和基于 OpenAI ChatGPT 技术的 Bing Chat。其它更新包括记事本应用加入了暗模式支持，多标签支持——可以在单一窗口打开多个标签；设置应用包含了能耗推荐部分，优化电池续航；等等。
https://blogs.windows.com/windowsexperience/2023/02/28/introducing-a-big-update-to-windows-11-making-the-everyday-easier-including-bringing-the-new-ai-powered-bing-to-the-taskbar/
