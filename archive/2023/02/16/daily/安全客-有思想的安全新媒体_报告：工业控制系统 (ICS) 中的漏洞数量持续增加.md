# 报告：工业控制系统 (ICS) 中的漏洞数量持续增加

译文仅供参考，具体内容表达以及含义原文为准。

![]()

根据工业网络安全公司 SynSaber 的一份新报告，在工业控制系统 (ICS) 中发现的漏洞数量持续增加，其中许多漏洞的严重性等级为“严重”或“高”。 评级为“严重”的漏洞数量增长更为显著，从2021年的186个增加到2022年的近300个。根据其 CVSS 评分，总共有近1,000个漏洞为“严重”或“高严重性”。

该报告比较了CISA在2020年和2022年期间发布的 ICS 和 ICS 医疗咨询数量。虽然2021年和2022年的咨询数量大致相同，均为350个，但去年发现的漏洞数量达到1342个，而2021年为1191个前一年。[[阅读原文]](https://hackernews.cc/archives/43270)
