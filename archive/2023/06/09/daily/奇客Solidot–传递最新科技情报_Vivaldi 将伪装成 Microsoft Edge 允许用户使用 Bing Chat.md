## Vivaldi 将伪装成 Microsoft Edge 允许用户使用 Bing Chat

[![人工智能](https://icon.solidot.org/images/topics/topicAI.png?123)](/search?tid=151 "人工智能")
[![Chromium](https://icon.solidot.org/images/topics/topicChromium.png?123)](/search?tid=155 "Chromium")

[Wilson](/~Wilson) (42865)发表于 2023年06月08日 17时21分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75192&appkey=1370085986&title=Vivaldi%20%E5%B0%86%E4%BC%AA%E8%A3%85%E6%88%90%20Microsoft%20Edge%20%E5%85%81%E8%AE%B8%E7%94%A8%E6%88%B7%E4%BD%BF%E7%94%A8%20Bing%20Chat%20 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自绿里**

基于 Chromium 的浏览器 Vivaldi 释出了 v6.1，创始人谭咏文表示新版的一项新功能是能伪装成 Microsoft Edge，允许用户绕过限制使用 Bing Chat。谭咏文称，很多网站和服务会根据用户代理字符串（User-Agent strings）限制其它浏览器，其中最著名的是 Google 的服务，因此在 2019 年他们修改了 Vivaldi 的用户代理字符串冒充 Google Chrome 以实现更好的网站兼容性。到了 2023 年，历史再次重复了。用户如果想尝试最近火爆的 AI 聊天机器人 Bing Chat，他们需要使用 Microsoft Edge。因此在最新版本中，Vivaldi 将能冒充 Microsoft Edge，帮助用户绕过使用限制。
https://vivaldi.com/blog/vivaldi-on-android-6-1/
