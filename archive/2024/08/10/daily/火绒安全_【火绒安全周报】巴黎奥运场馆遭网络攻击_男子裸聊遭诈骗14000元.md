# 巴黎奥运会场馆遭勒索软件攻击

#

近日，正在举办巴黎奥运会击剑和武术比赛的法国大皇宫博物馆服务器遭遇勒索软件攻击，影响了卢浮宫、凡尔赛宫、奥赛和毕加索博物馆在内的众多场馆。据悉，攻击者试图通过加密财务数据来索要赎金，并威胁如果不支付赎金将在48小时内发布加密数据。尽管发生了网络攻击，但法国警方和网络安全机构ANSSI迅速介入，且表示巴黎奥运会的运营并未受到影响以及尚未观察到任何数据泄露。同时已提交辞呈的法国总理Gabriel Attal表示，近期安全部门已经成功防御了68次网络攻击，其中包括针对奥运场馆的两次。

# 美国背调公司数据泄露影响近30亿人

#

近日，在知名黑客平台上，一名自称为“Fenice”的黑客泄露了29亿条国家公共数据，泄露的数据通过两个链接进行分享，总量达到277GB。经过分析，这些数据不仅包括用户的全名、地址、城市、县、州和邮政编码，还包括未加密的SSN。鉴于SSN在美国是识别个人身份和进行金融交易的关键信息，且具有终生性，因此这次泄露对美国用户构成了巨大的安全威胁，可能导致受害者面临身份被盗窃、信用账户被伪造、贷款欺诈和虚假报税等问题。

# 移动设备管理公司遭攻击影响上万台学校设备

#

近期，总部位于英国的移动设备管理公司Mobile Guardian遭遇黑客攻击，导致其平台上注册的部分iOS和ChromeOS设备数据被访问。此次事件影响了全球用户，其中新加坡教育部表示，26所学校的1.3万名学生的设备（包括iPad和Chromebook）均被攻击者远程抹除。目前事件仍在调查中，Mobile Guardian在声明中表示，没有证据表明攻击者获取了用户数据。但该事件引发了人们对Mobile Guardian平台的安全态势及其保护敏感用户数据的能力的严重质疑。

# 男子裸聊遭诈骗14000元

#

据公安部网安局报道，市民张某通过广告添加陌生人QQ好友，聊天中对方言语挑逗提出视频聊天的邀请，张某经不住诱惑，跟对方开视频裸聊。随后，对方以该视频威胁男子两次转账共14000元，张某意识到被骗遂报警。七夕将至，嫌疑人诈骗套路层出不穷，可能会通过社交平台引诱受害者安装木马软件，同时截取受害者面容合成不雅视频，对受害人进行敲诈、勒索钱财。建议广大用户加强个人信息保护意识，远离陌生链接、陌生软件、陌生二维码。

# 美国国务院1000万美元悬赏六名伊朗黑客

#

近期，美国国务院宣布对六名伊朗黑客悬赏1000万美元，这些黑客涉嫌去年对美国水务公司发起网络攻击。据悉，他们与伊朗伊斯兰革命卫队（IRGC）有关，并被指控破坏工业控制系统。美国财政部随后对这些黑客实施了制裁，冻结了他们在美的资产，并禁止美国公民与他们进行交易。网络安全局（CISA）已提醒美国水务运营商加强安全措施，更改设备默认密码，以防止未授权访问。尽管没有证据显示安全饮用水供应受损，但美国政府对这些网络攻击构成的威胁表示严重关切，并通过“正义奖赏”计划鼓励公众提供相关信息。
