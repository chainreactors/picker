ж №жҚ® CYFIRMA зҡ„еҲҶжһҗпјҢж”»еҮ»е§ӢдәҺдёҖдёӘз»ҸиҝҮж··ж·ҶеӨ„зҗҶзҡ„ JavaScript ж–Үд»¶пјҢиҜҘж–Үд»¶д»ҺдёҖдёӘејҖжәҗжңҚеҠЎдёӯжЈҖзҙўзј–з Ғеӯ—з¬ҰдёІгҖӮиҝҷдәӣеӯ—з¬ҰдёІдјҡи§ҰеҸ‘дёҖдёӘ PowerShell и„ҡжң¬пјҢиҜҘи„ҡжң¬иҙҹиҙЈдёӢиҪҪйўқеӨ–зҡ„жңүж•ҲиҪҪиҚ·пјҢе…¶дёӯеҢ…жӢ¬дёҖеј  JPG еӣҫеғҸе’ҢдёҖдёӘж–Үжң¬ж–Үд»¶гҖӮз„¶иҖҢпјҢиҝҷдәӣж–Үд»¶дёӯе·§еҰҷең°йҡҗи—ҸзқҖдҪҝз”ЁйҡҗеҶҷжҠҖжңҜеӨ„зҗҶиҝҮзҡ„жҒ¶ж„Ҹ MZ DOS еҸҜжү§иЎҢж–Үд»¶гҖӮ

дёҖж—Ұжү§иЎҢпјҢж”»еҮ»жңҖз»ҲдјҡйғЁзҪІStealerжҒ¶ж„ҸиҪҜд»¶пјҢиҜҘжҒ¶ж„ҸиҪҜд»¶дјҡж”¶йӣҶеҮӯжҚ®гҖҒжөҸи§ҲеҷЁж•°жҚ®е’Ңзі»з»ҹдҝЎжҒҜпјҢз„¶еҗҺе°ҶзӘғеҸ–зҡ„ж•°жҚ®дј иҫ“з»ҷж”»еҮ»иҖ…жҺ§еҲ¶зҡ„ Telegram жңәеҷЁдәәгҖӮиҝҷз§ҚеӨҡеұӮж··ж·Ҷд»ҘеҸҠеҜ№еҹәдәҺдә‘зҡ„еҗҲжі•еӘ’дҪ“жүҳз®ЎжңҚеҠЎзҡ„еҲ©з”ЁпјҢдҪҝеҫ—иҜҘжҒ¶ж„ҸиҪҜд»¶иғҪеӨҹйҒҝејҖдј з»ҹе®үе…ЁжңәеҲ¶зҡ„жЈҖжөӢгҖӮ

CYFIRMA жҠҘе‘Ҡз§°пјҡвҖңжӯӨж¬Ўж”»еҮ»еҲ©з”ЁдәҶеӨҡеұӮж··ж·ҶгҖҒз”ЁдәҺжүҳз®Ўжңүж•ҲиҪҪиҚ·зҡ„еҗҲжі•зҪ‘з»ңжңҚеҠЎд»ҘеҸҠеҠ еҜҶйҖҡдҝЎпјҢиҝҷз»ҷжЈҖжөӢе’Ңзј“и§Је·ҘдҪңеёҰжқҘдәҶжҢ‘жҲҳгҖӮвҖқ

жҒ¶ж„ҸиҪҜд»¶зҡ„dropperи„ҡжң¬з»ҸиҝҮдәҶй«ҳеәҰж··ж·ҶпјҢйҮҮз”ЁдәҶ Unicode еӯ—з¬ҰеӨ„зҗҶгҖҒиҝҗиЎҢж—¶д»Јз ҒжұӮеҖјпјҲeval ()пјүд»ҘеҸҠеҜ№ ActiveX зҡ„ж»Ҙз”ЁгҖӮиҝҷдҪҝеҫ—йқҷжҖҒе’ҢеҠЁжҖҒеҲҶжһҗйғҪж— жі•иҝӣиЎҢпјҢеўһеҠ дәҶжЈҖжөӢйҡҫеәҰгҖӮ

зҪ‘з»ңзҠҜзҪӘеҲҶеӯҗеҲ©з”ЁеҗҲжі•зҡ„дә‘жңҚеҠЎиҝӣиЎҢ C2 йҖҡдҝЎе’Ңжңүж•ҲиҪҪиҚ·жүҳз®ЎгҖӮдёҖдёӘејҖжәҗеӘ’дҪ“е№іеҸ°иў«з”ЁдәҺеӯҳеӮЁдёҖеј зңӢдјјж— е®ізҡ„ JPG еӣҫеғҸпјҢиҖҢе®һйҷ…дёҠиҜҘеӣҫеғҸдёӯеөҢе…ҘдәҶжҒ¶ж„Ҹзҡ„.NET зЁӢеәҸйӣҶгҖӮиҝҷдҪҝеҫ—жҒ¶ж„ҸиҪҜд»¶иғҪеӨҹж··е…ҘжӯЈеёёзҡ„зҪ‘з»ңжөҒйҮҸпјҢз»•иҝҮдј з»ҹзҡ„е®үе…Ёжү«жҸҸгҖӮ

дёәдәҶиҝӣдёҖжӯҘиәІйҒҝжЈҖжөӢпјҢж”»еҮ»иҖ…йҮҮз”ЁйҡҗеҶҷжңҜе°Ҷз»ҸиҝҮ Base64 зј–з Ғзҡ„.NET зЁӢеәҸйӣҶйҡҗи—ҸеңЁеӣҫеғҸдёӯгҖӮеҪ“иҝҷдәӣеӣҫеғҸиў«жҒ¶ж„ҸиҪҜд»¶еӨ„зҗҶж—¶пјҢйҡҗи—Ҹзҡ„д»Јз Ғдјҡиў«жҸҗеҸ–е№¶жү§иЎҢпјҢиҝҷдҪҝеҫ—дј з»ҹзҡ„еҶ…е®№жЈҖжөӢе·Ҙе…·ж— жі•е°Ҷе®ғд»¬ж Үи®°дёәжҒ¶ж„Ҹж–Үд»¶гҖӮ

дёҖж—ҰжҒ¶ж„Ҹжңүж•ҲиҪҪиҚ·иў«жҸҗеҸ–еҮәжқҘпјҢ.NET зЁӢеәҸйӣҶе°ұдјҡзӣҙжҺҘд»ҺеҶ…еӯҳдёӯжү§иЎҢпјҢиҝҷжҳҜдёҖз§Қиў«з§°дёәеҸҚе°„еҠ иҪҪзҡ„жҠҖжңҜгҖӮиҝҷз§Қж–№жі•йҒҝе…ҚеңЁзЈҒзӣҳдёҠз•ҷдёӢз—•иҝ№пјҢеҮҸе°‘дәҶеҸ–иҜҒиҜҒжҚ®пјҢд№ҹи®©еҹәдәҺж–Үд»¶зҡ„йҳІз—…жҜ’и§ЈеҶіж–№жЎҲйҡҫд»ҘжЈҖжөӢеҲ°гҖӮ

иҝҷеңәж”»еҮ»жҙ»еҠЁеҲ©з”ЁдәҶиҜёеҰӮ ActiveX еҜ№иұЎпјҲWScript.ShellгҖҒScripting.FileSystemObjectпјүе’Ң PowerShell и„ҡжң¬зӯүж—§жҠҖжңҜдёҺзі»з»ҹиҝӣиЎҢдәӨдә’гҖӮиҝҷдҪҝеҫ—ж”»еҮ»иҖ…иғҪеӨҹжү§иЎҢд»»ж„Ҹе‘Ҫд»ӨпјҢе№¶жңүеҠ©дәҺеҠЁжҖҒжҸҗеҸ–жңүж•ҲиҪҪиҚ·д»ҘеҸҠеңЁеҶ…еӯҳдёӯжү§иЎҢгҖӮ

дёҺйҖҡиҝҮиў«е…Ҙдҫөзҡ„ Web жңҚеҠЎеҷЁиҝӣиЎҢйҖҡдҝЎзҡ„дј з»ҹжҒ¶ж„ҸиҪҜд»¶дёҚеҗҢпјҢиҝҷеңәж”»еҮ»жҙ»еҠЁдҪҝз”Ёз”ұ Telegram жңәеҷЁдәәжҺ§еҲ¶зҡ„ C2 жңҚеҠЎеҷЁдј иҫ“зӘғеҸ–зҡ„ж•°жҚ®гҖӮиҝҷдёәзҪ‘з»ңзҠҜзҪӘеҲҶеӯҗжҸҗдҫӣдәҶдёҖз§ҚеҠ еҜҶдё”жҢҒд№…зҡ„ж–№жі•жқҘиҺ·еҸ–еҸ—е®іиҖ…ж•°жҚ®пјҢеҗҢж—¶з»•иҝҮдәҶи®ёеӨҡдј з»ҹзҡ„зҪ‘з»ңе®үе…ЁжҺӘж–ҪгҖӮ

CYFIRMA иӯҰе‘Ҡз§°пјҡвҖңеңЁиҝҷдёӘжЎҶжһ¶дёӯж•ҙеҗҲ.NET жңүж•ҲиҪҪиҚ·пјҢеҸҜиғҪдҪҝж”»еҮ»иҖ…иғҪеӨҹйғЁзҪІеҗ„з§ҚеҗҺз»ӯеЁҒиғҒпјҢеҢ…жӢ¬еӢ’зҙўиҪҜд»¶гҖҒдҝЎжҒҜзӘғеҸ–иҪҜд»¶е’ҢеҗҺй—ЁзЁӢеәҸгҖӮвҖқ
