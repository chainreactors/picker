# 山石安全技术研究院于CanSecWest 2023发表演讲议题

原创

信创安全实验室

山石网科安全技术研究院

‍‍山石网科安研院投递的议题《Hunting Windows Desktop Window Manager bugs for Privilege Escalation》成功入选CanSecWest 2023，我们的演讲作为压轴议题安排在最后第三天的下午，在本次议题中，山石网科安全研究将焦点集中在windows操作系统上，揭露一个全新的攻击面，并披露多个报告给微软漏洞的详细信息，两位山石网科安全研究院的研究员将登上演讲舞台。这是山石首次登上该会议，完整会议议题见：https://www.secwest.net/presentations-2023  CanSecWest 2023于3月22-24日在加拿大温哥华成功举办 。

CanSecWest创办于2000年，是北美地区最具影响力的信息安全大会，也是目前全球顶级的知名的老牌信息安全峰会之一，CanSecWest一直以其权威性、热点性、前沿性而备受行业关注。CanSecWest演讲议题挑选非常严格，只有在相关领域具有独到、深度的研究，并对安全行业未来发展有深入影响的议题才能入选。用TK教主的话说就是：CanSecWest 是纯技术人员高端的交流。不是科普性质的。不把广大群众作为受众。

![](https://mmbiz.qpic.cn/mmbiz_jpg/Gw8FuwXLJnRBp4MgSfvOJnPzKM89pqwLYKYxh1ROVDZWfqJjpT1nFfF1F6tS6Y8HMB8VIiaZZ5pIsVAibHGianq5Q/640?wx_fmt=jpeg)CanSecWest 2023 纪念衫

全球黑客顶尖赛事Pwn2Own也是由CanSecWest大会的创始人DragosRuiu组织的。多年来，CanSecWest与Pwn2Own的一赛一会广受互联网爱好者追捧，被视为极客界的殿堂级活动。此前，CanSecWest曾在日本举办过PacSec安全峰会，也曾将举办范围拓展至新加坡，还有香港的AsiaSecWest。
