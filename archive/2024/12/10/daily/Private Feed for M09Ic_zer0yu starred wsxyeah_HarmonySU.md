# HarmonySU

鸿蒙模拟器 root 工具，方便调试。

## 使用方法

1. 编译 su

```
# 指定鸿蒙 NDK 路径
export HARMONY_NDK_HOME=/path/to/your/harmony-ndk/native

# 编译
./build.sh

# 查看编译结果
ls -l build/*/su
```

2. Patch 系统镜像

```
sudo ./patch_system_img.sh /path/to/your/system.img build/arm64-v8a/su
```

Note

脚本目前仅支持 Linux 环境下运行
