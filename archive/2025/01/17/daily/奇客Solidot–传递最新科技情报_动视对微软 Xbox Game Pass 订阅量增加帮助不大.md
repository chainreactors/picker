## 动视对微软 Xbox Game Pass 订阅量增加帮助不大

[![游戏](https://icon.solidot.org/images/topics/topicgames.png?123)](/search?tid=13 "游戏")

[Wilson](/~Wilson) (42865)发表于 2025年01月16日 17时21分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=80350&appkey=1370085986&title=%E5%8A%A8%E8%A7%86%E5%AF%B9%E5%BE%AE%E8%BD%AF%20Xbox%20Game%20Pass%20%E8%AE%A2%E9%98%85%E9%87%8F%E5%A2%9E%E5%8A%A0%E5%B8%AE%E5%8A%A9%E4%B8%8D%E5%A4%A7 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自约纳丹号历险记**

微软斥资 750 亿美元收购动视暴雪，一个目的是帮助增加旗下订阅服务 Xbox Game Pass 的订阅量，但数据显示这一目标落空了。截至 6 月的一年中，Game Pass 收入增长了 5.7%，低于 11% 的内部目标。Game Pass 未能增加玩家对购买 Xbox 硬件的兴趣，促使微软将其第一方游戏作品移植到 PS5 等竞争对手的平台。微软还希望收购动视能吸引其它游戏开发商租用其云服务 Azure 的服务器，但这种局面没有发生。动视的年度大作《使命召唤》系列至今仍然使用的是亚马逊 AWS 和 Google 的 GCP 云服务，没有迁移到母公司旗下的 Azure。甚至微软在六年前收购的职业社交网络 LinkedIn，其使用的云服务至今没有迁移到 Azure。
