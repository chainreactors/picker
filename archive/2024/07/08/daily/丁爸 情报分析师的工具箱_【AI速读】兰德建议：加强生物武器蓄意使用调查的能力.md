文本是一份由兰德公司（RAND Corporation）出具的研究报告，主题是探讨如何将生物武器使用归咎于特定的个体或团体，并加强美国国防部调查故意生物事件的能力，以便支持联合国秘书长关于调查涉嫌使用化学和生物武器的机制（联合国秘书长机制）。该报告还研究了生物武器使用的重要性、提供有价值信息的证据、确定生物事件源头的科学状态以及国防部目前具备或可能开发的能力。

### 主要观点

生物武器使用归责的重要性

* 提升归责能力可作为威慑力，阻止生物武器的使用。
* 强化调查过程有助于理解对手的生物武器相关能力和作战概念。
* 改进生物防御能力的信息反馈和提升国际法规制。

决定生物事件源头的科学状态

* 确定生物事件的起因复杂，特别是在确定是否为人为事件时。
* 鉴定技术能力范围从识别事件发生到确定起源或归责。

国防部目前具备的能力与开发潜力

* 利用开放式来源文献和政策文件评估国防部的现有能力。
* 识别并优先处理增强能力的特定领域，以支持联合国秘书长机制。
* 建议增强灵活性和冗余性，保持对生物事件调查能力的透明度。

### 总结

文章核心在于强调美国国防部在加强联合国调查生物及生物毒素武器使用机制中的重要作用。为了有效应对潜在的生物武器威胁，尤其是来自ZG、伊朗、朝鲜和俄罗斯等国家及非国家行为者的威胁，美国必须增强自身的能力，特别是在生物武器使用的归责方面。通过提升调查和归责能力，不仅可以响应国际社会的期望，还能通过阻止生物武器的使用来维护国家安全和全球健康。

+ 美国国防部调查生物武器使用的能力加强计划
+ 美国国防部正在加强调查生物武器使用的能力建设，以支持联合国秘书长关于化学和生物武器调查机制的工作。
+ 生物武器（BW）使用的原因包括确定责任方，发展和维护强大的调查能力可以提供对使用生物武器的威慑，并改善对对手生物武器相关能力和教义的理解。
+ 在生物事件期间，包括生物武器使用的情况下，可能提供的证据可能会为方便归因提供有价值的信息。
+ 决定生物事件的原因和来源是困难的，特别是识别可能通过生物武器使用故意引起此类事件的行为者。
+ 国际防止和应对生物武器的机制不如化学武器那样强大，没有与禁止化学武器组织相对应的生物武器组织。

+ 美国国防部增强调查生物武器使用能力计划
+ 美国国防部正在加强其调查生物武器使用的能力建设，以更好地支持联合国秘书长机制进行调查。
+ 改进调查过程和建立强大的技术能力对于阻止生物武器的使用至关重要，这不仅可以防止生物恐怖主义的发生，也可以提高我们对敌人的了解并改进我们的防御措施。
+ 美国政府应继续投资于生物技术，包括微生物法医技术，并与其他国际伙伴保持透明合作，共同应对生物安全和生物防御问题。
+ 在发生生物事件时，收集证据并采取适当的处理措施可以有助于确定事件的责任方，并且提供更多的高质量信息可以帮助做出更准确的归因结论。
+ 随着双用途生物技术的发展，未来可能会出现新型的生物武器或传递方法，这将给检测、归因和治疗由生物武器暴露引起的伤害或疾病带来挑战，因此需要持续提升相关能力。

+ 美国国防部调查生物武器使用能力增强计划
+ 美国国防部正在加强其调查故意使用生物武器的能力，以支持联合国秘书长关于化学和生物武器使用调查机制（UNSGM）的努力。
+ 确定生物武器使用的责任方是一项具有挑战性的任务，需要收集并分析来自不同阶段的证据。
+ 提升美国在这一领域的技术能力和过程灵活性可以作为强化国际反对生物武器使用规范的一部分，并有助于提高对对手生物相关能力的理解。
+ 美国应保持透明，与国际伙伴合作进行生物安全和生物防御工作，同时警惕可能表明生物事件为人为的因素。
+ 美国防务部可以通过提升调查和归因生物事件的能力来帮助增强UNSGM的能力，包括通过投资于生物技术，特别是微生物法医技术。

+ 美国国防部加强调查生物武器使用能力的计划
+ 美国国防部正在加强其调查故意生物事件的能力，以支持联合国秘书长关于化学和生物武器调查机制（UNSGM）的工作。
+ 强化美国在这个领域的能力，特别是确定生物武器使用者的责任能力，可以帮助更好地支持UNSGM进行调查和归因，即使不是针对美国的目标。
+ 其他可能构成生物威胁的国家包括中华人民共和国、伊朗、朝鲜民主主义人民共和国和俄罗斯，非国家行为者也构成了额外的威胁。
+ 随着双用途生物技术在未来几年内的发展，可能会出现新型生物战剂或递送方法，这可能使检测生物战剂、归因生物袭击以及治疗由暴露于生物战剂引起的伤害或疾病变得复杂。
+ 国际防止和应对生物武器的机制不如化学武器那样强大，目前还没有与禁止化学武器组织类似的机构来提供实施《化学武器公约》的机构、专家中心和分析实验室的技术支持。
