иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

![CISA]()

CISA иӯҰе‘ҠзҫҺеӣҪиҒ”йӮҰжңәжһ„зЎ®дҝқе…¶зі»з»ҹе®үе…ЁпјҢйҳІжӯў Oracle WebLogic Server е’Ң Mitel MiCollab зі»з»ҹдёӯзҡ„е…ій”®жјҸжҙһиў«ж”»еҮ»еҲ©з”ЁгҖӮ

иҜҘзҪ‘з»ңе®үе…Ёжңәжһ„е°ҶеңЁ Mitel зҡ„ MiCollab з»ҹдёҖйҖҡдҝЎе№іеҸ°зҡ„ NuPoint Unified Messaging (NPM) з»„д»¶дёӯеҸ‘зҺ°зҡ„дёҖдёӘе…ій”®и·Ҝеҫ„йҒҚеҺҶжјҸжҙһпјҲCVE-2024-41713пјүж·»еҠ еҲ°е…¶е·ІзҹҘжјҸжҙһзӣ®еҪ•дёӯгҖӮ

иҝҷдёӘе®үе…ЁжјҸжҙһе…Ғи®ёж”»еҮ»иҖ…жү§иЎҢжңӘз»ҸжҺҲжқғзҡ„з®ЎзҗҶж“ҚдҪңе№¶и®ҝй—®з”ЁжҲ·е’ҢзҪ‘з»ңдҝЎжҒҜгҖӮ

вҖңжҲҗеҠҹеҲ©з”ЁжӯӨжјҸжҙһеҸҜдҪҝж”»еҮ»иҖ…иҺ·еҫ—жңӘз»ҸжҺҲжқғзҡ„и®ҝй—®жқғйҷҗпјҢд»ҺиҖҢеҜ№зі»з»ҹзҡ„дҝқеҜҶжҖ§гҖҒе®Ңж•ҙжҖ§е’ҢеҸҜз”ЁжҖ§йҖ жҲҗжҪңеңЁеҪұе“ҚгҖӮвҖқMiCollab и§ЈйҮҠиҜҙпјҡвҖңиҜҘжјҸжҙһж— йңҖйӘҢиҜҒеҚіеҸҜеҲ©з”ЁгҖӮвҖқ

вҖңеҰӮжһңиҜҘжјҸжҙһиў«жҲҗеҠҹеҲ©з”ЁпјҢж”»еҮ»иҖ…еҸҜд»ҘжңӘз»Ҹиә«д»ҪйӘҢиҜҒи®ҝй—®й…ҚзҪ®дҝЎжҒҜпјҢеҢ…жӢ¬йқһж•Ҹж„ҹз”ЁжҲ·е’ҢзҪ‘з»ңдҝЎжҒҜпјҢе№¶еңЁ MiCollab жңҚеҠЎеҷЁдёҠжү§иЎҢжңӘз»ҸжҺҲжқғзҡ„з®ЎзҗҶж“ҚдҪңгҖӮвҖқ

иў«иҝҪиёӘдёә CVE-2020-2883 зҡ„ Oracle WebLogic Server е…ій”®жјҸжҙһеңЁеӣӣе№ҙеүҚзҡ„ 2020 е№ҙ 4 жңҲе·Іиў«дҝ®иЎҘпјҢжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҸҜд»ҘиҝңзЁӢж”»еҚ жңӘжү“иЎҘдёҒзҡ„жңҚеҠЎеҷЁгҖӮ

зҫҺеӣҪзҪ‘з»ңе®үе…Ёжңәжһ„иҝҳиӯҰе‘ҠиҜҙпјҢз¬¬дәҢдёӘMitel MiCollabи·Ҝеҫ„йҒҚеҺҶжјҸжҙһпјҲCVE-2024-55550пјүеҸҜдҪҝжӢҘжңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„и®ӨиҜҒж”»еҮ»иҖ…еңЁжңүжјҸжҙһзҡ„жңҚеҠЎеҷЁдёҠиҜ»еҸ–д»»ж„Ҹж–Үд»¶гҖӮдёҚиҝҮпјҢз”ұдәҺжҲҗеҠҹеҲ©з”ЁиҜҘжјҸжҙһдёҚдјҡеҜјиҮҙжқғйҷҗеҚҮзә§пјҢиҖҢдё”еҸҜи®ҝй—®зҡ„ж–Үд»¶дёҚеҢ…еҗ«ж•Ҹж„ҹзҡ„зі»з»ҹдҝЎжҒҜпјҢеӣ жӯӨеҪұе“ҚжңүйҷҗгҖӮ

д»ҠеӨ©пјҢCISA е°ҶжүҖжңүиҝҷдёүдёӘжјҸжҙһж·»еҠ еҲ°е…¶е·ІзҹҘжјҸжҙһзӣ®еҪ•дёӯпјҢе№¶е°Ҷе…¶ж Үи®°дёәе·Іиў«з§ҜжһҒеҲ©з”ЁгҖӮж №жҚ® 2021 е№ҙ 11 жңҲеҸ‘еёғзҡ„зәҰжқҹжҖ§ж“ҚдҪңжҢҮд»Ө (BOD) 22-01 зҡ„и§„е®ҡпјҢиҒ”йӮҰж–ҮиҒҢиЎҢж”ҝйғЁй—Ё (FCEB) жңәжһ„еҝ…йЎ»еңЁ 1 жңҲ 28 ж—ҘеүҚзҡ„дёүе‘ЁеҶ…зЎ®дҝқе…¶зҪ‘з»ңе®үе…ЁгҖӮ

CISA е‘ЁдәҢиЎЁзӨәпјҡвҖңиҝҷдәӣзұ»еһӢзҡ„жјҸжҙһжҳҜжҒ¶ж„ҸзҪ‘з»ңиЎҢдёәиҖ…з»ҸеёёдҪҝз”Ёзҡ„ж”»еҮ»иҪҪдҪ“пјҢеҜ№иҒ”йӮҰдјҒдёҡжһ„жҲҗйҮҚеӨ§йЈҺйҷ©гҖӮвҖқ

иҷҪз„¶ KEV зӣ®еҪ•зҡ„йҮҚзӮ№жҳҜжҸҗйҶ’зҫҺеӣҪиҒ”йӮҰжңәжһ„еә”е°Ҫеҝ«дҝ®иЎҘзҡ„жјҸжҙһпјҢдҪҶе»әи®®жүҖжңүжңәжһ„дјҳе…Ҳзј“и§Јиҝҷдәӣе®үе…ЁжјҸжҙһпјҢд»Ҙйҳ»жӯўжҢҒз»ӯзҡ„ж”»еҮ»гҖӮ
