# 补天平台感恩季活动第二弹|事件型App专项

补天平台

补天平台

据统计，全球每年至少**新增150万种移动端恶意软件**，至少造成了**超1600 万件的移动恶意攻击事件**。近年来，伴随着移动互联网应用业务的高速发展，其产品安全漏洞风险日益突出。

    补天平台自2019年推出**“五星计划”**以来，一直联合监管、网络安全企业SRC及实验室等机构开展多方协同，共建漏洞响应生态，为提升APP安全防护能力提供有力支撑。

本次活动**补天平台将扩大收集范围，并对收到的APP漏洞进行规范化处理**，并向相关监管单位及国家漏洞库进行保送，为促进移动互联网产业安全有序发展贡献力量。

**事件型App漏洞提交方式**

补天平台官网，

https://www.butian.net/，提交选择**App品类**，漏洞标题请添加**前缀【事件型App活动】**

**收录规则评级标准及奖励**

App需为安卓系统最新版本且官方应用商店下载。漏洞分为高危、中危、低危三个等级，详情请参见【补天官网-网站公告】，公告链接：

https://www.butian.net/Article/content/id/599

**额外活动**

**活动时间**

2022年11月23日-11月30日

**活动范围及奖励**

**活动范围：**安卓系统事件型App漏洞

**活动奖励1：**有效高危漏洞均获得**2倍现金奖励。**

**活动奖励2：**累计漏洞奖励

|  |  |
| --- | --- |
| 有效高危个数 | 对应奖励 |
| 20+个 | 600元现金 |
| 10+个 | 60  kb |
| 5+个 | 25  kb |

**新人奖励**

**参与白帽：**2022年在补天平台没有提交过有效漏洞的白帽子。

**新人奖励：**活动期间提交一个有效高危事件型App漏洞即可获得**2023补天新春轻享版大礼包**。（以提交时间为准，限前10名新人白帽。）

**提交要求：**

1. 本次活动只收录服务端交互，修复在服务端进行的，请提交至补天平台，**选择App漏洞品类**，漏洞标题请添加**前缀****【事件型app活动】。**
2. 累计下载量1百万+（官方主流应用商店）。
3. 写明app下载地址或方式，漏洞报告附下载渠道分布图，可参考点点数据：https://app.diandian.com/
4. 单个app,同类型漏洞只收取 1个。
5. 单个app，收取总量不超过3个。
