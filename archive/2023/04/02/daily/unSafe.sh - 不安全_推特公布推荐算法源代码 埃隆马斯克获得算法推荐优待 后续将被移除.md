[之前埃隆马斯克就曾表示要将推特的推荐算法开源](https://www.landiannews.com/archives/97457.html)，让所有人都可以了解推特推荐算法的运作逻辑和方法，这是埃隆马斯克提升推特透明度的方式之一。

本周末推特已经在 GitHub 上公布了名为 The Algorithm 的仓库，许可证采用 GNU Affero General Public License v3.0。

仓库地址：<https://github.com/twitter/the-algorithm>

![破除信息茧房？马斯克称将在下周开源推特推荐算法](https://img.lancdn.com/landian/2023/02/97391.png)

推荐算法只要用途是推特 “为您推荐”/“For You” 部分的推荐，这部分功能主要根据用户浏览习惯或潜在兴趣，自动推荐其他推特用户的推文。

根据推特说明，推荐算法主要目的是筛选出流行推文，利用机器学习模型对推荐内容进行排名，当然这里面还有过滤器，过滤器是用来过滤掉 NSFW 推文和已屏蔽用户推文的，过滤完成后剩余的内容将会展示在用户为您推荐的时间线上。

比较搞笑的是虽然推特算法说是过滤用户已经屏蔽的其他用户的推文，但从反馈来看，不少用户在推荐里看到自己主动屏蔽掉的用户的推文，或被他人屏蔽后，这人的推文还出现在用户时间线上，相当于绕过了推特的屏蔽。

另外一个值得关注的是，检查源代码发现有多个与关键词 elon 有关的设置，这些代码实际上是用来提高埃隆马斯克账户和推文的权重，让他发布的内容出现更多用户的时间线中。

不过埃隆马斯克自称并不知道这件事，后面会在算法里移除对自己账户的有待。
