## Microsoft Edge 准备淘汰 Manifest V2 扩展开始禁用 uBlock Origin

[![Internet Explorer](https://icon.solidot.org/images/topics/topicIE.png?123)](/search?tid=31 "Internet Explorer")

[Wilson](/~Wilson) (42865)发表于 2025年02月28日 14时53分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=80676&appkey=1370085986&title=Microsoft%20Edge%20%E5%87%86%E5%A4%87%E6%B7%98%E6%B1%B0%20Manifest%20V2%20%E6%89%A9%E5%B1%95%E5%BC%80%E5%A7%8B%E7%A6%81%E7%94%A8%20uBlock%20Origin "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自泰坦的女妖**

在 Google Chrome 之后，Microsoft Edge 准备淘汰 Manifest V2 扩展开始禁用 uBlock Origin。最新的 Edge Canary 版本禁用了 Manifest V2 扩展，并显示信息称该扩展不再被支持，Microsoft Edge 建议将其移除。对于被自动禁用的扩展如 uBlock Origin，用户目前暂时还可以在扩展管理中手动启用。
