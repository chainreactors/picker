## [QakBot C2 Traffic](/?page=Blog&month=2023-03&post=QakBot-C2-Traffic)

In this video I analyze network traffic from a QakBot (QBot) infection in order to identify the Command-and-Control (C2) traffic.
The analyzed PCAP file is from [malware-traffic-analysis.net](https://malware-traffic-analysis.net/2023/02/27/index.html).
