# InnoSetup Format Package

We have released the [InnoSetup Format package](https://cerbero.io/packages/innosetupformat/) for all licenses of Cerbero Suite!

InnoSetup is a free, script-driven installation system for Windows, developed by Jordan Russell in 1997. It uses a Pascal-based scripting language, allowing for complex and customizable installers. Key features include graphical interfaces, file and registry operations, multilingual support, and the creation of encrypted and compressed executables.

Our support encompasses all released versions of InnoSetup, including the latest 6.3 version. We provide support for encrypted setups, file extraction, thorough inspection of all parts of the format, and [PascalScript disassembly](/ifps-format-package/).

![](/wp-content/uploads/2024/08/innosetup/0.png)

The inspection of the format’s inner details not only enables you to understand all the actions performed by the installer but also reveals forensic artifacts, such as timestamps.

![](/wp-content/uploads/2024/08/innosetup/1.png)

The inspection of the format covers the resources of the setup wizard itself.

![](/wp-content/uploads/2024/08/innosetup/2.png)

When a setup uses encryption, automatic decryption is attempted. For example, the malicious sample shown in the screenshot below was automatically decrypted.

![](/wp-content/uploads/2024/08/innosetup/3.png)

If you want to inspect the custom RemObjects PascalScript bytecode, make sure to install the [IFPS Format package](https://blog.cerbero.io/ifps-format-package/) as well.

![](/wp-content/uploads/2024/08/ifps/0.png)
