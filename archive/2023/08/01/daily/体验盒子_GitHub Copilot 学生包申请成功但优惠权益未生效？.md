# GitHub Copilot 学生包申请成功但优惠权益未生效？

* 发表于 2023年07月31日
* [周边](https://www.uedbox.com/web-security/safety/)
* 更新于 2023年10月26日 9:28:57 上午

**GitHub Copilot 学生包**是 GitHub 面向全球学生推出的一项优惠福利，提供给学生使用。很多学生申请了这个学生包，通过了学术验证，但是却未能成功获得优惠权益。这让他们非常困惑，不知道自己到底出了什么问题。

针对这个问题，GitHub 官方发布了最新的应用程序和再验证流程，以确保全球学生在他们的学术旅程中获得优秀的免费福利。

> 现在，当您提交申请时，您的学术认证将会被验证，并立即获得全球校园的访问权限。
> 您的教育福利将在此验证后**的72小时**内到达。
> 但是，在繁忙的时期，如新学期的开始，可能需要更长的时间。
> 您将收到一封电子邮件通知，告知您的学术福利已准备就绪。
> 如果您曾经申请了学生包，但是没有收到优惠权益，那么请耐心等待，一般权益在72小时内会生效可用。

若您发现认证成功后无法使用学生包权益，请打开[https://education.github.com/discount\_requests/student\_application](https://githubstu.com/go?_=3a369dc371aHR0cHM6Ly9lZHVjYXRpb24uZ2l0aHViLmNvbS9kaXNjb3VudF9yZXF1ZXN0cy9zdHVkZW50X2FwcGxpY2F0aW9u)

查看Approved标志是否为绿色，如果是绿色，则为认证成功但优惠未到账，一般3天左右会到账，到账后Approved图标会变成紫色。
