# MLEngineStub.exe lolbin

Posted on [2024-12-27](https://www.hexacorn.com/blog/2024/12/27/mlenginestub-exe-lolbin/ "12:07 am")  by  [adam](https://www.hexacorn.com/blog/author/adam/ "View all posts by adam")

When you run *MLEngineStub.exe* on Windows 2025, it will try to locate the following non-existing executable:

```
C:\Windows\UUS\amd64\MLEngine.exe
```

So, if we place our own executable there, it will be launched:

[![](https://www.hexacorn.com/blog/wp-content/uploads/2024/12/MLEngineStub_MLEngine.png)](https://www.hexacorn.com/blog/wp-content/uploads/2024/12/MLEngineStub_MLEngine.png)

The only caveat – only TrustedInstaller can write to this folder.
