# 哈雷疑似发生数据泄露事件

#

近日，美国摩托车制造商哈雷戴维森疑似发生一起重大数据泄露事件，该事件由名为“888”的网络犯罪组织所为。据报道，该组织声称泄露了66,700多名哈雷戴维森客户的个人详细信息。泄露的数据包括客户的全名、地址、电子邮件地址和手机号码，这些信息的暴露可能会导致身份盗用和其他网络犯罪的风险大大增加。目前哈雷戴维森尚未对这一事件发表官方声明。

# 罗德岛州遭遇勒索软件攻击

#

近期，Brain Cipher勒索软件团伙攻击了罗德岛州政府，导致其数字平台的数据被盗并泄露。目前，罗德岛州已启动全州范围的外联战略，鼓励可能受影响的居民保护自己的个人信息，并与德勤公司合作计划向受影响的人员名单发送相关通知。据悉，约有65万人使用了被攻击的国家系统，可能泄露的信息包括姓名、地址、出生日期、社会安全号码、银行信息等。

# 日本最大移动运营商遭网络攻击导致部分服务中断

#

近期，日本最大的移动运营商NTT Docomo遭受网络攻击导致部分服务中断。该公司表示，其系统受到DDoS攻击，攻击者通过发送大量垃圾流量淹没网络，使包括新闻网站、视频流媒体平台、移动支付、网络邮件服务和高尔夫爱好者网站在内的服务暂时不可用。目前，该公司表示大多数服务已恢复，但某些内容更新可能仍有延迟。NTT Docomo尚未将此次攻击归因于任何特定威胁行为者。

# 苹果9500万美元和解Siri“偷听”集体诉讼

#

#

据报道，苹果公司同意支付9500万美元解决一项集体诉讼，该诉讼指控苹果的Siri语音助手在未经用户同意的情况下记录对话并与第三方共享，侵犯了用户隐私。和解协议涉及2014年9月17日至2024年12月31日期间所有使用Siri的美国用户。根据协议，每位用户最多可申请5台设备的赔偿，每台设备最高可获赔20美元，同时苹果需在六个月内删除2019年10月前收集的所有Siri个人音频记录。

# GitHub出现310万虚假“星标”误导用户

#

近期，GitHub上出现了大量虚假“星标”（Star），星标通常被用来标记用户感兴趣的项目，类似于社交媒体的“点赞”。然而，现在它们被用来人为提升诈骗性和恶意软件项目的关注度，误导用户下载。研究人员使用StarScout工具对GHArchive数据进行了分析，分析结果显示，GitHub上有310万个星标确认为虚假星标，涉及278000个账号和15835个库。因此，建议大家在选择项目时应更仔细的阅读文档、检查项目内容、查看贡献记录，以免受到虚假星标的影响。

相关链接：

https://www.bleepingcomputer.com/news/security/over-31-million-fake-stars-on-github-projects-used-to-boost-rankings/
