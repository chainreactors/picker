*Moby Project з»ҙжҠӨиҖ…еҸ‘еёғзҡ„е…¬е‘ҠдёӯеҶҷйҒ“пјҡвҖңж”»еҮ»иҖ…еҸҜд»ҘдҪҝз”Ё Content-Length и®ҫзҪ®дёә 0 зҡ„ API иҜ·жұӮжқҘеҲ©з”Ёз»•иҝҮжјҸжҙһпјҢеҜјиҮҙ Docker е®ҲжҠӨзЁӢеәҸе°ҶжІЎжңүжӯЈж–Үзҡ„иҜ·жұӮиҪ¬еҸ‘еҲ° AuthZ жҸ’д»¶пјҢиҜҘжҸ’д»¶еҸҜиғҪдјҡй”ҷиҜҜең°жү№еҮҶиҜҘиҜ·жұӮгҖӮвҖңдҪҝз”Ёзү№еҲ¶зҡ„ API иҜ·жұӮпјҢEngine API е®ўжҲ·з«ҜеҸҜд»ҘдҪҝе®ҲжҠӨиҝӣзЁӢе°ҶиҜ·жұӮжҲ–е“Қеә”иҪ¬еҸ‘еҲ°жҺҲжқғжҸ’д»¶пјҢиҖҢж— йңҖжӯЈж–ҮгҖӮеңЁжҹҗдәӣжғ…еҶөдёӢпјҢжҺҲжқғжҸ’д»¶еҸҜиғҪдјҡе…Ғи®ёдёҖдёӘиҜ·жұӮпјҢеҰӮжһңжӯЈж–Үиў«иҪ¬еҸ‘з»ҷе®ғпјҢе®ғжң¬жқҘдјҡиў«жӢ’з»қгҖӮ*

2018 е№ҙеҸ‘зҺ°зҡ„дёҖдёӘзјәйҷ·е…Ғи®ёж”»еҮ»иҖ…дҪҝз”Ёжһ„е»әзҡ„ API иҜ·жұӮз»•иҝҮ Docker еј•ж“Һдёӯзҡ„жҺҲжқғ пјҲAuthZпјү жҸ’д»¶пјҢд»ҺиҖҢеҸҜиғҪеҜјиҮҙжңӘз»ҸжҺҲжқғзҡ„ж“ҚдҪңпјҢеҢ…жӢ¬жқғйҷҗжҸҗеҚҮгҖӮиҜҘжјҸжҙһе·ІеңЁ Docker Engine v18.09.1 зүҲжң¬дёӯеҫ—еҲ°и§ЈеҶіпјҢдҪҶжңӘеҢ…еҗ«еңЁеҗҺз»ӯзҡ„дё»иҰҒзүҲжң¬дёӯпјҢд»ҺиҖҢеҜјиҮҙеӣһеҪ’гҖӮжӯӨй—®йўҳдёҚдјҡеҪұе“Қ Docker EE v19.03.x жҲ–д»»дҪ•зүҲжң¬зҡ„ Mirantis е®№еҷЁиҝҗиЎҢж—¶гҖӮ

иҜҘжјҸжҙһдәҺ 2024 е№ҙ 4 жңҲиў«еҸ‘зҺ°пјҢе·ІеңЁ 2024 е№ҙ 7 жңҲ 23 ж—ҘеҸ‘еёғзҡ„зүҲжң¬ 23.0.14 е’Ң 27.1.0 дёӯеҫ—еҲ°и§ЈеҶігҖӮд»ҘдёӢжҳҜдҪҝз”Ё AuthZ жҸ’д»¶ж—¶еҸ—еҪұе“Қзҡ„ Docker Engine зүҲжң¬еҲ—иЎЁпјҡ

*вҖңдёҚдҫқиө–AuthZжҸ’д»¶зҡ„Dockerе•Ҷдёҡдә§е“Ғе’ҢеҶ…йғЁеҹәзЎҖи®ҫж–Ҫзҡ„з”ЁжҲ·дёҚеҸ—еҪұе“ҚпјҢвҖқиҜҘе…¬е‘Ҡз»§з»ӯиҜҙйҒ“гҖӮ*

Docker Desktop 4.32.0 еҸҠд№ӢеүҚзүҲжң¬еҢ…еҗ«жҳ“еҸ—ж”»еҮ»зҡ„ Docker Engine зүҲжң¬гҖӮдҪҶжҳҜпјҢз”ұдәҺеҲ©з”ЁиҜҘжјҸжҙһйңҖиҰҒи®ҝй—® Docker APIпјҢиҖҢ Docker API йҖҡеёёйңҖиҰҒеҜ№дё»жңәиҝӣиЎҢжң¬ең°и®ҝй—®пјҢеӣ жӯӨйҷҚдҪҺдәҶйЈҺйҷ©гҖӮй»ҳи®Өжғ…еҶөдёӢпјҢDocker Desktop дёҚеҢ…еҗ« AuthZ жҸ’д»¶пјҢд»ҺиҖҢйҷҗеҲ¶дәҶеҜ№ Docker Desktop VM иҖҢдёҚжҳҜеә•еұӮдё»жңәзҡ„жқғйҷҗжҸҗеҚҮгҖӮйў„и®Ў Docker Desktop v4.33 дёӯе°ҶжҸҗдҫӣ Docker Engine зҡ„еӣәе®ҡзүҲжң¬пјҢд»Ҙи§ЈеҶіиҝҷдәӣе®үе…Ёй—®йўҳгҖӮ
