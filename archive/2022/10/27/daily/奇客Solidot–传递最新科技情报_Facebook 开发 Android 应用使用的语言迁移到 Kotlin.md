## Facebook 开发 Android 应用使用的语言迁移到 Kotlin

[![程序](https://icon.solidot.org/images/topics/topicprogramming.png?123)](/search?tid=5 "程序")
[![Facebook](https://icon.solidot.org/images/topics/topicfacebook.png?123)](/search?tid=161 "Facebook")

[WinterIsComing](/~WinterIsComing) (31822)发表于 2022年10月26日 23时26分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73179&appkey=1370085986&title=Facebook%20%E5%BC%80%E5%8F%91%20Android%20%E5%BA%94%E7%94%A8%E4%BD%BF%E7%94%A8%E7%9A%84%E8%AF%AD%E8%A8%80%E8%BF%81%E7%A7%BB%E5%88%B0%20Kotlin%20 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自微光城市**

Meta/Facebook [透露](https://engineering.fb.com/2022/10/24/android/android-java-kotlin-migration/)它开发 Android 应用使用的语言从 Java 迁移到了 Kotlin，而 Kotlin 的代码行数突破了一千万。Kotlin 相对于 Java 的优势包括：内置为空性（nullability）处理，支持内联函数和 lambda 表达式，代码更短，能定义 DSL（Domain-specific language）。缺点是需要在很长时间内面对混合代码库，而 Kotlin 的流行度仍然远逊于 Java——Java 是世界第二大或第三大流行的语言。
