Deepseek R1 越狱版！无审查、无内容限制，无思想钢印，无道德制高点

DeepSeek 开源了多模态大模型，普通电脑可以直接安装使用，现在我们就来本地部署！支持图像识别和图像生成，性能非常强悍！

![](https://i0.wp.com/cdn.skyimg.de/up/2025/2/7/w2fekf.webp)

本地部署 deepseek 无限制版本！ 也就是DeepSeek R1的越狱版！deepseek-r1-abliterated ：无内容审查，无限制，无思想钢印，无道德….？

**越狱版的DeepSeek R1现已托管在Hugging face上，支持8B、14B、32B 【[链接直达](https://blog.upx8.com/go/aHR0cHM6Ly9odWdnaW5nZmFjZS5jby9odWlodWktYWk)】**

可能有人会觉得越狱版没有官方版聪明，这是一个比较大的误解，官方版可是671B的，普通人是跑不起来的，大多数人能本地运行的都是7b、8b、14b、32b这几大类型的模型，是被蒸馏压缩过的，无法直接和671B相比，但是自己本地使用已足够。

![](https://i0.wp.com/cdn.skyimg.de/up/2025/2/7/q7uquf.webp)

**当然如果你有高级的硬件配置，你可以直接上70B的, 在Ollama 【[链接直达](https://blog.upx8.com/go/aHR0cHM6Ly9vbGxhbWEuY29tL2h1aWh1aV9haS9kZWVwc2Vlay1yMS1hYmxpdGVyYXRlZDo3MGI)】**

**![](https://i0.wp.com/cdn.skyimg.de/up/2025/2/7/mjyjy8.webp)**

**本地部署越狱版步骤**

1、下载并安装Ollama 【**[点击下载](https://blog.upx8.com/go/aHR0cHM6Ly9vbGxhbWEuY29tLw)**】

2、在CMD终端下通过命令下载DeepSeek R1越狱版

以下五个模型：7b、14b、32b、70b，自己选一个

```
ollama run huihui_ai/deepseek-r1-abliterated:7b
ollama run huihui_ai/deepseek-r1-abliterated:8b
ollama run huihui_ai/deepseek-r1-abliterated:14b
ollama run huihui_ai/deepseek-r1-abliterated:32b
ollama run huihui_ai/deepseek-r1-abliterated:70b
```

3、安装 Web UI

就可以本地调用 deepseek r1 越狱版 AI 大模型 【[**点击下载**](https://blog.upx8.com/go/aHR0cHM6Ly9jaHJvbWV3ZWJzdG9yZS5nb29nbGUuY29tL2RldGFpbC9wYWdlLWFzc2lzdC0lRTYlOUMlQUMlRTUlOUMlQjAtYWktJUU2JUE4JUExJUU1JTlFJThCJUU3JTlBJTg0LXdlYi9qZmdmaWlncGtobGtibmZuYm9iYmtpbmVoaGZkaG5kbw)】
