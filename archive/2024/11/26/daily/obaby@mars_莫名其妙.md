# 莫名其妙

2024年11月25日
[71 条评论](https://h4ck.org.cn/2024/11/18629#comments)

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG735_e.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG735_e.jpg)

不得不说最近这个天是越来越冷了，本来都不准备写东西了。就写一篇技术文拉到，象征性的摸摸鱼，然鹅，事情的发展总是能出乎意料。

平淡的周末依然平淡，周五去给宝子开家长会，老师好一通表扬，让我也不由得心花怒放。说宝子字写得好看，有时候批改作业或者心情不好的时候就去会看宝子的作业，瞬间就让人心情愉悦了。不过说实话，宝子的字写的确实是好看，哪怕跟现在自己写的字相比都毫不逊色。这也不是第一次老师当着那么多人的面表扬宝子了。在家长会之前，还专门设置了个区域，让大家去看那些优秀的作业。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG749.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG749.jpg)

晚上，看了下第二天的小记者活动，原本已经报名满额的关爱罕见病的活动，竟然又空出来一个，正好就可以重新报名参加这个活动。留住海鸥的活动就不再参与了，毕竟，那个距离也有些远。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7451732516797_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7451732516797_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7461732516799_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7461732516799_.pic_.jpg)

排队领材料的时候，宝子依然一副漫不经心的样子。就那么站着，也不着急，有时候还东张西望，前面有人插队也不会去阻止。这一点真的不知道是遗传的谁，那种与世无争的状态，有时候真的让人看了来气。完全不知道维护自己的权利和利益，也从来不知道竞争是什么，这个好心态有时候也让自己无比的佩服。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7431732516796_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7431732516796_.pic_.jpg)

拿到材料，看的还蛮认真的。后面进到屋里做手工，家长不让陪同，也就只能在外面候着了。虽然已经是冬天，但是在风和日丽的天气里，晒着太阳，懒洋洋的暖暖的倒是也颇为自在。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7501732518120_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7501732518120_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7511732518123_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7511732518123_.pic_.jpg)

做完手工，后续又来了一位医生，给小朋友讲解罕见病的相关知识，还有采访环节，当然，不出意外，宝子还是在最外围。嗯，就是这么波澜不惊。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7471732516799_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7471732516799_.pic_.jpg)

不过这短短的一个小时，竟然又认识了一个姐姐，跟那个小姐姐玩的还挺开心的。活动结束，两个人还又去游乐场玩了一个多小时。

宝子看新闻的时候，总说想去滑雪了，从高德地图看了下滑雪场，都还没开放。不过风和日丽的暖天应该也不会太多了，再有一个多月应该就可以去滑雪了。下午带宝子去公园骑自行车，有朋友推荐国学公园，长途跋涉之后直奔国学公园。

然而，公园外并没有共享单车神马的，只能宝子一个人骑自己在后面跟着，不过这个公园面积的确不大，不一会儿就转了一圈了。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7421732516795_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7421732516795_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7411732516794_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7411732516794_.pic_.jpg)

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG752.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG752.jpg)

不过这个小圆子好处是人不多，拍婚纱照的蛮多的。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG748.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG748.jpg)

意犹未尽，只能去另外一个公园，奥林匹克公园。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7531732518654_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7531732518654_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7541732518654_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7541732518654_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7551732518656_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7551732518656_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7561732518657_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7561732518657_.pic_.jpg)

还有在草丛里吃草的猫咪：

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7571732518658_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7571732518658_.pic_.jpg)

在湖里的游船，后面跟着个大喇叭一边走一边喊：水深四米，请勿跳水……

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7581732518659_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7581732518659_.pic_.jpg)

不过真的有人会跳水吗？😂

下午回家，宝子的姥姥开始数落，：”看你们带孩子出去，把孩子给累的“

问题是她不吃饭 ，纯粹就是不想吃，跟我们没关系。

晚饭象征性的对付了过去，我也没怎么吃东西就喝了一碗豆浆。早上起床发现手机断网了，打开wifi 界面，其实所有的无线都无法上网，这尼玛就神了。

看了下网站，果然挂了，而服务器状态监控，家里的主机也都离线了。好不意外，肯定是家里的网络出问题了。重启路由器，无效，reset 依然无效，开始怀疑是不是宽带出问题了。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7381732516791_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7381732516791_.pic_.jpg)

光猫的光信号指示灯是灭的，然而，打开电视，iptv 却是正常的。这尼玛就离谱了。

打开电脑，获取 ip 地址依然是 168.254 网段的什么东西，这么看来是彻底完犊子了。看了下二手东的购买日期，去年的双十一，刚好过保，哈哈哈。这尼玛，到公司不死心继续折腾，发现竟然可以延保，通过延保服务提交售后，最后总算是解决这个问题了。不过需要把路由器寄回去。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG759-scaled-e1732519456162.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/WechatIMG759.jpg)

中午回去翻箱倒柜，把之前的路由器拿出来，先换上勉强用着，旧路由器拆下来，准备寄回。

[![](https://h4ck.org.cn/wp-content/uploads/2024/11/7371732516790_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7371732516790_.pic_.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2024/11/7361732516789_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2024/11/7361732516789_.pic_.jpg)

看着这一堆线真的头大，完全不想折腾啊。

顺丰下单，两点寄件，现在三点半了还没来取，哎？人呢。
