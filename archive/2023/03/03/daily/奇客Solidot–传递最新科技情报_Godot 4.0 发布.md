## Godot 4.0 发布

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")
[![游戏](https://icon.solidot.org/images/topics/topicgames.png?123)](/search?tid=13 "游戏")

[Wilson](/~Wilson) (42865)发表于 2023年03月02日 14时47分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74275&appkey=1370085986&title=Godot%204.0%20%E5%8F%91%E5%B8%83 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自小无知气球旅行**

开源游戏引擎项目 Godot 释出了 4.0 版本。该版本经历了四年时间开发，合并了 1.2 万个 pull requests，修正了 7000 个问题，有 1500 名贡献者。Godot 4.0 的新特性包括：新的 Vulkan 后端；支持 AMD 的 FSR 1.0，未来将支持 FSR 2.1；为大型开放世界引入了新实时全局照明技术 Signed Distance Field Global Illuminatio；新的渲染优化技术，等等。
https://godotengine.org/article/godot-4-0-sets-sail/
