## 微软测试资源管理器文件推荐功能

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Wilson](/~Wilson) (42865)发表于 2023年03月09日 15时49分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74343&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E6%B5%8B%E8%AF%95%E8%B5%84%E6%BA%90%E7%AE%A1%E7%90%86%E5%99%A8%E6%96%87%E4%BB%B6%E6%8E%A8%E8%8D%90%E5%8A%9F%E8%83%BD "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自黑暗之潮**

微软开始测试名为“文件资源管理器”的“文件推荐”功能，如名所示，当用户访问文件资源管理器主标签，它会在顶部展示你可能想要打开的特定文件。该功能目前只提供给 Windows 11 Insider Preview Build 23403 的部分测试者，且需要登陆 Azure Active Directory 账户。这意味着目前该功能针对的是企业用户。微软表示将会先收集反馈，然后决定如何推送给所有用户。软件巨人也认识到不是所有人都喜欢资源管理器里面出现意想不到的结果。
https://blogs.windows.com/windows-insider/2023/03/08/announcing-windows-11-insider-preview-build-23403/
