**Sneaky 2FAзҪ‘з»ңй’“йұјеҚіжңҚеҠЎе·Ҙе…·еҢ…жҳҜеҰӮдҪ•е·ҘдҪңзҡ„пјҹ**

Sneaky Log йҖҡиҝҮ Telegram дёҠзҡ„иҒҠеӨ©жңәеҷЁдәәй”Җе”®зҪ‘з»ңй’“йұје·Ҙе…·еҢ…гҖӮдёҖж—Ұе®ўжҲ·д»ҳж¬ҫпјҢSneaky Log е°ұдјҡжҸҗдҫӣ Sneaky 2FA жәҗд»Јз Ғзҡ„и®ҝй—®жқғйҷҗгҖӮSneaky Log дҪҝз”Ёиў«е…Ҙдҫөзҡ„ WordPress зҪ‘з«ҷе’Ңе…¶д»–еҹҹжқҘжүҳз®Ўи§ҰеҸ‘зҪ‘з»ңй’“йұје·Ҙе…·еҢ…зҡ„йЎөйқўгҖӮ

йӘ—еұҖеҢ…жӢ¬еҗ‘жҪңеңЁеҸ—е®іиҖ…еұ•зӨәдёҖдёӘиҷҡеҒҮзҡ„еҫ®иҪҜи®ӨиҜҒйЎөйқўгҖӮз„¶еҗҺпјҢSneaky 2FAдјҡжҳҫзӨәдёҖдёӘеёҰжңү вҖңйӘҢиҜҒжӮЁжҳҜдәәзұ» вҖқжҸҗзӨәжЎҶзҡ„Cloudflare TurnstileйЎөйқўгҖӮ

еҰӮжһңеҸ—е®іиҖ…жҸҗдҫӣдәҶиҙҰжҲ·дҝЎжҒҜпјҢд»–д»¬зҡ„з”өеӯҗйӮ®д»¶е’ҢеҜҶз Ғе°ұдјҡиў«еҸ‘йҖҒеҲ°й’“йұјжңҚеҠЎеҷЁгҖӮSneaky Log жңҚеҠЎеҷЁдјҡжЈҖжөӢ Microsoft 365 иҙҰжҲ·зҡ„еҸҜз”Ё 2FA ж–№жі•пјҢе№¶жҸҗзӨәз”ЁжҲ·жҢүз…§иҝҷдәӣж–№жі•ж“ҚдҪңгҖӮ

з”ЁжҲ·е°Ҷиў«йҮҚе®ҡеҗ‘еҲ°дёҖдёӘзңҹе®һзҡ„ Office365 URLпјҢдҪҶзҪ‘з»ңй’“йұјжңҚеҠЎеҷЁзҺ°еңЁеҸҜд»ҘйҖҡиҝҮ Microsoft 365 API и®ҝй—®з”ЁжҲ·зҡ„иҙҰжҲ·гҖӮ

еҰӮжһңй’“йұјзҪ‘з«ҷзҡ„и®ҝй—®иҖ…жҳҜжңәеҷЁдәәгҖҒдә‘жҸҗдҫӣе•ҶгҖҒд»ЈзҗҶгҖҒVPNгҖҒжқҘиҮӘж•°жҚ®дёӯеҝғжҲ–дҪҝз”Ё вҖңдёҺе·ІзҹҘж»Ҙз”Ёзӣёе…і вҖқзҡ„ IP ең°еқҖпјҢеҲҷйЎөйқўдјҡйҮҚе®ҡеҗ‘еҲ°дёҺеҫ®иҪҜзӣёе…ізҡ„з»ҙеҹәзҷҫз§‘жқЎзӣ®гҖӮе®үе…Ёз ”з©¶еӣўйҳҹ TRAC Labs дәҺ 2024 е№ҙ 12 жңҲеңЁдёҖдёӘеҗҚдёә WikiKit зҡ„зҪ‘з»ңй’“йұји®ЎеҲ’дёӯеҸ‘зҺ°дәҶзұ»дјјзҡ„жҠҖжңҜгҖӮ

Sekoia жҢҮеҮәпјҢSneaky Log зҡ„е·Ҙе…·еҢ…дёҺйЈҺйҷ©е№іеҸ°е…¬еҸё Group-1B дәҺ 2023 е№ҙ 9 жңҲеҸ‘зҺ°зҡ„еҸҰдёҖдёӘзҪ‘з»ңй’“йұје·Ҙе…·еҢ…е…ұдә«дёҖдәӣжәҗд»Јз ҒгҖӮиҜҘе·Ҙе…·еҢ…дёҺдёҖдёӘеҗҚдёә W3LL зҡ„еЁҒиғҒиЎҢдёәиҖ…жңүе…ігҖӮ

Sneaky Log д»ҘжҜҸжңҲ 200 зҫҺе…ғзҡ„д»·ж јеҮәе”® Sneaky 2FAпјҢд»ҘеҠ еҜҶиҙ§еёҒж”Ҝд»ҳгҖӮSekoia иЎЁзӨәпјҢиҝҷжҜ” Sneaky Log зҡ„зҠҜзҪӘз«һдәүеҜ№жүӢжҸҗдҫӣзҡ„е·Ҙе…·еҢ…иҰҒдҫҝе®ңдёҖдәӣгҖӮ

**еҰӮдҪ•жЈҖжөӢе’ҢеҮҸе°‘й¬јй¬јзҘҹзҘҹзҡ„ 2FA**

Sekoia иҜҙпјҢдёҺ Sneaky 2FA зӣёе…ізҡ„жҙ»еҠЁеҸҜд»ҘеңЁз”ЁжҲ·зҡ„ Microsoft 365 е®Ўи®Ўж—Ҙеҝ—дёӯжЈҖжөӢеҲ°гҖӮ

зү№еҲ«жҳҜпјҢе®үе…Ёз ”з©¶дәәе‘ҳеңЁи°ғжҹҘзҪ‘з»ңй’“йұјдјҒеӣҫж—¶пјҢеҸҜиғҪдјҡеҸ‘зҺ°еңЁиә«д»ҪйӘҢиҜҒжөҒзЁӢзҡ„жҜҸдёӘжӯҘйӘӨдёӯпјҢHTTPиҜ·жұӮзҡ„User-Agentеӯ—з¬ҰдёІйғҪжҳҜдёҚеҗҢзҡ„зЎ¬зј–з ҒгҖӮеҰӮжһңз”ЁжҲ·иә«д»ҪйӘҢиҜҒжӯҘйӘӨжҳҜиүҜжҖ§зҡ„пјҢе°ұдёҚеӨӘеҸҜиғҪеҮәзҺ°иҝҷз§Қжғ…еҶөгҖӮ

Sekoia еҸ‘еёғдәҶдёҖжқЎ Sigma жЈҖжөӢи§„еҲҷпјҢиҜҘи§„еҲҷ вҖңжҹҘжүҫ iOS е№іеҸ° Safari з”ЁжҲ·д»ЈзҗҶзҡ„ Login:login дәӢд»¶е’Ң Windows е№іеҸ° Edge з”ЁжҲ·д»ЈзҗҶзҡ„ Login:resume дәӢд»¶пјҢиҝҷдёӨдёӘдәӢд»¶е…·жңүзӣёеҗҢзҡ„зӣёе…і IDпјҢе№¶дё”еңЁ 10 еҲҶй’ҹеҶ…еҸ‘з”ҹгҖӮвҖқ

е®үе…Ёдё“дёҡдәәе‘ҳеҸҜд»ҘжҸҗйҶ’е‘ҳе·ҘйҒҝе…ҚдёҺеҸҜз–‘з”өеӯҗйӮ®д»¶дә’еҠЁпјҢеҢ…жӢ¬йӮЈдәӣеҗ¬иө·жқҘеҫҲзҙ§жҖҘжҲ–еҸҜжҖ•зҡ„з”өеӯҗйӮ®д»¶гҖӮSekoia еңЁдёҖе°ҒеҗҚдёә вҖңFinal Lien Waiver.pdf вҖқзҡ„жҒ¶ж„Ҹз”өеӯҗйӮ®д»¶йҷ„д»¶дёӯеҸ‘зҺ°дәҶ Sneaky 2FAпјҢиҜҘйҷ„д»¶еҢ…еҗ«дёҖдёӘдәҢз»ҙз ҒгҖӮдәҢз»ҙз ҒдёӯеөҢе…Ҙзҡ„URLжҢҮеҗ‘дёҖдёӘиў«е…Ҙдҫөзҡ„йЎөйқўгҖӮ

**иҝ‘жңҹй’ҲеҜ№еҫ®иҪҜзҡ„е…¶д»–зҪ‘з»ңй’“йұјдјҒеӣҫ**

еҫ®иҪҜзҡ„ж— еӨ„дёҚеңЁдҪҝе…¶жҲҗдёәеЁҒиғҒиЎҢдёәиҖ…зҡ„еӨ§жң¬иҗҘпјҢж— и®әд»–д»¬жҳҜзӣҙжҺҘе®һж–Ҫж”»еҮ»иҝҳжҳҜеҮәе”®зҪ‘з»ңй’“йұјеҚіжңҚеҠЎе·Ҙе…·гҖӮ

2023 е№ҙпјҢеҫ®иҪҜзҡ„еЁҒиғҒжғ…жҠҘеӣўйҳҹжҠ«йңІдәҶдёҖдёӘй’ҲеҜ№ Office жҲ– Outlook зӯүжңҚеҠЎзҡ„зҪ‘з»ңй’“йұје·Ҙе…·еҢ…гҖӮеҗҢе№ҙжҷҡдәӣж—¶еҖҷпјҢProofpoint жҸӯејҖдәҶ ExilProxy зҡ„йқўзәұпјҢиҝҷжҳҜдёҖж¬ҫеҸҜд»Ҙз»•иҝҮеҸҢеӣ зҙ иә«д»ҪйӘҢиҜҒзҡ„зҪ‘з»ңй’“йұје·Ҙе…·еҢ…гҖӮ
