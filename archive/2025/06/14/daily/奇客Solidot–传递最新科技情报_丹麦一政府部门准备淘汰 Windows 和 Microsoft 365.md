## 丹麦一政府部门准备淘汰 Windows 和 Microsoft 365

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2025年06月13日 18时03分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81552&appkey=1370085986&title=%E4%B8%B9%E9%BA%A6%E4%B8%80%E6%94%BF%E5%BA%9C%E9%83%A8%E9%97%A8%E5%87%86%E5%A4%87%E6%B7%98%E6%B1%B0%20Windows%20%E5%92%8C%20Microsoft%20365 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自暗黑童话**

丹麦数字事务部准备从下个月开始，淘汰 Windows 和 Microsoft 365，切换到 Linux 和 LibreOffice。该计划将逐步推进，首先半数员工改用 Linux 和 LibreOffice，如果一切顺利，所有员工将从秋季开始使用开源解决方案。部长 Caroline Stage 表示，此举旨在节省成本以及减少对美国软件的依赖，称数字主权是该部门的优先事项。此举也有助于应对 Windows 10 操作系统即将于今年 10 月终止支持所带来的安全问题。Stage 表示如果进展不顺利的话，还是会继续使用微软软件。
