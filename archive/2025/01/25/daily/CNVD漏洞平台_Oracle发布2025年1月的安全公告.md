# Oracle发布2025年1月的安全公告

原创

CNVD

CNVD漏洞平台

安全公告编号:CNTA-2025-0001

2025年1月21日，Oracle发布了2025年1月份的安全更新，修复了其多款产品存在的318个安全漏洞。受影响的产品包括：Oracle Database Server数据库（5个）、Oracle Application Express（1个）、Oracle GoldenGate（2个）、Oracle REST Data Services（1个）、Oracle Secure Backup（1个）、Oracle Commerce（1个）、Oracle Communications Applications（28个）、Oracle
Communications（85个）、Oracle Construction and Engineering（4个）、电子商务套装软件Oracle
E-Business Suite（4个）、Oracle Enterprise Manager（3个）、Oracle Financial Services Applications（31个）、中间件产品Fusion
Middleware（22个）、Oracle Analytics（26个）、Oracle Health Sciences Applications（2个）、Oracle
Hospitality Applications（1个）、Oracle Hyperion（2个）、Oracle Java SE（2个）、Oracle JD Edwards（23个）、Oracle MySQL数据库（39个）、Oracle PeopleSoft（16个）、Oracle Retail Applications（2个）、Oracle Siebel CRM（2个）、Oracle Siebel CRM（6个）、Oracle Systems（1个）、Oracle Utilities Applications（6个）和Oracle
Virtualization（2个）。

本次安全更新涉及的漏洞中，共包括133个高危漏洞，272个可被远程利用漏洞。CNVD提醒广大Oracle用户，请及时下载补丁更新，避免引发漏洞相关的安全事件。
