# IPv4 用完，如今花钱才能重新使用获取？

埃文科技
[行业](https://www.4hou.com/category/industry)
2022-11-14 10:16:35

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)126383

收藏

导语：“在2019年11月25日，我们终于、终于、终于用完了IPv4地址。”

“在2019年11月25日，我们终于、终于、终于用完了IPv4地址。”

负责欧洲、中东与中亚网络位址分发的互联网注册机构RIPE的说法，该机构周一宣布，"我们从可用地址池中剩余的最后一个地址中进行了最后的/22 IPv4分配。我们现在已经用完了IPv4地址"。

![1.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20221114/1668389097177786.png "1668389097177786.png")

不过IPv4的枯竭早已不是新鲜事了，除了那些已分配使用的IPv4，如果想要重新使用IPv4 "将继续恢复......从已经停业或关闭的组织，或从退回他们不再需要的地址的网络"。然而，这样的情况需要等待。所以，这也意味着全球 IPv4 位址正式用完，未来只能回收旧的IPV4重用。

但是如今的互联网，我们依旧在使用IPV4。工程师们会继续想办法让IPv4为他们服务，而且由于全球可用 IP 地址的匮乏导致IP地址已经成为了一种商品，IPv4销售的灰色市场越来越大。

每个IP地址在公开市场的售价都在15到25美元之间。从事IP地址销售的人都从中获益良多，同样的，那些专门从事网络犯罪与垃圾邮件的也想要未经所有者允许利用那些“休眠”的 IP 地址。

住在加利福尼亚的独立安全研究员 Ron Guilmette 自从 2016 年以来，一直在追踪留给非洲的 IP 地址块。这些 IP 地址块以某种方式流转到了基于互联网的营销公司的手中。

通过与在南非的记者进行合作，Guilmette 发现了成千上万个 IP 地址被盗。这些 IP 地址被 AFRINIC 的政策协调人（Policy Coordinator）创立的几家公司售出的。（注：AFRINIC 是世界上五个区域互联网注册管理机构之一，负责处理非洲和印度洋地区的IP地址分配事务）

那么为什么RIPE不断宣布我们已经用完了IPv4地址？

有两个原因：其一，他们在发放IPv4方面的传统角色实际上已经结束，他们并不完全清楚从这一点上他们该做什么。第二，因为他们相信，指出IPv4稀缺的里程碑会使人们看到曙光，转而使用IPv6。

![2.png](https://img.4hou.com/uploads/ueditor/php/upload/image/20221114/1668389112177351.png "1668389112177351.png")

与我国加速推广IPv6不同 ,对于IPV6在国外的形式正如RIPE在他们的公告中所指出的那样，缺乏向IPv6发展的行动开始削弱了长期以来互联网扩张没有真正限制的观念。"如果没有大规模的IPv6部署，我们就有可能进入这样一个未来：我们的互联网增长受到不必要的限制，不是因为缺乏熟练的网络工程师、技术设备或投资，而是因为缺乏独特的网络标识符。"
