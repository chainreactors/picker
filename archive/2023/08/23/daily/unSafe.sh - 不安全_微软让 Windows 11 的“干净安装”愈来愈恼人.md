以前 Windows 用户可通过“干净安装（Clean Install）或全新安装”移除 PC 厂商会捆绑到系统中的第三方软件。今天 Windows 的“干净安装”仍然是可能的，甚至比以前更方便了，因为 Windows Update 能提供计算机所需的所有驱动程序。但问题是“干净安装”并不干净，让 Windows 变得不干净的不是第三方而是微软本身。微软让 Windows 11 的“干净安装”愈来愈恼人，它会反复向用户推销它的服务如 Microsoft 365 试用版、OneDrive 云储存和 PC Game Pass 游戏订阅，第一次你说不需要，微软还会用第二次来烦你，并美名为“SCOOBE（second chance out-of-box experience）”。
