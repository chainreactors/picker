[Wilson](/~Wilson) (42865)发表于 2024年05月12日 00时05分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=78143&appkey=1370085986&title=%E6%B3%95%E5%AE%98%E9%A9%B3%E5%9B%9E%E4%BA%86%20X%20%E5%AF%B9%E6%95%B0%E6%8D%AE%E6%8A%93%E5%8F%96%E5%85%AC%E5%8F%B8%E7%9A%84%E8%AF%89%E8%AE%BC "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自火星之剑**

美国地区法官 William Alsup 驳回了马斯克(Elon Musk)的 X Corp 对数据抓取公司 Bright Data 的诉讼，Bright Data 被控不当访问 X/Twitter 系统，在抓取和出售数据时违反 X 的条款和州的法律。法官认为 X 的理由暴露了它既想要控制用户数据，又想要享受 Communications Decency Act 第 230 条款所赋予的平台无需对第三方内容承担责任的安全港好处。法官认为 X 其实就是想要利用用户数据牟利，同时阻止第三方复制其用户公开的内容。法官认为 X 不能发明自己的版权法，X 的条款并不能限制公开数据的抓取。
https://arstechnica.com/tech-policy/2024/05/elon-musks-x-tried-and-failed-to-make-its-own-copyright-system-judge-says/
