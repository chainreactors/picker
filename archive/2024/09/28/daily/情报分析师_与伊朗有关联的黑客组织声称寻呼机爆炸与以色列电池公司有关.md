情报分析师

Handala，是一个表达亲巴勒斯坦观点的黑客组织。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J65R8BzRfzHGVx9xL2UYH3szxMDonCgVZTPThYqIKJtK6J2oeicwhCFXg/640?wx_fmt=png&from=appmsg)

自2024年5月，Handala对以色列组织发起了一连串的攻击。他们的行动是通过泄露敏感数据和公之于众的方式。

但是他们的在线遭受了重创，网站和X账户双双被关闭。不仅如此，他们还失去了四个活跃的Telegram群组。

之后，Handala 在他们的 Tor 网站上发布了一份声明：

“

最近的行动是摩萨德和 8200 部队以及犹太复国主义政权的一些空壳公司的一系列联合行动！Handala 的黑客在最近几个小时的广泛黑客攻击中，能够从过去几天的操作中获取非常机密和机密的信息，所有文件将在未来几个小时内公布！

操作摘要如下：

* 此次供应链攻击是通过在生产商原产国用特殊类型的热敏爆炸材料污染 Pagers 设备的电池而发生的！
* IIB（以色列工业电池）公司在纳哈里亚的电池已经被这些炸药污染了！
* 摩萨德负责将受污染的电池运送到生产商的原产国！
* 由于爆炸物探测设备对这些电池很敏感，并且需要在多个国家/地区移动它们，摩萨德与 vidisco 空壳公司合作，已经转移了上述货物！
* Vidisco 公司是8200部队的附属公司，如今世界上超过 84% 的机场和海港在其安全单位中使用该公司制作的 X 射线，该单位实际上有一个 8200部队的专用后门！（本项目的完整源码将在接下来的几个小时内发布！
* 受污染的货物通过使用 Vidisco 后门并经过多个国家后到达黎巴嫩！

Handala 已确定此操作中涉及的所有因素，很快所有数据都将公布！

Handala 已成功入侵 Vidisco 和 IIB，他们的 14TB 数据将被泄露！

更多详细信息将在未来几个小时内发布。

”

他们随后发布了两篇帖子：

**Vidisco**

Vidisco 是一家以色列便携式数字 X 射线检查系统开发商和制造商。该公司隶属于政权的 8200 部队，目前全球 84% 的机场在其安检门中使用该公司的产品。

基本上，摩萨德使用嵌入在该公司软件中的后门，通过其认为合适的任何机场的任何货物的安检门。

世界各国的许多安全和军事部门都使用该公司的产品。世界是否知道该公司与摩萨德之间的联系？

接下来的几个小时将有大惊喜。该公司 8 TB 的绝密数据，包括所有代理商、客户、电子邮件、后门来源、软件来源、财务和行政文件等的姓名，现在都掌握在 Handala 手中，很快就会泄露！

**以色列工业电池**

IIB 是隶属于以色列国防部的公司之一，负责设计和生产军事和国防工业的许多重要储能基础设施，如雷达、电信设备、间谍设备等。该公司还用炸药污染了真主党电子设备的电池。

该公司的 6TB 敏感信息，包括所有电子邮件、财务和行政文件、设计和生产文件、客户、人力资源等，即将泄露！

对Handala Hack Team的了解

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/fakJbgvVaNe9bXU9AKWoSCqEuIfde1GoGyjLn8zweJHdvwGUbxN3dOcfaPrq4WATGD2pmmbkoklYh0iaFmdrp5w/640?from=appmsg)

目前尚不清楚，但他们之前的网络域名早期的网络流量来自伊朗 IP 地址。他们文章中的谈话要点与伊朗政府的谈话要点重叠。所有受害者目标都在以色列。

这些说法可信吗？

Handala 尚未提供这些组织数据泄露的证据。联系到上述一家公司时，他们表示他们正遭受“IT 问题”的困扰。

在 Handala 之前的声明中，他们在受害者姓名方面是可信的。

电池声明是否可信呢？目前无法评估，因为迄今为止没有提供任何证据。

黎巴嫩爆炸装置事件中被指控的一家设备制造商声称爆炸发生在电池舱中。日本共同社报道称，日本国际通信研究院院长榎本义树表示，从设备照片显示电池舱周围受损严重，电池可能在采购后被改装成炸药。

2024年9月19日下午 3 点，Handala 已开始在 Telegram 上转储数据，他们声称数据显示了 Mossad 和 Vidisco 之间的合作。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J6Aj5yyJ96JibnhhIFDGadIKTj5AI0ccvricJYZNzy5nUXticeibG16iboycw/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J6RiaGH8tJy8TgmGoRdgE1Ho5icevibU65gw0E98ao70H7W8ZDVcWBM2ynw/640?wx_fmt=png&from=appmsg)

2024年9月20日上午 10 点，Handala 声称已经发布了源代码，并似乎用伊朗无人机威胁 Vidisco。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J6ZO2t7be3F8TzjuwkyRmSzMk0ZJ10edb533GGsXQpO4U8npG4GEUlaQ/640?wx_fmt=jpeg&from=appmsg)

2024年9月23日晚上 7 点，向消息来源证实，Vidisco 的黑客攻击是真实的。他们正在运行重大网络安全事件，其中包括数据泄露。

Handala 已经第五次被 Telegram 踢出。

在被踢出之前，他们发布了关于 Benny Gantz 的指控，包括他们声称的 35k 电子邮件、身份证件和旅行行程。目前尚不清楚他们从哪里获得这些数据，或者他们是否篡改了这些数据，甚至还包含以色列政府的数据。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J61ob9IwvV4qfmgWVibjmxriaQ8Q362d08ac30xeMWvaObribN9rfNqP0pQ/640?wx_fmt=webp&from=appmsg)

他们还发布了各种其他文件转储，这些文件包括来自 Vidisco 的真实数据，包括 Vidisco 办公室内的录音。

Vidisco 或 IIB 与电池攻击没有任何联系，但很明显 Handala 已经获得了 Vidisco 网络的访问权限。

此外，Handala似乎已经获得了前以色列总理Ehud Barak的个人电话的访问权限，发布了一系列指控各种事情的信息以及大量的照片和身份文件。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/Mkvak7WiccmWAfwBOSImUoaicYO3BBS2J63r9Uj22y5rVRHSDgFYMkEiaDD1JC8ibseWIs7dnfrUBLBduykK08aFgg/640?wx_fmt=jpeg&from=appmsg)

就上述爆料，开源信息查了一下Vidisco这家公司。

Vidisco 是一家总部位于以色列的便携式数字 X 射线检测系统开发商和制造商。

该公司由 Shlomo Shapira 于 1988 年创立，旨在开发未爆弹药的数字便携式 X 射线检测，此后，该公司为安全和工业测试市场开发了一系列数字 X 射线检测设备。Vidisco 是最早将非晶硅平板集成到便携式 X 射线系统中的公司之一。

据该公司称，它为世界各地的许多安全和军事机构提供便携式数字射线照相检测系统，包括以色列国防军、英国国防部和美国武装部队。截至 2019 年 1 月，Vidisco 的首席执行官是 Alon Guttel。
