## Steam 停止支持 Windows 7 和 Windows 8

[![游戏](https://icon.solidot.org/images/topics/topicgames.png?123)](/search?tid=13 "游戏")

[Wilson](/~Wilson) (42865)发表于 2024年11月14日 18时42分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=79774&appkey=1370085986&title=Steam%20%E5%81%9C%E6%AD%A2%E6%94%AF%E6%8C%81%20Windows%207%20%E5%92%8C%20Windows%208 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自金刚**

Steam 客户端最新更新停止了对 Windows 7 和 Windows 8 的支持。此举并不出人意料，微软早已停止支持这些旧版本 Windows，而 Valve 提前了两年向客户通知了它的计划。Steam 现在只支持 Windows 10、Windows 11、Linux 和 macOS。根据 10 月的 Steam 硬件调查，只有 0.28% 的 Windows 玩家运行 Windows 7，而运行 Windows 8 和 Windows 8.1 的玩家人数则更少。
https://www.pcgamesn.com/steam/windows-7-and-8-stops-working
