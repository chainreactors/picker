你可以到本文结尾处下载最新版的「网页更新提醒」浏览器插件，该扩展支持 [Chrome](https://www.iplaysoft.com/google-chrome.html)、[Edge](https://www.iplaysoft.com/microsoft-edge.html)、[FireFox](https://www.iplaysoft.com/firefox.html)、360 浏览器。除了提供谷歌应用商店的链接外 (很多人无法访问)，也提供了 .crx 离线安装包，你可以[参考这个教程下载安装 .crx 插件](https://www.iplaysoft.com/p/chrome-install-crx)。

### 第二步：圈选需要监视的网页内容

* 自动模式：通过捕获鼠标停留在页面上的位置，获取需要监控的页面节点，鼠标悬停在页面任何位置后，对应的节点会高亮，点击高亮元素即可完成区域的圈选
* 手动模式：按住鼠标左键，然后在页面上画个圈就可以完成指定区域的圈选，程序就会智能地判断应该监控页面上的哪个节点了

### 第三步：配置规则和提醒方式

![网页更新提醒](https://img.iplaysoft.com/wp-content/uploads/2022/courier/wygxtx_screenshot.png)

* **选择器**：用于检测的时候提取内容，默认是不需要干预的，程序会根据圈选的区域自动生成，当然如果你懂 CSS / XPATH 也可以自行修改
* **选择器**：默认 10 分钟，可以根据需要修改，最低可以支持到 2 秒
* **提醒方式**：支持桌面通知、声音、邮件、钉钉、微信、飞书、企业微信、Telegram、Slack 及其他自定义的 Webhook
* **云端运行**：云端运行是指任务在官方的服务端上运行，不需要开机也能监控
* **性能模式**：高性能模式是指在检测的时候只加载页面必要的代码，不加载图片/媒体等非核心的资源，从而提高检测速度
* **匹配规则**：就是设置关键字，支持正则。匹配规则有两种模式：「全部」和「任意一个」，全部就是要命中全部的关键字才提醒，任意一个就是只要有一个关键字匹配就提醒
* **排除规则**：排除规则只在不存在「匹配规则」的情况才有效，用于忽略那些不感兴趣的变更（比如：时间、数量等）

**然后，插件就会自动定期检测网页内容的变化，一旦有符合条件的内容更新就会通过你配置的通知方式通知你**

### 高级功能

* **检测前预处理**：在提取内容之前，可以通过 Javascript 脚本进行预处理，用来准备提取的内容。使用场景如：有些需要设置查询条件然后查询才能看到数据；自动化测试等
* **更新后预处理**：检测到更新以后，可以通过 Javascript 脚本进行预处理，用来实现自动下单、自动化测试等操作
* **通知前预处理**：消息推送之前，可以通过 Javascript 脚本进行预处理，动态修改通知的人机通知的内容，达到个性化推送的目的
* **私有化部署**：如果你有[服务器资源](https://www.iplaysoft.com/go/vps)，可以通过 Docker 进行私有化部署，实现 24 小时不间断运行
