Forcepoint X вҖ“ Labs з ”з©¶еӣўйҳҹеҸ‘зҺ°дәҶдёҖеңәж–°зҡ„ AsyncRAT жҒ¶ж„ҸиҪҜд»¶ж”»еҮ»жҙ»еҠЁпјҢжӯӨж¬Ўжҙ»еҠЁеҲ©з”ЁеҗҲжі•зҡ„еңЁзәҝжңҚеҠЎжқҘдј йҖ’жҒ¶ж„Ҹжңүж•ҲиҪҪиҚ·гҖӮиҝҷеңәжҙ»еҠЁдёҺ 8 жңҲд»ҪеҸ‘зҺ°зҡ„дёҖж¬Ўж”»еҮ»зұ»дјјпјҢеҮёжҳҫдәҶзҪ‘з»ңзҠҜзҪӘеҲҶеӯҗжҢҒз»ӯеҲ©з”ЁеҸҜдҝЎеҹәзЎҖи®ҫж–ҪжқҘиәІйҒҝжЈҖжөӢе№¶ж¬әйӘ—жҜ«ж— йҳІеӨҮз”ЁжҲ·зҡ„и¶ӢеҠҝгҖӮ

В

ж”»еҮ»е§ӢдәҺдёҖе°ҒеҢ…еҗ« Dropbox зҪ‘еқҖзҡ„зҪ‘з»ңй’“йұјйӮ®д»¶гҖӮзӮ№еҮ»иҜҘй“ҫжҺҘдјҡдёӢиҪҪдёҖдёӘ ZIP ж–Үд»¶пјҢйҮҢйқўеҢ…еҗ«дёҖдёӘ Internet еҝ«жҚ·ж–№ејҸж–Үд»¶пјҢжү“ејҖжӯӨеҝ«жҚ·ж–№ејҸж–Үд»¶дҫҝдјҡеј•еҸ‘иҝһй”ҒеҸҚеә”гҖӮиҝҷдёҖзі»еҲ—ж“ҚдҪңеҢ…жӢ¬еңЁеҗҺеҸ°дёӢиҪҪе№¶жү§иЎҢеӨҡдёӘж–Үд»¶пјҢе…¶дёӯжңүдёҖдёӘ JavaScript ж–Үд»¶гҖҒдёҖдёӘ BAT ж–Үд»¶пјҢжңҖеҗҺжҳҜдёҖдёӘйғЁзҪІ AsyncRAT зҡ„ Python и„ҡжң¬гҖӮ

жҠҘе‘ҠжҢҮеҮәпјҡвҖңиҝҷеңә AsyncRAT ж”»еҮ»жҙ»еҠЁеҶҚж¬ЎиЎЁжҳҺпјҢй»‘е®ўжҳҜеҰӮдҪ•еҲ©з”Ё Dropbox зҪ‘еқҖе’Ң TryCloudflare зӯүеҗҲжі•еҹәзЎҖи®ҫж–ҪдёәиҮӘе·ұи°ӢеҲ©зҡ„гҖӮвҖқ

В

ж”»еҮ»иҖ…йҖҡиҝҮеҗҢж—¶жү“ејҖдёҖдёӘзңӢдјјеҗҲжі•зҡ„ PDF ж–Үд»¶пјҢе·§еҰҷең°жҺ©зӣ–е…¶жҒ¶ж„ҸиЎҢдёәпјҢи®©з”ЁжҲ·иҜҜд»ҘдёәеҸӘжҳҜдёӢиҪҪдәҶдёҖеј еҸ‘зҘЁгҖӮдёҺжӯӨеҗҢж—¶пјҢжҒ¶ж„ҸиҪҜд»¶е°ұеңЁз”ЁжҲ·зі»з»ҹдёӯжүҺдёӢдәҶж №гҖӮ

В

TryCloudflare жҳҜдёҖйЎ№дёә Web еә”з”ЁзЁӢеәҸжҸҗдҫӣеҝ«йҖҹдҫҝжҚ·йҡ§йҒ“зҡ„жңҚеҠЎпјҢжӯӨж¬Ўж”»еҮ»жҙ»еҠЁеҜ№е…¶зҡ„дҪҝз”Ёе°ӨдёәеҖјеҫ—е…іжіЁгҖӮж”»еҮ»иҖ…е°ҶжҒ¶ж„Ҹж–Үд»¶жүҳз®ЎеңЁ TryCloudflare зҡ„еӯҗеҹҹеҗҚдёҠпјҢдёәе…¶ж”»еҮ»иЎҢеҠЁеўһж·»дәҶдёҖеұӮеҗҲжі•жҖ§пјҢдҪҝеҫ—е®үе…ЁйҳІжҠӨж–№жЎҲжӣҙйҡҫе°ҶиҜҘжҙ»еҠЁж Үи®°дёәеҸҜз–‘иЎҢдёәгҖӮ

В

жңҖз»Ҳзҡ„жңүж•ҲиҪҪиҚ·дәӨд»ҳдҫқиө–дәҺ Python и„ҡжң¬пјҢиҝҷдәӣи„ҡжң¬ж—ЁеңЁжү§иЎҢеЈід»Јз ҒпјҢе№¶е°ҶжҒ¶ж„ҸиҝӣзЁӢжіЁе…ҘеҲ°еҗҲжі•еә”з”ЁзЁӢеәҸдёӯгҖӮж”»еҮ»иҖ…дҪҝз”Ё DLL жіЁе…ҘгҖҒиҝӣзЁӢз©әжҙһеҢ–е’Ң APC йҳҹеҲ—жҠҖжңҜжқҘйҡҗз§ҳең°еҗҜеҠЁжҒ¶ж„ҸиҪҜд»¶гҖӮ

Forcepoint и§ЈйҮҠйҒ“пјҡвҖңload.py ж–Үд»¶з»ҸиҝҮ Base64 зј–з ҒпјҢи§ЈеҜҶеҗҺжҲ‘д»¬еҫ—еҲ°е®Ңж•ҙд»Јз ҒгҖӮиҝҷдёӘ Python ж–Үд»¶зҡ„дё»иҰҒзӣ®зҡ„жҳҜйҒҚеҺҶжҜҸдёӘеӯҳеӮЁеЈід»Јз Ғзҡ„ bin ж–Үд»¶пјҢжҜҸдёӘж–Үд»¶еҢ…еҗ«дёҚеҗҢзұ»еһӢзҡ„жҒ¶ж„ҸиҪҜд»¶гҖӮвҖқ
