# [Wireshark 4.4.0 is now available](/forums/diary/Wireshark%2B440%2Bis%2Bnow%2Bavailable/31220/)

**Published**: 2024-08-31. **Last Updated**: 2024-08-31 05:56:41 UTC
**by** [Didier Stevens](/handler_list.html#didier-stevens) (Version: 1)

[0 comment(s)](/diary/Wireshark%2B440%2Bis%2Bnow%2Bavailable/31220/#comments)

This is the first 4.4 release. Many new features have been added, details are [here](https://www.wireshark.org/docs/relnotes/wireshark-4.4.0.html).

One feature I already highlighted are custom columns with field expressions: "[Wireshark 4.4.0rc1's Custom Columns](https://isc.sans.edu/diary/Wireshark%2B440rc1s%2BCustom%2BColumns/31174/)".

![](https://isc.sans.edu/diaryimages/images/20240603-123246.png)

Other new features I'll be looking at are:

* automatic profile switching based on display filter
* converting display filters in to BPF capture filters
* implementing display filter functions via plugins
