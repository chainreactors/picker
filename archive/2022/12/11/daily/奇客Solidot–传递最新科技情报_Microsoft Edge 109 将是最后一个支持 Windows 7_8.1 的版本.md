## Microsoft Edge 109 将是最后一个支持 Windows 7/8.1 的版本

[![Internet Explorer](https://icon.solidot.org/images/topics/topicIE.png?123)](/search?tid=31 "Internet Explorer")

[Wilson](/~Wilson) (42865)发表于 2022年12月10日 21时13分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73617&appkey=1370085986&title=Microsoft%20Edge%20109%20%E5%B0%86%E6%98%AF%E6%9C%80%E5%90%8E%E4%B8%80%E4%B8%AA%E6%94%AF%E6%8C%81%20Windows%207%2F8.1%20%E7%9A%84%E7%89%88%E6%9C%AC "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自火星众神**

微软宣布计划在 2023 年 1 月 12 日发布的 Microsoft Edge 109 将是最后一个支持 Windows 7/8.1 的版本，而 Windows 8/8.1 将于 2023 年 1 月 10 日终止扩展安全更新支持。微软称，Microsoft Edge 和 Webview2 Runtime versions 109 能在 Windows 8/8.1 上继续运行，但不会再收到新功能和安全更新。Microsoft Edge 未来的更新将需要用户将操作系统升级到 Windows 10 或 11。微软同时鼓励开发者停止支持 Windows 7 和8/8.1，以确保用户在终止支持前升级操作系统。根据 Statcounter GlobalStats 的统计，Windows 7 的市场份额仍然有 10.25%，Windows 8.1 则占 2.53%。
https://blogs.windows.com/msedgedev/2022/12/09/microsoft-edge-and-webview2-ending-support-for-windows-7-and-windows-8-8-1/
