千元级无线降噪耳机新选择：Beats Studio Buds + 首发体验

设计出彩、佩戴舒适、功能实用，这是 Beats 近年推出的耳机产品带给我的主要印象。不论是小巧精干的 Beats Studio Buds，还是全能强悍的 Beats Fit Pro，后者更是我每天日常在使用的一副耳机，频率远远高于我持有的另外几副 AirPods 耳机。

所以当得知 Beats Studio Buds 是 Beats 品牌成立以来销售速度最快的耳机产品时，我也毫不感到意外。或许也正因如此，Beats 才想将这副「更适合大多数人」的耳机产品打造得更加完善，并于两年后的现在，推出了这一系列的全新升级作品：Beats Studio Buds +。

![](https://cdn.sspai.com/2023/05/17/df26b8c333987a9159877375fc607ea6.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

选择 Beats 耳机的重要原因，外形设计一定会是于我而言相当重要的因素之一。毕竟作为每天高频使用甚至会佩戴出街的「配饰」，我自然也会希望它能更好地融入不同风格的服饰搭配之中，又不显得过于抢眼。

虽然 Beats Studio Buds + 相较于上一代产品而言并没有外形设计方面大的变化，新增加的透明配色还是足够让人眼前一亮。Beats Studio Buds + 采用的是具有雾面透明效果的外壳材质，因此我们实际上无法很清楚地看到充电盒及耳机内部的元器件结构，若隐若现的元件、隐约可见的发声单元…… Beats Studio Buds + 整体视觉效果的科技感也相当强烈，第一次拿在手上我就仔细观察了好一会。

![](https://cdn.sspai.com/2023/05/17/e545ae388c3e89d27209fe3eb11a4164.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

作为 Beats 耳机的老用户，我曾不止一次地表达过对于不支持无线充电功能的遗憾，如果你也像我一样期待在这款新的产品上看到任何变化，这一期望将会继续落空，Beats Studio Buds + 依然只能继续通过 USB-C 接口来为充电盒以及放入其中的耳机充电。

不过，Beats Studio Buds + 的续航能力相比前代产品又好了不少。搭配充电盒使用，官方宣称 Beats Studio Buds + 的续航时长能够达到惊人的 36 小时 (耳机续航时长 9 小时)，这一数字在前代产品上的测试数据则是 24 小时 (耳机续航时长 8 小时)，提升足足有 50% 之多。

![](https://cdn.sspai.com/2023/05/17/71acf539ec00865b9a46a30271fe44e4.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

推出 Beats Studio Buds 时，Beats 曾介绍说为了兼顾音质体验和佩戴感受，耳机根据大量人耳测量数据与之前积累的人体工学经验进行了设计。而在 Beats Studios Buds + 上，他们还想让佩戴舒适度跟进一步——即使在外形设计几乎没有变化的情况下——而且真的做到了。

虽然在看到官方宣称通过增加通气孔数量增强了 Beats Studio Buds + 的佩戴舒适性的相关介绍时我并不以为意，毕竟在两副几副一模一样的耳机上，让人很难想象只是几个小孔的数量区别能带来多直观的变化，但是实际戴上耳机之后感受还是相当直接的，能够明显感受到 Beats Studio Buds + 戴在耳中的压力更低。用更直观的例子来说的话，戴上 Beats Studio Buds 就像是坐高层电梯，耳内会有些微胀和闷的感觉，戴上 Beats Studio Buds + 则没有这种感受。

![](https://cdn.sspai.com/2023/05/17/ec3311d49fbba8d69a613fd2dbf10e7b.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

当然需要说明的是，这不是在说 Beats Studio Buds 的舒适性欠佳，这种区别只在同时依次佩戴两副耳机进行对比时能够感受出来，Beats Studio Buds 依然是一副足够满足日常需求的耳机，它佩戴稳定、小巧耐看，便携性也足够出色。

![](https://cdn.sspai.com/2023/05/17/0ccefbc89dd6e27d0ac5a96453d64dad.png?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

其次是可替换耳塞。Beats Studio Buds + 在 原有 S/M/L 三种尺寸耳塞的基础上，增加了更小一号的 XS 尺寸选项；虽然我并没有相关需求，相信这项变化也是根据前代产品的用户调研和反馈得来，能够有效满足更多元化的使用需求。

听感方面，新增加的气孔除了增强佩戴舒适性外，Beats 也表示这样的设计让新款耳机的音质表现能够更加出色。当然从我自己的聆听感受来说，两副耳机在听感方面的差距其实相差不大。不过，还是能够听出 Beats Studio Buds + 的低音表现相对更好一些，人声的层次感也更为分明，依然是在同价位产品中综合素质不错的一项选择。

![](https://cdn.sspai.com/2023/05/17/945b893f3f9f71b7202d77cc96d43891.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

而在通透模式以及主动降噪能力方面，Beats 宣称更大尺寸的麦克风、新增加的通气孔、升级之后的芯片都为这两项功能的提升有所助益。根据官方提供的数据，Beats Studio Buds + 在通透模式下的声音清晰度提升最高可达 2 倍，主动降噪效果提升最高可达 1.6 倍。当然，这些数字在日常使用中的意义我认为其实不大，生活场景下环境嘈杂度和复杂度的上限相对而言并没有那么高，很难发挥出硬件在极限测试环境下能够发挥出来的全部能力。即使是 Beats Studio Buds 的通透模式和主动降噪功能，在我看来已经非常好用了，不过 Beats Studio Buds + 现在能做得更好，自然也是一件好事。

既然说 Beats Studio Buds 系列是「更适合大多数人」的耳机，那就不得不提它在 iOS 和 Android 平台上功能体验的诸多一致性，这样的一致性也得益于 Beats 团队为其专门设计打造的芯片。Android 设备不仅能够通过 Beats 官方应用实现类似 iOS 平台上的一键配对 (Google Fast Pair) 能力，更是能使用查找功能定位 Beats Studio Buds + 的位置。耳机也充分利用了不同平台提供的原生功能，比如在 iOS 上还支持 Hey Siri 语音唤醒，在 Android 上可以利用音频切换功能在登录了同一 Google 账号的设备之间无缝切换聆听。

![](https://cdn.sspai.com/2023/05/17/1bb1103aca7b3a3251b79dc893a2f8b4.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

要实现平台体验如此的「一致性」，必然就会有所牺牲。除了之前提过的无线充电的功能缺失以外，由于没有采用 H 系列芯片以及缺少相关传感器，我们并不能使用 Beats Studio Buds + 在 Apple 设备上体验到自动切换、自动播放/暂停等功能，也无法享受到支持动态头部追踪的个性化空间音频功能带来的奇妙体验。

最后说说购买建议。

Beats Studio Buds + 依然是一款综合素质十分出色的 TWS 耳机产品，虽然相比前代 Beats Studio Buds，它的提升并没有那么显著：听感好了一点点，佩戴更舒服一些，功能没有大的变化。

![](https://cdn.sspai.com/2023/05/17/4a2b6e1620dbb2a1389fce6d3e87109c.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

全新的 Beats Studio Buds + 在价格方面也是向前代产品看齐，1349 元的售价相比后者 1199 元的价格高出了 200 元都不到。如果你在物色一副千元级的真无线蓝牙降噪耳机，Beats Studio Buds + 绝对是充满竞争力的选项之一。除了新增加的透明配色足够吸睛，黑白配色也较前代产品做了调整，视觉效果更加优秀。尤其是对于 Android 或者多平台用户而言，Beats Studio Buds + 在保持主要功能一致性的同时，又尽量利用到了平台各自的原生能力，能够带来更好的功能体验。

如果你是 Apple 用户，那么 Beats Studio Buds 系列的两副耳机吸引力就没那么强了。同价位的第三代 AirPods 搭载了 Apple H1 芯片，还能选配 MagSafe 无线充电盒，带来 Apple 生态下最好的设备协作体验。当然，后者在通透模式和降噪功能方面的缺失，也是不得忽视的一大因素，如果你真的非常看重降噪功能，而预算又相对有限，Beats Studio Buds + 相对而言确实是对 Apple 用户来说一个不错的方案。

![](https://cdn.sspai.com/2023/05/17/c5c7624932e1a452c9d26655952c7619.JPG?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

虽然 Beats Studio Buds + 还达到了 IPX4 级的抗汗抗水能力，但是我对这类没有额外固定设计的耳机在运动场景下的使用依然抱有疑虑。如果你需要一副无线降噪运动耳机，Beats Fit Pro 是我更推荐的产品。就像我说的，这副耳机在佩戴的舒适度和稳定性之间达到了一个较好的平衡，运动和日常都好用，Apple H1 芯片也能带来 Apple 生态下最好的使用体验。

对了，老款的 Beats Studio Buds 依然会在新款上市之后继续以 1199 元的价格销售。在千元级的真无线蓝牙降噪耳机产品中，它有着不错的外观，功能也完全够用，还有更多配色选项，也依然还是值得考虑入手的一款产品。
