##### иҜ‘ж–ҮеЈ°жҳҺ

жң¬ж–ҮжҳҜзҝ»иҜ‘ж–Үз« пјҢж–Үз« еҺҹдҪңиҖ… do sonпјҢж–Үз« жқҘжәҗпјҡsecurityonline

иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

Mozilla дёәе…¶ Firefox жөҸи§ҲеҷЁеҸ‘еёғдәҶдёҖдёӘзҙ§жҖҘе®үе…Ёжӣҙж–°пјҢд»Ҙи§ЈеҶідёҖдёӘзӣ®еүҚеңЁйҮҺеӨ–иў«еҲ©з”Ёзҡ„е…ій”®жјҸжҙһгҖӮиҜҘжјҸжҙһиў«иҝҪиёӘдёә CVE-2024-9680пјҢCVSS еҫ—еҲҶдёә 9.8пјҢе…Ғи®ёж”»еҮ»иҖ…еңЁз”ЁжҲ·зі»з»ҹдёҠжү§иЎҢд»»ж„Ҹд»Јз ҒгҖӮ

![Firefox Vulnerability]()

вҖңдёңйғЁж—¶й—ҙе‘ЁдәҢдёҠеҚҲ 8 зӮ№е·ҰеҸіпјҢжҲ‘д»¬ж”¶еҲ°еҸҚз—…жҜ’е…¬еҸё ESET зҡ„иӯҰе‘ҠпјҢд»–д»¬жҸҗйҶ’жҲ‘д»¬еңЁйҮҺеӨ–еҸ‘зҺ°дәҶдёҖдёӘ Firefox жјҸжҙһпјҢвҖқMozilla еңЁдёҖд»Ҫе®үе…Ёе…¬е‘ҠдёӯиҜҙгҖӮвҖңжҲ‘д»¬иҰҒиЎ·еҝғж„ҹи°ў ESET дёҺжҲ‘д»¬еҲҶдә«д»–д»¬зҡ„еҸ‘зҺ°вҖ“жӯЈжҳҜиҝҷж ·зҡ„еҗҲдҪңи®©зҪ‘з»ңеҜ№жҜҸдёӘдәәжқҘиҜҙйғҪжӣҙеҠ е®үе…ЁгҖӮ

иҜҘжјҸжҙһеӯҳеңЁдәҺеҠЁз”»ж—¶й—ҙиҪҙз»„д»¶дёӯпјҢе®ғжҳҜ Firefox Web Animations API дёӯзҡ„дёҖдёӘжңәеҲ¶пјҢз”ЁдәҺжҺ§еҲ¶е’ҢеҗҢжӯҘзҪ‘йЎөдёҠзҡ„еҠЁз”»гҖӮжӣҙе…·дҪ“ең°иҜҙпјҢе®ғжҳҜдёҖдёӘ вҖңйҮҠж”ҫеҗҺдҪҝз”ЁвҖқпјҲuse-after-freeпјүжјҸжҙһпјҢиҝҷжҳҜдёҖз§ҚеҶ…еӯҳжҚҹеқҸжјҸжҙһпјҢеҪ“зЁӢеәҸйҮҠж”ҫеҶ…еӯҳдҪҚзҪ®еҗҺз»§з»ӯдҪҝз”ЁиҜҘдҪҚзҪ®ж—¶е°ұдјҡеҮәзҺ°иҝҷз§ҚжјҸжҙһгҖӮиҝҷж ·пјҢж”»еҮ»иҖ…е°ұеҸҜд»ҘжіЁе…ҘжҒ¶ж„Ҹд»Јз Ғе№¶жҺ§еҲ¶еҸ—еҪұе“Қзҡ„зі»з»ҹгҖӮ

вҖңMozilla и§ЈйҮҠиҜҙпјҡвҖқESET еҸ‘йҖҒз»ҷжҲ‘д»¬зҡ„ж ·жң¬еҢ…еҗ«дёҖдёӘе®Ңж•ҙзҡ„жјҸжҙһеҲ©з”Ёй“ҫпјҢе…Ғи®ёеңЁз”ЁжҲ·и®Ўз®—жңәдёҠиҝңзЁӢжү§иЎҢд»Јз ҒгҖӮвҖңеңЁж”¶еҲ°ж ·жң¬зҡ„дёҖдёӘе°Ҹж—¶еҶ…пјҢжҲ‘д»¬е°ұеҸ¬йӣҶдәҶдёҖдёӘз”ұе®үе…ЁгҖҒжөҸи§ҲеҷЁгҖҒзј–иҜ‘еҷЁе’Ңе№іеҸ°е·ҘзЁӢеёҲз»„жҲҗзҡ„еӣўйҳҹпјҢеҜ№иҜҘжјҸжҙһиҝӣиЎҢйҖҶеҗ‘е·ҘзЁӢпјҢиҝ«дҪҝе®ғи§ҰеҸ‘жңүж•ҲиҪҪиҚ·пјҢе№¶дәҶи§Је®ғжҳҜеҰӮдҪ•е·ҘдҪңзҡ„гҖӮвҖқ

е°Ҫз®ЎжІЎжңүжҸҗеүҚйҖҡзҹҘпјҢиҖҢдё”жјҸжҙһеҲ©з”ЁйқһеёёеӨҚжқӮпјҢдҪҶ Mozilla иҝҳжҳҜеңЁзҹӯзҹӯ 25 е°Ҹж—¶еҶ…ејҖеҸ‘е№¶еҸ‘еёғдәҶдҝ®еӨҚзЁӢеәҸгҖӮиҝҷз§Қеҝ«йҖҹеҸҚеә”еҮёжҳҫдәҶжјҸжҙһзҡ„дёҘйҮҚжҖ§д»ҘеҸҠз«ӢеҚіжӣҙж–°еҲ°жңҖж–°зүҲжң¬зҒ«зӢҗжөҸи§ҲеҷЁзҡ„йҮҚиҰҒжҖ§гҖӮ

дҝ®иЎҘзүҲжң¬еҢ…жӢ¬ Firefox 131.0.2гҖҒFirefox ESR 128.3.1гҖҒFirefox ESR 115.16.1гҖҒThunderbird 115.16гҖҒThunderbird 128.3.1 е’Ң Thunderbird 131.0.1гҖӮ

д»ӨдәәжӢ…еҝ§зҡ„жҳҜпјҢMozilla е·ІзЎ®и®ӨиҜҘжјҸжҙһжӯЈиў«з§ҜжһҒеҲ©з”ЁжқҘж”»еҮ» Tor жөҸи§ҲеҷЁз”ЁжҲ·гҖӮдёҚиҝҮпјҢжңүе…іиҝҷдәӣж”»еҮ»зҡ„жҖ§иҙЁе’Ңж”»еҮ»иҖ…иә«д»Ҫзҡ„иҜҰз»ҶдҝЎжҒҜд»Қз„¶еҫҲе°‘гҖӮ

жң¬ж–Үзҝ»иҜ‘иҮӘsecurityonline еҺҹж–Үй“ҫжҺҘгҖӮеҰӮиӢҘиҪ¬иҪҪиҜ·жіЁжҳҺеҮәеӨ„гҖӮ
