## Chromium 开始支持使用 Rust 语言

[![Chromium](https://icon.solidot.org/images/topics/topicChromium.png?123)](/search?tid=155 "Chromium")

[Wilson](/~Wilson) (42865)发表于 2023年01月13日 15时48分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73884&appkey=1370085986&title=Chromium%20%E5%BC%80%E5%A7%8B%E6%94%AF%E6%8C%81%E4%BD%BF%E7%94%A8%20Rust%20%E8%AF%AD%E8%A8%80 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自星髓**

Google 安全博客宣布，Chromium 项目开始支持使用 Rust 语言。Rust 是一种高性能、内存安全语言，而软件项目发现的大部分漏洞都属于内存安全 bug。它最早是 Mozilla 为 Firefox 浏览器开发的，现在它开始帮助 Chromium 项目变得更安全。Google 开发者对 Mozilla 表达了感谢。目前 Chromium 仅支持第三方 Rust 库，这些库是作为独立组件开发的，Google 现阶段不会扩大 Rust 的使用。
https://security.googleblog.com/2023/01/supporting-use-of-rust-in-chromium.html
