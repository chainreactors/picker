# Linux ELF BOF Template

This repository hosts the core files needed to create ELF Beacon Object Files for Linux.

A Linux Beacon Object File (BOF) is a compiled C program, written to a convention that allows it to execute within an Outflank C2 process (part of [Outflank's OST offering](https://outflank.nl/ost)).
