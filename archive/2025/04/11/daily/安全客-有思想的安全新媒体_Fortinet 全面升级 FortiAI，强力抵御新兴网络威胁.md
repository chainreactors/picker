# Fortinet 全面升级 FortiAI，强力抵御新兴网络威胁

发布时间 : 2025-04-10 10:47:32

![Story image]()

Fortinet 宣布大幅扩展其 FortiAI 功能，在其 Security Fabric 平台中嵌入新的人工智能（AI）特性，以增强对新出现的网络威胁的防护能力，并简化操作流程。

Fortinet 公司创始人、总裁兼首席技术官 Michael Xie 表示：“Fortinet 的人工智能优势源自我们人工智能生态系统的广度和深度，这是十多年来人工智能创新的成果，而且我们拥有的专利数量比其他任何网络安全供应商都要多，这进一步巩固了我们的优势。”

他还称：“通过在 Fortinet Security Fabric平台中全面嵌入 FortiAI，包括新的智能体人工智能功能，我们让客户能够减轻其安全和网络分析师的工作负担，同时提高其安全和网络运营的效率、速度和准确性。”

Michael Xie 补充道：“与此同时，我们扩大了在 Security Fabric 生态系统中的覆盖范围，使客户能够监控和控制其组织内使用生成式人工智能（GenAI）服务的情况。”

据 Fortinet 公司称，人工智能在网络攻击中的应用日益增多，比如那些旨在绕过防御、自动化攻击途径以及利用系统漏洞的攻击，这就要求各组织部署能够抵御攻击，并保护自身人工智能系统不被攻陷的人工智能驱动的安全工具。

Fortinet 公司在过去 15 年里开发了自己的人工智能系统，目前已获得或正在申请的人工智能相关专利超过 500 项。

新扩展的 FortiAI 现在涵盖了 Fortinet 公司采用的全人工智能驱动的安全和网络运营方法。它集成在 Fortinet Security Fabric 平台中，为威胁检测、运营效率提升和人工智能模型保护提供自主且智能的能力。

此次扩展的核心是两项重大增强功能：FortiAI-Assist 和 FortiAI-Protect。

FortiAI-Assist 将生成式人工智能、智能体人工智能以及用于 IT 运营的人工智能（AIOps）相结合，通过自动化和数据分析来简化安全和网络任务。

新功能包括用于网络运营的智能体人工智能应用程序，其中生成式人工智能助手可自主管理网络配置更新，验证并纠正现有配置，并且无需人工干预即可解决问题。该系统还能主动识别并处理有线、无线和软件定义广域网（SD-WAN）环境中的网络问题。

在安全运营方面，智能体人工智能用于对警报进行分类筛选，根据上下文和历史风险数据对通知进行优先级排序。它能抑制重复警报并标记高可信度的威胁。该系统还支持使用人工智能驱动的推理进行自适应威胁搜寻和事件根源追踪，同时威胁情报增强功能会提供更多背景信息，以帮助预先防范对手的攻击策略。

另一方面，FortiAI-Protect 专注于高级威胁检测和情境风险评估。它通过监控 6500 多个与人工智能相关的网址的使用情况，并提供诸如训练模型、用例和数据地理位置等详细信息，来增强安全情报服务。

该系统应用零信任原则来阻止未经授权或高风险的人工智能应用程序，使管理员能够了解并控制其组织内的人工智能使用情况。Fortinet 公司还强调了通过先进的机器学习和数据分析在恶意软件检测方面的增强功能，以及在入侵防范系统（IPS）方面的改进，以识别新的攻击方法。

各组织可以使用 FortiAI-SecureAI 进一步保护其人工智能系统，该功能可保护跨供应商的人工智能基础设施和云原生工作负载，保护模型和知识产权，并实施访问控制。

数据隐私仍然是一个核心关注点。Fortinet 公司表示，FortiAI 采用严格的隐私保护措施，以防止与生成式人工智能助手共享的数据被用于训练底层语言模型。所有查询都在本地处理，确保敏感信息不会离开组织的网络。

随着 FortiAI-Assist、FortiAI-Protect 和 FortiAI-SecureAI 的联合推出，Fortinet 公司旨在支持企业在安全采用人工智能的同时，跟上日益复杂的网络威胁的发展步伐。
