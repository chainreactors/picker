自 2020 年 Slack 在欧盟投诉微软捆绑销售 Microsoft Teams 后，经过三年的拉扯后，最终欧盟还是绝对对微软发起反垄断调查，即便微软此前已经表示可以将 Microsoft Teams 从 Microsoft 365 中剥离进行单独销售。

剥离也没用的原因在于，微软已经利用过去几年的机会破坏了市场竞争，现在这种剥离产品的补救措施实际意义没那么大。

先将时间拨回 2020 年，彼时因为疫情原因，大量企业需要居家办公，这让 Zoom 这类在线会议软件迅速崛起，而传统企业协作软件包括 Slack、Microsoft Teams、钉钉、企业微信、腾讯会议等软件也快速崛起。

在国外，原本 Slack 有较大优势，在疫情期间 Slack 也获得大量企业用户，不过 Microsoft Teams 增长率更高，因为微软的策略是安装 Microsoft 365/Office 的时候自动安装 Microsoft Teams，而且微软还在 Windows 11 里预装 Microsoft Teams 的轻量版本 Chat，最关键的是 Teams 对不少用户来说是 “免费” 的，当然不是真完全免费，而是企业购买的 Microsoft 365 授权里包含 Microsoft Teams。

这导致 Microsoft Teams 的增长率远高于 Slack，这对 Slack 来说是个巨大的打击，加之微软的竞争行为确实有反竞争倾向了，[所以 Slack 决定在欧盟投诉微软](https://www.landiannews.com/archives/77440.html)。

[![剥离产品也没用 欧盟就Microsoft Teams对微软发起正式发垄断调查](https://img.lancdn.com/landian/2020/07/77440.png)](https://img.lancdn.com/landian/2020/07/77440.png)

这几年欧盟和也微软沟通过不少次，[最近一次的沟通就是微软从 Windows 11 里允许用户删除 Chat](https://www.landiannews.com/archives/98511.html) (当前用户只能隐藏但不能卸载，而且 Chat 还强制接管了 Win+C 快捷键组合)，同时微软可以将 Microsoft Teams 从 Microsoft 365 拆分出来，不捆绑安装，也不捆绑销售，即后续企业需要单独购买 Microsoft Teams 授权。

但现在来看谈判是破裂了，据欧委员发布的最新消息，欧盟已经决定对微软发起正式的反垄断调查。

**欧委会：**

*委员会担心微软可能会通过不让客户在其订阅其生产套件 (也就是 Microsoft 365) 时选择是否包含对 Microsoft Teams 的授权，并且微软可能限制其生产力套件与竞争产品之间的互操作性。*

*这些做法可能构成反竞争搭售或捆绑，这会阻止其他通信和协作工具的供应商参与竞争，从而损害欧洲经济区的客户。*

从欧委会的态度来看，欧盟认为企业订阅 Microsoft 365 时，其价格是包含 Teams 的，而微软并没有给客户提供去除 Teams 价格的版本，同时 Teams 没有提供与其他产品的互操作性，也就是提供某些 API 进行兼容、转移之类的。

对微软来说，一旦欧委会的反垄断调查成立，那么微软将会被处以巨额罚款，以微软这体量和 Microsoft Teams 的客户量来算，估计微软也要大出血。
