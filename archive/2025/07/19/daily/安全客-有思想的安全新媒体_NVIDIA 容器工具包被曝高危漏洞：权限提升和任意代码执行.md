NVIDIA еҸ‘еёғе…ій”®е®үе…Ёжӣҙж–°пјҢдҝ®еӨҚе…¶ Container Toolkit е’Ң GPU Operator дёӯеӯҳеңЁзҡ„дёӨйЎ№й«ҳеҚұжјҸжҙһпјҢиҝҷдәӣжјҸжҙһеҸҜиғҪиў«ж”»еҮ»иҖ…еҲ©з”ЁпјҢд»ҘжҸҗеҚҮжқғйҷҗжү§иЎҢд»»ж„Ҹд»Јз ҒгҖӮ

жң¬ж¬Ўдҝ®еӨҚзҡ„жјҸжҙһзј–еҸ·дёә CVE-2025-23266 е’Ң CVE-2025-23267пјҢеҪұе“ҚжүҖжңүиҝҗиЎҢ NVIDIA Container Toolkit вүӨ 1.17.7 зүҲжң¬д»ҘеҸҠ GPU Operator вүӨ 25.3.0 зүҲжң¬зҡ„зі»з»ҹе№іеҸ°гҖӮ

### иҰҒзӮ№жҰӮи§Ҳпјҡ

1. **Container Toolkit еӯҳеңЁжјҸжҙһпјҢеҸҜиғҪиў«з”ЁдәҺеңЁе®№еҷЁзҺҜеўғдёӯд»ҘжҸҗжқғж–№ејҸжү§иЎҢд»»ж„Ҹд»Јз Ғ**пјӣ
2. **еҸ—еҪұе“ҚзүҲжң¬иҢғеӣҙдёә Container Toolkit 1.17.7 еҸҠд»ҘдёӢпјҢGPU Operator 25.3.0 еҸҠд»ҘдёӢ**пјӣ
3. **е®ҳж–№е»әи®®з«ӢеҚіеҚҮзә§иҮі Container Toolkit 1.17.8 е’Ң GPU Operator 25.3.1пјҢжҲ–йҖҡиҝҮзҰҒз”Ё `enable-cuda-compat` hook е®һж–Ҫдёҙж—¶зј“и§ЈжҺӘж–Ҫ**гҖӮ

иҝҷдёӨйЎ№жјҸжҙһеҰӮиў«еҲ©з”ЁпјҢеҸҜиғҪйҖ жҲҗдёҘйҮҚе®үе…ЁйЈҺйҷ©пјҢеҢ…жӢ¬пјҡ**жқғйҷҗжҸҗеҚҮпјҲж”»еҮ»иҖ…иҺ·еҫ—жӣҙй«ҳзі»з»ҹжқғйҷҗпјүгҖҒж•°жҚ®зҜЎж”№гҖҒж•Ҹж„ҹдҝЎжҒҜжі„йңІгҖҒжӢ’з»қжңҚеҠЎж”»еҮ»пјҲDoSпјү зӯү**гҖӮе»әи®®зӣёе…із”ЁжҲ·е°Ҫеҝ«е®ҢжҲҗеҚҮзә§пјҢд»ҘйҷҚдҪҺжҪңеңЁеЁҒиғҒгҖӮ

### **е…ій”®е®№еҷЁжјҸжҙһеҲҶжһҗ**

жңҖдёәдёҘйҮҚзҡ„жјҸжҙһ **CVE-2025-23266** зҡ„ CVSS v3.1 еҹәзЎҖиҜ„еҲҶдёә **9.0**пјҢиў«еҪ’зұ»дёәвҖңдёҘйҮҚвҖқзә§еҲ«гҖӮиҜҘжјҸжҙһеӯҳеңЁдәҺз”ЁдәҺе®№еҷЁеҲқе§ӢеҢ–зҡ„йғЁеҲҶ Hook дёӯпјҢж”»еҮ»иҖ…еҸҜеҖҹжӯӨд»ҘжҸҗеҚҮзҡ„жқғйҷҗжү§иЎҢд»»ж„Ҹд»Јз ҒгҖӮ

е…¶ж”»еҮ»еҗ‘йҮҸжҸҸиҝ°дёәпјҡвҖң**AV:A/AC:L/PR:L/UI:N/S:C/C:H/I:H/A:H**вҖқпјҢж„Ҹе‘ізқҖж”»еҮ»иҖ…еҸҜйҖҡиҝҮйӮ»иҝ‘зҪ‘з»ңи®ҝй—®пјҢеҖҹеҠ©дҪҺеӨҚжқӮеәҰзҡ„ж”»еҮ»и·Ҝеҫ„пјҢеңЁж— йңҖз”ЁжҲ·дәӨдә’зҡ„жғ…еҶөдёӢйҖ жҲҗдёҘйҮҚзҡ„ж•°жҚ®жі„йңІгҖҒе®Ңж•ҙжҖ§з ҙеқҸе’ҢжңҚеҠЎдёӯж–ӯгҖӮиҜҘжјҸжҙһеҪ’зұ»дёә **CWE-426пјҡдёҚеҸ—дҝЎд»»зҡ„жҗңзҙўи·Ҝеҫ„дҪҝз”Ё**гҖӮ

еҸҰдёҖйЎ№жјҸжҙһ **CVE-2025-23267** зҡ„ CVSS иҜ„еҲҶдёә **8.5**пјҢиў«е®ҡдёәвҖңй«ҳеҚұвҖқзә§еҲ«гҖӮиҜҘжјҸжҙһеҪұе“Қ `update-ldcache` HookпјҢж”»еҮ»иҖ…еҸҜеҲ©з”Ёзү№еҲ¶зҡ„е®№еҷЁй•ңеғҸеҸ‘иө·з¬ҰеҸ·й“ҫжҺҘи·ҹиёӘж”»еҮ»гҖӮ

иҜҘжјҸжҙһеҪ’еұһ **CWE-59пјҡж–Үд»¶и®ҝй—®еүҚдёҚеҪ“й“ҫжҺҘи§Јжһҗ**гҖӮ

дёҠиҝ°дёӨйЎ№жјҸжҙһеқҮйҖҡиҝҮиҙҹиҙЈд»»зҡ„ж–№ејҸжҠ«йңІпјҡ

* CVE-2025-23266 з”ұ Trend Zero Day Initiative зҡ„ Nir Ohfeld е’Ң Shir Tamari жҠҘе‘Ҡпјӣ
* CVE-2025-23267 еҲҷз”ұеҚҺдёәдә‘ Nebula е®үе…Ёе®һйӘҢе®Өзҡ„ Lei Wang е’Ң Min Yao жҠҘе‘ҠгҖӮ

![]()

### **е®үе…Ёжӣҙж–°дҝЎжҒҜ**

дёәеә”еҜ№дёҠиҝ°жјҸжҙһпјҢNVIDIA е·ІеҸ‘еёғж–°зүҲиҪҜд»¶иҝӣиЎҢдҝ®еӨҚгҖӮе»әи®®жүҖжңүдҪҝз”Ё NVIDIA Container Toolkit зҡ„з”ЁжҲ·е°ҶзүҲжң¬еҚҮзә§иҮі 1.17.8пјҢд»Ҙжӣҝд»ЈжӯӨеүҚзҡ„жүҖжңүзүҲжң¬пјҲеҢ…жӢ¬ 1.17.7 еҸҠжӣҙж—©зүҲжң¬пјүгҖӮ

еҜ№дәҺиҝҗиЎҢеңЁ Linux е№іеҸ°дёҠзҡ„ NVIDIA GPU OperatorпјҢе»әи®®е°ҶзүҲжң¬еҚҮзә§иҮі **25.3.1**пјҢиҰҶзӣ–жӯӨеүҚзҡ„жүҖжңүзүҲжң¬пјҲеҢ…жӢ¬ 25.3.0 еҸҠжӣҙж—©зүҲжң¬пјүгҖӮ

йңҖиҰҒзү№еҲ«жіЁж„Ҹзҡ„жҳҜпјҢ**CDI жЁЎејҸдёӢзҡ„жјҸжҙһ**д»…еҪұе“Қд»ҘдёӢзүҲжң¬пјҡ

* Container Toolkit 1.17.5 д№ӢеүҚзҡ„зүҲжң¬пјӣ
* GPU Operator 25.3.0 д№ӢеүҚзҡ„зүҲжң¬гҖӮ

еңЁиғҪеӨҹз«ӢеҚіиҝӣиЎҢеҚҮзә§еүҚпјҢжңәжһ„дәҰеҸҜйҖҡиҝҮ**зҰҒз”ЁеӯҳеңЁжјҸжҙһзҡ„ enable-cuda-compat hook**жқҘе®һж–Ҫдёҙж—¶зј“и§ЈжҺӘж–ҪгҖӮ

й’ҲеҜ№дҪҝз”Ё NVIDIA Container Runtime зҡ„з”ЁжҲ·пјҢеҸҜйҖҡиҝҮзј–иҫ‘й…ҚзҪ®ж–Үд»¶ `/etc/nvidia-container-toolkit/config.toml`пјҢе°Ҷ `features.disable-cuda-compat-lib-hook` еҠҹиғҪйЎ№и®ҫзҪ®дёә `true` жқҘзҰҒз”Ёзӣёе…іеҠҹиғҪгҖӮ
