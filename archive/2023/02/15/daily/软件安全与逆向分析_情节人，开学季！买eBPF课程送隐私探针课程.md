##

**课程应用场景**

> * 游戏反外挂人员高效对抗基于 eBPF 实现的游戏外挂
> * 云原生安全相关人员掌握基于 eBPF 的容器探测与可观测技术
> * 系统安全人员分析 eBPF 相关的病毒木马
> * 代码安全分析沙箱开发人员编写 App 动态分析 Hook 插件
> * 隐私合规相关人员掌握全面的 App Hook 点与数据分析
> * 逆向工程人员使用 eBPF 实现底层的 App Hook 技术
> * 爬虫开发人员基于 eBPF 技术绕过安全限制

***2***

##

**课程受众人员**

> * 隐私合规从业人员
> * 游戏安全从业人员
> * App 逆向分析人员
> * 安全沙箱开发人员
> * 安全爱好者
> * 爬虫与反爬相关从业人员
> * 业务风控从业人员

***3***

##

**课程目标**

完成本课的学习后，您将具备以下能力：

> * Frida 一些高级功能的使用
> * 使用 Frida 快速调试 API 的技巧
> * 网络安全相关的法律法规
> * 安卓设备指纹 API 接口的认识
> * 安卓设备指纹 API 接口数据的获取
> * 自己动手打造隐私合规数据采集工具
> * 不同版本中敏感 API 差异
> * 安卓敏感行为的分析

***4***

##

**课程特色**

本系列课程采用代码实战讲解，循序渐进的讲解 eBPF 技术与安全沙箱相关技术：

> * 每一节实战部分都有实例代码
> * 一些未公开的技术技巧
> * 全面覆盖 eBPF 与沙箱技术点
> * 代码工具可操作性与利用性强
> * VIP 群答疑，解决学习遇到的问题
> * 课程无限重复观看，直到学会为止
