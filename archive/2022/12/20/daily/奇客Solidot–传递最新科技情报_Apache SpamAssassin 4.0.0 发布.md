## Apache SpamAssassin 4.0.0 发布

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[wanwan](/~wanwan) (42055)发表于 2022年12月19日 17时41分 星期一 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73692&appkey=1370085986&title=Apache%20SpamAssassin%204.0.0%20%E5%8F%91%E5%B8%83 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自尘埃记**

Apache SpamAssassin 垃圾邮件过滤器释出了 v4.0.0 版本。新版包含了大量变更和调整，显著改进了对国际语言文本的处理，包含了完整的 Unicode 字符支持。它整合了过去数年的修正，显著改进了分类和性能，已在生产系统中进行了全面测试。开发者强烈建议用户尽快升级。
https://spamassassin.apache.org/news.html
