OpenAI 目前已经为全球初创公司推出专享计划，初创公司可以向 OpenAI 申请免费的 API 额度，用来测试、构建基于 GPT 的应用程序。

要申请免费额度并不算复杂但有一定的门槛，至少没有真实的初创公司那不太可能成功白嫖，另外还有个门槛是初创企业必须在 TechCrunch 旗下的 Crunchbase 数据库里建档了。

[![[申请] OPENAI向初创公司赠送免费API额度 最高15000美元但有一定门槛](https://img.lancdn.com/landian/2023/03/98055.png)](https://img.lancdn.com/landian/2023/03/98055.png)

**免费额度方面：**

* 一般性的初创公司最高可以获得 2500 美元的 API 额度，相当于 12.5 亿个 gpt-3.5-turbo tokens
* 由 OpenAI VC 投资的初创公司最高可以获得 15000 美元的 API 额度，相当于 75 亿个 gpt-3.5-turbo tokens
* 以上额度的有效期都是 1 年，即审批通过后 1 年内有效，过期作废

**其他要求：**

* 必须是公司
* 成立 7 年以内
* 之前没有获得 OpenAI 赠送的额度
* 需要有一个有效的公司网站
* Crunchbase
* 必须提供产品演示 / 概念验证
