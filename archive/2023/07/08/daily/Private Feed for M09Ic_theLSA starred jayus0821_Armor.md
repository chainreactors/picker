# 蜜罐溯源技术原理

蜜罐溯源这部分用到的技术主要是jsonp
jsonp全称是 JSON with Padding ，是基于 JSON 格式的为解决跨域请求资源而产生的解决方案。

如果某些站点存在jsonp劫持漏洞，如web1这个站点有个jsonp接口暴露者，功能就是返回用户的姓名：

```
<?php
$callback = $_GET['callback'];
print $callback.'({"id" : "1","name" : "readname"});';
?>
```
