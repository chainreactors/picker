# Golconde HTTP 下载器

Golconde 是一个轻量级的 HTTP 下载工具，使用纯汇编语言编写，专为 x86\_64 Linux 系统设计。它主要用于在受限环境中实现文件下载功能的第一阶段工具。

## 特性

* 纯汇编语言实现，无需任何依赖
* 支持通过命令行参数指定目标 IP 和端口
* 固定下载保存到根目录的 /wget 文件
* 极小的程序体积，适合通过字符串方式传输
* 专门设计用于获取完整版的 wget 工具

## 使用场景

Golconde 设计用于以下场景：

1. 在无法直接上传二进制文件的受限环境中
2. 通过 Base64/Echo/Printf 等方式将 Golconde 的小型二进制文件转成字符串
3. 在目标系统重新组装 Golconde
4. 使用 Golconde 下载完整的 wget 工具（推荐使用 busybox wget，仅 90KB）
5. 后续可使用完整版 wget 下载其他所需文件

## 使用方法

```
./golconda <IP地址> <端口>
```

例如：

```
./golconda 127.0.0.1 8080
```

## 编译说明

要获得最小的程序体积，请按以下步骤编译：

```
# 汇编（禁用所有警告）
nasm -f elf64 -w-all golconda.asm -o golconda.o

# 链接（禁用额外特性以减小体积）
ld -s -z norelro --no-eh-frame --build-id=none -o golconda golconda.o

# 使用 UPX 进行压缩
upx --best golconda
```

## 推荐的 wget 源

建议下载 busybox 的 wget 版本：

```
https://www.busybox.net/downloads/binaries/1.35.0-x86_64-linux-musl/busybox_WGET
```

该版本仅约 90KB，功能完整，适合作为第二阶段工具。

## 工作流程

1. 准备阶段：

   * 编译 Golconde 获得最小体积的二进制文件
   * 将二进制转换为 Base64 或其他字符串形式
2. 部署阶段：

   * 在目标系统通过字符串重建 Golconde
   * 执行 Golconde 下载完整的 wget
   * wget 成功下载后，获得完整的文件下载能力
3. 后续操作：

   * 使用获得的 wget 下载其他所需工具
   * 建立完整的工具链

## 限制说明

* 不支持 DNS 解析，需要直接使用 IP 地址
* 固定下载到 /wget 路径
* 不支持 HTTPS
* 建议配合小型的 wget 版本使用
* 需要目标系统有写入根目录的权限

* 使用系统调用实现网络通信
