根据微软的产品政策，[IE 浏览器之前已经结束支持](https://www.landiannews.com/archives/96577.html)，所以微软通过 Windows 更新强制禁用 IE 浏览器，如果用户试图启动 IE 浏览器则会被强制跳转到 Microsoft Edge 浏览器。

同时微软还通过 Windows 更新删除 IE 浏览器图标、相关元素和设置，也就是要把 IE 相关的东西从 Windows 10/11 里彻底清除。

[![鉴于企业客户的要求 微软宣布IE11将在特殊情况下继续运行不会再被删除](https://img.lancdn.com/landian/2022/04/93301.png)](https://img.lancdn.com/landian/2022/04/93301.png)

但这种做法也引起企业客户的不满，毕竟目前确实还有相当多的企业客户依赖 IE 浏览器，最终微软觉得还是要听取一些企业客户的意见，为企业客户增加配置选项，如果客户确实需要继续保留 IE 相关的内容，可以通过配置选项保留。

**微软在产品公告中表示：**

*根据客户反馈，IE11 视觉参考例如开始菜单和任务栏中的 IE11 图标将不再会像之前所说的那样被 Windows 更新删除。组织 (政企、教育机构等) 可以控制 IE 相关元素被删除的时间，如果客户没有配置这些选项，那么系统将继续删除 IE11 的视觉参考元素。*

根据说明企业 IT 管理员需要修改组策略对内网中的设备进行控制，如果没有主动设置那就默认沿用微软的设置，所以有需要的企业最好尽快操作一下。

***PS：IE 11 本身就没有被删除，只是被禁用和删除图标，现在保留图标后看起来就像没被删似的。***
