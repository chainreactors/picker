## 前言

这几年用过的各种网盘程序不下 20 款,始终没有一款满意的,直到前段时间在某论坛看到有人分享了`vue-file-manager`源码,部署使用了一段时间后觉得功能,颜值都很满意,于是推荐给大家.

## 官方文档

博主并没有详细的写篇教程来教大家部署,主要原因是该程序对环境要求相对复杂,所以建议有一定基础的朋友自行参考官方文档部署.

**官方网站**
[https://vuefilemanager.com/](https://blog.upx8.com/go/aHR0cHM6Ly92dWVmaWxlbWFuYWdlci5jb20v)

**官方文档地址:**
[https://gist.github.com/MakingCG/a702a112be63bc6f0032dd55522327cf](https://blog.upx8.com/go/aHR0cHM6Ly9naXN0LmdpdGh1Yi5jb20vTWFraW5nQ0cvYTcwMmExMTJiZTYzYmM2ZjAwMzJkZDU1NTIyMzI3Y2Y)

`值得注意是 PHP 环境要求,同时需要 MySQL 数据库的支持:`

* Intl
* GD
* BCMath
* PDO
* SQLite3
* Ctype
* Fileinfo
* JSON
* Mbstring
* OpenSSL
* Tokenizer
* XML
* Exif
