# 【联盟动态】第七届中国网信产业前锋汇成功举办

[行业](https://www.4hou.com/category/industry)
2023-05-29 18:45:27

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)96217

收藏

导语：5月28日，第七届中国网信产业前锋汇（以下简称“前锋汇”）在京成功举办。

2023年5月28日下午，作为2023年中关村国际技术交易大会重点场次活动之一，第七届中国网信产业前锋汇（以下简称“前锋汇”）在中关村软件园国际会议中心报告厅成功举办。中关村国际技术交易大会自2020年起成为中关村论坛的重要组成部分，本届前锋汇被纳入2023年中关村国际技术交易大会重要活动，以“全球数字经济发展与数据安全关键技术”为主题，围绕如何增强网络安全防护能力、提升数据安全保障水平、有效防范各类安全威胁等问题展开深入的思想交流。前锋汇上，专家观点精彩纷呈，为与会嘉宾及观众奉上一场干货满满的行业盛宴，成为中关村国际技术交易大会中最具亮眼的活动之一，再度提升了前锋汇在业界的影响力。

![第七届中国网信产业前锋汇-致辞全景.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414401193395.jpeg "1685414136181743.jpeg")

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414401332712.jpeg "1685414184203172.jpeg")

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414402928634.jpeg "1685414184140092.jpeg")![第七届中国网信产业前锋汇-于锐致辞.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414402196859.jpeg "1685414224320994.jpeg")![第七届中国网信产业前锋汇-于晴致辞.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414403123521.jpeg "1685414236892135.jpeg")

本届前锋汇由公安部网络安全保卫局原局长顾建国主持，北京科技创新促进中心党委书记王红，公安部第一研究所副所长、中国计算机学会计算机安全专业委员会主任于锐，中国（中关村）网络安全与信息化产业联盟理事长于晴为前锋汇发表致辞，阐述了网信产业发展的重要意义，并对数据安全产业发展对数字经济保障的厚重寄托。

![第七届中国网信产业前锋汇-云晓春主旨演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414403719483.jpeg "1685414292832577.jpeg")

前锋汇主旨演讲环节，中关村实验室主任助理/首席科学家/网络安全应急技术国家工程研究中心主任云晓春带来《数字替身——一种APT防御新思路》主旨演讲，系统思考并提出一种新型技术的用于防御APT攻击的框架方法，其主旨思想是用“替身”替代“本体”，并技术对比了蜜罐等技术的优劣点等，并介绍了不同场景下的技术部署问题。

![第七届中国网信产业前锋汇-张阳线下主旨演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414404201323.jpeg "1685414309464089.jpeg")

德国CISPA亥姆霍兹信息安全中心终身教授张阳带来《针对机器学习模型的攻击》主旨演讲，在国际视角下分享机器学习发展现况，并从多层技术讨论了机器学习技术上的薄弱环节，引发了AI大数据模型时代下的安全思考。

![第七届中国网信产业前锋汇-张继春桔皮书《数据安全》发布.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414404214506.jpeg "1685414325130742.jpeg")

在联盟产业研究成果发布环节，联盟发布中国网信产业桔皮书《数据安全》发展现状与趋势（以下简称“《数据安全》”），联盟网信战略研究中心主任张继春代表编委会现场发布并解读桔皮书，就数据安全多维战略背景，数据安全技术趋势，以及产业发展做出系统性介绍。在技术趋势与产业发展方面，桔皮书围绕数字经济落地过程中的技术问题和市场化趋势做了系统概括，凝聚集体智慧集准确性、权威性于一身，值得产业各界借鉴参考。

![第七届中国网信产业前锋汇-桔皮书参编单位颁发证书合影.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414405106679.jpeg "1685414345212904.jpeg")

《数据安全》的发布离不开各参编单位的集思广益、群策群力，前锋汇上也举行了参编单位授证仪式，由北京科技创新促进中心党委书记王红、中国（中关村）网络安全与信息化产业联盟理事长于晴为各参编单位代表颁发《参编证书》。

![第七届中国网信产业前锋汇-孙保成《2023网络安全企业科技能力报告》发布.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414405537166.jpeg "1685414366349041.jpeg")

第七届前锋汇还发布了《2023网络安全企业科技能力报告》，该报告由联盟牵头，安全419与中知数通作为支撑单位联合发布，由安全419 COO孙保成现场解读。该报告基于可公开获得的知识产权大数据，挖掘各网络安全企业主体知识产权的获得情况、质量与价值、转化及增长，提供了全新的洞察视角。据了解，该报告系统调研了国内520家工商登记注册主体，报告统计有效专利15709件，有效专利总估值超过106亿元人民币，其中高价值专利约占三分之二，受支撑的专利创造和储备势头较为强劲。

![第七届中国网信产业前锋汇-陈广勇主题演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414418264112.jpeg "1685414418264112.jpeg")

在主题演讲环节，公安部信息安全等级保护评估中心总工程师陈广勇带来《当前网络安全形势和有关工作要求》主题演讲，从当前网络安全面临的新形势、新挑战，国家网络安全工作新任务、新要求两方面展开分享，并指出要加强顶层设计、坚持依法保护，以总体国家安全观为统领，坚持安全和发展，建立网络和数据安全综合防御体系。

![第七届中国网信产业前锋汇-李滨线下主题演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414439134547.jpeg "1685414439134547.jpeg")

腾讯云安全总经理李滨在《企业数据安全风险治理与建设实践》主题演讲中指出，数据安全正面临的五大挑战分别为企业经营模式与IT生命周期变化、新技术及架构的演进、外部攻击威胁日趋严峻、合规驱动企业数据安全治理新架构、碎片化与技术复杂度，并对腾讯云数据安全体系框架进行了系统全面的介绍。

![第七届中国网信产业前锋汇-檀林线下主题演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414406101971.jpeg "1685414383166641.jpeg")

海尔海创汇首席生态官/中关村新场景MA Club发起人檀林做《生成式AI与企业数智化转型机会》主题演讲，从生成式AI是人工智能的侧枝进化、人工智能应用新物种的大爆炸两方面阐述了ChatGPT带来的改变，并从有哪些职业和工作无法被AI取代及企业如何利用生成式AI加速数智化转型两方面说明我们应该如何拥抱这个新的“范式革命“。

![第七届中国网信产业前锋汇-翟胜军线下主题演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414469181597.jpeg "1685414469181597.jpeg")

奇安信科技集团首席安全专家翟胜军在《奇安信数据安全防护运营实践分享》主题演讲中指出，数字城市建设的首要挑战就是网络安全，而当前我国城市安全运营存在明显短板，半数以上城市处在基础和入门阶段。如何构建一套行之有效的网络安全保障体系，保护这个庞大的网络，降低其带来的威胁至关重要。

![第七届中国网信产业前锋汇-张伟军线下主题演讲.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414485127068.jpeg "1685414485127068.jpeg")

安芯网盾（北京）科技有限公司技术副总裁张伟军在《基于内存安全技术应对未知威胁攻击的应用分享》主题演讲中提出，在未来的半年或一年里，高级APT组织所形成的攻击频次预测可能会翻一倍、五倍甚至十倍。安芯网盾作为国内最早专注于内存安全领域的企业，以新视角来应对日益增长的未知威胁问题，为内存安全技术发展提供新思路。

![第七届中国网信产业前锋汇-圆桌对话合影.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230530/1685414500356183.jpeg "1685414500356183.jpeg")

圆桌对话环节由国家信息中心研究员李新友主持，中国科学院信息工程研究所副所长/大数据安全研究室主任王伟平、中国科学院计算机网络信息中心首席科学家/研究员廖方宇、中国互联网络信息中心工程实验室主任张海阔、北京大学计算机学院教授/北京大学网络与信息安全实验室主任陈钟、北京理工大学网络空间安全学院党委书记/教授祝烈煌以“数据流通与安全”为主题，就数据流通指标要求、制度建设、监管创新、数据跨境流通等相关问题进行研讨，分享权威观点，为当前我国数据安全治理难点问题提供有益指导和参考。

中国（中关村）网络安全与信息化产业联盟作为前锋汇的主办单位，未来将继续保持与中关村论坛和中关村国际技术交易大会的紧密合作与联系，主动融入，深度参与，紧跟国家发展战略，发挥桥梁纽带作用，为我国网信事业发展贡献力量。

**中国网信产业前锋汇**

中国网信产业前锋汇，是由中国（中关村）网络安全与信息化产业联盟自2016年发起组织的重要行业特色峰会活动，旨在为在产业发展、企业做强、技术创新和政策导向等方面更好的为产业、企业、政府和社会服务。前锋汇始终秉承“聚焦网信的前沿前端方向，聚集专业人士研究探索”的办会宗旨，为行业治理提供智力支持，为行业发展做出更明确的方向引领。经过七届前锋汇举办，“中国网信产业前锋汇”已经成为联盟在行业内的一张名片。

从“全球背景下的中国网信企业崛起”，到“全球数字经济发展与数据安全关键技术”，前锋汇已连续成功举办七届。本届前锋汇重点指出，近年来国家高度重视数据安全，合规升级、监管趋势成为主旋律，发展数据安全产业对于提高各行业各领域数据安全保障能力，加速数据要素市场培育和数据价值释放，夯实数字中国建设和数字经济发展基础有着重要意义。
