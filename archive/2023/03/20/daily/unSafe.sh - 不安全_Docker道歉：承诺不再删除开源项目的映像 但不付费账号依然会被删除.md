本周蓝点网提到 Docker 向所有注册类型为组织的开发者发邮件通知：如果用户不升级到团队付费计划，[那么账号将被删除，账号下的所有镜像也会被一并删除](https://www.landiannews.com/archives/97869.html)。

这项新政策不影响个人免费版用户，个人免费用户政策继续使用 2020 年版的，即如果镜像超过 6 个月都未被使用，那么镜像会被删除。

在 Docker Hub 上有不少开源项目的开发者注册为组织类型，在新政策下要么选择付费 480 美元 / 年，要么等着删账号，这引起了极大的争议，也引起开源社区极大的恐慌。

[![Docker道歉：承诺不再删除开源项目的映像 但不付费账号依然会被删除](https://img.lancdn.com/landian/2023/03/97869.png)](https://img.lancdn.com/landian/2023/03/97869.png)

经过一周发酵后 Docker 官方今天发布公开信道歉，**新政策改为不付费账号依然会被冻结并删除**，但已经发布的 Docker 映像不会被删除。

另外开源项目开发者可以重新尝试申请免费的 DSOS 项目，之前申请失败也可以重新申请，如果申请成功账户可以继续用也不必付费。

**Docker 对这次新政策进行了详细解释和说明，大概内容如下：**

Free Teams 订阅也就是组织类型的免费订阅计划已经被弃用，弃用原因是该订阅计划针对性比较差，也不如 Docker 推出的 DSOS 计划，DSOS 即 Docker 为开源项目赞助的订阅层级，开源项目需要主动申请，审批通过后即可免费使用。

Docker 最新承诺只有维护者主动删除公开映像时，Docker 才会从 Docker Hub 中删除映像。如果维护者不采取任何行动，那公开映像不会被删除。

至于冻结问题 Docker 并未让步，如果申请 DSOS 失败且开发者仍然不愿意付费升级的话，那在 30 天后账户将被冻结无法操作。此时公开存储库会继续运行但不能发布或修改，私人存储库会被停用，Docker 称本来私人存储库就是付费使用的。

以上时间节点为 4 月 13 日，建议使用 Free Teams 的开发者尽快申请 DSOS 计划，否则到期后账户将无法再操作，**特别强调：到期后账户会被删除，但项目名称不会被释放，防止其他人占用。**
