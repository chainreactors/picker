# 重磅来袭 | 2023数字供应链安全大会邀请函（DSS 2023）

2023数字供应链安全大会（DSS 2023）将于8月10日在北京·国家会议中心隆重开幕。本次大会由悬镜安全主办，ISC互联网安全大会组委会、中国软件评测中心（工业和信息化部软件与集成电路促进中心）、中国信息通信研究院云计算与大数据研究所、CCF计算机安全专业委员会联合发起，OpenSCA开源社区、XRASP代码疫苗社区共同协办，**以“开源的力量”为主题**，打造以“数字供应链安全”技术论道、产业变革、创新发展为核心交付价值，立足世界、规模宏大、影响力深远的安全盛会，擘画开源驱动下数字安全发展新蓝图。

![]()

数字时代，随着AI、大数据、云计算等信息技术的高速发展和广泛普及，以数字化应用为代表的数字资产要素已成为支撑社会运转的核心因素之一，决定着企业数字化转型进程，渗透到业务发展、技术研发、企业管理等各个场景，贯穿于企业发展全生命周期。传统的软件供应链已演进成为数字供应链，并成为企业重点关注的新焦点。加快落地数字供应链安全治理工作，提升数字供应链安全风险的发现能力、分析能力、处置能力、防护能力以及数字供应链安全管理水平，迫在眉睫。

悬镜安全作为DevSecOps敏捷安全领导者，顺应数字产业时代的发展规律，作为行业先行者，持续引领技术领先优势的同时，也将以更加长远的战略眼光布局未来，更中立、更国际化的视角，在更大的范围，赋能更多的学术同仁、行业用户和生态伙伴。悬镜安全已深耕行业赛道近十年，作为2023数字供应链安全大会（DSS 2023）Digital Supply-chain Security 唯一主办单位，将联合学术界、产业界等权威组织和国际知名开源社区，以人为本，科技驱动，与业界同仁共襄盛会，共话数字经济创新发展新篇章。

**时代使命 护航数字中国建设**

数字时代，数字化转型已成为全球发展的命脉。在我国，数字中国建设是数字时代推进中国式现代化的重要引擎，是构筑国家竞争新优势的有力支撑。加快数字中国建设，对全面建设社会主义现代化国家、全面推进中华民族伟大复兴具有重要意义和深远影响。当下，几乎所有数字产品和服务背后都有供应链，数字供应链促进了人们在日常生活中使用和依赖的大多数基本的社会服务。因此，数字供应链对我们最基本和最重要的社会功能和运作，对安全建设数字中国而言至关重要，其本身的安全性和稳定性需要得到保障。

DSS 2023大会作为业界影响力最大的聚焦数字供应链安全的大会，邀请来自第三方研究机构、开源机构、产业生态侧以及金融、泛互联网、车联网、通信等行业的专家智囊、意见领袖，分享他们在数字供应链安全领域的研究与实践成果。

在大会现场，悬镜安全将联合权威研究机构、高校发布全球首个《数字供应链安全白皮书》，对数字供应链安全定义、发展趋势、面临风险、最佳治理实践以及发展前景等进行全面深入地洞察剖析和图文并茂地呈现。

站在百年未有之大变局的历史关口，DSS 2023大会放眼世界发展局势，立足国家战略需要，旨在助力打造一个汇集“国家、行业、机构、企业“等综合力量且“同向、同心”的数字供应链安全保障生态体系，为数字中国建设筑牢安全屏障。

**齐心聚力 推动开源生态发展**

数字时代，凭借开放、自由、共享等特性，开源已经成为全球科技创新式发展、产业数字化升级的核心引擎，在加速数字化应用迭代升级、促进IT产用协同创新、推动IT产业生态完善上发挥着主导作用，是企业进行数字化转型的重要选择。国家高度重视开源生态发展，持续将开源写入国家规划，上升到战略高度，突出强调支持具有自主核心技术的开源社区、开源平台、开源项目发展，推动创新资源共建共享，促进创新模式开放化。

但与此同时，数字供应链开源化导致供应链各个环节都不可避免受到开源问题的影响。尤其是开源组件中存在的众多安全问题，直接导致数字供应链的安全隐患大大增加，安全形式更加严峻。

在此背景下，DSS 2023大会以“开源的力量”为主题，围绕悬镜安全“用开源的方式做开源风险治理”理念，联合OpenSCA开源社区、XRASP代码疫苗社区以及国内外开源组织，共同深化开源安全文化理念，分享开源数字供应链风险管理方面的技术研究和应用实践成果，探索一站式解决方案，助力保障开源社区的安全性和稳定性，推动开源生态健康发展。

**关于“悬镜安全”**

悬镜安全，起源于北京大学网络安全技术研究团队“XMIRROR”，创始人子芽。作为DevSecOps敏捷安全领导者，始终专注于以“代码疫苗”技术为内核，凭借原创专利级“全流程数字供应链安全赋能平台+敏捷安全工具链”的第三代DevSecOps智适应威胁管理体系，创新赋能金融、车联网、泛互联网、智能制造、通信及能源等行业用户，构筑起适应自身业务弹性发展、面向敏捷业务交付并引领未来架构演进的共生积极防御体系，持续守护中国数字供应链安全。了解更多信息请访问悬镜安全官网：www.xmirror.cn
