# 【工具】情报分析师必备工具之：虚拟机

原创

丁爸

丁爸 情报分析师的工具箱

虚拟机（Virtual Machine）是通过软件模拟的完整计算机系统，是运行在一个完全隔离环境中的计算机系统，通俗来讲就是虚拟出来的电脑，它和真实的电脑几乎一模一样，但虚拟机的硬盘是在一个文件中虚拟处理的，所以可以随意修改虚拟机的设置，这样就可以有效规避基于浏览器插件或电脑指纹的用户追踪。因此，虚拟机也是情报分析师必备工具之一。

**1、VirtualBox**

网站网址：https://www.virtualbox.org/

VirtualBox是由德国Innotek公司开发，由Sun Microsystems推出的一款免费开源的虚拟机软件。现由甲骨文公司进行开发，是甲骨文公司xVM虚拟化平台技术的一部分。它可以在虚拟机里安装各个系统，包括Solaris、Windows、DOS、Linux、OS/2 Warp、BSD等。VirtualBox支持跨平台运行，可以在Windows、Mac、Linux系统上使用。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QjwWINH6f9JuQnRyibt8hJCJfBuBk8MU0XJawnnWvoC6d3DJsceGVibJg/640?wx_fmt=png&from=appmsg)

**2. VMware Workstation**

网站网址：https://www.vmware.com/

VMware Workstation是一款功能强大的电脑桌面虚拟计算机软件，提供用户可在单一的桌面上同时运行不同的操作系统。它适用于进行开发、测试、部署新的应用程序。VMware Workstation能够在一部实体机器上模拟完整的网络环境，具有灵活的网络配置和先进的技术支持，具有Windows、Linux版本。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QgPzE5R6kjRGibeS3QB8gjFCOw2xN65wlcWsQdVJibLUQwIV14eUia7Y1g/640?wx_fmt=png&from=appmsg)

**3、ubuntu**

网站网址：https://ubuntu.com/

Ubuntu是基于Debian，以桌面应用为主的Linux发行版。Ubuntu有三个正式版本，包括桌面版、服务器版及用于物联网设备和机器人的Core版。前述三个版本既能安装于实体电脑，也能安装于虚拟电脑。从17.10版本开始，Ubuntu以GNOME为默认桌面环境。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QagJLK1icicFroGBPxeTI5icYlia2Og9qSffpHDM6GNvibWzbzkHoLXhgicAw/640?wx_fmt=png&from=appmsg)

**4. Java虚拟机（JVM）**

网站网址：https://java-virtual-machine.net/download.html

Java虚拟机（JVM）是一个虚构出来的计算机，是通过在实际的计算机上仿真模拟各种计算机功能模拟来实现的。它是Java语言底层实现的基础，使得Java语言能够在不同平台上运行而不需要重新编译。Java虚拟机以堆栈结构机器来实现。最早由Sun微系统所研发并实现第一个实现版本，是Java平台的一部分，能够执行以Java语言写作的软件程序。Java虚拟机有自己完善的硬件架构，如处理器、堆栈、寄存器等，还具有相应的指令系统。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QVKaeucHF9iaPk7w1bV77cib0upH3bXu14Q4Wfo9AfnmfibDR94qvtWQmg/640?wx_fmt=png&from=appmsg)

**5、基于内核的虚拟机 (KVM)**

网站网址：https://linux-kvm.org/

基于内核的虚拟机(Kernel-based Virtual Machine)是一种用于Linux内核中的虚拟化基础设施，可将Linux内核转化为一个虚拟机监视器。KVM于2007年2月5日被并入Linux 2.6.20核心中。KVM需要支持硬件虚拟化拓展特性的处理器。KVM起初支持x86平台处理器并随后被移植到了S/390、PowerPC、和IA-64平台上。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QO5mokueF6A7TnlqyK1kuySHdZ2LmnMzjGa0iah9ST4MgOKlPat96rUw/640?wx_fmt=png&from=appmsg)

**6、Microsoft
Hyper-V**

网站网址：https://learn.microsoft.com/en-us/virtualization/hyper-v-on-windows/

Microsoft Hyper-V，Windows Server和Windows 10专业版中提供的虚拟化功能，适合企业环境，支持高效的虚拟机管理。Hyper-V，代号Viridian，旧称Windows Server Virtualization，是Microsoft的本地虚拟机管理程序，它可以在运行x86-64位的Windows上创建虚拟机。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QshEekYG8HDYQYbEBakS85ticK7mROticCTaWnJCpiaW4s1DwcYQXgxHaQ/640?wx_fmt=png&from=appmsg)

**7、QEMU**

网站网址：https://www.qemu.org/

QEMU是一款免费开源模拟器，由法布里斯·贝拉等人编写。其与Bochs，PearPC类似，但拥有高速、跨平台的特性。QEMU是一个托管的虚拟机，它使用动态二进制转换技术来模拟处理器，并且提供多种硬件和外设模型，这使它能够运行多种未修改的客户机操作系统，能与KVM配合以接近本地速度运行虚拟机，支持多种硬件架构的虚拟化。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QGEDicOMId81zoeXto68ibh1y24prKcZtcclGVbFjQmNZfpPClQjnI6PA/640?wx_fmt=png&from=appmsg)

**8、Parallels
Desktop**

网站网址：https://www.parallels.cn/

Parallels Desktop for Mac是由Parallels推出的一款为苹果电脑Mac系统提供硬件虚拟化的软件。用户可以通过Parallels
Desktop for Mac在苹果电脑上安装Windows、Linux发行版、FreeBSD、MS-DOS、EComStation、OS/2、Solaris等系统。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QCDrfSibZOqxXROU3MEtNkDFdJIp5IbTGLgkoBYhVVXzficic767Hibxhjw/640?wx_fmt=png&from=appmsg)

**9、Citrix
Hypervisor**

网站网址：https://www.citrix.com/

Citrix Hypervisor (以前称为XenServer)，企业级虚拟化解决方案，适合数据中心和云计算环境。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QexREaxicU5ibj6Fkac3WXrsq5CzI9JicFH8C68NHbvMqEweXibTC4qyIHQ/640?wx_fmt=png&from=appmsg)

**10、Proxmox
VE**

网站网址：https://www.proxmox.com/

Proxmox Virtual Environment，是一个开源的服务器虚拟化环境Linux发行版。Proxmox VE基于Debian，使用基于Ubuntu的定制内核，包含安装程序、网页控制台和命令行工具，并且向第三方工具提供了REST API，在Affero通用公共许可证第三版下发行。结合了KVM和LXC容器，适合于虚拟机和容器的管理。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0Qtib7RiaEoRMfRNVAiar0LDESuk8q1BibjGCJGeKgia3EEXt0W9HRBjyWyvA/640?wx_fmt=png&from=appmsg)

**11、Docker**

网站网址：https://www.docker.com/

Docker是一个开放源代码的开放平台软件，用于开发应用、交付应用和运行应用。Docker允许用户将基础设施中的应用单独分割出来，形成更小的颗粒，从而提高交付软件的速度。Docker容器与虚拟机类似，但二者在原理上不同。容器是将操作系统层虚拟化，虚拟机则是虚拟化硬件。虽然严格来说Docker是一个容器化平台而非传统虚拟机软件，但它允许开发者在独立的容器中打包和运行应用程序，具有轻量级、快速部署的特点。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QCN5ic65PL0OnyGydiax4N7JjyqXwpSfCUrPvgvia3jrrlXa5iaf7CTbDSw/640?wx_fmt=png&from=appmsg)

**12、linuxmint**

网站网址：https://linuxmint.com/

Linux Mint是一种基于Ubuntu的Linux发行版，由Linux Mint Team团队于2006年开始发行。Linux Mint的宗旨是提供一个免费开源、现代、优雅、功能强大却也易于使用的操作系统。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0QrWy0jslhmLtdV3cz5Z0BB7aDaMkJbqsBia195ePQpjptibr0vnLTq9Hg/640?wx_fmt=png&from=appmsg)

**13、Kali
virtualization**

网站网址：https://www.kali.org/docs/virtualization/

Kali Linux是一个专为数字取证和渗透测试而设计的Linux发行版。该软件基于Debian Testing分支。Kali 使用的大多数软件包都是从Debian存储库导入的。

Kali Linux拥有大约600个渗透测试程序，包括Armitage（图形化网络攻击管理工具）、Nmap（端口扫描器）、Wireshark（数据包分析器）、metasploit（渗透测试框架）、John the Ripper（密码破解器）、sqlmap（自动SQL注入和数据库接管工具）、Aircrack-ng（无线局域网渗透测试软件套件）、Burp suite 和OWASP ZAP Web 应用程序安全扫描程序等。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5wasmN9RMJsW4tBIMeYlj0Q7fEtAoHptX8u2leagx9IoxZiccMOZSEtq3INMUp4mrdHX2qVAItyKQg/640?wx_fmt=png&from=appmsg)
