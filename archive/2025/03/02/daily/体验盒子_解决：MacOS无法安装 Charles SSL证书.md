## 问题：**无法安装**Charles SSL证书，一闪而过

> **You may need to configure your browser or application to trust the Charles Root Certificate. See SSL Proxying in the Help menu.**

在MacOS中，不管是在： “Help” -> “SSL Proxying” -> “Install Charles Root Certificate” 中，还是保存本地双南或拖入钥匙应用安装都是一闪而过。

![解决：MacOS无法安装 Charles SSL证书](https://www.uedbox.com/wp-content/uploads/2025/03/20250301-141904@2x.png)

## 解决：重置证书：Reset Charles Root Certificate

“Help” -> “SSL Proxying” -> “Reset Charles Root Certificate”，重置完后，重启一下Charles，再安装就好了。

![解决：MacOS无法安装 Charles SSL证书](https://www.uedbox.com/wp-content/uploads/2025/03/20250301-142316@2x.png)
