## Forgejo v8.0 释出

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2024年08月01日 18时46分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=78858&appkey=1370085986&title=Forgejo%20v8.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自失眠**

软件开发平台 Forgejo 释出了 v8.0。主要变化包括：由于许可证不兼容移除了两个前端，作为具有自由开源理念的软件项目移除了对私有的 Microsoft SQL Server 支持，改进了稳定性，新的 UI 团队大幅减少了随机的 UI 变化，等等。Forgejo v8.0 不是长期支持版本，关注稳定性的用户可选择上一个版本 Forgejo v7.0，它将一直支持到 2025 年 7 月 16 日，Forgejo v8.0 将支持到 2024 年 10 月 16 日 Forgejo v9.0 发布时。
https://forgejo.org/2024-07-release-v8-0/
