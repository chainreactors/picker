# 【安全圈】发布3年后Windows 11终于超越Windows 10成为最受欢迎的PC游戏操作系统

安全圈

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGylgOvEXHviaXu1fO2nLov9bZ055v7s8F6w1DD1I0bx2h3zaOx0Mibd5CngBwwj2nTeEbupw7xpBsx27Q/640?wx_fmt=png&from=appmsg "微信图片_20230927171534.png")

**关键词**

操作系统

2012 年微软跳过 Windows 9 直接推出 Windows 10 系统，并且声称 Windows 10 是最后一个 Windows 系统，后续将在 Windows 10 的基础上推出年度版本更新。

不过在 2021 年 10 月微软正式推出 Windows 11 系统，这款基于 Windows 10 设计但又进行了大量 UI 更改的新系统褒贬不一，但至少对游戏玩家来说这不是一个好消息：稳定性较差和卡顿。

大多数游戏玩家都不会选择 Windows 11，因为 Windows 11 总有些奇奇怪怪的问题，比如间歇性的卡顿，这种问题对游戏玩家来说是非常不好的，所以在 Steam 游戏平台大多数游戏玩家都使用 Windows 10。

这种情况在 Windows 11 发布 3 年后终于变了，Steam 公布的 8 月份操作系统统计数据显示，Windows 11 超越 Windows 10 成为最受欢迎的 PC 游戏操作系统。![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGyliaDCTA3nG7ibcdGzNjxkibVFvlFquMM2IXH8Ch9tosAuQkEcU9teS3oFwwtRaEibGCQuMYtCogibV6NHA/640?wx_fmt=png&from=appmsg)

在 2024 年 8 月份 Windows 10 系统市场份额 47.09%，环比下滑 3.07%；Windows 11 市场份额 49.17%，环比增长 3.36%；其他操作系统例如 macOS 和 Linux 基本没有多少变化。

当然这也算是在预期之类，就像 Windows 10 发布时仍然有大量用户使用 Windows 7 和 Windows 8，彼时 Windows 10 的稳定性也非常糟糕，在后续的持续更迭中才逐渐稳定。

现在 Windows 11 的情况也是如此，通过 3 年的更迭后 Windows 11 的稳定性也逐渐好转，当然更关键的是新推出的笔记本电脑已经无法预装 Windows 10，这也在快速推进 Windows 11 市场份额的上涨。

到明年 10 月 Windows 10 系统将主流安全支持，到时候可能还会有不少用户逐渐转入 Windows 11 — 即便硬件不支持 Windows 11 也得想办法升级，因为随着时间的推移部分游戏和平台都会陆续停止支持 Windows 10。
