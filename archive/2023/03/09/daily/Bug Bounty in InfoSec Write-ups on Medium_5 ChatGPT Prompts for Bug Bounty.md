# 5 ChatGPT Prompts for Bug Bounty

## JS, XSS, CSRF, and Decoding Made Easy with ChatGPT

[![Mike Takahashi (TakSec)](https://miro.medium.com/v2/resize:fill:64:64/1*oeaksoi-GPEkdXjmR0uMFA.jpeg)](https://taksec.medium.com/?source=post_page---byline--6b7365d61b58---------------------------------------)

Are you a bug bounty hunter, penetration tester, or developer, but looking for ways to augment your abilities with the power of AI? You are in luck because [ChatGPT](https://chat.openai.com/chat) can assist you in the following ways.

### 1. AI-powered Scan for Vulnerabilities in Javascript

Javascript can be complicated and time-consuming to read, but is a gold mine for vulnerabilities. Let ChatGPT solve both for you.

**Prompt:**

> Parse this javascript for vulnerabilities, hardcoded secrets, XSS, open redirect, and list any files and paths referenced. be specific with where the vulnerability is and a PoC for each. Use rich text formatting to make it easier to read:
>
> `<insert js>`

Press enter or click to view image in full size

![]()

If the Javascript is too long for ChatGPT, you can use snippets or [OpenAI Playground](https://platform.openai.com/playground).
