基于 Windows 7/8/8.1 已经结束生命周期，Google Chrome、Microsoft Edge 都已经停止支持这些系统，Firefox 则会在明年 9 月停止支持这些系统。

对谷歌来说自家其他软件也没有继续支持这些系统的必要，所以谷歌云端硬盘 (Google Drive) 客户端将在 8 月份停止支持 Windows 8 和 Windows 8.1 版。

[![谷歌云端硬盘不再支持Windows 8/8.1以及32位版Windows 10系统](https://img.lancdn.com/landian/public/thumb/workspace.png)](https://img.lancdn.com/landian/public/thumb/workspace.png)

谷歌云端硬盘桌面客户端主要提供文件同步功能，在谷歌停止支持旧系统后，继续在旧系统上将无法正常连接，所以这会导致文件同步出现问题，谷歌建议用户升级系统。

另外谷歌云端硬盘还将结束对 32 位版系统的支持，这里主要指的是 32 位版的 Windows 10 系统，Windows 11 只有 64 位版没有 32 位版所以不用担心。

和 Chrome 不同的是，停止支持旧系统后，旧版 Chrome 还是可以在 Windows 7/8/8.1 上使用的，毕竟这只是个浏览器。Google Drive 则需要联网和登录谷歌账号，谷歌直接掐掉了这些通道，因此用户除非升级系统否则就只能使用网页版 Google Drive 上传或下载文件了。
