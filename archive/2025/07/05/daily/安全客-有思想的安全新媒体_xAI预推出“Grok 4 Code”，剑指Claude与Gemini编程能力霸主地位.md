# xAI预推出“Grok 4 Code”，剑指Claude与Gemini编程能力霸主地位

xAI 正在筹备推出全新一代模型 Grok 4，以取代此前的 Grok 3，成为其最新的旗舰模型。

在正式发布前，有测试者在 X 平台上再次发现了多个 Grok 4 相关模型的引用。

其中一个模型名为 “**grok-4-prod-mimic**”，据称在企业级应用场景中表现出色，擅长处理数据提取、代码生成、文本摘要等任务，并具备金融、医疗、法律、科学等多个垂直领域的深度知识。

但“grok-4-prod-mimic”并非 Grok 4 系列的唯一新成员。另一款新模型名为 “**Grok 4 Code**”，根据 xAI 控制台描述，其定位是用户的“**编程伴侣**”，专为开发者提供代码辅助支持而打造。

还有一款最基础、也就是“日常对话”模型，命名为 **“Grok 4”**。据介绍，它旨在提供“在自然语言处理、数学计算和逻辑推理方面无与伦比的性能，是一款真正的多面手通用模型。”

据悉，xAI 计划在**美国独立日（7月4日）之后**发布 Grok 4，但该版本**不包含视觉和图像处理能力**。
