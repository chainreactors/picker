## 微软向 OpenAI 投资数十亿美元

[![商业](https://icon.solidot.org/images/topics/topicbiz.png?123)](/search?tid=54 "商业")
[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Wilson](/~Wilson) (42865)发表于 2023年01月24日 17时12分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73947&appkey=1370085986&title=%E5%BE%AE%E8%BD%AF%E5%90%91%20OpenAI%20%E6%8A%95%E8%B5%84%E6%95%B0%E5%8D%81%E4%BA%BF%E7%BE%8E%E5%85%83 "新浪微博分享")

**来自精灵王之女**

微软和 OpenAI 宣布扩大合作关系，在投资了 10 亿美元之后微软将在未来几年再次向 OpenAI 投资数十亿美元。早些时候有传言称微软有意向 OpenAI 投资百亿美元，最新的声明只是笼统的称为数十亿美元。OpenAI 表示它将会继续使用基于 Azure 的超算训练其模型，而 Azure 将始终是其所有产品和服务的唯一云服务商。OpenAI 称微软的产品和服务将整合其技术，其中包括 GitHub Copilot 和 Microsoft Designer。
https://openai.com/blog/openai-and-microsoft-extend-partnership/
