# 【安全圈】微软正在修复Windows 11更换主板后无法激活的问题 遇到该问题的用户还需稍等

安全圈

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGylgOvEXHviaXu1fO2nLov9bZ055v7s8F6w1DD1I0bx2h3zaOx0Mibd5CngBwwj2nTeEbupw7xpBsx27Q/640?wx_fmt=other&from=appmsg&tp=webp&wxfrom=5&wx_lazy=1&wx_co=1)

**关键词**

Windows 11

12月份微软向测试频道的用户推出 Windows 11 Beta 22635.4580 版，对应的累积更新为 KB5046733，此次更新主要是修复多个已知的错误。

例如部分用户更换主板后系统无法激活，正常请看下更换主板后确实会失去激活，但只需要点击激活设置页面的疑难杂症按钮并选择更换了主板就可以激活，这个错误导致疑难杂症选项也无法修复激活。

不过这个修复程序暂时仅面向测试频道的用户推出，按照更迭节奏以及圣诞假期，新的修复更新最快也要到 2025 年 1 月份才能抵达正式版，如果还需要发布可选更新测试的话，那在正式版里大概要到 2025 年 2 月份的月度例行更新中才能完成修复。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGyljdvYibffqfqPicPMQN0HSqqoVh334fvgH98tojFIgxNvTSAelawQx9ZEMuOKN3yy4dZGHy2ib7wVJvg/640?wx_fmt=png&from=appmsg)上图为示意图，LTSC 版不支持迁移激活

**下面是此次更新的功能变化：**

由于此前测试版中的时间模块短日期存在错误，在新版本中已被禁用即恢复为长日期

微软正在更新资源管理器，默认情况下打开文件夹时将合并到当前已经打开的文件资源管理器窗口，具体请看：微软给 Windows 11 添加新选项允许打开任意文件夹最终都在新选项卡中打开

资源管理器中的 OneDrive 备份图标现在有新选项可以禁用，禁用后地址栏不再闪烁蓝色的备份按钮，具体请看：微软终于允许关闭 Windows 11 资源管理器的备份广告 不用删除OneDrive

**下面是修复的各种错误：**

解决部分低功耗蓝牙设备无法传输蓝牙音频问题，该问题在某些助听器上容易出现

修复微软拼音输入法在窗口切换时会意外将中文输入切换为英文输入

修复基于 internet 打印协议 (IPP) 打印机通过 USB 连接时可能导致系统卡死
