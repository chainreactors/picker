# 博智安全为第十七届全国大学生信息安全竞赛创新实践能力赛--华东北分区选拔赛保驾护航

嘶吼专业版

![](https://mmbiz.qpic.cn/mmbiz_gif/wpkib3J60o297rwgIksvLibPOwR24tqI8dGRUah80YoBLjTBJgws2n0ibdvfvv3CCm0MIOHTAgKicmOB4UHUJ1hH5g/640?wx_fmt=gif)

6月24日，由教育部高等学校网络空间安全专业教学指导委员会主办，南京信息工程大学承办，**博智安全科技股份有限公司（以下简称“博智安全”）提供平台技术支撑**，奇安信科技集团股份有限公司协办的第十七届全国大学生信息安全竞赛创新实践能力赛--华东北分区选拔赛（以下简称“分区赛”）于南京信息工程大学圆满结束。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQaGLJdib5iaDqmaEZcg54Hanuzbo93lHIZ2rqxFqVTam4GwJXIrh8m8ww/640?wx_fmt=jpeg&from=appmsg)

分区赛旨在积极响应国家网络空间安全人才需求，加快攻防兼备创新人才培养步伐，实现以赛促学、以赛促教、以赛促用，从而推动网络空间安全人才培养和产学研用的生态体系。

分区赛采用 AWD PLUS赛制，综合考核参赛团队攻击、防御技术能力、即时策略的攻防兼备比赛模式。每个参赛队互为攻击方和防守方，充分体现比赛的实战性、实时性和对抗性，对参赛队的渗透能力和防护能力进行综合全面的考量。

此外，分区赛围绕Web 安全、二进制漏洞挖掘等网络安全知识技能，重点考察参赛选手SQL 注入、命令执行、反序列化等漏洞，Php、Asp、Java、Python、Go 等语言代码审计问题，渗透测试，Linux 的二进制漏洞挖掘与利用技术等相关能力的掌握情况。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQv1UqEAOolWuhmOAcZsJBCibvmONRa85lFE5qLWBtLSpOxDk9HcXbpZA/640?wx_fmt=jpeg&from=appmsg)

经过6.5小时的激烈角逐，分区赛圆满收官！

# ![](https://mmbiz.qpic.cn/sz_mmbiz_png/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQtxphkIPL09WAvVd6YPXJgfqR7IyTnU8xiaWuWCkdiaJXMC1ZnyrI6kdw/640?wx_fmt=png&from=appmsg)闭幕式

分区赛闭幕式邀请到江苏省教育厅“双减”办公室主任、江苏省教育信息化中心主任孙其华，南京信息工程大学党委副书记闵锦忠致辞，南京市公安局二级巡视员 、原副局长蒋平担任大赛裁判长作总结发言，南京信息工程大学沃特福德学院院长王保卫担任闭幕式主持人。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQnSnNs5uT5ZwLYrMOGbR15gVXc3PichJCVSUTXmah17R6bmF9yiagTUSA/640?wx_fmt=jpeg&from=appmsg)

江苏省教育厅“双减”办公室主任、江苏省教育信息化中心主任  孙其华

孙其华指出，目前，我国网络安全人才培养取得了重要进展，但与高质量推进网络强国、数字中国建设目标之间还存在一定差距，这对新时代网络安全人才培养提出了新的更高要求。近年来，省教育厅认真贯彻落实中央网信办等六部委《关于加强网络安全学科建设和人才培养的意见》，进一步推动我省高校加强网络安全人才培养，促进网络安全教育深入开展，积极引导高校增设网络安全相关学科，在网络安全人才培养上取得明显成效。

本次赛事的开展不仅给广大参赛学生提供了网络安全实战平台，同时也搭建了发现人才和人才交流的平台。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQibTnAtRODgvW9YLaPnT60vjoVic85mq9icYx0x7ic2ib2l9Te9aQn4Z6QcQ/640?wx_fmt=jpeg&from=appmsg)

南京信息工程大学党委副书记  闵锦忠

闵锦忠指出，全国大学生信息安全竞赛是国家A类赛事，是我国级别最高、覆盖范围最广、影响力最大的大学生信息安全类竞赛。大赛的举办对于加快攻防兼备创新人才培养步伐，实现以赛促学、以赛促教、以赛促用具有重要意义。

南京信息工程大学高度重视网络空间安全学科的建设和发展，积极参与组织承办本次比赛。在学校信息工科跨越式发展的进程中，计算机学院（网络空间安全学院）是重要的中坚力量。欢迎各位本科生、硕士研究生同学积极报考我校的计算机科学与技术、信息安全等专业的硕士和博士生研究生。欢迎博士生毕业后积极入职我校，为南信大的发展添砖加瓦。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQZaFl06CbQIIbqKS6Bl2PX4gLf3WUyZGVgSVekibKicrD5sAwEpeNGakA/640?wx_fmt=jpeg&from=appmsg)

南京市公安局二级巡视员、原副局长  蒋平

蒋平表示，在各有关部门的密切配合、通力合作下，分区赛取得了丰硕的成果：一是竞赛涉及面广，大赛的参与性广泛，汇集江苏省、安徽省、山东省三大省份，初赛筛选后的**80支优秀的竞赛队伍，参赛人数高达320余人**。二是大赛筹备周密细致，组织工作严谨有序。大赛制定了技术实施方案、裁判规则等技术文件，起草了会议手册，并严格按照手册实施，做到了大赛有方案、落实有措施、安全有保障，确保了本次大赛的公平公正。三是大赛促进人才成长，本次大赛最重要的成果是选拔了一批技艺超群的信息安全方向的高校队伍，一大批青年人才脱颖而出，这是我们倍感欣慰的。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQYDgtjKuDvoJfOdqHELXUuM1TxHDFnqxg85fQfLPEkeW4ZoVsfcRdYA/640?wx_fmt=jpeg&from=appmsg)

南京信息工程大学沃特福德学院院长  王保卫

江苏省教育厅“双减”办公室主任、江苏省教育信息化中心主任孙其华为获一等奖的战队颁奖，南京市公安局二级巡视员原副局长蒋平为获二等奖的战队颁奖，南京信息工程大学计算机学院、网络空间安全学院副院长、公共计算机教学部主任庄伟，博智安全科技股份有限公司副总裁郑轶，奇安信集团江苏市场总监周子佳为获三等奖战队颁奖。

# ![](https://mmbiz.qpic.cn/sz_mmbiz_png/wpkib3J60o29M8ZvdyGaaBhKIZxXEV3wQtxphkIPL09WAvVd6YPXJgfqR7IyTnU8xiaWuWCkdiaJXMC1ZnyrI6kdw/640?wx_fmt=png&from=appmsg)博智安全持续为建设网络强国提供人才保障

近年来，博智安全成功支撑上百场网络安全赛事，覆盖政府、能源、通信、电力、交通、教育、国防等行业，形成“以赛促学、以赛促用、以赛促改、以赛促建”的联动效果，选拔出一批又一批技术能力突出，攻防兼备的新型尖端人才。

博智安全在网安人才培养领域的经验和能力，依托于自研的**博智孪生仿真靶场**——以孪生仿真技术为基础、威胁模拟生成为手段、攻防推演验证为目标的一体化综合演训平台，能够构建工业自动化网络、电信、电力、轨道交通、IOT、卫星遥感、智能制造、传统网络等多种行业的仿真场景，开展知识训练、技能训练、比武竞赛、实战演训、渗透测试、安全评估、技术验证等活动，实现学、练、赛、训、评全流程的统一管理，全面提高人员及机构的网络安全意识、防护水平和实战能力，保障国家关基重保行业的安全运行。

网络安全就是国家安全，培养高素质网络安全人才是根基所在。未来，博智安全将基于“博智孪生仿真靶场”，充分发挥自身技术优势，助力政府、高校、产业界培养攻防兼备的网络安全人才，为“网络强国”战略持续供能。
