# Bitdefender收购网状，增强电子邮件安全平台

Bitdefender 已达成收购全球电子邮件安全公司 Mesh 的协议，计划将 Mesh 的电子邮件安全技术集成到 Bitdefender 的扩展检测和响应平台以及托管检测和响应服务中。

**电子邮件威胁**

与电子邮件相关的网络安全风险在数字攻击中占很大比例，包括勒索软件、网络钓鱼和商业电子邮件泄露 （BEC）。2024 年的威胁形势是电子邮件的持续利用，FBI 报告称，仅 BEC 诈骗就造成了约 28 亿美元的损失。2024 年 Bitdefender 网络安全评估报告基于对全球 1,200 名网络安全专业人士的调查，将网络钓鱼和社会工程列为组织面临的最普遍的威胁之一。

该交易受监管和惯例成交条件的约束。此次集成将使 Mesh 的电子邮件安全功能整合到 Bitdefender GravityZone 中，这是一个统一的安全、合规和风险分析平台。Mesh 实施双层安全方法，通过其安全电子邮件网关将基于边界的保护与由基于 API 的部署模型管理的邮箱级防御相结合。此次集成有望扩大威胁可见性并增强 Bitdefender 运营的全球威胁情报网络。

**平台改进**

Bitdefender 的全球网络由超过 41,000 个频道和托管服务提供商 （MSP） 合作伙伴组成。Mesh 功能的添加有望实现针对 MSP 优化的集中式平台，支持高效的多租户电子邮件安全管理。主要功能包括 24×7 全天候保护、减少运营开销、自动化策略实施、实时威胁洞察以及与现有工作流程的兼容性，从而为广泛的客户群提供可扩展的安全解决方案。

> Bitdefender 商业解决方案集团总裁兼总经理 Andrei Florescu 表示：“我们很高兴地宣布我们打算收购 Mesh，这是一项战略举措，将补充我们的 GravityZone XDR 平台，并为我们的 MDR 服务提供支持，以帮助企业在不断发展时应对电子邮件传播的威胁。Mesh 从经验丰富的团队中带来了领先的创新，他们与我们一样致力于有效的实际安全。我们将共同进一步加强我们的能力，为我们的全球客户群提供成熟且值得信赖的电子邮件保护。

Mesh 成立于 2020 年，得到了投资者 Elkstone 和 Enterprise Ireland 的支持。该公司已在全球数百家 MSP 合作伙伴和数千家最终客户中建立起来，以有效检测、以 MSP 为中心的架构和作简单的服务设计而闻名。Mesh 的解决方案专注于应对基于电子邮件的威胁带来的独特挑战。

> “这是更大事情的开始，”Mesh 首席执行官兼联合创始人 Brian Byrne 说。“我们一直专注于构建实用、强大的电子邮件安全解决方案 – Bitdefender 也有同样的理念。我们很高兴能联手为更多组织提供更强大的保护。

**市场背景**

电子邮件仍然是被利用最广泛的网络攻击媒介。即将将 Mesh 的架构与 Bitdefender 的平台集成，旨在解决企业客户日益增长的担忧，提供针对检测效果并符合 MSP 要求的新产品。

没有披露与此次交易有关的财务条款。
