еңЁдёҖж¬ЎеӨҚжқӮзҡ„дҫӣеә”й“ҫж”»еҮ»дёӯпјҢжҒ¶ж„ҸиЎҢдёәиҖ…жҪңе…ҘдәҶе№ҝжіӣдҪҝз”Ёзҡ„ JavaScript еә“ lottie-playerпјҢжіЁе…ҘдәҶеңЁеҗҲжі•зҪ‘з«ҷдёҠжү“ејҖ Web3 й’ұеҢ…иҝһжҺҘжҸҗзӨәзҡ„д»Јз ҒгҖӮеҰӮжһңз”ЁжҲ·жҺҘеҸ—иҜҘжҸҗзӨәпјҢж”»еҮ»иҖ…е°ұеҸҜд»ҘзӣҙжҺҘд»ҺеҠ еҜҶиҙ§еёҒй’ұеҢ…дёӯжҠҪиө°иө„дә§гҖӮlottie-player зҡ„дёӢиҪҪйҮҸи¶…иҝҮ 400 дёҮж¬ЎпјҢеңЁдј—еӨҡй«ҳжөҒйҮҸзҪ‘з«ҷзҡ„еҠЁз”»еөҢе…Ҙдёӯжү®жј”зқҖйҮҚиҰҒи§’иүІпјҢжӯӨж¬ЎдәӢд»¶зҡ„еҪұе“ҚеҮёжҳҫдәҶејҖжәҗеә“еӣәжңүзҡ„е®үе…ЁйЈҺйҷ©гҖӮ

дёҖеҗҚз”ЁжҲ·йҰ–е…ҲеңЁ GitHub дёҠжҠҘе‘ҠдәҶиҝҷдёҖжјҸжҙһпјҢд»–еңЁзҪ‘з«ҷдёҠйӣҶжҲҗ lottie-player ж—¶еҸ‘зҺ°дәҶж„ҸеӨ–зҡ„ Web3 й’ұеҢ…жҸҗзӨәгҖӮиҝӣдёҖжӯҘи°ғжҹҘеҸ‘зҺ°пјҢеҸ‘еёғдәҺж јжһ—е°јжІ»ж ҮеҮҶж—¶й—ҙ 2024 е№ҙ 10 жңҲ 30 ж—ҘжҷҡдёҠ 8:12 иҮі 9:57 д№Ӣй—ҙзҡ„ lottie-player 2.0.5гҖҒ2.0.6 е’Ң 2.0.7 зүҲжң¬е·ІйҒӯеҲ°е…ҘдҫөгҖӮж”»еҮ»иҖ…еңЁжңӘз»ҸжҺҲжқғи®ҝй—®дёҺиҜҘеә“з»ҙжҠӨиҖ…д№ӢдёҖ Aidosmf зӣёе…іиҒ”зҡ„д»ӨзүҢеҗҺпјҢжҲҗеҠҹжіЁе…ҘдәҶжҒ¶ж„Ҹд»Јз ҒгҖӮиҝҷж ·пјҢд»–д»¬е°ұеҸҜд»ҘйҖҡиҝҮ npm е’Ңдё»иҰҒ CDN дј ж’ӯеҸ—ж”»еҮ»зҡ„зүҲжң¬пјҢеҪұе“ҚйӮЈдәӣеңЁжІЎжңүзүҲжң¬й”Ғзҡ„жғ…еҶөдёӢиҺ·еҸ–жңҖж–°зүҲжң¬зҡ„зҪ‘з«ҷгҖӮ

жіЁе…Ҙзҡ„д»Јз ҒдјҡжҳҫзӨәдёҖдёӘиҷҡеҒҮзҡ„ Web3 й’ұеҢ…иҝһжҺҘиҜ·жұӮпјҢиҜ•еӣҫеҲ©з”ЁеҸ—еҪұе“ҚзҪ‘з«ҷзҡ„и®ҝй—®иҖ…пјҢе°Өе…¶жҳҜеҠ еҜҶиҙ§еёҒйўҶеҹҹзҡ„и®ҝй—®иҖ…гҖӮеҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢеҺ»дёӯеҝғеҢ–йҮ‘иһҚпјҲDeFiпјүе№іеҸ° 1inch иҜҒе®һпјҢдёҖдәӣз”ЁжҲ·еҸҜиғҪйҒҮеҲ°дәҶй’“йұјжҸҗзӨәгҖӮжҚ® Scam Sniffer жҠҘйҒ“пјҢдёҖеҗҚеҸ—е®іиҖ…е·Із»ҸжҠҘе‘Ҡз§°пјҢз”ұдәҺжҒ¶ж„Ҹй’ұеҢ…иҝһжҺҘдјҒеӣҫпјҢд»–жҚҹеӨұдәҶ 10 дёӘжҜ”зү№еёҒпјҲд»·еҖјзәҰ 723,436 зҫҺе…ғпјүгҖӮ

еҸ‘зҺ°ж”»еҮ»еҗҺпјҢlottie-player еӣўйҳҹиҝ…йҖҹеҸ‘еёғдәҶдёҖдёӘе№ІеҮҖзҡ„ 2.0.8 зүҲжң¬пјҢејҖеҸ‘иҖ…еҸҜд»Ҙз”Ёе®ғжқҘжӣҝжҚўиў«ж”»еҮ»зҡ„ж–Үд»¶гҖӮжӯӨеӨ–пјҢ2.0.5 иҮі 2.0.7 зүҲжң¬е·Ід»Һ npm е’Ң CDN жҸҗдҫӣе•ҶпјҲеҰӮ unpkg е’Ң jsdelivrпјүдёӯеҲ йҷӨпјҢд»ҘйҒҸеҲ¶жјҸжҙһе№¶йҷҗеҲ¶жҡҙйңІгҖӮ

дҪҶжҳҜпјҢйҷӨйқһжӣҙж–°еҲ°е®үе…ЁзүҲжң¬пјҲ2.0.4 жҲ– 2.0.8пјүпјҢеҗҰеҲҷд»»дҪ•зӣҙжҺҘеј•з”ЁеҸ—еҪұе“ҚзүҲжң¬зҡ„зҪ‘з«ҷд»Қз„¶еӯҳеңЁжјҸжҙһгҖӮ

жӯӨж¬Ўж”»еҮ»еҸ‘з”ҹеҗҺпјҢе®үе…Ёеӣўйҳҹеә”йҮҮеҸ–д»ҘдёӢжҺӘж–Ҫпјҡ

* **е®Ўи®Ўдҫқиө–е…ізі»пјҡ** иҝӣиЎҢеҪ»еә•е®Ўи®ЎпјҢжүҫеҮәдҫқиө–дәҺеҸ—ж”»еҮ»зҡ„ lottie-player зүҲжң¬зҡ„д»»дҪ•йЎ№зӣ®гҖӮйүҙдәҺиҜҘеә“зҡ„е№ҝжіӣдҪҝз”ЁпјҢиҝҷдёҖжӯҘйӘӨеҜ№дәҺдҝқжҠӨзҪ‘з»ңиө„дә§е’Ңз”ЁжҲ·дҝЎжҒҜиҮіе…ійҮҚиҰҒгҖӮ
* **жӣҙж–°жҲ–иҝҳеҺҹпјҡ** з«ӢеҚіжӣҙж–°еҲ°е®үе…ЁзүҲжң¬ 2.0.8 жҲ–жҒўеӨҚеҲ°еҸ—еЁҒиғҒеүҚзҡ„зүҲжң¬ 2.0.4гҖӮйј“еҠұд»ҚеңЁдҪҝз”ЁеҸ—еҪұе“ҚзүҲжң¬зҡ„зҪ‘з«ҷз«ӢеҚійҮҮеҸ–иЎҢеҠЁпјҢйҳІжӯўжҪңеңЁзҡ„жјҸжҙһгҖӮ
* **е®һж–ҪзүҲжң¬й”Ғе®ҡпјҡ** йҒҝе…ҚеңЁжңӘжҢҮе®ҡе®үе…Ёе’Ңз»ҸиҝҮйӘҢиҜҒзҡ„зүҲжң¬зҡ„жғ…еҶөдёӢдҪҝз”ЁжңҖж–°еҸ‘еёғзҡ„зүҲжң¬гҖӮе°Ҷдҫқиө–е…ізі»еӣәе®ҡеҲ°з»ҸиҝҮйӘҢиҜҒзҡ„зү№е®ҡзүҲжң¬пјҢеҸҜд»ҘйҷҚдҪҺиҮӘеҠЁжҡҙйңІдәҺжңӘжқҘдҫӣеә”й“ҫж”»еҮ»зҡ„йЈҺйҷ©гҖӮ
* **зӣ‘жҺ§ејӮеёёдәӨжҳ“пјҡ** йүҙдәҺиҮіе°‘жңүдёҖиө·йҮҚеӨ§жҚҹеӨұзҡ„жҠҘе‘ҠпјҢдјҒдёҡеә”з§ҜжһҒзӣ‘жҺ§з”ЁжҲ·дәӨжҳ“дёӯзҡ„еҸҜз–‘жҙ»еҠЁпјҢе°Өе…¶жҳҜ Web3 й’ұеҢ…иҝһжҺҘжҸҗзӨәе’ҢзӯҫеҗҚгҖӮ
