# 【安全圈】OpenAI 被币圈黑客盗号，15 个月内第四起

**关键词**

OpenAI

美国东部时间周一晚上，OpenAI 在社交平台 X 上的一个官方账号被黑客入侵，用于宣传加密货币诈骗活动。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/aBHpjnrGyljvHleU06H1K5XEeEtyFNIRvVZm3yTSfiblT7qkaDpkewCvpWN1U1LWKdddQDNkZBa3R8e2YJXeib0g/640?wx_fmt=jpeg&from=appmsg)

美国东部时间周一晚上 6:24，拥有近 5.4 万名关注者的 OpenAI 新闻资讯账号 ( @OpenAINewsroom ) 发布了一条现已删除的帖子，推广一种名为 "$OPENAI" 的代币。

帖子称：" 我们很高兴宣布推出 $OPENAI，该代币将桥接人工智能和区块链技术。所有 OpenAI 用户均有资格认领初始供应的一部分 $OPENAI 代币。持有 $OPENAI 将使您能够访问我们所有未来的测试计划。" 帖子还包含了一个指向虚假 OpenAI 网站的链接，该网站的网址并openai.com。

访问该网站时，发现其中包含一个认领 $OPENAI 加密货币的部分。点击相关按钮后，网站要求连接钱包。

截至发稿，OpenAI 和社交平台 X 尚未对此进行回应。OpenAI 新闻资讯账号也未发布任何解释该事件经过的帖子。该账号于本月初上线。
