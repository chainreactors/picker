安全内参5月15日消息，澳大利亚联邦政府日前公布了2024-2025财年预算，其中有多项投资用于改进IT，投资项目包括即将进行的人口普查的安全数据存储、改善监管机构的数据能力和网络安全等。

从2024-2025财年开始，澳大利亚政府计划三年内投入1.968亿澳元（约合人民币9.44亿元），以支持**2026年人口普查**。具体工作包括确保人口普查数据的安全收集和存储，组织宣传活动以提高普查参与度，并通过myGov网站使人口普查更加便捷。

为提升**澳大利亚审慎监管局（APRA）**和**澳大利亚证券与投资委员会（ASIC）**的数据能力和网络安全，继续稳定商业登记，推动澳大利亚税务局（ATO）、工业、科学与资源部、国库部和财政部的遗留系统的现代化，从2024-2025财年开始，澳大利亚政府将在四年内投资2.064亿澳元（约合人民币9.9亿元），然后每年持续投资720万澳元。相关成本将部分由APRA和ASIC通过行业征费来支付。

为了检测、预防并减缓针对税务和养老金系统的欺诈行为，澳大利亚政府承诺投入7870万澳元用于升级信息和通信技术，使**澳大利亚税务局**能够实时识别和阻止可疑活动。

从本财年开始，澳大利亚政府将在五年内提供3990万澳元，用于制定政策、提升能力，支持以安全和负责任的方式吸收、使用AI技术。其中，自2024-2025财年起，三年内将投入260万澳元，以**应对和减轻与AI相关的国家安全风险**。

联邦政府承诺，自2024-2025财年起，四年内向**议会部门**提供1280万澳元，并在此后每年持续投入230万澳元，以加强对商业生产力和网络安全的IT支持。

**澳大利亚民政服务部（Services Australia）**将获得以下投资：

* 三年内投入18亿澳元，以改善网络安全环境，并增加前线和服务交付人员以管理索赔、持续应对紧急情况。
* 自2024-2025财年起，两年内投入2350万澳元，继续推动反诈融合小组（Fraud Fusion Taskforce）展开欺诈调查和应对活动 。
* 两年内投入4600万澳元，继续运营和改进澳大利亚政府数字身份系统（Digital ID System）进行交易所需的身份交换功能，并执行数字身份系统管理员的功能。

**澳大利亚政府正在扩大数字身份系统**，为人们提供一种安全、方便、自愿和包容的方式在线验证身份。一旦拥有了数字身份，人们将不再需要向企业或政府提供身份证件副本来确认自己的身份，企业也不再需要存储这些信息，从而减少数据泄露和网络攻击的风险和影响。超过1200万澳大利亚人已经拥有了myGovID。

Secure Code Warrior首席执行官Pieter Danhieux在一份声明中说：“如果数字身份能够执行得当，将是一项有效的措施，可以减轻在线欺诈和身份盗窃的影响。然而，这是澳大利亚持续网络安全战略中被动应对性内容。我担心我们对于积极的、预防性的安全措施投入的资金过少。”

自2024-2025财年起，**澳大利亚卫生和老龄化部门**将在三年内投入800万澳元，以支持澳大利亚体育基金会完成技术升级，包括增强筹款平台的网络安全。

此外，为了让**国家残障保险计划（NDIS）**重新回到正轨，澳大利亚政府承诺，自2024-2025财年起，四年内将投入1.607亿澳元，并在之后每年持续投入2460万澳元，用于升级NDIS质量和保障委员会的IT系统，以更好地保护NDIS参与者的安全，减少NDIS提供者的监管负担，并改善网络安全。此外，从上一财年开始，两年内将投入8390万澳元用于提升国家残障保险局（NDIA）的欺诈检测信息技术系统，以进一步保障NDIS的完整性。

澳大利亚外交部长Penny Wong表示，这份预算“将在前瞻性预测期（三年）投资超过2.278亿澳元，用于**升级澳大利亚的通信基础设施和海外财产**，从而来确保我们关键的外交网络安全，包括加强我们对网络威胁的弹性。”

澳大利亚政府还将从2024-2025财年起，四年内提供420万澳元，并在此后每年持续投入90万澳元，用于**建立一个由内政部领导的技术外国干预小组**。该小组将与澳大利亚的技术部门合作，保护敏感和专有信息免受间谍活动、破坏行动和外国干预的影响。这项成本将由内政部的现有资金支付。
