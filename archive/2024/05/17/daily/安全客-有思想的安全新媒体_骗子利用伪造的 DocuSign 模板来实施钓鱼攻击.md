# 骗子利用伪造的 DocuSign 模板来实施钓鱼攻击

发布时间 : 2024-05-16 11:55:46

本文是翻译文章

原文地址：<https://www.darkreading.com/threat-intelligence/scammers-fake-docusign-templates-blackmail-steal-companies>

译文仅供参考，具体内容表达以及含义原文为准。

由于伪造模板和登录凭据的地下市场蓬勃发展，模仿 DocuSign 的网络钓鱼电子邮件不断增加。

过去一个月，Abnormal Security 的研究人员声称发现，旨在模仿合法 DocuSign 请求的网络钓鱼攻击显着增加。他们很快就进入了一个俄罗斯网络犯罪论坛，那里的卖家兜售各种类似于真实电子邮件和文档的模板。

**网络钓鱼的地下市场**
市场领先的文档签名软件长期以来一直为网络钓鱼者提供了肥沃的土壤。它的受欢迎程度有所帮助，并且它经常用于存储和传输包含敏感数据的有价值的文档。 DocuSign 电子邮件往往是通用的，这使得伪造它们变得轻而易举，带有一个黄色的大按钮，吸引用户在三思而行之前点击。

Abnormal Security 的首席信息安全官 Mike Britton 解释说：“每个人都已经习惯了，尤其是在工作场所工作一段时间后，DocuSign 链接看起来一定是这样的。” “它有蓝色背景、‘DocuSign’ 徽标、[特色] 外观和感觉。在任何特定的一周内，我可能会处理六种必须签署 DocuSign 的不同事物 — 无论是来自供应商还是合作伙伴，无论如何——我有点习惯于看到它，点击它，然后进入自动驾驶状态。”

为了实现完美的外观和感觉，让受害者陷入自动驾驶状态，攻击者可能会花时间从头开始制作看起来合法的 DocuSign 电子邮件和文档模板。业余的、懒惰的、过度劳累的或者只是逻辑和高效的黑客可能会从在线市场购买现成的恶意软件。毕竟，Britton 说，DocuSign、Amazon、PayPal 等的新模板的成本仅为 10 美元。

有了如此廉价的资源，攻击者就可以制作网络钓鱼电子邮件，以多种方式欺骗目标组织的员工。例如，他们可以发送虚假文档，并提示用户输入其个人身份信息 (PII)，或者可以将用户重定向到虚假登录页面以提交真实的 DocuSign 登录凭据。然后他们可以利用获得的数据，或者更有可能将其出售给食物链中的下一个买家。

正如布里顿所说，“网络犯罪分子掌控[攻击]整个生命周期的时代已经一去不复返了。现在，如果我想攻击 10,000 名受害者并从他们那里窃取金钱，我只需去购买凭证即可，[和]购买访问权——这是缩短它的必要资产。”

因此，除了电子邮件和文档模板之外，网络钓鱼者收集的登录凭据也有一个蓬勃发展的市场。这就是攻击开始变得丑陋的地方。

**对公司的影响**
通过廉价的登录凭据，黑客可以探测员工的 DocuSign 历史记录，以获取他们最近几个月使用过的所有敏感文档。他们可以使用雇主合同、供应商协议和付款信息中的信息作为勒索攻击中的勒索素材，或者他们可以将其出售给更进一步的攻击者。他们还可以使用它来识别新的、更高价值的目标，并冒充公司或合作伙伴公司中的特定个人。

例如，攻击者可以在公司通常每月向供应商付款的时间左右超时发送汇款请求。利用受感染员工的 DocuSign 历史记录中的信息，他们可以冒充直接上级或供应商财务部门的负责人，并将特定的真实文档附加到电子邮件中以供参考。

为了防止这种情况或任何其他潜在的最坏情况的发生，Abnormal Security 建议员工始终留意可疑的电子邮件发件人和链接地址、客观的电子邮件问候语以及异常简短的 DocuSign 安全代码，并直接从公司网站打开文档而不是通过电子邮件。最后，不要打开您不期望的文档。

“每个人都很忙，”布里顿承认。 “无论你是在办公室，还是在有个人生活的混合工作环境中，最安全的选择就是拿起电话说：‘嘿，我刚刚收到了你发来的这封电子邮件。合法吗？’”
