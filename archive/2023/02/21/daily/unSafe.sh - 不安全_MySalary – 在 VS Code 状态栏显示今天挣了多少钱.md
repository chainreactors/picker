## mySalary

开发者 @hzjjg 很谨慎的标记了：**仅供娱乐**。 😂

仅供娱乐的vscode小插件，可以再vscode状态栏上显示今日实时赚取的工资。以及工作状态小提示，感觉工作更有动力了呢！

### 特点：

* 显示你当天到此时所获得的工资
* 在不同工作时段，显示不同的小提示
* 可自定义工资前缀，后缀
* 可自定义工作小提示

### 安装：

* [Visual Studio Code 商店](https://marketplace.visualstudio.com/items?itemName=hzjjg.mysalary)

### 使用：

安装之后，点击 VS Code 左下角状态栏的工资区域，即可设置具体工资、工作时间。
