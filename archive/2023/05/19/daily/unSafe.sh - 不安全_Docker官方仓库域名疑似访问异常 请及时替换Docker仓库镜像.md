据网友分享的信息，Docker 官方仓库域名 https://hub.docker.com/ 在国内疑似出现访问异常，PING 检测显示该域名被返回各种乱七八糟的 IP 地址，这些 IP 都是都不是 Docker 官方服务器的，因此无法正常访问。

如果你频繁使用 Docker 请及时替换第三方提供的仓库镜像，以免出现无法正常下载、更新的情况。
