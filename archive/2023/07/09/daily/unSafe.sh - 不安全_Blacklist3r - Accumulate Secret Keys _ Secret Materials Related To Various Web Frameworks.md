The goal of this project is to accumulate the [secret keys](https://www.kitploit.com/search/label/Secret%20Keys "secret keys") / secret materials related to various web frameworks, that are publicly available and potentially used by developers. These secrets will be utilized by the Blacklist3r tools to audit the target application and verify the usage of these pre-published keys.

We are releasing this project with.Net machine key tool to identify usage of pre-shared Machine Keys in the application for [encryption](https://www.kitploit.com/search/label/Encryption "encryption") and [decryption](https://www.kitploit.com/search/label/Decryption "decryption") of forms [authentication](https://www.kitploit.com/search/label/Authentication "authentication") cookie.

Note: Requires Visual Studio 2019, not 2022. Visual Studio 2022 does not support .NET Framework 4.5, which this repo relies on.
