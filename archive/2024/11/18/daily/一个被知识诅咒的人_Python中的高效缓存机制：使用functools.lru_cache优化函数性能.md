##### 1. 缓存机制概述

缓存是一种用于存储频繁访问数据的机制，通过避免重复计算或数据提取来减少计算时间和系统开销。缓存的应用场景广泛，如网络请求、数据库查询、复杂计算的结果保存等。在Python编程中，缓存机制尤为重要，特别是在处理大规模数据和重复计算时，缓存能够显著提高系统响应速度。

##### 2. `functools.lru_cache`简介

`functools.lru_cache`是Python标准库中的一个缓存装饰器，用于缓存函数调用的结果。LRU的含义是“最近最少使用”（Least Recently Used），即当缓存达到指定容量时，最近最少使用的数据会被优先移除。这种机制适合数据访问频率具有时间关联的场景，例如递归计算和动态规划等。

`lru_cache`的基本使用示例如下：

```
from functools import lru_cache

@lru_cache(maxsize=32)
def fibonacci(n):
    if n < 2:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

print(fibonacci(10))  # 输出: 55
```

在此示例中，`fibonacci`函数的每次计算结果都被缓存起来，以便在重复调用相同参数时直接从缓存中获取。
