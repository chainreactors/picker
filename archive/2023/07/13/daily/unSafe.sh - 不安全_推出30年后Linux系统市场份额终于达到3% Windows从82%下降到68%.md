据 [SataCounter](https://gs.statcounter.com/os-market-share/desktop/worldwide) 的统计数据，在 2018 年达到 1.62% 后，Linux 系统花了 5 年将自己的份额翻倍到 3%，看起来 5 年才翻倍是不是很慢？但要知道在过去二十多年里 Linux 份额才 1.62%，这样来看是不是快很多了？

目前操作系统领域，微软的 Windows 操作系统依然占据着极高的份额 --- 68.15%，比较有趣的是 Windows 份额与 Chrome 在浏览器中的份额走势有些类似。

Windows 在 2018 年市场份额高达 82%，Chrome 之前市场份额接近 90%，现在 Windows 份额已经下降到 68%，Chrome 在浏览器中的份额也下降到 64% 左右。

[而 Safari 浏览器份额快速](https://www.landiannews.com/archives/99368.html)提升与 macOS 系统的份额提升也有直接关系，统计数据显示自 2018 年以来 macOS 系统的份额已经增长到 21.38%，2018 年时为 12.33%。

[![推出30年后Linux系统市场份额终于达到3% Windows从82%下降到68%](https://img.lancdn.com/landian/2023/07/99447-1.png)](https://img.lancdn.com/landian/2023/07/99447-1.png)

说回 Linux，Linux 桌面平台用户其实非常少，主要还是开发者和企业在服务器上使用 Linux 系统，在消费级市场 Linux 桌面系统依然是个非常冷门的选择，通常有些 OEM 会预装 Linux 节省 Windows 授权费用，但用户拿到手基本都还是 Windows。

不过现在选择 Linux 作为基础而开发的设备也越来越多了，NAS 设备一般采用 Linux 系统进行开发，Steam Deck 也是基于 Linux 的，这些设备的流行倒不是可以帮助 Linux 提高多少份额，但 Linux 始终在背后默默助力着互联网的发展。

最后，如果将移动平台也算在内的话，那 Linux 系统就属于其他了，因为算上移动端的话，Android 排名第一占比 40.16%，Windows 排名第二占比 28.59%，iOS 排名第三占比16.8%，macOS 排名第四占比 8.97%，ChromeOS 排名第五占比 1.74%。

[![推出30年后Linux系统市场份额终于达到3% Windows从82%下降到68%](https://img.lancdn.com/landian/2023/07/99447-2.png)](https://img.lancdn.com/landian/2023/07/99447-2.png)

注：SataCounter 的统计数据有时候会比较迷，所以仅供参考。
