# [Wireshark 4.4.1 Released](/forums/diary/Wireshark%2B441%2BReleased/31346/)

**Published**: 2024-10-13. **Last Updated**: 2024-10-13 06:05:59 UTC
**by** [Didier Stevens](/handler_list.html#didier-stevens) (Version: 1)

[0 comment(s)](/diary/Wireshark%2B441%2BReleased/31346/#comments)

Wireshark release [4.4.1](https://www.wireshark.org/docs/relnotes/wireshark-4.4.1.html) fixes 2 vulnerabilities and 27 bugs. One of these bugfixes is for the missing IP address plugin on Windows, see "[Wireshark 4.4's IP Address Functions](https://isc.sans.edu/diary/Wireshark%2B44s%2BIP%2BAddress%2BFunctions/31250)".

And there is a new feature in TShark:

*"The TShark syntax for dumping only fields with a certain prefix has changed from -G fields prefix to -G fields,prefix. This allows tshark -G fields to again support also specifying the configuration profile to use."*
