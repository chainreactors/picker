微软今天向 Windows Insider Canary Channel 通道的用户推出 Windows 11 Can Build 25352 版，本次更新倒是没什么更新内容，但版本代号已经从 rs 变更为 Zn，即锌。

根据前期消息目前微软已经在着手启用镓 (Ga) 代号，从目前金丝雀通道的变更来看，至少目前应该还不是 Windows 12。

[![Windows 11 Can金丝雀通道更新为Zn(锌) 过渡期间部分功能暂时被移除](https://img.lancdn.com/landian/2023/04/98499.png)](https://img.lancdn.com/landian/2023/04/98499.png)

目前金丝雀通道还属于过渡期间，因此部分功能暂时被移除，例如微软拼音输入法的云建议功能相关改进就被移除了，但不用着急，后面这类功能会重新支持。

版本代号的变更通常都涉及重大版本之间的跨越，微软在博客中强调目前金丝雀通道的所有功能都不应该归因于任何已知版本，也就是所谓的动态功能更新，测试的新功能准备完毕后就会直接推送，而不是非得等到某个大版本发布。

话虽如此，重大版本跨越时也会带来更多新功能和改进，所以接下来几周的金丝雀版新增功能应该会逐渐增多。

目前 Windows Insider Dev Channle 通道的代号是镍 (Ni)，原本我们预计今年的 Windows 11 23H2 版会是铜 (Cu) 分支，不过目前还看不到铜分支的实际版本，不知道后面镍分支升级为铜分支还是铜分支被抛弃。
