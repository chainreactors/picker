## Stack Exchange 迁移到云端

[![云计算](https://icon.solidot.org/images/topics/topicicloud.png?123)](/search?tid=146 "云计算")

[Edwards](/~Edwards) (42866)发表于 2025年07月27日 16时11分 星期日 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81901&appkey=1370085986&title=Stack%20Exchange%20%E8%BF%81%E7%A7%BB%E5%88%B0%E4%BA%91%E7%AB%AF "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自月池**

编程问答平台 Stack Exchange 宣布迁移到云端，放弃使用自己的服务器。Stack Exchange 自 2010 年起就在新泽西州的数据中心托管旗下网站，它使用了大约 50 台服务器。如果服务器出现问题，工程师需要去现场更换或重启硬件。2023 年它的 Stack Overflow for Teams 迁移到了微软的 Azure 云，现在 Stack Overflow 和 Stack Exchange 网络托管在了 Google Cloud 云服务上。Stack Overflow 从此不再拥有任何物理数据中心或办公室，完全在云端远程工作。
hardware.slashdot.org/story/25/07/26/1946217/stack-exchange-moves-everything-to-the-cloud-destroys-servers-in-new-jersey
