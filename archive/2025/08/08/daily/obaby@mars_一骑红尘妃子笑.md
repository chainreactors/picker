# 一骑红尘妃子笑

2025年8月8日
[45 条评论](https://h4ck.org.cn/2025/08/21108#comments)

[![](https://h4ck.org.cn/wp-content/uploads/2025/07/WechatIMG1651.jpg)](https://h4ck.org.cn/wp-content/uploads/2025/07/WechatIMG1651.jpg)

自上篇文章到现在已经累计超过一个月的时间，七月七日接到一个电话，后续就是这一个月的沉寂。不要问为什么，也不要问该怎么办。

在那个烈日炎炎的中午，来到了市南区。刺目的阳光晒进车内，经过一层玻璃都能感受到炽热的阳光，仿佛要把自己融化一般。

[![](https://h4ck.org.cn/wp-content/uploads/2025/07/16411753063460_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2025/07/16411753063460_.pic_.jpg)

市南的建筑没有太大的变化，之前在市南上班的时候，喜欢将车停在家乐福顶楼的停车场。说是喜欢，其实是因为便宜，毕竟奥帆的停车场没了年卡之后 8 块钱一小时，80 一天。这已经不是停车不停车的问题了，真的停不起。

[![](https://h4ck.org.cn/wp-content/uploads/2025/07/16401753063459_.pic_.jpg)](https://h4ck.org.cn/wp-content/uploads/2025/07/16401753063459_.pic_.jpg)

楼顶的热浪一波波的袭来，跟进了烤箱一样，甚至连呼吸都变得困难。楼下的家乐福早在四年前就已经撤出，现在已经成了苏宁，只是那个标牌还依然矗立在那里，似乎诉说着当年的历史。

下午回到公司，已经没什么时间吃饭了。不过自己倒是也不是很饿，所以吃不吃也就无所谓了。裁员的阴霾依旧笼罩在上方，只是，这次来的更猛烈一些。定了名单，自然就是杀伐果断，没有什么可以协商的余地，不过这长痛和短痛，却别不大。

我手上负责的项目，已经在五月被其他人带着人马拆分了出去，于是，我得到了这片刻的清闲，我也得到了各种领导的关注。在某一天，我的领导跟我说，在集团会上，大领导当场问：“钟\*\*现在在干嘛？工作严重不饱和！”

当然，我的领导替我揶揄了几句，但是这解决不了我成为众矢之的。我这早就不在庙堂，竟然还有人能记得我。这也是属实不容易了，只是，我不做大哥好多年。

在某天下班之后，领导叫我到一个无人的角落说：“你跟\*\*，现在很危险，W 已经说了，下次裁员的目标很可能要把你们俩给报上去了。你是以为没有长期项目，他是因为在某论坛太活跃了，以至于集团都发现了这个人才，找到了咱们这边。只是，咱们不是个技术公司，也不需要有那么多的时间去技术论坛发文章”

“没事，其实也没什么的，如果说真到了那一天，也就只能拿补偿走人了。这个也没什么选择的余地不是吗？”我答道。

“这个裁员持续了这么长时间，后面还会继续，我也一样，没什么所谓的项目管理，专门的研发经理了”他继续说道。

“嗐，其实也没什么，也的确是没什么新的业务，要缩减成本，那自然是向研发开刀了”我附和道，“毕竟现在要搞什么所谓的生产力工具，不就是为了缩减程序员嘛”

自这天起，开始在集团里找各种工作岗位，包括之前的关系不错的，都联系了一下。然而，投了几份之后，得到的答复都是，我很优秀，但是，和岗位匹配度不高。说白了就是那些所谓的 java 以及 ai 技术要求都更直接一些，然而，这方面我的确了解，要能做，其实也能搞，本来就没什么高深的东西。

就这样，我收获了一堆“好人卡”。

好人卡多了之后，说实话，我也不大想继续折腾了。集团内，所谓的那些领导们，也多数都是半桶水，并不见得比我高明多少。然而，现在却没什么好办法来解决这个问题。我开始等那头上的刀落下来。

之前自己团队里的人已经走的差不多了，从十几个人，到现在三四个人。多数人跟着自己都超过三年时间了，唯一一个时间比较短的是 23 年招的那个应届生。想着这时日无多，也不见得什么时候是个头。在某天中午决定带招来的这姐妹去吃饭。

只是，这一顿饭吃完之后，下午，第一波的裁员名单下来了，自己出去买个饮料的功夫。回来之后，她过来跟我说，自己被约谈了，在这一波的裁员名单里，要么人力帮忙找岗，要么拿赔偿走人。只是现在这个情况下，找岗的确是比较困难，一时间我竟然不知道该说什么。

下午下班之后，陆陆续续的也大概知道了名单里面的几个人。这下，自己熟知的人就更少了，虽然活少了，人少了，但是整体开发的工作量却并不会少太多，毕竟，还有那些有的没的项目，乱七八糟一大堆，而至于工作，自然是变得更加具体。当然，得益于领导的关注，给安排了个新活，之前没干过，但是也不见得不能干，毕竟就那么点破事，还能有什么难的呢？

是时候多写写代码了，至于框架，自然是转到了 java。至于单体的 java 框架还好，只需要直接运行个 jar 包就 ok 了，而现在要用微服务框架，就需要解决各种依赖问题，虽然是基于开源的框架开发和部署，但是在部署的时候还是遇到了各种问题。说实话就是，第一次接触这个东西真的没那么熟练，尤其是各种文档对新手的确不友好。

在折腾了两天之后还是给部署成功了，项目总算是跑了起来。这，也算是入门了吧。只是，这个也是暂时的吧，哪有什么安稳。

大姐家的孩子虽然高考成绩不好，但是，还是要回去表达一下心意。主要也是聊聊后面上学怎么办。学历日益贬值，现在本科生已经没什么优势，研究生尚可，至于专科？那自然是更难。早上八点多就出门了，毕竟路程不算远，两个来小时的车程。宝子 8 月初要考级，自然不能带她，还要去上钢琴课。

而至于不带宝子还有另外一个原因，那就是没地方住。二姐家的孩子又把之前住的房间占了，弄得乱七八糟，也实在是不想去折腾。所以，自己回去也没打算在家住。回家之后，姐姐们陆陆续续也就来了，即使是最亲近的人，有的话题也无法聊到一起。好几次聚餐没见到四姐了，这次她反而是最早到的。

他的大儿今年也参加高考，同样，也是差不多的成绩，离本科线差 100 分。四姐夫还因为这个成绩和孩子闹翻了，孩子一气之下，去了他叔叔家。这次聚会自然也来不了。四姐夫还在复读与打工之间纠结，其实就这个成绩反而也没什么纠结的了，复读意义不大。就这个天分，浪费一年时间而已。除此之外没什么别的用处，明年还是会重复今天的故事。小儿子手上打着绷带，挂在脖子上在那里玩手机。说是被被人撞到了，有点软骨挫伤，没有骨折已经算是万幸。说去镇上的医院贴膏药，那个膏药多好多少。这种话题我一般不愿意参与，因为我一旦参与多数是反对者。既然他们觉得有用，那就继续吧。

午饭满桌子的菜，我吃的并不多，这半年减肥的效果就是食量下降了很多。吃了点凉拌黄瓜，还有青菜。最终的话题还是选学校和专业的问题，其实这个东西，我也不知道该怎么选，而至于以后的出路我也看不清，自己的路都看不清楚，又如何能看清楚别人的。不过大姐的想法是让闺女去学什么针灸中医之类的。也算是个选择吧，毕竟人口老龄化越来越严重了，这两个最起码以后应该也不会失业。而至于哪个学校心理没底，至少去学校的招生办网站搜电话，让她打电话问。在这之前她也不知道该怎么办，问谁。这个还能问谁呢？要想能靠谱，不出问题，那自然还是问学校招生办，大概就有数了。毕竟，一年一万六到三万多的学费，上了就得交钱不是吗？

不过这一圈问下来心理倒是也大概就有了数了，不在那么迷茫。至于学完之后有用吗？这谁又知道，只是这个年龄的孩子，不去上学，能干什么呢？哪怕没什么用，也要度过这几年的时光，去大概了解下这个社会不是吗？

下午大姐一家走了之后，四点左右自己也走了。有时候，呆在一起，也挺难找到一个合适的话题。县城的空调温度显示室温 34 度， 不过好在能远程启动，到家之后也不至于跟进了烤箱一样。一个人在家，也挺无聊的，到家之后也有那么一丝丝后悔，应该吃完晚饭再回来的。

这跟一个人在路上感觉还是不同的，一个人的旅行，开车其实没那么寂寞，然而，一个人呆在家里却倍感寂寞。也不想出去吃饭，订了份外卖，然而烤串的质量真的很一般，尤其是那两串羊肉，竟然是肉片，食之无味。

吃完饭，这漫漫长夜也的确无聊。吃完饭，找个地方按了一下。回来的时候发现进不了小区了，小区门口围了一群人。把车停到路边，走回来才发现的确是出事故了，别倒了一辆二轮电动车，还别住了一辆三轮电动车。这个操作技术难度应该蛮高的。

[![](https://h4ck.org.cn/wp-content/uploads/2025/07/138ca6377bafaaf9c259f8bf0ec8a5.jpg)](https://h4ck.org.cn/wp-content/uploads/2025/07/138ca6377bafaaf9c259f8bf0ec8a5.jpg) [![](https://h4ck.org.cn/wp-content/uploads/2025/07/a6dbf685291faf4cb7d5d730aa006a.jpg)](https://h4ck.org.cn/wp-content/uploads/2025/07/a6dbf685291faf4cb7d5d730aa006a.jpg)

手机的镜头膜现在跟神经病一样，竟然拍出了离奇的效果。

晚上自然空调也是不敢关，一旦关上，温度上升的比热得快还快。这的确让人有些无语。

一觉醒来，天已大亮。关掉空调，洗刷一下准备往回走，这空调关掉之后，稍微一活动就已经开始冒汗。不过这早上的温度自然还是要低一些，返程第一件事还是要先去加油，到加油站，使用壳牌的 app 付款的时候，却一直闪退，知道后来才知道，原来已经切换到小程序了。至于 app 已经好久不用了，一番折腾之后才完成付款。只是，不管是 app 还是小程序，壳牌的 ui 和交互总是让人觉得有些别扭，很不方便。

后面的日子，依旧是一天天的重复，似乎什么都不同了，似乎什么都没变。不过心态稍微平和了一点， 内心的焦虑稍微缓和了一些。毕竟，眼下似乎危机没那么严重了。当然，这个世界离了谁一样运行，没什么是不可或缺的，更何况到处都是草台班子。

每天依然是上班、下班，上个周末对象家表弟结婚。时间也早已经通知，宝子和姥姥、姥爷周五就回老家了，家里自然就剩下我们两个。这大好的时光似乎应该做点什么，我提议去看电影。搜了一通发现也没什么特别喜欢的片子，最终选了《侏罗纪世界：重生》。

这一场电影下来，两个多小时，就是标准的商业片的感觉。周五晚上的影院竟然也没多少人，在我买票的时候只卖出去两张。等快开始了，看了下，也不过十来张，整个影厅就那么稀稀拉拉的坐了十来个人。

银座也的确是没什么人气，甚至连放映的音响设备都有问题，右侧的第二个音响低频共振非常严重，还有杂音，让观影体验又降低了不少。不过现在好莱坞的电影，确实难以满足国人的审美和口味了，两个多小时，比较拖沓，情节也没那么引人入胜。

周五就是婚礼了，我们两个其实都不想去参加这个婚礼。然而，宝子的姥姥、姥爷带着孩子回去了，要求周日的时候有人给接回来，既然总是要回去的，那还是对象去接吧。而至于我，自然是不想去。大好的时光怎么能浪费在这里呢？

更何况在知道这天是表弟婚礼的时候，我已经预定了那天的写真。有什么能比拍写真更重要呢。

对象七点多就出发了，自己收下东西，洗刷完打车到公司停车场开车去拍写真。约的十点，到的时候发现才九点四十，按了下门铃没反应，问了下才知道店里还没开门，嗯。这次来早了。

这次比上次拍的时候又轻了七八斤，感觉效果还是有那么一丢丢的。上次的之后肚子稍微有点大，摄影师问，你的束腰呢？问题是我真的没带啊，自从决心减肥之后，之前买的束腰就都扔掉了。这次似乎不用束腰感觉也能稍微好那么一点点，毕竟减肥真的减了。

不过这个天也实在不想拍外景，室内也蛮好的啊。就这样，每三个月拍一次，当然，这也是减肥的目标之一，为了好看。

[![](https://h4ck.org.cn/wp-content/uploads/2025/07/psc-scaled.jpeg)](https://h4ck.org.cn/wp-content/uploads/2025/07/psc.jpeg)

这次尝试了一下丸子头，然后就爱上这个发型了，感觉真的好看。半年的减肥之后，竟然能看到锁骨了，真的不容易。

减肥，那么多的付出，这时候才是最开心的。

三套衣服全部拍完之后，最中意的还是这套，一共拍了 120 多张，然而，套餐里面只有 16 张精修。这拍都拍了 16 张怎么够呢，删了半天最后还剩了 60 张。至于多出来的就只能加钱解决了。

等选完照片已然到了两点多，下午的时间还有不少。打定主意再去看一场电影，选来选去时间比较近的只有《长安的荔枝》。

荔枝这个东西，之前电视剧版的看了一些，不过后来因为时间有限，剧情推进的没那么快。就没再继续跟，再后来出了个什么上戏的那什么的姐妹。也懒得再去跟了。

之所以选这个，还有一个原因是上面有个喜剧的标签，想着既然是喜剧，怎么看完也能高兴点。

周五的下午，影院并没有多少人。看了一下偌大的 imax 厅也只有十来个人，甚至都不到。

只是，这喜剧的标签贴的有些无厘头了，看了之后真的没什么笑点。作为一个牛马，这不过是现实的真是写照而已。“圣人”一句话，各个阶层就开始折腾。至于这些折腾，层层都是为...
