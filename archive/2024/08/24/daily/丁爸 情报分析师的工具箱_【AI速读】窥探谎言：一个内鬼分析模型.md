# 【AI速读】窥探谎言：一个内鬼分析模型

丁爸 情报分析师的工具箱

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xmhj3vHPBKNPljUYMoWYibok2UHDsvtNpYzYEzCyDULahicfDJ9krYibTqjqVzCVRcib5tcGL1Q1TX1w/640?wx_fmt=png&from=appmsg)

间谍行动的威胁常常来自于内部人员，一个单位和组织内部哪些人更容易成为内鬼？本研究的目的是建立一个“风险个人”的理论模型，并利用该模型、人格评估和文本分析技术，开发一种工具，进行自我评估测试，可用于评估一个人是否更有可能实施间谍行为。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xmhj3vHPBKNPljUYMoWYiboAZc19SEFNsI2wY3klEaZrIKC35s1EEeibic5llGwMCthSZ1ic2nUc537A/640?wx_fmt=png&from=appmsg)

本研究收集了52个个体的数据，并用各种定量方法进行分析。在对理论模型提出的风险因素进行内部可靠性检验时，8个因素中有7个具有良好的可靠性。

研究策略采用问卷调查法。该工具由三个模块组成，每个模块测试与一个内部风险个体相关的不同人格特征。通过文献研究，建立了风险个体的理论模型。在构建每个模块时，使用了各种来源，包括专家意见、国际人格项目库和瑞典安全局的指南。

所有个体都是独一无二的，拥有不同的价值观、兴趣、能力和经历。有些人可能更容易冒险，经常做出冲动的举动，而另一些人则喜欢规划自己的生活。有些人可能会对某些物质上瘾，而其他人则不会。在某种物质的影响下，人们往往会比清醒时更具冲动性，从而增加冒险的可能性。我们对外部刺激的反应和表现情感的方式各不相同。在某些情况下，拥有双重国籍可以被视为一种文化丰富，而在其他情况下则相反，因为忠诚度的分裂可能会给某人的雇主带来安全问题。在一些环境中，擅长说谎可能非常有益，但在其他环境中则可能导致问题。个体差异取决于环境，可能是祝福，也可能是诅咒。通过使用不同的方法和技巧，如性格评估、文本分析、以往的研究以及安全许可指南，可以衡量个体差异并计算风险。

五大特征模型，也被称为五因素模型(FFM)，已经在研究者和心理学家之间获得了共识。五大人格包括五种人格结构，它们是跨主要人格清单和跨文化的强大结构。这五个特征，通常被记作OCEAN，代表每个特征的第一个字母，分别是：开放性、严谨性、外向性、宜人性和神经质。最后一个概念，神经质，在近代被贴上了稳定性的标签。

开放的人往往具有想象力强、机智、原创和艺术性强的特点，而在这一特质上得分低的人往往会显得肤浅或简单。高水平的尽职尽责与个体更加谨慎、可靠、有组织和负责任相关。在这一特质上得分低的人倾向于更冲动、粗心、混乱和不靠谱。外向型个体更具活力、热情、支配力和健谈，而对于内向者来说情况相反，可以描述为更为害羞、顺从和安静。在宜人性上取得高分与某人友好、合作和信任有关，而低分则与冷漠、好争吵和不友善的人有关。最后一个结构稳定性上取得高分与情绪稳定、平静和满足感强相关，而低分的个体在情感上更容易不稳定、紧张和不满足。

作为五大人格模型的补充，有人提出了一个被称为黑暗三合一体的人格模型，其中包含了反社会和恶意行为。黑暗三合一体包括五大因素无法解释的三个性格特征：马基雅维利主义、自恋和精神变态。这三个特质共享一些熟悉的个性特征，如“社交上的恶毒角色、自我提升、情感冷漠、虚伪和攻击性”。

在马基雅维利主义上得分高的人具有操纵性的性格，会使用不同的策略来追求自利、欺骗和操控，即使别人可能会被利用，因为这种人对他人的同理心不强。一个自恋主义者（不要与人格障碍的临床定义混淆）是在自恋性格特质上得分高的人，他们感到自己有权获得某些东西，有支配和优越的需求，表现出傲慢和主要关注自我。在精神病态上得分高的人（同样，这个术语不是指人格障碍的临床意义）是冲动的、寻求刺激的，结合了低度的同理心和低焦虑。这种性格的其他特征包括傲慢、欺诈和不负责任的行为。

尽管黑暗三合会特质与五大特质不同，但它们之间仍发现了相关性。黑暗三合会的所有三种特质都与和悦性（五大）呈负相关。在心理变态量表上得分高的人在神经质量表上的得分低，而在马基雅维利主义和心理变态量表上得分高的人“都显示出较低的责任心水平，而自恋与认知能力显示出轻微的正相关”。

因此，在评估某人性格时，将五大人格与黑暗三合一体结合使用可以揭示组织或机构在雇用该个人之前想要了解的有关此人的信息。

**内部人员风险分析模型（内鬼分析模型）**

|  |  |  |
| --- | --- | --- |
| **风险因素** | **定义** | **行为示例** |
| **压力** | 一种情绪上和生理上的状态，由苛求或挑战环境的经历引起 | 与财务问题(如债务人超支)、个人问题(如离婚)或专业问题(如由于缺乏知识或经验而无法完成与工作相关的任务)有关的压力。 |
| **不满情绪** | 由于感觉受到不公平待遇、不公正或未实现的期望而产生的不满或不满情绪。 | 糟糕的工作态度(例如旷工或错过截止日期)，对工作感到沮丧，对同事或管理层心存怨恨。 |
| **权威问题/违反规范** | 自暴自弃的由于对上级的不满或反叛而难以接受或遵守上级的规定和指示的 | 对权威生气，对权威不屑一顾。违反了规范，不总是循规蹈矩。 |
| **优越感** | 认为个人应该享有特权和利益，而不管这些特权和利益是否是通过努力得来的。 | 自恋型人格类型(例如，感觉自己比同龄人优越)，认为自己凌驾于法律和/或其他人之上 |
| **滥用物质** | 过度使用有害物质或酒精，导致身体、心理和社会问题。 | 酗酒或吸毒 |
| **缺乏共情能力** | 难以理解或不能理解或分享他人的感觉、情绪和观点的。 | 不顾及别人的感受，在事业上和个人生活上出人头地而不顾及别人。 |
| **道德灵活性** | 有能力调整他们的道德标准和行为，以适应不同的情况和背景，以获得个人利益。 | 是非不分或不道德的行为 |
| **孤独感** | 孤立感缺乏有意义的社会联系，与母亲隔绝或疏远的感觉 | 在工作中独来独往，不参加社交活动。 |

该分析模型由三个不同的模块组成。

**第一个模块**是分析模型的基础，包括基于五大人格模型的（规范的）性格评估测试。

**第二个模块**是在第一模块基础之上，通过八个风险因素和专家意见来测试个人的脆弱性。在国际人格项目池的帮助下，选择了针对每个风险因素的量表（偶尔进行修改以更精确地定位每个风险因素），以测试个体在心理特质上与理论模型所提出的风险个体的比较情况。

**第三个模块**是基于当前指导方针的个人安全许可测试。这些问题对公众开放，用于面试需要安全许可的工作职位（瑞典安全警察，2021）。

为了更深入地了解个体，模块三还包含了一些由个体自行书写的开放式问题。以往的研究表明，人们的书写方式可以提供一些关于他们是否说实话的洞见，而能够高效地说谎对内幕风险个体或间谍来说可能是有帮助的。与一些诚实的人相比，说谎者更为规避，表达不清楚，提供的细节较少，并试图使自己远离他们的写作（例如，不用“我”作为代词，而是使用“一个”来代替）。说谎者也倾向于比说实话的人表达更多的情感，无论是消极的还是积极的。

共有52名参与者提供了数据，包括学生、政府机构内的员工以及其他人员。

在将理论模型提出的风险因素应用于内部可靠性测试时，八个因素中有七个显示出良好的可靠性。其中一个因素，压力，表现不佳。这可能是因为所提问题的范围太广，从个人压力到职业压力都有。因此，压力被排除在进一步测试之外。剩余的七个因素相互关联，只有一个因素除外，即权利感。这个风险因素与道德灵活性有关，但与其他六个风险因素无关。为了测试五大人格特征与风险的相关性，我们计算了一个风险个体的平均值，并将其与OCEAS（开放式经验评估系统）的五个因素进行了比较。这五个因素都与风险呈负相关，其中宜人性具有最高负相关，外向性具有最低负相关。在将得分最高的十个参与者和得分最低的十个参与者进行比较时，可以发现差异。在五大人格特征方面，差异较小；也就是说，五大人格特征和理论模型都能用来筛选高风险个体。然而，两者结合使用比单独使用其中一种效果更好，能够提供更深入的洞察。
