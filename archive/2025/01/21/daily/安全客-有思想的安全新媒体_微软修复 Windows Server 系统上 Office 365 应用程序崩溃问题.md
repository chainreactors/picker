еҫ®иҪҜе·Із»Ҹдҝ®еӨҚдәҶдёҖдёӘе·ІзҹҘй—®йўҳпјҢиҜҘй—®йўҳдјҡеҜјиҮҙ Microsoft 365 еә”з”ЁзЁӢеәҸе’Ңз»Ҹе…ё Outlook еңЁ Windows Server 2016 жҲ– Windows Server 2019 зі»з»ҹдёҠеҙ©жәғгҖӮ

ж №жң¬еҺҹеӣ жҳҜжңҖиҝ‘зҡ„дёҖж¬Ў Office жӣҙж–°пјҢиҜҘжӣҙж–°йӣҶжҲҗдәҶ React Native жЎҶжһ¶пјҢд»Ҙж”ҜжҢҒ Microsoft 365 еә”з”ЁзЁӢеәҸзҡ„жҹҗдәӣеҠҹиғҪпјҢиҖҢеҪұе“Қд»…йҷҗдәҺе°Ҷз»Ҹе…ё Outlook е’Ң Microsoft 365 еә”з”ЁзЁӢеәҸжӣҙж–°еҲ° 2412 зүҲжң¬пјҲBuild 18324.20168пјүзҡ„ Microsoft 365 з”ЁжҲ·гҖӮ

жң¬е‘ЁдәҢпјҢи®ёеӨҡз”ЁжҲ·жҠҘе‘ҠиҜҙпјҢд»–д»¬зҡ„ Outlook з”өеӯҗйӮ®д»¶е®ўжҲ·з«Ҝе’Ң Office 365 еә”з”ЁзЁӢеәҸпјҲеҰӮ ExcelгҖҒWordгҖҒOutlook е’Ң PowerPointпјүеңЁеҗҜеҠЁеҗҺ 15 з§’еҶ…еҙ©жәғпјҢе…¬еҸёйҡҸеҗҺйҰ–ж¬ЎзЎ®и®ӨдәҶиҝҷдёҖй—®йўҳгҖӮ

е‘ЁеӣӣпјҢйӣ·еҫ·и’ҷеҫ·еҸ‘еёғдәҶдёҖдёӘ Office еҘ—д»¶жӣҙж–°пјҢдҝ®еӨҚдәҶеңЁ Windows Server зі»з»ҹдёҠеј•еҸ‘еә”з”ЁзЁӢеәҸеҙ©жәғзҡ„й”ҷиҜҜгҖӮ

вҖңеҰӮжһңдҪ зҡ„ Microsoft 365 Apps е®ўжҲ·з«Ҝиў«й…ҚзҪ®дёәд»Һ Office еҶ…е®№дәӨд»ҳзҪ‘з»ңпјҲCDNпјүиҮӘеҠЁжӣҙж–°пјҢеҲҷж— йңҖйҮҮеҸ–д»»дҪ•жҺӘж–ҪгҖӮеҰӮжһңжӮЁзӣҙжҺҘз®ЎзҗҶжӣҙж–°пјҢзҺ°еңЁе°ұеҸҜд»ҘдёӢиҪҪжңҖж–°жӣҙж–°е№¶ејҖе§ӢйғЁзҪІгҖӮвҖқд»ҠеӨ©пјҢMicrosoft 365 з®ЎзҗҶдёӯеҝғж–°еўһдәҶдёҖжқЎж¶ҲжҒҜгҖӮ

вҖңд»ҺеӨӘе№іжҙӢж ҮеҮҶж—¶й—ҙ 2025 е№ҙ 1 жңҲ 16 ж—ҘејҖе§ӢпјҢжҲ‘д»¬е°ҶйҖҗжӯҘеҗ‘иҜҘжӣҙж–°йҖҡйҒ“дёҠзҡ„з”ЁжҲ·жҺЁеҮә Microsoft 365 еә”з”ЁзЁӢеәҸзҡ„иҝҷдёҖжӣҙж–°гҖӮвҖқ

![Microsoft Outlook crash on Windows Server]()
еҫ®иҪҜ Outlook еңЁ Windows жңҚеҠЎеҷЁдёҠеҙ©жәғпјҲдәҡеҺҶе…Ӣж–Ҝ-зҰҸж ји’Ӯпјү

**д№ҹжңүи§ЈеҶіж–№жі•**

жңӘиҮӘеҠЁеә”з”Ёжӣҙж–°дё”ж— жі•з«ӢеҚіеңЁзі»з»ҹдёҠе®үиЈ…жӣҙж–°зҡ„е®ўжҲ·д№ҹеҸҜд»ҘйҖҡиҝҮд»ҘдёӢжӯҘйӘӨжүӢеҠЁжҒўеӨҚеҲ°зүҲжң¬ 2411пјҲBuild 18227.20162пјүпјҢдҪңдёәдёҙж—¶дҝ®еӨҚжҺӘж–Ҫпјҡ

1. еңЁз®ЎзҗҶе‘ҳдёҠдёӢж–Үдёӯжү“ејҖе‘Ҫд»ӨжҸҗзӨәз¬ҰгҖӮ
2. еңЁ вҖңе‘Ҫд»ӨжҸҗзӨәз¬Ұ вҖқзӘ—еҸЈдёӯй”®е…ҘжҲ–зІҳиҙҙдёӨжқЎе‘Ҫд»ӨпјҢе№¶еңЁжҜҸжқЎе‘Ҫд»ӨеҗҺжҢү Enter й”®пјҡ
   cd %programfiles%\Common Files\Microsoft Shared\ClickToRun
   officec2rclient.exe /update user updatetoversion=16.0.18227.20152
3. иҰҒйҳІжӯў Office жӣҙж–°еӣһжңҖж–°зүҲжң¬пјҢдҪ еҸҜд»ҘзӮ№еҮ»ж–Үд»¶гҖҒOffice еёҗжҲ·гҖҒжӣҙж–°йҖүйЎ№е’ҢзҰҒз”Ёжӣҙж–°жқҘе…ій—ӯжӣҙж–°гҖӮ
4. иҜ·еңЁж—ҘеҺҶдёҠжіЁжҳҺ 2 жңҲ 11 ж—ҘпјҢд»ҘдҫҝйҮҚж–°еҗҜз”Ёжӣҙж–°гҖӮ

жң¬е‘Ёж—©дәӣж—¶еҖҷпјҢеҫ®иҪҜзј“и§ЈдәҶеӨҡеӣ зҙ иә«д»ҪйӘҢиҜҒпјҲMFAпјүж•…йҡңпјҢиҜҘж•…йҡңеҜјиҮҙз”ЁжҲ·ж— жі•и®ҝй—®Microsoft 365 Officeеә”з”ЁзЁӢеәҸгҖӮ

12жңҲеҲқпјҢз”ұдәҺд»ӨзүҢз”ҹжҲҗй—®йўҳеҜјиҮҙиә«д»ҪйӘҢиҜҒиҜ·жұӮеӨұиҙҘпјҢд№ҹйҖ жҲҗдәҶOffice Webеә”з”ЁзЁӢеәҸе’ҢMicrosoft 365з®ЎзҗҶдёӯеҝғзҡ„зҳ«з—ӘгҖӮ
