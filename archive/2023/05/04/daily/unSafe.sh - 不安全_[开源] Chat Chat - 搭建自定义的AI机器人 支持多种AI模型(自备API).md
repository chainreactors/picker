Chat Chat 是开发者 okisdev 推出的一个开源项目，这个开源项目实际上是为用户提供一种功能丰富的交互界面，同时可以在线或离线部署，只要用户有各个 AI 模型的 API 即可。

AI 模型方面支持 OPENAI、微软的 Azure OPENAI、Claude、Cohere、Hugging Face 等等，用户提前配置好 API 后可以随时在不同的模型之间切换。

[![[开源] Chat Chat - 搭建自定义的AI机器人 支持多种AI模型(自备API)](https://img.lancdn.com/landian/2023/05/98518.png)](https://img.lancdn.com/landian/2023/05/98518.png)

目前已经支持的功能包括：TTS、与文件聊天、分享对话、支持 SSE 流信息、Markdown 格式化、代码语言语法高亮、System Prompt、快捷菜单、聊天记录、封装的 API、创建功能。

未来会提供诸如语音输入、深色模式、多语言支持、支持更多 API 等。

因此借助 Chat Chat 你可以快速搭建一个供自己或别人使用的 AI 对话网页，当然无论是你自己使用还是别人使用都需要提供 API，Chat Chat 项目并不支持白嫖功能。

部署方面，支持通过 Vercel 快速部署或 Docker 快速部署，也支持本地部署，简单来说基本就是两分钟就可以完成搭建。

有兴趣的用户可以点击这里查看项目主页：<https://github.com/okisdev/ChatChat>

你也可以点击这里使用开发者提供的网站：<https://chat.okisdev.com/> 请注意：无论使用哪种模型，只要是模型本身需要 API 交互的，那你就需要在左下角设置里配置 API 后才能使用。
