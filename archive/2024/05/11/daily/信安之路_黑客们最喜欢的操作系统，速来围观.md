原创

xazlsec

信安之路

工欲善其事，必先利其器，对于老司机而言，更多是喜欢自己安装部署使用得心应手的工具集，每个老司机都有自己的工具集和操作系统，但是对于初学者而言，使用的工具很少，也不知道哪些工具适合自己，而且每个工具的部署所依赖的环境和基础组件都有所不同，很容易耗费大量精力在工具的安装部署上，所以这些集成黑客系统就变成了初学者的首选。

今天为大家收集整理了 9 款适用于黑客的操作系统，包括渗透测试、安全评估、IoT、安全取证等领域，总有一款适合你，当然，这也不是全部，国内也有不少团队整理这类操作系统，没在统计范围呢，大家自行搜索。

### 0x01 ArchStrike – 一键部署黑客常用软件

严格遵循 Arch Linux 标准，保持软件包简洁、安全且易于维护，官网：

> https://archstrike.org/

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibyP7zNamoCUc3PdI6ccN5dXhpBkCa1JCnLcXe4V7FrmlYlTtTKkpajA/640?wx_fmt=png&from=appmsg)

从 Github 项目中可以看到所有部署的软件列表：

> https://github.com/ArchStrike/ArchStrike/blob/master/archstrike/utc.patch

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibQfq4ibfiazOlv2ibp3sqWibuQwoqjykc4MXS7iaCJgRM3A6QRLHCks6Q3tg/640?wx_fmt=png&from=appmsg)

### 0x02 AttifyOS – 一款基于 Linux 系统的 IoT 渗透系统

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibHibbwhQ1ibGSs5Giby8ZEYDc2t7IeqXx4UTGJkXcYIQSNSVvibpcndKkeg/640?wx_fmt=png&from=appmsg)

AttifyOS 帮助安全专业人才对 IoT（物联网）设备进行安全评估，预装软件包括：

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibuEQsibCOicNm6I1giaDmDGXNNp7bsTqF0nicpJ3jFu71FaIw1tv42Glkgg/640?wx_fmt=png&from=appmsg)

目前更新至 4.0 版本，项目地址：

> https://github.com/AttifyOS/AttifyOS

### 0x03 BackBox – 一款面向渗透测试和安全评估的 Linux 系统

其中安装部署了最常见/最常用的安全和分析工具，从 web 应用分析到网络分析、压力测试、抓包嗅探、漏洞评估、计算机取证、汽车漏洞利用等，基于 Ubuntu 系统。

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibYWJRnM4D9kYneibWSrq9dxFn0qPyulRr1I5ctZPBT4KzFr465uLVleA/640?wx_fmt=png&from=appmsg)

### 0x04 BlackArch – 一款面向渗透测试人员等操作系统

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibYhr9dnWIR1Wo1HSzHhR7D0Z9cp1o2zbohR6LWAyyDEKOyRZCwSjhtA/640?wx_fmt=png&from=appmsg)

> https://blackarch.org/

![](https://mmbiz.qpic.cn/mmbiz_png/sGfPWsuKAffPbbmUJfrib0ToHISTOhhPibxUbUkTeV8BzrVHBC7sQXxWr2qqER735548exibEbmicVm1vZp5PRV6PQ/640?wx_fmt=png&from=appmsg)

> https://parrotsec.org/
