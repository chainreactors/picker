еңЁе№ҝеҸ—ж¬ўиҝҺзҡ„ TP-Link и·Ҝз”ұеҷЁдёӯпјҢе·ІеҸ‘зҺ°еӨҡдёӘжјҸжҙһпјҢз”ұдәҺе…¶ Web з®ЎзҗҶз•ҢйқўеӯҳеңЁ SQL жіЁе…Ҙзјәйҷ·пјҢиҝҷдәӣжјҸжҙһдҪҝз”ЁжҲ·йқўдёҙдёҘйҮҚзҡ„е®үе…ЁйЈҺйҷ©гҖӮ

иҝҷдәӣжјҸжҙһжҳҜз”ұе®үе…Ёз ”з©¶е‘ҳ вҖңThe VeteranвҖқ еҸ‘зҺ°зҡ„пјҢе®ғдҪҝеҫ—иҝңзЁӢж”»еҮ»иҖ…иғҪеӨҹз»•иҝҮиә«д»ҪйӘҢиҜҒпјҢеңЁж— йңҖжңүж•ҲеҮӯжҚ®зҡ„жғ…еҶөдёӢеҜ№и®ҫеӨҮиҝӣиЎҢжңӘз»ҸжҺҲжқғзҡ„жҺ§еҲ¶гҖӮ

****TP-Link и·Ҝз”ұеҷЁжјҸжҙһжҰӮиҝ°****

****CVE-2025-29648пјҡTP-Link EAP120 SQL жіЁе…ҘжјҸжҙһ****

жӯӨжјҸжҙһеҪұе“Қ TP-Link EAP120 и·Ҝз”ұеҷЁпјҲзүҲжң¬ 1.0пјүгҖӮе…¶зҷ»еҪ•д»ӘиЎЁжқҝжңӘиғҪжӯЈзЎ®жё…зҗҶиә«д»ҪйӘҢиҜҒеӯ—ж®өдёӯзҡ„з”ЁжҲ·иҫ“е…ҘгҖӮ

еӣ жӯӨпјҢжңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҸҜд»ҘйҖҡиҝҮиҝҷдәӣеӯ—ж®өжіЁе…ҘжҒ¶ж„Ҹзҡ„ SQL иҜӯеҸҘгҖӮ

дёҖж—ҰжҲҗеҠҹеҲ©з”ЁиҜҘжјҸжҙһпјҢж”»еҮ»иҖ…е°ұиғҪеӨҹз»•иҝҮиә«д»ҪйӘҢиҜҒпјҢе№¶жңүеҸҜиғҪиҺ·еҸ–и®ҫеӨҮзҡ„з®ЎзҗҶе‘ҳи®ҝй—®жқғйҷҗгҖӮ

****CVE-2025-29649пјҡTP-Link TL-WR840N SQL жіЁе…ҘжјҸжҙһ****

TP-Link TL-WR840N и·Ҝз”ұеҷЁпјҲзүҲжң¬ 1.0пјүд№ҹе®№жҳ“еҸ—еҲ°зұ»дјјзҡ„ SQL жіЁе…Ҙзјәйҷ·зҡ„еҪұе“ҚгҖӮ

зҷ»еҪ•д»ӘиЎЁжқҝжҺҘеҸ—еңЁз”ЁжҲ·еҗҚе’ҢеҜҶз Ғеӯ—ж®өдёӯжңӘз»Ҹжё…зҗҶзҡ„иҫ“е…ҘпјҢиҝҷдҪҝеҫ—жңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…иғҪеӨҹжіЁе…Ҙд»»ж„Ҹзҡ„ SQL д»Јз ҒгҖӮ

иҝҷеҸҜд»Ҙз»•иҝҮзҷ»еҪ•жҺ§еҲ¶пјҢеңЁжІЎжңүжңүж•ҲеҮӯжҚ®зҡ„жғ…еҶөдёӢи®ҝй—®и·Ҝз”ұеҷЁзҡ„з®ЎзҗҶз•ҢйқўгҖӮ

****CVE-2025-29650пјҡTP-Link M7200 4G LTE з§»еҠЁж— зәҝи·Ҝз”ұеҷЁ SQL жіЁе…ҘжјҸжҙһ****

жӯӨжјҸжҙһеҪұе“ҚиҝҗиЎҢеӣәд»¶зүҲжң¬ 1.0.7 Build 180127 Rel.55998n зҡ„ TP-Link M7200 4G LTE з§»еҠЁж— зәҝи·Ҝз”ұеҷЁгҖӮ

иҜҘи®ҫеӨҮзҡ„зҷ»еҪ•з•ҢйқўжңӘиғҪжӯЈзЎ®жё…зҗҶз”ЁжҲ·еҗҚе’ҢеҜҶз Ғеӯ—ж®өдёӯзҡ„иҫ“е…ҘпјҢдҪҝеҫ—жңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…иғҪеӨҹжіЁе…ҘжҒ¶ж„Ҹзҡ„ SQL иҜӯеҸҘгҖӮ

еҲ©з”ЁиҜҘжјҸжҙһеҸҜиғҪдјҡеҜјиҮҙеҜ№и·Ҝз”ұеҷЁз®ЎзҗҶжҺ§еҲ¶еҸ°зҡ„жңӘз»ҸжҺҲжқғи®ҝй—®гҖӮ

****CVE-2025-29653пјҡTP-Link M7450 4G LTE з§»еҠЁж— зәҝи·Ҝз”ұеҷЁ SQL жіЁе…ҘжјҸжҙһ****

TP-Link M7450 4G LTE з§»еҠЁж— зәҝи·Ҝз”ұеҷЁпјҢзү№еҲ«жҳҜеӣәд»¶зүҲжң¬ 1.0.2 Build 170306 Rel.1015nпјҢе…¶зҷ»еҪ•йЎөйқўдёҠзҡ„з”ЁжҲ·еҗҚе’ҢеҜҶз Ғеӯ—ж®өе®№жҳ“еҸ—еҲ° SQL жіЁе…Ҙж”»еҮ»гҖӮ

жңӘз»Ҹиә«д»ҪйӘҢиҜҒзҡ„ж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁжӯӨзјәйҷ·жіЁе…Ҙд»»ж„Ҹзҡ„ SQL е‘Ҫд»ӨпјҢиҝҷжңүеҸҜиғҪеҜјиҮҙеҜ№и®ҫеӨҮз®ЎзҗҶеҠҹиғҪзҡ„е®Ңе…ЁжҺ§еҲ¶гҖӮ

иҝҷдәӣжјҸжҙһеҜ№е®¶еәӯз”ЁжҲ·е’ҢдјҒдёҡз”ЁжҲ·йғҪжһ„жҲҗдәҶйҮҚеӨ§зҡ„е®үе…ЁйЈҺйҷ©гҖӮдёҖж¬ЎжҲҗеҠҹзҡ„ж”»еҮ»еҸҜиғҪдјҡи®©ж”»еҮ»иҖ…е®Ңе…ЁжҺ§еҲ¶еҸ—еҪұе“Қзҡ„и·Ҝз”ұеҷЁпјҢд»–д»¬жңүеҸҜиғҪдјҡпјҡ

1.зӣ‘жҺ§е’ҢжӢҰжҲӘзҪ‘з»ңжөҒйҮҸгҖӮ

2.дҝ®ж”№ DNS и®ҫзҪ®пјҢе°Ҷз”ЁжҲ·йҮҚе®ҡеҗ‘еҲ°жҒ¶ж„ҸзҪ‘з«ҷгҖӮ

3.е°Ҷиў«ж”»йҷ·зҡ„и·Ҝз”ұеҷЁз”ЁдҪңж”»еҮ»е…¶д»–и®ҫеӨҮзҡ„и·іжқҝгҖӮ

4.и®ҝй—®йҖҡиҝҮзҪ‘з»ңдј иҫ“зҡ„ж•Ҹж„ҹдҝЎжҒҜгҖӮ

TP-Link е·Іиў«е‘ҠзҹҘиҝҷдәӣжјҸжҙһпјҢдҪҶжҲӘиҮіжң¬ж–ҮеҸ‘еёғж—¶пјҢе°ҡжңӘзЎ®и®Өжңүе®үе…ЁиЎҘдёҒеҸ‘еёғгҖӮ

еңЁе®ҳж–№дҝ®еӨҚзЁӢеәҸеҸҜз”Ёд№ӢеүҚпјҢе®үе…Ёдё“е®¶е»әи®®йҮҮеҸ–д»ҘдёӢйў„йҳІжҺӘж–Ҫпјҡ

1.жӣҙж”№й»ҳи®Өзҡ„з®ЎзҗҶе‘ҳеҮӯжҚ®гҖӮ

2.зҰҒз”ЁиҝңзЁӢз®ЎзҗҶгҖӮ

3.зЎ®дҝқи·Ҝз”ұеҷЁеӣәд»¶жӣҙж–°еҲ°жңҖж–°еҸҜз”ЁзүҲжң¬гҖӮ

4.иҖғиҷ‘дҪҝз”ЁзҪ‘з»ңзӣ‘жҺ§е·Ҙе…·жқҘжЈҖжөӢејӮеёёжҙ»еҠЁгҖӮ

еҸ—еҪұе“Қи®ҫеӨҮзҡ„з”ЁжҲ·еә”е…іжіЁ TP-Link зҡ„е®үе…Ёе…¬е‘ҠпјҢд»ҘиҺ·еҸ–еҚіе°ҶеҸ‘еёғзҡ„иЎҘдёҒпјҢе№¶з«ӢеҚійҮҮеҸ–жҺЁиҚҗзҡ„е®үе…ЁжҺӘж–ҪгҖӮ
