[0 comment(s)](/diary/Quickie%2BCyberChef%2BSorting%2BBy%2BString%2BLength/29328/#comments)

I like to sort the output of the strings command by string length, so that the longest strings (and potentially more interesting) appear at the end of the command output.

[CyberChef](https://gchq.github.io/CyberChef/) included a small PR I made to add sorting by length to the sort operation:

![](https://isc.sans.edu/diaryimages/images/20221211-210205.png)

And I have a couple new operations in the pipeline.
