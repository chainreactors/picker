On behalf of the team, it is my pleasure to announce the release of the very first Sales Cloud Digital First course:

**C4H41B - [Discovering SAP Sales Cloud](https://saplearninghub.plateau.com/learning/user/common/viewItemDetails.do?OWASP_CSRFTOKEN=IULD-OTOY-52YV-PCDJ-3BIS-X8HI-TRRS-GJMR&componentID=C4H41B_EN_Col2205&componentTypeID=E-Learning&fromSF=Y&revisionDate=1357992000000&menuGroup=Learning&menuItem=Cur&fromDeepLink=true&hideItemDetailsBackLink=true)**

#### ![](/legacyfs/online/storage/blog_attachments/2020/10/course_highlights.png)

* + The following course has been created based on the 2205 release.In this course you will learn how to use the different features and functions of SAP Sales Cloud. The topics are explained with interactive animations, videos and quizzes. This is a beginner, foundational level course.

    The course covers topics such as :

    - Navigating the user interface

    - Features of Leads, opportunities, quotes , sales orders

    - Activity management and site visits

    - Sales planning and forecasting

    - Pricing

    - Intro to Add ons in SAP Sales Cloud

    - Utilizing reports and analytics

This course is available on SAP [Learning Hub](https://learninghub.sap.com/?languagePrompt=true), which requires a subscription . Here you can do a simple search on the course code (C4H41B) to get to the course.
