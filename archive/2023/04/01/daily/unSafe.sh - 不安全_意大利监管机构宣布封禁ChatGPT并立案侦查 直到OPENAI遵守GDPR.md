意大利隐私监管机构今天下午宣布以涉嫌侵犯隐私为由、封禁流行的人工智能对话应用 ChatGPT，该应用由没有人工智能科技公司 OPENAI 开发。

这次禁令属于临时性的，但具体封禁到什么时候暂时还不清楚，意大利隐私监管机构称将对 OPENAI 立案调查。

[![意大利监管机构宣布封禁ChatGPT并立案侦查 直到OPENAI遵守GDPR](https://img.lancdn.com/landian/2023/03/98055.png)](https://img.lancdn.com/landian/2023/03/98055.png)

封禁 ChatGPT 和进行立案调查的原因主要是意大利隐私监管机构认为 OPENAI 没有遵守欧盟通用数据保护条例 (GDPR) 处理意大利用户的数据，例如监管机构认为 OPENAI 缺乏法律依据来证明大量收集和存储个人数据以训练 ChatGPT 的算法 (即没有证明这是必要的)，同时监管机构还认为 OPENAI 不准确的处理用户数据。

意大利监管机构还提到 ChatGPT 上周出现的数据泄露问题，当时 ChatGPT 由于系统问题不慎暴露部分用户的聊天会话标题和少部分用户的支付数据。

还有个关键问题可能会引起欧盟其他成员国的共同关注：ChatGPT 注册账号时不需要验证年龄，即未成年人也可以注册使用，而 ChatGPT 提供内容没有经过分级处理，可能会向未成年人提供不适合的内容。

在欧盟有关未成年人数据保护是重点关注对象，之前 Microsoft 365 就因为收集学生用户数据并传输到美国遭到德国黑森州数据监管机构的禁用。

OPENAI 在欧盟没有办事处，但在欧洲经济区有专门的代表，这个代表将有 20 天向监管机构传达该公司怎么样让 ChatGPT 符合 GDPR 规定保护用户隐私。

如果超期未答复或未进行整改的话，那么 OPENAI 可能会被欧盟处以 2000 万欧元或全球营收 4% 的巨额罚款，以最高额为准，目前 OPENAI 尚未就此事发布回应。
