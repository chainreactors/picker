# PiPiName

根据三才五格，从诗经、楚辞、论语、周易、唐诗、宋词给宝宝取名。

相关阅读可以[看这里](https://juejin.cn/post/6868186071260856334)。

# 使用方法

## 1. 安装第三方库

安装简繁体转换库 OpenCC：

```
pip install opencc
```

打开 config.py，进行参数配置：
