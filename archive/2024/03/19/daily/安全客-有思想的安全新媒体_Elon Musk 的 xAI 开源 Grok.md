# Elon Musk 的 xAI 开源 Grok

xAI 聊天机器人 Grok 的基本模型现已在 GitHub 上发布，任何人都可以在自己的项目中免费使用。

该模型是在 Apache 2.0 许可证下发布的，该许可证允许将 Grok 背后的底层技术用于商业用途。但是，该版本不包括用于模型的训练代码。

“我们正在发布大型语言模型 Grok-1 的基本模型权重和网络架构。Grok-1 是一个由 xAI 从头开始训练的 3140 亿参数混合专家模型，”该公司在博客文章中表示。

该公司表示，“原始”基础模型来自 Grok 的预训练阶段，该阶段于去年 10 月结束。“这意味着该模型没有针对任何特定应用进行微调，例如对话，”xAI 说。

亿万富翁埃隆·马斯克 (Elon Musk) 上周宣布，他将开源他的 ChatGPT 竞争对手，就在起诉他帮助创建的 OpenAI 几天后。马斯克指责这家微软支持的公司及其首席执行官萨姆·奥尔特曼放弃了最初的非营利使命，转而采用赚钱模式。

OpenAI 通过发布马斯克的旧电子邮件来回应诉讼，其中他似乎主张营利性设置，甚至建议该合资企业应与特斯拉合并，以便成为其“摇钱树”。

xAI 去年向 X Premium+ 订阅者发布了 Grok。其他开源部分人工智能模型的公司包括 Meta、Mistral 和 Google。
