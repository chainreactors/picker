## ChatGPT 的错误答案给 OpenCage 短暂增加新用户

[![Bug](https://icon.solidot.org/images/topics/topicbug.png?123)](/search?tid=53 "Bug")

[Wilson](/~Wilson) (42865)发表于 2023年02月24日 12时23分 星期五 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74222&appkey=1370085986&title=ChatGPT%20%E7%9A%84%E9%94%99%E8%AF%AF%E7%AD%94%E6%A1%88%E7%BB%99%20OpenCage%20%E7%9F%AD%E6%9A%82%E5%A2%9E%E5%8A%A0%E6%96%B0%E7%94%A8%E6%88%B7 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自冲破黎明**

提供地理编码 API 的 OpenCage 项目披露 AI 聊天机器人 ChatGPT 的错误答案给它短暂增加了新用户，但很快又失去了这些用户。*OpenCage 发现有很多新用户在短暂试用 API 后停止了使用，它认为需要展开调查，结果发现原因是 ChatGPT 错误的推荐。ChatGPT 告诉其用户 OpenCage 提供了电话号码转换到地理位置的服务，然而用户使用后发现根本没有该功能。它认为 ChatGPT 的信息来自于一则错误的 YouTube 视频教程，OpenCage 以前为此澄清过，它现在希望用户不要对 ChatGPT 的答案深信不疑，OpenCage 并没有提供电话位置的查询服务。*
https://blog.opencagedata.com/post/dont-believe-chatgpt
