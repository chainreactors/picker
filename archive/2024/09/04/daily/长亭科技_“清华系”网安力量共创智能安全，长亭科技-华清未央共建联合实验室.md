# “清华系”网安力量共创智能安全，长亭科技-华清未央共建联合实验室

长亭科技

![](https://mmbiz.qpic.cn/sz_mmbiz_gif/Fuleibl6qMuqjgDBtJiaQXYQ0GeUy94hZmXGMrH7ibjalralgxnASkVrKVZh1bH4hWHnGXKLPzw1eruVJaqIGMRuw/640?from=appmsg)

2024年9月3日，长亭科技与华清未央在北京市中关村国家自主创新示范区会议中心举行了联合实验室的签约仪式。在签约仪式现场，**长亭科技高级合伙人、战略发展副总裁余慧英，华清未央CEO朱文宇分别代表长亭科技和华清未央签署了合作协议。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/Fuleibl6qMupm4OiaTwSbaCPka9jhqIMiaiarqm1fwy94Ou51Oul7yiaBMors9b2fTXj2bibAQhH0AcRGCRIZVNYHVMA/640?wx_fmt=png&from=appmsg)

**双方宣布，将发挥各自优势，进一步深化合作，建立长亭科技—华清未央联合实验室（以下简称“联合实验室”），共同研发web安全大模型，推动大模型在网络安全领域的技术进步和落地应用。**双方期待通过共同努力，为安全行业的发展贡献力量，共同构建更加安全、智能的未来。

据介绍，联合实验室的宗旨为重点致力于研发网络安全领域的垂直大模型应用。此次合作的Web安全大模型，旨在通过技术创新，提升安全事件的识别和响应速度，在应对高级持续性威胁（APT）、处理海量数据、自动化实时响应、预测性防御及个性化安全策略制定等方面取得明显成效。长亭科技将提供专业的安全大模型能力和高质量实战攻防数据，助力华清未央训练出可用于实战的Web安全大模型。

余慧英表示，通过此次与华清未央合作，长亭科技将在长期一线实践中的先进实战化攻防能力、安全大模型能力与华清未央的大模型技术相结合，确保Web安全大模型真正贴合当前网络安全的实战化需求，实现大模型在该领域的成功应用，为安全行业带来革命性的变革。

朱文宇对双方的合作前景表示看好，他表示，华清未央很早就开展AI for Security方向的科技攻关，此次通过与长亭科技的紧密合作，将进一步拓展大模型在安全领域的应用场景，将大模型技术的优势发挥到极致，为安全行业提供更加精准和高效的大模型底座。

**长亭科技**

*知攻善防，智能安全*

作为一家发端于网络攻防实战赛场，创始团队来自我国知名的清华“蓝莲花”CTF战队的新生代网络安全领军企业，长亭科技早在2016年便全球首创了具有独立自主知识产权的智能语义分析技术，多年来在智能攻防能力、网络安全未知威胁发现与处置等方面，具有丰富经验。

当前，公司已拥有完备的行业资质认证，建设有覆盖全国三十多个省市的服务网络和2000多家合作伙伴的生态体系，全面适配国家自主可控生态。依托知攻善防的实战攻防能力，长亭科技构筑起以多款自研产品为核心的新一代智能安全防护体系，并通过专业的安全测试及咨询服务，帮助客户建设全方位、领先的实战化、体系化、常态化的智能安全运营能力。十年间，长亭科技为来自金融、通信、政府、能源、互联网等重要行业的3000多家用户提供了可靠有效的安全解决方案，具有优质的行业资源优势。这为联合实验室的技术、资源保障以及业务支持，均提供了坚实的保障。

**华清未央**

*机器理解机器，机器服务机器*

本次联合实验室的合作方华清未央，成立于2023年，为全球首家机器语言大模型供应商，是一家主要从事科技推广和应用服务业的“清华系”新锐企业。

华清未央秉持让网络空间更美好的愿景，专注于突破人工智能赋能软件和安全的技术，是国内外最早开展Al for Software （AI赋能软件）和Al for Security （AI赋能安全）研究的团队之一。公司自主研发的机器语言大模型MLM (Machine Language Model)，首次全面支持对多指令集、多操作系统、多文件格式的可执行程序/二进制程序/闭源软件进行结构分析、语义理解、安全分析，不依赖软件源代码或调试信息，解决二进制程序信息缺失和语义理解困难等难题，突破软件分析技术多年来面临的瓶颈，于多个关键技术群实现零的突破，在全球市场尚无竞品。

当前，产业数字化转型与行业智能化进程不断深入，网络安全垂直领域大模型为网络安全与AI产业的融通和落地化建设，提供了新的机遇。以本次联合实验室的建立为契机，长亭科技将与华清未央将继续秉持清华学子的实干精神，深入打造耦合网络安全一线业务的实用型web安全大模型，把握时代脉络，深耕智能安全，以自强不息的拼搏精神与创新力琢铸产品与服务，以AI驱动企业安全运营迈入智能化时代。
