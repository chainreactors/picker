# 丰田再次出现数据泄露事件

#

近日，丰田公司遭遇黑客攻击，导致大约240G敏感信息泄露。该事件由名为ZeroSevenGroup的黑客组织发起，该组织声称成功入侵了丰田美国的一个分支机构，并窃取了大量涉及员工、客户、财务及网络基础设施的信息，据悉，此前丰田公司也曾发生过多起数据泄露事件。目前，丰田称此次事件的影响范围有限，未波及整个系统，同时表示将与已受影响的相关方取得联系，在必要时提供帮助。

# 男子伪造死亡证明逃避子女抚养费

#

近日，一男子通过盗取医生的登录凭证访问夏威夷死亡登记系统，成功伪造了自己的死亡证明，逃避其未支付的子女抚养费。此外，该男子还利用盗取的凭证非法登录其他州的死亡登记系统、访问政府和私人企业的网络，并在暗网出售相关信息。目前，该男子以计算机欺诈罪和严重身份盗窃罪被判处81个月有期徒刑，其对政府和企业计算机系统造成的损害以及未能支付的子女抚养费共计195758.65美元。

# 美国知名建筑公司数据被泄露

#

近期，美国知名建筑设计公司CannonDesign向其13,000多名在职及离职员工发送了数据泄露通知。据悉，该公司于2023年初发现入侵行为，2024年5月结束调查工作，调查结果显示，攻击者可能获取了包括姓名、地址、社会安全号码等在内的个人信息。为了降低个人数据泄露的风险，受影响的用户将获得Experian提供的为期24个月的信用监控服务。CannonDesign没有明确指出攻击者的身份，但其发言人表示，此次披露的信息与2023年初发生的Avos Locker勒索软件攻击有关。

# 三人制售游戏外挂被刑拘

#

据公安部网安局报道，江西省鹰潭市网安部门成功破获一起非法售卖游戏外挂的网络犯罪案件。经调查，该犯罪团伙由3名年轻人组成，三人共同谋划制作游戏外挂，并通过建立稳定的客户群体和多个千人规模的QQ群，利用卡网销售的方式，自动分发“永久卡”、“周卡”、“天卡”等产品牟取利益。经公安机关辗转多地侦查取证，三名犯罪嫌疑人目前均已被依法刑事拘留。

# 黑客攻击俄勒冈州动物园网站盗取超10万人信息

#

近期，俄勒冈动物园发生数据泄露事件，泄露的数据包括姓名、支付卡号、CVV等，共有117815人受到此次事件影响。动物园方表示将为受到影响的用户提供一年的免费信用监控和身份保护服务。据悉，俄勒冈州动物园是继多伦多动物园和坦帕湾动物园后最新面临网络犯罪分子攻击的动物园。目前，该动物园已向俄勒冈州和德克萨斯州的监管机构提交了违规通知。
