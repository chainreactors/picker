# Sessions zur SAP Integration Suite

## [Programm] Mittwoch, 23.03.2023

**V001: SAP Integration Suite – Status-Update, neueste Entwicklungen, Roadmap & Application Integration**

Datum: Mittwoch, 22 März 2023
Uhrzeit: 13:00 - 13:45
Kategorie: Application Integration; Roadmap

Beschreibung:
SAP Integration Suite – Status-Update, neueste Entwicklungen, Roadmap & Application Integration

Sprecher: Udo Paltzer, SAP

**V002: Globaler Digital Integration Hub für Produkt- und Katalogdaten auf Basis von SAP BTP @ Miele**

Datum: Mittwoch, 22 März 2023
Uhrzeit: 14:15 - 15:00
Kategorie: Application Integration; Erfahrungsbericht

Beschreibung:
Der Vortrag gibt einen Überblick über die Implementierung eines zentralen Integration Hubs für die Transformation und Verteilung von Produkt- und Katalogdaten bei Miele. Die Umsetzung erfolgte auf einem API-basierten Ansatz und einen zentralen, standardisierten Produktdatenmodell auf Basis von SAP BTP. Die globale Lösung umfasst über 59 internationale Vertriebsgesellschaften mit Verteilung von über 350 Publikationen an interne und externe Webshops, Händler als auch direkte API-basierten Zugriff aus mobilen Apps. Die Datentransformation von täglich über 100 GB Katalogdaten erfolgte über SAP Cloud Integration, welche über OData APIs und das Cloud Application Programming Model (CAP) auf die In-Memory-Datenbank auf Basis von SAP HANA Cloud zugreift. Der Vortrag erläutert die Erfahrungen aus der Implementierung und Betrieb einer komplexen, hochvolumigen und hochintegrierten Cloud-Anwendung.

Sprecher:
Manuel Männle, Miele & Cie. KG, Enterprise Architect
Holger Himmelmann, cbs Corporate Business Solutions GmbH, Consulting Director
Jonas Guntermann-Schandelle, Miele & Cie. KG

SAP Integration Suite – SAPs hybride Integrationsplattform für die Integration von SAP- und Nicht-SAP-Systemen: Was wird heute unterstützt und welche Anforderungen hätten Sie?

Milena Mathyl, SAP
