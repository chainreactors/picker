![Vulnhuntr]()

еңЁеҪ“д»ҠдёҚж–ӯеҸҳеҢ–зҡ„зҪ‘з»ңе®үе…ЁзҺҜеўғдёӯпјҢиҜҶеҲ«д»Јз Ғеә“дёӯзҡ„жјҸжҙһеҜ№дәҺз»ҙжҠӨиҪҜд»¶е’ҢеҹәзЎҖи®ҫж–Ҫзҡ„е®үе…ЁиҮіе…ійҮҚиҰҒгҖӮVulnhuntr жҳҜ GitHub дёҠзҡ„дёҖж¬ҫејҖжәҗе·Ҙе…·пјҢе®ғеҲ©з”ЁеӨ§еһӢиҜӯиЁҖжЁЎеһӢпјҲLLMпјүе’ҢйқҷжҖҒд»Јз ҒеҲҶжһҗжқҘиҜҶеҲ«еҹәдәҺ Python зҡ„йЎ№зӣ®дёӯеҸҜиў«иҝңзЁӢеҲ©з”Ёзҡ„жјҸжҙһгҖӮVulnhuntr и®ҫи®Ўз®ҖжҙҒгҖҒеҠҹиғҪејәеӨ§пјҢеңЁжҷәиғҪиҮӘеҠЁеҢ–е’Ңж·ұеәҰд»Јз ҒеҲҶжһҗд№Ӣй—ҙжһ¶иө·дәҶдёҖеә§жЎҘжўҒпјҢжҳҜејҖеҸ‘дәәе‘ҳгҖҒе®үе…Ёдё“дёҡдәәе‘ҳе’ҢдјҒдёҡзҡ„е®қиҙөиө„жәҗгҖӮ

**Vulnhuntr еҰӮдҪ•е·ҘдҪң**

Vulnhuntr йҮҮз”ЁзӢ¬зү№зҡ„еӨҡйҳ¶ж®өж–№жі•иҝӣиЎҢжјҸжҙһжЈҖжөӢпјҡ

1. **LLM й©ұеҠЁзҡ„ README еҲҶжһҗпјҡ** LLM йҰ–е…ҲеҲҶжһҗйЎ№зӣ®зҡ„ README ж–Үд»¶пјҢдәҶи§Јд»Јз Ғеә“зҡ„еҠҹиғҪе’ҢжҪңеңЁжјҸжҙһгҖӮиҝҷдәӣдҝЎжҒҜе°Ҷз”ЁдәҺжҢҮеҜјеҗҺз»ӯеҲҶжһҗгҖӮ
2. **еҲқе§Ӣд»Јз Ғжү«жҸҸпјҡ** LLM еҜ№ж•ҙдёӘд»Јз Ғеә“иҝӣиЎҢеҲқжӯҘжү«жҸҸпјҢж №жҚ®е…¶еҜ№е®үе…Ёзј–з Ғе®һи·өе’Ңеёёи§ҒжјҸжҙһжЁЎејҸзҡ„зҗҶи§Јж Үи®°жҪңеңЁжјҸжҙһгҖӮ
3. **дёҠдёӢж–Үж·ұеәҰжҢ–жҺҳпјҡ** еҜ№дәҺжҜҸдёӘжҪңеңЁжјҸжҙһпјҢVulnhuntr йғҪдјҡеҗ‘ LLM жҸҗдҫӣзү№е®ҡзҡ„жјҸжҙһжҸҗзӨәпјҢд»ҺиҖҢи§ҰеҸ‘жӣҙж·ұе…Ҙзҡ„еҲҶжһҗгҖӮLLM дјҡд»Һзӣёе…іж–Үд»¶дёӯжҷәиғҪең°иҜ·жұӮйўқеӨ–зҡ„дёҠдёӢж–ҮпјҢиҝҪиёӘд»Һз”ЁжҲ·иҫ“е…ҘеҲ°жңҚеҠЎеҷЁз«ҜеӨ„зҗҶзҡ„ж•°жҚ®жөҒгҖӮиҝҷж ·пјҢе®ғе°ұиғҪиҜҶеҲ«и·Ёи¶ҠеӨҡдёӘж–Үд»¶е’ҢеҠҹиғҪзҡ„жјҸжҙһгҖӮ
4. **е…Ёйқўзҡ„жјҸжҙһжҠҘе‘Ҡпјҡ** Vulnhuntr дјҡз”ҹжҲҗдёҖд»ҪиҜҰз»ҶжҠҘе‘ҠпјҢжҰӮиҝ°е…¶еҸ‘зҺ°гҖӮиҜҘжҠҘе‘ҠеҢ…жӢ¬

* жҜҸдёӘж–Үд»¶зҡ„еҲқе§ӢиҜ„дј°з»“жһң
* еҢ…еҗ«дёҠдёӢж–ҮеҮҪж•°е’Ңзұ»еј•з”Ёзҡ„дәҢж¬ЎиҜ„дј°з»“жһң
* жҜҸдёӘжјҸжҙһзҡ„еҸҜдҝЎеәҰеҲҶж•°
* еҲҶжһҗиҝҮзЁӢж—Ҙеҝ—
* еҜ№е·ІйӘҢиҜҒжјҸжҙһзҡ„жҰӮеҝөйӘҢиҜҒ (PoC) еҲ©з”Ё

**еә“дёӯеҸ‘зҺ°зҡ„жјҸжҙһзӨәдҫӢ**

еңЁжңҖиҝ‘зҡ„жү«жҸҸдёӯпјҢVulnhuntr еҸ‘зҺ°дәҶеӨҡдёӘзҹҘеҗҚйЎ№зӣ®дёӯзҡ„жјҸжҙһпјҢеұ•зӨәдәҶе…¶жңүж•ҲжҖ§пјҡ

* **gpt\_academic (64k stars)пјҡ** LFIгҖҒXSS
* **ComfyUIпјҲ5 дёҮйў—жҳҹпјүпјҡ** XSS
* **FastChatпјҲ3.5 дёҮйў—жҳҹпјүпјҡ** SSRF
* **REDACTED (29k stars)пјҡ** RCEгҖҒIDOR
* **Ragflow (16k stars)пјҡ** RCE

иҝҷдәӣеҸ‘зҺ°иҜҙжҳҺдәҶ Vulnhuntr иғҪеӨҹжЈҖжөӢзҡ„жјҸжҙһзұ»еһӢзҡ„еӨҡж ·жҖ§пјҢд»ҺеӯҰжңҜз ”з©¶е·Ҙе…·дёӯзҡ„ LFI еҲ°жңәеҷЁеӯҰд№ йЎ№зӣ®дёӯзҡ„ RCEгҖӮ

**еұҖйҷҗжҖ§**

иҷҪз„¶ Vulnhuntr д»ЈиЎЁдәҶжјҸжҙһжү«жҸҸйўҶеҹҹзҡ„дёҖеӨ§иҝӣжӯҘпјҢдҪҶе®ғд№ҹжңүдёҖдәӣеұҖйҷҗжҖ§пјҡ

* **ж”ҜжҢҒ Pythonпјҡ** зӣ®еүҚпјҢиҜҘе·Ҙе…·д»…ж”ҜжҢҒ Python д»Јз Ғеә“гҖӮ
* **жјҸжҙһзұ»еҲ«пјҡ** Vulnhuntr еҸҜиҜҶеҲ«дёҖз»„зү№е®ҡзҡ„жјҸжҙһзұ»еҲ«пјҢеҢ…жӢ¬ LFIгҖҒAFOгҖҒRCEгҖҒXSSгҖҒSQLIгҖҒSSRF е’Ң IDORгҖӮ

**з»“и®ә**

йҖҡиҝҮе°Ҷ LLM дёҺйқҷжҖҒд»Јз ҒеҲҶжһҗзӣёз»“еҗҲпјҢVulnhuntr дёәжјҸжҙһжЈҖжөӢеёҰжқҘдәҶдёҖз§Қе…Ёж–°зҡ„ж–№жі•пјҢж—ўиғҪжҸҗдҫӣй«ҳзә§еҲҶжһҗпјҢеҸҲиғҪжҸҗдҫӣж·ұе…Ҙзҡ„и§Ғи§ЈгҖӮVulnhuntr иғҪеӨҹд»Һд»Јз Ғеә“зҡ„зӣёе…ійғЁеҲҶеҠЁжҖҒиҜ·жұӮдёҠдёӢж–ҮпјҢзЎ®дҝқе…ЁйқўиҰҶзӣ–пјҢиҖҢе…¶жңҖз»ҲеҲҶжһҗдёҺ PoC жјҸжҙһеҲ©з”Ёе’ҢзҪ®дҝЎеәҰиҜ„еҲҶдёәејҖеҸ‘дәәе‘ҳе’Ңе®үе…ЁеӣўйҳҹжҸҗдҫӣдәҶеҸҜж“ҚдҪңзҡ„дҝЎжҒҜгҖӮ
