## API-2 У║ФС╗йУ«цУ»Ђтц▒ТЋѕ(Broken Authentication)

тюе2023-RCуЅѕуџёТЏ┤Тќ░СИГ№╝їсђїAPI-2 ућеТѕиУ║ФС╗йУ«цУ»Ђтц▒ТЋѕсђЇ(Broken User Authentication)УбФС┐«Тћ╣СИ║С║єТЏ┤т╣┐С╣ЅуџёсђїУ║ФС╗йУ«цУ»Ђтц▒ТЋѕсђЇ(Broken Authentication)№╝їТїЄтЄ║APIуџёУ«┐жЌ«УђЁТЌаУ«║Тў»т«ъС║║У┐ўТў»уеІт║Ј№╝їжЃйжюђУдЂт»╣тЁХУ┐ЏУАїУ║ФС╗йТаАжфїсђѓ

тюет«ъУихСИГ№╝їтЁ│С║јAPIуџёУ«цУ»ЂТю║тѕХтњїУ║ФС╗йт«џС╣ЅСИђУѕгСИјСИџтіАтю║ТЎ»уЏИтЁ│№╝їт┐ФжђЪУ┐ГС╗БуџёAPIС┐ЮТїЂУ«цУ»ЂТю║тѕХуџётіеТђЂТђДтњїТюЅТЋѕТђДТў»СИђСИфТїЉТѕўсђѓСИЇтљѕуљєуџёУ«цУ»ЂТю║тѕХС╝џт»╝УЄ┤ТюфТјѕТЮЃУ«┐жЌ«жЌ«жбў№╝їС╗ЦтЈіУ«цУ»ЂУбФу╗ЋУ┐Є№╝ЏУђїУ┐Єт║дтцЇТЮѓуџёУ«цУ»ЂТю║тѕХС╝џтй▒тЊЇућеТѕиуџёСй┐ућеТЋѕујЄС╗ЦтЈіСИІТИИт»╣ТјЦт║ћућеуеІт║Јуџёу╗┤ТіцТѕљТюгсђѓ

**У«ЙУ«АУ«цУ»ЂТю║тѕХТЌХт║ћУђЃУЎЉС╗ЦСИІтєЁт«╣№╝џ**

Рђб Сй┐ућеТаЄтЄєтїќуџёУ«цУ»ЂТю║тѕХ№╝ѕтдѓJWTсђЂOAuthуГЅ№╝Ѕ№╝їт╣ХтЈѓУђЃтЁХТюђСй│т«ЅтЁет«ъУихсђѓжЂ┐тЁЇСй┐ућеBasic AuthуГЅу«ђтЇЋУ«цУ»ЂТЅІТ«хсђѓ

Рђб тї║тѕєућеТѕиУ║ФС╗йСИјт«бТѕиуФ»У║ФС╗й№╝їућеТѕитюеУ«цУ»ЂУ║ФС╗йтљј№╝їтіеТђЂУјитЈќAPIуџёУ«┐жЌ«С╗цуЅїсђѓ

Рђб УђЃУЎЉТџ┤тіЏуа┤УДБсђЂУ┤дтЈиТъџСИЙсђЂжђЪујЄжЎљтѕХтњїжћЂт«џС┐ЮТіцсђѓ

## API-1 т»╣У▒Ау║ДтѕФТјѕТЮЃтц▒ТЋѕ(Broken Object Level Authorization)

тЇ│Т░┤т╣│УХіТЮЃ№╝їС╗ј2019уЅѕтѕ░2023уЅѕ№╝їт»╣У▒Ау║ДтѕФТјѕТЮЃтц▒ТЋѕСИђуЏ┤УбФуй«С║јAPIжБјжЎЕуџёждќСйЇсђѓсђїт»╣У▒Ау║ДтѕФсђЇТїЄуџёТў»жђџУ┐ЄтЈѓТЋ░ТјДтѕХуџёУ«┐жЌ«т»╣У▒А№╝їтдѓжњѕт»╣GET /system/{user}/info/ТјЦтЈБуџёжђџС┐АСИГ№╝їу│╗у╗ЪУ«ЙУ«АСИіТ»ЈСИфућеТѕитЈфУЃйУ«┐жЌ«УЄфти▒уџётєЁт«╣№╝їСйєтдѓТъюТјѕТЮЃСИЇтйЊ№╝їТћ╗тЄ╗УђЁтЈ»С╗ЦжЂЇтјє{user}тЈѓТЋ░№╝їУХіТЮЃТЅ╣жЄЈУ«┐жЌ«С╗ќС║║ТЋ░ТЇ«сђѓ

У┐ЎСИфжБјжЎЕт«џС╣ЅСИјУ║ФС╗йУ«цУ»ЂСИЇтљї№╝їтЇ│Сй┐тюеу╗ЈУ┐ЄУ║ФС╗йУ«цУ»ЂуџёућеТѕи(user-1)С╣ЪтЈ»С╗ЦтѕЕућеТГцТ╝ЈТ┤ъУ«┐жЌ«(user-2)уџёТЋ░ТЇ«сђѓТГцжБјжЎЕуџёжў▓УїЃуѓ╣СИ╗УдЂтюеС║јтіЪУЃй-тцџуДЪТѕиуџёТЮЃжЎљУ«ЙУ«АСИјС╗БуаЂт«ъуј░сђѓ

Рђб тюетіЪУЃйУ«ЙУ«АтЈіжюђТ▒ѓУ»ёт«АУ┐ЄуеІуј»Уіѓт░єтцџуДЪТѕиУ«ЙУ«АСИјтіЪУЃйТЮЃжЎљУ«ЙУ«АУъЇтљѕ№╝їт╣ХтЈѓУђЃт«ЅтЁетЏбжўЪТёЈУДЂсђѓ

Рђб Сй┐ућежџЈТю║тђ╝сђЂСИЇтЈ»жбёТхІуџётђ╝СйюСИ║ТЪЦУ»бТЮАС╗Х№╝їжў▓ТГбУбФуїюТхІТѕќТъџСИЙсђѓ

Рђб жњѕт»╣тіЪУЃйтњїтцџуДЪТѕиТЮЃжЎљУ«ЙУ«А№╝їт║ћу╝ќтєЎт»╣т║ћуџёТхІУ»ЋућеСЙІУДёжЂ┐ТГцжБјжЎЕ№╝їт╣Хт╝ЋтЁЦт«ЅтЁеТБђТЪЦсђѓ

Рђб жњѕт»╣СИЇтљїуДЪТѕиуџёСй┐ућеУАїСИ║тіаС╗ЦуЏЉТјД№╝їтЈЉуј░ТюфУбФУДѓТхІтѕ░уџёТ░┤т╣│УХіТЮЃУАїСИ║сђѓ

## API-5 тіЪУЃйу║ДтѕФТјѕТЮЃтц▒ТЋѕ(Broken Function Level Authorization)

┬а

тЇ│тъѓуЏ┤УХіТЮЃ№╝їТГцжА╣тюе2023уЅѕТ▓АТюЅтЈўтіесђѓСЙІтдѓТЎ«жђџућеТѕитЈ»С╗ЦУ«┐жЌ«у«АуљєтЉўжЃетѕєтіЪУЃйAPIсђЂжњѕт»╣RestFulжБјТа╝ТјЦтЈБТЪљС║ЏтЈфУ»╗ућеТѕи(GET)тЈ»С╗ЦУХіТЮЃУ┐ЏУАїТЋ░ТЇ«ТЊЇСйю(UPDATE, DELETE)уГЅсђѓ

┬а

тЁ│С║јТГцжЌ«жбўуџёжў▓Тіцуѓ╣СИ╗УдЂтюеС║ју│╗у╗ЪуџётіЪУЃйСИјућеТѕиУДњУЅ▓уџёС║цтЈЅУ«ЙУ«АтЈіС╗БуаЂт«ъуј░сђѓ

┬а

Рђб жњѕт»╣т║ћућеу│╗у╗Ът«їТЋ┤уџёТб│уљєAPIУхёС║Дт▒ѓТгАу╗ЊТъё№╝їт╣Хжњѕт»╣СИЇтљїAPIсђЂСИЇтљїућеТѕиУ┐ЏУАїТЮЃжЎљуџёС║цтЈЅТБђТЪЦсђѓ

Рђб уА«С┐ЮТЅђТюЅуџёУ«┐жЌ«УАїСИ║жЃйУбФу║│тЁЦућеТѕиУ║ФС╗йУ«ЙУ«АС╣ІСИГ№╝їудЂТГбТюфТјѕТЮЃУ«┐жЌ«сђѓ
