# Update: oledump.py Version 0.0.71

# [Didier Stevens](https://blog.didierstevens.com/)

## Friday 11 November 2022

### Update: oledump.py Version 0.0.71

Filed under: [Uncategorized](https://blog.didierstevens.com/category/uncategorized/) — Didier Stevens @ 0:00

A new plugin and an updated plugin.

Plugin plugin\_dttm is a plugin for Word documents: it searches for [Dop](https://learn.microsoft.com/en-us/openspecs/office_file_formats/ms-doc/841b5f72-487e-4fe7-8657-ec90d5af8750) structures. They contain [DTTM](https://learn.microsoft.com/en-us/openspecs/office_file_formats/ms-doc/164c0c2e-6031-439e-88ad-69d00b69f414) timestamps.

And plugin plugin\_metadata has been updated to parse digital signatures (option -s).
