**关键词**

Windows

微软刚刚推出 Windows Server vNext Build 26280 版，由于 Windows Server 2025 RTM 版即将发布，所以这些测试版包含的新功能应该不会在首发版本中提供。

此次更新其实更重要的是微软调整时间炸弹，此前所有 Windows Server 2025 预览版的到期时间都是 2024 年 9 月 16 日。

如果用户仍然使用旧版本则会在 9 月 16 日后系统每隔 4 小时就强制弹窗提醒版本已经到期，这种弹窗提醒是无法彻底禁用的，除非更新系统。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGyliayAgNUicp8k6tBdiaTt9Ne0tbbeNaiaibcj5iaaFCJEXMpjtdKHpdBSUK8WFXxhkFSbSHrX7OAwRnusiaw/640?wx_fmt=png&from=appmsg)

而从 Build 26280 版开始接下来的所有预览版时间炸弹是到 2025 年 9 月 15 日，也就是现有的预览版可以接着用一年不会碰到时间炸弹。

有兴趣的开发者和 IT 管理员可以在这里注册 Windows Server Insider 计划并下载 ISO 镜像文件，同时配合微软公布的激活密钥激活系统。

下载地址：‍https://www.microsoft.com/en-us/software-download/windowsinsiderpreviewserver‍

Windows Server Standard 标准版激活密钥：MFY9F-XBN2F-TYFMP-CCV49-RMYVH

Windows Server Datacenter 数据中心版激活密钥：2KNJJ-33Y9H-2GXGX-KMQWH-G6H67

另因制裁相关原因，俄罗斯 IP 地址无法访问 Windows Server Insider 预览计划和下载镜像文件，其他受制裁的地区情况也是相同的。
