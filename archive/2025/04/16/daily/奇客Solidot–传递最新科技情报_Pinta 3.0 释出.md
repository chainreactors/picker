## Pinta 3.0 释出

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2025年04月15日 14时08分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=81049&appkey=1370085986&title=Pinta%203.0%20%E9%87%8A%E5%87%BA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自荷鲁斯崛起**

图像编辑器 Pinta 释出了 3.0 版本。Pinta 是 Paint.Net 3.0 的 GTK 克隆，Paint.Net 是用 C# 编写的，而 Pinta 最初是基于.NET 开源框架 Mono，现在迁移到了微软开源的 Microsoft .NET 框架。Pinta 3.0 的主要变化包括：移植到 GTK 4.0 和 libadwaita，.NET 版本最低需求提高到 8.0；恢复在 2.0 版本中因技术限制而禁用的插件支持，新的 Dithering、Voronoi Diagram、Vignette、Dents、Feather Object、Align Object 特效，等等。
github.com/PintaProject/Pinta/releases
