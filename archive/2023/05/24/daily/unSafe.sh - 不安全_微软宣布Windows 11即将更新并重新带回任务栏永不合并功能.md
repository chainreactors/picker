Windows 11 任务栏的自动合并问题已经遭到大量用户的长期吐槽，感谢这些坚持吐槽和反馈的用户，现在微软终于意识到把 Windows 11 任务栏应用图标合并是个非常糟糕的主意。

在 Build 2023 上，微软宣布即将推送更新重新带回任务栏的永不合并功能，以前这需要用户使用第三方软件才能实现，现在微软终于又重新支持了。

[![微软宣布Windows 11即将更新并重新带回任务栏永不合并功能](https://img.lancdn.com/landian/public/thumb/Windows11.png)](https://img.lancdn.com/landian/public/thumb/Windows11.png)

同时微软宣布任务栏还会改进通知区域，可选隐藏时间和日期模块，微软称新的选项可以让用户保持专注，也就是不用老盯着时间看看自己的工 (摸) 作 (鱼) 时间到了没。

可选隐藏时间和日期模块还有个好处是录制视频的时候可以更简洁，实际上用户录制视频的时候可以有意隐藏时间，当然从观众角度来说，能看到视频的实际录制时间可能并不是坏事。

时间和日期模块还有个改进是可选显示秒功能，此前微软称显示秒会导致系统某些组件无法休眠进而耗费资源降低续航，不过这也是在用户强烈要求下，微软重新支持显示秒，[用户只需要在设置里勾选即可](https://www.landiannews.com/archives/97933.html)。
