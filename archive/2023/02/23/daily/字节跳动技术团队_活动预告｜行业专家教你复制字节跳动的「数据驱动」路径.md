# 活动内容

### **议题一：从业务出发，解读字节跳动万亿级数据** **中台** **实战**

演讲嘉宾：蔡猛 火山引擎 数据产品解决方案专家

演讲概要：

从数据库、数据仓库、数据平台到数据中台，经过多年发展，数据中台也已从概念落到实处。字节跳动数据中台服务于内部海量业务场景，在实践检验下抽象和沉淀出一套解决方案，并复用到金融、泛互联网等领域。

那么，数据中台如何解决行业、企业从“有数到看数”，再从“看数到用数”的问题？本次分享将围绕以下展开：

* 揭秘承载字节跳动爆发式数据增长和业务需求的数据中台矩阵和架构
* 详细解读字节跳动数据中台内部最佳实践
* 聚焦金融领域数字化转型实践，拆解银行客户核心痛点、落地方案和最终效果

### **议题二：轻松管数，一站式** **数据治理** **助力企业数字化升级**

演讲嘉宾：陈金玉 火山引擎 DataLeap 产品专家

演讲概要：

数据中台能够帮助企业全面管理多源数据，打破数据壁垒，而数据治理是数据中台建设重要组成部分。本次分享聚焦字节跳动数据治理实践，从数据集成、开发、运维、治理、资产、安全等全链路环节，并拆解实际应用案例。本次分享将围绕以下展开：

* 大数据研发领域，从集成到治理等全环节痛点及解决方案
* 数据治理避坑指南，复盘字节跳动内部数据建设全过程
* 解读数据治理行业案例，如何帮助外部客户治理提效

### 议题三：**如何算数？****计算引擎****提速，** **ByteHouse** **实时计算能力升级**

演讲嘉宾：沈曈 火山引擎 ByteHouse 企业版产品负责人

演讲概要：

为什么字节跳动选择Clickhouse作为核心分析技术？ByteHouse在Clickhouse社区版基础上做了哪些增强、重构？随着业务数据量扩大，场景覆盖变得广泛，如何满足复杂查询和实时性要求？本次分享将围绕以下展开：

* 海量数据、复杂场景，需要企业拥有高性能、稳定、实时性强的计算引擎
* 揭秘字节跳动基于ClickHouse的性能优化和能力升级
* 如何利用ClickHosue提升决策分析效率，并应用于企业自身场景

### **议题四：高效看数、智能决策，解密企业级可视化** **BI** **平台**

演讲嘉宾：常成 火山引擎 DataWind 产品负责人

演讲概要：

在企业数字化转型的浪潮中，数据分析无疑是推动数字化转型进程的关键之一。越来越多的企业及分析师通过BI平台实现数据可视化，并辅助完成智能化决策。本次分享将围绕以下展开：

* 从有数、管数、算数到看数，字节内部数字化决策实践现状
* 企业内部实现数字化决策必备方法论和实操方案
* 详细介绍智能数字化决策场景下的新一代BI架构和能力
