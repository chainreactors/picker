SocketиӯҰе‘ҠиҜҙпјҡвҖңВ *pumptoolforvolumeandcomment дёӯзҡ„еҠ иҪҪзЁӢеәҸдјҡи§Јз Ғж··ж·Ҷзҡ„жңүж•ҲиҪҪиҚ·пјҢиҜҘиҪҪиҚ·дјҡжҗңзҙў Base58 зј–з Ғзҡ„еҠ еҜҶиҙ§еёҒеҜҶй’ҘгҖҒй’ұеҢ…ж–Үд»¶е’ҢвҖңBullXвҖқдәӨжҳ“ж•°жҚ®вҖҰвҖҰз„¶еҗҺйҖҡиҝҮ Telegram жңәеҷЁдәәзӘғеҸ–ж•Ҹж„ҹж•°жҚ®гҖӮвҖқ*

жӯӨж¬Ўж”»еҮ»зҡ„йҮҚзӮ№жҳҜ BullXпјҢиҝҷжҳҜдёҖдёӘж·ұеҸ—еҠ еҜҶиҙ§еёҒдәӨжҳ“иҖ…ж¬ўиҝҺзҡ„е№іеҸ°пјҢз”ЁдәҺеҝ«йҖҹиҺ·еҸ–ж–°еҸ‘иЎҢзҡ„еҠ еҜҶиҙ§еёҒгҖӮSocket з ”з©¶дәәе‘ҳиҜҒе®һпјҢиҜҘжҒ¶ж„ҸиҪҜд»¶дјҡжҳҺзЎ®жү«жҸҸдёҺвҖңBullXвҖқзӣёе…ізҡ„ж–Үд»¶пјҢе№¶ж”¶йӣҶжүҖжңүдёҺиҜҘеӯ—з¬ҰдёІеҢ№й…Қдё”д»Ҙ .txt з»“е°ҫзҡ„ж–Үд»¶гҖӮ

pumptoolforvolumeandcomment зҡ„дё»зЁӢеәҸдёӯйҡҗи—ҸзқҖдёҖдёӘз»ҸиҝҮ base64 ж··ж·Ҷзҡ„и„ҡжң¬пјҢиҜҘи„ҡжң¬йҡҗи—ҸеңЁ parts.txt ж–Үд»¶дёӯгҖӮдёҖж—Ұиў« index.js еҠ иҪҪеҷЁи§Јз Ғе№¶жү§иЎҢпјҢиҜҘи„ҡжң¬е°ұдјҡејҖе§ӢеңЁ Linux е’Ң macOS ж–Үд»¶зі»з»ҹдёӯз§ҜжһҒжҗңзҙўеҮӯиҜҒпјҢе№¶дҪҝз”ЁдёҺ Base58 зј–з Ғеӯ—з¬ҰдёІеҢ№й…Қзҡ„жӯЈеҲҷиЎЁиҫҫејҸпјҲиҝҷз§Қж јејҸеңЁй’ұеҢ…з§ҚеӯҗзҹӯиҜӯе’Ңз§Ғй’ҘдёӯеҫҲеёёи§ҒпјүгҖӮ

* дҪҝз”Ёж”»еҮ»иҖ…зҡ„д»ӨзүҢе’ҢиҒҠеӨ© ID йҖҡиҝҮ Telegram жңәеҷЁдәәеҸ‘йҖҒиў«зӣ—ж•°жҚ®

иў«зӣ—ж•°жҚ®йҖҡиҝҮ Telegram Bot API дј иҫ“пјҢдҪҝж”»еҮ»иҖ…иғҪеӨҹеҚіж—¶иҺ·еҸ–еҮӯиҜҒе№¶и·ЁеўғеҢҝеҗҚж“ҚдҪңгҖӮиҝҷз§Қе®һж—¶ж•°жҚ®жі„йңІжЁЎеһӢе…је…·йҖҹеәҰе’Ңйҡҗи”ҪжҖ§пјҢжҳҜж”»еҮ»й«ҳд»·еҖјеҠ еҜҶй’ұеҢ…зҡ„зҗҶжғійҖүжӢ©гҖӮ

дёәдәҶжү©еӨ§еҪұе“ҚиҢғеӣҙпјҢж”»еҮ»иҖ…иҝҳеҸ‘еёғдәҶ debugdogsпјҢиҝҷжҳҜдёҖдёӘеҢ…иЈ…еҢ…пјҢе…¶еҠҹиғҪд»…йҷҗдәҺйңҖиҰҒе’Ңжү§иЎҢ pumptoolforvolumeandcommentгҖӮ

Socket жҢҮеҮәпјҡвҖң*йҖҡиҝҮзӣҙжҺҘжҢҮеҗ‘ pumptoolforvolumeandcommentпјҢдҪңиҖ…зЎ®дҝқд»»дҪ•е®үиЈ… debugdogs зҡ„дәәйғҪдјҡиҮӘеҠЁжӢүе…Ҙе№¶жү§иЎҢзңҹжӯЈзҡ„жңүж•ҲиҪҪиҚ·гҖӮ вҖқ*
