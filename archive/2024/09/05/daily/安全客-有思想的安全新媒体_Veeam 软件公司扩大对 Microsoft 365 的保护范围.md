# Veeam 软件公司扩大对 Microsoft 365 的保护范围

Veeam Software 宣布推出 Veeam Backup for Microsoft 365 v8，可为 Microsoft 365 数据提供全面、灵活的不变性。

现在，组织可以采用零信任、多层不可变策略确保其 Microsoft 365 数据具有弹性，确保备份数据免受潜在更改或删除的影响，从而保持其原始完整性。Veeam 目前保护着超过 2100 万 Microsoft 365 用户，保护客户的关键 Microsoft 365 数据，以确保他们的业务在任何发生什么情况下都能继续运行。

“丢失 Microsoft 365 中的关键数据、文件和通信对任何组织来说都是灾难性的，”Veeam 首席风险官 John Jester 说。“这就是为什么我们今天保护了超过 2100 万用户，比市场上的任何供应商都多，使 Veeam 成为 Microsoft 365 的 #1 数据弹性解决方案。Veeam Backup for Microsoft 365 v8 可确保，即使预期会发生网络攻击和数据中断，组织仍可随时访问关键业务信息，以确保业务连续性。现在，凭借 Microsoft 365 最全面的备份不变性，此版本包括专为效率和规模而设计的新架构，以及直接基于客户请求的附加支持。

Veeam Backup for Microsoft 365 v8 将不可变备份与现有不可变副本相结合，为企业的备份提供全面防御。它提供了将备份数据存储在任何对象存储上的灵活性，包括 Azure Blob Storage、Amazon S3、IBM Cloud Object Storage 或 S3 兼容存储。

除了增强的不变性外，Veeam 还通过 Veeam 代理池提高了企业规模和效率。此架构更新通过在多个代理之间分配流量来提高备份处理速度。通过智能分担负载并保持不受限制，企业可以实现更好的备份性能，并有效地扩展拥有数万个用户的大型环境。

为了响应客户，Veeam 通过多项新功能扩展了其支持。组织现在能够使用基于 Linux 的备份代理，从而提供更多选择和更低的总拥有成本。此外，Veeam Backup for Microsoft 365 v8 现在支持私有和共享 Microsoft Teams 频道，为这一流行的通信和协作平台提供全面保护。

Veeam Backup for Microsoft 365 v8 的主要特性包括：

* 全面的不变性：市场上最全面的 Microsoft 365 备份不变性
* 企业规模：专为处理最大的企业数据集而构建的架构
* 增加的支持：私有和共享 Teams 频道、基于 Linux 的备份代理以及对 UI 的 MFA 访问

“保护驻留在 Microsoft 365 中的业务关键型数据对于各种规模的组织都至关重要。Enterprise Strategy Group 已反复验证，Veeam Backup for Microsoft 365 可在整个 Microsoft 365 中提供全面的保护选项。对于每个新版本，Veeam 都引入了广泛的新安全措施，版本 8 也不例外。Veeam 使组织能够拥有一个由公司治理驱动的数据保护和恢复解决方案，涵盖其所有关键数据，“TechTarget 企业战略小组业务总监 Tony Palmer 说。

“多年来，我们一直依赖和信任 Veeam。其中一个主要原因是 Veeam 不断发展，并在新功能方面处于市场领先地位。凭借 Veeam Backup for Microsoft 365 v8 中提供的备份不变性，我们看到了 Veeam 致力于增强 Microsoft 365 数据保护安全层的进一步证据，“FHNW 瑞士西北应用科学与艺术大学数据中心系统工程师兼团队负责人 Patrick Gebhard 说。

California Dairies， Inc. 基础设施和安全总监 Matthew Prieto 表示：“我们已经对 Veeam Backup for Microsoft 365 充满信心，现在有了 v8 以及 Linux 代理、代理池和改进的不变性，我们期待着实施进一步的弹性和安全性。
