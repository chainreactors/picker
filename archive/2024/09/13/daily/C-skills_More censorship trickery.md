### More censorship trickery

[![](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgo4cIM84U8Y5GgESr4yZH_D5y2BfV_1lhPnWDgM6wHvBZhOfmd8cy8z-NAE-UbeXFzKWSurNAJuSkfyc7gU5Eo4mo0cKwgIkB3qLhfGkH1-fIXiQacj5y0hoJzZsTJold3Roc7qpZ2XjmET2KN8wPKwNlfdKBqr5Z-jEZoaqI5RlJM93NKJrO5Q0iQcgIs/w602-h385/passport.jpg)](https://blogger.googleusercontent.com/img/b/R29vZ2xl/AVvXsEgo4cIM84U8Y5GgESr4yZH_D5y2BfV_1lhPnWDgM6wHvBZhOfmd8cy8z-NAE-UbeXFzKWSurNAJuSkfyc7gU5Eo4mo0cKwgIkB3qLhfGkH1-fIXiQacj5y0hoJzZsTJold3Roc7qpZ2XjmET2KN8wPKwNlfdKBqr5Z-jEZoaqI5RlJM93NKJrO5Q0iQcgIs/s642/passport.jpg)

I updated some of my git repos. [crash](https://github.com/stealth/crash) can now be used with Disguise Filters to serve an innocent redirect webpage when certain secrets are not passed beforehand so nobody except you will ever see that there is a shell server inside, even if they observe the SNIs that are in use. I also updated the docu on how you setup WA proxies with it since this changed in newer versions of the app.

Then there is an entirely [new PoC repo](https://github.com/c-skills/passport) to tunnel traffic through passkey servers. Thx to the ppl that tested it with me. Check it out, its fancy new [Perl](https://perl.org) code!
