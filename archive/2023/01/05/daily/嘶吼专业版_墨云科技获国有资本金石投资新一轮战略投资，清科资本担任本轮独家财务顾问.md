# 墨云科技获国有资本金石投资新一轮战略投资，清科资本担任本轮独家财务顾问

嘶吼专业版

![](https://mmbiz.qpic.cn/mmbiz_gif/wpkib3J60o297rwgIksvLibPOwR24tqI8dGRUah80YoBLjTBJgws2n0ibdvfvv3CCm0MIOHTAgKicmOB4UHUJ1hH5g/640?wx_fmt=gif)

近日，**「墨云科技」获得了国内头部国有资本新一轮战略投资，投资方为中信证券旗下专业私募股权投资平台金石投资，清科资本担任独家财务顾问。**本轮融资资金将主要用于持续加大研发投入，巩固智能自动化攻防产品矩阵，为客户提供全方位的信息安全服务，引领新一代以攻促防的安全服务模式。

墨云科技成立于2017年，作为全球领先的AI智能网络攻防安全服务提供商，墨云通过主动攻击验证用户安全防御体系状态，以攻促防。墨云也是国内唯一覆盖了从外网到防御体系、从防御体系到内网、从内网到操作系统、从操作系统到文件，从文件到函数的全链条自动化安全验证厂商。

目前，已在北京、上海、美国硅谷等10地设有研发中心及分支机构。公司获得多项荣誉资质，2022年度“红鲱鱼（Red Herring）”全球100强、被Gartner发布的《Hype Cycle for Security in China, 2022》评选为 BAS与SCA双领域标杆企业、国家信息安全漏洞库（CNNVD）技术支撑单位、IDC 人工智能安全解决方案创新者、德勤北京明日之星、GEI中国潜在独角兽企业等。

墨云科技具有显著的先发技术优势和卓越的研发实力，拥有四大人工智能攻防安全实验室，曾多次收到Oracle，Google，VMware的感谢信，自主研发了多个国内知名的网络攻防产品及解决方案。墨云科技正在成为全球领先的智能攻防领域领导者。

墨云科技创始人刘兵表示：“墨云科技通过AI算法进行安全验证，模拟黑客行为，以攻促防，能有效减少超过70%以上的外部漏洞报告；基于高于行业水准的数万个‘黑客剧本’，结合人工智能技术生成攻击杀伤链，以黑客视角全面验证用户安全系统的有效性；同时提供7\*24小时持续性安全验证服务，帮助安全专家集中精力攻坚克难。”

近年来全球网络安全环境日益复杂，网络战争频发，国家网络安全面临极大挑战。中国网络安全行业正在高速增长中，市场前景巨大。墨云构建了先进的AI与攻防相结合的技术及产品体系，率先研发了国内首款自主可控的虚拟黑客机器人，颠覆传统安全防护模式，是首次将人工智能算法与网络攻防技术相结合的智能攻防产品，技术国际领先。墨云解决了网络攻防技术上的诸多难题，填补了我国在网络安全智能攻防领域多年空白，引领中国信息安全行业的创新升级。

墨云科技打造了智能自动化渗透测试平台Vackbot与智能自动化攻击模拟平台VackBAS等多个颠覆性智能网络攻防产品，以“产品+SaaS订阅制”的商业模式落地，目前已累计服务逾200家大型行业客户，覆盖金融、通信、能源、政府、互联网等多个领域，包括中国人民银行、中国银联、国家电网、南方电网、中石油、中国移动、中国联通、中国电信、省级公安等国家安全关键领域头部客户。行业标杆客户连续多年复购增购，过去四年收入年复合增长率达近90%，内生增长力强劲，市场领先优势显著。

# ![](https://mmbiz.qpic.cn/sz_mmbiz_png/wpkib3J60o2ibR2vSLibcFv8TmXTcicFBvZTM9FX0UpRcHibbsQwLtbnia1k9aYmpwq2Nj3O2ibRIQcOPmLVkGDf9HdZg/640?wx_fmt=png)金石投资介绍

金石投资有限公司（以下简称：金石投资）成立于2007年10月，是中信证券股份有限公司（证券代码600030，以下简称“中信证券”）的专业投资子公司，也是首批获得中国证监会批准直投试点资格的两家企业之一。作为国内成立最早、资本金规模最大的券商直接投资机构，金石投资具备强大投资管理能力。金石投资合并管理基金超过30只，管理规模超过2300亿元。人民币基金累计投资项目超过560单，已退出项目超过220单。金石投资具有300+投资团队，依托中信证券1500+投行团队，200+研究团队以及中信集团全方位产业协同优势，已成为了中国最具影响力的私募股权投资机构之一。

# ![](https://mmbiz.qpic.cn/sz_mmbiz_png/wpkib3J60o2ibR2vSLibcFv8TmXTcicFBvZTM9FX0UpRcHibbsQwLtbnia1k9aYmpwq2Nj3O2ibRIQcOPmLVkGDf9HdZg/640?wx_fmt=png)资本市场观点

**金石投资**表示：“随着全球网络安全局势复杂化，网络安全已上升至国家战略层面。得益于政策及新技术的双驱动，网络安全已驶入高质量发展“快车道”，网络安全向智能化、自动化演进是大势所趋。墨云科技在网络安全攻防技术上具有明显突破，以攻击者视角为客户提供持续的自动化的系列安全验证产品，是具有颠覆性影响的创新网络安全企业。我们相信，墨云即将成为网络安全行业的领航者”

**清科资本合伙人韩晶**表示：“我们非常荣幸为墨云科技完成本轮融资，近年来，国际形势的变化将自动化攻防提升至国家安全核心战略层面，墨云科技在自动化攻击领域的全链条专精产品和先发技术优势具有唯一性，为国家关键安全领域头部客户创造了不可替代的实战应用价值。我们相信，在团队极具前瞻性的战略布局和非凡行业洞见下，墨云科技将把握时代机遇，乘风而起。”
