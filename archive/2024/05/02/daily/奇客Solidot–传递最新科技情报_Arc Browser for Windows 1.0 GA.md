## Arc Browser for Windows 1.0 GA

[![软件](https://icon.solidot.org/images/topics/topicsoftware.png?123)](/search?tid=15 "软件")

[Wilson](/~Wilson) (42865)发表于 2024年05月01日 10时29分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=78060&appkey=1370085986&title=Arc%20Browser%20for%20Windows%201.0%20GA "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自秘密团伙**

Arc 浏览器正式面向所有 Windows 11 用户开放。此前想使用 Arc 浏览器需要加入 Waitlist 才能进行测试。Arc 浏览器也是基于 Chromium 内核，默认采用竖直标签格局，支持 Chrom/Edge 扩展。Arc 浏览器最大的不同就是引入了“Space”概念（类似于“Groups”），用户可以创建不同的 “Space”来满足不同场景的浏览需求，每个 Space 下的网址集合可以一次性分享给他人。如果想从 Edge 切换到不同的浏览器，又不想使用 Chrome，那么可以试试 Arc。
https://resources.arc.net/hc/en-us/articles/22513842649623-Arc-for-Windows-2023-2024-Release-Notes
https://windiscover.com/posts/arc-browser-is-generally-available-for-windows-11.html
