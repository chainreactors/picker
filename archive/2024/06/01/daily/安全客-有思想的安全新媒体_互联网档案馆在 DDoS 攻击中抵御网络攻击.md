# 互联网档案馆在 DDoS 攻击中抵御网络攻击

互联网档案馆是历史最悠久的网站、电影、书籍、软件等在线目录之一，目前正面临网络攻击，导致其服务中断超过三天。互联网档案馆网络攻击被认定为分布式拒绝服务 (DDoS) 攻击，该攻击已围攻该服务并使其服务器充斥着重复请求。

虽然该组织向用户保证其藏品是安全的，但其允许用户浏览历史网页的工具 Wayback Machine 的可访问性已经受到影响。

互联网档案馆网络攻击针对多个系统
根据互联网档案馆 5 月 28 日分享的一篇博客文章，过去几天报告了间歇性服务中断，档案馆官员在社交媒体平台上分享的更新证实了这一点。尽管努力减轻攻击，但确切的来源仍未披露。

针对此次DDoS 攻击，互联网档案馆创始人兼数字图书管理员 Brewster Kahle 对大家的支持表示感谢，同时重申该组织将加强防御。Kahle 在博文中将此次攻击描述为“持续、影响深远、有针对性、适应性强，更重要的是，非常凶狠”。

缓解互联网档案馆 DDoS 攻击
互联网档案馆是免费访问各种媒体内容（包括历史和当代内容）的用户的宝贵资源。然而，其使知识获取民主化的使命遇到了法律挑战，该组织去年面临美国图书出版和唱片行业协会的诉讼。

这些法律诉讼指控侵犯版权并要求赔偿巨额损失，给全球图书馆的未来运营蒙上了阴影。互联网档案馆遭受的网络攻击反映了全球图书馆和知识机构遭受攻击的令人不安的趋势。最近的受害者包括大英图书馆、加州索拉诺县公共图书馆、柏林自然历史博物馆、安大略省伦敦公共图书馆，以及本周的西雅图公共图书馆。

鉴于持续不断的网络攻击和法律纠纷，卡尔强调了世界各地图书馆面临的更广泛影响。他警告称，出版业和唱片业的行为有可能破坏图书馆的生存，给全世界的读者带来严重担忧。
