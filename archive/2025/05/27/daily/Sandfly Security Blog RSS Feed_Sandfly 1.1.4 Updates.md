# Sandfly 1.1.4 Updates

26 April 2018

Product Update

Sandfly 1.1.4 is available. Includes 10 new sandflies to expand Linux stealth rootkit detection, process anti-forensics, log file tampering, suspicious user activity, and [process masquerading attacks](https://attack.mitre.org/wiki/Technique/T1036).
