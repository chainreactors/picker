Upscayl v2.5 Linux 预览版发布，免费开源的 AI 图像放大工具，可自定义模型了。

[Upscayl](https://www.appinn.com/upscayl/) 是一款跨平台的 AI 图像放大工具，免费、开源，下载后即开即用，不联网。几个小时以前发布了 v2.5 Linux 预览版。此版本仅针对 Linux 用户，新增**无限的新AI模型**，即现在可以导入自己的模型了。@[Appinn](https://www.appinn.com/upscayl-v2-5-for-linux/)

![Upscayl v2.5 Linux 预览版发布，免费开源的 AI 图像放大工具，可自定义模型了。 1](https://static1.appinn.com/images/202304/upscayl.jpg!o "Upscayl v2.5 Linux 预览版发布，免费开源的 AI 图像放大工具，可自定义模型了。 1")

以下是机翻更新内容：

* 无限的新AI模型! 🤖 Upscayl 现在允许导入你自己的模型
* 记住设置，即使你关闭了它✍（是的，以前不行 😂
* 新的设置标签 😎
* 自 v2.0.1 版起缩小了尺寸。🙃
* 更新了 Ultrasharp 模型 ⚡
* 新的自定义模型资源库! 📂
* 终于有了取消放大的按钮! 🛑
* 更明显的进度条 🚧
* 新的日志区。现在你可以直接看到日志和报告 bug。👨‍💻
* 批量模式现在可以跨会话持续存在。👬
* 输出文件夹现在可以跨时段保存。👬
* 新的试验性规模选项。⚖
* 删除了锐化选项（它毫无用处） 🥶
* 修正了对名称中有特殊字符的图像进行升格的问题。🉐
* 大幅改进了日志记录。💆

对于 Linux 用户，快去试试吧。

另外，使用 Upscayl 需要兼容 Vulkan 的 GPU，主要包括：

### NVIDIA

* GeForce 600 系列及更新产品
* Quadro K 系列及更新产品
* Tesla K/M/X 系列
* GRID K2/K520

### AMD

* Radeon HD 7000 系列及更新产品
* Radeon R7/R9 200 系列及更新产品
* Radeon RX 400/500/5000 系列及更新产品
* FirePro W 系列及更新产品

### Intel

* 第六代 Core 处理器及更新产品，包括 Haswell、Broadwell、Skylake、Kaby Lake、Coffee Lake 等

不确定也没关系，试试又没什么坏处。
