# Windows 10 支持即将结束，但可选扩展安全更新服务仍可获取

Microsoft 的扩展安全更新计划将在 2025 年 10 月 14 日之后为 Windows 10 提供付费补丁，但仅适用于 22H2 版设备。

运行 Windows 10 的企业和组织如果在 2025 年 10 月 14 日支持结束后继续使用该作系统，则需要支付安全更新费用，每台设备 61 美元起，费用每年翻倍。此产品/服务（包括面向消费者的产品/服务）是 Microsoft 扩展安全更新计划的一部分，将在有限的时间内提供保护。

Microsoft 高管 Yusuf Mehdi 将该计划描述为通往 Windows 11 体验的桥梁。

## 适用于企业的 Microsoft ESU

Microsoft 的企业扩展安全更新 （ESU） 计划将在支持结束后为 Windows 10 设备提供每月安全补丁。ESU 仅涵盖关键安全补丁;它不提供新功能、设计更新或技术支持。该计划通过批量许可和批准的云平台提供，仅适用于运行 22H2 版本的设备。

第一年的订阅起价为每台设备 61 美元。ESU 订阅必须以全年为增量购买;部分年度保险不可用。

**包括用于基于云的 Windows 10 的 ESU**

ESU 覆盖范围将免费提供给托管在受支持的云环境中的 Windows 10 虚拟机。这包括：

* **Windows 365 云环境：**Windows 365 云电脑和通过其有效订阅连接到它们的 Windows 10 终结点，最长可使用三年。
* **基于 Azure 的服务：**Azure 虚拟桌面、Azure 虚拟机、Azure 专用主机、Azure Local（以前称为 Azure Stack HCI）、Azure Stack Hub 和 Azure Stack Edge。
* **合作伙伴平台：**Azure VMware 解决方案和 Azure 上的 Nutanix Cloud 集群。

企业现在可以通过 Microsoft 批量许可进行注册，从 2025 年 9 月 1 日开始，通过云解决方案提供商 （CSP） 进行注册。

## 面向消费者的 Microsoft ESU

Microsoft 还首次将其 ESU 覆盖范围扩展到消费者，允许个人设备在对 Windows 10 的支持结束后以有限的方式保持保护。该计划为尚未准备好升级的用户提供一年的修补选项，这与企业可用的三年保修不同。

从 7 月开始，个人用户可以通过 Windows 设置中的引导式设置工具选择加入，预计在 8 月中旬之前即可获得一般访问权限。用户可以以 30 美元的价格激活该计划，兑换 Microsoft Rewards 积分，或与 Windows 备份同步。选择后，该过程将在设备上自动完成。

面向消费者的 ESU 计划已向 Windows 预览体验成员参与者提供。与企业版一样，它提供必要的更新，并且仅作为短期解决方案。

个人用户可以在今年夏天开始通过系统通知和标准许可注册。

## Windows 11 没有回头路

ESU 计划提供额外的时间，但强制实施限制，将用户推向 Microsoft 的较新平台。计划详细信息可能会更改，因此有关完整和最新的信息，请参阅 Microsoft 的官方公告。
