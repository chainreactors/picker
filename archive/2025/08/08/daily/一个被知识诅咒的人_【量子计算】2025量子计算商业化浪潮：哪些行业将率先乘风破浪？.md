### 量子计算基础与算法

#### 量子比特与量子门

量子计算的基本单元是量子比特。使用QuTiP库，我们可以模拟一个量子比特的初始化和操作。

```
# 导入QuTiP库，用于量子信息模拟
import qutip as qt
import numpy as np

# 创建一个基态量子比特 |0>
qubit = qt.basis(2, 0)
print("初始量子比特状态：", qubit)

# 应用Hadamard门，创建叠加态
H = qt.hadamard_transform()
superposition = H * qubit
print("叠加态：", superposition)
```
