# 重磅发布！美亚柏科取证3.0新模式解决方案重构取证

企业资讯
[行业](https://www.4hou.com/category/industry)
2023-07-19 15:54:42

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)73503

收藏

导语：美亚柏科基于取证3.0新模式通过技术+业务融合创新以“五大重构”全面提升取证产品能力。

大数据时代下，数据体量爆炸式增长，物联设备种类日益繁多，犯罪团伙的作案手法随之“全面升级”。传统取证单一技术、单兵装备、单警种作战难以满足打击新型网络犯罪需求，使得新型涉网犯罪侦办“难上加难”。作为国内电子数据取证行业龙头企业，美亚柏科凭借自身的技术资源优势以及经验积累，围绕全景数据下的取证与研判创新性地推出全新的行业解决方案——取证3.0新模式解决方案。

取证3.0新模式解决方案以终端取证技术、云取证技术、区块链取证、物联网取证以及流量取证技术等前沿技术为依托，开展“云、管、端”取证建设。具有取证装备和各级实验室全互联；采集、取证、分析结果数据全汇聚；取证流程业务流程全对接；取证生态全开放等特点。

![微信图片_20230721103745.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230721/1689907089168837.jpg "1689907089168837.jpg")

取证3.0新模式在取证3.0新模式下，美亚柏科构建全国统一的取证VPN网，在取证网内搭建统一的取证云平台，提供数据赋能、AI赋能和知识库赋能，提供SaaS化的云取证、App分析、区块链分析和调证服务等。

美亚柏科基于取证3.0新模式通过技术+业务融合创新以“五大重构”全面提升取证产品能力：

**一、超级精灵重构取证模式**

针对新型网络犯罪检材多、勘验难的问题，全新推出小巧全能的电子数据固证设备——超级精灵，该设备具有“全能固定证、高效应用、合规存证”等特点，以“一次固证，多次使用”的工作模式重构取证业务。

**二、AI大模型重构取证分析**

美亚柏科重磅发布国内首个公共安全大模型——“天擎”美亚公共安全大模型。以“天擎”美亚公共安全大模型为智能基座打造“小美”MYGPT，技术性变革用户交互体验，实现 “用提问代替操作”，重构电子数据取证分析。

**三、取证标准件重构取证流程**

基于20余年的案件支撑服务，美亚柏科技术支持团队，通过汇聚更新大量知识库并归纳整理犯罪生态链，形成百余种针对不同犯罪类型、不同角色的取证标准件，取证标准件用于指导取证分析策略，例如重点提取的应用、研判关键词、关键要素指引等。

**四、QKOS重构取证平台**

乾坤大数据操作系统（QKOS）提供跨行业、跨领域的大数据资源生产、开发、运维、运营、安全等全生命周期的服务能力，支撑各行业大数据基础能力、领域中台、业务应用的快速构建，同时解决各行业大数据项目跨领域难、交付质量低和实施成本高等问题，满足城市级新型数字基础设施、行业大数据基座、大数据产品研发基座等多场景需求。除了“九天揽月”实战工具、“彩云”电子数据汇聚与智能分析平台、“擎天”电子数据分析战训一体化平台等“成员”，取证云平台整体正式融入QKOS“大家庭”。

**五、MYOS重构取证生态**

MYOS取证操作系统是美亚柏科融合取证领域多年积累的取证能力，包括计算机取证能力、手机取证能力、云取证能力、汽车取证能力、物联取证能力等，倾力打造的一款提供全证据链电子数据提取、恢复、分析和检验鉴定能力的操作系统，以“统一架构、统一板卡、统一标准、统一应用、统一生态”重构取证生态。依托MYOS，美亚柏科将与生态伙伴快速构建全行业广义取证解决方案。

基于取证3.0新模式和“五大重构”，美亚柏科坚持自主研发，围绕取证大数据平台、实验室取证、现场勘查、广义取证等，重磅研发推出系列产品，赋能产品升级优化，包括“擎天”电子数据分析战训一体化平台、基于MYOS开放的云取证工作站、神剑侦勘一体化系、“亮剑一号”数据安全检查工具箱等，未来还将助力开发更多适用于各类业务场景的新产品新应用。

基于全新的取证3.0新模式和产品体系，美亚柏科将持续联合生态企业，持续领航市场，共同拓展新行业新领域，迈向取证3.0+广义取证的新时代。
