# 明年2月起谷歌Chrome将不再支持Windows 7

[新闻](https://www.4hou.com/category/news)

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)183783

收藏

导语：谷歌宣布自2023年2月起Chrome浏览器将不在支持Windows 7和Windows 8.1系统。

谷歌宣布自2023年2月起Chrome浏览器将不在支持Windows 7和Windows 8.1系统。

2023年2月7日，谷歌将发布Chrome 110版本，之后谷歌Chrome将停止对Windows 7和Windows 8.1系统的支持。谷歌称，Chrome用户必须确保其设备至少运行在Windows 10以上系统版本，才能继续使用未来的Chrome更新版本。

谷歌决定不再支持Windows 7和Windows 8.1系统版本与微软的Windows生命周期策略是一致的。微软宣布将于2023年1月10日起针对Windows 7和Windows 8.1系统的扩展安全更新（Extended Security Update）项目将不再支持。

根据Statcounter GlobalStats的统计数据，目前全球有10%的Windows系统用户使用Windows 7系统，有2.7%的Windows系统用户使用Windows 8.1系统。

而谷歌Chrome web浏览器的市场份额超过65%，Safari浏览器的市场份额大约18%，微软edge的市场份额大约为4.32%。

![Windows versions market share](https://www.bleepstatic.com/images/news/u/1109292/2022/Desktop%20Windows%20Version%20Market.png)

图 Windows版本市场份额

虽然老版本的谷歌Chrome版本仍然可以在Windows 7和Windows 8.1系统上运行，但谷歌建议用户对系统进行升级，以能够持续收到安全更新。

微软也建议用户更新系统，因为扩展安全更新项目不再支持后，系统用户将不再能收到技术帮助和系统更新，可能会使其计算机和数据暴露在安全风险之下。
