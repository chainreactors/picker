**01**

**Lazarus 利用 macOS 扩展文件属性隐藏恶意代码**

**披露时间：**2024年11月13日

**情报来源：**https://www.group-ib.com/blog/stealthy-attributes-of-apt-lazarus/

**相关信息：**

Group-IB 的研究人员发现 Lazarus 组织正在使用一种新技术，滥用 macOS 文件的扩展属性来传播研究人员称之为 RustyAttr 的新木马。攻击者将恶意代码隐藏在自定义文件元数据中，并使用诱饵 PDF 文档来帮助逃避检测。新技术类似于 2020 年 Bundlore 广告软件将其有效载荷隐藏在资源分支中以隐藏 macOS 有效载荷的方式。

macOS 扩展属性（EA）表示通常与文件和目录相关联的隐藏元数据，这些元数据不能通过 Finder 或终端直接看到，但可以使用“xattr”命令提取以显示、编辑或删除扩展属性。在 RustyAttr 攻击案例中，EA 名称为“test”，并包含一个 shell 脚本。存储 EA 的恶意应用程序是使用 Tauri 框架构建的，该框架结合了可以调用 Rust 后端上的函数的 Web 前端（HTML、JavaScript）。

当应用程序运行时，它会加载一个包含 JavaScript（“preload.js”）的网页，该网页从“测试”EA 中指示的位置获取内容并将其发送到“run\_command”函数以执行 shell 脚本。为了在此过程中降低用户的怀疑程度，一些样本会启动诱饵 PDF 文件或显示错误对话框。

**02**

**伊朗“梦想工作”活动披露**

**披露时间：**2024年11月12日

**情报来源：**https://www.clearskysec.com/irdreamjob24/

**相关信息：**

ClearSky 的研究识别了一个名为“伊朗梦想工作活动”的网络钓鱼活动，攻击者TA455通过提供虚假工作机会针对航空业。

攻击者使用假招聘网站和LinkedIn个人资料分发包含恶意文件的ZIP文件。一旦ZIP文件被下载，受害者点击一个突出显示的EXE文件。EXE文件通过DLL侧加载加载恶意DLL文件“secur32.dll”。恶意软件检查受害者的IP地址，并从一个包含C&C服务器域名地址的GitHub账户下载信息。

该活动分发了SnailResin恶意软件，该软件激活了SlugResin后门。ClearSky将这两种恶意程序归因于Charming Kitten的一个子群组，但也有一些网络安全公司认为属于朝鲜的Kimsuky/Lazarus APT组织。这种类似的“梦想工作”诱饵、攻击技术和恶意软件文件表明，Charming Kitten可能在冒充Lazarus以隐藏其活动，或者朝鲜与伊朗分享了攻击方法和工具。

**03**

**WIRTE 继续针对中东并转向破坏性活动**

**披露时间：**2024年11月12日

**情报来源：**https://research.checkpoint.com/2024/hamas-affiliated-threat-actor-expands-to-disruptive-activity/

**相关信息：**

Check Point 发现与哈马斯有关联的WIRTE威胁行为者在中东地区的活动并未因地区冲突而中断，他们继续利用地区事件进行间谍活动，目标可能是巴勒斯坦权力机构、约旦、伊拉克、埃及和沙特阿拉伯的实体。

自 2023 年 10 月以来，研究人员观察到的多个案例使用 IronWind 加载程序作为感染媒介，通过与命令和控制 (C2) 服务器进行通信并执行隐藏在 HTML 元素中的恶意代码。2024年9月，研究人员发现了一条新的感染链，攻击者在PDF中嵌入模仿 URL 缩短服务的嵌入式 URL，该链接将目标用户重定向至一个RAR文档，其中包含三个旨在使用 DLL-Sideloading 的文件，最终下发 Havoc，允许攻击者保持对受感染系统的持续访问，从而促进各种恶意活动。

2024 年 10 月，以色列ESET 经销商的合法电子邮件地址发送了恶意电子邮件活动 。该电子邮件包含 SameCoin Wiper 的新版本，该版本已于2月的时候在针对以色列的攻击中部署。分析发现SameCoin擦除器组件中使用的独特 XOR 函数可以在较新的 IronWind 加载器变体 ( propsys.dll) 代码中找到，表明这两种工具是由同一个参与者开发，并且可能在同一个环境中进行编译的。SameCoin 是一款多平台擦除器，具有 Android 和 Windows 版本，在这两种情况下，它都冒充了 INCD 安全更新。

**相关信息：**

近日，奇安信威胁情报中心发布报告针对最近几年捕获到的金眼狗样本以及攻击手法进行了介绍，并讨论了金眼狗的常用攻击手法和样本演变。

**披露时间：**2024年11月7日

近日，根据 SentinelOne 披露， 朝鲜Lazarus APT组织的一个分支 BlueNoroff 正在针对加密相关企业发起代号为“Hidden Risk（隐藏风险）”的攻击活动。该活动开始于 2024 年 7 月，攻击者采用了独特的策略来逃避检测并入侵受害者系统。

**01**

**攻击者利用 Excel 文件传播 Remcos RAT 变种**

**披露时间：**2024年11月8日

**情报来源：**https://www.fortinet.com/blog/threat-research/new-campaign-uses-remcos-rat-to-exploit-victims

**相关信息：**

Fortinet 研究人员发现了一个危险的网络钓鱼活动，该活动正在传播一种新的Remcos RAT（远程访问木马）变种。这种功能强大的恶意软件在网上商业销售，针对 Microsoft Windows 用户，并允许威胁行为者远程控制受感染的计算机。

此次攻击活动以伪装成订单通知（OLE Excel 文档）的欺骗性钓鱼电子邮件发起。打开附加的恶意 Excel 文档后，CVE-2017-0199漏洞就会被利用来下载并执行 HTML 应用程序 (HTA) 文件。该 HTA 文件经过多层混淆处理，其代码以不同的脚本编写，包括 JavaScript、VBScript、Base64 编码、URL 编码和 PowerShell。然后，下载恶意可执行文件 (dllhost.exe)，并通过 32 位 PowerShell 进程执行。它通过进程挖空以在新进程“Vaccinerende.exe”中运行恶意代码，这不仅可以确保在受害者的设备上持久存在，还可以下载和解密 Remcos 有效负载文件。

此外，恶意代码使用的反分析技术还包括矢量异常处理程序、动态获取的 API、动态计算的常量、API ZwSetInformationThread() 和 ZwQueryInformationProcess() 以及 API 挂钩。

**02**

**黑客使用 ZIP 文件串联来逃避检测**

**披露时间：**2024年11月7日

**情报来源：**https://perception-point.io/blog/evasive-concatenated-zip-trojan-targets-windows-users/

**相关信息：**

近日，Perception Point研究人员发现黑客正在使用 ZIP 文件连接技术在压缩档案中传递恶意负载，从而避免安全解决方案的检测。此方法涉及将多个 ZIP 存档附加到单个文件中。据悉，研究人员在分析一次以虚假发货通知引诱用户的网络钓鱼攻击时，发现了一个隐藏着木马的串联 ZIP 存档。该附件被伪装成 RAR 存档，并且恶意软件利用 AutoIt 脚本语言自动执行恶意任务。

首先，攻击者创建两个或更多单独的 ZIP 存档，并将恶意负载隐藏在其中一个中，其余部分则保留无害内容。然后通过将一个文件的二进制数据附加到另一个文件，将单独的文件连接成一个文件，并将其内容合并为一个组合的 ZIP 存档。虽然最终结果显示为一个文件，但它包含多个 ZIP 结构，每个结构都有自己的中心目录和结束标记。

攻击的下一阶段依赖于 ZIP 解析器如何处理串联档案。Perception Point 对 7zip、WinRAR 和 Windows 文件资源管理器进行了测试，当使用7zip打开时，只显示了一个看似无害的PDF文件，而使用Windows文件资源管理器或WinRAR打开时，则完全暴露了隐藏的危险，包括恶意可执行文件。

为了防御连接的 ZIP 文件，Perception Point 建议用户和组织使用支持递归解包的安全解决方案。
