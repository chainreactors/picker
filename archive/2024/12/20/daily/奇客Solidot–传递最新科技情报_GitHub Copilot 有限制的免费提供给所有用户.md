## GitHub Copilot 有限制的免费提供给所有用户

[Wilson](/~Wilson) (42865)发表于 2024年12月19日 14时12分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=80097&appkey=1370085986&title=GitHub%20Copilot%20%E6%9C%89%E9%99%90%E5%88%B6%E7%9A%84%E5%85%8D%E8%B4%B9%E6%8F%90%E4%BE%9B%E7%BB%99%E6%89%80%E6%9C%89%E7%94%A8%E6%88%B7 "新浪微博分享")

**来自棕榈上的霜**

GitHub 宣布其辅助编程助手 Copilot 免费提供给所有用户，但免费并非意味着用户可以无限制的使用，GitHub 此举还是希望用户在用惯 Copilot 之后会选择付月费。免费用户在一个月内可以使用最多 2000 次代码辅助完成以及 50 次聊天请求，支付 10 美元月费则不再有限制。

https://github.com/features/copilot
