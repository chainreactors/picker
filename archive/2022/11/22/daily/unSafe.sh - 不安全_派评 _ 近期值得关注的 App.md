## 不容错过的新 App

除了「新鲜」App，App Store 中的许多老面孔也在不断迭代、更新，增加更多有趣和实用的功能。少数派希望帮你筛选 App Store 中值得关注的 App 动态，让你快速了解 App 和开发者们的最新动态。

### Happy:Days：让每一个日子都值得期待

* 平台：iOS
* 关键词：倒数日、纪念日

[@Peggy\_](https://sspai.com/u/5isr02uh)：重要亲人的生日还有多久，和恋人相知相伴的时间已有几时，临近考试复习的时间还有多少…… 在我们的生活当中，有太多美好的日子值得期待，也有许许多多的重要事件需要记录。各类倒数日应用也应运而生，我们往往会把各种值得纪念的日子交给倒数日类应用去管理，但是，赋予日子意义的不仅仅是记录的动作，还有一颗诚挚期待的心。

Happy:Days 是一款新上架的倒数日类应用，和早先少数派介绍的运动习惯养成应用 Grow 师出同门，相比于其他倒数日应用，Happy:Days 在设计上下足了功夫，我们不妨看看作为后起之秀的它有何独特之处。谈到倒数日，首先离不开的就是基础的记录功能，Happy:Days 在这一点上几乎给了你想要的一切功能。Happy:Days 支持记录的时间类型包含倒数日、纪念日、生日乃至年龄等等，任何值得铭记的时间你都可以放心地记录在内。

![boxcnxumd6lXulv3UzqSu6nUZyb](https://cdn.sspai.com/editor/u_/cdtk6ktb34t9mfr1itu0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

除了基础的公历外，Happy:Days 还支持我们常用的农历，这样一来我们便无需再费心自己转换某些农历时间。其次，Happy:Days 也给出丰富的重复设定，对于生日、固定日期发生的事件而言，利用 Happy:Days 的重复设定，便能让我们做到一次设定，永久使用。颇为用心的是，Happy:Days 支持丰富的时间格式选项，我们可以任意选择自己喜欢的记录方法，方便对于事件已经或是即将到来的事件有更加清晰的认知。

![boxcnWkDwZKUkiAMWdWh1WAd9Hf](https://cdn.sspai.com/editor/u_/cdtk6l5b34t9m86tai0g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Happy:Days 整体设计称得上赏心悦目，细节也完全拉满。针对生日这类特殊的事件，Happy:Days 额外增加了显示年龄和星座的选项；如果你记录了重要的人的年龄，Happy:Days 除了能够展示年龄外，还会根据年龄的变化，动态显示不同的人物图标。

![boxcnP8VV8nZbSfhWcrMC3IVJTS](https://cdn.sspai.com/editor/u_/cdtk6ldb34t9m86tai10?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

与其他应用只提供一些简单的记录模版不同，Happy:Days 从不同的方面充分提供个性化选项，让我们记录的每个日子都有所不同。除了基础的模版外，Happy:Days 还允许我们设定丰富的主题，并内置多种字体以供选择，甚至在事件标题记录上，Happy:Days 还额外支持了添加照片、emoji 等选项。这样用心制作的事件卡片仅仅存在于应用内多可惜，Happy:Days 提供了丰富的小组件支持，并且精心设计了锁屏桌面的显示样式，让你的重要日子时刻呈现在最醒目的位置。

![boxcn39cP7nix7RemNE4iMEbf97](https://cdn.sspai.com/editor/u_/cdtk6llb34t9mc7if7ig?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

你可以通过 [App Store](https://apps.apple.com/cn/app/happy-days/id1564858029) 下载使用 Happy:Days。

### Anybox：效率至上、诚意满满的书签管理工具

* 平台：iOS / iPadOS / macOS
* 关键词：稍后读、网络书签

[@Vanilla](https://sspai.com/u/whoops)：在书签管理工具这个领域，前有 Pinboard、Raindrop 等前辈，后有 Cubox、GoodLinks 等后起之秀，想要在这片本就小众的领域分得一碗羹不是一件易事。我一开始不以为然，只把 Anybox 当作是又一款普普通通的个人开发者练手作品而已。然而，经过一段时间的重度使用以及对 Anybox 官网的「地毯式」探索，我认为 Anybox 已经称得上一款完成度很高的书签管理工具，并且特色鲜明。

Anybox 支持导入链接、便条、图片和文件等四类内容，导入后会在应用的侧边栏按照类型单独进行分类。Anybox 可以在导入链接时自动抓取标题、网站图标、预览图片和描述等元数据，并且可以在导入完成后进行「编辑链接元数据」的操作。元数据包含了三部分内容——文本、图片和归档，文本指的是标题和描述，图片指的是网站图标和预览图片，而归档指的是 Web Archive、PDF 或者图片文件。

![boxcnhgJGUxD1OVXpJiPgnohUud](https://cdn.sspai.com/editor/u_/cdtk6ltb34t9mc7if7j0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

作为一款书签管理工具，导入书签的便利程度是考量这一类型应用的重要因素之一。据本人的不完全统计，Anybox 支持导入书签的方式超过了 14 种，包括 HTML / JSON 文件导入、快捷键粘贴、拖拽、浏览器扩展、快捷指令、Apple Script 等，可以说是我见过最夸张的应用了。Anybox 几乎适配了 macOS 系统中所有与 I/O 相关的系统特性，除此之外还有不少创新的功能令人眼前一亮。

作为一款效率工具，Anybox 也支持了丰富的 URL Schemes，我们可以通过[官方文档](https://anybox.app/url-schemes)来进行查询，同时可以搭配「快速链接」和 Anydock 这两个功能来实现对链接的快速添加。「快速链接」是「关键词展开」功能的一个延伸，可以在触发关键词展开时直接打开相应的链接，比如说 Anybox 的 URL Schemes。例如，将链接 `anybox://save-tab` 设为关键词 `ttt` 的扩展，那么只需连续三次按下 `t` 即可将当前标签页添加到 Anybox 中。在进行操作前，请确保「系统偏好设置」>「隐私与安全」>「辅助功能」中已经添加了 Anybox，并且点击菜单栏图标，在弹出的菜单中启用了「关键词展开」和「快速链接」这两个功能。

![boxcnY0GElQ3bw1rqzt1ZFqthrg](https://cdn.sspai.com/editor/u_/cdtk6nlb34t9mjt23sb0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

想要打开 Anybox 中的链接非常简单，只需在应用内双击链接条目即可。不过注意不要双击链接的预览图片，否则会弹出该图片的预览窗口。另外，也可以右键点击链接条目并选择「使用 XXX 浏览器打开」的操作。

![boxcnYTSWV2Hznvm01w8h5zjVEd](https://cdn.sspai.com/editor/u_/cdtk6o5b34t9mfr1itv0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

另外，也可以通过 Anybox 的快速查找功能来搜索指定的链接，然后按下键盘按键 `Return` 或者鼠标单击来打开链接。

![boxcnjr1oTtMJm8TR8HXF6ae5H4](https://cdn.sspai.com/editor/u_/cdtk6o5b34t9m86tai1g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

前文提到过 Anybox 有个功能叫做 Anydock，它就像 macOS 系统中的 Dock 一样可以显示在屏幕上。可以将 Anybox 中的任意链接添加到 Anydock 上，然后点击 Anydock 上的图标即可快速打开链接。Anydock 支持添加网页链接、URL Schemes（包括其它第三方 App）和 Anybox 配置文件。

![boxcnRFPUKF1pHSl6AVKNdNQ6Ug](https://cdn.sspai.com/editor/u_/cdtk6olb34t9mfr1itvg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Anydock 是 Anybox 非常成功的一个创新，它以链接为载体，打通了 Anybox 与其它应用之间的联系。比如说，我将 Things 的一个 URL Scheme `things:///add?use-clipboard=replace-title` 添加到 Anydock 上，使用时点击 Anydock 上的 Things 图标就可以直接将剪贴板上的链接作为一条新的待办事项添加到 Things 中。

![boxcnjh9vCcbpmvHOFPXg4VPa1c](https://cdn.sspai.com/editor/u_/cdtk6otb34t9m86tai20?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

虽然 Anybox 的重心放在了书签管理上，但是也可以直接在应用内阅读链接的内容。选中想要阅读的文章，在右键菜单中选中「下载文章」，等待「下载完成」的消息提示后再次右键点击链接，然后选择「显示阅读器」即可进入阅读模式。

想要分享链接也很简单，直接在链接列表右键点击链接，即可在右键菜单中选择「拷贝为」或者「分享」两个动作。在「快速查找」窗口中，也可以通过右键点击或者按下键盘按键 Tab 来触发操作菜单，然后选择拷贝或者分享的操作。

![boxcn0XxQ3xlVOkBb7PEq2c0Vnh](https://cdn.sspai.com/editor/u_/cdtk6rdb34t9mfr1iu00?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Anybox 对链接的分类管理主要通过「文档集」来实现，它的功能类似于标签，也就是说可以将一个链接同时添加到多个文档集中。虽然文档集无法实现多级嵌套，但是可以通过「主题」来进行归类。也就是说，Anybox 的链接管理是两个层级。

![boxcnlznoMGV9ed1cTjnhh7oa5g](https://cdn.sspai.com/editor/u_/cdtk6rtb34t9mc7if7jg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

在通过「文档集」和「主题」进行手动分类的基础上，Anybox 还有「智能列表」的功能。Anybox 支持筛选种类、星标、关键词设定、文档集、文档集关系、来源、查找关键词、文件扩展名、添加日期、修改日期、上次打开日期等非常丰富的选项，基本可以满足大家个性化的需求。

![boxcnVdtDe8omHBMu7dnG2SZEK5](https://cdn.sspai.com/editor/u_/cdtk6s5b34t9mfr1iu0g?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Anybox 可以在 [App Store](https://apps.apple.com/cn/app/id1593408455) 免费下载，支持了 iOS、iPadOS 和 macOS 三个平台。免费用户可以使用大部分功能，付费后则可以解锁 iCloud 同步、文档集、智能列表、归档、自动化、Deeplinks 等功能，iOS / iPadOS 客户端则可以解锁键盘扩展和小组件等功能。Anybox Pro 的订阅价格为每月 8 元、每年 68 元，或者可以以 248 元买断，个人认为价格还算合理。

### Airshow：过于轻量级的播客播放器

* 平台：iOS / iPadOS
* 关键词：播客

[@黎明前线Alan](https://sspai.com/u/safari_cheung)：Airshow 是一款刚刚上架 App Store、主打轻量级的播客播放器，来自 RSS 服务 Feedbin 的开发团队。其最主要的特征，就是采用了高斯模糊设计的主播放界面，整个播放界面除了播客封面，快进后退 15s / 暂停按钮，以及进度条之外，找不到任何任何播放控制元素；任何其他播客添加订阅/下载剧集等选项，都被收进了底部的二级菜单中。

![boxcnuSydRkpNXyLAKRAXRAhdDu](https://cdn.sspai.com/editor/u_/cdtk6slb34t9mjt23sbg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

Airshow 可以使用 Feedbin 账户同步播客内容，应用内置也有播客订阅 / 播放进度同步服务，不过这些都不影响 Airshow 本身播客功能的正常使用。

![boxcnjl4LkLtmPTsPHtbltA1Z2c](https://cdn.sspai.com/editor/u_/cdtk6tdb34t9mc7if7k0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

对于很多此类新 App 来讲，「轻量级」往往也意味着大量功能缺失：当前 Airshow 在很多播客 App 关键功能上也是如此：比如播放速度控制、睡眠倒计时，以及连续播放等功能，在 Airshow 中都不见踪影。

对我而言，Airshow 目前使用起来最不方便的地方是 Airshow 没有直接播放 Podcast 剧集的选项：你只能直接下载对应的剧集，等待下载完毕之后才可以直接播放。对于习惯 Spotify、Pocketcast 的用户来讲或多或少会有些不便。

![boxcnAAjAsXZlLB4HYallok8nph](https://cdn.sspai.com/editor/u_/cdtk6ttb34t9mc7if7kg?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

除了上述问题，很多播客老饕中意的人声增强功能，Airshow 也是没有的；这些功能稍稍严苛一些来讲的话，已经是如今播客移动端 App 的基本配置，这些直接决定了收听播客的实际体验，加上开发者对开发 Airshow 这款应用的态度也只是「看看能不能做出比现成选项更简单好用的选项」，Airshow 后续的新功能开发以及维护进度，可能也要打一个问号。

但如果你非常中意 Airshow 的播放界面，或是你最近也在寻找轻量级播客播放器给内存空间吃紧的 iOS 设备，或许 Airshow 仍然是一个值得你尝试的选择。

![boxcndBdXFoUjxR40shbaM7LOdg](https://cdn.sspai.com/editor/u_/cdtk6u5b34t9mjt23sc0?imageView2/2/w/1120/q/40/interlace/1/ignore-error/1)

你可以在 [App Store](https://apps.apple.com/cn/app/airshow-lightweight-podcasts/id1584582270) 下载到 Airshow，应用支持 iOS / iPadOS，应用内使用 Feedbin 账户同步完全免费，但如果是使用应用内置的跨设备同步服务，则需要付费购买，售价为 133 元 / 年。
