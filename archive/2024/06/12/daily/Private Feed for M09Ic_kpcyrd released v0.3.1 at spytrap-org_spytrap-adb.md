## What's new

* Update dependencies (this is a minor release)
  + Fix [RUSTSEC-2024-0003](https://rustsec.org/advisories/RUSTSEC-2024-0003)
  + Fix [RUSTSEC-2024-0332](https://rustsec.org/advisories/RUSTSEC-2024-0332)
  + Fix [RUSTSEC-2024-0019](https://rustsec.org/advisories/RUSTSEC-2024-0019)
  + Fix [RUSTSEC-2024-0336](https://rustsec.org/advisories/RUSTSEC-2024-0336)
  + Fix [RUSTSEC-2023-0075](https://rustsec.org/advisories/RUSTSEC-2023-0075)
* Update forensic-adb to 0.7.0
  + Use `exec:` instead of `shell:` adb instructions

```
git clone https://github.com/spytrap-org/spytrap-adb
cd spytrap-adb
git checkout v0.3.1
make
sha256sum target/x86_64-unknown-linux-musl/release/spytrap-adb
```
