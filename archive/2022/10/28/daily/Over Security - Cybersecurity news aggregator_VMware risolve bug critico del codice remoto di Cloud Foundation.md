# VMware risolve bug critico del codice remoto di Cloud Foundation

* Share
* Tweet

VMware ha rilasciato oggi aggiornamenti di sicurezza per correggere una vulnerabilità critica in VMware Cloud Foundation, una piattaforma cloud ibrida per l’esecuzione di app aziendali in ambienti privati ​​o pubblici.

“A causa di un endpoint non autenticato che sfrutta XStream per la serializzazione dell’input in VMware Cloud Foundation (NSX-V), un hacker può ottenere l’esecuzione di codice in remoto nel contesto di ‘root’ sull’appliance”, [spiega](https://www.vmware.com/security/advisories/VMSA-2022-00031.html) l’azienda in un comunicato di oggi .

VMware fornisce anche una [soluzione temporanea](https://kb.vmware.com/s/article/89809) per coloro che non possono applicare immediatamente le patch ai propri dispositivi.
