## 律师宣布调查 GitHub Copilot 版权侵犯问题

[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[WinterIsComing](/~WinterIsComing) (31822)发表于 2022年10月20日 16时58分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73117&appkey=1370085986&title=%E5%BE%8B%E5%B8%88%E5%AE%A3%E5%B8%83%E8%B0%83%E6%9F%A5%20GitHub%20Copilot%20%E7%89%88%E6%9D%83%E4%BE%B5%E7%8A%AF%E9%97%AE%E9%A2%98 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自火星超人**

GitHub 的 AI 编程助手 Copilot 使用公开的代码库进行训练，在开发者写代码时根据函数名等上下文自动补充完后续代码。很多时候 Copilot 补充的是公开代码库中代码片段的拷贝。这就引起了版权相关的问题。律师 Matthew Butterick 本周宣布与律所 Joseph Saveri 合作[调查对 Copilot 提起版权索赔的可能性](https://www.theregister.com/2022/10/19/github_copilot_copyright/)，包括：GitHub 用开源代码训练 Copilot 是否有不当行为；在补充代码时是否不当的推荐了其他人的版权作品。自由软件基金会（FSF）此前曾[呼吁](https://www.solidot.org/story?sid=68448)对 GitHub Copilot 所产生代码的许可授权问题展开调查。
