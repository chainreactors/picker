各种互联网网站为了更好地了解用户的爱好和习惯，都会对用户的设备码、IP、浏览器版本及插件、操作系统、屏幕分辨率等进行记录，并由此形成每个用户的电子指纹，从而对用户的上网行为进行记录和分析，方便推送精准的广告或者进行其他分析应用。因此，如何保护好包括IP在内的各种信息，也是情报分析师需要了解的知识。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xoaXOsr5fLvFtYUbtla4DhTicqRbnQYWInUYA0PKibuhk6eIXF7TPNcEQgqxYoWuxDwcsyMloxHhVA/640?wx_fmt=png)

**一、什么是 IP 地址？**

Internet 协议 (IP) 地址本质上是连接到任何网络的机器的标识号。我们可以将IP视为上网时的收件和发件地址。自 1970 年代首次开发以来，IP 地址已经演变为定义设备在互联网上的交互方式。IP 地址类似于任何网络上设备（包括服务器）的 GPS 位置。无论您是简单地上网、观看 YouTube 上的视频，还是预订航班，肯定都涉及到IP 地址。

**二、IP 地址是如何工作的？**

IP 地址通常由 Internet 服务提供商 (ISP) 分配给上网设备。由于 ISP 为您提供互联网访问权限，因此他们有责任为您的设备分配 IP 地址。假设电信公司或移动公司是您的 ISP，那么他们将负责为您的设备分配 IP。当您尝试连接到互联网时，您的在线活动首先会转到您的 ISP，并立即通过您的 IP 地址路由到您的设备。IP地址目前有IPv6 和 IPv4两种。

**三、IPv6 和 IPv4的主要区别**

IPv4 地址设计为 32 位，因此最多可能有4294967296个地址，因而随着联网的设备越来越多，IPv4 地址已经不够分配了。

IPv6 地址为 128 位，IPv6旨在成为修复IPv4的限制和缺陷的下一代协议。其地址数量号称可以为全世界的每一粒沙子编上一个地址。

**四、如何查看自己的 IP 地址**

**1、查看设备的网络属性**

无论是手机上网还是电脑上网，都可以通过查看网络属性的方式查看自己设备的IP。若设备是通过WIFI上网的，通常看到的IP是路由器分配给设备的局域网IP，而不是在公网上的IP。

**五、如何更改自己的 IP 地址**

**1、重启路由器**

目前国内绝大部分用户使用的都是动态IP，因此只要重启一次路由器，就会更新一次IP。但这类动态IP还是能反映用户的大致区域位置。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xoaXOsr5fLvFtYUbtla4DhoM9EYoic0qgiaLyOFY4DoOU2r6oiaeWOgDTic1rUXm4LFBviaQTv429llEA/640?wx_fmt=png)

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xoaXOsr5fLvFtYUbtla4DhzW7Vf8TwRQ1ejMCN7tbc9xSTYnPnyvXG8DvbFqERibqeMMghVOYPwGg/640?wx_fmt=png)

**2、更换网络服务商**

假如刚才您的电脑是利用电信的WIFI上网，现在可以换成利用自己的移动手机做热点上网，这样电脑的上网IP就会改变。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xoaXOsr5fLvFtYUbtla4DhAulCKb75sVHuziaEBnGEhCCibRjrggmU1qRZvhsyzdgjZibxPYVXTglXQ/640?wx_fmt=png)

**3、使用VPN**

虚拟专用网络 (VPN) 提供了一种可靠的方式来更改或屏蔽您的 IP 地址，这使得包括您的 ISP 在内的第三方很难监控您的在线活动。VPN 提供了许多好处，包括保护您的在线身份。

![](https://mmbiz.qpic.cn/mmbiz_png/B0AKMb5va5xoaXOsr5fLvFtYUbtla4DhW9V5NibgymXOFRdvY6sMsv6T9taGhsL2wGFVKIsIiaiclddTnV6fCia0RQ/640?wx_fmt=png)

**4、使用 Tor 浏览器**

每次打开 Tor 浏览器时，都会为您分配一个不同的 IP 地址。您通过 Tor 浏览器访问的所有网站都将看到分配的 IP，而不是您的真实 IP 地址。
