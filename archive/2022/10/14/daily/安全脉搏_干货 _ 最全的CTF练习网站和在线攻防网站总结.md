最全的CTF和在线攻防网站总结

这是一份红队/蓝队CTF在线平台列表来测试你学习的技能

# 1800多个攻防实验室

1800 多个实验室！涵盖 Windows 安全、云安全、密码破解、逆向工程等等。我强烈推荐这个平台！

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650339.png)

地址：https://attackdefense.com/members

# 加密黑客

一个有趣、免费的现代密码学学习平台

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650340.png)

地址：https://cryptohack.org/

# CMD挑战

CMD挑战

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650342.png)

地址：https://cmdchallenge.com/

# exploit.education

exploit.education 提供了多种资源，可用于了解漏洞分析、漏洞利用开发、软件调试、二进制分析和一般网络安全问题

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650345.png)

地址：https://exploit.education/

# Defend the Web

Defend the Web 是一个交互式安全平台，可以在其中学习和挑战自己的技能。尝试并完成所有 60 多个黑客挑战级别。

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650347.png)

地址：https://defendtheweb.net/

# hacksplaining

学习和练习不同的漏洞

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650349.png)

地址：https://www.hacksplaining.com/lessons

# TryHackMe

学习网络安全的一种有趣方式

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650351.png)

地址：https://tryhackme.com/

# HackTheBox

一个巨大的黑客游乐场

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650353.png)

地址：https://www.hackthebox.com/

# VulnHub

提供漏洞镜像，让任何人都能在数字安全、计算机软件和网络管理方面获得实际的“动手实操”经验。

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650355.png)

地址：https://www.vulnhub.com/

# Root-Me

训练你的黑客技能

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650358.png "null")

地址：https://www.root-me.org/

# Root In Jail

与HTB类似

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650359.png)

地址：https://rootinjail.com/index.html#

# OverTheWire

OverTheWire提供的兵棋推演可以帮助您以充满乐趣的游戏形式学习和练习安全概念。

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650362.png)

地址：https://overthewire.org/wargames/

# Cryptopals 加密挑战

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650365.png)

地址：https://cryptopals.com/

# LetsDefend

LetsDefend 通过调查模拟 SOC 中的真实网络攻击，帮助练习者建立具有实践经验的蓝队职业生涯

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650368.png)

地址：https://letsdefend.io/

# Cyber Defenders

蓝队挑战

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650369.png)

地址：https://cyberdefenders.org/blueteam-ctf-challenges/

# W3Challs

提供学习和练习黑客攻击的安全挑战

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-189011-1665650370.png)

地址：https://w3challs.com/
