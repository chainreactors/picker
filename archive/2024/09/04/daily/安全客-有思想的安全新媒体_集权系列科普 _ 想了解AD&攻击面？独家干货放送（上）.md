Active DirectoryпјҲADпјүжҳҜз”ұеҫ®иҪҜејҖеҸ‘зҡ„дёҖз§Қзӣ®еҪ•жңҚеҠЎзі»з»ҹпјҢиҮӘ 1999 е№ҙеҸ‘еёғд»ҘжқҘпјҢе®ғе·ІжҲҗдёәи®ёеӨҡдјҒдёҡ IT еҹәзЎҖи®ҫж–Ҫзҡ„ж ёеҝғз»„д»¶гҖӮAD жҸҗдҫӣдәҶйӣҶдёӯеҢ–зҡ„з”ЁжҲ·з®ЎзҗҶгҖҒиә«д»ҪйӘҢиҜҒгҖҒи®ҝй—®жҺ§еҲ¶д»ҘеҸҠеҗ„з§Қе®үе…ЁеҠҹиғҪпјҢдёәдјҒдёҡзҪ‘з»ңзҡ„з®ЎзҗҶе’ҢдҝқжҠӨжҸҗдҫӣдәҶејәеӨ§зҡ„ж”ҜжҢҒгҖӮ

жң¬ж–Үе°Ҷд»Ӣз»Қ AD зҡ„еҹәжң¬жҰӮеҝөгҖҒжһ¶жһ„гҖҒзү№зӮ№еҸҠе…¶дҪҝз”Ёзҡ„е…ій”®еҚҸи®®пјҢе№¶жҺўи®Ёе…¶еңЁдјҒдёҡзҺҜеўғдёӯзҡ„еә”з”ЁгҖӮ

# **йӣҶжқғи®ҫж–ҪADзҡ„е®ҡд№үдёҺеҠҹиғҪ**

Active Directory жҳҜдёҖдёӘзӣ®еҪ•жңҚеҠЎпјҢж—ЁеңЁз”ЁдәҺз®ЎзҗҶи®Ўз®—жңәзҪ‘з»ңдёӯзҡ„з”ЁжҲ·е’Ңиө„жәҗгҖӮзӣ®еҪ•жңҚеҠЎжҳҜдёҖз§ҚеҸҜд»ҘеӯҳеӮЁе…ідәҺз”ЁжҲ·гҖҒи®Ўз®—жңәгҖҒеә”з”ЁзЁӢеәҸе’Ңе…¶д»–иө„жәҗзҡ„дҝЎжҒҜпјҢе№¶е…Ғи®ёз®ЎзҗҶе‘ҳе’Ңз”ЁжҲ·иҪ»жқҫең°жҹҘжүҫе’Ңи®ҝй—®иҝҷдәӣдҝЎжҒҜзҡ„зі»з»ҹгҖӮ

##

## еҠҹиғҪ

**йӣҶдёӯиә«д»ҪйӘҢиҜҒе’ҢжҺҲжқғпјҡ**

йҖҡиҝҮеҹҹжҺ§еҲ¶еҷЁпјҲDomain Controller, DCпјүиҝӣиЎҢйӣҶдёӯиә«д»ҪйӘҢиҜҒпјҢзЎ®дҝқз”ЁжҲ·е’Ңи®Ўз®—жңәзҡ„е®үе…Ёи®ҝй—®гҖӮ

**зӣ®еҪ•жңҚеҠЎпјҡ**

еӯҳеӮЁе’Ңз»„з»Үе…ідәҺеҹҹеҶ…иө„жәҗзҡ„дҝЎжҒҜпјҢж–№дҫҝжҹҘжүҫе’Ңз®ЎзҗҶгҖӮ

**з»„зӯ–з•Ҙз®ЎзҗҶпјҡ**

е…Ғи®ёз®ЎзҗҶе‘ҳй…ҚзҪ®е’Ңз®ЎзҗҶз”ЁжҲ·е’Ңи®Ўз®—жңәзҡ„и®ҫзҪ®гҖӮ

## еҹәжң¬жҰӮеҝө

**еҹҹпјҲDomainпјүпјҡ**

еҹҹжҳҜ AD зҡ„еҹәжң¬еҚ•дҪҚпјҢд»ЈиЎЁдёҖдёӘйҖ»иҫ‘з»„зҡ„з”ЁжҲ·гҖҒи®Ўз®—жңәе’Ңе…¶д»–иө„жәҗгҖӮжҜҸдёӘеҹҹйғҪжңүдёҖдёӘе”ҜдёҖзҡ„ DNS еҗҚз§°пјҢдҫӢеҰӮnetstar.comгҖӮеҹҹдёӯзҡ„жүҖжңүеҜ№иұЎйғҪеӯҳеӮЁеңЁеҹҹжҺ§еҲ¶еҷЁдёҠпјҢе№¶дё”йҖҡиҝҮзӣ®еҪ•жңҚеҠЎиҝӣиЎҢз®ЎзҗҶгҖӮеҹҹжҸҗдҫӣдәҶдёҖз§ҚйӣҶдёӯеҢ–зҡ„з®ЎзҗҶж–№ејҸпјҢдҪҝеҫ—еҹҹз®ЎзҗҶе‘ҳеҸҜд»Ҙз»ҹдёҖз®ЎзҗҶеҹҹеҶ…жүҖжңүиө„жәҗгҖӮ

**еҹҹж ‘пјҲDomain Treeпјүпјҡ**

еҹҹж ‘жҳҜ Active Directory дёӯеӨҡдёӘе…·жңүдҝЎд»»е…ізі»зҡ„еҹҹзҡ„йӣҶеҗҲпјҢиҝҷдәӣеҹҹе…ұдә«дёҖдёӘе…¬е…ұзҡ„ DNS е‘ҪеҗҚз©әй—ҙгҖӮжҜҸдёӘеҹҹйғҪжңүдёҖдёӘе”ҜдёҖзҡ„ DNS еҗҚз§°пјҢеҪўжҲҗеұӮж¬ЎеҢ–зҡ„з»“жһ„пјҢдҫӢеҰӮ netstar.com дёӢзҡ„ it.netstar.com е’Ң hr.netstar.comгҖӮеҹҹж ‘йҖҡиҝҮиҮӘеҠЁдҝЎд»»е…ізі»пјҢе®һзҺ°дәҶи·Ёеҹҹзҡ„иө„жәҗи®ҝй—®е’Ңз®ЎзҗҶгҖӮ

**еҹҹжЈ®жһ—пјҲDomain Forestпјүпјҡ**

еҹҹжЈ®жһ—жҳҜ Active Directory зҡ„жңҖй«ҳеұӮж¬Ўз»“жһ„пјҢе®ғеҢ…еҗ«дёҖдёӘжҲ–еӨҡдёӘеҹҹж ‘пјҢиҝҷдәӣеҹҹж ‘е…ұдә«дёҖдёӘе…ЁеұҖзӣ®еҪ•гҖҒжһ¶жһ„е’Ңй…ҚзҪ®гҖӮжЈ®жһ—дёӯзҡ„жүҖжңүеҹҹй»ҳи®ӨйҖҡиҝҮеҸҢеҗ‘дј йҖ’дҝЎд»»е…ізі»иҝһжҺҘпјҢе…Ғи®ёи·Ёеҹҹиө„жәҗи®ҝй—®е’Ңиә«д»ҪйӘҢиҜҒгҖӮиҝҷдәӣдҝЎд»»е…ізі»еҸҜд»Ҙж №жҚ®дјҒдёҡзҡ„е®үе…Ёзӯ–з•ҘиҝӣиЎҢи°ғж•ҙпјҢд»ҘйҷҗеҲ¶жҲ–е…Ғи®ёзү№е®ҡеҹҹд№Ӣй—ҙзҡ„и®ҝй—®гҖӮжЈ®жһ—жҸҗдҫӣдәҶз»ҹдёҖзҡ„з®ЎзҗҶжЎҶжһ¶пјҢж”ҜжҢҒеӨҚжқӮзҡ„дјҒдёҡзҪ‘з»ңз»“жһ„гҖӮ

**з»„з»ҮеҚ•дҪҚ (Organizational Unit, OU)пјҡ**

OU жҳҜеҹҹдёӯзҡ„е®№еҷЁпјҢз”ЁдәҺз»„з»Үе’Ңз®ЎзҗҶеҜ№иұЎгҖӮеҹҹз®ЎзҗҶе‘ҳеҸҜд»ҘдҪҝз”Ё OU жқҘеҲҶз»„з”ЁжҲ·гҖҒи®Ўз®—жңәе’Ңе…¶д»–иө„жәҗпјҢе№¶еә”з”Ёз»„зӯ–з•ҘгҖӮOU з»“жһ„е…Ғи®ёзҒөжҙ»зҡ„з®ЎзҗҶжқғйҷҗе§”жҙҫе’ҢеҲҶеұӮж¬Ўзҡ„з®ЎзҗҶгҖӮ

# **7еӨ§ж ёеҝғз»„д»¶**

#

##

## з”ЁжҲ·е’Ңз»„

**з”ЁжҲ·**жҳҜActive DirectoryдёӯжңҖеҹәжң¬зҡ„еҜ№иұЎд№ӢдёҖпјҢд»ЈиЎЁзҪ‘з»ңдёӯзҡ„дёӘдҪ“пјҢеҸҜд»ҘжҳҜдәәжҲ–жңҚеҠЎиҙҰжҲ·гҖӮжҜҸдёӘз”ЁжҲ·еҜ№иұЎеҢ…еҗ«д»ҘдёӢе…ій”®дҝЎжҒҜпјҡ

з”ЁжҲ·еҗҚпјҡз”ЁжҲ·зҡ„е”ҜдёҖж ҮиҜҶгҖӮ

еҜҶз Ғпјҡз”ЁдәҺиә«д»ҪйӘҢиҜҒгҖӮ

дёӘдәәдҝЎжҒҜпјҡеҰӮе§“еҗҚгҖҒиҒҢеҠЎгҖҒиҒ”зі»з”өиҜқзӯүгҖӮ

з»„жҲҗе‘ҳиә«д»Ҫпјҡз”ЁжҲ·жүҖеұһзҡ„з»„пјҢз”ЁдәҺжқғйҷҗз®ЎзҗҶгҖӮ

**з»„**жҳҜз”ЁжҲ·зҡ„йӣҶеҗҲпјҢз”ЁдәҺз®ҖеҢ–жқғйҷҗз®ЎзҗҶгҖӮз»„еҲҶдёәдёӨз§Қзұ»еһӢпјҡ

е®үе…Ёз»„пјҡз”ЁдәҺеҲҶй…Қжқғйҷҗе’Ңи®ҝй—®жҺ§еҲ¶гҖӮйҖҡиҝҮе°Ҷз”ЁжҲ·ж·»еҠ еҲ°е®үе…Ёз»„пјҢеҸҜд»Ҙз®ҖеҢ–жқғйҷҗз®ЎзҗҶиҝҮзЁӢгҖӮ

еҲҶеҸ‘з»„пјҡдё»иҰҒз”ЁдәҺз”өеӯҗйӮ®д»¶еҲҶеҸ‘пјҢдёҚе…·еӨҮе®үе…ЁеҠҹиғҪгҖӮ

## еҹҹеҗҚзі»з»ҹпјҲDNSпјү

DNSпјҲDomain Name SystemпјүеңЁActive Directoryдёӯжү®жј”дәҶйҮҚиҰҒи§’иүІгҖӮе®ғз”ЁдәҺе°ҶеҹҹеҗҚиҪ¬жҚўдёәIPең°еқҖпјҢеё®еҠ©е®ўжҲ·з«ҜжүҫеҲ°еҹҹжҺ§еҲ¶еҷЁе’Ңе…¶д»–зҪ‘з»ңиө„жәҗгҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

жңҚеҠЎе®ҡдҪҚпјҲSRVпјүи®°еҪ•пјҡADдҪҝз”ЁSRVи®°еҪ•её®еҠ©е®ўжҲ·з«Ҝе®ҡдҪҚеҹҹжҺ§еҲ¶еҷЁгҖҒе…ЁеұҖзј–еҪ•жңҚеҠЎеҷЁзӯүгҖӮ

е‘ҪеҗҚи§ЈжһҗпјҡжҸҗдҫӣеҹҹеҗҚеҲ°IPең°еқҖзҡ„иҪ¬жҚўпјҢзЎ®дҝқзҪ‘з»ңйҖҡдҝЎзҡ„йЎәеҲ©иҝӣиЎҢгҖӮ

еҹҹеұӮж¬Ўз»“жһ„пјҡж”ҜжҢҒADзҡ„еҹҹе‘ҪеҗҚз©әй—ҙпјҢдҪҝзҪ‘з»ңиө„жәҗе’ҢжңҚеҠЎзҡ„и®ҝй—®жӣҙеҠ ж–№дҫҝгҖӮ

##

## з»„зӯ–з•ҘпјҲGroup Policyпјү

з»„зӯ–з•ҘжҳҜдёҖз§ҚйӣҶдёӯз®ЎзҗҶз”ЁжҲ·е’Ңи®Ўз®—жңәи®ҫзҪ®зҡ„еҠҹиғҪгҖӮйҖҡиҝҮз»„зӯ–з•ҘпјҢз®ЎзҗҶе‘ҳеҸҜд»ҘжҺ§еҲ¶еҗ„з§Қй…ҚзҪ®е’Ңе®үе…Ёи®ҫзҪ®гҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

зӯ–з•Ҙи®ҫзҪ®пјҡй…ҚзҪ®е®үе…Ёи®ҫзҪ®гҖҒиҪҜд»¶е®үиЈ…гҖҒи„ҡжң¬зӯүгҖӮ

з»§жүҝе’ҢиҰҶзӣ–пјҡз»„зӯ–з•ҘеҸҜд»Ҙеә”з”ЁдәҺеҹҹгҖҒOUпјҲз»„з»ҮеҚ•дҪҚпјүе’Ңжң¬ең°и®Ўз®—жңәпјҢе№¶дё”ж”ҜжҢҒз»§жүҝе’ҢиҰҶзӣ–жңәеҲ¶гҖӮ

з®ЎзҗҶжЁЎжқҝпјҡжҸҗдҫӣйў„е®ҡд№үзҡ„зӯ–з•Ҙи®ҫзҪ®жЁЎжқҝпјҢз®ҖеҢ–й…ҚзҪ®иҝҮзЁӢгҖӮ

з»„зӯ–з•ҘеҜ№иұЎпјҲGPOпјүпјҡз»„зӯ–з•ҘеҜ№иұЎжҳҜз»„зӯ–з•Ҙзҡ„е…·дҪ“е®һзҺ°гҖӮжҜҸдёӘGPOеҢ…еҗ«еӨҡдёӘзӯ–з•Ҙи®ҫзҪ®пјҢеҸҜд»Ҙй“ҫжҺҘеҲ°еҹҹгҖҒOUжҲ–з«ҷзӮ№гҖӮ

## иҪ»йҮҸзә§зӣ®еҪ•и®ҝй—®еҚҸи®®пјҲLDAPпјү

LDAPпјҲLightweight Directory Access ProtocolпјүжҳҜз”ЁдәҺи®ҝй—®е’Ңз®ЎзҗҶзӣ®еҪ•жңҚеҠЎзҡ„еҚҸи®®гҖӮе®ғжҳҜADзҡ„ж ёеҝғйҖҡдҝЎеҚҸи®®пјҢж”ҜжҢҒзӣ®еҪ•ж•°жҚ®зҡ„жҹҘиҜўе’Ңдҝ®ж”№гҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

зӣ®еҪ•жҹҘиҜўпјҡе…Ғи®ёе®ўжҲ·з«ҜжҹҘиҜўзӣ®еҪ•жңҚеҠЎдёӯзҡ„ж•°жҚ®пјҢеҰӮз”ЁжҲ·дҝЎжҒҜгҖҒз»„жҲҗе‘ҳиә«д»ҪзӯүгҖӮ

ж•°жҚ®ж“ҚдҪңпјҡж”ҜжҢҒеҜ№зӣ®еҪ•еҜ№иұЎзҡ„ж·»еҠ гҖҒеҲ йҷӨгҖҒдҝ®ж”№зӯүж“ҚдҪңгҖӮ

зӣ®еҪ•еҗҢжӯҘпјҡз”ЁдәҺеңЁдёҚеҗҢзӣ®еҪ•жңҚеҠЎд№Ӣй—ҙеҗҢжӯҘж•°жҚ®гҖӮ

##

## еҹҹжҺ§еҲ¶еҷЁпјҲDCпјү

еҹҹжҺ§еҲ¶еҷЁDCжҳҜиҝҗиЎҢActive DirectoryжңҚеҠЎзҡ„жңҚеҠЎеҷЁпјҢиҙҹиҙЈеӨ„зҗҶиә«д»ҪйӘҢиҜҒиҜ·жұӮе’Ңзӣ®еҪ•жңҚеҠЎзҡ„жүҖжңүж“ҚдҪңгҖӮе®ғжҳҜеӯҳеӮЁADж•°жҚ®еә“зҡ„ең°ж–№пјҢеҢ…еҗ«е…ідәҺеҹҹдёӯзҡ„з”ЁжҲ·гҖҒи®Ўз®—жңәгҖҒз»„еҸҠе…¶д»–еҜ№иұЎзҡ„жүҖжңүдҝЎжҒҜгҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

иә«д»ҪйӘҢиҜҒпјҡDCиҙҹиҙЈеҜ№з”ЁжҲ·е’Ңи®Ўз®—жңәиҝӣиЎҢиә«д»ҪйӘҢиҜҒпјҢзЎ®дҝқеҸӘжңүз»ҸиҝҮйӘҢиҜҒзҡ„е®һдҪ“жүҚиғҪи®ҝй—®зҪ‘з»ңиө„жәҗгҖӮе®ғж”ҜжҢҒKerberosе’ҢNTLMзӯүиә«д»ҪйӘҢиҜҒеҚҸи®®гҖӮ

жҺҲжқғпјҡж №жҚ®з”ЁжҲ·жҲ–и®Ўз®—жңәзҡ„иә«д»ҪеҸҠе…¶еңЁADдёӯзҡ„жқғйҷҗпјҢDCжҺҲдәҲжҲ–жӢ’з»қеҜ№иө„жәҗзҡ„и®ҝй—®гҖӮ

зӣ®еҪ•жңҚеҠЎж“ҚдҪңпјҡDCеӨ„зҗҶжүҖжңүзӣ®еҪ•жңҚеҠЎзҡ„ж“ҚдҪңпјҢеҢ…жӢ¬еҜ№иұЎзҡ„еҲӣе»әгҖҒеҲ йҷӨгҖҒдҝ®ж”№е’ҢжҹҘиҜўгҖӮ

еӨҚеҲ¶пјҡеңЁжӢҘжңүеӨҡдёӘDCзҡ„зҺҜеўғдёӯпјҢADйҖҡиҝҮеӨҡдё»еӨҚеҲ¶пјҲMulti-Master ReplicationпјүеңЁеҗ„DCд№Ӣй—ҙеҗҢжӯҘж•°жҚ®пјҢзЎ®дҝқдёҖиҮҙжҖ§е’ҢеҶ—дҪҷжҖ§гҖӮ

## е…ЁеұҖзӣ®еҪ•жңҚеҠЎеҷЁпјҲGCSпјү

е…ЁеұҖзӣ®еҪ•жңҚеҠЎеҷЁжҳҜдёҖдёӘзү№ж®Ҡзҡ„еҹҹжҺ§еҲ¶еҷЁпјҢе®ғеҢ…еҗ«еҹҹеҶ…жүҖжңүеҜ№иұЎзҡ„йғЁеҲҶеұһжҖ§пјҢе№¶жҸҗдҫӣи·ЁеҹҹжҹҘиҜўжңҚеҠЎгҖӮGCSеңЁеӨҡеҹҹзҺҜеўғдёӯе°ӨдёәйҮҚиҰҒпјҢеӣ дёәе®ғиғҪжҸҗдҫӣеҝ«йҖҹгҖҒе…Ёйқўзҡ„зӣ®еҪ•жҹҘиҜўгҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

и·ЁеҹҹжҹҘиҜўпјҡGCSеҢ…еҗ«жүҖжңүеҹҹдёӯеҜ№иұЎзҡ„йғЁеҲҶеұһжҖ§пјҢеҸҜд»Ҙе“Қеә”жқҘиҮӘд»»дҪ•еҹҹзҡ„жҹҘиҜўиҜ·жұӮпјҢжҸҗдҫӣе…ЁеұҖзӣ®еҪ•жңҚеҠЎгҖӮ

зҷ»еҪ•йӘҢиҜҒпјҡеңЁз”ЁжҲ·зҷ»еҪ•ж—¶пјҢGCSжҸҗдҫӣеҝ«йҖҹйӘҢиҜҒз”ЁжҲ·еңЁе…¶д»–еҹҹдёӯзҡ„зҫӨз»„жҲҗе‘ҳиә«д»ҪгҖӮ

йғЁеҲҶеұһжҖ§йӣҶпјҲPASпјүпјҡGCSеӯҳеӮЁжҜҸдёӘеҜ№иұЎзҡ„дёҖйғЁеҲҶеұһжҖ§пјҢиҖҢдёҚжҳҜе…ЁйғЁеұһжҖ§пјҢиҝҷдәӣеұһжҖ§йҖҡеёёз”ЁдәҺжҹҘиҜўж“ҚдҪңгҖӮ

## иҜҒд№ҰжңҚеҠЎпјҲADCSпјү

ADCSжҳҜдёҖдёӘжү©еұ•з»„д»¶пјҢжҸҗдҫӣж•°еӯ—иҜҒд№Ұзҡ„еҲӣе»әгҖҒз®ЎзҗҶе’ҢеҲҶеҸ‘еҠҹиғҪпјҢеўһејәзҪ‘з»ңе®үе…ЁгҖӮдё»иҰҒеҠҹиғҪеҰӮдёӢпјҡ

иҜҒд№ҰйўҒеҸ‘жңәжһ„пјҲCAпјүпјҡз®ЎзҗҶе’ҢйўҒеҸ‘ж•°еӯ—иҜҒд№ҰгҖӮ

иҜҒд№ҰжіЁеҶҢзӮ№пјҡеё®еҠ©з”ЁжҲ·е’Ңи®ҫеӨҮиҜ·жұӮе’ҢиҺ·еҫ—иҜҒд№ҰгҖӮ

иҜҒд№ҰеҗҠй”ҖеҲ—иЎЁпјҲCRLпјүпјҡз®ЎзҗҶе’ҢеҸ‘еёғиҜҒд№ҰеҗҠй”ҖдҝЎжҒҜгҖӮ

# **6еӨ§е…ій”®еҚҸи®®**

**LDAP (Lightweight Directory Access Protocol)**пјҡLDAP жҳҜ AD и®ҝй—®е’ҢжҹҘиҜўзӣ®еҪ•дҝЎжҒҜзҡ„дё»иҰҒеҚҸи®®гҖӮе®ғжҸҗдҫӣдәҶдёҖз§Қж ҮеҮҶеҢ–зҡ„ж–№ејҸжқҘи®ҝй—®е’Ңж“ҚдҪңзӣ®еҪ•жңҚеҠЎпјҢдҪҝеҫ— AD еҸҜд»ҘдёҺе…¶д»– LDAP е…је®№зҡ„зі»з»ҹе’Ңеә”з”ЁзЁӢеәҸйӣҶжҲҗгҖӮ

**Kerberos**пјҡKerberos жҳҜ AD зҡ„дё»иҰҒиә«д»ҪйӘҢиҜҒеҚҸи®®пјҢжҸҗдҫӣејәеӨ§зҡ„е®үе…ЁжҖ§е’ҢеҚ•зӮ№зҷ»еҪ•пјҲSSOпјүеҠҹиғҪгҖӮKerberos дҪҝз”ЁзҘЁжҚ®жҺҲдәҲжңәеҲ¶пјҢзЎ®дҝқз”ЁжҲ·иә«д»ҪйӘҢиҜҒзҡ„е®үе…ЁжҖ§е’Ңжңүж•ҲжҖ§гҖӮ

**DNS (Domain Name System)**пјҡDNS жҳҜ AD е‘ҪеҗҚз©әй—ҙзҡ„йҮҚиҰҒз»„жҲҗйғЁеҲҶпјҢз”ЁдәҺи§ЈжһҗеҹҹеҗҚе’Ң IP ең°еқҖгҖӮAD дҫқиө– DNS жқҘе®ҡдҪҚеҹҹжҺ§еҲ¶еҷЁе...
