## GIMP 3.0 RC1 开始测试

[![The Gimp](https://icon.solidot.org/images/topics/topicgimp.png?123)](/search?tid=108 "The Gimp")

[Wilson](/~Wilson) (42865)发表于 2024年11月06日 14时54分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=79693&appkey=1370085986&title=GIMP%203.0%20RC1%20%E5%BC%80%E5%A7%8B%E6%B5%8B%E8%AF%95 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自月海沉船**

图像编辑自由软件项目 GIMP 终于发布了 v3.0 的第一个 RC 版本，但目前只能通过 Git 获取，官网尚未提供二进制版本下载。GIMP 3.0 是一次用户期待已久的重大更新，其变化包括从 GTK2 移植到 GTK3，从 Python 2 过渡到 Python 3，基本实现非破坏性编辑，等等。开发者原计划在 2023 年内发布，但之后又推迟到了 2024 年 5 月，RC1 之后正式版可能要到明年了。**更新**：二进制下载已释出。
https://www.gimp.org/news/2024/11/06/gimp-3-0-RC1-released/
https://gitlab.gnome.org/GNOME/gimp/-/commit/76036f4833f1131fe6a74c1a7837e246bb0ee680
