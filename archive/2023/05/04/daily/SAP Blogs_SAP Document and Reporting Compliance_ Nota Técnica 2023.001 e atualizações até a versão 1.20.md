Oi pessoal,

O time de SAP Document and Reporting Compliance, opção de emissão de fatura de entrada e de faturamento de saída para o Brasil entregou atualizações sobre regras de mapeamento e validação de arquivos XML. A mudança atende a **Nota Técnica 2023.001** da SEFAZ até a versão 1.20. A atualização alterou o layout do XML adicionando novos campos no grupo Combustível. Também foram criados novos grupos de impostos de ICMS e adicionados campos de ICMS no grupo Total.

![](/legacyfs/online/storage/blog_attachments/2023/05/image_pt-min.png)

SAP Document and Reporting Compliance: Nota Técnica 2023.001

Você pode implementar essas alterações na opção de emissão de fatura de entrada e de faturamento de saída. Siga as instruções nas seguintes SAP Notes.

Se você usa **opção de faturamento de saída**:

* SAP Note [3311653](https://launchpad.support.sap.com/#/notes/3311653)

* SAP Note [3311652](https://launchpad.support.sap.com/#/notes/3311652)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

Se você usa **opção de emissão de fatura de entrada**:

* SAP Note [3317013](https://launchpad.support.sap.com/#/notes/3317013)

* SAP Note [3319139](https://launchpad.support.sap.com/#/notes/3319139)

* SAP Note [3325410](https://launchpad.support.sap.com/#/notes/3325410)

* SAP Note [3328353](https://launchpad.support.sap.com/#/notes/3328353)

A nova versão entra em vigor em 1º de maio de 2023, conforme a secretaria da fazenda (SEFAZ).

Para mais informações sobre SAP Document and Reporting Compliance, opção de emissão de fatura de entrada e de faturamento de saída para o Brasil, consulte as seguintes páginas do SAP Help Portal:

* [SAP Document and Reporting Compliance, opção de emissão de fatura de entrada para o Brasil](https://help.sap.com/docs/DC_NFe/75478ef83fbe47b6836f33609268796b/ffaf0ea3291d4ddd9be5e5ba2c641e27.html)

* [SAP Document and Reporting Compliance, opção de faturamento de saída para o Brasil](https://help.sap.com/docs/SLH_NFe/71b3b178e5cb4a50a88cf2985a14f17c/b2f5ad5578574616855dda8638e2ee92.html)

Você pode implementar as mudanças na solução SAP Electronic Invoicing for Brazil (SAP NFE). Siga as instruções nas seguintes SAP Notes.

Se você usa **SAP NFE outbound nas versões superiores**:

* SAP Note [3311653](https://launchpad.support.sap.com/#/notes/3311653)

* SAP Note [3311652](https://launchpad.support.sap.com/#/notes/3311652)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

Se você usa **SAP NFE outbound nas versões inferiores**:

* SAP Note [3315005](https://launchpad.support.sap.com/#/notes/3315005)

* SAP Note [3314894](https://launchpad.support.sap.com/#/notes/3314894)

* SAP Note [3324374](https://launchpad.support.sap.com/#/notes/3324374)

* SAP Note [3327881](https://launchpad.support.sap.com/#/notes/3327881)

Se você usa **SAP NFE inbound nas versões superiores e inferiores**:

* SAP Note [3317026](https://launchpad.support.sap.com/#/notes/3317026)

* SAP Note [3325411](https://launchpad.support.sap.com/#/notes/3325411)
