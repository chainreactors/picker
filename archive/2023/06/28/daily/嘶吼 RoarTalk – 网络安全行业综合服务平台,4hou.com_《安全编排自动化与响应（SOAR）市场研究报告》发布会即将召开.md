# 《安全编排自动化与响应（SOAR）市场研究报告》发布会即将召开

嘶吼
[活动专区](https://www.4hou.com/category/xactivity)
2023-06-27 12:00:00

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)119879

收藏

导语：报告发布会直播时间：6月29日14:00，欢迎预约观看！

由国家网络安全产业园区（通州园）和嘶吼安全产业研究院联合撰写的**《安全编排自动化与响应（SOAR）市场研究报告》**将于**6月29日14:00**举行线上发布会，欢迎大家提前预约，以便进入直播间进行观看。

![微信图片_20230627101502.jpg](https://img.4hou.com/uploads/ueditor/php/upload/image/20230627/1687837648144509.jpg "1687835700417312.jpg")随着安全团队在安全操作中对应用自动化需求的不断提高，SOAR加速建设生态圈，通过持续不断连接不同的解决方案，大幅提升了安全运营的效能和成熟度。

2022年，Gartner正式发布了2022年《Market Guide for Security Orchestration，Automation and Response Solutions》报告，详细分析了SOAR的市场发展情况并给出相关建议，帮助政企组织安全负责人正确评估SOAR如何支持和优化其更广泛的安全运营能力，并将SOAR定义为“将事件响应、流程编排自动化及威胁情报平台管理功能集中在一起的解决方案”。

Gartner 对于这一市场的描述为：“SOAR 解决方案是三种历史上不同的技术的融合，这些技术具有一些常见的属性和一些使用它们的普通用户。这些技术在历史上是独特的，并以一种形式为安全运营团队提供实用程序，可以减轻大量体力劳动以提高安全性的产品运营职能。”

对此，嘶吼安全产业研究院经过深入调研与访谈，精心撰写《安全编排自动化与响应（SOAR）市场研究报告》。本报告涵盖SOAR的发展背景、技术演变、挑战、市场态势、企业竞争分析、未来趋势等内容，在此次发布会上，嘶吼安全产业研究院分析师将针对报告内容，进行详细阐释。

此外，嘶吼特别邀请到深度参与本次报告调研的三家优秀安全厂商：天融信、众智维和雾帜智能，为大家带来精彩的议题分享，全面解读SOAR技术的应用落地情况。
