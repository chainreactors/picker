2022 年 4 月微软初次在 Microsoft Edge 中向极少数用户推出名为安全网络的新服务，其实就是 VPN 服务，启用后 Microsoft Edge 的所有连接都将通过 VPN 加密传输，但这仅限于 Edge 的连接，并不影响系统本身的网络流量。

![](https://www.landiannews.com/wp-content/uploads/2022/04/image-1-2022-04-28-12-18-45-56.jpg)

经过一年的测试后，微软这几天似乎在广泛推出 Microsoft Edge VPN 服务，不少用户升级 Microsoft Edge 110 版后出现安全网络提示，**比较逗的是虽然微软向不少用户推送了 VPN，但不少用户是看到了提示然而还不能连接，估计后面一段时间微软才会支持大多数用户使用。**

区域支持方面，之前微软主要是在美国测试的，现在不支持美国用户，更多国家 / 地区的用户也收到了推送，不过微软关于 Edge VPN 的支持文档里并未提到哪些国家 / 地区支持此服务。

数据流量方面：用户每月可以免费使用 1GB 的流量，总共提供 15GB 流量，超过了就得付费使用了，现在套餐价格信息也还没公布。

技术细节方面：Edge VPN 是 CloudFlare 提供的，微软的目的是加密流量而非隐藏用户身份，加密流量指的是避免用户浏览数据被窃取、避免被广告商跟踪。

CloudFlare 会给用户分配最近的节点 (一般是同城市的)，因此用户搜索天气时搜索引擎还能自动判断用户所在的城市给出当地天气，但 IP 地址是 CloudFlare 节点地址，广告网络并不能根据这个 IP 来归因用户身份，至少无法单纯的通过 IP 地址推送精准广告。

另外在美国下载盗版电影 / 音乐都是非法的，Edge VPN 大概率也不能保证出现这类操作时不会被执法机构上门。
