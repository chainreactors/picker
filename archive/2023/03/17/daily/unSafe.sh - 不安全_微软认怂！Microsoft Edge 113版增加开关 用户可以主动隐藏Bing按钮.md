在本周推出的 Microsoft Edge 111 正式版中，[微软在浏览器右上角添加一个硕大的 Bing 按钮](https://www.landiannews.com/archives/97837.html)，这个是新功能名为 Edge Copilot。

Edge Copilot 集成 Bing Chat，可以快速发起会话，也可以读取网页内容提取内容，用户如果在网页写东西也可以让 Edge Copilot 帮忙扩展内容等。

[![微软认怂！Microsoft Edge 113版增加开关 用户可以主动隐藏Bing按钮](https://img.lancdn.com/landian/2023/03/97866-2.png)](https://img.lancdn.com/landian/2023/03/97866-2.png)

但并非所有用户都喜欢这个功能，微软一如既往地不提供设置项让用户关闭，结果引起不少用户的抱怨。

所幸微软还预留了个企业政策可以关闭，通过修改注册表后可以移除 Bing 按钮让浏览器右上角恢复正常。事实证明微软应该是看到了用户的抱怨，所以在今天发布的 Microsoft Edge Canary 113.x 版中，微软在设置里增加了选项，用户无需修改注册表也可以直接关闭 Bing 按钮。

不过 Microsoft Edge 113 抵达稳定版大约还需要一个月，[所以暂时看着碍眼的用户还是修改注册表先将它删除吧](https://www.landiannews.com/archives/97833.html)，后面有设置项了可以再把注册表键值修改下。

设置路径：Microsoft Edge 113+、设置、侧栏、应用和通知设置、发现 (Discover)、显示发现按钮，关闭即可。
