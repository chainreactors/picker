# 首发x86\_64电脑上最优雅的学习Linux/Android平台eBPF方案

原创

丰生强

软件安全与逆向分析

很多学习eBPF的学员是Linux云开发服务器开发的朋友，他们常年使用macOS与Linux作为工作主力机，没有Windows电脑，这给课程的观看带来不便。

为了解决这个问题，花了一周时间，研究出了在x86\_64电脑上部署测试与学习eBPF的最优雅方案！

首先，设备的选择要兼容Windows的课程播放器，其次设备价格要优惠，且性能要强，便捷性好。最终选中了零刻的ser5pro迷你电脑，16g内存500g ssd的配置，只要2k出头！并且是R7 5800h的处理器，所有的配置都满足课程播放与学习的需求！‍

![](https://mmbiz.qpic.cn/mmbiz_png/k9S5z61JPnaQ6ib6fia7ibPLoFovN5vVqkf3na3L7oaL2aLFxfVnEq7tf8PwbOqCNmUDpfXS8uEC4ic5UT3iaaUNY3g/640?wx_fmt=png)

Linux的eBPF学习环境搭建相对简单，只要处理好几个内核配置，多种方案可供选择。最终，选择与课程同步的环境，更新到最新的Ubuntu 22.04.2 LTS，内核是5.19了，与5.15一样的用，5.19也没加啥新的eBPF特性。‍‍‍‍‍‍

Android eBPF的配置就麻烦一些，主要体现在多种方案的测试与踩坑，最终的实现方法不难，但不多测试就不会有这个方案。目前，完美的安装各种arm64的apk，运行稳定性、效率与eBPF兼容性都很美。

目前，手上折腾的两台serpro5设备搞个活动，方便没有Windows设备的学员弄走。

**eBPF课程全季（含第1、2、3季）加上全新8核16G迷你电脑ser5pro一台，已经配置好Linux与安卓eBPF开发环境，也可作为课程观看设备，开箱即用！加上App隐私合规与安全分析探针实战课程第一季与第二季课程，价值3499+1268+2299元，现活动价格5299元。**

购买本套餐的朋友，找本人要本方案的视频内容。

**此方案的全部细节将会在eBPF系列课程第4季eBPF Main Feature完全篇揭开！**敬请期待。‍‍‍

本周五更新安卓eBPF环境搭建之rock5b开发板方案，方便喜欢折腾与想玩云手机的朋友。届时，将会揭开一切相关秘密！‍
