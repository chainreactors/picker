AhnLab е®үе…Ёжғ…жҠҘдёӯеҝғпјҲASECпјүиҝ‘жңҹеҸ‘зҺ°пјҢдёҖеңәеӨ§и§„жЁЎжҒ¶ж„ҸиҪҜд»¶дј ж’ӯжҙ»еҠЁжӯЈеңЁеҲ©з”Ё GitHub еҲҶеҸ‘ **SmartLoader**вҖ”вҖ”дёҖз§Қеёёиў«з”ЁдәҺжҠ•ж”ҫдҝЎжҒҜзӘғеҸ–зЁӢеәҸпјҲInfoStealerпјүзҡ„еҠ иҪҪеҷЁпјҢеҢ…жӢ¬ RhadamanthysгҖҒRedline е’Ң Lumma Stealer зӯүгҖӮ

жҚ® ASEC д»Ӣз»ҚпјҢеЁҒиғҒиЎҢдёәиҖ…дјҡвҖңзІҫеҝғдјӘиЈ…д»“еә“пјҢдҪҝе…¶зңӢиө·жқҘеғҸжҳҜеҗҲжі•йЎ№зӣ®вҖқпјҢе№¶й”Ғе®ҡжёёжҲҸеӨ–жҢӮгҖҒиҪҜд»¶з ҙи§ЈгҖҒиҮӘеҠЁеҢ–е·Ҙе…·зӯүзғӯй—Ёжҗңзҙўе…ій”®иҜҚиҝӣиЎҢжҠ•ж”ҫгҖӮжҜҸдёӘд»“еә“йғҪеҢ…еҗ«дёҖдёӘ README ж–Үд»¶е’ҢдёҖдёӘеҺӢзј©еҢ…пјҢе…¶дёӯзҡ„еҺӢзј©еҢ…жҡ—и—ҸжҒ¶ж„ҸиҪҪиҚ·гҖӮ

еҪ“з”ЁжҲ·жҗңзҙўзү№е®ҡе…ій”®иҜҚж—¶пјҢвҖңеҢ…еҗ« SmartLoader жҒ¶ж„ҸиҪҜд»¶зҡ„ GitHub д»“еә“еҫҖеҫҖдјҡеҮәзҺ°еңЁжҗңзҙўз»“жһңеүҚеҲ—вҖқпјҢжһҒжҳ“и®©жҜ«ж— йҳІеӨҮзҡ„еҸ—е®іиҖ…дёӢиҪҪгҖӮ

иҝҷдәӣ README ж–Үд»¶еҶ…е®№е®Ңж•ҙгҖҒж’°еҶҷдё“дёҡпјҢйҖҡеёёеҢ…еҗ«йЎ№зӣ®жҰӮиҝ°гҖҒзӣ®еҪ•гҖҒдё»иҰҒеҠҹиғҪгҖҒе®үиЈ…дёҺдҪҝз”ЁиҜҙжҳҺзӯүпјҢиҗҘйҖ еҮәдёҖз§ҚеҸҜдҝЎзҡ„еҒҮиұЎгҖӮз„¶иҖҢпјҢжҢүиҜҙжҳҺж“ҚдҪңеҗҺпјҢз”ЁжҲ·дјҡдёӢиҪҪеҲ°дёҖдёӘеҢ…еҗ«еӣӣдёӘж–Үд»¶зҡ„ ZIP еҺӢзј©еҢ…вҖ”вҖ”е…¶дёӯеҢ…жӢ¬дёӨдёӘеҗҲжі•еҸҜжү§иЎҢж–Үд»¶гҖҒдёҖдёӘжҒ¶ж„Ҹжү№еӨ„зҗҶж–Үд»¶пјҢд»ҘеҸҠдёҖдёӘж··ж·ҶиҝҮзҡ„ Lua и„ҡжң¬гҖӮ

ж”»еҮ»й“ҫд»ҺеҸ—е®іиҖ…жү§иЎҢ **Launcher.cmd** ејҖе§ӢпјҢиҜҘжү№еӨ„зҗҶж–Үд»¶дјҡйҖҡиҝҮеҗҲжі•зҡ„ **luajit.exe** еҠ иҪҪеҷЁиҝҗиЎҢжҒ¶ж„Ҹ Lua и„ҡжң¬гҖӮSmartLoader йҡҸеҗҺдјҡе°ҶиҮӘиә«еӨҚеҲ¶еҲ° **%AppData%\ODE3** зӣ®еҪ•пјҢе№¶еҲӣе»әеҗҚдёә **SecurityHealthService\_ODE3** зҡ„и®ЎеҲ’д»»еҠЎпјҢд»Ҙе®һзҺ°жҢҒд№…еҢ–гҖӮ

дёҖж—ҰиҝҗиЎҢпјҢSmartLoader дјҡвҖңеҗ‘ C2 жңҚеҠЎеҷЁеҸ‘йҖҒеҸ—ж„ҹжҹ“з”өи„‘зҡ„жҲӘеӣҫеҸҠзі»з»ҹдҝЎжҒҜвҖқпјҢз„¶еҗҺж №жҚ®жңҚеҠЎеҷЁжҢҮд»Өжү§иЎҢжӣҙеӨҡжҒ¶ж„Ҹе‘Ҫд»ӨгҖӮе…¶дёҺ C2 зҡ„жүҖжңүйҖҡдҝЎеқҮйҮҮз”Ё Base64 зј–з Ғе№¶з»ҸиҝҮж··ж·ҶеӨ„зҗҶгҖӮ

ASEC еҲҶжһҗеҸ‘зҺ°пјҢSmartLoader дјҡд»Һ GitHub дёӢиҪҪдёүдёӘиҪҪиҚ·ж–Үд»¶пјҢе…¶дёӯеҢ…жӢ¬ **\_x64.bin** е’Ң **\_x86.bin**пјҢдәҢиҖ…еқҮиў«зЎ®и®Өдёә Rhadamanthys дҝЎжҒҜзӘғеҸ–зЁӢеәҸзҡ„ shellcodeгҖӮиҜҘжҒ¶ж„ҸиҪҜд»¶дјҡжіЁе…Ҙ **rundll32.exe** е’Ң **dllhost.exe** зӯүиҝӣзЁӢпјҢд»ҺиҖҢзӘғеҸ–з”өеӯҗйӮ®д»¶гҖҒFTP д»ҘеҸҠзҪ‘дёҠй“¶иЎҢиҙҰжҲ·зҡ„еҮӯиҜҒдҝЎжҒҜгҖӮ

AhnLab иӯҰе‘Ҡз§°пјҡвҖңSmartLoader дё»иҰҒз”ЁдәҺдёӢиҪҪдҝЎжҒҜзӘғеҸ–зұ»жҒ¶ж„ҸиҪҜд»¶пјҢд№ҹжңүи®ёеӨҡжЎҲдҫӢжҳҫзӨәе®ғиў«з”ЁжқҘжү§иЎҢ RhadamanthysгҖҒRedlineгҖҒLumma Stealer зӯүе…¶д»–жҒ¶ж„ҸзЁӢеәҸгҖӮвҖқ
