今天给大家推送的是美军指挥与参谋学院2023年7月发布的写作指南：《专业写作》。

**快速指南：**

|  |  |  |
| --- | --- | --- |
| 流程 | 写作活动 | 标准 |
| 制定计划 | 分析任务  制定写作计划 | 为实现目标而写作 |
| 收集资料 | 收集强有力的证据  避免偏见  有条理地做笔记  记录信息来源 | 连贯地提出论点  用有力的证据和合理的推理支持论点  引用所有资料来源，并正确设置引用格式 |
| 起草内容 | 撰写摘要、引言、主体内容和结论  按逻辑组织主体内容  清楚地陈述主张，用证据和推理来支持这些主张  考虑反驳意见  使用引语、释义和摘要整合资料来源 | 为实现目标而写作  连贯一致地提出论点  用有力的证据和合理的推理支持论点  撰写清晰的引言、主体内容和结论  在引言中尽早阐明论点  按逻辑顺序排列段落和章节 |
| 修改内容 | 修改内容和条理  写作就是思考，如有必要，可做更多的研究和起草工作。  不要对任何内容过于偏爱  修订段落  使用段落过渡 | 连贯一致地提出论点  用有力的证据和合理的推理支持论点  撰写清晰的引言、主体内容和结论  在引言中尽早阐明论点  按逻辑顺序排列段落和章节  围绕一个主题组织段落 |
| 编辑 | 使文章清晰明了  擅长使用简单的词语和短句  使用具体的语言  最好使用主动语态  书写有连贯性的句子  删除不必要的词语  避免模糊表达和吞吞吐吐  避免称谓语  保持专业语气 | 使用简洁明了的句子  使用主动语态  使用简单的词语，省略不必要的词语  保持专业的语气，正式但不失对话性和自信  使用常规的标点、拼写和语法  引用所有资料来源，格式正确 |
| 发布 | 在发布或上交之前，请确认：  论文达到了目的  摘要和结论一致  论文符合所有管理要求（如字数）  论文上有姓名或身份证号码  文件格式正确  文件类型正确（有些教授可能会要求 PDF 格式）  引用了所有资料来源，引用资料完整、准确 | 为实现目标而写作  引用所有资料来源，并正确设置引文格式  文件格式正确 |
