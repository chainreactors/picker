## Thunderbird 预览新 UI

[![开源](https://icon.solidot.org/images/topics/topicopensource.png?123)](/search?tid=3 "开源")

[Wilson](/~Wilson) (42865)发表于 2023年02月15日 22时02分 星期三 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74144&appkey=1370085986&title=Thunderbird%20%E9%A2%84%E8%A7%88%E6%96%B0%20UI "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自梦境救援**

Thunderbird 开发者此前宣布正在重新构建软件的 UI，今年夏天发布 Thunderbird 115“Supernova” 时将带来这一努力的首个成果。现在它预览了新 UI 的文件夹面板。新面板 Unified Folder 模式中各个按钮之间的空隙更大，密度更低。这可能会惹恼喜欢紧凑界面的老用户。开发者表示在发怒前“先听我一言”，Unified Folder 模式可以关闭，关闭后的布局和旧版基本一致。它如此设计是为了满足不同用户的需求，部分用户喜欢现有的紧凑界面，看一眼就能掌握所有需要的信息；但还有部分用户不喜欢所有信息都挤在一起。
https://blog.thunderbird.net/2023/02/thunderbird-115-supernova-preview-the-new-folder-pane/
