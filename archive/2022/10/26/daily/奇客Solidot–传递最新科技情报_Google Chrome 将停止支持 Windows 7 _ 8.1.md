## Google Chrome 将停止支持 Windows 7 / 8.1

[![Chrome](https://icon.solidot.org/images/topics/topicchrome.png?123)](/search?tid=154 "Chrome")

[WinterIsComing](/~WinterIsComing) (31822)发表于 2022年10月25日 16时42分 星期二 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73162&appkey=1370085986&title=Google%20Chrome%20%E5%B0%86%E5%81%9C%E6%AD%A2%E6%94%AF%E6%8C%81%20Windows%207%20%2F%208.1 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自机器岛**

Google [宣布](https://www.bleepingcomputer.com/news/google/google-chrome-to-drop-support-for-windows-7-81-in-feb-2023/)其 Chrome 浏览器将从 2023 年 2 月起停止支持 Windows 7 / 8.1。Google Chrome 支持经理称，计划于明年 2 月 7 日发布的 Chrome 110 将终止对 Windows 7 / 8.1 的支持，用户设备需要 Windows 10 及以上版本才能运行 Chrome 未来版本。Google 终止支持的决定与微软 Windows 生命周期政策一致，软件巨人计划于 2023 年 1 月结束 Windows 7 / 8.1 的扩展安全更新支持。目前 Windows 7 在操作系统市场的占有率仍然逾 10%，而 Windows 8.1 只有 2.7%。
