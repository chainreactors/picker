Proton Pass – 免费、开源密码管理器，隐私优先的新选择

**Proton Pass** 是一款来自瑞士的免费、开源的密码管理器，支持 iOS、Android，以及 Chrome、Edge、Firefox、Brave 浏览器扩展，旗下还有隐私邮箱、日历、网盘等服务。适合于隐私优先用户。@[Appinn](https://www.appinn.com/proton-pass/)

![Proton Pass - 免费、开源密码管理器，隐私优先的新选择](https://static1.appinn.com/images/202307/appinn-feature-images-2023-07-30t124442-880.jpg!o "Proton Pass - 免费、开源密码管理器，隐私优先的新选择 1")

Proton Pass 在[博客](https://proton.me/blog/pass-open-source-security-audit)中说：

鉴于使用密码管理器保护敏感信息，因此准确了解其中发生的情况至关重要。由于 Proton Pass 是开源的，任何人都可以检查我们的代码并确保应用程序按描述工作。

Proton Pass 在 GitHub 开源了 iOS、Android 客户端源码、浏览器扩展源码。

并且通过了独立安全审核。Proton Pass 的代码在 5 月和 6 月接受了德国安全公司 Cure53 的安全审核。Cure53 对 Proton Pass 的评价是：“Cure53 对 Proton 进行的广泛而彻底的安全评估表明了他们对维持高水平安全的承诺。由于调查结果数量适中，而且大多数安全漏洞的严重程度有限，Proton 应用程序和平台的整体安全状况值得称赞。”

「[安全审核中报告](https://res.cloudinary.com/dbulfrlrz/images/v1689754709/wp/Cure53-proton-pass-20230717_41141c8436/Cure53-proton-pass-20230717_41141c8436.pdf?_i=AA)的所有问题均已解决，但中等严重性问题 PRO-01-003 WP1 除外，遗憾的是，由于 Android 平台限制，该问题目前无法解决（Android 操作系统当前不提供该信息）这将需要解决这个问题）。您可以亲自阅读 Proton Pass 审核报告。您还可以找到所有 Proton 服务的审核报告。」
