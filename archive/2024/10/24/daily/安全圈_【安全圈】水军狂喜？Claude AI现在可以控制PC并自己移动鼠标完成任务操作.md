**关键词**

人工智能

人工智能技术开发商 Anthropic 日前为 Claude AI 推出一项名为计算机使用的实验性公开测试版 API，这个 API 允许 Claude 控制计算机并移动鼠标完成任务操作。

简单来说就是你可以向 Claude AI 输入一串指令，接下来 Claude AI 将自己控制计算机并移动鼠标进行点击、使用键盘进行输入以完成特定任务。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/aBHpjnrGyliaTWdGRD6vLKNsGd99DcNtsL05Vf6pODyFxCkLicCLvEnAnuwPvf20oicBf3wicYN710oaACBMkPIU6Q/640?wx_fmt=png&from=appmsg)

**这个功能在多个场景都有实用意义：**

* 开发者可以借助此功能进行软件测试
* 研究人员可以借助改功能自动收集数据
* 办公人士可以用来批量处理某些数据
* 甚至网络水军还可以用来批量发布内容

**Anthropic 在博客中表示：**

*对于计算机的使用我们正在尝试一些全新的功能，我们并没有制作特定的工具来帮助 Claude 完成个别任务，而是教它一般的计算机技能，让它能够使用那些为人类设计的各种标准工具和软件程序。*

*开发者也可以集成这个 API 让 Claude 将特定指令转换为计算机指令，例如使用我的计算机和在线数据来填写此表格：检查表格、移动光标打开浏览器、打开某个网页、使用这个网页中的数据填写表格等。*

不过现阶段由于某些限制 Claude 的使用计算机功能还不够完善，例如不能执行某些常见的操作：滚动、拖动和缩放，而且这个 API 也可能会被安全软件识别为攻击从而影响使用。

当然最重要的就是我们在标题中提到的诸如网络水军这类非善意操作，Anthropic 开发了新的分类器用于检测和识别诸如发送垃圾邮件、批量伪造并发布错误信息、钓鱼或欺诈等，当检测到这类用途时任务就会被终止。
