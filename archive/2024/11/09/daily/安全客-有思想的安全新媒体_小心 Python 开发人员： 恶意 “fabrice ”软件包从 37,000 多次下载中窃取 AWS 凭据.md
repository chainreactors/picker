еҜ№дәҺ Python ејҖеҸ‘дәәе‘ҳе’Ңдә‘з®ЎзҗҶе‘ҳжқҘиҜҙпјҢдёҖдёӘд»ӨдәәжӢ…еҝ§зҡ„иҝӣеұ•жҳҜпјҢSocket з ”з©¶еӣўйҳҹеҸ‘зҺ°дәҶдёҖдёӘеҗҚдёә fabrice зҡ„жҒ¶ж„ҸиҪҜд»¶еҢ…пјҢе®ғдјӘиЈ…жҲҗеҗҲжі•дё”е№ҝжіӣдҪҝз”Ёзҡ„ Fabric SSH иҮӘеҠЁеҢ–еә“гҖӮиҮӘ 2021 е№ҙеңЁ PyPI дёҠеҸ‘еёғд»ҘжқҘпјҢfabrice е·ІжӮ„ж— еЈ°жҒҜең°еӨ–жі„дәҶж•Ҹж„ҹзҡ„ AWS еҮӯжҚ®пјҢеҪұе“ҚдәҶйӮЈдәӣеңЁдёҚзҹҘжғ…зҡ„жғ…еҶөдёӢе®үиЈ…дәҶиҜҘжҺ’еҚ°й”ҷиҜҜиҪҜд»¶еҢ…зҡ„з”ЁжҲ·гҖӮfabrice иҪҜд»¶еҢ…зҡ„дёӢиҪҪйҮҸе·Іи¶…иҝҮ 37000 ж¬ЎпјҢеҮёжҳҫдәҶжҒ¶ж„ҸиҪҜд»¶еңЁејҖжәҗиҪҜд»¶жәҗдёӯеёҰжқҘзҡ„жҢҒз»ӯйЈҺйҷ©гҖӮ

жҒ¶ж„Ҹ fabrice иҪҜд»¶еҢ…йҮҮз”ЁдәҶеӨҡз§ҚжҠҖжңҜжқҘдәӨд»ҳе…¶жңүж•ҲиҪҪиҚ·пјҢе№¶й’ҲеҜ№ Linux е’Ң Windows зҺҜеўғе®ҡеҲ¶дәҶдёҚеҗҢзҡ„жҒ¶ж„Ҹж“ҚдҪңпјҡ

1. **еңЁ Linux зі»з»ҹдёҠпјҡ** иҜҘиҪҜд»¶еҢ…дҪҝз”ЁдёҖдёӘеҗҚдёә linuxThread() зҡ„еҮҪж•°д»ҺеӨ–йғЁжңҚеҠЎеҷЁдёӢиҪҪе№¶жү§иЎҢйҡҗи—Ҹи„ҡжң¬гҖӮз ”з©¶еҸ‘зҺ°пјҢвҖңlinuxThread()еҮҪж•°дјҡеҲӣе»әдёҖдёӘйҡҗи—Ҹзӣ®еҪ•пјҲ~/.local/bin/vscodeпјүжқҘеӯҳеӮЁдёӢиҪҪзҡ„жңүж•ҲиҪҪиҚ·вҖқпјҢд»ҺиҖҢдҪҝжЈҖжөӢеҸҳеҫ—еӣ°йҡҫгҖӮжӯӨеӨ–пјҢиҜҘиҪҜд»¶еҢ…дјҡиҝһжҺҘеҲ°дёҖдёӘ IP ең°еқҖдёә 89.44.9.227 зҡ„ VPN жңҚеҠЎеҷЁжқҘдёӢиҪҪиҝҷдәӣжҒ¶ж„Ҹи„ҡжң¬пјҢе№¶дҪҝз”Ёж··ж·ҶжҠҖжңҜжқҘиәІйҒҝжЈҖжөӢгҖӮ
2. **еңЁ Windows зі»з»ҹдёҠпјҡ** Windows зүҲжң¬дҪҝз”Ё winThread() еҮҪж•°пјҢдҫқиө–дәҺеӯҳеӮЁеңЁеҸҳйҮҸ vv е’Ң zz дёӯзҡ„ base64 зј–з Ғжңүж•ҲиҪҪиҚ·гҖӮз ”з©¶и§ЈйҮҠиҜҙпјҢвҖңvv еҸҳйҮҸи§Јз Ғдёә VBScript (p.vbs)пјҢиҝҗиЎҢйҡҗи—Ҹзҡ„ Python и„ҡжң¬ (d.py)вҖқпјҢз„¶еҗҺдёӢиҪҪжӣҙеӨҡжҒ¶ж„ҸеҸҜжү§иЎҢж–Үд»¶гҖӮzz и„ҡжң¬йҖҡиҝҮи°ғеәҰд»»еҠЎйҮҚж–°жү§иЎҢжҒ¶ж„Ҹд»Јз ҒжқҘе»әз«ӢжҢҒд№…жҖ§пјҢеҚідҪҝеңЁзі»з»ҹйҮҚеҗҜеҗҺд№ҹиғҪдҝқжҢҒж”»еҮ»зҡ„еӯҳеңЁгҖӮ

fabrice зҡ„дё»иҰҒзӣ®зҡ„дјјд№ҺжҳҜзӘғеҸ– AWS еҮӯжҚ®гҖӮйҖҡиҝҮеҲ©з”Ё boto3 еә“пјҢиҜҘиҪҜд»¶еҢ…еҸҜд»Ҙд»Һиў«е…Ҙдҫөзҡ„зҺҜеўғдёӯжҸҗеҸ– AWS и®ҝй—®е’ҢеҜҶй’ҘгҖӮдёҖж—ҰжҚ•иҺ·пјҢиҝҷдәӣеҮӯжҚ®е°ұдјҡиў«дј иҫ“еҲ° VPN з«ҜзӮ№пјҢд»ҺиҖҢдҪҝеҸ—е®іиҖ…йҡҫд»ҘиҝҪиёӘеӨ–жі„ж•°жҚ®гҖӮжӯЈеҰӮжҠҘе‘ҠжүҖејәи°ғзҡ„пјҢвҖңйҖҡиҝҮж”¶йӣҶ AWS еҜҶй’ҘпјҢж”»еҮ»иҖ…иҺ·еҫ—дәҶеҜ№жҪңеңЁж•Ҹж„ҹдә‘иө„жәҗзҡ„и®ҝй—®жқғйҷҗвҖқпјҢжңүеҸҜиғҪе°ҶйҮҚиҰҒж•°жҚ®е’Ңдә‘иө„жәҗжҡҙйңІз»ҷжңӘз»ҸжҺҲжқғзҡ„и®ҝй—®гҖӮ

иҜҘжҒ¶ж„ҸиҪҜд»¶зҡ„и®ҫи®ЎдёҺе№іеҸ°ж— е…іпјҢе…¶ test() еҮҪж•°еҸҜжЈҖжҹҘж“ҚдҪңзі»з»ҹе№¶жү§иЎҢзӣёеә”зҡ„жҒ¶ж„ҸзәҝзЁӢгҖӮиҝҷдёҖеҠҹиғҪдҪҝжҒ¶ж„ҸиҪҜд»¶иғҪеӨҹеҗҢж—¶й’ҲеҜ№ Linux е’Ң Windows з”ЁжҲ·пјҢжү©еӨ§дәҶе…¶еҪұе“ҚиҢғеӣҙгҖӮвҖңжҠҘе‘ҠжҢҮеҮәпјҡвҖқиҝҷз§ҚдёҺе№іеҸ°ж— е…ізҡ„и§ҰеҸ‘еҷЁзЎ®дҝқж”»еҮ»дёҚеҸ—ж“ҚдҪңзі»з»ҹзҡ„еҪұе“ҚпјҢд»ҺиҖҢжү©еӨ§дәҶе…¶жҪңеңЁеҪұе“ҚгҖӮ

Typosquatting ж”»еҮ»пјҢеҚіжҒ¶ж„ҸиҪҜд»¶еҢ…зҡ„е‘ҪеҗҚдёҺеҸҜдҝЎеә“зӣёдјјпјҢд»Қз„¶жҳҜејҖжәҗиҪҜд»¶еә“дёӯдёҖдёӘж—ҘзӣҠеўһй•ҝзҡ„йЈҺйҷ©гҖӮз”ұ bitprophet еҲӣе»әзҡ„еҗҲжі• Fabric еә“жӢҘжңүи¶…иҝҮ 2.01 дәҝж¬ЎдёӢиҪҪпјҢеҸ—еҲ°ејҖеҸ‘иҖ…зҡ„е№ҝжіӣдҝЎд»»гҖӮfabrice иғҢеҗҺзҡ„ж”»еҮ»иҖ…еҲ©з”ЁдәҶиҝҷз§ҚдҝЎд»»пјҢеңЁжҜ«ж— жҲ’еӨҮзҡ„зі»з»ҹдёӯеӨ§иӮҶзӘғеҸ–еҮӯжҚ®е№¶е®үиЈ…еҗҺй—ЁгҖӮ
