дёҖеңәж–°еҸ‘зҺ°зҡ„жҒ¶ж„ҸиҪҜд»¶ж”»еҮ»жҙ»еҠЁжӯЈзһ„еҮҶ Docker зҺҜеўғпјҢйҮҮз”ЁдәҶдёҖз§ҚеӨҚжқӮзҡ„гҖҒеӨҡеұӮзә§зҡ„ж··ж·ҶжҠҖжңҜжқҘйҖғйҒҝжЈҖжөӢпјҢе№¶еҠ«жҢҒи®Ўз®—иө„жәҗд»ҘиҝӣиЎҢеҠ еҜҶеҠ«жҢҒгҖӮ

жқҘиҮӘ Darktrace е’Ң Cado Security Labs зҡ„е®үе…Ёз ”з©¶дәәе‘ҳеҜ№иҝҷеңәж”»еҮ»жҙ»еҠЁиҝӣиЎҢдәҶеҲҶжһҗпјҢжҸӯзӨәдәҶж”»еҮ»иҖ…еңЁжҠҖжңҜдёҠзҡ„зІҫе·§жүӢж®өпјҢд»ҘеҸҠе®№еҷЁеҢ–еҹәзЎҖи®ҫж–ҪжүҖйқўдёҙзҡ„ж—ҘзӣҠеўһй•ҝзҡ„йЈҺйҷ©гҖӮ

****DockerпјҡжҒ¶ж„ҸиҪҜд»¶зҡ„дё»иҰҒзӣ®ж Ү****

Docker дҪңдёәйўҶе…Ҳзҡ„е®№еҷЁеҢ–е№іеҸ°пјҢжӯЈи¶ҠжқҘи¶ҠеӨҡең°жҲҗдёәзҪ‘з»ңзҠҜзҪӘеҲҶеӯҗзҡ„ж”»еҮ»зӣ®ж ҮпјҢиҝҷжҳҜеӣ дёәе®ғиў«е№ҝжіӣйҮҮз”ЁпјҢиҖҢдё”е®№еҷЁеҸҜд»ҘеҫҲе®№жҳ“ең°д»Һе…¬е…ұжіЁеҶҢдёӯеҝғиҝӣиЎҢйғЁзҪІгҖӮ

ж”»еҮ»иҖ…еҲ©з”Ёй…ҚзҪ®й”ҷиҜҜжҲ–жҡҙйңІзҡ„ Docker жңҚеҠЎжқҘеҗҜеҠЁжҒ¶ж„Ҹе®№еҷЁпјҢд»–д»¬еёёеёёдҪҝз”Ёжүҳз®ЎеңЁ Docker Hub дёҠзҡ„й•ңеғҸгҖӮ

иҝҷеңәж”»еҮ»жҙ»еҠЁе§ӢдәҺдёҖдёӘд»Һ Docker Hub иҝҗиЎҢе®№еҷЁзҡ„иҜ·жұӮпјҢе…·дҪ“жҳҜдҪҝз”Ё kazutod/tene:ten иҝҷдёӘй•ңеғҸгҖӮ

иҝҷдёӘе®№еҷЁиў«и®ҫи®Ўз”ЁжқҘжү§иЎҢдёҖдёӘ Python и„ҡжң¬ ten.pyпјҢиҜҘи„ҡжң¬еөҢе…ҘеңЁй•ңеғҸеұӮдёӯгҖӮе®үе…ЁеҲҶжһҗеёҲдҪҝз”Ё Docker зҡ„еҶ…зҪ®е·Ҙе…·жҸҗеҸ–е№¶еҲҶжһҗдәҶиҝҷдёӘй•ңеғҸпјҢеҸ‘зҺ°дәҶдёҖдёӘеӨҚжқӮзҡ„ж··ж·Ҷж–№жЎҲгҖӮ

ж··ж·ҶиҝҮзЁӢеҰӮдёӢпјҡ

1.ten.py и„ҡжң¬е®ҡд№үдәҶдёҖдёӘ lambda еҮҪж•°пјҢиҜҘеҮҪж•°еҜ№дёҖдёӘз»ҸиҝҮ Base64 зј–з Ғзҡ„еӯ—з¬ҰдёІиҝӣиЎҢеҸҚиҪ¬пјҢз„¶еҗҺеҜ№е…¶иҝӣиЎҢи§Јз ҒпјҢдҪҝз”Ё zlib еә“еҜ№е…¶и§ЈеҺӢзј©пјҢжңҖеҗҺжү§иЎҢеҫ—еҲ°зҡ„д»Јз ҒгҖӮ

2.иҝҷдёӘиҝҮзЁӢдјҡйҖ’еҪ’йҮҚеӨҚпјҡи§Јз ҒеҗҺзҡ„жңүж•ҲиҪҪиҚ·дјҡеҶҚж¬Ўи°ғз”ЁзӣёеҗҢзҡ„и§Јз ҒеҮҪж•°пјҢжҜҸж¬Ўдј йҖ’дёҖдёӘж–°зҡ„ж··ж·Ҷеӯ—з¬ҰдёІгҖӮ

3.еҲҶжһҗеёҲеҸ‘зҺ°пјҢиҝҷдёӘи§Јз ҒеҫӘзҺҜйңҖиҰҒиҝӣиЎҢ 63 ж¬Ўиҝӯд»ЈжүҚиғҪжңҖз»ҲжҸӯзӨәеҮәе®һйҷ…зҡ„жҒ¶ж„Ҹд»Јз ҒгҖӮ

иҝҷз§Қж·ұеәҰзҡ„еӨҡеұӮж··ж·ҶжҳҜдёҚеёёи§Ғзҡ„гҖӮиҷҪз„¶еҚ•иҪ®ж··ж·ҶйҖҡеёёи¶ід»Ҙз»•иҝҮеҹәдәҺзӯҫеҗҚзҡ„жЈҖжөӢпјҢдҪҶж”»еҮ»иҖ…дҪҝз”ЁеҮ еҚҒеұӮж··ж·Ҷдјјд№ҺжҳҜдёәдәҶи®©дәәе·ҘеҲҶжһҗеёҲе’ҢиҮӘеҠЁеҢ–е·Ҙе…·йғҪж„ҹеҲ°жЈҳжүӢгҖӮ

е°Ҫз®Ўж”»еҮ»иҖ…иҙ№е°ҪеҝғжҖқпјҢз ”з©¶дәәе‘ҳиҝҳжҳҜиғҪеӨҹе®һзҺ°еҺ»ж··ж·ҶиҝҮзЁӢзҡ„иҮӘеҠЁеҢ–пјҢе№¶еңЁеҮ еҲҶй’ҹеҶ…жҸҗеҸ–еҮәжңҖз»Ҳзҡ„жңүж•ҲиҪҪиҚ·гҖӮ

****еҠ еҜҶеҠ«жҢҒзҡ„ж–°жүӢж®ө****

дёҺдј з»ҹзҡ„еҠ еҜҶеҠ«жҢҒжҒ¶ж„ҸиҪҜд»¶дёҚеҗҢпјҢдј з»ҹжҒ¶ж„ҸиҪҜд»¶дјҡйғЁзҪІеғҸ XMRig иҝҷж ·зҡ„е·Ҙе…·жқҘзӣҙжҺҘжҢ–жҺҳеҠ еҜҶиҙ§еёҒпјҢиҖҢиҝҷж¬Ўзҡ„ж”»еҮ»жҙ»еҠЁйҮҮз”ЁдәҶдёҖз§Қж–°йў–зҡ„ж–№жі•гҖӮ

еҺ»ж··ж·ҶеҗҺзҡ„д»Јз ҒдјҡиҝһжҺҘеҲ° teneo.proпјҢиҝҷжҳҜдёҖе®¶еҗҲжі•зҡ„ Web3 еҲқеҲӣе…¬еҸёпјҢиҝҗиҗҘзқҖдёҖдёӘеҺ»дёӯеҝғеҢ–зҡ„зӨҫдәӨеӘ’дҪ“ж•°жҚ®зҪ‘з»ңгҖӮ

йҖҡиҝҮиҝҗиЎҢдёҖдёӘиҠӮзӮ№е№¶жҢҒз»ӯеҸ‘йҖҒ вҖңдҝқжҢҒжҙ»еҠЁвҖқ зҡ„ ping иҜ·жұӮпјҢжҒ¶ж„ҸиҪҜд»¶еҸҜд»ҘиөҡеҸ– вҖңTeneo зӮ№ж•°вҖқвҖ”вҖ” иҝҷжҳҜдёҖз§Қз§ҒжңүеҠ еҜҶд»ЈеёҒпјҢдјҡеҘ–еҠұз»ҷеңЁзҪ‘з»ңдёҠдҝқжҢҒеңЁзәҝж—¶й—ҙе’ҢиҝӣиЎҢжҙ»еҠЁзҡ„з”ЁжҲ·гҖӮ

еҖјеҫ—жіЁж„Ҹзҡ„жҳҜпјҢжҒ¶ж„ҸиҪҜд»¶е№¶дёҚеғҸеҗҲжі•зҡ„иҠӮзӮ№иҪҜд»¶йӮЈж ·жү§иЎҢд»»дҪ•е®һйҷ…зҡ„ж•°жҚ®жҠ“еҸ–ж“ҚдҪңгҖӮзӣёеҸҚпјҢе®ғеҸӘжҳҜжЁЎжӢҹжҙ»еҠЁд»ҘжңҖеӨ§еҢ–д»ЈеёҒеҘ–еҠұгҖӮ

иҝҷз§Қж–№жі•дҪҝж”»еҮ»иҖ…иғҪеӨҹиҺ·еҲ©пјҢеҗҢж—¶еҸҲдёҚдјҡи§ҰеҸ‘дј з»ҹжҢ–зҹҝж“ҚдҪңдёӯеёёи§Ғзҡ„й«ҳиө„жәҗдҪҝз”ЁзҺҮжҲ–зҪ‘з»ңејӮеёёжғ…еҶөгҖӮ

ж №жҚ®жҠҘе‘ҠпјҢиҝҷеңәж”»еҮ»жҙ»еҠЁеҮёжҳҫдәҶдёҖдёӘжӣҙе№ҝжіӣзҡ„и¶ӢеҠҝпјҡж”»еҮ»иҖ…жӯЈд»Һе®№жҳ“иў«жЈҖжөӢеҲ°зҡ„зҹҘеҗҚжҢ–зҹҝе·Ҙе…·пјҢиҪ¬еҗ‘ж»Ҙз”ЁеҗҲжі•зҡ„еҺ»дёӯеҝғеҢ–е№іеҸ°е’ҢеҘ–еҠұзі»з»ҹгҖӮ

иҝҷдәӣз§Ғжңүд»ЈеёҒзҡ„е°Ғй—ӯжҖ§дҪҝеҫ—еҫҲйҡҫиҝҪиёӘжҲ–йҮҸеҢ–ж”»еҮ»иҖ…зҡ„иҺ·еҲ©жғ…еҶөгҖӮ

е®үе…Ёдё“е®¶ејәи°ғпјҢDocker зҺҜеўғд»Қз„¶жҳҜжһҒе…·еҗёеј•еҠӣзҡ„ж”»еҮ»зӣ®ж ҮпјҢе№¶ж•Ұдҝғеҗ„з»„з»Үпјҡ

1.йҷӨйқһз»қеҜ№еҝ…иҰҒпјҢеҗҰеҲҷдёҚиҰҒе°Ҷ Docker жңҚеҠЎжҡҙйңІеңЁдә’иҒ”зҪ‘дёҠгҖӮ

2.дҪҝз”ЁеӨ§зҡ„иә«д»ҪйӘҢиҜҒжңәеҲ¶е’ҢйҳІзҒ«еўҷжқҘйҷҗеҲ¶и®ҝй—®гҖӮ

3.е®ҡжңҹе®Ўи®Ўе’Ңзӣ‘жҺ§е®№еҷЁжҙ»еҠЁпјҢд»ҘеҸ‘зҺ°ејӮеёёжғ…еҶөгҖӮ

4.еҸӘд»ҺеҸ—дҝЎд»»зҡ„жқҘжәҗжӢүеҸ–й•ңеғҸпјҢе№¶еҜ№е…¶иҝӣиЎҢжҒ¶ж„ҸиҪҜд»¶жү«жҸҸгҖӮ

йҡҸзқҖж”»еҮ»иҖ…дёҚж–ӯеҲӣж–°пјҢйҳІеҫЎиҖ…еҝ…йЎ»дҝқжҢҒиӯҰжғ•пјҢе№¶и°ғж•ҙд»–д»¬зҡ„е®үе…ЁжҺӘж–ҪпјҢд»ҘдҝқжҠӨе®№еҷЁеҢ–еҹәзЎҖи®ҫж–Ҫе…ҚеҸ—ж—ҘзӣҠеӨҚжқӮзҡ„еЁҒиғҒгҖӮ
