## What

go-sql-stmt 库是使用go函数来构建 SQL 语句。

类似的项目有:

* <https://github.com/huandu/go-sqlbuilder>
* <https://github.com/knex/knex>

go-sql-stmt 在使用上更加方便，文法更接近原生的sql。通过If语句也可以现实条件生成 SQL 语句。

## Example

`stmt/node_test.go`
