## Windows 11 设置将显示 Game Pass 广告

[![商业](https://icon.solidot.org/images/topics/topicbiz.png?123)](/search?tid=54 "商业")
[![微软](https://icon.solidot.org/images/topics/topicMicrosoft.png?123)](/search?tid=150 "微软")

[Wilson](/~Wilson) (42865)发表于 2024年06月29日 22时30分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=78561&appkey=1370085986&title=Windows%2011%20%E8%AE%BE%E7%BD%AE%E5%B0%86%E6%98%BE%E7%A4%BA%20Game%20Pass%20%E5%B9%BF%E5%91%8A "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自向世界倾诉爱**

根据最新的预览版补丁 KB5039302 (OS Builds 22621.3810 和 22631.3810)说明，微软将在 Windows 11 设置页展示 Game Pass 广告（软件巨人称之为推荐）。如果用户经常玩游戏，且通过 Microsoft account 登陆 Windows，那么家庭版和专业版的设置页将会有游戏广告。其它变化包括：默认在任务栏展示桌面按钮；文件资源管理器加入了通过上下文菜单创建 7-Zip 和 Tape Archive (TAR）压缩档案的功能；支持 Emoji 15.1；支持通过 Windows Share 窗口复制文件；等等。
https://support.microsoft.com/en-us/topic/june-25-2024-kb5039302-os-builds-22621-3810-and-22631-3810-preview-0ab34e3f-bca9-4a52-a1a4-404bf8162f58
