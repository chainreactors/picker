AI FOR CODE

**大赛介绍**

**「稀土掘金 AI FOR CODE 创意挑战赛」**是稀土掘金和 Trae 共同发起的创意征集大赛，面向所有富有创意的工程师和广泛的科技爱好者，携手跨越想象与技术的边界。希望大家能以创新的思维，利用高效的编程工具 Trae，搭乘AI新浪潮，探索并实践AI领域新方向。在这里不论你是有经验的开发者，还是不懂编程的小白，都可以通过智能协作伙伴 Trae 实现你的灵感！

**赛道说明**

赛事总共分【创意征集】和【应用实现】两个赛道，如果你有想法，但是没时间完成创意到落地开发，可以选择只参与【创意征集】赛道，如果你可以独立或者和小伙伴一起完成独立项目开发，即可报名【应用实现】赛道（应用实现赛道包含创意征集部分）。

**【创意征集】**赛道：创意征集可以为参赛队伍提供灵感，更重要的是即使你无法抽身参与亲自开发，也有机会让自己的灵感变为现实。

**【应用实现】**赛道：此赛道鼓励无限创意，不设边界，尽展想象，在算法、特效、新能力、工具、玩法、平台、应用及效率上大胆革新。
