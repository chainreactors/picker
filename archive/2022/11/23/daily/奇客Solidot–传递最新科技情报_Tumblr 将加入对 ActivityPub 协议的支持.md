## Tumblr е°ҶеҠ е…ҘеҜ№ ActivityPub еҚҸи®®зҡ„ж”ҜжҢҒ

[![дә’иҒ”зҪ‘](https://icon.solidot.org/images/topics/topicinternet.png?123)](/search?tid=17 "дә’иҒ”зҪ‘")

[WinterIsComing](/~WinterIsComing) (31822)еҸ‘иЎЁдәҺ 2022е№ҙ11жңҲ22ж—Ҙ 12ж—¶33еҲҶ жҳҹжңҹдәҢ [ж–°жөӘеҫ®еҚҡеҲҶдә«](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=73436&appkey=1370085986&title=Tumblr%20%E5%B0%86%E5%8A%A0%E5%85%A5%E5%AF%B9%20ActivityPub%20%E5%8D%8F%E8%AE%AE%E7%9A%84%E6%94%AF%E6%8C%81 "ж–°жөӘеҫ®еҚҡеҲҶдә«")
![](https://icon.solidot.org/images/a7c7.png)

**жқҘиҮӘең°зҗғзҡ„е‘је”Ө**

Automattic CEO Matt Mullenweg е®ЈеёғпјҢиҪ»еҚҡе®ўжңҚеҠЎ Tumblr е°ҶеҠ е…ҘеҜ№ ActivityPub еҚҸи®®зҡ„ж”ҜжҢҒгҖӮActivityPub жҳҜдёҖдёӘејҖж”ҫеҺ»дёӯеҝғеҢ–зӨҫдәӨзҪ‘з»ңеҚҸи®®пјҢиў« MastodonгҖҒPixelfed е’ҢдёІжөҒжңҚеҠЎ PeerTube зӯүеә”з”ЁдҪҝз”ЁгҖӮActivityPub жҸҗдҫӣдәҶдёҖдёӘе®ўжҲ·з«ҜеҲ°жңҚеҠЎеҷЁзҡ„ API з”ЁдәҺеҲӣе»әгҖҒжӣҙж–°е’ҢеҲ йҷӨеҶ…е®№пјҢд»ҘеҸҠдёҖдёӘиҒ”йӮҰеҲ¶зҡ„жңҚеҠЎеҷЁеҲ°жңҚеҠЎеҷЁ API з”ЁдәҺдј йҖ’йҖҡзҹҘе’Ңи®ўйҳ…еҶ…е®№гҖӮиҝҷдёҖеҒҡжі•дёҺд»ҠеӨ©еӣҙеўҷиҠұеӣӯејҸзҡ„зӨҫдәӨзҪ‘з»ңиғҢйҒ“иҖҢй©°гҖӮеҰӮжһң Tumblr еҠ е…ҘеҜ№ ActivityPub зҡ„ж”ҜжҢҒпјҢиҝҷж„Ҹе‘ізқҖ Mastodon з”ЁжҲ·еҸҜд»Ҙи®ўйҳ… Tumblr з”ЁжҲ·зҡ„её–еӯҗпјҢ Tumblr з”ЁжҲ·д№ҹиғҪиҝӣе…ҘиҒ”йӮҰе®Үе®ҷиҖҢж— йңҖеңЁ Mastodon зҡ„дёҚеҗҢжңҚеҠЎеҷЁдёҠжіЁеҶҢгҖӮ
