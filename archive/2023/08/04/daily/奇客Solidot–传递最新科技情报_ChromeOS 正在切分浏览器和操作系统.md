## ChromeOS 正在切分浏览器和操作系统

[![操作系统](https://icon.solidot.org/images/topics/topicos.png?123)](/search?tid=94 "操作系统")
[![Chrome](https://icon.solidot.org/images/topics/topicchrome.png?123)](/search?tid=154 "Chrome")

[Wilson](/~Wilson) (42865)发表于 2023年08月03日 11时16分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75689&appkey=1370085986&title=ChromeOS%20%E6%AD%A3%E5%9C%A8%E5%88%87%E5%88%86%E6%B5%8F%E8%A7%88%E5%99%A8%E5%92%8C%E6%93%8D%E4%BD%9C%E7%B3%BB%E7%BB%9F "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自大臣号遇难者**

ChromeOS 正在将浏览器和操作系统分离开来。该项目代号 Lacros——代表 **L**inux **A**nd **C**h**R**ome **OS**。顾名思义，也就是将 ChromeOS 的 Linux OS 和 Chrome 浏览器分开，允许 Google 独立更新这两大组件。今天的 Chrome OS 系统 UI(包括 ash window manager、login screen 等) 和浏览器都位于相同的二进制程序中，分开之后将变成两个二进制程序，分别为 ash-chrome (系统 UI) 和 lacros-chrome (浏览器)，此举的一个意图是加入更多对 Wayland 的支持。ChromeOS 此前使用了一个 Google 自己开发的图形栈 Freon，而 Wayland 是今天包括 Android 在内的 Linux 系统和应用广泛使用的图形栈，它将作为 ChromeOS 的原生图形使用。Google 也将无需为 ChromeOS 定制 Chrome，可以直接使用 Linux 版本。
https://arstechnica.com/gadgets/2023/08/google-is-finally-separating-chrome-from-chromeos-for-easier-updates/
