# Sandfly 5.0.6 Maintenance Release

09 June 2024

Product Update

Sandfly 5.0.6 was released and is a maintenance update. It fixes the following bugs.

* Increased size for operating system reconnaissance data to allow for larger result sets for hosts being added.
* Improved error reporting in scanning log.

This is not a critical release for most customers. **If you are not experiencing the above two issues, you do not need to upgrade.**

Upgrade instructions are available below:

[Sandfly Security Upgrade Documentation](https://docs.sandflysecurity.com/docs/upgrading-sandfly)

Thank you for using Sandfly.
