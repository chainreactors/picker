说起主机管理工具，你是否用过资源监控、终端管理、主机面板、远程桌面、堡垒机之类的工具来管理和监控主机，却依旧面临以下问题：

笔记本没带，服务器的 SSH 认证私钥只在笔记本上，没有私钥登不上去，咋办？
家里有台机器，但是网络受 NAT 的影响，出了门就连不回家里的内网，无法远程连接内网主机，咋办？
出于工作需要，在阿里云、腾讯云、华为云等等各种云上面买了一大堆机器，经常忘了自己有哪些机器，还不能统一管理，咋办？
运维要求 SSH 端口不能对外开放，连个 SSH 需要好几层堡垒机，太麻烦了，咋办？
出门在外没有电脑，只有手机，想看看服务器的状态，咋办？
……

## 项目起源

作为一位研发&运维&安全技术爱好者，长期饱受以上主机管理问题的困扰。一次酒足饭饱后，与三个程序员一拍即合，决定做一个供自己使用的主机管理小工具。几天后发布了第一版，没想到吸引了有同样困扰的几个人，三个程序员裂变成了六个程序员，于是我们正式给项目命名：牧云（collie），希望通过本工具可以像放羊一样地管理云主机。

## 牧云（collie）

![]()

有别于其他相似的产品，牧云主机管理助手致力于让技术爱好者能方便、快捷、优雅的管理远程主机，而非傻瓜式管理，主要优势在于：

连接远程主机不再受到内网 NAT 的限制
不用记密码，不用存私钥，认证都交给 ”微信登录“ 来托管
轻松解决性能监控问题
统一管理大批量主机

#### 1.1 资源监控

借鉴了微软的任务管理器，这里从”磁盘空间“，”CPU 使用率“、”内存使用率“、”网络 IO“、”磁盘 IO“ 这五个维度提供图标展示，还在服务端保留了 7 天的历史数据。

![]()

#### 1.2 在线终端

体验与本地的终端基本一致，感受不到延迟，还能支持选择配色，与传统 SSH 最大的区别在于，不用开放 22 端口，不用设置密码，不用配置公私钥，最大的遗憾在于受浏览器的限制，ctrl-w 快捷键没法使用。

![]()

#### 1.3 文件管理

交互细节抄（参考）了微软的资源管理器（希望能比mac的finder好用点），现在只有列目录和上传下载，后面还会增加快速查看和在线编辑的功能。

![]()

## 打开方式

微信扫描登录百川平台，绑定你的主机。
<https://rivers.chaitin.cn/?promotion=a46555d83883b379c6e77c7a085dc1da>

牧云提供的所有功能对于普通个人用户都是免费的，但是如果你非常有钱，也十分感谢你能不吝打赏五毛图片。

自牧云内测以来，装机量持续上升就是我们加班的动力，目前工具仍在 ”公测“ 阶段，许多地方尚不成熟，内测收到的需求已经撑爆了 todo list，至少接下来的一年之内都会保持高速迭代状态，每周一次小更新，每月一次大更新。

## 版本预告

* Docker 管理，可视化远程管理容器和镜像，至少具备 docker desktop 的能力
* 点对点认证，加强工具的安全性
* 内网穿透，映射主机开放的端口到本地，允许访问主机内网的服务
* Windows 版本，支持管理 Windows 主机
* 私有化版本，提供独立部署能力，给暂时不方便使用 SaaS 服务的用户使用
* 开源，等待功能相对完善后，项目源码最终我们会以开源的方式回馈技术社区
