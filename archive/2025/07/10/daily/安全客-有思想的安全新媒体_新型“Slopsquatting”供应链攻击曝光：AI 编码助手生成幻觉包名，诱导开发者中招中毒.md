### **Slopsquatting еЁҒиғҒеҺҹзҗҶи§Јжһҗ**

Slopsquatting зҡ„жң¬иҙЁеңЁдәҺж”»еҮ»иҖ…**еҲ©з”Ё AI зј–з ҒеҠ©жүӢз”ҹжҲҗиҷҡжһ„еҢ…еҗҚзҡ„жјҸжҙһ**пјҢд»ҘжӯӨиҜұеҜјејҖеҸ‘иҖ…еңЁдёҚзҹҘжғ…зҡ„жғ…еҶөдёӢе®үиЈ…жҒ¶ж„Ҹдҫқиө–еҢ…гҖӮ

дёҚеҗҢдәҺдј з»ҹ typosquatting й’ҲеҜ№зҡ„жҳҜз”ЁжҲ·жӢјеҶҷй”ҷиҜҜпјҢslopsquatting зӣҙжҺҘеҲ©з”Ё AI з”ҹжҲҗзҡ„\*\*вҖңеҗҲзҗҶдҪҶе№¶дёҚеӯҳеңЁвҖқзҡ„еҢ…еҗҚ\*\*пјҢдҫӢеҰӮеңЁз”ҹжҲҗд»Јз Ғж—¶е»әи®®дҪҝз”Ё `graphorm` иҝҷж ·зҡ„иҷҡжһ„дҫқиө–гҖӮ

ж”»еҮ»жөҒзЁӢйҖҡеёёеҰӮдёӢпјҡ

1. ж”»еҮ»иҖ…йҖҡиҝҮеҲҶжһҗдё»жөҒ AI зј–з ҒеҠ©жүӢзҡ„вҖңе№»и§үжЁЎејҸвҖқпјӣ
2. жҸҗеүҚеңЁ PyPI зӯүе…¬е…ұд»“еә“дёӯжіЁеҶҢиҝҷдәӣиҷҡжһ„еҢ…еҗҚпјӣ
3. еҪ“ејҖеҸ‘иҖ…иҝҗиЎҢ AI иҮӘеҠЁз”ҹжҲҗзҡ„ `pip install xxx` е‘Ҫд»Өж—¶пјҢеҚідјҡд»Һж”»еҮ»иҖ…жҺ§еҲ¶зҡ„еҢ…дёӯдёӢиҪҪе®үиЈ…е№¶жү§иЎҢжҒ¶ж„Ҹд»Јз ҒгҖӮ

з ”з©¶дәәе‘ҳеңЁдёҖж¬Ўе®һйӘҢдёӯи§ӮеҜҹеҲ°пјҢдёҖж¬ҫе…Ҳиҝӣзҡ„зј–з Ғд»ЈзҗҶз”ҹжҲҗдәҶдёҖдёӘ**зңӢиө·жқҘе®Ңе…ЁеҗҲзҗҶдҪҶдёҚеӯҳеңЁ**зҡ„еҢ…еҗҚпјҢжңҖз»ҲеҜјиҮҙжһ„е»әеӨұиҙҘгҖӮиҖҢжӣҙдёҘйҮҚзҡ„йЈҺйҷ©еңЁдәҺпјҡдёҖж—Ұж”»еҮ»иҖ…жҸҗеүҚжіЁеҶҢиҝҷдәӣеҢ…еҗҚпјҢ**AI е»әи®®е°ұеҸҜиғҪзӣҙжҺҘиҪ¬еҢ–дёәдҫӣеә”й“ҫе…Ҙдҫө**гҖӮ

### **еӨҡе№іеҸ°еӯҳеңЁе№»и§үеҢ…еҗҚжјҸжҙһ**

з ”з©¶еӣўйҳҹеҜ№еӨҡдёӘ AI зј–з Ғе№іеҸ°зҡ„вҖңе№»и§үзҺҮвҖқиҝӣиЎҢдәҶеҲҶжһҗпјҢж¶үеҸҠпјҡ

* **Anthropic зҡ„ Claude Code CLI**
* **OpenAI зҡ„ Codex CLI**
* **йӣҶжҲҗ Model Context Protocol (MCP) йӘҢиҜҒжңәеҲ¶зҡ„ Cursor AI**

жөӢиҜ•з»“жһңеҸ‘зҺ°пјҡ

* **еҹәзЎҖжЁЎеһӢ**еңЁйқўеҜ№еӨҚжқӮд»»еҠЎж—¶жӣҙе®№жҳ“дә§з”ҹ 2пҪһ4 дёӘиҷҡжһ„еҢ…еҗҚпјҢе°Өе…¶еңЁз”ҹжҲҗеӨҡеә“з»„еҗҲд»Јз Ғж—¶пјҢеҖҫеҗ‘дәҺе°ҶвҖңgraphвҖқвҖңormвҖқвҖңlibвҖқзӯүзҶҹиҜҚжӢјжҺҘдёәзңӢдјјзңҹе®һзҡ„еҗҚеӯ—гҖӮ
* **й«ҳзә§зј–з Ғд»ЈзҗҶ**е…·еӨҮжӣҙејәзҡ„жҺЁзҗҶиғҪеҠӣдёҺдёҠдёӢж–ҮиҜҶеҲ«пјҢе№»и§үзҺҮжҜ”еҹәзЎҖжЁЎеһӢдҪҺзәҰ 50%пјҢдҪҶд»ҚеңЁйғЁеҲҶеңәжҷҜдёӯпјҲеҰӮдёҠдёӢж–ҮиЎҘе…ЁгҖҒеҗҚз§°жӢҹжҖҒпјүеҮәзҺ°жјҸжҙһгҖӮ
* еҚідҫҝжҳҜе…·еӨҮе®һж—¶йӘҢиҜҒжңәеҲ¶зҡ„ Cursor AIпјҢд№ҹдјҡеңЁи·ЁиҜӯиЁҖз”ҹжҖҒжҲ–иҜҚзҙ жӢјжҺҘпјҲеҰӮд»Һ JavaScript еҖҹз”Ё Python еҢ…еҗҚз»“жһ„пјүзӯүиҫ№з•ҢеңәжҷҜдёӢйҒ—жјҸејӮеёёгҖӮ

### **йҳІжҠӨе»әи®®пјҡжһ„е»әеӨҡеұӮж¬Ўдҫқиө–е®үе…ЁжңәеҲ¶**

дёәйҳІиҢғжӯӨзұ» AI й©ұеҠЁзҡ„дҫӣеә”й“ҫж”»еҮ»пјҢз ”з©¶дәәе‘ҳжҸҗеҮәд»ҘдёӢе®үе…ЁеҜ№зӯ–пјҡ

1. **еҗҜз”ЁиҪҜд»¶зү©ж–ҷжё…еҚ•пјҲSBOMпјү**пјҡиҝҪиёӘдҫқиө–жқҘжәҗпјҢеҪўжҲҗе®Ўи®Ўй“ҫгҖӮ
2. **дҪҝз”Ёдҫқиө–жјҸжҙһжү«жҸҸе·Ҙе…·**пјҡеҰӮ Safety CLIпјҢеҸҜеңЁе®үиЈ…еүҚжЈҖжөӢе·ІзҹҘжјҸжҙһпјҲCVEпјүгҖӮ
3. **ејәеҲ¶ AI з”ҹжҲҗе‘Ҫд»ӨеңЁйҡ”зҰ»зҺҜеўғдёӯжү§иЎҢ**пјҡдҫӢеҰӮйҖҡиҝҮдёҙж—¶ Docker е®№еҷЁжҲ–иҷҡжӢҹжңәе®ҢжҲҗдҫқиө–е®үиЈ…пјҢйҒҝе…ҚжұЎжҹ“дё»жңәгҖӮ
4. **еҠ е…ҘвҖңеҚіж—¶еҢ…йӘҢиҜҒвҖқжңәеҲ¶**пјҡеңЁ AI з”ҹжҲҗд»Јз ҒеүҚпјҢйҖҡиҝҮ prompt й©ұеҠЁжҹҘиҜўеҢ…еҗҚжҳҜеҗҰзңҹе®һеӯҳеңЁгҖӮ
5. **дәәе·Ҙе®Ўжү№жөҒзЁӢ**пјҡеҜ№дёҚзҶҹжӮүзҡ„еҢ…иҝӣиЎҢдәәе·ҘеӨҚж ёпјҢйҒҝе…Қе®Ңе…Ёдҫқиө–иҮӘеҠЁеҢ–жөҒзЁӢгҖӮ
6. **жІҷз®ұиҝҗиЎҢдёҺзҪ‘з»ңйҷҗеҲ¶**пјҡеҸҜдҪҝз”Ёжүҳз®Ўдә‘жІҷз®ұпјҢе№¶и®ҫзҪ®зҪ‘з»ңеҮәеҸЈжҺ§еҲ¶пјҢйҷҚдҪҺж”»еҮ»жҲҗеҠҹзҺҮгҖӮ
7. **е…Ёйқўж—Ҙеҝ—е®Ўи®ЎдёҺиЎҢдёәзӣ‘жҺ§**пјҡи®°еҪ•е®үиЈ…е‘Ҫд»ӨеҸҠзӣёе…іж“ҚдҪңпјҢзӣ‘жөӢејӮеёёжЁЎејҸгҖӮ

###

йҡҸзқҖ AI зј–з ҒеҠ©жүӢзҡ„е№ҝжіӣеә”з”ЁпјҢ**slopsquatting жӯЈејҸжҲҗдёәиҮӘеҠЁеҢ–ејҖеҸ‘жөҒзЁӢдёӯзҡ„ж–°еһӢдҫӣеә”й“ҫйЈҺйҷ©зӮ№**гҖӮе°Ҫз®Ўзӣ®еүҚзҡ„ AI е·Ҙе…·жңүжүҖиҝӣжӯҘпјҢиғҪдёҖе®ҡзЁӢеәҰеҮҸе°‘вҖңе№»и§үеҢ…еҗҚвҖқзҡ„з”ҹжҲҗпјҢдҪҶиҝҷдёҖйЈҺйҷ©е№¶жңӘж №йҷӨгҖӮ

з»„з»Үеә”и®ӨиҜҶеҲ°пјҡ**з®ҖеҚ•зҡ„еҢ…еҗҚеӯҳеңЁжҖ§жҹҘиҜўдёҚи¶ід»Ҙжһ„жҲҗе®үе…ЁйҳІзәҝ**пјҢеҝ…йЎ»е°Ҷдҫқиө–и§ЈжһҗдёҺйӘҢиҜҒи§Ҷдёә**еҸҜе®Ўи®Ўзҡ„е®үе…ЁжөҒзЁӢ**пјҢд»Һж №жәҗдёҠйҷҚдҪҺиў«е…Ҙдҫөзҡ„йЈҺйҷ©гҖӮ

иҝҷдёҖз ”з©¶дёәејҖеҸ‘еӣўйҳҹж•Іе“ҚдәҶиӯҰй’ҹпјҡ**AI з”ҹжҲҗд»Јз ҒеёҰжқҘзҡ„дҫҝеҲ©пјҢдёҚиғҪд»ҘзүәзүІдҫӣеә”й“ҫе®үе…Ёдёәд»Јд»·**гҖӮ
