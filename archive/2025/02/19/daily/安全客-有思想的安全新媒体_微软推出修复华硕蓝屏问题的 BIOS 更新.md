# 微软推出修复华硕蓝屏问题的 BIOS 更新

![ASUS]()

在安装了一个 BIOS 更新后，更多华硕（ASUS）用户现在可以安装 Windows 11 24H2 系统了，该 BIOS 更新解决了十月份所确认的蓝屏死机（BSOD）问题。

正如微软去年所披露的那样，当时微软在受影响的系统上添加了一个兼容性限制，阻止了 Windows 11 2024 更新的安装。在一些配备特定硬件组件的华硕设备上尝试更新到 Windows 11 24H2 系统时，就会出现这些问题。

微软当时表示，这个已知问题主要影响华硕 X415KA 和 X515KA 笔记本电脑型号。微软称：“在更新过程中，设备可能会出现带有错误信息的蓝屏，并且无法完成 24H2 版本的安装。”

然而，在最近对 Windows 发布健康状况仪表板的一次更新中，微软表示，现在正在推出由华硕提供的 BIOS 更新，该更新解决了蓝屏死机问题，使得更多华硕设备能够安装最新的 Windows 11 版本。

微软表示：“运行 BIOS 版本低于 311 的华硕 X415KA 和 X515KA 型号设备，需要安装最新的 BIOS 更新，以解除安全防护措施，并继续进行 Windows 11 24H2 版本的升级。”

“该 BIOS 更新被归类为关键更新（CU），可通过 Windows 更新获取。虽然安全防护措施仍然存在，但随着更多设备收到更新，它将逐步解除。”

建议希望让自己的电脑能够进行 Windows 11 24H2 升级的华硕用户，首先通过 “设置”>“Windows 更新” 来安装 BIOS 更新。
