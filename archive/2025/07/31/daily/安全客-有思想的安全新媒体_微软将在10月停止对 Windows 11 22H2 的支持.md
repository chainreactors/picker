# 微软将在10月停止对 Windows 11 22H2 的支持

![]()

微软今天提醒客户，Windows 11 22H2 的最后支持版本将于**10月14日**结束服务。

这一公告适用于2022年9月20日发布的 Windows 11 22H2 企业版、教育版和物联网企业版，继去年的家庭版和专业版停止服务后发布。

微软表示：“2025年10月的每月安全更新将是此版本的最后一次更新。此日期之后，运行此版本的设备将不再接收每月的安全和预览更新，无法获得来自最新安全威胁的保护。”

“为了帮助您保持安全和生产力，Windows 更新将在设备达到服务终止时自动启动 Windows 11 功能更新，适用于消费设备和非托管的业务设备。这将确保您的设备持续获得支持并接收每月的关键安全和生态系统健康更新。”

不过，微软表示，客户可以选择在非活动时间进行重启，完成 Windows 24H2 功能更新。

您还可以通过访问 Windows 生命周期常见问题页面或使用生命周期政策搜索工具，获取有关其他 Windows 版本终止服务日期的更多信息。微软还提供了一份即将在未来几个月内停止支持或退役的所有产品清单。

Windows 11 版本 24H2（也称为 Windows 11 2024 更新），是 Windows 11 的最新版本，已于2024年5月开始向加入 Windows Insider 发布预览渠道的企业客户推出，并于2024年10月发布给符合条件的 Windows 11 22H2/23H2 设备。

为了尽早接收最新的功能更新，请前往设置 > Windows 更新，并启用“获取最新更新，一旦可用”选项。

然而，需要注意的是，**微软已对具有不兼容驱动程序和软件的设备添加了多个 Windows 11 24H2 的保护性暂停**。

仍然存在的升级阻止措施已经应用于具有不兼容的 SenseShield 技术代码混淆驱动程序、英特尔智能声音技术（SST）音频驱动程序、壁纸自定义软件、集成摄像头和 Dirac 音频改进软件的计算机。
