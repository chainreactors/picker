# Trustwave 推出专为 Microsoft 用户打造的钓鱼防护服务

发布时间 : 2025-07-15 18:20:09

原文地址：<https://securitybrief.asia/story/trustwave-unveils-phishing-protection-service-for-microsoft-users>

![]()

**Trustwave 宣布推出专为 Microsoft 用户打造的托管钓鱼防护服务，旨在增强使用 Microsoft Office 365 与 Defender for Office 的组织应对钓鱼攻击的能力。**

钓鱼攻击仍是全球报告最多的网络犯罪类型。攻击者正利用愈加先进的人工智能技术，绕过主流平台的默认电子邮件安全防护机制。

为应对此趋势，Trustwave 开发了这项托管式网络安全解决方案，为企业提供持续的安全防护与员工安全意识提升。该服务与 Microsoft 原生防护系统协同运作，提供额外的风险缓解和邮件安全管理能力。

### 功能与集成能力

“Managed Phishing for Microsoft” 服务包含多项功能，用于减少组织面对复杂钓鱼活动的暴露风险，包括：

* 端到端的技术管理
* 多层次的威胁检测系统
* 定期的仿真钓鱼演练
* 全天候的威胁响应服务

Trustwave 表示，其技术管理功能可完成钓鱼相关策略和规则的全流程配置与管理，降低内部 IT 团队的运维负担，使其能够专注于其他优先事务。

检测系统则依托 AI 驱动的引擎以及 Trustwave SpiderLabs 威胁研究团队的情报支持，能够阻止超过 99% 的威胁。同时，该服务还为员工提供定制化的钓鱼演练，根据不同组织的业务环境量身设计，提高员工警觉性。

此外，服务还包含 24/7 的邮件分析、事件调查与响应能力，尤其针对那些成功绕过 Microsoft 原生防护机制的钓鱼邮件。服务还提供持续的报告分析，包括用户行为和攻击趋势等洞察，由 Trustwave 的网络安全专家提供可操作的建议，帮助企业及时调整安全策略。

该服务可无缝集成至现有 Microsoft 环境中，尤其强调通过其主动防御与情报分析能力，进一步释放 Microsoft E5 安全投资的潜力。

### 行业观点

**Trustwave 产品管理与解决方案工程高级副总裁 Jesse Emerson 表示：“钓鱼攻击在数量和复杂度上的持续增长，正给各类组织带来巨大压力。我们的托管钓鱼防护服务能帮助组织最大化利用其 Microsoft 安全投资，同时通过先进检测、快速响应与员工持续教育，弥补关键防护缺口。”**

数据显示，钓鱼攻击仍是企业与公共部门面临的长期安全难题。攻击者越来越多地使用自动化和个性化策略，试图欺骗终端用户，绕过传统安全控制措施。

### 威胁态势与研究

Trustwave 旗下 SpiderLabs 威胁研究团队正在持续追踪钓鱼攻击技术的快速演进。如今，越来越多钓鱼活动利用 AI 工具来提升成功率，并躲避旧有的邮件过滤系统和安全规则。

Trustwave 指出，其托管钓鱼防护服务正是为了解决原生电子邮件安全产品可能存在的防护盲区，旨在通过技术、专家支持和用户意识相结合的方式提供更全面的防护方案。

服务所提供的定期钓鱼仿真演练，也根据客户的实际业务环境定制，帮助员工建立起更强的警觉性与防御文化。

### 持续分析与风险应对

**订阅该服务的组织将持续收到关于用户行为和攻击趋势的分析报告。Trustwave 表示，这些洞察将帮助客户及时应对变化中的威胁态势，进一步降低其网络安全风险敞口。**

通过对 Microsoft Office 365 与 Defender for Office 原生功能的补强，“Managed Phishing for Microsoft” 服务致力于帮助组织提升事件响应能力，并显著降低钓鱼攻击成功的可能性。
