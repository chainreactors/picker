ж”»еҮ»д»ҺеЁҒиғҒиЎҢдёәиҖ…жү«жҸҸ Internet д»ҘжҹҘжүҫзјәд№ҸжӯЈзЎ®иә«д»ҪйӘҢиҜҒзҡ„ Docker API з«ҜзӮ№ејҖе§ӢгҖӮж”»еҮ»иҖ…дҪҝз”Ё masscan е’Ң zgrab зӯүе·Ҙе…·иҜҶеҲ«жҳ“еҸ—ж”»еҮ»зҡ„ Docker е®№еҷЁпјҢе№¶жү§иЎҢз”ҹжҲҗж–°е®№еҷЁзҡ„жҒ¶ж„Ҹе‘Ҫд»ӨпјҢд»ҺиҖҢз ҙеқҸзі»з»ҹгҖӮж”»еҮ»иҖ…дҪҝз”ЁиҝҷдәӣеҸ—ж„ҹжҹ“зҡ„е®№еҷЁжқҘйғЁзҪІ XMRigпјҢиҝҷжҳҜдёҖз§ҚжөҒиЎҢзҡ„еҠ еҜҶиҙ§еёҒжҢ–зҹҝиҪҜд»¶пјҢе®ғдёәж”»еҮ»иҖ…зҡ„й’ұеҢ…жҢ–жҺҳй—ЁзҪ—еёҒ пјҲXMRпјүгҖӮ

иҜҘжҙ»еҠЁж¶үеҸҠзҡ„е…ій”®е·Ҙе…·д№ӢдёҖжҳҜеҲқе§ӢеҢ–и„ҡжң¬ пјҲinit.shпјүпјҢдёҖж—Ұжү§иЎҢпјҢе®ғе°ұдјҡдёәиҝӣдёҖжӯҘзҡ„е…Ҙдҫөи®ҫзҪ®е®№еҷЁгҖӮиҜҘи„ҡжң¬д»Һж”»еҮ»иҖ…зҡ„е‘Ҫд»Өе’ҢжҺ§еҲ¶ пјҲC2пјү жңҚеҠЎеҷЁдёӢиҪҪйўқеӨ–зҡ„жңүж•ҲиҙҹиҪҪпјҢе®үиЈ…еҝ…иҰҒзҡ„ж•°жҚ®дј иҫ“е·Ҙе…·пјҢеҰӮ curl е’Ң wgetпјҢз”ҡиҮійғЁзҪІиҮӘе®ҡд№үиҝӣзЁӢйҡҗи—ҸзЁӢеәҸд»ҘйҳІжӯўиў«еҸ‘зҺ°гҖӮ

жҒ¶ж„ҸиҪҜд»¶е№¶дёҚжӯўдәҺж„ҹжҹ“еҚ•дёӘ Docker е®һдҫӢгҖӮдёҖж—Ұиҝӣе…ҘпјҢе®ғе°ұдјҡдҪҝз”ЁдёҖзі»еҲ—жЁӘеҗ‘з§»еҠЁжҠҖжңҜеңЁдә‘еҹәзЎҖи®ҫж–Ҫдёӯдј ж’ӯпјҢзү№еҲ«жҳҜй’ҲеҜ№ Docker е’Ң KubernetesгҖӮkube.lateral.sh е’Ң spread\_docker\_local.sh зӯүи„ҡжң¬дјҡжү«жҸҸеұҖеҹҹзҪ‘дёӯзҡ„ејҖж”ҫ KubernetesпјҲз«ҜеҸЈ 10250пјүе’Ң DockerпјҲз«ҜеҸЈ 2375гҖҒ2376 е’Ң 2377пјүз»Ҳз«ҜиҠӮзӮ№пјҢд»ҺиҖҢе…Ғи®ёжҒ¶ж„ҸиҪҜд»¶еғҸи •иҷ«дёҖж ·еңЁеӨҡдёӘзі»з»ҹдёӯдј ж’ӯгҖӮ

еҸ—ж„ҹжҹ“зҡ„ Kubernetes йӣҶзҫӨйҖҡиҝҮд»Ҙ Kubelet API дёәзӣ®ж ҮиҝӣдёҖжӯҘеҸ—еҲ°жҚҹе®іпјҢиҜҘ API е…Ғи®ёж”»еҮ»иҖ…зӣҙжҺҘеңЁйӣҶзҫӨеҶ…зҡ„е®№еҷЁдёҠз®ЎзҗҶ Pod е№¶жү§иЎҢжҒ¶ж„ҸиҪҜд»¶гҖӮйҖҡиҝҮеҲ©з”ЁжӯӨ APIпјҢеЁҒиғҒиЎҢдёәиҖ…еҸҜд»ҘйғЁзҪІдё“й—Ёз”ЁдәҺжҢ–жҺҳеҠ еҜҶиҙ§еёҒзҡ„йўқеӨ–е®№еҷЁпјҢд»ҺиҖҢжҳҫзқҖжү©еӨ§е…¶иҝҗиҗҘи§„жЁЎгҖӮ

иҜҘжҒ¶ж„ҸиҪҜд»¶иҝҳдҪҝз”ЁжөҒиЎҢзҡ„е®№еҷЁй•ңеғҸеӯҳеӮЁеә“ Docker Hub жқҘжүҳз®Ўе’ҢеҲҶеҸ‘е…¶жҒ¶ж„ҸиҙҹиҪҪгҖӮDatadog з ”з©¶дәәе‘ҳеҸ‘зҺ°пјҢз”ЁжҲ·еҗҚдёә nmlmweb3 зҡ„ Docker Hub з”ЁжҲ·еҸ‘еёғдәҶеҢ…еҗ«жҒ¶ж„Ҹи„ҡжң¬зҡ„еӣҫеғҸгҖӮжҒ¶ж„ҸиҪҜд»¶дјҡдёӢиҪҪе№¶жү§иЎҢиҝҷдәӣеӣҫеғҸпјҢд»ҘиҝӣдёҖжӯҘдј ж’ӯж”»еҮ»гҖӮ

жңүи¶Јзҡ„жҳҜпјҢж”»еҮ»иҖ…иҝҳйҖҡиҝҮејәеҲ¶еҸ—ж„ҹжҹ“зҡ„ Docker е®һдҫӢзҰ»ејҖзҺ°жңүзҡ„ Swarm зҪ‘з»ңе№¶еҠ е…Ҙз”ұеЁҒиғҒиЎҢдёәиҖ…жҺ§еҲ¶зҡ„жҒ¶ж„Ҹ Swarm жқҘж“Қзәө Docker Swarm зҺҜеўғгҖӮиҝҷдҪҝд»–д»¬иғҪеӨҹеҚҸи°ғе’ҢжҺ§еҲ¶еӨ§йҮҸеҸ—ж„ҹжҹ“зҡ„дё»жңәпјҢдҪңдёәдё“й—Ёз”ЁдәҺеҠ еҜҶиҙ§еёҒжҢ–жҺҳзҡ„еғөе°ёзҪ‘з»ңзҡ„дёҖйғЁеҲҶгҖӮ

иҜҘжҙ»еҠЁејәи°ғдәҶдә‘зҺҜеўғдёӯй…ҚзҪ®й”ҷиҜҜзҡ„е…ій”®й—®йўҳпјҢзү№еҲ«жҳҜжңӘз»Ҹиә«д»ҪйӘҢиҜҒе°ұжҡҙйңІзҡ„ Docker API з«ҜзӮ№гҖӮиҝҷдәӣејҖж”ҫзҡ„е…ҘеҸЈзӮ№е……еҪ“еҲқе§Ӣе…Ҙдҫөзҡ„ж”»еҮ»еӘ’д»ӢгҖӮз„¶еҗҺпјҢеЁҒиғҒиЎҢдёәиҖ…дҪҝз”Ёе…¶д»–и„ҡжң¬пјҲдҫӢеҰӮ spread\_ssh.shпјүжқҘе®ҡдҪҚ SSH жңҚеҠЎеҷЁпјҢжү«жҸҸй…ҚзҪ®й”ҷиҜҜзҡ„зі»з»ҹе№¶дҪҝз”Ёе®ғд»¬иҝӣдёҖжӯҘдј ж’ӯгҖӮ

иҜҘжҙ»еҠЁзҡ„жҲҗеҠҹеңЁдәҺе®ғиғҪеӨҹеҲ©з”Ёиҝҷдәӣеёёи§Ғзҡ„й”ҷиҜҜй…ҚзҪ®пјҢдҪҝе…¶иғҪеӨҹеңЁдә‘еҹәзЎҖи®ҫж–Ҫдёӯиҝ…йҖҹдј ж’ӯгҖӮжҜҸдёӘеҸ—ж„ҹжҹ“зҡ„иҠӮзӮ№йғҪжңүеҠ©дәҺеӨ§и§„жЁЎзҡ„еҲҶеёғејҸеҠ еҜҶжҢ–зҹҝж“ҚдҪңпјҢиҜҘж“ҚдҪңз»§з»ӯдёәж”»еҮ»иҖ…еҲӣйҖ еҲ©ж¶ҰпјҢе№Іжү°жңҖе°ҸгҖӮ

дёәдәҶйҳІиҢғиҝҷдәӣеЁҒиғҒпјҢдә‘з®ЎзҗҶе‘ҳеҝ…йЎ»зЎ®дҝқйҖӮеҪ“зҡ„е®үе…Ёй…ҚзҪ®пјҢеҢ…жӢ¬еҜ№ Docker API дҪҝз”Ёиә«д»ҪйӘҢиҜҒгҖҒдҝқжҠӨ Kubernetes йӣҶзҫӨд»ҘеҸҠзӣ‘жҺ§еҸҜз–‘жҙ»еҠЁгҖӮйҡҸзқҖеҠ еҜҶеҠ«жҢҒжҙ»еҠЁзҡ„дёҚж–ӯеҸ‘еұ•пјҢдё»еҠЁе®үе…ЁжҺӘж–ҪеҜ№дәҺдҝқжҠӨдә‘зҺҜеўғиҮіе…ійҮҚиҰҒгҖӮ
