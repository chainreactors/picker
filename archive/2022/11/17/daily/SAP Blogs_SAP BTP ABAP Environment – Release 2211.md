While SAP TechEd 2022 is in full swing (see all ABAP-related content [here](https://blogs.sap.com/2022/10/28/abap-at-sap-teched-in-2022/)), customers and partners of SAP BTP ABAP Environment can already enjoy latest features as part of the recent release 2211.

As always, please enjoy my little overview of most important features and their respective details.

All available new features can be found on the [official release notes](https://help.sap.com/whats-new/cf0cb2cb149647329b5d02aa96303f56?Environment=ABAP&sel4=ABAP&locale=en-US&Valid_as_Of=2022-11-01%3A2022-11-12) page.

**ABAP Development Tools**

* Support for navigation to the **base behavior definition** of an **interface** or **projection behavior definition** in the **Relation Explorer** ([details](https://help.sap.com/docs/BTP/5371047f1273405bb46725a417f95433/59c28262323f49968e246e2239e5e11a.html#loio59c28262323f49968e246e2239e5e11a__section_base_BDEF))

* Enhancements to the **ABAP Object search** including **new filters** and **grouping options** in the search view as well as new supported **object types** ([details](https://help.sap.com/docs/BTP/5371047f1273405bb46725a417f95433/59c28262323f49968e246e2239e5e11a.html#loio59c28262323f49968e246e2239e5e11a__section_abap_object_search))

* Support for displaying the transport **action log** in the **Transport Request editor** ([details](https://help.sap.com/docs/BTP/5371047f1273405bb46725a417f95433/baf612fb01674f8da73a60a298c4a9d5.html?version=Cloud))

* Support for an **additional level** in the **business object composition tree**

* Support for **new scenario options** such as adding copy actions, deprecate actions and data consistency checks in the **Business Configuration Maintenance Object wizard**

* Support for **quick assist** to add a **friends declaration** to a **global class** for a **test class** ([details](https://help.sap.com/docs/BTP/5371047f1273405bb46725a417f95433/d57b6fe571a649dd84fe90aa94dc1398.html))

* Support for new ABAP repository object type **Metric Provider** which can be used to **define own application-specific metrics** for **health monitoring** ([details](https://help.sap.com/docs/PRODUCTS/922bf2dbe0b646aaaa8cb5e077cfd799/e4de41eaea9d49a5be76bfca81f736ab.html?locale=en-US&version=Cloud))

**ABAP RESTful Application Programming Model**

* Support for the definition of **static feature control** for fields in a **RAP BO interface** ([details](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/7caa146ddc4b4f24a58efd33f0f789b4.html))

* Support for usage of **released CDS views** as **value help** for existing factory calendars, holiday calendars and holidays

* Support for **excluding fields** as **trigger condition** (see details [here](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/1c9cbfcecef74d588e68a2552812a278.html) & [here](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/f70e8ef245dc4bda8353a4a113843361.html))

* Availability of documentation for the **modeling of parameters** and **usage of their representation** ([details](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/d891bf474d5444cb939a85d222b75fdf.html))

* Support for **dynamic** and **static feature control** for **large objects** in **OData V2** ([details](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/10a3eb645b83413cbbebe4fc1d879a62.html))

* Enhancements to the **RAP BO authorization control** via **own authorization context** ([details](https://help.sap.com/viewer/923180ddb98240829d935862025004d6/Cloud/en-US/2864fea800e44f648bdfd8751c369fce.html))

**Administration**

* Enhancements to the **ATC Configurator App** including the **export of changes** as JSON file and new options for the **Transport Check Policy** ([details](https://help.sap.com/docs/BTP/65de2977205c403bbc107264b8eccf4b/22c26ff27b9f44b7b7229a01e8e8ed25.html))

* Support for the **download** of a **list of communication users** in the **Maintain Communication Users** app ([details](https://help.sap.com/docs/BTP/65de2977205c403bbc107264b8eccf4b/eef80dda3867461c92ac1273689ed36f.html))

* Support for creating **drafts** and **versions** of **key user adaptations** ([details](https://help.sap.com/docs/PRODUCTS/922bf2dbe0b646aaaa8cb5e077cfd799/b9de62a11b7941a08c549e0d4e196839.html?locale=en-US&version=Cloud))

* Support for the new **Health Monitoring** app to **locally monitor** the **health of the system** based on KPIs ([details](https://help.sap.com/docs/PRODUCTS/922bf2dbe0b646aaaa8cb5e077cfd799/bbf6b336bcb84d4185429dd76dfdfaf8.html?locale=en-US&version=Cloud))

* Support for the new **Maintain Customizing Translations** app which allows **translating client-specific customizing** ([details](https://help.sap.com/docs/PRODUCTS/922bf2dbe0b646aaaa8cb5e077cfd799/136acbbe40e74b80906e2e486e49c747.html?locale=en-US&version=...
