# Microsoft 在 Edge Stable 中预览 Game Assist 游戏内浏览器

![Gamer]()

微软日前宣布，其最近发布的游戏内浏览器 Game Assist（游戏辅助）现在也可以为 Microsoft Edge 稳定版用户提供预览版。

Game Assist 是专为 PC 游戏优化的 Edge 特别版本，当按下 Win+G 键盘快捷键时，它将以 “游戏感知 ”覆盖的形式出现在游戏栏的顶部。

据微软 Edge 团队的高级产品经理 William Devereux 称，它旨在为支持的游戏列表提供提示和指南，并共享相同的浏览器数据。

Devereux 说：“随着 Edge 稳定版 132 的发布，游戏辅助功能无需任何 Insider 软件，也无需更改默认浏览器即可使用，这使它在游戏 PC 上的使用更加方便。”

“要选择加入游戏辅助（预览版），请打开 Edge，进入设置及其他（“……”）> 设置，在搜索框中搜索游戏辅助，找到游戏辅助选项，然后选择安装小工具。”

11 月份推出的第一个预览版要求使用 Microsoft Edge 的 Beta 或 Preview 频道作为默认网页浏览器，而今天推出的版本将默认使用 Edge 稳定版。不过，对于最新更新，你也可以将其设置为使用 Edge Beta、Dev 或 Canary。

除了 11 月份推出的《博德之门 3》、《暗黑破坏神 4》、《堡垒之夜》、《地狱之刃 2：塞努亚传奇》、《英雄联盟》、《Minecraft》、《守望先锋 2》、《Roblox》和《Valorant》之外，游戏辅助功能现在还能为更多游戏提供指南和上下文提示，包括《龙腾世纪：面纱护卫》、《夺宝奇兵与大圆环》、《奇迹对决》、《Metaphor： ReFantazio》和《S.T.A.L.K.E.R. II: Heart of Chernobyl》。

在微软为更多热门游戏添加内容之前，你还可以在玩任何游戏时使用 “游戏辅助 ”功能查找指南和其他网站上的游戏提示。

Devereux 补充说：“我们还对浏览器体验进行了一系列改进，并增加了对广告拦截器等一些扩展程序的支持。”

“我们目前正在进行下一波更新，包括支持键盘快捷键、带有常用浏览器控件（包括将当前标签页添加为侧边栏上的应用程序）的’设置和更多’（……）菜单、右键上下文菜单、改进的画中画体验等。”

微软还要求用户通过 Game Assist 的 “发送反馈 ”对话框（可在 “设置和更多 ”菜单中找到）、Game Assist 社区反馈门户、r/MicrosoftEdge 和其他社交媒体渠道提供反馈。
