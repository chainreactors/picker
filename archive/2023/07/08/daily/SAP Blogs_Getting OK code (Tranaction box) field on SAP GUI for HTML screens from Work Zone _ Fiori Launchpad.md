I like SAP Build Work Zone (Fiori Launchpad) and the Horizon theme-based user interface for running my day-to-day application.

However, for R/3-era dinosaurs like me, moving away from the age-old SAP GUI (thick client) was a big change, especially since there was no way to access the T-codes (transactions) that I had memorized over the years.

Someone showed me how to get the "OK code field" (i.e. Transaction box) on SAP GUI for HTML tcodes accessed via Fiori Launchpad, and that made my day‚Ä¶

![](/legacyfs/online/storage/blog_attachments/2023/07/OK-code-GIF-1.gif)

OK Code on SAP GUI for HTML screen

Here are the steps...

1. Open the tile that will launch SAP GUI for HTML (example: Post Goods Movement [MIGO])

2. Click on ‚ÄòMore‚Äô![](/legacyfs/online/storage/blog_attachments/2023/07/1-6.jpg)

3. Navigate to ‚ÄúGUI Actions and Settings > Settings‚Äù

![](/legacyfs/online/storage/blog_attachments/2023/07/2-5.jpg)

4. Select ‚ÄúShow OK Code Field:‚Äù and ‚ÄúEnable enhanced search for OK code field:‚Äù and click ‚ÄúSave‚Äù

![](/legacyfs/online/storage/blog_attachments/2023/07/3-2.jpg)

5. OK filed is now visible

![](/legacyfs/online/storage/blog_attachments/2023/07/4-2.jpg)

6. Enhance search is pretty good as well

![](/legacyfs/online/storage/blog_attachments/2023/07/5-2.jpg)

Limitation: In the system I used transactions SMEN or SESSION\_MANAGER are locked so I can‚Äôt get back to the ‚ÄòTree menu‚Äô.
