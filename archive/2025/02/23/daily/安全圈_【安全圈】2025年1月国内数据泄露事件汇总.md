# 2025年1月涉国内数据泄露事件汇总

**数世咨询联合零零信安**共同发布了《数据泄露态势》月度报告，基于0.zone安全开源情报系统，报告监控了约10万个威胁源，包括明网、深网、暗网、匿名社群等。报告显示，2025年1月共监控到全球DWM（Dark Web Market）深网和暗网有效情报 **96,082** 份，其中 **4,749** 份为高价值买卖情报。

## 1. 主要行业数据泄露情况

* **行业属性数据**占泄露数据总量的约 \*\*62%\*\*，主要涉及：

+ 金融行业
+ 信息和互联网行业
+ 党政军与社会
+ 批发零售业
+ 教育行业等

* **无明显行业属性数据**占 \*\*38%\*\*，包括：

+ 邮箱密码二要素数据
+ 公民个人信息数据
+ 批量的企业工商数据等

## 2. 具体数据泄露事件

* **发布时间**：2025年1月23日
* **泄露数量**：**15,000条**

* **事件描述**：该数据泄露自2024年7月21日，卖家未能售卖此数据，最终在2025年1月23日通过暗网发布。

## 3. 涉及中国数据泄露的类型

* **泄露类型**：

+ 开房信息
+ 相亲信息
+ 屋主信息
+ 票务信息
+ 投资信息
+ 理财信息
+ 贷款信息
+ 车主信息等

* **匿名社交社群数据泄露**：

+ 每个月在匿名社交社群中发布的我国数据泄露的样例数据约 **10万条**，全数据量估算在 **1000万条** 以上。

* **活跃用户信息泄露**：

+ 使用匿名社交软件的活跃用户中，以 **“86”**（我国区号）开头的手机号共发信息 **92,189条**。

## 4. 安全防护建议

### 1. **数据安全策略**

* 企业应制定包括数据安全、数据可用性和访问控制的数据保护策略。这包括：

+ 合法合规采集数据
+ 数据备份
+ 数据加密存储
+ 数据销毁

### 2. **事件响应机制**

* 企业应建立暗网监测机制和数据泄露事件响应机制，确保在发生安全事件时，能迅速应对并减少损失。

### 3. **数据防泄漏（DLP）解决方案**

* 部署DLP技术，实时监控、识别和保护敏感数据，防止数据通过网络、邮件、外部设备等渠道泄露。

### 4. **限制数据访问**

* 采用访问控制技术，确保只有经过授权的人员才能访问敏感数据。

### 5. **选择安全的第三方供应商**

* 在与第三方共享数据时，选择安全可靠的供应商，并要求其采取适当的安全措施。
