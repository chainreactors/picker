***漏洞详情***

**1.****用友网络科技股份有限公司用友NC存在文件上传漏洞**

漏洞介绍：

用友NC是一款大型erp企业管理系统与电子商务平台。

漏洞危害：

用友网络科技股份有限公司用友NC存在文件上传漏洞，攻击者可利用该漏洞获取服务器权限。

影响范围：

用友网络科技股份有限公司 用友NC 6.5

修复方案：

及时测试并升级到最新版本或升级版本

来源:CNVD

2.******SolarWinds Access Rights Manager反序列化漏洞******

漏洞介绍：

SolarWinds Access Rights Manager是SolarWinds公司的一个轻量级的审查管理系统。

漏洞危害：

SolarWinds Access Rights Manager存在反序列化漏洞，攻击者可利用该漏洞远程执行代码。

漏洞编号：

CVE-2024-40057

影响范围：

Solarwinds Access Rights Manager <=2023.2.2

修复方案：

及时测试并升级到最新版本或升级版本

来源：CNVD

****3.****Microsoft DNS Server远程代码执行漏洞********

漏洞介绍：

Microsoft DNS Server是美国微软（Microsoft）公司的一个服务。

漏洞危害：

Microsoft DNS Server存在远程代码执行漏洞，攻击者可利用该漏洞在系统上执行任意代码。

漏洞编号：

CVE-2024-26221

影响范围：

Microsoft Windows Server 2016

Microsoft Windows Server 2019

Microsoft Windows Server 2022

修复方案：

及时测试并升级到最新版本或升级版本

来源：CNVD

****4.****Apache HTTP Server代码执行漏洞********

漏洞介绍：

Apache HTTP Server是美国阿帕奇（Apache）基金会的一款开源网页服务器。该服务器具有快速、可靠且可通过简单的API进行扩充的特点。

漏洞危害：

Apache HTTP Server存在代码执行漏洞，攻击者可利用该漏洞在配置允许的目录中执行脚本。

漏洞编号：

CVE-2024-26221

影响范围：

Apache HTTP Server >=2.4.0，<=2.4.59

修复方案：

及时测试并升级到最新版本或升级版本

来源：CNVD
