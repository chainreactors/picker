作为削减开支应对全球经济疲软的一部分，代码托管平台 GitHub 之前已经进行了裁员，但裁员并未停止：GitHub 开启的新裁员计划在印度裁剪了 100 多个工作岗位，这实际上就是将 GitHub 在印度的工程团队全部裁掉。

[![GitHub再次裁员！位于印度的GitHub工程团队数百名员工几乎全部被裁](https://img.lancdn.com/landian/2022/12/96646.png)](https://img.lancdn.com/landian/2022/12/96646.png)

知名科技网站 [TechCrunch](https://techcrunch.com/2023/03/27/github-slashes-engineering-team-in-india/) 报道了这起裁员，而最初消息是由前软件工程师兼《Pragmatic Engineer》电子报的撰稿人 Gergely Orosz 发布的，目前相关消息已经得到证实。

GitHub 的一位发言人告诉 TechCrunch，此次裁员是公司在今年 2 月份公布的精简计划的一部分。GitHub 曾表示，将在截止到 3 月的这一季度裁减大约 10% 的员工。

该发言人表示：“作为 2 月份分享的重组计划的一部分，我们今天做出了困难但必要的决策和调整，进行了人员削减。这既是保护我们短期业务健康的措施，也为我们未来长期战略的投入赋予了能力。”

印度是全球最大的开发者市场之一，也是许多大公司的关键工程中心。此次 GitHub 在印度的裁员无疑给当地开发者市场带来了一定的冲击。作为微软旗下的子公司，GitHub 的此次举措也引发了业界对全球市场未来走势的关注。
