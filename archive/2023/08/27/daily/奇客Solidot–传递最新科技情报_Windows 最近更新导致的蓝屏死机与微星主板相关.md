## Windows 最近更新导致的蓝屏死机与微星主板相关

[![Bug](https://icon.solidot.org/images/topics/topicbug.png?123)](/search?tid=53 "Bug")

[Wilson](/~Wilson) (42865)发表于 2023年08月26日 19时07分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75903&appkey=1370085986&title=Windows%20%E6%9C%80%E8%BF%91%E6%9B%B4%E6%96%B0%E5%AF%BC%E8%87%B4%E7%9A%84%E8%93%9D%E5%B1%8F%E6%AD%BB%E6%9C%BA%E4%B8%8E%E5%BE%AE%E6%98%9F%E4%B8%BB%E6%9D%BF%E7%9B%B8%E5%85%B3 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自天风怒**

部分 Windows 10 和 Windows 11 用户在安装了最新预览版更新 KB5029351 后遭遇了错误信息为 UNSUPPORTED\_PROCESSOR 的蓝屏死机。微星公司现在证实该问题与其主板相关，它在官方声明中称使用 MSI 600/700 系列主板的用户遭遇了蓝屏死机，如果该补丁没有自动卸载，那么建议用户回滚 BIOS 到上一个版本，然后卸载 KB5029351。微软表示受影响的操作系统版本包括 Windows 10 21H2/22H2 和 Windows 11 22H2，它表示在多次蓝屏死机之后系统应该会自动卸载该补丁。
