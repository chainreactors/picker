# 知名伏特加品牌遭勒索攻击后申请破产

#

近期，全球知名伏特加品牌斯托利（Stolichnaya）制造商Stoli集团遭受了勒索软件攻击，导致其ERP系统瘫痪，核心业务被迫转为手动操作。此次网络攻击不仅造成运营中断，还使得公司无法向贷方提供财务报告，面临7800万美元的违约债务指控。此外，Stoli集团在俄罗斯的最后两家酒厂也被当地政府没收，资产价值高达1亿美元。这一系列事件迫使Stoli集团在美国的子公司申请了破产保护。

# 19岁少年黑客涉嫌入侵电信公司被起诉

#

近日，美国一名19岁少年因涉嫌与网络犯罪组织Scattered Spider合作遭到检方起诉。据称，该男子试图通过利用电信公司系统的访问权限向数百万用户发送钓鱼短信的方式窃取加密货币。同时，他通过访问该公司的电话和短信管理系统，对特定客户实施精准攻击。此外，他还入侵了一家美国金融机构，通过欺骗员工的方式获得账户权限，并窃取敏感金融数据。报道称，该男子的攻击行为造成电信公司与金融机构共计损失400万美元。

# 英国国防部员工登录凭证在暗网被发现

#

近期，英国国防部（MOD）约600名员工登录凭证在暗网被发现，这些凭证可用于访问MOD的防御门户网站，涉及人力资源、教育和培训等模块。据报道，这些被盗凭证影响了包括伊拉克、卡塔尔、塞浦路斯以及欧洲和英国本土访问网站的员工。尽管网站采用多因素身份验证（MFA）增加了安全屏障，但目前仍未可知这些相关凭证是如何被盗取的，以及被盗取的凭证是否被恶意使用。

# 福建公安破获重大侵犯公民个人信息案件

#

近期，福建公安网安部门破获了一起侵犯公民个人信息的重大案件，犯罪分子非法窃取了网购平台的快递面单，非法获取了包括姓名、手机号及收货地址在内的公民个人信息，并将其出售给商户或中介以牟取不当利益。该行为不仅侵犯了公民的个人信息安全，还破坏了网络经济的正常秩序，对社会造成了极大的负面影响。友情提示广大用户提高网络安全意识，保护个人信息，也不要非法获取或交易他人个人信息！

# 非法加密通信服务组织Matrix被国际警方联合捣毁

#

在国际联合行动“Passionflower”中，法国和荷兰警方成功捣毁了非法加密通信服务组织Matrix。据悉，Matrix因其高度匿名性和复杂的技术架构被犯罪分子用于毒品走私、军火交易和洗钱等不法行为。通过使用创新技术，警方截获了超过230万条加密消息，揭露了该组织犯罪活动的细节，在多国展开行动逮捕了包括主要运营者在内的三名嫌疑人，查获了现金、加密货币和设备，并关闭了Matrix的核心服务器，切断了其运营能力。
