# 2022-ISRC年终冲榜活动！大点干！

[资讯](https://www.secpulse.com/archives/category/news)

[PIG-Z](https://www.secpulse.com/newpage/author?author_id=26210)
![]( https://www.secpulse.com/wp-content/themes/secpulse2017/img/renzheng3.png)

2022-12-12

9,408

**为感谢2022年所有白帽子积极地为ISRC发现外部风险，小合决定这次活动"大点干"！**

 活动时间

2022年12月12日-2022年12月26日

 测试范围

|  |  |
| --- | --- |
| **产品名称** | **域名** |
| 名片全能王 | \*.camcard.com |
| 扫描全能王 | \*.camscanner.com |
| 启信宝 | \*.qixin.com |
| TextIn | \*.textin.com |
| 蜜蜂试卷APP | 应用市场搜索"蜜蜂试卷 |
| 蜜蜂家校小程序 | 微信小程序搜索"蜜蜂家校" |
| 其他 | \*.intsig.net、\*.bertadata.com |

 活动一：双倍金币，多多多多多倍积分！

双倍金币活动：活动期间，所有有效漏洞均享受基础奖励**2倍金币**奖励；

多倍积分活动：中、低危：**2倍积分**、高危：**3倍积分**、严重：**4倍积分**

活动二：猛男奖励！

**奖励产品范围：**启信宝个人版、启信慧眼、扫描全能王、蜜蜂试卷、蜜蜂家校、名片全能王、名片全能王企业版、TetxIn

|  |  |
| --- | --- |
| 产品数量 | 奖励（要求：每款各1个高危及以上漏洞） |
| 两款 | 2000元京东卡 |
| 三款 | 4000元京东卡 |
| 四款 | 10000元京东卡 |

四款产品有效漏洞达到奖励要求送京东卡，最高可达**一万元**！

注：上述每款产品包括所属小程序、app、web等资产，若同一款产品发现多个漏洞，则不适用上述规则。

例如：小华在启信宝、扫描全能王2款产品中各发现1个高危漏洞，则本次额外奖励**2000**元京东卡

小帅在扫描全能王1款产品中发现2个高危漏洞，则本次不享受额外奖励

以此类推……

 活动三：新年爆款！

活动期间：凡提交有效漏洞，均可获得**ISRC定制新款帽子**一个！

![](https://secpulseoss.oss-cn-shanghai.aliyuncs.com/wp-content/uploads/1970/01/beepress-image-193394-1670837765.jpeg)
