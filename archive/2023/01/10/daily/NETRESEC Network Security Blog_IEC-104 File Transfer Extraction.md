## [IEC-104 File Transfer Extraction](/?page=Blog&month=2023-01&post=IEC-104-File-Transfer-Extraction)

Did you know that the SCADA protocol IEC 60870-5-104 (IEC-104) can be used to transfer files? This file transfer feature is primarily used for retrieving disturbance data from electric grid protection devices,
such as [protective relays](https://en.wikipedia.org/wiki/Protective_relay), but can in practice be used to transfer any type of data.

In this video I demonstrate how IEC-104 file transfers can be extracted from network traffic with [NetworkMiner](https://www.netresec.com/?page=NetworkMiner).

[![

The video cannot be played in your browser.
](https://www.netresec.com/images/iec-104-file-transfer_1248x648.png)](https://media.netresec.com/videos/iec-104-file-transfer_1248x648.webm)

The network traffic that was captured with NetworkMiner in this video can be downloaded here:
[NM\_2022-12-13T14-16-00.pcap](https://media.netresec.com/pcap/NM_2022-12-13T14-16-00.pcap)

The IEC-104 software used in the video was the IEC 104 RTU Server Simulator and IEC 104 Client Simulator from [FreyrSCADA](https://freyrscada.com/).
