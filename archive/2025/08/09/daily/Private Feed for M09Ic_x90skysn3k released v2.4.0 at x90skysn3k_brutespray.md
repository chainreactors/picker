# Brutespray v2.4.0 Release Summary

## Major New Features

* Added HTTP/HTTPS brute force support with dedicated wordlists
* Implemented performance monitoring and metrics module
* Complete threading and concurrency system revamp
* Added RDP domain authentication support
* Added no-color output option for logging/CI environments

## Technical Improvements

* Enhanced SOCKS5 proxy handling and error reporting
* Improved connection pooling and resource management
* Capped retry attempts with better failure handling
* Major telnet module reliability improvements
* Significantly improved output formatting and progress reporting

## Dependency Updates

* Go toolchain: 1.22.4 → 1.23.0 (toolchain 1.24.0)
* mysql driver: v1.8.1 → v1.9.3
* gosnmp: v1.38.0 → v1.42.1
* pterm: v0.12.80 → v0.12.81
* go-ora: v2.8.22 → v2.9.0
* mongo-driver: v1.17.1 → v1.17.4
* golang.org/x/crypto: v0.31.0 → v0.40.0
* golang.org/x/net: v0.33.0 → v0.41.0

## Bug Fixes

* Fixed SOCKS5 proxy connection issues
* Resolved multiple linting errors
* Enhanced connection stability across all modules
* Better timeout and network failure handling

## Statistics

* 21 files changed: +2,402 insertions, -427 deletions
* Major refactoring of core brute force engine
* New HTTP/HTTPS wordlist directories added

* New HTTP/HTTPS brute forcing capability
