![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C3thlY5N3Ec94lWdJ1zBUjBvD6Qtra8gb4TVxmWn8iagZJX9587prtPGQ/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C32gNV3Anh6jLJuaABWbj9vT95Wibj08b11RQ843Rs2YrdYS9VV3skh6Q/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C3VYCiberUAf1tQH0At7PMc1OOcmpMydyRapu1M9HyZZcu4Gia0jiau4hxw/640?wx_fmt=png&from=appmsg)

翻翻功能点，抓抓包，看看有没有 sql，运气不好，没有。

其次也可能通过 druid 去查看有没有注入参数

在 sql 监控里面，会有 sql 语句，随便点一个就可以看到，传参方式都是？,代表是预编译

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C37gP2RmaYJzemsa474GGyVyEMGDdLc8KZJkDyM0J6xTccJ2WTn5N4vg/640?wx_fmt=png&from=appmsg)

所以就不会有注入漏洞。

继续翻数据包，发现一个 user 接口，刚好 有个 id

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C3yxfbpDFibn9fKpptiaWmcDuWI7ialcQjdQhdU5qdicXTxiacKnLuYpfTT9g/640?wx_fmt=png&from=appmsg)

那越权不就来了吗，遍历 id 便会得到其他用户信息，同时也可以搜集用户名

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C3ZhrPH6gsCAnSKCw4KM5LzT5leibwyFE833I10CL6BibFeN3t15rhb43A/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/BAby4Fk1HQa6YicsjJtUBH7ulIvRqB7C3KbWTBCCWD8SMG6zzmfygfjEClJDMbHnwhN54VFWorySib0NTuKEOLCw/640?wx_fmt=png&from=appmsg)
