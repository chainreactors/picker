对不少用户来说鼠标手势还是个非常实用的功能，在早前的测试版中微软就逐步向部分 Microsoft Edge 用户推送了鼠标手势功能，到现在 Microsoft Edge 114 正式版发布，鼠标手势虽然未被默认启用，但也可以通过追加启动命令的方式开启。

至少从功能更迭方面来说，Microsoft Edge 还是一款非常不错的浏览器的，就是每次打开总是各种弹窗广告着实让人不胜其烦。

比如在测试鼠标手势的时候，打开 Microsoft Edge 首先来俩弹窗，就这种情况用户欢迎那是不现实的，即便有各种各样的新功能，估计不少用户还是会选择 Google Chrome 和 Mozilla Firefox。

**鼠标手势功能如何启用：**

右键点击 Microsoft Edge 图标、属性，然后在目标里加上空格再粘贴下面这行代码保存即可。如果图标固定在任务栏上，则右键点击任务栏上的 Edge 图标再点属性，操作方法基本一致。

```
--enable-features=msEdgeMouseGestureDefaultEnabled,msEdgeMouseGestureSupported
```

保存后启动 Microsoft Edge 浏览器，然后转到设置，搜索关键词 鼠标 即可找到配置页面，默认情况下鼠标手势已经被开启，至于不同的手势，用户亦可自定义。

**如何使用：**

按住鼠标右键滑动即可，比如按住鼠标右键向下滑动是向下滚动页面。
