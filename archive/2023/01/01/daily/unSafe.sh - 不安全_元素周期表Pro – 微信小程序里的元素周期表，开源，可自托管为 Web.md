元素周期表Pro – 微信小程序里的元素周期表，开源，可自托管为 Web

**元素周期表Pro** 是一个开源的元素周期表项目，可以让用户在微信小程序中查看到高颜值的元素周期表，并且还拥有溶解性表、元素百科、深色模式、元素图片、物理性质等信息。而开源项目的好处在于，你可以将其自托管为 Web 项目，在浏览器中直接使用。@[Appinn](https://www.appinn.com/periodic-table-pro/)

## 元素周期表Pro [微信小程序]

在小程序中，由于屏幕尺寸限制，只能显示部分元素周期表，通过拖动浏览，以及搜索来查找所有的元素。

![元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 3](https://static1.appinn.com/images/202212/img_246dd19b024b-1.jpg!o "元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 4")

包含的元素信息非常丰富，不次于上面提到的那个 app **[元素周期表](https://www.appinn.com/periodic-table/)**，包括：

* 物理性质
* 原子性质
* 电磁性质
* 其他属性
* 含量
* 发现年份

在小程序中，会偶尔显示广告。

![元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 4](https://static1.appinn.com/images/202212/gh_56cdd91ee838_430.jpg "元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 5")

### 自托管一个元素周期表

开发者实际上提供了一个 [Web](https://periodic-table-pro.netlify.app/pages/index/index) 页面，即开即用。如果有大屏幕，那用起来还是很爽的。

![元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 5](https://static1.appinn.com/images/202212/screen-appinn2022-12-31_17_31_54.jpg!o "元素周期表Pro - 微信小程序里的元素周期表，开源，可自托管为 Web 6")

而想要自己托管，也挺容易的，只需要以下几条命令：

```
git clone https://github.com/baotlake/periodic-table-pro.git
yarn install
npm run build:h5 -- --watch
```

然后，浏览器会自动打开 http://192.168.1.2:10086/ 效果和上面的图片一样。
