**01**

**Operation DevilTiger：APT-Q-12 使用 0day 漏洞技战术披露**

**披露时间：**2024年8月26日

**情报来源：**https://mp.weixin.qq.com/s/vB-4LJqTddWohS5g2zwVVA

**相关信息：**

APT-Q-12，又名伪猎者，是一个具有东北亚背景的 APT 组织，其攻击目标主要在东亚地区。奇安信威胁情报中心自 2021 年起对其进行追踪，发现 APT-Q-12 与其他几个有朝鲜半岛背景的攻击集合有重叠，认为它们可能是 Darkhotel 组织的子集。

文章详细介绍了 APT-Q-12 组织的信息收集过程，包括探测受害者使用的邮件品牌和平台信息，以及如何利用 0day 漏洞进行攻击。在 Windows 平台上，APT-Q-12 利用基于 CEF 框架的软件脆弱性，通过 XSS 漏洞和内部接口调用执行恶意代码。攻击者使用特殊的技术来绕过安全检测，收集用户代理信息，从而确定目标使用的邮件客户端和平台。在 Android 平台上，APT-Q-12 利用邮件客户端的 XSS 漏洞执行恶意代码，实现对目标手机的长期控制，收集与中朝贸易有关的情报。

此外，还对 APT-Q-12 使用的恶意软件进行了分析，包括其功能、C2 通信和插件类型。这些插件包括键盘记录器、浏览器窃密插件、隧道工具和屏幕截图插件等，用于收集敏感信息和进行长期监控。

**披露时间：**2024年8月19日

**情报来源：**https://cert.gov.ua/article/6280422

**相关信息：**

乌克兰计算机应急响应小组(CERT-UA)近日发布警告称，UAC-0020 组织正发动新一轮网络钓鱼攻击，通过散布包含所谓库尔斯克地区战俘照片的钓鱼邮件，引诱收件人下载并打开包含恶意代码的 ZIP 压缩包(spysok\_kursk.zip)。一旦受害者打开其中的 CHM 文件(list of vp Dropout.kursk.chm)，恶意JavaScript代码便会启动混淆的 PowerShell 脚本，进一步安装间谍软件 SPECTR 及其新型同类软件 FIRMACHAGENT。而FIRMACHAGENT的主要功能是检索 SPECTR 盗取的数据并将其发送至远程管理服务器。

**03**

**APT-C-60 利用 WPS Office 零日漏洞部署恶意软件**

**披露时间：**2024年8月28日

**情报来源：**https://www.welivesecurity.com/en/eset-research/analysis-of-two-arbitrary-code-execution-vulnerabilities-affecting-wps-office/

**相关信息：**

ESET 研究人员发现 APT-C-60 在野外利用 WPS Office for Windows 中的代码执行漏洞（CVE-2024-7262）针对东亚国家/地区发起攻击，最终的有效载荷是一个自定义后门，内部命名为 SpyGlace。

APT-C-60创建了电子表格文档（MHTML 文件），其中嵌入了隐藏在诱饵图像下的恶意超链接，以诱骗受害者点击它们，从而触发漏洞。处理后的 URL参数包括一个 base64 编码的命令，用于执行特定插件 (promecefpluginhost.exe)，该插件试图加载包含攻击者代码的恶意 DLL(ksojscore.dll)。该DLL是APT-C-60的下载器组件，用于从攻击者的服务器获取最终的有效负载（TaskControler.dll），这是一个名为“SpyGlace”的自定义后门。

漏洞CVE-2024-7262源于攻击者提供的文件路径缺乏清理和加载插件的验证不足，研究人员在对 CVE-2024-7262 补丁进行分析，发现通过劫持同一个 WPS Office 插件组件的控制流可导致另一个代码执行漏洞CVE-2024-7263。

**Pioneer Kitten组织对美国发起勒索攻击**

**披露时间：**2024年8月28日

**情报来源：**https://www.cisa.gov/news-events/cybersecurity-advisories/aa24-241a

**相关信息：**

美国发布联合安全公告称，Pioneer Kitten于本月侵入美国和外国网络，窃取敏感数据并部署勒索软件。攻击者通过 Check Point、Citrix、Palo Alto Networks 和其他制造商的易受攻击的 VPN 和防火墙设备入侵。攻击目标包括美国学校、银行、医院、国防部门组织和政府机构，以及以色列、阿塞拜疆和阿拉伯联合酋长国的目标。

**APT33利用新型Tickler恶意软件攻击美国多个部门**

**披露时间：**2024年8月28日

**情报来源：**https://www.microsoft.com/en-us/security/blog/2024/08/28/peach-sandstorm-deploys-new-custom-tickler-malware-in-long-running-intelligence-gathering-operations/

**相关信息：**

据微软安全研究人员观察，APT33组织（也被追踪为 Peach Sandstorm 和 Refined Kitten）在 2024 年 4 月至 7 月期间使用新型Tickler恶意软件对美国和阿联酋的政府、国防、卫星、石油和天然气部门组织的网络进行后门攻击。

这些攻击过程中，攻击者在通过密码喷洒攻击或社会工程学进行初始访问后，该组织部署了一个新的自定义多阶段后门 Tickler，并利用托管在攻击者控制的欺诈性 Azure 订阅中的 Azure 基础设施进行命令和控制 (C2)。

发现的第一个 Tickler 恶意软件样本从主机收集网络信息，并通过 HTTP POST 请求将其发送到 C2 URI。随后，攻击者改进了该恶意软件，第二个样本 sold[.]dll 是一个木马植入程序，它会从 C2 服务器下载其他负载。这包括一个后门、一个用于设置此后门持久性的批处理脚本以及一些合法文件。成功入侵后，攻击者会进行：利用服务器消息块 (SMB) 横向移动、下载并安装远程监控和管理 (RMM) 工具、拍摄 Active Directory (AD) 快照等操作。

**01**

**ShinyHunters 勒索软件背后的威胁组织Bling Libra分析**

**披露时间：**2024年8月23日

**情报来源：**https://unit42.paloaltonetworks.com/shinyhunters-ransomware-extortion/

**相关信息：**

Bling Libra 组织在 2020 年首次出现，并与多起重大数据泄露事件有关，例如 Microsoft GitHub 和 Tokopedia 攻击。他们的攻击目标跨越不同行业和地区，但攻击手法保持一致：获取合法凭证，针对数据库基础设施收集个人信息（PII），然后在暗网市场上转售

近期，Unit 42 在处理一起事件响应时发现，Bling Libra（ShinyHunters 勒索软件背后的组织）改变了策略，从出售被盗数据转向敲诈受害者，并将目标锁定在云环境上。

据悉，Bling Libra 利用公开存储库中的合法凭证获取对组织 AWS 环境的初始访问权限，一旦 Bling Libra 获得对组织 AWS 环境的访问权限，他们就会执行各种 API 调用，以确定被盗凭证所包含的权限范围。之后攻击者使用 S3 Browser 和 WinSCP 等工具收集有关 S3 存储桶配置的信息、访问并删除数据。删除所有 S3 存储桶后，威胁行为者使用 AWS CLI 的自动脚本尝试创建新的 S3 存储桶，最后向受害组织发送勒索电子邮件。

**针对医疗保健、教育和公共部门的勒索软件攻击**

**披露时间：**2024年8月19日

**情报来源：**https://www.zscaler.com/blogs/security-research/5-key-takeaways-ransomware-attacks-healthcare-education-and-public-sector

**相关信息：**

ThreatLabz研究显示，由于生成式AI技术的进步，过去一年中勒索软件攻击的频率和大胆程度显著增加，全球勒索软件事件增长了17.8%。特别令人关注的是，通过恶意数据泄露站点分析识别的勒索软件攻击激增了57.8%。这些攻击包括有史以来最大胆的勒索要求和支付记录，包括2024年创纪录的7500万美元勒索支付。

医疗保健和教育行业在2023年4月至2024年4月期间成为受勒索软件攻击最严重的行业之一，政府机构也经历了勒索软件攻击的显著同比增长。以下是 ThreatLabz 2024 年勒索软件报告中的五个关键要点：

* 医疗保健行业成为勒索软件攻击的第二大目标行业；
* 教育机构作为受勒索软件影响第四大的行业，面临越来越大的压力；
* 政府机构遭受勒索软件攻击同比增长近 50%；
* 企业面临着一年内遭受多次勒索软件攻击的威胁；
* LockBit 仍然是一股强大的勒索软件力量，对关键行业构成持续威胁。

**03**

**Microsoft Sway 在大规模二维码网络钓鱼活动中被滥用**

**披露时间：**2024年8月27日

**情报来源：**https://www.netskope.com/blog/phishing-in-style-microsoft-sway-abused-to-deliver-quishing-attacks

**相关信息：**

2024 年 7 月，Netskope 威胁实验室发现通过 Microsoft Sway 传递的网络钓鱼页面流量增加了 2,000 倍。调查的大多数凭证获取页面都使用了“Quishing”，这是一种使用二维码诱骗用户访问恶意网站的网络钓鱼形式。网络钓鱼活动以 MS Office 凭证为目标，使用文档诱骗用户登录。这些活动的受害者主要来自亚洲和北美，涉及技术、制造业和金融等多个领域。

这些电子邮件将潜在受害者重定向到托管在 sway.cloud.microsoft 域上的网络钓鱼登录页面，攻击者通常鼓励受害者使用移动设备扫描二维码，而移动设备的安全措施通常较弱，从而增加了绕过安全控制的机会，并允许他们不受限制地访问钓鱼网站。

此外，这些二维码钓鱼活动还采用了透明网络钓鱼和 Cloudflare Turnstile 两种技术。透明网络钓鱼可确保受害者访问合法登录页面的确切内容，并允许他们绕过多因素身份验证等其他安全措施。同时，Cloudflare Turnstile 用于向静态内容扫描器隐藏网络钓鱼负载，从而维护其域的良好声誉。
