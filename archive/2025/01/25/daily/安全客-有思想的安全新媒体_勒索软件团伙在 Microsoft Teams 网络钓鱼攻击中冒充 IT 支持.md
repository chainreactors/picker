иҜ‘ж–Үд»…дҫӣеҸӮиҖғпјҢе…·дҪ“еҶ…е®№иЎЁиҫҫд»ҘеҸҠеҗ«д№үеҺҹж–ҮдёәеҮҶгҖӮ

![Ransomware gangs pose as IT support in Microsoft Teams phishing attacks]()

еӢ’зҙўиҪҜд»¶еӣўдјҷи¶ҠжқҘи¶ҠеӨҡең°йҮҮз”Ёз”өеӯҗйӮ®д»¶иҪ°зӮёзҡ„ж–№ејҸпјҢз„¶еҗҺеңЁ Microsoft Teams йҖҡиҜқдёӯеҶ’е……жҠҖжңҜж”ҜжҢҒдәәе‘ҳпјҢиҜұйӘ—е‘ҳе·Ҙе…Ғи®ёиҝңзЁӢжҺ§еҲ¶е№¶е®үиЈ…жҒ¶ж„ҸиҪҜд»¶пјҢд»ҺиҖҢжҸҗдҫӣеҜ№е…¬еҸёзҪ‘з»ңзҡ„и®ҝй—®жқғйҷҗгҖӮ

еЁҒиғҒеҲҶеӯҗдјҡеңЁзҹӯж—¶й—ҙеҶ…еҸ‘йҖҒж•°еҚғе°ҒеһғеңҫйӮ®д»¶пјҢз„¶еҗҺд»ҺеҜ№жүӢжҺ§еҲ¶зҡ„ Office 365 е®һдҫӢдёӯе‘јеҸ«зӣ®ж ҮпјҢеҒҮиЈ…жҸҗдҫӣ IT ж”ҜжҢҒгҖӮ

иҮӘеҺ»е№ҙдёӢеҚҠе№ҙд»ҘжқҘпјҢдәәд»¬е°ұеңЁ Black Basta еӢ’зҙўиҪҜд»¶зҡ„ж”»еҮ»дёӯеҸ‘зҺ°дәҶиҝҷз§ҚжҲҳжңҜпјҢдҪҶзҪ‘з»ңе®үе…Ёе…¬еҸё Sophos зҡ„з ”з©¶дәәе‘ҳеҸ‘зҺ°пјҢеҸҜиғҪдёҺ FIN7 з»„з»Үжңүе…іиҒ”зҡ„е…¶д»–еЁҒиғҒиҖ…д№ҹеңЁдҪҝз”ЁеҗҢж ·зҡ„ж–№жі•гҖӮ

дёәдәҶдёҺе…¬еҸёе‘ҳе·ҘеҸ–еҫ—иҒ”зі»пјҢй»‘е®ўеҲ©з”ЁдәҶзӣ®ж Үз»„з»Үзҡ„й»ҳи®Ө Microsoft Teams й…ҚзҪ®пјҢиҜҘй…ҚзҪ®е…Ғи®ёд»ҺеӨ–йғЁеҹҹжӢЁжү“з”өиҜқе’ҢиҒҠеӨ©гҖӮ

**и§ӮеҜҹеҲ°зҡ„жҙ»еҠЁ**

Sophos и°ғжҹҘзҡ„з¬¬дёҖдёӘжҙ»еҠЁдёҺз ”з©¶дәәе‘ҳеҶ…йғЁи·ҹиёӘзҡ„ STAC5143 е°Ҹз»„жңүе…ігҖӮй»‘е®ўйҰ–е…Ҳеҗ‘зӣ®ж ҮеҸ‘йҖҒдәҶеӨ§йҮҸйӮ®д»¶пјҢ45 еҲҶй’ҹеҶ…еҸ‘йҖҒдәҶ 3,000 е°ҒйӮ®д»¶гҖӮ

дёҚд№…д№ӢеҗҺпјҢзӣ®ж Үе‘ҳе·ҘжҺҘеҲ°дәҶдёҖдёӘеҗҚдёә вҖңеё®еҠ©еҸ°з»ҸзҗҶ вҖқзҡ„еӨ–йғЁеӣўйҳҹз”өиҜқгҖӮеЁҒиғҒиҖ…иҜҙжңҚеҸ—е®іиҖ…йҖҡиҝҮ Microsoft Teams и®ҫзҪ®иҝңзЁӢеұҸе№•жҺ§еҲ¶дјҡиҜқгҖӮ

ж”»еҮ»иҖ…еңЁдёҖдёӘеӨ–йғЁ SharePoint й“ҫжҺҘдёҠжҠ•ж”ҫдәҶдёҖдёӘ Java еҪ’жЎЈ (JAR) ж–Үд»¶пјҲMailQueue-Handler.jarпјүе’Ң Python и„ҡжң¬пјҲRPivot еҗҺй—ЁпјүгҖӮ

JAR ж–Үд»¶жү§иЎҢ PowerShell е‘Ҫд»ӨдёӢиҪҪеҗҲжі•зҡ„ ProtonVPN еҸҜжү§иЎҢж–Үд»¶пјҢиҜҘж–Үд»¶дҫ§иҪҪдәҶжҒ¶ж„Ҹ DLL (nethost.dll)гҖӮ

иҜҘ DLL дёҺеӨ–йғЁ IP еҲӣе»әдәҶдёҖдёӘеҠ еҜҶзҡ„е‘Ҫд»ӨдёҺжҺ§еҲ¶пјҲC2пјүйҖҡдҝЎйҖҡйҒ“пјҢдёәж”»еҮ»иҖ…иҝңзЁӢи®ҝй—®иў«е…Ҙдҫөи®Ўз®—жңәжҸҗдҫӣдәҶдҫҝеҲ©гҖӮ

ж”»еҮ»иҖ…иҝҳиҝҗиЎҢдәҶ Windows Management Instrumentation (WMIC) е’Ң whoami.exe жқҘжЈҖжҹҘзі»з»ҹз»ҶиҠӮпјҢе№¶йғЁзҪІдәҶз¬¬дәҢйҳ¶ж®ө Java жҒ¶ж„ҸиҪҜд»¶жқҘжү§иЎҢ RPivot вҖ“ дёҖз§Қе…Ғи®ёйҖҡиҝҮ SOCKS4 д»ЈзҗҶйҡ§йҒ“еҸ‘йҖҒе‘Ҫд»Өзҡ„жё—йҖҸжөӢиҜ•е·Ҙе…·гҖӮ

![Obfuscated RPivot code]()
**з»ҸиҝҮж··ж·Ҷзҡ„ RPivot д»Јз Ғ**
жқҘжәҗпјҡSophos

FIN7 иҝҮеҺ»жӣҫеңЁж”»еҮ»дёӯдҪҝз”ЁиҝҮ RPivotгҖӮжӯӨеӨ–пјҢжүҖдҪҝз”Ёзҡ„ж··ж·ҶжҠҖжңҜд№ҹжӣҫеңЁ FIN7 зҡ„жҙ»еҠЁдёӯеҮәзҺ°иҝҮгҖӮ

з„¶иҖҢпјҢз”ұдәҺ RPivot е’Ңж··ж·Ҷж–№жі•зҡ„д»Јз ҒйғҪжҳҜе…¬ејҖзҡ„пјҢSophos ж— жі•д»Ҙй«ҳеәҰзҡ„еҸҜдҝЎеәҰе°Ҷ STAC5143 ж”»еҮ»дёҺ FIN7 зҡ„жҙ»еҠЁиҒ”зі»иө·жқҘпјҢе°Өе…¶жҳҜдј—жүҖе‘ЁзҹҘ FIN7 иҝҮеҺ»жӣҫе°Ҷе…¶е·Ҙе…·еҮәе”®з»ҷе…¶д»–зҪ‘з»ңзҠҜзҪӘеӣўдјҷгҖӮ

з ”з©¶дәәе‘ҳи§ЈйҮҠиҜҙпјҡвҖңSophos иҜ„дј°и®ӨдёәпјҢиҝҷж¬Ўж”»еҮ»дёӯдҪҝз”Ёзҡ„ Python жҒ¶ж„ҸиҪҜд»¶дёҺ FIN7/Sangria Tempest иғҢеҗҺзҡ„еЁҒиғҒеҸӮдёҺиҖ…жңүе…іпјҢеҸҜдҝЎеәҰдёәдёӯзӯүгҖӮвҖқ

з ”з©¶дәәе‘ҳи®ӨдёәпјҢй»‘е®ўзҡ„зӣ®зҡ„жҳҜзӘғеҸ–ж•°жҚ®пјҢз„¶еҗҺйғЁзҪІеӢ’зҙўиҪҜд»¶гҖӮ

з¬¬дәҢдёӘж”»еҮ»жҙ»еҠЁжқҘиҮӘдёҖдёӘиў«иҝҪиёӘдёә вҖңSTAC5777 вҖқзҡ„з»„з»ҮгҖӮиҝҷдәӣж”»еҮ»еҗҢж ·д»Ҙз”өеӯҗйӮ®д»¶иҪ°зӮёејҖе§ӢпјҢйҡҸеҗҺжҳҜ Microsoft Teams ж¶ҲжҒҜпјҢеЈ°з§°жқҘиҮӘ IT ж”ҜжҢҒйғЁй—ЁгҖӮ

дҪҶеңЁиҝҷз§Қжғ…еҶөдёӢпјҢеҸ—е®іиҖ…дјҡиў«иҜұйӘ—е®үиЈ… Microsoft Quick AssistпјҢд»ҺиҖҢдҪҝж”»еҮ»иҖ…иҺ·еҫ—й”®зӣҳи®ҝй—®жқғйҷҗпјҢе№¶еҲ©з”ЁиҜҘжқғйҷҗдёӢиҪҪжүҳз®ЎеңЁ Azure Blob Storage дёҠзҡ„жҒ¶ж„ҸиҪҜд»¶гҖӮ

жҒ¶ж„ҸиҪҜд»¶пјҲwinhttp.dllпјүиў«дҫ§иҪҪеҲ°еҗҲжі•зҡ„ Microsoft OneDriveStandaloneUpdater.exe иҝӣзЁӢдёӯпјҢPowerShell е‘Ҫд»ӨдјҡеҲӣе»әдёҖдёӘжңҚеҠЎпјҢеңЁзі»з»ҹеҗҜеҠЁж—¶йҮҚж–°еҗҜеҠЁгҖӮ

жҒ¶ж„Ҹ DLL дјҡйҖҡиҝҮ Windows API и®°еҪ•еҸ—е®іиҖ…зҡ„жҢүй”®ж“ҚдҪңпјҢд»Һж–Үд»¶е’ҢжіЁеҶҢиЎЁдёӯиҺ·еҸ–еӯҳеӮЁзҡ„еҮӯжҚ®пјҢе№¶йҖҡиҝҮ SMBгҖҒRDP е’Ң WinRM жү«жҸҸзҪ‘з»ңд»ҘеҜ»жүҫжҪңеңЁзҡ„ж”ҜзӮ№гҖӮ

Sophos и§ӮеҜҹеҲ° STAC5777 иҜ•еӣҫеңЁзҪ‘з»ңдёҠйғЁзҪІ Black Basta еӢ’зҙўиҪҜд»¶пјҢеӣ жӯӨиҜҘеЁҒиғҒиЎҢдёәиҖ…еҫҲеҸҜиғҪдёҺиҝҷдёӘиҮӯеҗҚжҳӯи‘—зҡ„еӢ’зҙўиҪҜд»¶еӣўдјҷжңүжҹҗз§ҚиҒ”зі»гҖӮ

з ”з©¶дәәе‘ҳи§ӮеҜҹеҲ°еЁҒиғҒиЎҢдёәиҖ…и®ҝй—®дәҶж–Үд»¶еҗҚдёӯеҗ«жңү вҖңеҜҶз Ғ вҖқзҡ„жң¬ең°и®°дәӢжң¬е’Ң Word ж–ҮжЎЈгҖӮй»‘е®ўиҝҳи®ҝй—®дәҶдёӨдёӘиҝңзЁӢжЎҢйқўеҚҸи®®ж–Үд»¶пјҢеҫҲеҸҜиғҪжҳҜеңЁеҜ»жүҫеҸҜиғҪзҡ„еҮӯиҜҒдҪҚзҪ®гҖӮ

йҡҸзқҖиҝҷдәӣзӯ–з•ҘеңЁеӢ’зҙўиҪҜд»¶йўҶеҹҹеҸҳеҫ—и¶ҠжқҘи¶Ҡжҷ®йҒҚпјҢдјҒдёҡеә”иҜҘиҖғиҷ‘йҳ»жӯўеӨ–йғЁеҹҹеңЁ Microsoft Teams дёҠеҸ‘иө·ж¶ҲжҒҜе’Ңе‘јеҸ«пјҢе№¶еңЁе…ій”®зҺҜеўғдёҠзҰҒз”Ёеҝ«йҖҹеҚҸеҠ©еҠҹиғҪгҖӮ
