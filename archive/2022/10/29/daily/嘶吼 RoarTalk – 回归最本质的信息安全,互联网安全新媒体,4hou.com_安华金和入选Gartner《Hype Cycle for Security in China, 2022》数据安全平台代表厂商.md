# 安华金和入选Gartner《Hype Cycle for Security in China, 2022》数据安全平台代表厂商

导语：10月10日，国际权威机构Gartner正式发布《Hype Cycle for Security in China, 2022》，在数据安全平台（DSP）领域，安华金和凭借深厚的技术积累作为样本供应商被列入该报告。

10月10日，**国际权威机构Gartner正式发布《Hype Cycle for Security in China, 2022》，在数据安全平台（DSP）领域，安华金和凭借深厚的技术积累作为样本供应商被列入该报告。**

当下，各类数据安全产品之间缺乏有效的联动和统一调度管理，安全风险应对能力难以得到真正提升，这促使数据安全厂商将单体产品安全能力聚合到数据安全平台（DSP）中，使其能够帮助组织更好地进行数据安全建设。**该报告指出，DSP涵盖了各种场景下的数据安全保护需求，以数据发现和数据分类分级为基础，混合了多种技术来实现数据安全防护****，**例如：数据访问控制，数据脱敏，文件加密等，成熟的DSP也可能包含了数据活动监控和数据风险评估的功能。

此外，DSP可以提高数据的可视性和控制能力，以确保个人、组织和政府之间的数据流动安全。DSP能够帮助组织做出更明智的决策，并为企业和社会提供更好的数据安全保护。

2016年，**安华金和于国内首倡数据安全治理这一理念，近年来推出数据安全运营平台、数据安全协同平台、数据安全监测平台，支撑数据安全治理框架的落地实践，**目前已在政府、金融、医疗、运营商等多个行业得到实际应用。

**安华金和数据安全平台**

**01数据安全运营平台（DSOP）**

安华金和数据安全运营平台（DSOP）是以“可实用、可持续”为设计初衷，“一站式、体系化”的数据安全运营管控平台。可集合包括数据资产梳理、数据库防火墙、数据库审计、数据脱敏、数据库运维管理、数据库加密等在内的各类数据安全产品优势于一身，通过可视化的信息呈现与工作引导，真正实现“统一部署、统一监控、统一管理、统一运营”的数据安全日常化、可持续的运营管控目标。

**02数据安全协同平台（DSCP）**

提供一站式数据分析运算能力，具备敏感数据自动识别、数据自动分级分类管控、数据操作留痕审计、输出结果申报审核等功能的安华金和数据安全协同平台（DSCP），搭建了统一规范、互联互通、安全可控的数据开放环境，多方协同全程加密传输、缓存和运算，保护数据的安全和隐私，促使数据流通价值最大化，安全合规的推动跨部门、跨机构、跨行业的数据开放与共享，实现了提供数据安全流通和共享的协同生态圈。

**03数据安全监测平台（DSMP）**

安华金和数据安全监测平台（简称：DSMP）面向数据安全监管机构和大型企业集团数据安全总体责任部门，依托国家数据安全法规、行业数据安全监管政策和企业数据安全总体策略，以分布式的数据安全评估、数据库审计、应用审计为基础，通过平台化的设备管理以及关联分析能力，打破应用和数据库安全监测的壁垒，实现以数据为中心，全面的数据分类分级安全合规监测。

**屡获国际权威机构认可**

作为中国专业的数据安全产品与解决方案提供商，中国“数据安全治理”理念、体系的提出者和践行者，**安华金和在技术实力与行业领先性方面已连续多年、十数次获得Gartner的认可：**

2017-2021年，安华金和累计入选Gartner涵盖“数据安全、数据管理、威胁应对、应用安全、风险管理、隐私保护”六个领域及“企业内部关键数据的分级加密”技术在内的共**12份技术成熟度曲线研究报告，作为DAP（数据库审计与保护）领域代表厂商，同时也是进入该领域第一家中国数据安全企业。**

2020-2021，安华金和连续两年进入Gartner 发布的 Market Guide for Data Masking（数据脱敏市场指南），跻身数据脱敏领域全球代表厂商行列，同时也是该领域**代表厂商中唯一的中国数据安全企业。**

**2022年3月，安华金和入选Gartner中国数据安全代表厂商名录，**其静态数据脱敏（SDMS）、动态数据脱敏（DDMS）、数据库安全审计（DAS）与数据安全运营平台（DSOP）**四款产品入选报告**。**2022年6月，安华金和数据安全平台入选Gartner《数据安全平台创新洞察报告》**，被列为具有代表性的供应商之一。

多次的权威认可，体现了业界对安华金和在数据安全领域突出的专业能力和成熟度的肯定，**充分彰显安华金和在数据安全领域的产品、解决方案实力及体系化的服务能力。**

Gartner,Hype Cycle for Security in China, 2022，Data Security Platforms in China，Analysis By: Anson Chen，日期：2022年10月10日
