## AI视频换脸工具roop

roop是一款使用python开发的深度AI视频换脸工具，不过目前工具没有开箱即用，需要一点点的python开发命令知识，不过官方也提供文档步骤，建议使用python 3.10，安装完必须的包即可，安装使用过程中遇到其他错误善于使用搜索哈。

[![](https://img.imgdd.com/f210f3.34219fab-9789-463e-885a-5abd15fba6f4.gif)](https://img.imgdd.com/f210f3.34219fab-9789-463e-885a-5abd15fba6f4.gif)

## 安装步骤

### 1、必备的工具环境，自行搜索安装

* python (3.10 recommended)
* pip
* git
* ffmpeg
* visual studio 2022 runtimes (windows)

### 2、克隆仓库

```
git clone https://github.com/s0md3v/roop
```

### 3、安装依赖

使用cd命令进入刚刚克隆的仓库，然后安装依赖，建议使用venv虚拟环境

```
pip install -r requirements.txt
```

![](https://img.imgdd.com/f210f3.3d13ffa4-0410-425b-b9ba-884c0bad2f5a.png)

### 4、启动运行

安装完依赖使用`python run.py`命令运行 roop即可。首次运行该程序时，程序会下载一些模型，这可能需要一些时间，具体取决于你的网络连接。

![](https://img.imgdd.com/f210f3.0892ec3e-d115-4732-a32c-0dcb18f42671.png)
