**活动时间**

2022年11月1日——11月11日

(以漏洞提交时间为准)

**活动范围**

天融信所有产品加业务

（详见官网漏洞评级标准与处置流程）

**活动内容**

**一：贡献累计奖励**

仅统计活动期间的总安全币/贡献值，不对活动前期已有的安全币/贡献值进行累计，活动结束后，贡献值总和前三白帽分别获得额外奖励￥3000、￥2000、￥1000

**获奖条件：**个人至少提交中危及以上有效漏洞3个或累计贡献值达500

**二：新人特别奖励**

|  |  |  |
| --- | --- | --- |
| 奖励 | | 达成条件 |
| 新人奖励 | 现金￥300  额外赠送任意漏洞奖励1.5倍翻倍卡1张  （供12月使用） | 首次在TOPSRC提交中危及以上有效漏洞 |
| 老带新 | 现金￥200  额外赠送任意漏洞奖励1.5倍翻倍卡1张  （供12月使用） | 邀请新人，新人在TOPSRC提交中危及以上有效漏洞 |
| 注：老带新需要新人在提交时邮箱名称备注老白帽子的平台ID，反之无效 | | |

**翻倍卡使用规则：**

使用时间：2022年12月1日——12月31日

使用要求：每个漏洞仅可使用一张翻倍卡，翻倍卡不可叠加其他活动一同使用，后期使用翻倍卡时在漏洞标题处备注【11月翻倍卡+翻倍倍数】即可（反之无效，请注意）

**例如:**老白帽A拉新白帽B挖TOPSRC漏洞，当B首次在TOPSRC提交中危及以上有效漏洞时，除漏洞本身奖励，还加获现金￥300，同时额外赠送任意漏洞奖励1.5倍翻倍卡1张（供12月使用），当B在提交时邮箱名称备注老白帽子A的平台ID时，A可加获现金￥200，同时额外赠送任意漏洞奖励1.5倍翻倍卡1张（供12月使用），若B在邮箱名称提交时未加A的平台ID，则A不获奖励，仅B获得该新人奖励。

**三：文章转发抽奖**

1、关注“天融信安全漏洞响应中心”公众号；

2、转发本文至朋友圈（不分组且保留至开奖）；

3、扫描下方二维码即可参与活动抽奖。

**温馨提示：**

**·**白帽子在漏洞测试中应遵守《SRC行业安全测试规范》；

**·**TOPSRC的各位白帽请尽量使用同一账号提交漏洞，工作人员会不定期核实该账号的有效性，若发现一人注册多个账号，将取消该用户活动资格；

**·** 请注意及时在“个人中心”正确填写收货方式，以便于您兑换礼品的收发。
