## Claude Code 中文开发套件概览

**Claude Code 中文开发套件**为中国开发者定制，提供完整中文化与零门槛体验。集成 MCP 服务器、智能上下文管理与安全扫描，支持通过 AnyRouter 转发节点使用，无需翻墙即可获得稳定的 AI 编程体验。基于 Claude Code Development Kit 的本地化版本，安装到配置全链路中文可用。

## Claude Code 中文开发套件关键特性

### 完全中文化

* 中文 AI 指令与提示词，全局对话与上下文均为中文
* 三层中文文档体系：基础层 / 组件层 / 功能层
* 中文错误信息与帮助文档
* 从安装到配置的中文引导

### 智能上下文管理

* 三层文档分级管理，结构清晰
* **自动上下文注入**：子智能体读取项目上下文
* **智能文档路由**：依据任务复杂度加载合适文档
* **跨会话状态管理**：任务交接与状态保持更连贯

### 开发工具集成

* **Hook 系统**：中文化自动化脚本
* **MCP 服务器支持**：接入 Gemini 咨询、Context7 文档等能力
* **安全扫描**：MCP 调用前自动检查敏感信息
* **通知系统**：关键事件的系统级提醒

### 完整模板库

* 项目模板：覆盖多语言的项目结构
* 文档模板：标准化中文文档
* 配置示例：开箱即用的配置文件

## Claude Code 中文开发套件使用指南

### 快速上手

安装完成后，在任意项目中启动：

```
claude
```

开启中文对话，提示与上下文均已本地化。

### MCP 服务器能力

#### 🧠 Gemini 深度咨询

**触发**：对 Claude 说“咨询 Gemini”或“请 Gemini 分析”
**适用**：复杂架构设计、性能优化、多文件重构、深度技术分析
**发送**：问题描述 + 相关代码文件 + 期望建议类型
**能力**：多方案对比、代码审查与优化、架构最佳实践、跨技术栈经验

#### 📚 Context7 文档查询

**触发**：询问任意开源库的最新用法
**适用**：学习新框架、查询最新 API、处理版本兼容
**发送**：库名（如“React 的最新 hooks 用法”）+ 具体问题
**能力**：获取官方文档、提供代码示例、解释新特性与变更

## 增强功能

### 🎵 自定义通知音效

默认路径：`.claude/hooks/sounds/`，支持 `.mp3/.wav/.aiff`
替换示例：

```
# 任务完成音效
cp your-sound.mp3 .claude/hooks/sounds/complete.mp3
# 输入提示音效
cp your-sound.mp3 .claude/hooks/sounds/input.mp3
```

### 🔒 安全扫描

MCP 调用前自动检查**API 密钥、令牌、密码、敏感配置、个人信息、私有代码片段**等，降低泄露风险。

### 🤖 智能上下文管理（进阶）

新任务自动加载项目文档，智能挑选关联上下文，跨多轮会话保持状态一致。
