**Lively Wallpaper** 是一款开源的 Windows 动态壁纸工具，近日新增音乐壁纸功能，桌面上会增加一个小电视，并实时显示当前播放的音乐信息，支持 SPotify、foobar2000、原生媒体播放器 与 Chrome 等，并且还可以调整桌面小电视的大小与方向。[@Appinn](https://www.appinn.com/lively-wallpaper-music/)

![Lively Wallpaper 音乐壁纸：在桌面小电视上，显示当前播放的音乐[Windows]](https://static1.appinn.com/images/202302/lively-wallpaper-music.jpg!o "Lively Wallpaper 音乐壁纸：在桌面小电视上，显示当前播放的音乐[Windows] 1")

感谢 @[chaso ] 同学的推荐。

小众软件曾经介绍过：[Lively Wallpaper – 为 Windows 创建动态桌面壁纸：视频、网页、流媒体等](https://www.appinn.com/lively-wallpaper-for-windows/)

* <https://www.appinn.com/lively-wallpaper-for-windows/>

近日的更新非常有意思，它新增了一个名为 Music TV（LQ）的壁纸：

![Lively Wallpaper 音乐壁纸：在桌面小电视上，显示当前播放的音乐[Windows] 1](https://static1.appinn.com/images/202302/screen-appinn2023-02-27_17_40_06.jpg!o "Lively Wallpaper 音乐壁纸：在桌面小电视上，显示当前播放的音乐[Windows] 2")

开启后会在桌面上出现一个没信号的小电视，当你使用 Spotify、foobar2000、媒体播放器等播放器，或者 Chrome 浏览器播放音乐/视频的时候，它就能获取到当前的音乐/视频信息，然后显示在小电视上。

更多播放器并未测试，网易云音乐暂不支持。
