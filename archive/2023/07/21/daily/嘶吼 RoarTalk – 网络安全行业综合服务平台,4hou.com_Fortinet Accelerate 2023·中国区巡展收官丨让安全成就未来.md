# Fortinet Accelerate 2023·中国区巡展收官丨让安全成就未来

科技产业资讯
[行业](https://www.4hou.com/category/industry)
2023-07-20 17:42:17

![](https://img.4hou.com/article/%E6%B5%8F%E8%A7%88.png)91268

收藏

导语：7月18日，2023 Fortinet Accelerate Summit在上海成功举办！

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842779174567.png)

7月18日，2023 Fortinet Accelerate Summit在上海成功举办！这亦象征着“Fortinet Accelerate·2023中国区巡展”圆满收官。Fortinet携手来自多个典型行业的百余位代表客户，以及Telstra - PBS 太平洋电信、Tenable等多家生态合作伙伴，共话网安融合大势、共商应对新兴威胁方案。在国际权威咨询机构的数据支持下，网络安全融合这一Fortinet核心理念在数字化转型的大背景下，已经被赋予了行业级别的意义。同时，Fortinet混合部署防火墙等产品和方案的发布再次引领行业发展方向。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842780136350.png)

**平台演进，安全成就未来**

在致辞中，Fortinet北亚区总经理陈鸿翔（Spencer Chen）指出，网络和安全融合已经成为趋势，预计安全组网市场2026年可以达到2800亿。与此同时，企业攻击平面在物联网、混合办公模式等快速发展中，变得愈加广泛。整合多种防御手段应对各种威胁攻击是企业解困的关键。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842780701534.png)

Fortinet北亚区总经理 陈鸿翔

IDC中国企业级研究部副总裁周震刚在《网安融合，数字时代新动能》中表示，数字化转型进入新阶段，推动算力网络向云转移，容器与云原生应用的崛起，以及多云、混合云的普及让云中漏洞随着云数目的增长而增长，跨核心和边缘的云、网和安全的融合成为用户的主要网络议程。这种变化也对网络和安全提出了新的需求，用户体验、云边协同、IT/OT融合、AI驱动等成为新的需求关键词，云时代网安融合技术已经成为主导性技术，其能够带来安全防护更精准、用户使用更便捷、管理运维更简单等优势。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842781198464.png)

IDC中国企业级研究部副总裁 周震刚

Fortinet中国区技术总监张略表示，高级可持续性威胁不断进化，投资于研发与创新已经成为应对最新威胁的关键。当前，边缘无处不在使得整个攻击面的可见性至关重要，但传统手段无法做到全局威胁可见。因此，网络安全融合也促使网络安全平台演进，通过SOC平台实现NOC平台、应用安全平台、端点安全平台等融合，并实现自动化运营，通过生态合作实现威胁情报共享，借助安全运营中心SOC自动化，保护数据、用户和整个社会免受网络犯罪的侵害，用安全成就未来。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842781122579.png)

Fortinet中国区技术总监 张略

**眼手同步，贯通网安通路**

Fortinet北亚区首席技术顾问谭杰进一步指出，网安融合包含融合（Convergence）与整合（Consolidation），融合是把安全能力和安全技术融入到每一个安全环节，整合是采用一致性、易用性、有效性的管理手段，把这样一个网安融合的体系管理起来。同时，Fortinet网安融合的关键还有AI驱动的威胁情报，担当网安融合方案的“大脑”，而网安融合方案则是“眼”与“手”，大脑与眼、手配合就能打通感知威胁、应对威胁、处置威胁的通路。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842782550133.png)

Fortinet北亚区首席技术顾问 谭杰

当前，许多企业正在寻求整合单点产品、整合供应商和功能，以提高其安全性的各种手段。Gartner报告显示到2026年超过60%的组织将会部署不止一种类型的防火墙，这将促使Hybrid Mesh Firewall（混合式部署防火墙，简称HMF）的应用。本届Accelerate大会宣布扩展Fabric安全架构，构建跨五个关键市场的集成解决方案的统一网络安全平台，以实现跨网络、端点和云的高级威胁防御和协调响应，并发布HMF、端点安全、SOC 自动化、威胁情报、身份和访问以及应用程序安全产品和方案。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842782191326.png)

Fortinet 混合式部署防火墙是一款跨企业 IT 多个区域提供协同保护的安全平台，支持硬件设备、虚拟设备、云原生防火墙及防火墙即服务（FWaaS）多种部署模式，可全面保护企业分支机构、园区和数据中心、公有云和私有云及远程办公用户。作为本次大会发布亮点，HMF秉承Fortinet多年来独特而先进的网络融合理念，持续向用户传递HMF理念，赋能混合网络下所有安全组网产品统一管理和分析特性，降低混合网络复杂性。

**融合赋能，全场景落地**

从平台到产品，从功能到场景，网安融合目的是安全赋能。工业互联网、高级持续性威胁攻击、云安全等热门场景愈加复杂，网络和安全融合赋能是守护业务可持续性运营的关键。

专业能力联动，OT场景赋能。数字化转型推动万物互联发展，催生数据互联、柔性制造、数字中台等形态，也使得针对OT网络的攻击呈指数型上升的趋势。Fortinet华东区技术总监卜婵敏在分析了“OT安全模型选择”、“OT设备监控与管理”、“OT安全态势感知”等三个OT安全痛点后，分享了Fortinet OT安全解决方案。该方案具备SAA安全访问架构、FortiNAC实现IoT的持续监控、OT安全态势感知平台、主动安全以及SD-WAN多场景的安全互联等五大OT专业能力。其中SAA安全架构的“横跨模型，从第0层到第5层的微分段架构”专业能力业界唯一。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842783133766.png)

Fortinet华东区技术总监 卜婵敏

全面防御阵法，APT场景赋能。Fortinet中国区首席攻防专家王涛分享了对待APT这种高级持续威胁，Fortinet自己基于AI技术的“全面防御”阵法，涵盖攻防两端。应对APT不是防火墙、杀毒软件等孤立产品和方案能够胜任的，需要结合SOAR、防火墙、EDR、沙箱、邮件安全网关等全面防御手段，覆盖整个攻击的杀伤链，包括攻击者在探测、交付和利用全过程，都有Fortinet相应的产品，来满足安全需求，实现APT场景赋能。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842783131795.png)

Fortinet中国区首席攻防专家 王涛

迈向ZTNA之旅，零信任赋能。ZTNA是未来的发展趋势，但很多企业仍然面临方案落地的挑战，攻击平面暴增、接入形式多样等挑战都让企业落地ZTNA方案时不知从何下手。Fortinet渠道资深安全顾问徐立分享业内最快捷、最方便交付的Fortinet ZTNA解决方案，落地最快速、最简便、最易用、最灵活的ZTNA场景，帮助客户在现有的架构中能够一步一步过渡到各种ZTNA场景里。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842784307149.png)

Fortinet渠道资深安全顾问 徐立

多云环境挑战，分级各个击破。从公有云的挑战角度来看，数据中心从大机、小机，到后面X86虚拟化架构，到现在公有云架构，在不断变迁的同时，也给业务带来了巨大时效性提升，还有成本的降低。同时，不可否认的是也带来了云边界、API、云灾备、多云安全态势统一等来安全性的挑战。Fortinet 渠道资深安全顾问钟屹达针对这四个问题，认为企业应该从宏观（初创企业、中型企业、大型企业）的角度把客户分为三个大类，进行分级各个击破。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842785198524.png)

Fortinet渠道资深安全顾问 钟屹达

**拥抱合作伙伴：网安融合，数字化新动能**

会上，太平洋电信企业发展部高级总监张宇光分享了基于Fortinet产品和方案，提供安全服务和安全实践，打造企业无处不在的安全访问。太平洋电信发挥全球网络覆盖的基础优势，结合Fortinet网络和安全融合服务和方案，共同为客户提供PBS SASE。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842785281145.png)

太平洋电信企业发展部高级总监 张宇光

数字化是一把双刃剑，为企业带来很多好处同时也带来众多安全挑战。Tenable中国区总经理赵阳指出，数字资产快速增长、弹性变化、互联等给黑客带来更多的攻击机会。企业需要在事前、事中、事后做平衡的安全投资，不能只关注在事中防护，需要更全面的了解企业暴露面的全量资产及安全风险，因为你不能保护不知道的资产。Tenable携手Fortinet全方位守护汽车、半导体、食品、能源、高科技等高端制造业客户，成功案例遍布全球。

![](https://img.4hou.com/uploads/ueditor/php/upload/image/20230720/1689842786214076.png)

Tenable中国区总经理 赵阳

当前安全领域需要全面覆盖“云、边、端”全体系，“生态开放”已经成为行业前进基础。Fortinet Security Fabric 安全框架既是 Fortinet 所有产品和方案的基础，也是Fortinet 与上下游安全伙伴完成了互补合作的前提。网络融合与整合也将成为企业和工业 OT 场景、云安全场景等快速发展的安全新动能。

伴随Fortinet Accelerate Summit的落幕，历时2个月、横跨15城市的Accelerate 2023·中国区巡展胜利收官！Fortinet持续贯彻网安融合的不变初心，携手各大合作伙伴，力求通过全方位的安全解决方案及防御生态体系，为广大企业组织打造安全且具备高度韧性的网络环境，护航数字化转型之旅。
