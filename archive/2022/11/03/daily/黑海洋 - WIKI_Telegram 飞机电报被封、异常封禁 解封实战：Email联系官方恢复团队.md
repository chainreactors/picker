12号中午发现之前的小号不见了，一进去说手机号被封禁

因为是纯GV注册，群也没加，就和一个人对话过，居然给封了，于是决定找TG对线

联系该邮箱

|  |  |
| --- | --- |
| Telegram Recovery <recover@telegram.org> | |

|  |  |
| --- | --- |
|  | Dear telegram staff, I found out that I was banned when logging in to TG with my mobile phone number today. I was surprised because my account is basically in a low active state, so I chatted privately with a few friends. Did not send ads, did not join any group, suddenly disappeared, and then logged in and found that it was blocked |
|  | I hope to get help and guidance, I am confused |
|  | My phone number is +1 xxxxx 【US】 |

在13日上午回复

|  |  |
| --- | --- |
|  | Hello, |
|  |  |
|  | The issue with your phone number should be solved now, please check. |
|  |  |
|  | We apologize for all the inconvenience caused. |
|  | Do not hesitate to contact us again if you encounter any similar problem. |
|  |  |
|  | Please contact our support service (Settings - Ask a Question), should |
|  | you have any questions about Telegram. |
|  |  |
|  | Regards, |
|  | Telegram Recovery Team |

然后再去登录发现一切正常，数据已经恢复

之前最后使用记录是11号中午，不清楚什么时候不见的
