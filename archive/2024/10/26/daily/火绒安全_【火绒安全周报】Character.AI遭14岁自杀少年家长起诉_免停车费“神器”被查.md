# Character.AI因14岁少年自杀事件面临诉讼

#

据外媒报道，今年2月美国佛罗里达州14岁少年塞维尔·塞泽在与AI公司Character.AI的聊天机器人长时间对话后自杀。据悉，塞维尔患有焦虑症及破坏性情绪失调症，与AI的长期互动为塞维尔带来了情感上的慰藉，还是加速了他的悲剧，这一点目前尚不可知。塞维尔的母亲指控Character.AI应对塞维尔的死负责，认为其技术“危险且未经测试”，并指出公司收集青少年数据来训练模型，使用“上瘾”的设计功能增加用户参与度。10月23日晚Character.AI对此发表致歉声明，并表示对未成年人模型进行了更改，降低接触敏感内容的可能性。

# 免停车费“神器”被查处

#

近期，执法部门查处了一起大规模非法销售免停车费遥控器的案件，缴获了1.2万套设备。这些设备可在车辆道闸附近对周边信号进行自动搜索扫描以获取车闸控制权限。经过缜密的侦查和研判，民警已成功抓获涉案团伙并捣毁制造窝点2处、仓库3个，扣押涉案手机25部、电脑18台，涉案金额达1000余万元。温馨提示广大车主不要使用此类设备，以免触犯法律！

# Meta公司重启面部识别技术

#

据路透社报道，Meta（原Facebook）公司近日宣布重启面部识别技术，用以打击平台上假冒名人账号进行诈骗的犯罪行为。Meta表示，将招募约50,000名公众人物参与试验，该技术将自动比对Facebook头像与疑似诈骗广告中的图像，如果检测到诈骗，相关广告将被阻止。据悉，Meta曾因非法收集生物识别数据、未能有效制止名人诱饵诈骗等指控被诉讼。在新的试验中，Meta承诺，试验中收集的面部数据将被立即删除，无论是否发现诈骗行为。

# 勒索软件团伙攻击非营利组织，索要百万美金

#

近日，臭名昭著的Rhysida勒索软件团伙攻击了非营利组织Easterseals，并索要130万美元的赎金。Easterseals是一家为残疾儿童、老年人、退伍军人等提供帮助的非营利慈善组织。目前，该组织已向监管机构报告了这一网络攻击事件，并采取了措施，包括聘请网络安全公司和IT专家进行调查。据悉，此次攻击导致14,855人的敏感信息被泄露，包括全名、地址、驾驶执照、社会安全号码、医疗信息等。Easterseals为受害者提供了12个月的身份保护服务，并加强了安全措施，如端点安全软件和多因素身份验证等。

# 卡西欧遭勒索软件攻击导致产品交付延迟

#

日本钟表制造商卡西欧因10月5日遭受勒索软件攻击导致产品交付延迟，预计影响将持续至11月。此次攻击不仅导致卡西欧维修物品交付严重延迟，还造成了数据泄露，泄露的数据包括临时工、合同工的个人信息以及与其关联的公司和业务合作伙伴的员工相关信息。目前，卡西欧已暂停个人产品维修业务，直至系统恢复，且卡西欧发言人表示他们在日本的运输也受到了相应的影响。
