Telegram 新增群组话题功能 Topics in Groups

Telegram 今日更新了一个新功能：Topics in Groups（群组话题），可以为超过 200 人的群组设置不同的话题，有点类似 Discord 的感觉。@[Appinn](https://www.appinn.com/tg-topics-in-groups/)

![Telegram 新增群组话题功能 Topics in Groups 1](https://static1.appinn.com/images/202211/topics-in-groups.jpg!o "Telegram 新增群组话题功能 Topics in Groups 1")

对于超过数万用户的庞大 Telegram 群组，想要在里面正经聊天是非常痛苦的事情，无论是话题，还是消息数量，都对阅读造成了干扰。

Telegram 的 Topics in Groups 功能，目前可以在超过 200 人的群组开通，大概就是这个样子的：

开启 Topics in Groups 需要最新版本客户端，目前各个平台都有更新，但自己的设备上不一定会推送，可能需要应用市场手动更新一下。

之后，就可以在群组中打开 Topics in Groups 功能了：

![Telegram 新增群组话题功能 Topics in Groups 2](https://static1.appinn.com/images/202211/screen-appinn2022-11-06-17-55-32.jpg!o "Telegram 新增群组话题功能 Topics in Groups 2")

注意目前并不能在与频道绑定的群组中开启，比如[小众软件频道群组](https://t.me/appinnfeed/)，暂时还不能开通。不过 [Appinn Talk](https://t.me/appinn) 群组是可以开通的，但是…开了几分钟以后，青小蛙决定关掉它了。

![Telegram 新增群组话题功能 Topics in Groups 3](https://static1.appinn.com/images/202211/screen-appinn2022-11-06-11-28-31.jpg!o "Telegram 新增群组话题功能 Topics in Groups 3")

开启之后原本的群就不能说话了，进群会直接看到所有话题分组，未更新客户端的用户更是完全无法使用群组，相当于炸群 😂 另外需要至少创建一个 Topic，不然没有聊天入口。

每一个 Topic 会有一个单独的链接，类似 https://t.me/appinn/235454

管理员多了一个创建与管理话题权限。以及在关闭 Topics in Groups 功能后，Topics 里的消息实际上还在，并且是以回复 Topics 标题的形式，存在于原群组中，也就是关闭 Topics 消息不会丢，还好。

是的，青小蛙决定等几天看看，话题都想好了，一般情况下就分三个就好了：正经话题、键正话题、瑟瑟话题，我觉得话题数量应该会依次增多，嗯。

最后，有没有哪个群组正式上了话题功能，欢迎留言让我们体验一下。
