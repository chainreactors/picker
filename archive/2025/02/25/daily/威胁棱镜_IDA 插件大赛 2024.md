# IDA 插件大赛 2024

原创

Avenger

威胁棱镜

Hex-rays 每年都会为 IDA 举办插件大赛，该大赛每年都会涌现出各种类型的插件，有安全团队也有个人安全研究员，为了解决各种问题从而开发各种插件。2024 年的插件大赛一共入围了二十款插件。评审团最终评出的前三名为：

* 🥇hrtng
* 🥈aiDAPal
* 🥉idalib Rust bindings

以下是本届插件大赛上二十款插件的简要介绍：

aiDAPal

aiDAPal是一个针对IDA伪代码进行微调的、本地运行的大模型辅助代码分析插件。该插件可以提供对伪代码的解释、变量的重命名等功能，为复杂函数提供易懂的概述表达。受限于硬件条件，该插件目前生成速度较慢，但显示了AI时代逆向分析工作流的巨大潜力。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEdK6kickk9l4nxfhY4a1sgOrS42WWJXvktIslCXwdhYqNJjVgIoCceSw/640?wx_fmt=png&from=appmsg)

```
https://plugins.hex-rays.com/aidapal
```

Assemport

将IDB中的函数单独导出到独立文件的插件，该插件只输出函数的第一个基本块，后续基本块会被忽略。

![图形用户界面, 文本, 应用程序  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEfA2UmTf94d7De1kSyAuib2IY6Bu4ZWoJTHcVhElwpTzF7EpRPytD8zw/640?wx_fmt=png&from=appmsg)

```
https://github.com/Bizarrus/Assemport
```

Community\_base

用户脚本与IDA Python API间的润滑剂，在API发生变更时，社区适应后对外提供统一的接口，用户脚本就不需要再修改。

```
https://plugins.hex-rays.com/Community_base
```

Delphi Helper

Delphi Helper是辅助分析使用Delphi编写的Windows程序文件。它可以解析并格式化RTTI结构并将标准函数重命名、解析已编译的DFM、支持新版Delphi的IDR签名。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLELicVvNicRwAm2vWaJnCJBO59JROnEmehSge0Cwq08dJ8OfBrNZlribnUg/640?wx_fmt=png&from=appmsg)

```
https://github.com/eset/DelphiHelper
```

Graffiti

创建自定义调用图的插件，跟踪文件中各部分的交叉引用和链接跳转。该插件能够与多种IDA和开发人员工具集成使用。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEySiahBUfKlmVcl6fQsxLsoXeofialmHRuGWbQuJicN4rBtHI1IlwXKG6g/640?wx_fmt=png&from=appmsg)

```
https://github.com/yoavst/Graffiti
```

hexhinlay

自动为函数调用参数注解名称，丰富反编译器的输出信息的插件。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEicQx82Pkhd3hBtUlQyhcqTBBs6V4Wl1k7cLMTXfY1kZxIzMIsvAAwCQ/640?wx_fmt=png&from=appmsg)

```
https://github.com/milankovo/hexinlay
```

hrtng

卡巴斯基出品的多功能插件，支持解密、去混淆、库函数识别以及伪代码转换等，每个功能都提供了视频的使用说明。自动控制流平坦化令评委会印象深刻。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEdSnxlb1ctB6oVox2unANr43DiaFbUmPIE9Uoqicc3CJJS9KyP0UiaoD4g/640?wx_fmt=png&from=appmsg)

```
https://github.com/KasperskyLab/hrtng
```

IDA-Notepad-Plus

替换IDA默认的记事本功能，可以为每个IDA窗口或者每个函数创建一个独立的记事本。

![图形用户界面, 文本, 应用程序, 电子邮件  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLESIQPyGhIKpT9waqxs3Nwhicqaxoe9qE9bibFdj3PBpMIwDHjAPAzmqIQ/640?wx_fmt=png&from=appmsg)

```
https://github.com/singleghost2/IDA-Notepad-plus
```

idalib Rust bindings

IDA公共API的Rust接口，支持用户用Rust编写基于IDA SDK的独立工具。通过接口支持查询有关文件的任何信息、生成任意函数的伪代码等功能，对构建批量分析能力很有价值。目前尚未全面覆盖所有IDA的API，但仍然是一个好的起点。

```
https://github.com/binarly-io/idalib
```

IDAscope

IDAScope将NeoVim与IDA集成，从而直接从Neovim有效地利用IDA的功能。支持通过模糊搜索查看伪代码、汇编代码，也支持将反编译代码导出。

```
https://plugins.hex-rays.com/assets/contest2024/idascope-main.zip
```

IDAxLM

2023年插件大赛冠军的新作，能够使用大模型提示的插件。该插件支持多个大模型（OpenAI、Mistral、Groq），通过大模型为函数增加注释或者生成等效代码等。

![图形用户界面, 文本, 应用程序, 电子邮件  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEL2K8XoF7OOJC34QG8iasUD8mbNn14z5dDLuShO4iaFUiaqiaq8olxP929A/640?wx_fmt=png&from=appmsg)

```
https://plugins.hex-rays.com/assets/contest2024/IDAxLM.zip
```

instrlen

instrlen支持更改数据库中指令的有效长度为自定义值，与解码的实际长度无关。这在处理带有模糊代码的文件时非常有用，但目前只能手动处理。

![文本  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLE4uV2oryib8dJp2acmcr9Szq3WpibesaNob7c6RicIvmCeicZiavb0r6qDxQ/640?wx_fmt=png&from=appmsg)

```
https://github.com/milankovo/instrlen
```

LabSync

LabSync用于在分析相同文件的不同用户间同步IDB数据库，同步函数名称等重要信息给所有分析人员。

![图形用户界面, 文本  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEY145ZASpjSLVA2ecB4GYuAg2BOENDBau7DZvqjtKpjcjWpt1CnTenQ/640?wx_fmt=png&from=appmsg)

```
https://github.com/cellebrite-labs/LabSync
```

Mnemocrypt

使用随机森林检测识别32位x86文件中的加密函数，为不同置信度的函数提供不同的颜色标识。

![表格  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEdd4olS3UNbiaTmRcCv4VTibHOcqiajL96V7ibmZVBkcNGVFf9hXu6cOO0A/640?wx_fmt=png&from=appmsg)

```
https://github.com/theneonai/mnemocrypt
```

Navcolor

该插件通过位函数提供不同的着色来增强导航栏，为函数和反汇编情况提供快速导览。

![图形用户界面, 应用程序  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEEodI2gUIu7ia1d3qVMpkGmrAhqBDoCPzVxjSqHStGylDcsLhYBg15Aw/640?wx_fmt=png&from=appmsg)

```
https://github.com/milankovo/navcolor
```

RevEng.AI

AI驱动的二进制分析平台的接口，用户上传二进制文件可以在云端进行识别，返回壳信息、恶意判断和漏洞情况等。

![图形用户界面, 文本  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEK79mJYkVqTeI8QicaJIqia9yz3nzSHaAJmJLCkdEDoeybPuDD69LEhHw/640?wx_fmt=png&from=appmsg)

```
https://plugins.hex-rays.com/assets/contest2024/reai-ida-1.2.0.zip
```

Shannon

分析Shannon基带的插件，大大节省分析人员的时间。

![图形用户界面, 文本  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLE7SmkLibsbURmMJ1VG5xiaHjubxgW0ia52GMwSl2ibJozB6JWiaUBgjAp7gQ/640?wx_fmt=png&from=appmsg)

```
https://github.com/alexander-pick/shannon_modem_loader
```

Unknown Cyber

Unknown-Cyber
-Inc在云上提供了分析样本相似性和生成Yara规则的功能，该插件是该功能的API接口。

![图形用户界面, 网站  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLEfKgXX0GpX9Z140VTMGBjQDOg77iclaDvv3qGMbFKG5ATNRmIFIOVXFw/640?wx_fmt=png&from=appmsg)

```
https://github.com/Unknown-Cyber-Inc/ida
```

Xrefer

该插件用于辅助静态分析，可以导入大量信息也能够与大模型集成。

![文本  AI 生成的内容可能不正确。](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLE8O3KFcaSWbppXcSDpZx3P9h32S40ibbicX5VyEt1b2foykxTRuY2SBYg/640?wx_fmt=png&from=appmsg)

```
https://github.com/mandiant/xrefer
```

YaraVM

用于分析已编译Yara规则的插件。

![](https://mmbiz.qpic.cn/mmbiz_png/dlhiccJOdNYb59hgeNKRadycNlmsmcVLER0CyXEVlwcxt6lzic7cMt9e6VWibQpmmA03PP3hSamO7pkseR8vggk3A/640?wx_fmt=png&from=appmsg)

```
https://github.com/milankovo/YaraVM
```
