# Menlo Security收购Votiro，全面筑牢企业数据安全防线

Menlo Security宣布收购 Votiro，Votiro 是一家专注于内容解除武装与重构（CDR）以及数据丢失防护的数据和文件安全平台。

Menlo Security和 Votiro 携手，将使企业能够在不降低员工工作效率和业务生产力的前提下，实施数据安全最佳实践。

在现代企业中，数据存储在众多位置，包括软件即服务（SaaS）应用程序和多种云存储解决方案中，并且分布在多个云服务提供商（CSP）和私有云中。传统的数据安全工具需要繁琐的设置规则和配置，这让安全团队倍感困扰，抑制了员工的生产力，还使企业面临数据泄露、合规性违规和运营中断等代价高昂的后果。

通过将 Votiro 的数据和文件安全平台纳入Menlo Security浏览器安全解决方案套件，Menlo Security提供了一个全面的工作空间安全解决方案，无论现代员工身在何处，都能保障他们的安全，并适应他们的工作方式。用户通常 80% 以上的工作时间都在通过浏览器访问应用程序。

当文件在企业工作空间内外流动，以及通过浏览器、电子邮件、协作工具和 API 流等多种渠道传输时，Votiro 可解决文件安全问题。扩展后的Menlo Security解决方案提供零信任访问，允许用户安全地访问他们的应用程序，同时防止数据泄露并确保文件传输的安全。该解决方案独特地实现了高安全效能和透明的用户体验。在保障用户安全的同时，不会给他们带来困扰，也无需改变他们的工作方式。

Menlo Security首席执行官Amir Ben-Efraim表示：“收购 Votiro 扩展了我们的产品功能，提供了一种市场上独一无二的工作空间安全解决方案。这巩固了我们的市场领导地位，使企业能够在每一个关键业务工作流程中保护他们的数据，同时让他们能够以现代工作的速度推进业务。Menlo在企业浏览器安全领域领先超过 10 年。现在，我们正在改变工作空间安全的未来。”

Votiro 和Menlo Security有着相同的理念，即提供从底层设计就易于配置、部署和管理的强大安全解决方案。结合Menlo Security的 HEATShield AI，Votiro 的数据检测与响应（DDR）解决方案为数据丢失防护和数据屏蔽带来了管理上的简便性。这两种解决方案中对人工智能的创新性应用，能够检测并防止更广泛的网络钓鱼和恶意软件 / 勒索软件攻击，以及敏感数据的丢失，而无需复杂和静态的规则配置。

Votiro 首席执行官Ravi Srinivasan表示：“Votiro 针对现代工作环境中的数据保护建立了一种独特且高效的方法，这种方法与Menlo Security的技术和理念非常契合。作为值得信赖的行业领导者，Menlo在支持全球数百万用户方面有着出色的业绩记录，是 Votiro 的理想归宿。我期待与Menlo的优秀团队携手合作，为我们的客户、员工和合作伙伴实现平稳过渡，并且相信新篇章将为所有人带来丰硕且富有成效的体验。”

在年度经常性收入（ARR）达到 1 亿美元之后，此次收购拓展了Menlo Security的潜在市场总量，显著增加了收入机会。
