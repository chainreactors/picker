## Microsoft Edge 移除删除云端同步数据的选项

[![Internet Explorer](https://icon.solidot.org/images/topics/topicIE.png?123)](/search?tid=31 "Internet Explorer")

[Wilson](/~Wilson) (42865)发表于 2023年06月10日 20时09分 星期六 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=75212&appkey=1370085986&title=Microsoft%20Edge%20%E7%A7%BB%E9%99%A4%E5%88%A0%E9%99%A4%E4%BA%91%E7%AB%AF%E5%90%8C%E6%AD%A5%E6%95%B0%E6%8D%AE%E7%9A%84%E9%80%89%E9%A1%B9 "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自哈特拉斯船长历险记**

Microsoft Edge 从 v114 开始不再提供“重置同步（reset sync）”——从微软服务器上删除同步数据——的按钮，取而代之的是“重新同步（Re-sync now）”。如果你对隐私很注重，那么现在你将没有办法从微软的云端删除储存的浏览数据。类似 Google Chrome，当用户登陆 Microsoft Edge 时会同步收藏夹、设置历史、密码、打开的标签、扩展等各种类型的数据。用户可通过重置同步解决同步问题，或者删除储存的云端数据。但 v114 删除了这一选项。
https://techdows.com/2023/06/microsoft-edge-removes-reset-sync.html
