Ivanti и§ЈеҶідәҶеҪұе“Қе…¶дә‘жңҚеҠЎи®ҫеӨҮпјҲCSAпјүи§ЈеҶіж–№жЎҲзҡ„дёҖдёӘе…ій”®иә«д»ҪйӘҢиҜҒз»•иҝҮжјҸжҙһпјҢиҜҘжјҸжҙһиў«иҝҪиёӘдёә CVE-2024-11639пјҲCVSS иҜ„еҲҶ 10пјүгҖӮ

иҝңзЁӢжңӘи®ӨиҜҒзҡ„ж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһиҺ·еҫ—з®ЎзҗҶе‘ҳи®ҝй—®жқғйҷҗгҖӮ

иҝҷдёӘжјҸжҙһжҳҜз”ұ CrowdStrike й«ҳзә§з ”з©¶еӣўйҳҹеҸ‘зҺ°зҡ„пјҢеҪұе“Қ 5.0.2 еҸҠд№ӢеүҚзүҲжң¬гҖӮ

вҖңеңЁ Ivanti CSA 5.0.3 д№ӢеүҚзҡ„зүҲжң¬зҡ„з®ЎзҗҶе‘ҳзҪ‘з»ңжҺ§еҲ¶еҸ°дёӯеӯҳеңЁдёҖдёӘиә«д»ҪйӘҢиҜҒз»•иҝҮй—®йўҳпјҢе…Ғи®ёиҝңзЁӢжңӘи®ӨиҜҒзҡ„ж”»еҮ»иҖ…иҺ·еҫ—з®ЎзҗҶе‘ҳи®ҝй—®жқғйҷҗгҖӮвҖқе…¬еҸёеҸ‘еёғзҡ„е…¬е‘ҠдёӯеҶҷйҒ“гҖӮ

иҜҘе…¬еҸёиҝҳдҝ®еӨҚдәҶдёҖдёӘе…ій”®зҡ„ SQL жіЁе…ҘжјҸжҙһпјҢиҜҘжјҸжҙһиў«иҝҪиёӘдёә CVE-2024-11772пјҲCVSS иҜ„еҲҶ 9.1пјүпјҢеҪұе“Қ Ivanti CSA 5.0.3 д№ӢеүҚзүҲжң¬зҡ„з®ЎзҗҶе‘ҳзҪ‘з»ңжҺ§еҲ¶еҸ°гҖӮе…·жңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„иҝңзЁӢи®ӨиҜҒж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһжү§иЎҢд»»ж„Ҹ SQL иҜӯеҸҘгҖӮ

е…¬еҸёи§ЈеҶізҡ„з¬¬дёүдёӘй—®йўҳжҳҜеҸҰдёҖдёӘе…ій”®зҡ„ SQL жіЁе…ҘжјҸжҙһпјҢиў«иҝҪиёӘдёә CVE-2024-11773пјҲCVSS иҜ„еҲҶ 9.1пјүпјҢеҗҢж ·еҪұе“Қ Ivanti CSA 5.0.3 д№ӢеүҚзүҲжң¬зҡ„з®ЎзҗҶе‘ҳзҪ‘з»ңжҺ§еҲ¶еҸ°гҖӮе…·жңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„иҝңзЁӢи®ӨиҜҒж”»еҮ»иҖ…д№ҹеҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһжү§иЎҢд»»ж„Ҹ SQL иҜӯеҸҘгҖӮ

Ivanti еҸ‘еёғдәҶ CSA 5.0.3 зүҲжң¬жқҘи§ЈеҶідёҠиҝ°й—®йўҳпјҢе№¶жҢҮеҮәе…¬еҸёдёҚзҹҘйҒ“жңүеңЁйҮҺеҲ©з”ЁиҝҷдәӣжјҸжҙһзҡ„ж”»еҮ»гҖӮ

вҖңеңЁжҲ‘д»¬е…¬ејҖжҠ«йңІд№ӢеүҚпјҢжІЎжңүе®ўжҲ·иў«иҝҷдәӣжјҸжҙһжүҖеҲ©з”Ёзҡ„ж¶ҲжҒҜгҖӮиҝҷдәӣжјҸжҙһжҳҜйҖҡиҝҮжҲ‘д»¬зҡ„иҙҹиҙЈд»»жҠ«йңІи®ЎеҲ’иў«жҠ«йңІзҡ„гҖӮвҖқе…¬е‘ҠжҖ»з»“йҒ“пјҢвҖңзӣ®еүҚпјҢжІЎжңүе·ІзҹҘзҡ„е…¬ејҖеҲ©з”ЁиҝҷдәӣжјҸжҙһзҡ„жғ…еҶөпјҢд№ҹж— жі•жҸҗдҫӣеҰҘеҚҸжҢҮж ҮгҖӮвҖқ

ж—©еңЁ 10 жңҲеҲқпјҢиҝҷе®¶иҪҜд»¶е…¬еҸёе°ұиӯҰе‘ҠдәҶе…¶дә‘жңҚеҠЎи®ҫеӨҮпјҲCSAпјүдёӯзҡ„дёүдёӘж–°зҡ„е®үе…ЁжјҸжҙһпјҲCVE-2024-9379гҖҒCVE-2024-9380 е’Ң CVE-2024-9381пјүпјҢиҝҷдәӣжјҸжҙһжӯЈеңЁйҮҺеӨ–ж”»еҮ»дёӯиў«з§ҜжһҒеҲ©з”ЁгҖӮ

д»ҘдёӢжҳҜиҝҷдёүдёӘжјҸжҙһзҡ„жҸҸиҝ°пјҡ

* CVE-2024-9379пјҲCVSS иҜ„еҲҶ 6.5пјүвҖ”вҖ” Ivanti CSA 5.0.2 д№ӢеүҚзүҲжң¬зҡ„з®ЎзҗҶе‘ҳзҪ‘з»ңжҺ§еҲ¶еҸ°дёӯзҡ„ SQL жіЁе…ҘгҖӮе…·жңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„иҝңзЁӢи®ӨиҜҒж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһжү§иЎҢд»»ж„Ҹ SQL иҜӯеҸҘгҖӮ
* CVE-2024-9380пјҲCVSS иҜ„еҲҶ 7.2пјүвҖ”вҖ” Ivanti CSA 5.0.2 д№ӢеүҚзүҲжң¬зҡ„з®ЎзҗҶе‘ҳзҪ‘з»ңжҺ§еҲ¶еҸ°дёӯзҡ„ж“ҚдҪңзі»з»ҹе‘Ҫд»ӨжіЁе…ҘжјҸжҙһгҖӮе…·жңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„иҝңзЁӢи®ӨиҜҒж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһе®һзҺ°иҝңзЁӢд»Јз Ғжү§иЎҢгҖӮ
* CVE-2024-9381пјҲCVSS иҜ„еҲҶ 7.2пјүвҖ”вҖ” Ivanti CSA 5.0.2 д№ӢеүҚзҡ„зүҲжң¬дёӯзҡ„и·Ҝеҫ„йҒҚеҺҶй—®йўҳгҖӮе…·жңүз®ЎзҗҶе‘ҳжқғйҷҗзҡ„иҝңзЁӢи®ӨиҜҒж”»еҮ»иҖ…еҸҜд»ҘеҲ©з”ЁиҜҘжјҸжҙһз»•иҝҮйҷҗеҲ¶гҖӮ

еЁҒиғҒиЎҢдёәиҖ…е°ҶиҝҷдәӣдёүдёӘжјҸжҙһдёҺ CSA йӣ¶ж—ҘжјҸжҙһ CVE-2024-8963пјҲCVSS иҜ„еҲҶ 9.4пјүй“ҫжҺҘиө·жқҘпјҢиҜҘжјҸжҙһеңЁ 9 жңҲиў«иҪҜд»¶е…¬еҸёи§ЈеҶігҖӮ

еЁҒиғҒиЎҢдёәиҖ…еҸҜд»ҘеҲ©з”ЁиҝҷдәӣжјҸжҙһеңЁжҳ“еҸ—ж”»еҮ»зҡ„ CSA зҪ‘е…ідёҠжү§иЎҢ SQL жіЁе…Ҙж”»еҮ»гҖҒйҖҡиҝҮе‘Ҫд»ӨжіЁе…Ҙжү§иЎҢд»»ж„Ҹд»Јз ҒпјҢе№¶еҲ©з”Ёи·Ҝеҫ„йҒҚеҺҶејұзӮ№з»•иҝҮе®үе…ЁйҷҗеҲ¶гҖӮ

вҖңжҲ‘д»¬дәҶи§ЈеҲ°жңүйҷҗзҡ„е®ўжҲ·иҝҗиЎҢ CSA 4.6 иЎҘдёҒ 518 еҸҠд№ӢеүҚзҡ„зүҲжң¬пјҢеҪ“ CVE-2024-9379гҖҒCVE-2024-9380 жҲ– CVE-2024-9381 дёҺ CVE-2024-8963 й“ҫжҺҘж—¶пјҢиҝҷдәӣе®ўжҲ·йҒӯеҲ°дәҶеҲ©з”ЁгҖӮвҖқIvanti еҸ‘еёғзҡ„е…¬е‘ҠдёӯеҶҷйҒ“пјҢвҖңжҲ‘д»¬жІЎжңүиҜҒжҚ®иЎЁжҳҺе…¶д»–жјҸжҙһеңЁйҮҺеӨ–иў«еҲ©з”ЁгҖӮиҝҷдәӣжјҸжҙһдёҚеҪұе“Қ Ivanti зҡ„д»»дҪ•е…¶д»–дә§е“ҒжҲ–и§ЈеҶіж–№жЎҲгҖӮвҖқ
