# 冒充学术论文写作高手

原创

沈沉舟

青衣十三楼飞花堂

我不是学术界的，也不是工程界的，按当年某次会议体制内某人不屑鄙视的说法，我们属于社会闲散人员中的一个。不知道这个哏的，可去渣浪问TK。

但我认识几个学术界的，其中有个跟我比较熟，哈，懂？她那些无处倾诉的张狂，只能找我和cby听，cby八卦有余、知识有限，所以我就成了最佳垃圾桶。这位有时候非拉着我听她的科研思路、标书写作、论文逻辑，假装我听得懂，其实只是无处倾诉。

耳濡目染之下，偶尔也会关注学术相关的东西。有天，我注意到一本电子书:

如何成为学术论文写作高手\_针对华人作者的18周技能强化训练\_13874587.pdf

这书看上去是投SCI时的一些技术处理，只有前言、目录是中文的，内容没有翻译，保持在英文状态。从目录看，应该是讲了些方法论的东西。

当时我就想到了她那几个活宝学生，可能从中有所借鉴。每天听她说她的那些娃，可不就这样了么。于是，我发给她一份电子版。她正在临床带教，无响应。晚上见着人了，我就问了问她。

scz:  我给你发的书，看到没
issy: 啊，看到了，我正说要好好翻翻来着
issy: (假装正好要翻)
scz:  不是给你看的，是让你给你学生看看，说不定有用
issy: 啊，这样啊

从这位64K小U盘的脑容量及过往经验判断，如果当时没有转发，过后就会忘记。有个学生之前学R语言时与我有过交道，我就补转了一份给这个学生。用不用得上，另说。

渣浪小号那边我知道有几位网友也是学术界的，TA们可能已是个中好手，但我本着相遇是缘的心态，在渣浪上分享了一个Gofile的临时下载点:

```
https:///gofile.io/d/5BBFGI
```

按我估计，两只手数得过来的下载量。第二天一看，一百三十多的下载，要知道，渣浪小号总共才三千多的关注，发这书的时候仅关注可见，无法转发，属于小众中的小众。这种情况，肯定不是因为有这么多学术界的，而是因为，这拨人有一种病，囤鼠病，TA们见我发个啥，管它沾不沾边，先下为敬，下完之后束之高阁，并不翻看。我咋知道的？之前有次类似的小众电子书分享，有网友直接这么说的呗。这一个个都病得不轻，不好治，估计也治不好。书，还是欲看再下。我很同情这些囤鼠病友们。

虽然我也是非正常人类研究中心被研究对象，也病得不轻，但我没这囤鼠病。就说这本吧，要不是想到那几个活宝娃，即便送到我U盘上，我都立马给删了，跟我挨得着嘛，搞得好像我有生之年会发一区的几十分的SCI似的。

我这里胡乱分享的书，大部分其实在Z、A、L站上就有，不存在奇货可居。没有创造知识，只是传播知识，带着没有愧疚的罪恶。
