## Twitter API 将不再提供免费访问

[![Twitter](https://icon.solidot.org/images/topics/topicTwitter.png?123)](/search?tid=158 "Twitter")

[Wilson](/~Wilson) (42865)发表于 2023年02月02日 16时50分 星期四 [新浪微博分享](//service.weibo.com/share/share.php?url=//www.solidot.org/story?sid=74021&appkey=1370085986&title=Twitter%20API%20%E5%B0%86%E4%B8%8D%E5%86%8D%E6%8F%90%E4%BE%9B%E5%85%8D%E8%B4%B9%E8%AE%BF%E9%97%AE "新浪微博分享")
![](https://icon.solidot.org/images/a7c7.png)

**来自消失吧**

Twitter 宣布从 2 月 9 日开始它的 API（包括 v2 和 v1.1）将不再支持免费访问，使用其 API 将必须付费。这事实上杀死了第三方客户端和利用免费 API 对 Twitter 推文进行研究的项目，以及各种在多平台上交叉发帖的工具。在马斯克(Elon Musk)收购 Twitter 之后，API 访问已经多次出现问题，上个月 Twitter 修改了 API 规则， 明确禁止开发者开发第三方客户端。API 付费只是最后一根稻草。
https://twitter.com/TwitterDev/status/1621026986784337922
