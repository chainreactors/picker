# 奋斗者协议

## 一 风起

这几年，大多数的普通人似乎过得都并不太好。年轻人的种种的压力，都在摧毁着所有的期许。

"这人世间，所有的美好，我都不配拥有"正在成为很多人的内心。

有时我也在想：我在我所处的行业，也算是人五人六的主儿，虽不说混得大富大贵，但起码敢对着不爽的人与事去骂街......可如果让二十多岁的我来到现今，我是否能抗得住这压力？

不好意思，我TM认怂！

为此，希望你能明白奋斗者协议的初衷：

```
你，用嘴，向我们承诺你会好好整！
你，行动，向世界证明你的不甘心！
我，给你，提供一个公平的环境！
我，给你，擦着成本边缘降价！
```

## 二 奋斗者协议

下述内容，你需在农夫大本营报道后三天内，以手写并签字按手印的方式向我们提交。

```
我承诺：
   1 在过往，我没有从事过任何黑灰产或违法犯罪的事项。
   2 以自己最大的精力投入到学习当中。
   3 课余时间，不外出游玩，不玩游戏等。
   4 以诚实守信的原则，去面对未来的求职。
若违反，可随意开除，绝无二话！
                              签字：    手印：
```

迷途的羔羊啊，签了这份魔鬼契约，我们真的会给你降价哦。

学费降至8000/期,低于所有历史学员月平均薪资约20%！

## 三 最后

我参与个红蓝对抗的项目，最多也就是搞来几个内网权限。

我收的学费再贵，人数收再多，也就是多挣几个臭钱。

这哪有把一帮臭鱼烂虾整出去卖个万八儿来的爽啊，尤其是在现今的形势下。
