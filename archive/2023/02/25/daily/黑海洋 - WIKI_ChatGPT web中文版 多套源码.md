最近ChatGPT非常火，具说这个人工智能可以完成很多事情，这里提供有PHP和python等版本

**一、【开源】在Replit上运行Chatgpt，支持多Key轮询**

Replit为免费容器平台，无限流量。

演示地址：https://chatgpt.moeyyteam.repl.co
项目地址：https://replit.com/@moeyyteam/chatgpt

如何使用：先fork，然后修改message.php文件里的key。

**二、ChatGPT修复版源码**

体验地址：https://ai.orange.icu

下载：https://pan.orange.icu/s/8vUb 密码：orange

下载后修改 chatgptApi.php 文件的第四行的$key（把你的内容写进去）即可

**三、开源：ChatGPT连续对话Python Web版，docker部署，在线更新Token**

https://github.com/vastsa/AIChatWeb
有python
pip install fastapi
然后直接启动就能用

Docker一键部署

> AMD
>
> docker run -d --restart=always -p 8000:8000 --name aichat lanol/aichat:latest
>
> ARM
>
> docker run -d --restart=always -p 8000:8000 --name aichat lanol/aichat:arm
