## **1. Hardware**

* **Laptop/PC**: At least 16GB RAM, SSD storage (512GB+), and a modern multi-core processor (e.g., Intel i7/AMD Ryzen 7 or better).
* **External Monitors**: Multiple monitors (dual or triple setup) to manage tasks, tools, and VMs efficiently.
* **External Wi-Fi Adapters**: Compatible with monitor mode and packet injection (e.g., Alfa AWUS036ACH, TP-Link TL-WN722N v1).
* **USB Drives**: Bootable USB drives for OSes like Kali Linux or live USB for emergency scenarios.
* **Portable Battery Banks**: If you’re working in the field or performing Wi-Fi penetration testing, portable power is helpful.

## **2. Primary OS**

* **Host OS**: Ubuntu 22.04 (with your preferred XFCE desktop environment). Ubuntu is stable and customizable, allowing easy setup for hacking tools.
* **VM Management**: VirtualBox or VMware Workstation for running different VMs. It provides isolation for testing and avoids polluting your main OS with malware or dangerous tools.
