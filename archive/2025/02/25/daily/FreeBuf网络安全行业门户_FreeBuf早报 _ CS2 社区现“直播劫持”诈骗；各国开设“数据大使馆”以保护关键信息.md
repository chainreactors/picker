所属地 上海

## 全球网安事件速递

### 1. 3 名跨境电诈首要分子被判无期，最高人民法院发布电信网络诈骗犯罪典型案例

据新华社报道，2月24日，最高人民法院发布电信网络诈骗犯罪典型案例，对 3 名跨境电诈首要分子判处无期徒刑。 【[阅读原文](https://www.ithome.com/0/833/140.htm)】

### 2. CS2 社区现“直播劫持”诈骗：骗取观众加密货币及 Steam 贵重资产

网络安全公司 Bitdefender 的研究人员日前发现，网络犯罪分子正在针对《反恐精英 2》（CS2）社区实施“直播劫持”诈骗。据 Bitdefender 实验室研究员 Ionuț Băltăriu 介绍，这些攻击利用被劫持和伪造的电竞主播账号作为信任载体，导致大量玩家的 Steam 账号被盗、加密货币失窃以及珍贵游戏道具损失。 【[阅读原文](https://www.ithome.com/0/833/073.htm)】

### 3. 朝鲜黑客窃取了价值超百亿元的加密货币，金额创历史新高

Bybit首席执行官Ben Zhou称，黑客控制了公司一个未联网的“冷钱包”，将资金转移至未知地址；区块链情报平台Arkham援引ZachXBT提供的信息称，朝鲜黑客组织拉撒路集团是这起事件的幕后黑手；这是迄今为止最大的加密货币盗窃案，超越了此前Ronin Network和Poly Network的黑客事件，分别损失6.24亿美元和6.11亿美元。 【[阅读原文](https://www.secrss.com/articles/75971)】

### 4. 规模激增614%！AI深度伪造视频引发新型“自骗”攻击浪潮

攻击者通过攻陷的YouTube频道发布合成人物视频，诱导受害者执行恶意PowerShell命令，窃取加密货币钱包和浏览器凭据，甚至完全控制受害者系统。 【[阅读原文](https://www.secrss.com/articles/75874)】

### 5. 澳大利亚因国家安全和间谍问题禁止卡巴斯基软件

继美国之后，澳大利亚已成为最新一个以国家安全为由禁止安装俄罗斯公司卡巴斯基安全软件的国家。 【外刊-[阅读原文](https://thehackernews.com/2025/02/australia-bans-kaspersky-software-over.html)】

### 6. 苹果在英国放弃 iCloud 的高级数据保护功能，以应对加密后门要求

在政府要求通过后门访问加密用户数据后，苹果将从英国立即删除其 iCloud 的高级数据保护 （ADP） 功能。 【外刊-[阅读原文](https://thehackernews.com/2025/02/apple-drops-iclouds-advanced-data.html)】

### 7. 黑客利用 Confluence 服务器漏洞部署 LockBit 勒索软件

已经发现了一种利用关键 Atlassian Confluence 漏洞（CVE-2023-22527、CVSS 10.0）的复杂勒索软件攻击，最终在初始入侵后的两小时内将 LockBit Black 勒索软件部署到企业网络中。【外刊-[阅读原文](https://cybersecuritynews.com/hackers-exploited-confluence-server-to-deploy-lockbit/)】

### 8. Exim 邮件传输漏洞让攻击者注入恶意 SQL 查询

安全研究人员在 Exim 中发现了一个严重的 SQL 注入漏洞 （CVE-2025-26794），Exim 是一种广泛使用的邮件传输代理 （MTA），为超过 60% 的互联网邮件服务器提供支持。 【外刊-[阅读原文](https://cybersecuritynews.com/exim-mail-transfer-vulnerability/)】

### 9. Fluent Bit 0 day 漏洞使数十亿个生产环境面临网络攻击

研究人员在 Fluent Bit 中发现了关键的零日漏洞，Fluent Bit 是一种无处不在的日志记录实用程序，嵌入在 AWS、Google Cloud 和 Microsoft Azure 等主要提供商的云基础设施中。 【外刊-[阅读原文](https://cybersecuritynews.com/fluent-bit-0-day-vulnerabilities-exposes-billions-of-production-environments/)】

### 10. 各国开设“数据大使馆”以保护关键信息

由于担心在境内保护数据安全，越来越多的国家（通常是较小的国家）想到了一个大主意：将其公民的信息冗余托管在另一个地区的“数据大使馆”中，但保留对它的管辖权。 【外刊-[阅读原文](https://www.darkreading.com/cyber-risk/nations-data-embassies-protect-critical-info)】
