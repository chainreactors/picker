微软向 Dev Channel 测试者释出了新版 Windows 11，版本号为 Build 26120.961，它的一大变化是新的开始菜单。在新开始菜单中，微软将它的账号置于其核心位置，让菜单原有的功能日益边缘化。当你点击单击用户名或账户图片，它会移动到菜单的中心位置，凸显与其产品和服务 Microsoft 365、Xbox Game Pass 和 OneDrive，以及账号恢复相关的文字和链接。
