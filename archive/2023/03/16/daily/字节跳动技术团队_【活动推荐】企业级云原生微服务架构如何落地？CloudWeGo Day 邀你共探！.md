#### CloudWeGo Day #2

微服务理念自流行以来，很多企业想要通过构建微服务架构来帮助业务迭代升级。但微服务化后如何进行多个微服务之间的链路设计，如何构建高可靠、高稳定、可扩展、业务亲和性强的一整套微服架构，成了非常多企业的痛点。

CloudWeGo 开源的高性能微服务框架和高性能工具库，能够帮助企业用户降低在服务调用带来的开销，同时解决高并发、稳定性的问题。该项目已经吸引了全球 8 万多用户关注，以及近 2 万 GitHub star。数美科技、贪玩游戏、理想汽车、森马、晶泰科技、欢聚时代等超过20家企业用户均已落地使用。

那么，如何根据不同行业的业务差异性，通过 CloudWeGo 及 CloudWeGo 周边的生态项目，来搭建一整高可用微服务架构呢？2023年3月25日， CloudWeGo 社区携手稀土掘金，邀请了贪玩游戏、数美科技、字节跳动业务部门的一线架构师，为大家分享，在 Java 转 go 场景下、企业高可用性挑战等场景下，如何通过 CloudWeGo 来落地微服务架构。
